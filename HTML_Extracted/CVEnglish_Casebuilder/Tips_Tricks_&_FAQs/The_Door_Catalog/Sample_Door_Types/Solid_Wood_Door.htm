Creating a Solid Wood Door

To create a Solid Wood Door follow these steps:

  1. Select a Category to store the Door on, use the "Slab" category.

  2. Click on the New option in the ribbon.

  3. Name and give a Description for the Door.

  4. Select Multi Piece for the Type.

  5. Click Next.

  6. Select Raised for the Panel Type.

  7. Set the Panel Inset to 0.

  8. Change the Stile and Rail Widths to 0.

  9. Click Finish.

At this point the Door is complete. Just make sure you use a Door Material Schedule with the Door where the Raised Panel Part is using a properly configured board stock material.
