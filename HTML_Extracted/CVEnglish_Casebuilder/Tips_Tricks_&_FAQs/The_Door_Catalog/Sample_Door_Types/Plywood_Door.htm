Creating a Plywood Door

  1. Select a Category to store the Door on, use the "Slab" category.

  2. Click on the New option in the ribbon.

  3. Name and give a Description for the Door.

  4. Select Slab for the Type.

  5. Click Next.

  6. Click Finish.

At this point the Door is complete. Just make sure you use a Door Material Schedule with the Door where the Slab Part is using a properly configured plywood material.
