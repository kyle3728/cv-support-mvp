Part Manager

The Part Manager is relatively straight forward in design and usage. It is primarily responsible for holding information on Parts that you need within Casebuilder. It is relatively complete as it stands, containing the majority of Parts that most shops will use.

Part Tab

Name

The Name is an abbreviated code assigned by Casebuilder to the Part. Although you can change the Part Description, you can never change this Name once it is entered.

Description

The Description of the Part can be changed by changing the text here. The Description of the Part here is the "Name" used throughout Casebuilder including all output such as Cut Lists.

This will change the name of the Part everywhere it is used.

Right Click Options

Right clicking the Column Headers will bring up the Columns window allowing you to hide and rearrange (drag) the Columns displayed.

The Ribbonbar

Return - Closes the Part Manager and returns you to the opening screen.
