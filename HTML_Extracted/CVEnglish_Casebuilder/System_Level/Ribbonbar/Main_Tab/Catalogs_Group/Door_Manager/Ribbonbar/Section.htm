 3D View

In the 3D View you will see a graphic representation of the Door that is similar to the graphic representation that will appear in a Job. This view may be rotated, zoomed, and rendered as if it were in a Job. Also, the Door may be exploded or bent to give you a more detailed view. Following are the sidebar options when in 3D View.

  1. Sample Material - Material - Use this option to specify the Material to view this Door in this 3D View.
  2. Sample Profiles - Outside - Use this option to specify the Outside Edge Profile on this Door in this 3D View. This option will apply only to those Edges defined as Outside in the Door's Properties.
  3. Sample Profiles - Inside - Use this option to specify the Inside Edge Profile on this Door in this 3D View.
  4. Sample Profiles - Panel - Use this option to specify the Panel Edge Profile on this Door in this 3D View.
  5. Sample Profiles - Route - Use this option to specify the Route to assign to a Route Path on this Door in this 3D View.
  6. Sample Profiles - Fingerpull - Use this option to specify the Finger Pull Profile on this Door in this 3D View.This option will apply only to those Edges defined as Finger Pull in the Door's Properties.
  7. Width - This slider allows you to change the visual width the Door is displayed in this 3D View.
  8. Height - This slider allows you to change the visual height the Door is displayed in this 3D View. 
  9. Explode - This slider allows you to incrementally Explode your Door.
  10. Curve - This slider allows you to incrementally Curve/Bend your Door.

 Section

The Section View is is where your Door will be designed by adding Stiles and Rails if required, as well as modifying their size, shape, and joinery.

The idea behind the Door Section Editor is that you would add Stiles and Rails, if that were the type of Door you are working on. And then you would click on each Part (Stiles, Rails, Panel) one at a time and make sure that they are the size you would cut them if you were going to make this sample size Door. CABINET VISION will then use these sample Part sizes to give you the correct cut-list for whatever size Doors your Job requires.

When entering the sample Part sizes you should enter finished Part sizes not rough cut Part sizes. The exception being, that some parts have an "Extra box, so you can add in an extra amount to the Part.

If you are buying your Doors you don’t need to worry about the Parts being the correct size. Just make the Door look like the Door you want.

Panel Type

The first step in the Door Section Editor would be to click in the middle of the Door and select the "Type" of panel for your Door. In the screen below I have selected "Raised Panel" as the type of panel for this Door.

You can also right click in the middle of the Door to select the panel type.

By selecting "Raised Panel" as my panel type, I have performed two functions. First I have told CABINET VISION I want this Panel to be a Raised Panel "Type" Panel. And second I have told CABINET VISION that I want to make this Panel out of whatever Material I selected for "Raised Panel" when I filled out my Door Material Schedules.

There are seven panel types to choose from

  • None - Most commonly used if your Door has no Panel and consists only of Stiles and Rails.

  • Raised - most commonly used for solid lumber.

  • Flat - most commonly used for thin plywood or thin panel stock.

  • Glass - most commonly used for glass.

  • H Slats - most commonly used for tambour Doors with the slats going left and right. If you never use this type of panel you could assign it as cloth or some other Material that you do use.

  • V Slats - most commonly used for tambour door with the slats going up and down. If you never use this type of panel you could assign it as cloth or some other Material that you do use.

  • Slab - most commonly used for thick plywood or thick panel stock.

With the panel highlighted by clicking on it, the Door Properties in the sidebar display the width and height of the panel. Additionally, you may change the amount of inset the Panel has. Panel Inset is that portion of the panel which fits into the Stile and Rail. The values you enter here is the overall inset.

Horizontal Grain

This option is only available when the panel is highlighted. The default value is vertical grain. Selecting Yes in this box will change the panel to horizontal grain.

Panel Height/Width Locking

If your Door has more than one panel and you select one of them, there will be two sidebar properties named Lock Width and Lock Height. These properties allow you to lock the width and/or height of a panel so that when you are in a Job the locked panel will remain the same size when the Door grows or shrinks in width or height.

Once you have checked the part sizes for the sample Door. The next step is to click on the "Profile" tab and select the Profiles you wish to use with this Door.

Stiles & Rails

The second step is to add any necessary Mid-Stiles and/or Mid-Rails. To do this simply select the Panel and use the Split options in the ribbonbar.

The order in which you add the parts determines their joinery. If you add the Rails before the Stiles, the Rails will run long.

If you choose, you can miter the Stiles and Rails by clicking off of the Door and onto the screen enabling the Miter Corner Property in the sidebar.

Section Splits

Following are the Section options in the ribbonbar:

  1. Horizontal Split - Will split the selected panel with a Mid Rail.
  2. Vertical Split - Will split the selected panel with a Mid Stile.
  3. Horizontal Split - Will split the selected panel with Mid Rails creating the number of specified Sections/Panels.
  4. Vertical Multi Split - Will split the selected panel with Mid Stiles creating the number of specified Sections/Panels.
  5. Delete - Allows the deletion of the selected Stile/Rail.

 Part List

Use the Part List are to double check your Part Sizes for the current Door Style. You can select your Inside Edge Profile here because it can, depending upon the Extended setting, impact your part list. When you first enter the Part List window the sample Door Size will be used. Simply change the Door Width and/or Height and click Apply to test other Door Sizes.

Right Click Menu

When right clicking the screen in 3D or Section View you will be presented with View Properties.

When in Section View and you right click a Stile or Mid Stile you will get the Edit option which will take the Part to the CAM Editor.

When in Section View and you right click a Rail or Mid Rail you will get the following possible options:

  1. Edit - Takes the selected Part to the CAM Editor.
  2. Remove Shape - Removes any shape that has been applied to the selected Rail.
  3. Arch - Adds an Arch to the bottom of the selected Rail.
  4. Edit Shape - Take the selected Part to the Shape Editor. Lines that are red can not be edited.

When in Section VIew and you right click on a Panel or Mid Panel you will get the following possible options:

  1. Edit - Takes the selected Part to the CAM Editor.
  2. Lock Width - Locks the Width of the selected Panel.
  3. Lock Height - Locks the Height of the selected Panel.
  4. Panel - Allows you to select the Panel Type for the Panel.
