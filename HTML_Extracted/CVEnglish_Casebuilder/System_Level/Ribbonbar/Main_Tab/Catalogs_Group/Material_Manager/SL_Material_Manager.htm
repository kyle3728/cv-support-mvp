Material Manager

The Material Manager is simply a list of all Materials that could possibly be used in a Casebuilder Job. This list can be separated into Groups and Sub-Groups that you define so that you can find specific items more quickly.

  • ALL Materials must be listed in the Material Manager BEFORE they will be available inside a Job.

  • Make sure that every sheet good or board good that you enter has a valid THICKNESS. The Thickness is the most important characteristic of a Material.

Do not use any characters in the Material name except letters, numbers and a / character in fractions. The use of quotation marks (") and the pound symbol (#) can cause errors in the S2M Center as well as other areas of Solid.
