Pulls

This topic covers all the Properties for the Pulls Material Type.

Material Group

Previously covered in the Banding Material Type topic.

Pull Group

  • Type - The Type of Pull being defined.

  • Horizontal Inset - The Horizontal Position of the Pull (Reference Point of Graphic) on the Door/Drawer Front.

  • Vertical Inset - The Vertical Position of the Pull (Reference Point of Graphic) on the Door/Drawer Front.

Operations Tab

  • Diameter - The Diameter of the hole(s) to mount the Pull.
