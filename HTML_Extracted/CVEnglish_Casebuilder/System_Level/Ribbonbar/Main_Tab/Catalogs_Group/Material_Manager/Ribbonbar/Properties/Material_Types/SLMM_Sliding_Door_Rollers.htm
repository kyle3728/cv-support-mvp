Sliding Door Rollers

This topic covers all the Properties for the Sliding Door Rollers Material Type.

Material Group

Previously covered in the Banding Material Type topic.

Sliding Door Roller Group

  • Type - The Type of Sliding Door Roller being defined.

  • Door Edge Inset - The Mounting Plates location from the Edge of the Door.

  • Door End Inset - The Mounting Plates location from the End of the Door.

  • Mount On Back - Specifies if the Sliding Door Roller mounts to the back of the Door.
