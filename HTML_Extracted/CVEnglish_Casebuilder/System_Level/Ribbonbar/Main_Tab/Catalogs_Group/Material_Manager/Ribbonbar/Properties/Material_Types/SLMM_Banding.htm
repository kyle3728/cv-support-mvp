Banding

This topic covers all the Properties for the Banding Material Type.

Material Group

  • ID - Returns internal ID number for the selected Material. Not available for editing.

  • Name - The name of the Material, this name can be anything as long as it helps you to identify the Material correctly.

Do not use " marks in the Name of the Material as it causes errors in the optimization of a Job. For example, 1/2" Melamine would not be an appropriate name for a Material; use 1/2in Melamine to be safe.

  • Description - The Description of the Material.

  • Type - Returns the Material Type for the selected Material. Not available for editing.

  • Unit of Issue - Allows you to set the Unit in which you buy the Material. There are nine options here:

    • BD FT - Per Board Foot

    • BD M - Per Board Meter

    • Cubic FT - Per Cubic Foot

    • Cubic M - Per Cubic Meter

    • Each - Per Each

    • Pair - Per Pair (Useful for Drawer Guides or other Hardware sold in Pairs)

    • Per FT - Per Foot

    • Per M - Per Meter

    • Sheet - Per Sheet

    • SQ FT - Per Square Foot

    • SQ M - Per Square Meter

Banding Group

  • Length Trim - Allows you to set a extra amount of Banding length to compensate for wasted Length on an edgebander for feeding and end trimming.

  • Thickness - Allows you to set the Thickness of the selected Banding Material. The Thickness is the most important property of a Material because the construction of your Cabinets will depend on the Material Thickness in order to give you a correct cut list.

The Width of banding is determined by the Part it is attached to's Thickness. The overall width of the Banding is not considered, instead, you may wish to enter banding that is 13/16 or 7/8 wide by referencing that fact in the Material Name.

It is permissible to define solid stock as banding in the event you need solid wood edges on your Cabinets, Doors, or Drawer faces. If you do so, do not confuse the Thickness with the Width, the Width will default to the net finished Thickness of the Part it is attached to.
