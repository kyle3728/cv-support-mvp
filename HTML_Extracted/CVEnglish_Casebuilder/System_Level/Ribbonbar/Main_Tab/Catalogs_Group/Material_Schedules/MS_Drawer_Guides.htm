Drawer Guide

The Drawer Guides Material Schedule allows selecting Guides with different properties for different applications in your Assemblies. By choosing this Drawer Guide Material Schedule within a Job you will assign the Drawer Guide listed in the Schedule to the object it is selected for.

Sidebar/Properties

The following Properties are available for Drawer Guides:

  • ID - The ID number for the selected Material Schedule.

  • Name - The Name of the selected Material Schedule.

  • Description - The Description of the selected Material Schedule.

Multi-Length Drawer Guide Schedules

Drawer Guide Length is based off of the Drawer Box Depth. The first step in the process is to verify that Drawer Box Depths are being calculated correctly. Once you have verified that the Depths of the Drawer Boxes are correct then focus on the Drawer Guide Lengths.

Steps to set up a Multi-Length Schedule

First, make sure that the parent (the line that is labeled Drawer Guide) is set to either the "no guide for this size" Material or the shortest guide that you would use in this Material Schedule.

Next, right click the Drawer Guide Category and select Variable Sizing | Add Size.

You will then be prompted to enter the minimum Depth Drawer Box that the new Guide will be used on. In the example above, the first depth is 10. When you click OK, you will have a line under Drawer Guide that says ">= 10." This means that any Drawer Box that is less than 10" deep will use the parent guide and any Drawer Box that is 12" deep or greater will use the >= 12 guide (12" guide).

Then, continue adding new Drawer Box Depths until you have complete coverage for your needs.

Multi-Height Drawer Guide Schedules

Drawer Guide Height is based off of the Drawer Box Height when using System Drawer Box and System Roll Out Construction Methods.

Steps to set up a Multi-Height Schedule

First, make sure you have created your Lengths (above). Each Drawer Guide Height needs to correspond to a Length.

Next, right click the Length breakout you want to add a Height to and select Variable Sizing | Add Size.

Next, enter a Height this Guide is available for. If you enable the "Add this height for each length of this part" option, this New Guide Height will be added to every Guide Length in the Schedule you are in.

Then, continue adding new Drawer Box Heights until you have complete coverage for your needs.
