Assembly Manager

The Assembly Manager is the utility that you will use to "teach" CABINET VISION the manner in which you build your Assemblies. The following Assembly Types can be configured here:

  • Assembly - Includes both Cabinets and Closets
  • Drawer Box - Roll Out - Includes both Drawers and Roll Outs

The answers that you give to the questions in this area will create a set of rules that your Assemblies will follow when they are constructed. If you have a situation where you need a specific Job, Room, or even an Assembly to follow a different set of rules, there are several ways that you can make changes to each of those levels. Therefore, the set of rules that you create in the Assembly Manager should reflect the "normal" manner in which Assemblies of this type are built.

You are not limited to a certain number of different Construction Methods, therefore, you may create as many different Construction Methods as you may need to cover all the types of construction that you use.

Ribbonbar

Following are the options available in the Assembly Manager ribbonbar.

 Return

Closes the Assembly Manager and returns you to the CABINET VISION opening screen.

 New

This option is used to create a new Construction Method.

 Copy

This option is used to copy the current Construction Method.

 Rename

This option is used to rename the current Construction Method.

 Delete

This option is used to delete the current Construction Method..

Assembly Type Selector 

This option is used to select the Assembly Type you want to work with. There are three options here:

Assembly - Used to define Cabinet and Closet Construction Methods.

Drawer Box - Roll Out - Used to define Drawer and Roll Out Construction Methods.

Construction Method Selector 

This option is used to select which of your Construction Methods you want to edit.

 Modify

Used to modify the selected Part/Branch.
