General

The following diagram shows the options available in the General object. Click a specific option in the diagram for more information or continue through this topic for information on all Objects and options.

Roll Out Height Specifications (Standard #3)

Because the Height of the Roll Out cannot be controlled by the opening size, there is only one box to be filled out for the Height of the Roll Out. Enter the most common height for a Roll Out in this area. Then, if a specific Roll Out needs to be a different Height, it can be changed in the Section Editor.

Roll Out Set Back (Standard #117)

Use this option to establish a Set Back dimension to set the Roll Out back from the the Face of the Assembly.

 

Roll Out Front

The following diagram shows the options available in the Roll Out Front object. Click a specific option in the diagram for more information or continue through this topic for information on all Objects and options.

Roll Out Front Banding Definition (Standard #153)

Roll Out Front Laminate (Standard #36)

Specify here whether you Laminate your Roll-Out Front or not.

Roll Out Swoop Shape (Standard #37)

You have two options for the Swoop option:

  • No - Use this option if you do not Swoop your Roll Out Front.

  • Yes - Use this option if you do Swoop your Roll Out Front. Once this option is selected you will see the Edit button, click this button to control the shape of your Roll Out Swoop.

Swoop Part Shaper

See the Shape Editor Topic for more information.

Roll Out Front Scribe (Standard #38)

You have two options for the Front Scribe:

  • Touches Box Front

  • Space - A positive number represents how far in your want the Roll Out Front to be from the front of the Box. A negative number represents how far outside the front of the Box you want the Roll Out Front to be located.

Roll Out Front Top Reference (Standard #41)

You have two options for the Roll Out Front Top Reference:

  • Flush to Roll Out Top

  • Space - A positive value adjusts the Roll Out Front below the top of the Box. A negative value adjusts the Roll Out Front above the top of the Roll Out.

Roll Out Front Face Machining (Standard #131)
