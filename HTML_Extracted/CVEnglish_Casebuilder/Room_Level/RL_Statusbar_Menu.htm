Statusbar Menu

The Statusbar menu gives you quick access to control the status of certain Drawing aids. There are six possible options here:

Unit of Measure

Allows you to control the active Unit of Measure (Inches or Millimeters). The Unit displayed on the button represents the Unit of Measure that is currently active.

Unit Precision

Allows you to choose the Precision to display for the currently displayed Unit of Measure.

You can turn this button On/Off via the Preferences | Units tab.

Reset Ruler

This option allows you to quickly remove any Snap Marks you have placed on your Drawing.

Snap Options

The last three tools allow you to select which Snap mode is active on screen. The options are:

Snap to Marks

This feature allows snapping to a Snap Point set by use of the Ruler Tool. If this button is highlighted in blue it means it is active simply click it again to disable it.

Snap Grid

This option turns Grid Snapping on and off. You will see a grid pattern in the window where you layout Jobs, edit Assemblies, or CAD that you can Snap to. If this button is highlighted in blue it means it is active simply click it again to disable it. Go to Right Click Properties to set the Grid Spacing.

Snap to Points

This is perhaps the most useful tool when using the measure option of the Ruler Tool. Using this snap feature allows precise snapping of points. This option turns on the Dynamic Snapping functionality in Casebuilder. The snap to points appear as small dark dots in the corner of the objects and all parts within the object. These are the snap to points. If these points are present and not desired, reset the snapping mode to OFF or another mode to eliminate them. If this button is highlighted in blue it means it is active simply click it again to disable it.

Snap Tips

  • Use Ruler Points to place complex Peninsula Wall layouts

  • Use Ruler Points for placing Assemblies, Parts, etc.

  • Use Ruler Points to create Architectural Symbols for saving into your Symbol Library

  • Use Ruler Points to draw construction details.

  • Use Ruler Points to edit complex Assembly shapes.

  • All of the CAD Functions in CAD can Snap to Points, Grid or Ruler Points.

  • Use Grid Points to place items on the Grid.

  • Use Points to measure distance on Assemblies, Parts, Walls, etc.

  • Use the Points on an object to place Ruler Points

Build Warnings

This option will display any Build Warnings that are currently active.  Simply click this option to bring up the Message Log to view the active Build Warnings.
