Help Group

Help gives you access to the following options:

Help Topics...

Accesses this Help System. For information on how to use the Solid Help file see the Help topic.

eSupport

Access the eSupport web site. You must have an Internet Connection setup to use the eSupport web site.

Training Information

Access the Training Options for Solid. You must have an Internet Connection setup to view the Training Options.

 Team Viewer

This option allows you to start a Team Viewer desktop sharing session with a Casebuilder Technician. Use this option when instructed by your Casebuilder Technician and provide the displayed ID and Password.
