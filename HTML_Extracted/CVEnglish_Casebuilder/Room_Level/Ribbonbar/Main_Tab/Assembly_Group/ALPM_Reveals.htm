Reveals - (Frameless/32mm Style)

The Reveals Tab at Cabinet Properties has seven options:

  • Top - This controls the Top Reveal value. See image below:

  • Bottom - This controls the Bottom Reveal value. See image below:

  • Left - This controls the Left Reveal value. See image below:

  • Right - This controls the Right Reveal value. See image below:

  • Horizontal - This controls the Horizontal Reveal value. See image below:

  • Vertical - This controls the Vertical Reveal value. See image below:

  • Equalize Door Widths and Drawer Heights - Enabling this option will make all Doors equal in Width and all Drawer fronts equal in Height for the active Assembly.

 

Remember changes made here will only affect the current Assembly. Also, changes made here are attached as permanent (unless you remove them) overrides on the Assembly; which means that if you Save this Assembly into a Catalog these overrides will also be Saved.
