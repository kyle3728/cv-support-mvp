Material

The Materials Tab at Cabinet Properties has the following options:

  • Base Standard Cabinet Material Schedule - Allows you to change the Base Standard Cabinet Assembly Parts Material Schedule for the active Cabinet.

  • Base Exposed Cabinet Material Schedule - Allows you to change the Base Exposed Cabinet Assembly Parts Material Schedule for the active Cabinet.

    • To utilize the Exposed Interior Material Schedule on the active Cabinet the Cabinet must be set to a Finished Interior. To do this, simply go to the Case Tab and enable the Finished Interior option.

  • Upper Standard Cabinet Material Schedule - Allows you to change the Upper Standard Cabinet Assembly Parts Material Schedule for the active Cabinet.

  • Upper Exposed Interior Cabinet Material Schedule - Allows you to change the Upper Exposed Cabinet Assembly Parts Material Schedule for the active Cabinet.

  • Drawer Box - Allows you to change the Drawer Box Material Schedule for the active Assembly.

  • Roll Out - Allows you to change the Roll Out Material Schedule for the active Assembly.

 

Remember changes made here will only affect the current Assembly. Also, changes made here are attached as permanent (unless you remove them) overrides on the Assembly; which means that if you Save this Assembly into a Catalog these overrides will also be Saved.
