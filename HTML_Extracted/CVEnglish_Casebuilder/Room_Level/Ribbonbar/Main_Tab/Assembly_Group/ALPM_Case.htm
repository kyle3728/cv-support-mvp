Case

The Case Tab at Cabinet Properties has three main options:

  • Top - Allows you to change the Properties for the Top of the active Assembly. You have five options:

    • No Top

    • Full Top

    • Stretchers - Make sure you have a valid width dimension for Stretchers in your Construction Method. You do not have to have them as your default but they should have valid dimensions.

    • Web Frame - Make sure you have a valid width dimension for Web Frame members in your Construction Method. You do not have to have them as your default but they should have valid dimensions.

    • Gussets – Make sure you have Gussets configured in your Construction Method. You do not have to have them as your default but they should have valid dimensions.

  • Bottom - Allows you to change the Properties for the Bottom of the active Assembly. You have two options:

    • No Bottom - Removes the Deck/Bottom from the Assembly.

    • Standard - Follows the Construction Method for your Deck/Bottom.

  • Scribes - Allows you to change the Properties for Scribes of the active Assembly. You have seven possible options:

    • Top - Sets the value for the Top scribe of the active Assembly.

    • Bottom - Sets the value for the Bottom scribe of the active Assembly.

    • Left - Sets the value for the Left scribe of the active Assembly. This field is only active if your Construction Method for Unfinished Ends or Finished Ends is set to reference Exterior.

    • Left Inside - Sets the value for the Left Interior Scribe of the active Assembly. This field is only active if your Construction Method for Unfinished Ends or Finished Ends is set to reference Interior.

    • Right - Sets the value for the Right scribe of the active Assembly. This field is only active if your Construction Method for Unfinished Ends or Finished Ends is set to reference Exterior.

    • Right Inside - Sets the value for the Right Interior Scribe of the active Assembly. This field is only active if your Construction Method for Unfinished Ends or Finished Ends is set to reference Interior.

    • Back Scribe - Sets the value for the Back scribe of the active Assembly.

  • Exposed Interior - Selecting this option will instruct Solid to utilize the Exposed Interior Material Schedule selected for this Assembly in the Materials Tab.

 

Remember changes made here will only affect the current Assembly. Also, changes made here are attached as permanent (unless you remove them) overrides on the Assembly; which means that if you Save this Assembly into a Catalog these overrides will also be Saved.
