Frame - (Face Frame/Frame Overlay Style)

The Frame Tab at Cabinet Properties has eight options:

  • Part Widths Top Rail - Allows you to specify the Width of the Top Rail.

  • Part Widths Bottom Rail - Allows you to specify the Width of the Bottom Rail.

  • Part Widths Left Stile - Allows you to specify the Width of the Left Stile.

  • Part Widths Right Stile - Allows you to specify the Width of the Right Stile.

  • Frame Joints Top Left - Allows you to specify the Frame Joint on the Top Left of your Cabinet.
