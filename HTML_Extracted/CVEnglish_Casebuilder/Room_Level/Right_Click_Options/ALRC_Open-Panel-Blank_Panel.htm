Right Click Open/Panel/Blank Panel Section View - Section Face

 

Following are the possible options when right clicking on an Opening, Panel, or a Blank Panel in Section Face view:

  • Split Vertically - Allows you to split the selected Face Opening vertically.

  • Split Horizontally - Allows you to split the selected Face Opening horizontally.

  • Lock Width - Allows you to lock the Width of the selected Face Opening.

  • Equalize Width - Allows you to equalize the Width of the selected Face Opening with another Face Opening(s) you set with this option.

  • Lock Height - Allows you to lock the Height of the selected Face Opening.

  • Equalize Height - Allows you to equalize the Height of the selected Face Opening with another Face Opening(s) you set with this option.
