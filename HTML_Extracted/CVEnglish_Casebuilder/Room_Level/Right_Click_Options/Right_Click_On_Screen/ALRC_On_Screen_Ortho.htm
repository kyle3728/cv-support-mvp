Right Click On Screen -

 

Following are the options when right clicking the screen in a Face, Plan, or End Ortho view:

Save

The reference to an Object in Casebuilder includes Cabinets, and other items that you save to your library. Appliances, Cabinets, Closet Objects, Furniture, etc., are all objects. After creating a special Assembly or other object decide whether this will be a desirable item to have available in future Jobs. If this is a one time only creation, it is not necessary to save it as it will be saved when the Job is saved. However, it will only be available in the Job it is saved with. If it is possible that the item will be required for other Jobs it must be saved back into your library.

Save versus Save As

Right clicking on the screen at the Assembly Level, will provide a menu option similar to the one shown above. Note that there are two options under the "save" feature. It is important to understand the difference between the two options. In the simplest terms, Save keeps the new item in its new form at the expense of loosing the prior item (the pre-edit form). Save As allows for the creation of a new item. In effect, Save As allows for two items when finished, whereas, Save allows for only one. Save overwrites the original item, whereas Save As creates a new item in addition to the original one. Use the Save feature when you want a permanent change to an existing item or Assembly, and the Save As feature when you want the new item in addition to the existing one. Save As will allow you to rename the new item and place it in other libraries you may have made.

Features Common to Save and Save As

The first option allows choosing of a Library by clicking on the Library Symbols in the left window as shown on the above screen capture. Thereafter, if desired, the icon can be changed by typing the selected name in the "Name" box at the top. The Description can be changed to represent your Assembly. An Icon should be chosen from the list at the right that represents the Assembly you are saving. This is done by clicking on the one of your choice.

Preserve Finish

This feature will save the item you are saving to your Library with any Finish overrides you have applied to the Object or its Parts.

Preserve Height

This feature will save the item you are saving to your library at the Height you have edited it to, instead of the default Assembly Height. If you have created an object that is of a different Height than a default Assembly, it is advisable to select this option. You will still be able to stretch or re-size it's Height after you bring it into a new Job regardless of whether you select this option or not.

Preserve Depth

This feature will save the item you are saving to your library at the Depth you have edited it to, instead of the default Assembly Depth. If you have created an object that is of a different Depth than a default Assembly, it is advisable to select this option. You will still be able to stretch or re-size it's Depth after you bring it into a new Job regardless of whether you select this option or not.

The Advanced Button

The Class Tab

The Class tab allows selection of the Class and Type of the object to be saved.

It is a good idea to keep Base Class objects, Wall Class objects, etc within their original class as other areas of Casebuilder identify objects by their Class.

The Placement Tab

There is one area on the Placement Tab:

Stretching

By enabling or disabling the Width, Height, and/or Depth options you can control whether the Object can be re-sized in those dimensions.

The Preserves Tab

Clicking the Advanced button will present menus that allow preservation of other attributes, such as Material or Construction Methods.

Using this feature will effectively lock your Cabinet from further changes in other Jobs, therefore, it should be used sparingly.

Properties

Following are the options in Properties.

Grid Spacing - Controls the size of the grid squares on the screen when using the Grid Snap option from the statusbar menu.

Grid Always Visible - Normally the Grid is only displayed when Grid Snapping is on. Simply enable this option to have the Grid Always display.

Reset Ruler - This removes any snap marks (red Xs) on the view.
