Right Click On Line Item -

Following are the options here:

 

 Add Copy

Will add a Copy of the selected line item to the list.

 Delete

Will delete the selected line item.

 Rehinge

Selecting this option will allow you change the Hinge side for a Assembly, if it is a Single Door Assembly.

 Properties

Following are the Properties options.

General Tab - Here you can specifiy the Name and Description of the Cabinet.

Comment Tab - You add special notes on this Part here.

Overrides Tab - The Overrides Tab will display and allow removal of any Overrides on the selected Cabinet.
