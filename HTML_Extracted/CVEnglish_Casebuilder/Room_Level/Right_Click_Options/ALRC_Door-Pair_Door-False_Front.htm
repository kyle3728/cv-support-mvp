Right Click Door/Pair Door/False Front Section View - Section Face

 

In addition to the standard six right click Opening options you get the Properties option. The Properties option for a Door has 3 possible tabs:

Door Tab

There are 2 possible options on the Door Tab:

  • Type - Will allow you to select the Door level to use on Tall Cabinets.

  • Change - Will bring up the Door Style selector and allow you to change the Door Style for the selected Door.

  • Preserve Material - If the selected Door's Material has been overridden then the Preserve Material option is available. Selecting this will keep the Material from changing to something else "preserving" the Material when the Assembly is Saved to your Catalog.

Hinges Tab

There are 6 options on the Hinges Tab:

  • Change - Will bring up the Hinge selector and allow you to change the Hinge for the selected Door.

  • Minimum Distance from Door Top - Allows you to specify a different distance for the Hinge from the Top of the selected Door. The default value comes from the selected Hinges specifications in the Material Catalog.

  • Minimum Distance from Door Bottom - Allows you to specify a different distance for the Hinge from the Bottom of the selected Door. The default value comes from the selected Hinges specifications in the Material Catalog.

  • Quantity - Allows you to specify the Quantity of Hinges on the selected Door.

  • Create Cup Holes - Allows you to specify whether or not the Hinge Cup and Anchor holes are machined on the selected Door.

  • Create Plate Holes - Allows you to specify whether or not the Hinge Plate Holes are machined on the associated Part for the selected Door.

Pulls Tab

There are 7 options on the Hinges Tab:

  • Change - Will bring up the Pull selector and allow you to change the Pull for the selected Door.

  • Reference - Allows you to specify where the Pulls position is referenced from on the selected Door. (Top, Center, or Bottom)

  • Horizontal - Allows you to specify the Pulls Horizontal Position from the selected Reference on the selected Door.

  • Vertical - Allows you to specify the Pulls Vertical Position from the selected Reference on the selected Door.

  • Quantity - Allows you to specify the Quantity of Pulls on the selected Door. The valid range is between 0 and 3.

  • Rotate Pull - Will Rotate the Pull on the selected Door 90 degrees.

  • Create Holes - Allows you to specify whether or not the Pull holes are machined on the selected Door.

Sub-Frame Tab -

There are 6 options on the Sub-Frame tab:

  • Add Sub-Frame - Turns the selected Door/Opening into a Sub-Frame.

  • Frame Recess - Specifies the amount the Sub-Frame recesses in the selected Opening.

  • Top Rail - Specifies the Width of the Sub-Frames Top Rail.

  • Bottom Rail - Specifies the Width of the Sub-Frames Bottom Rail.

  • Left Stile - Specifies the Width of the Sub-Frames Left Stile.

  • Right Stile - Specifies the Width of the Sub-Frames Right Stile.
