Catalogs

The Custom Cabinets Library will display 4 icons representing 4 Assembly classes, Base, Wall, Tall and Vanity. Select the class icon representing the Assembly class you wish to place, and click on it. The Class will open presenting the members of that Class, as shown in the right hand graphic. Here the Standard Base Cabinet is highlighted. To place this Cabinet in Order Entry, click on the icon representing the Cabinet, and drag the icon from the sidebar into the List View.

           

Right Click Options

When you right-click either on an Object or in an empty space on the sidebar. Each option will include in its description where it would be available.

 

 Add Category - (Right-Click on Item/Right-Click in Empty Space)

This option allows you to add another Category to the list of Objects. You will have the option to enter a label (name) for the Category and select an icon from the list on the right. Click OK to complete the process.

 Edit Item - (Right-Click on Item)

This option allows the user to edit the label and icon for the currently selected item.

If the item is an Assembly, the Edit Item option will open the Save Object dialog box for the Assembly.

 Move Item

This option allows the user to change the position of this item in the list.

 Delete Item

This option will remove the item from the list of objects.

 Add Accessory (Right-Click on Item/Right-Click in Empty Space)

This option allows the user to add a Non-Graphic Accessory (NGA) to the list of Objects. The user can then drag that Accessory out of the Object Catalog and add it to the Room or to another Object. When the user creates a new Accessory in this manner, they will see the window displayed below. In this window, the user may enter values for all of the parameters shown.
