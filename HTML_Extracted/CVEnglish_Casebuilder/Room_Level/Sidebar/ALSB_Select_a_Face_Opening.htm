Select a Face Opening Section View - Section Face

When selecting a Face Opening in the Section Editor you are presented with the following sidebar:

You can, if you select one of the Face Openings within an Assembly, control four different Dimensional properties as well as seven Opening Types:

Dimensional Properties

  • Width - Allows you to change the Width of the selected Opening.

  • Height - Allows you to change the Height of the selected Opening.

  • Lock Width - Allows you to lock the Width of the selected Opening.

  • Lock Height - Allows you to lock the Height of the selected Opening.

Opening Types

  • Door - Selecting this option will change the selected Face Opening to a Door and present you with the following seven Side Bar options:

    1. Door Hinge - Lets you choose which side of the Door to put the Hinge on. (None, Left, Right, Top, or Bottom)

    2. Door Mullions/Lites - Lets you choose how many Mullions/Lites the Door needs.

    3. Adjustments Inset - Allows you to specify an Inset Adjustment for the Door in to or away from the Face (Front to Back).

    4. Overlay Left - Allows you to adjust the Width of the Door to or from the Left.

    5. Overlay Right - Allows you to adjust the Width of the Door to or from the Right.

    6. Overlay Top - Allows you to adjust the Height of the Door to or from the Top.

    7. Overlay Bottom - Allows you to adjust the Height of the Door to or from the Bottom.

  • Pair Door - Selecting this option will change the Face Opening to a Pair of Doors and has the same Properties as the single Door above with the exception that Door Hinge is replaced with Door Pair Gap. Door Pair Gap allows you to control the Vertical dimension between the Pair of Doors.

  • Slide Doors - Selecting this option will change the Face Opening to Sliding Doors and presents you with the following three sidebar options:

    1. Doors Qty - Allows you to set the Quantity of Sliding Doors. The default value comes from your Construction Method.

    2. Doors Overlap - Allows you to set the middle Overlap value for Sliding Doors. The default value comes from your Construction Method.

    3. Left Door on Front - Allows you to set whether the Left Door is on the Front or not. The default value comes from your Construction Method.

  • Drawer - Selecting this option will change the selected Face Opening to a Drawer and present you with the following eleven sidebar options:

    1. Front Hinge - Lets you choose which side of the Drawer Front to put the Hinge on (None, Left, Right, Top, or Bottom). For a Drawer, of course, the default value is None.

    2. Adjustments Inset - Allows you to specify an Inset Adjustment for the Drawer in to or away from the Face (Front to Back).

    3. Drawer Box Depth - Allows you to specify the Drawer Boxes Depth. You can type a value here to override your standard Depth settings or select a Standard Depth from the drop down menu. If you are using a Standard Drawer your Standard Depths will come from the Drawer Wizard. If you are using a System Drawer your Depths will come from the Drawer Guide Material Schedule selected for the current Assembly.

    4. Clearances Top - Allows you to specify the Drawer Boxes Top Clearance.

    5. Clearances Bottom - Allows you to specify the Drawer Boxes Bottom Clearance.

    6. Clearances Left Side - Allows you to specify the Drawer Boxes Left Side Clearance.

    7. Clearances Right Side - Allows you to specify the Drawer Boxes Right Side Clearance.

    8. Overlay Left - Allows you to adjust the Width of the Drawer Front to or from the Left.

    9. Overlay Right - Allows you to adjust the Width of the Drawer Front to or from the Right.

    10. Overlay Top - Allows you to adjust the Height of the Drawer Front to or from the Top.

    11. Overlay Bottom - Allows you to adjust the Height of the Drawer Front to or from the Bottom.

  • False Front - Selecting this option will change the selected Face Opening to a False Front and present you with the following six previously covered Side Bar options.

  • Open - Has the Standard Face Opening options.

  • Panel and Blank Panel - Selecting either of these options will change the selected Face Opening to a Panel or Blank Panel and present you with the following five previously covered sidebar options.

When the "Panel" Face Opening type is selected Solid places a Panel that represents the current Door Style for the Assembly. When the "Blank Panel" Face Opening type is selected, Solid places a Slab of Material as defined by your Panel part in the Assembly Parts Material Schedule.
