Select a Horizontal Split Section View - Section Interior

When selecting an Object placed utilizing the Horizontal Split option in a Case Opening in the Section Editor you are presented with the following sidebar:

There are 9 possible Horizontal Part Types:

  • Shelf - Selecting Shelf will change the selected Part to a Shelf and present you with the following four sidebar options:

    1. Position Relative - Lets you choose which side of the Opening the Shelf is oriented to. (Float, Top, Center, Bottom, or Default)

    2. Adjustments Depth - Allows you to adjust the Depth of the Shelf to or from the Cases Face. (Default value Controlled by Construction Method)

    3. Adjustments Left - Allows you to adjust the Width of the Shelf to or from the Left. (Default value Controlled by Construction Method)

    4. Adjustments Right - Allows you to adjust the Width of the Shelf to or from the Left. (Default value Controlled by Construction Method)

  • Adjustable Shelf - Selecting Adjustable Shelf will change the selected Part to an Adjustable Shelf and present you with the following four previously covered sidebar options. In addition you get the option to establish whether the Adjustable Shelf aligns with the Shelf holes (Line Boring).

  • Glass Adj Shelf - Selecting Glass Adj Shelf will change the selected Part to a Glass Adjustable Shelf Part (Which will follow the Adjustable Shelf Construction Method Properties) and present you with the following four previously covered sidebar options. In addition you get the option to establish whether the Glass Adj Shelf aligns with the Shelf holes (Line Boring).

  • Fixed Shelf - Selecting Fixed Shelf will change the selected Part to a Fixed Shelf and present you with the following four previously covered sidebar options.

  • Roll Out Shelf - Selecting Roll Out Shelf will change the selected Part to a Roll Out Shelf and present you with the following four sidebar options:

    1. Position Relative - Previously covered.

    2. Float Shelf/Holes - Determines if the object must align to Shelf Holes (Linebore).

    3. Adjustments Depth - Previously Covered. (Default value Controlled by Shelf Construction Method)

    4. Adjustments Left - Previously Covered. (Default value Controlled by the selected Drawer Guide’s Side Clearance.)

    5. Adjustments Right - Previously Covered. (Default value Controlled by the selected Drawer Guide’s Side Clearance.)

  • Roll Out Tray - Selecting Roll Out Tray will change the selected Part to a Roll Out Tray and present you with the following 6 sidebar options:

    1. Position Relative - Previously covered.

    2. Float Shelf/Holes - Previously covered.

    3. Adjustments Depth - Previously Covered. (Default value Controlled by Roll-out Construction Method Back Clearance)

    4. Adjustments Left - Previously Covered. (Default value Controlled by the selected Drawer Guide’s Side Clearance.)

    5. Adjustments Right - Previously Covered. (Default value Controlled by the selected Drawer Guide’s Side Clearance.)

    6. Adjustments Set Back - Allows you to adjust the amount that the Roll Out Tray sets back from the front of the face. (Default value is 0)

  • Drawer Stretcher - Selecting Drawer Stretcher will change the selected Part to a Drawer Stretcher and present you with the following four previously covered sidebar options.

  • Top - Selecting Top will change the selected Part to a Top and present you with the following four previously covered sidebar options.

  • Bottom - Selecting Bottom will change the selected Part to a Bottom and present you with the following four previously covered sidebar options.
