Keyboard Shortcuts

The following hot keys help in getting to different features quickly:

- Backs up a level. The same action as clicking the Return option in any toolbar/ribbonbar.

The following shortcut keys allow you to select and copy one or more Assemblies:

+ Left Mouse Click  - Selects Multiple Assemblies

+  - Cuts Selected Assemblies

+ - Copies Selected Assemblies

+ - Pastes Copied Assemblies

 

The following hot keys assist you in the Ortho areas:

 - Refreshes the View  (This key can be changed at the Preferences | Tools Tab)

- Snaps Object to Grid (This key can be changed at the Preferences | Tools Tab)

- Snaps Object to Point (This key can be changed at the Preferences | Tools Tab)

- Placing your cursor over an item while pressing the spacebar quickly centers the View on the cursor.
