Working In List Views

There are a couple of important features to note when working in List Views throughout Casebuilder.

Selection Copy

When in a List View is possible to copy text from one cell into multiple cells by grabbing the origin cells copy handle and dragging it up or down into other cells. See the example illustrated below.

Here I have several sheets of Poplar plywood where I would like their Descriptions to all be Poplar. Normally I would need to type the Description for each or use normal Windows Copy and Paste functionality.

Using the new functionality i can hover near the bottom right corner of the origin field and will get the copy handle as shown below.

Then simply drag to the last cell I want to copy to as shown below.

Then simply let go of the click and you will be prompted to accept the changes.

Click Yes and you will see the change.

Cells which must be unique will not allow this functionality.

Cell Hover Hints

When hovering on a cell which includes a button for selection you will see an outline of the button. Clicking on that outline will activate the button and clicking outside that outline will activate the field. Notice below I am hovering over the first Unit of Issue field in the illustration below.
