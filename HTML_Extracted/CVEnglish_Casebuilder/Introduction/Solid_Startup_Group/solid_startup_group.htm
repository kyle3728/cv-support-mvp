Casebuilder Start Group

In Windows 10 click the Windows Start Button then select All Apps then scroll to the Casebuilder Group.

 

In previous version of Windows go to Start | All Programs | Casebuilder

 

Following are options in the Startup Group:

  1. Backup Casebuilder Settings - Allows the creation of a backup of the Casebuilder Database Files.

  1. Casebuilder - Launches Casebuilder.

  2. Help - Opens this Help System.

  3. Restore Settings - Allows the Restoration of backups created by using the Casebuilder Backup Utility or Automatic Backups that are created automatically during Installations.

  1. System Information - A utility utilized by Technical Support to diagnose, troubleshoot, and repair software settings.

  2. Update Version - Launches the Update Version Utility which compacts and repairs Casebuilder Database Files.
