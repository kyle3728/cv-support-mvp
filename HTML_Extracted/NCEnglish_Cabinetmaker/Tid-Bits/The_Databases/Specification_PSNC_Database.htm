The PSNC.mdb Database

This is the Master Database for the S2M Center.

The following is the breakdown of the tables and their fields:

CommonLineCutting

Set part spacing to outline tool diameter and allow for a common cut between two parts

CommonLineMaxRetrace

Set the maximum distance a common line path can retrace over an already cut path.  A larger value will reduce the number of rapids in the outline paths

CommonLineRetraceBySmallParts

 

A boolean value set true to allow retracing around small parts.  Setting to false will increase the number of rapid moves in the output but will reduce the chance of nicking a small part that has moved

A boolean value set true to favor part placement to optimize common line cutting over material utilization

Optimize drill sequence to minimize the number of spindle configuration changes instead of the shortest path between holes

A boolean value when set true will force an interior open path to be milled centerline.  If false additional moves will be added above material to set comp prior to the cut.
