Help

Help gives you access to the following options:

Accesses this Help System. For information on how to use the S2M Center Help file see the Using Help topic.

Access the eSupport web site. You must have an Internet Connection setup to use the eSupport web site.

Access the Training Options for the S2M Center. You must have an Internet Connection setup to view the Training Options.

This option allows you to start a Team Viewer desktop sharing session with a Hexagon Technician. Use this option when instructed by your Hexagon Technician and provide the displayed ID and Password.
