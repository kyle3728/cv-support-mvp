View Mode

There are two options in View Mode:

Part View

This is the Default View when only outputting to a Secondary Machine. The Part View displays the Materials and Parts list in the sidebar and the selected Part and it's machining in the Pattern/Part Display Area.

Right Click Menu

     

Properties - Clicking on the Properties command brings up the following window.

General Tab

There are two main options on the General Tab:

      1. You can control whether Operations are visible on the Pattern printouts.

      2. You can, via the slide bar, control the speed of the CNC Player when using Show Tool Path in the Patterns section of the S2M Center.

Annotations Tab

On the Annotations Tab there are seven main options:

      1. Font - You can control the displayed Font.

      2. Dimensions Horizontal Offset - You can control the Horizontal Offset of your Dimensions.

      3. Dimensions Vertical Offset - You can control the Vertical Offset of your Dimensions.

      4. Dimensions Text Height - You can control the Text Height in your Dimensions.

      5. Dimensions Tick Height - You can control the Tick Height in your Dimensions.

      6. Annotations Heading Text Height - You can control the Text Height in the Heading Annotations.

      7. Annotations Part Labels Text Height - You can control the Text Height in your Part Labels (Labels here refers to the Text marking your Part on the Sheet not the Label Reports you Print.).

Pattern View

This is the Default View when Optimizing or Nesting. The Pattern View displays the Patterns list in the sidebar and the selected Pattern and it's machining in the Pattern/Part Display Area.

This area displays your Sheet/Pattern layouts.

The lower Section is the Part List area.  The Parts on the active sheet are listed in the lower section of the screen. There are several columns of information which you can sort the list by clicking on the column heading.

Right Click Menu

     

General Tab

Following are the options on the General Tab:

      1. Dimension Style - You can control whether the output Pattern printouts Dimension Style is "Width x Length" or "Length x Width".

      2. Dimension Type - You can control if you want Dimensions to display on the Parts or in a Legend on the side of your printouts.
