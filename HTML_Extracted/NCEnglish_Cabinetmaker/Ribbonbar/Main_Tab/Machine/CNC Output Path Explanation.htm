CNC Output Path Explanation

There are 3 different types of output paths you can produce with Intelli-CAM. They come from a combination of the tool catalog (Centerline vs. Offset Path) and the machine catalog (Tool Center vs. Tool Radius)

  1. Centerline - Tool Center NO G41/G42

  2. Centerline - Tool Center G41/G42

  3. Offset - Tool Radius

CENTERLINE - TOOL CENTER NO G41/G42:

The outline path of the part is larger than the part by the tool radius.  This path is to be followed by the center of the tool.

CENTERLINE - TOOL CENTER G41/G42:

The outline path is identical to Centerline - Tool Center NO G41/G42.  The difference is it triggers to look up tool compensation values on the machines control.

Tool comp is important when you start sharpening tools.  When you sharpen a tool it gets smaller.  Tool comp allows you to inform the control that the bit has changed size and to compensate for the change.

For example: If you use a ½” router bit (.25 radius) and as a result of the bit being sharpened it has a new size (.23 radius).  You need to account for this tool size change somewhere.

>> With Centerline - Tool Center NO G41/G42 you would go back to the Tool Catalog in the S2M Center and enter the new size of the tool (.46 diameter) and re-output the program.  Otherwise your part would cut out larger than it should be.

>> With Centerline - Tool Center G41/G42 you can enter a value of .02 at the control (.25 - .23 = .02) and the control will offset the tool path by .02 and correctly cut out the part.

It boils down to user preference whether the sharpened tool size is accounted for in the S2M Center or at the machine although you have advantages at the machine since if you need to swap bits you can compensate for different sizes on the fly rather than having to re-output from the S2M Center.

OFFSET - TOOL RADIUS:

This path follows the outline of the part exactly.  Thus if the router were to follow the path centerline, the part would be too small.  This path must be compensated for at the control.  If you are using a 1/2 bit, the control will need an offset of .25 (radius).  If you sharpen the bit you would need to change the comp value at the control to (.23).  In either case the control will offset the path of the tool by the tool radius resulting in the part being milled to the correct size.
