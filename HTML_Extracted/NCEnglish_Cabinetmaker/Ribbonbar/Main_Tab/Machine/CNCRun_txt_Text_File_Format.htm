CncRunXXX.txt Text File Format

The CNCRun.txt file is placed in the Output directory for Primary and Secondary output and displays information about what was output. Following is the format specification for the file:

^Job         : Job Name

^Description : Job Description

^Date        : Monday, Jun 26 2000

^Parts       : 1

r01b0001.dxf,Bottom(BO),304.00,889.00,19.05,3/4 2s White Melamine,5,1,0,B,0,

Field 1 - Program Name

Field 2 - Part Name (Internal Part Name)

Field 3- Part Width

Field 4- Part Length

Field 5- Material Thickness

Field 6- Material Name

Field 7- Cabinet Number

Field 8- Part Qty

Field 9 - Material Grain (0 – False, 1 – True)

Field 10- Program Face (F – Front, B – Back)

Field 11- Material Face-Dependent (0 – False, 1 – True)

Field 12 - Machine Tag Code

Field 13 - Machine Name
