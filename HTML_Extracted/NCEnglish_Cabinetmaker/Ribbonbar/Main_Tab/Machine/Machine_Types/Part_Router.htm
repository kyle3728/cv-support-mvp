Part Router

This Machine Type is specifically for Part Routers. Following are the Setup Tab options for Part Routers:

Setup Tab - Part Router

Intelli-CAM - Rapid Group

See Sheet Router

Intelli-CAM - File Info Group

See Sheet Router

Intelli-CAM - Tool Movement Group

  • No Sloping Lead In/Outs - With this option enabled Sloping Lead Ins and Outs will not be generated.

See Sheet Router for other settings

Intelli-CAM - Sheet Reference Group

See Sheet Router

Intelli-CAM - Offset Group

  • Offset by 1/2 EDX/EDY - This option is important when using EDX/EDY to oversize a Part and the Part Router is setup to Route the full outline. Use this option to make sure the oversized amount is compensated for.

Intelli-CAM - Banding Operation

  • Banding Tool - Select the Tool that applies (represents) the Banding Operation. You post-processor must be setup properly in order to use this option.

See Sheet Router for other options
