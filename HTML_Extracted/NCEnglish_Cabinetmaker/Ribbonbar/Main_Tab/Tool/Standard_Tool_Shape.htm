Standard Tool Shape

The Tool Shape toolbar has the following options:

Add Points

Use this tool to Add new Points to your Tool's Shape.

Delete Points

Use this tool to remove Points.

Move Points

Use this tool to Move Points.

Parallel Line

Use this Tool to move parallel points at the same time.

Arc Line

Use this tool to pull an Arc.

Radius Point

Use this tool to enter a radius value for a selected point.

Reference Point

Reset the reference point on the object. This changes the Reference Point temporarily during editing.

Show Grid

This tool allows you to turn on/off the display of the Grid.

Snap to Grid

This tool turns on the option to Snap to the Grid lines.

Zoom In

This tool allows you to Zoom In.

Zoom out

This tool incrementally Zooms Out your image.

Import DXF

Use this tool to browse to and Import a DXF of the Tool shape.

Prior to Import you can, if needed, Mirror, Rotate, and/or Scale the DXF.

Grid Unit Box

Allows for resetting the pattern size/snap precision of the Grid.

Right Clicking on any point will allow precision placement of the selected Point in either Absolute Dimensions or from the current Reference (Green Dot) Point. The Absolute Position is referenced from the Cross Point of the Main Grid.
