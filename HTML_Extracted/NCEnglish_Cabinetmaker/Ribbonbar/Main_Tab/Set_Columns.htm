Set Columns

The Set Columns option is available in the Part and Material List area as well as the Pattern View.

Set Columns - Parts and Material List Area

The Set Columns option allows you to change which Columns are displayed in the Part List area.

The Show/Hide Columns option also allows you to change which Columns are displayed in the Material List area.

The Show/Hide Columns option also allows you to change which Columns are displayed in the Pattern View.

In addition to controlling which items display you can also control the order in which the columns appear. Simply click and drag the item you want to move into its new position.
