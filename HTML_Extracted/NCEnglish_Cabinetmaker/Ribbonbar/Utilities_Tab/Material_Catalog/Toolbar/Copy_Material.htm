Copy Material

When clicking this tool it will create a copy of the selected Material and place it directly below it in the Material List as shown in the figure below.

Once the copied Material has been added to the Material List simply change its Properties in the Material Properties section.

Note that the copied Material's type is S2M Center. This means that it is separated from your Cabinet Vision or Cabnetware Material Database and will not be available there. If you wanted this new Material available in both the S2M Center and your Design/Manufacturing product you would want to create it in your Design/Manufacturing product and Import it using the Import Materials Tool in the S2M Center.
