Material Properties

Let's take a look at the Material Properties area.

The first thing you should make note of is that the Product the Material was imported is notated in the header.

There are four sections in the Material option section:

  • Product Properties - There are five options here:

    • Name - Enter a unique name to identify the Material here.

    • Description - Enter a Material description here.

    • Type - Allows you to set the Materials Type.

    • Grain - Select Yes/No to choose whether or not to limit part rotation on the Material due to grain.

    • Cost - Enter the Cost of this Material at the specified size here.

  • Size - There are three options here:

    • Width - Enter the Materials sheet Width here.

    • Length - Enter the Materials sheet Length here.

    • Thick - Enter the Materials Thickness here.

  • Face Properties - There are five options here:

    • Face Surface - Description of the Materials Face Surface.

    • Back Surface - Description of the Materials Back Surface.

    • Edge Surface - Description of the Materials Edge Surface.

    • Clean Face - If Yes, the S2M Center will automatically select the proper Tool to minimize the blowout on the Material's Face.

    • Clean Back - If Yes, the S2M Center will automatically select the proper Tool to minimize the blowout on the Material's Back.

  • Tool Movement - There is one option here:

    • Climb Cut - Use this option to specify whether to utilize a Climb or Conventional Cut. Typically, when cutting Hardwoods you will receive the cleanest edge utilizing a Climb Cut. When cutting Plywoods you will receive the cleanest edge utilizing a Conventional Cut.
