Preferences

There are 5 tabs in the Preferences screen:

General Tab

There are three sections at the General Tab:

  • Show Parts Not Set for Optimization or Output - This option allows you to turn on the capability to control whether a Part set to "Not Optimize" or "Not CNC" can be overridden. Doing this will turn on Right Click menu options in the Part List. See the Parts topic for more information.

  • One Click Optimize To Pattern Results - Enable this option to go straight from clicking the Optimize button to the Pattern area.

  • Use Radial Menus

Tool Warnings Tab

This option allows you to turn off Warnings that may display during Nesting. Simply de-select the Check Boxes for the Warnings you do not want displayed.

Geometry

Following are options on the Geometry Tab:

  • Extend Dado Lengths by Tool Diameter - This option will extend any rectangle path Dadoes on your Parts by the Tool Diameter.

  • Extend Single Pass Dado Lengths by Tool Diameter -  This option will extend any single pass Dadoes on your Parts by the Tool Diameter.

  • Combine Edge Operations with Part Outline - This option will combine any Routes that cut through the Part and touch one of the Part edges together into a single Part Outline cut.

  • Combine Boring to Through Boring - This option will combine any Boring operations that originate from both the Front and Back face that meet or cross in depth and are the same diameter and in the exact same location. This can reduce the number of Secondary programs necessary to generate through bores.

  • Mill Operations from the Unfinished Face Where Possible - This option will machine through Operations from the Unfinished Face.

  • Ignore Operations Outside of Part's Nest Boundary - This option, if enabled, will only machine Operations that fall within the Parts Outline. This does not include Operations which overlap the Outline to be combined into the Outline.

  • Pre-Mill Edgebanding - The S2M Center will compensate for Edgebanders with Pre-Mill Stations; by selecting this option you are indicating to the S2M Center that your Edgebander has a Pre-Mill Station. If Max is selected then the value is the maximum amount the banding machine will trim off the edge of the part during the banding operation.  Typically the banding machine trims off a value equal to the banding thickness. If Fixed is selected this is the amount the banding machine will trim off of the edge of the part that include banding.

Modifying the Max pre-mill edge banding will dynamically change the part size of banded parts.

What Edges can be Pre-Milled / Pre-Banded

Pre-Banded edges can only be the Left, Right, Top & Bottom of the part while it is still a rectangle.  Even if the part is shaped and banded on all edges only the L, R, T & Bottom edges will show the part size extended as pre-banded since the part would be banded prior to being placed on the P2P / Router to be shaped.

Pre-Milled edges can be any pre-banded edge as well as any other edge that could be fed into a banding machine. Edges that are excluded would be arcs or any edge that couldn’t be fed into the machine due to a collision with another edge of the part.  The part will be resized with only valid pre-mill edges extended.

  • Soft Corner Radius - Sets the Soft Corner radius for Doors set through MDF DoorMaker as having Soft Corners.

The maximum Radius allowed is 1/4" (6.35mm).

Reports

Following are the main areas in the Reports Tab:

Part Report - Select which Report to use when Printing Secondary Part Sheets

Pattern Printouts

Nest Report - This options allow you to determine which Pattern Printout Reports to use by default.

Measurement Units

The Measurement Units tab allows access to the area where your standard display unit of measure and its precision are set. Keep in mind that the S2M Center stores everything as a double integer in the Imperial (Inches) unit of measure. What you are selecting is how the S2M Center displays this stored information not how it is stored.

A double Integer consists of 15 unit places, meaning a measurement entered as 56mm is actually stored as 2.20472440944881” by the S2M Center. The measurement entered by you at 56mm would display as follows:

Units as Millimeters

Precision as 1 = 56

Precision as 1.5 = 56.0

Precision as 1.1 = 56.0

Precision as 1.01 = 56.00

Precision as 1.001 = 56.000

 

Units as Inches

Precision as 1 = 2

Precision as 1.1 = 2.2

Precision as 1.01 = 2.20

Precision as 1.001 = 2.205

Precision as 1.0001 = 2.2047

Precision as 1.00001 = 2.20472

Precision as 1.000001 = 2.204724

Precision as 1/8 = 2 1/4

Precision as 1/16 = 2 3/16

Precision as 1/32 = 2 7/32

Precision as 1/64 = 2 13/64

The important thing to remember is the S2M Center sees this number (56mm), no matter how you have the display units set, as 2.20472440944881”.

The "Trim trailing decimal zeros" will simply trim any trailing zeroes off of displayed decimal measurements.

The "Allow unit-switch button on Status Bar" options turns on/off the Unit-switch button in the Statusbar Menu.

Tools Tab (Accessible from Part Edit area)

Smart-lines. In this section you can control the Snap Angle as well as the Tolerance setting of a drawn Line.

A Snap Angle of 45 and a high Tolerance (Sliding the bar all the way to the right) will snap lines only in 45 degree increments.

A Snap Angle of 45 and a low Tolerance (Sliding the bar all the way to the left) will basically disable the Snap feature. This will basically allow you to draw a free-hand line.

Snap Lines. In this section you can control the Color and Type (Dashed or Solid) of line that Dynamic Snapping will use.

Snapping Options. In this section you can control the Snap Hot-Keys as well as which Hot-Points are on for Dynamic Snapping.

Grid Snap - Snaps what you are currently placing to Grid points.

Point Snap - Snaps what you are currently placing to Snap points.

Short-cut Keys. In this section you can modify the default Short-cut Keys for quick commands inside Solid. You may want to change these if they interfere with Short-cut Keys you may already have setup in Windows. There are five functions that can be controlled by short-cut keys:

Pan - The Pan function allows you to set a new View center. When activated the Pan feature will center your current view to the location of the cursor on the screen.

Refresh - The Refresh function instructs Solid to redraw the current View.

Zoom Mode. In this section you control how the Windows for a Zoom Window is drawn.

From Corner - When using this and performing a Zoom Window you specify the corners of your Zoom area diagonally.

From Center - When using this and performing a Zoom Window you start from the Center of the View. This is the way Solid has normally performed a Zoom Window.

Factor - This slider sets the Zoom Factor for Scroll Wheel Zooming. The lower (left) the setting the smaller the Factor.

Reverse Scroll Wheel Zoom Direction - An option allowing you to reverse the Zoom In/Zoom Out direction on your Mouse Scroll Wheel.
