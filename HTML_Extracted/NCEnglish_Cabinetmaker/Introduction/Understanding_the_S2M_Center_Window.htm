Understanding the S2M CENTER Window

When you start a new Job in the S2M CENTER, the main S2M CENTER window opens. This window is your primary work space. It contains a list of all the Parts that will be sent to your machines. The following illustration shows the main parts of the S2M CENTER window.

File Menu

The File menu is located in the upper-left corner of the S2M CENTER. When you click the file menu, you see the same basic commands available in Cabinetmaker.

Quick Access Toolbar

The Quick Access Toolbar is a toolbar containing a set of commands that are independent of the ribbonbar tab that is currently displayed.

The Quick Access Toolbar can be located in one of two places:

  • Upper-left corner above the File menu (default location)

  •  Below the ribbonbar

If you don't want the Quick Access Toolbar to be displayed in its current location, you can move it to the other location. If you find that the default location above the File menu is too far from your work area to be convenient, you may want to move it closer to your work area. The location below the ribbonbar encroaches on the work area. Therefore, if you want to maximize the work area, you may want to keep the Quick Access Toolbar in its default location.

  1. Click Customize Quick Access Toolbar  .

  2. In the list, click Show Below the Ribbon.

Titlebar

The Titlebar displays the Job you have open.

Ribbonbar

This section gives quick and easy access to tools specific to the area of the S2M CENTER you are in. The ribbon will change depending on which area of the S2M CENTER you are in. Tools with small  triangles on the right side have flyouts containing tools that invoke commands related to the first tool shown. Click and hold down the first button to display the flyout.

Minimize, Maximize, and Close Buttons

These buttons hide the S2M CENTER window, enlarge it to fill the whole screen, and close it, respectively.

Style Selection

Allows you to choose the Style that the S2M CENTER interface will utilize. Following are the choices.

  • Colorful - The default style

  • White

  • Dark Gray

  • Black

Sidebar

This section gives access to functions specific to the area of the S2M CENTER you are in. This section will change depending on which area of the S2M CENTER you are in.

Parts and Materials List Area

This is the actual working area of the S2M CENTER. This area's size varies, depending on the size of the S2M CENTER window and on the number of other elements that are displayed.

Statusbar

This bar contains your software license information.

Statusbar Menu

The Statusbar menu gives you quick access to control the status of 4 possible options:

  • Unit Selection or - Allows you to toggle between Inches (Imperial) and Metric Units on the screen. You can turn this button On/Off and set its default Unit via the Preferences | Measurement Units tab.

  • Unit Precision - Selecting this button will allow you to change the Precision of the Unit selected. The default Unit Precision is set via File Menu | Preferences | Measurement Units tab.

  • Snap Grid - This option turns the Grid display on. You will see a grid pattern in the window where you Edit Part Shapes or Edit Operations that you can Snap to. If this button is highlighted it means it is active simply click it again to disable it. Go to Right Click Properties to set the Grid Spacing.

  • Snap to Points - Using this snap feature allows precise snapping of drawn entities. This option turns on the Dynamic Snapping functionality in the S2M Center. The snap to points appear as small dark dots in the corner of the objects and all parts within the object. These are the snap to points. If these points are present and not desired, reset the snapping mode to OFF or another mode to eliminate them. If this button is highlighted it means it is active simply click it again to disable it.
