Reports

The S2M Center comes with thirteen predefined reports, simply select the report from the sidebar you would like to view.

  1. Material Summary - Material summary that groups the sheets used in this run by the material name, than provides a breakout of the sheet sizes used.

  2. Part Labels (Opt) - Part labels (for optimized parts) for Avery label style 5366.

  3. Part Labels (Opt) - Part labels (for optimized parts) with enlarged bar-code field for Avery label style 4013.
