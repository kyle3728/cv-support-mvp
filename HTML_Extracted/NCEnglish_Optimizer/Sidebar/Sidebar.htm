At the Pattern/Part View

There are three selectable areas in the Optimizing sidebar:

  • Cut Patterns - This area displays all optimized sheets. To view the Patterns simply click on the stack image in this area.

  • Reports - Allows you to view and print the S2M Center Reports.

  • NC Code & Labels - Allows you to output NC Code & Labels.

Right Click Options

In the Cut Patterns are there is one right click option:

  • Print - Clicking the Print command will Print the selected stack.
