Optimize

The Optimize button, once depressed, will begin the Optimization, once selected the Optimizing Screen will appear.

With a Saw Selected

 

You can override the Run Number here if necessary.  This field is incremented each time you optimize. This number is then used to determine the actual run number for the Job. The RunCounter is divided by the user defined Run Number Range and its remainder is added to the minimum range.

For example:

If the RunCounter is 723. That number is divided by 100 (600-500=100) and you get a remainder of 23.

That remainder is added to the minimum range (500) and the result is 523.

The Run Number is displayed at each of the Optimize dialogs. You can override the current run number by typing in a new value (disabled for a preset number).

If you change the run number to 100 for example. The next time a/the job is optimized the Run Number will default back to your normal range and the next increment based on the calculations above.

  1. Click on the Run button to begin the Optimization process.

  1. Once the Optimizing is completed this screen will display each Material optimized with the total number of sheets, the Used percentage, the Offcut percentage, and the Waste percentage.

You can at this point Print the completed section by clicking the Print button or by clicking the OK button you can accept the results and move on to the Patterns Area of the S2M Optimizer.
