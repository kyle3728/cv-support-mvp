S2M Optimizer Startup Group

In Windows 10 click the Windows Start Button then scroll to and select the S2M Optimizer 11 Group.

In previous versions of Windows go to Start | All Programs | S2M Optimizer 11

There are four possible options in the Startup Group:

  1. S2M Optimizer 11 - Launches the S2M Optimizer.

  2. S2M Optimizer 11 Help - Opens this Help System.

  3. System Information - Opens the Vero System Information Utility.

  4. Update S2M Optimizer 11 Databases - Launches the Update Database Utility which compacts and repairs S2M Optimizer Database Files.
