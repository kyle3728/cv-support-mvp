Using a Mouse in the S2M Optimizer

You can choose options by left clicking them with your mouse. You also can use the mouse to select objects on the screen. With a two-button mouse, the left button is the select button used to select objects on the screen. The right button typically displays a shortcut menu.

Scroll Wheel Mouse

Mouse Button Functions

The mouse provides feedback to the S2M Optimizer regarding a variety of functions. Although a Scroll Wheel Mouse is optional, it is highly recommended. The functions are listed below:

Left Button

  • Click to select objects

  • Drag to select multiple objects

  • Click to specify locations

Right Button

  • Click to cancel an event

  • Click within a window for window context sensitive (short cut) menu

  • Click on object for object context sensitive (short cut) menu

Wheel Button (Optional)

  • Scroll forward and backward to zoom in and out.

  • Double click to Zoom All

  • Drag to Pan view

A Scroll Wheel Mouse is typically a two-button mouse with a small wheel between the buttons. The left and right buttons behave the same as they do on a standard mouse. You can rotate the wheel by discrete values. You can use the wheel to zoom and pan in your drawing without using any S2M Optimizer tools.

The following table lists the mouse actions supported with a Scroll Wheel Mouse in the S2M Optimizer.

Do this...

Zoom In or Out

Rotate the wheel forward to Zoom In, backward to Zoom Out.

Zoom All
