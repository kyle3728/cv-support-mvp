Help

Help gives you access to the following options:

Accesses this Help System. For information on how to use the S2M Optimizer Help file see the Using Help topic.

Access the eSupport web site. You must have an Internet Connection setup to use the eSupport web site.

Access the Training Options for the S2M Optimizer. You must have an Internet Connection setup to view the Training Options.

This option allows you to start a Team Viewer desktop sharing session with a Vero Technician. Use this option when instructed by your Vero Technician and provide the displayed ID and Password.
