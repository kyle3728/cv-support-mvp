Parts

This is the default opening view for the S2M Optimizer.

You will notice in the figure above that the "Left End" has a Red icon next to it on the Left. Any parts in the list denoted with a red icon is a part that will not be optimized. In this case the part is longer than the 96" long Material that is specified.

To find the exact meaning of a particular Part icon simply hover over it to receive a description.

By default all items that can be sent to the Optimizer will be sent to them. However, if you just wanted to send certain items to the Optimizer you can select them individually. To select a part simply click the icon next to the part. To select more than one simply hold while selecting. You can also select a range of parts by selecting the first part in the list you would like to send and then hold and select the last part. This will select all parts in between the first and last parts selected.

You can also change the Material of any part in the list. Simply click on the Material Name and a drop down will appear and allow you to change the current Part to any other Material available in the current Job.

Any item in the list can be modified. You can change the Quantity of parts, the parts size, you can even add an Overage using the Over column.

The red arrow in the view above is there to let you know you can add a new Part to this Part list.

Right Click Options

Right clicking on a Part in the Part list brings up the following menu:

There are three possible commands available in this Right Click menu:

    1. Edit

    2. Delete - Clicking the Delete command will delete the selected part from the Parts List.

    3. OK to Optimize - This option allows you to determine if the selected Part is OK to go to the Optimizer.

If you do not see this option check to make sure Show Parts Not Set for Optimization or Output is enabled in Preferences.
