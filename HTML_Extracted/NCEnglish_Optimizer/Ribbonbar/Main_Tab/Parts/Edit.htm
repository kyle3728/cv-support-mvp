Sidebar

There is one option in the sidebar:

Edgeband

The Edgeband Editor allows you to assign Edgebanding to your Parts. You must have Edgeband Materials and have setup Edgeband Indicators in the Material Catalog in order to fully utilize this facility.

The Edge Banding option allows you define which edge of a Part gets edge banded with which edgeband indicator. Once the Edge Band option is active you will be placed in Edge Band mode with the following 4 options in the ribbonbar.

  • Select - When entering Edgeband mode this tool is selected by default. Simply click on the edge you would like to apply the Edgeband to and then select its type in the sidebar.

You can use Windows + Click to select multiple edges.

  • Select All - To quickly change all Edges to a specific Edgeband select this option then select their type in the sidebar.

  • Remove Banding - Will remove Edgebanding from the selected edge(s).

  • Set Banding - Will expose the Edgeband type selector in the sidebar without having to select an edge first.

It is important to note that the default values in the sidebar options are NOT representative of the current Edge Banding on the selected Part. To use this function simply select which Edgebanding type you would like.

Once the Banding options are chosen simply select any of the Part Edges you would like these Edgeband properties assigned to. In the image below the Front Edge of the End has been selected.

Note that the front edge is now red which indicates an Edgeband has been applied. Once you are through defining the Edges simply click the Return option to save the new Edge Properties.
