View Mode

There are two options in View Mode:

Part View

Part View is not available in the S2M Optimizer.

Pattern View

This is the Default View when Optimizing. The Pattern View displays the Patterns list in the sidebar and the selected Pattern and it's machining in the Pattern/Part Display Area.

This area displays your Sheet/Pattern layouts.

The lower Section is the Part List. The Parts on the active sheet are listed in the lower section of the screen. There are several columns of information which you can sort the list by clicking on the column heading. A Part selected from either the Pattern are or Part list area will cause that Part to be highlighted in both Views.

Right Click Menu

There is one option in the Sheet Display Area right click menu.

Properties

General Tab

When using an Optimization machine there are ten main options in the General Tab:

      1. Dimension Style - You can control whether the output Pattern printouts Dimension Style is "Width x Length" or "Length x Width".

      2. Dimension Type - You can control if you want Dimensions to display on the Parts or in a Legend on the side of your printouts.

Parts

Legend

None

      1. Set Origin at Bottom Left Corner - You can set the Origin at the Bottom Left corner on your printouts.

      2. From Corner/From Center - You can control if, when using the Zoom Window tool, whether the origin of the Zoom window begins From the Corner of where you pick or the Center.

      3. Factor - Determines the percentage the Scroll Wheel will Zoom the view In and Out.

      4. Reverse Scroll Wheel Zoom Direction - Determines which direction the Scroll Wheel Zooms In and Out.

Annotations Tab

On the Annotations Tab there are seven main options:

      1. Font - You can control the displayed Font.

      2. Dimensions Horizontal Offset - You can control the Horizontal Offset of your Dimensions.

      3. Dimensions Vertical Offset - You can control the Vertical Offset of your Dimensions.

      4. Dimensions Text Height - You can control the Text Height in your Dimensions.

      5. Dimensions Tick Height - You can control the Tick Height in your Dimensions.

      6. Annotations Heading Text Height - You can control the Text Height in the Heading Annotations.

      7. Annotations Part Labels Text Height - You can control the Text Height in your Part Labels (Labels here refers to the Text marking your Part on the Sheet not the Label Reports you Print.).
