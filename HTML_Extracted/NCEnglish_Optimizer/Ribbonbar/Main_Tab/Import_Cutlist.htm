Import Cutlist

Import Cutlist allows you to bring in a list of Parts in a CSV format for Optimization. You can use your favorite spreadsheet application or design application to create a CSV that can be imported directly into the S2M Optimizer.

This utility examines the text file that you are importing and helps you ensure that the data is imported in the way that you want. You can use this Utility to define what each field in your Text file represents so it is imported properly.

Toolbar

There are four options in the toolbar:

Select Configuration

Allows you to select which of your Saved data field configurations to use.

Save

Allows you to Save the current configuration for re-use.

Rename

Allows you to rename the currently selected configuration.

Delete

Allows you to delete the currently selected configuration.

General Options

There are four general options available to configure your Text file:

Fields - This option allows you to establish how the fields in the Text file are defined. If items in the text file are separated by tabs, colons, semicolons, spaces, or other characters, select Delimited; then select what the delimiter is (Tab, Comma, Semicolon, Space, Other). If all of the items in each column are the same length, select Fixed width.

Text Qualifier - Select the character that encloses values in your text file. When Solid encounters the text qualifier character, all of the text that follows that character and precedes the next occurrence of that character is imported as one value, even if the text contains a delimiter character. For example, if the delimiter is a comma (,) and the text qualifier is a quotation mark ("), "Tuscaloosa, Alabama" is imported into one field as Tuscaloosa, Alabama.

Dimensions - Select whether the file you are importing is Inches or Metric.

Start Import at Line - Type or select a line number to specify the first line of the data that you want to import.

Import Preview Area

In this are you will see all of the columns that the S2M Optimizer has available for your Text file. Click the column header of the column(s) that you do not want Ignored and set its type. There are eleven options:

Ignore - Will ignore this column when importing.

Name - The Part's Name/Description.

Width - The Part's Width.

Length - The Part's Length.

Thickness - The Part's Thickness.

Quantity - The Part's Quantity.

Material - The Material the Part will be cut from.

Grain - Establishes if the Part is Grained or not.

Assembly - The Name/Description of the Assembly the Part is used in.

Assembly Number - The Assembly Number of the Assembly the Part is used in.

Banding - The banding instructions for the Part. The banding instructions should only include Banding Codes you have defined within the S2M Center Material Catalog (lower case letters). N or 0 (zero) would equal an Edge with no Banding. The banding instructions will only define four edges. In the example below would be the "Banding" for the illustrated Part.

4NbNN

4 Edges starting from the reference point and going clockwise you get None, b , None, None.
