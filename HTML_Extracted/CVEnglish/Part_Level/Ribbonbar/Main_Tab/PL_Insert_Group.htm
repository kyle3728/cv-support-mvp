Insert Group

There is one option in the Insert Group at the Part Level:

ALPHACAM

Allows you to select and place an ALPHACAM drawing that is saved into the S2M CENTER Part Library.

When selecting an ALPHACAM Drawing there are five options in the sidebar Properties area:

  • Pos X - Establishes the X Position of the start point for the ALPHACAM Drawing.

  • Pos Y - Establishes the Y Position of the start point for the ALPHACAM Drawing.

  • Pos Z - Establishes the Z Position for the ALPHACAM Drawing.

  • Angle - Establishes the Line Bore angle. Rotation occurs around the start point.

  • Description - The Description of this ALPHACAM Drawing.
