Drawing Group

Following are the options in the Drawing Group at the Part Level.

Line Bore - When selecting the Line Bore option you will left click to establish the start point of your row of Line Bores then drag and left click to establish the end point.

When selecting a Line Bore operation these are the options in the sidebar Properties area:

    • Pos X - Establishes the X Position of the start point for the Line Bore.

    • Pos Y - Establishes the Y Position of the start point for the Line Bore.

    • Pos Z - Establishes the Z Position for the Line Bore.

    • End X -  Establishes the X Position of the end point for the Line Bore.

    • End Y - Establishes the Y Position of the end point for the Line Bore.

    • Length - Establishes the Length of the Line Bore.

    • Diameter - Establishes the Line Bore Diameter.

    • Depth - Establishes the Line Bore Depth.

    • Angle - Establishes the Line Bore angle. Rotation occurs around the start point.

    • Spacing - Establishes the Line Bore spacing.

    • Repeat - Establishes the number of Line Bores.

    • Primary - Establishes whether this is a Primary Operation or not.

    • Tool - Allows you to assign a specific tool to the Line Bore.

    • Description - The Description of this Line Bore.

When selecting a Hole operation these are the options in the sidebar Properties area:

Clamex - Use this option to place machining for Lamello Clamex™ connectors.

When selecting a Clamex operation there are ten options in the sidebar Properties area:

    • Pos X - Establishes the X Position of the Clamex.

    • Pos Y - Establishes the Y Position of the Clamex.

    • Pos Z - Establishes the Z Position of the Clamex.

    • Width - Displays the Width of the Clamex. The Type selected determines its Width.

    • Length - Displays the Length of the Clamex. The Type selected determines its Length.

    • Depth - Displays the Clamex Depth. The Type selected determines its Depth.

    • Angle - Establishes the Clamex angle. Rotation occurs around the center point.

    • Type - Establishes the Type of Clamex to machine for. There are four types available:

      • P10

      • P15

      • P14

      • 15/10

    • Primary - Establishes whether this is a Primary Operation or not.

    • Tool - Allows you to assign a specific tool to the Clamex.

    • Description - The Description of this Clamex.

Intelli-Joint - Allows you to select and place an Intelli-Joint from the Intelli-Joint Manager.

When selecting an Intelli-Joint these are the options in the sidebar Properties area:

    • Pos X - Establishes the X Position of the start point for the Intelli-Joint.

    • Pos Y - Establishes the Y Position of the start point for the Intelli-Joint.

    • Pos Z - Establishes the Z Position for the Intelli-Joint.

    • End X -  Establishes the X Position of the end point for the Intelli-Joint.

    • End Y - Establishes the Y Position of the end point for the Intelli-Joint.

    • Width - Establishes the Width of the Intelli-Joint.

    • Length - Establishes the Length of the Intelli-Joint.

    • Depth - Establishes the Intelli-Joint Depth.

    • Angle - Establishes the Intelli-Joint angle. Rotation occurs around the start point.

    • Type - Allows you to choose which Intelli-Joint from the Intelli-Joint Catalog to use.

    • Scope - There are two options to establish the Intelli-Joints Search Scope:

      • Assembly - The Default selection, this option only evaluates Secondary Parts within the Assembly the Intelli-Joint is located.

      • Room - Lets the selected Intelli-Joint search outside of the Assembly to adjacent Assemblies to apply Machining. Very handy for Assembly Connector Machining.

    • Description - The Description of this Intelli-Joint.

Line - Using the Line option allows you to draw one Line at a time by selecting the start point then the end point. See the Line option in the CAD Main Tab topic for more information.

When selecting a Line these are the options in the sidebar Properties area:

    • Pos X - Establishes the X Position of the start point for the Line.

    • Pos Y - Establishes the Y Position of the start point for the Line.

    • Pos Z - Establishes the Z Position for the Line.

    • End X - Establishes the X Position of the end point for the Intelli-Joint.

    • End Y - Establishes the Y Position of the end point for the Intelli-Joint.

    • Width - Establishes the Width of the Line.

    • Length - Establishes the Length of the Line.

    • Depth - Establishes the Line Depth.

    • Angle - Establishes the Line angle. Rotation occurs around the start point.

    • Cut - There are three options for Cut:

      • Centerline - Cuts along the Centerline of the Line.

      • Inside - Cuts along the Inside of the Line.

      • Outside - Cuts along the Inside of the Line.

    • Tool - Allows you to assign a specific tool to the Line.

    • Description - The Description of this Line.
