Right Click Line Item in the Reports View -

 

 Delete

If available this option will delete the selected CNC Operation.

 Cut

+

If available this option will Cut the selected Operation to the Windows Clipboard. You can not Cut System Generated Operations.

 Copy

+

This option will Copy the selected Operation to the Windows Clipboard.

 Paste

+

This option will Paste an Operation from the Widows Clipboard that has been Cut or Copied as a new Editable Operation.

  Properties

There are 4 tabs available on the Operation Properties screen:

  • Tool - This Tab allows you to select which of the available Tools you would like CABINET VISION to assign to the selected Operation. Only those Tools available for the Operation type are selectable. Auto Select is the recommended and the System's default selection. See the S2M CENTER Help system for more information on Tools, Tool Properties, and Automatic Tool Selection Logic.

  • Route/Hole - This tab allows you to specify the Tool-path Type for a Route/Hole.

  • Overrides - See the Assembly Level Overrides topic for more information.

  • Parameters - See the Assembly Level Overrides topic for more information.
