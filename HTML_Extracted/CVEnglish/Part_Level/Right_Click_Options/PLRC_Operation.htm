Right Click an Operation -

 

Following are the possible options here:

 Edit...

Opens the Shape Editor options in the ribbonbar for the Operation.

Click the options below for more information

 Cut

This option will Cut the selected CAD Object(s) to the Windows Clipboard.

 Copy

This option will Copy the selected CAD Object(s) to the Windows Clipboard.

 Paste

This option will Paste the CAD Object(s) stored in the Windows Clipboard to the current View.

 Delete

This option will Delete the selected CAD object(s).

 Transform

Depending on the active Operation type the following options are possibly available in the Transform fly-out:

 

 Flip - Will Flip the selected CAD Object.

 Mirror - Will Mirror the selected CAD Object.

 Reverse - Will Reverse the selected tool path on the selected Operation.

 Rotate... - Allows you to specify a Rotation for the selected CAD Object.

 Select All

The Select All menu option will select all CAD Drawn objects in the current View.

 Select None

The Select None menu option will deselect all of the CAD Drawn objects in the current View.

 Reverse Selection

The Reverse Selection option will deselect all the current Selections and in turn Select all the CAD objects that were not in the previous selection set.

  Properties...

There are 4 tabs available on the Operation Properties screen:

  • Tool - This Tab allows you to select which of the available Tools you would like Solid to assign to the selected Operation. Only those Tools available for the Operation type are selectable. Auto Select is the recommended and the System's default selection. See the S2M Center Help system for more information on Tools, Tool Properties, and Automatic Tool Selection Logic.

  • Route/Hole - This tab allows you to specify the Tool-path Type for a Route/Hole.

  • Overrides - See the Assembly Level Overrides topic for more information.

  • Parameters - See the Assembly Level Overrides topic for more information.
