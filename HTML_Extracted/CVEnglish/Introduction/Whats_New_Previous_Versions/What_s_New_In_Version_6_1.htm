What's New in Solid Version 6.1 (2011 R2)

Click the links below to get more information on these new features. For quick demonstrations of these new features you can visit the Cabinet Vision Solid 2011 R2 area of the Cabinet Vision YouTube Channel .

Assembly Level

  • Recent Tab is the Default Tab When Setting Part Materials

  • Option to Ignore Edgeband When Exploding an Assembly

  • New Options to Set Line Bore Type by Assembly in Section Editor

 

Assembly Wizard

  • Bore By Opening Available for Closet Objects

 

Bid Center

  • Export Bid Data to a Database for 3rd Party Use

 

CAD

  • Shift + Drag Rectangle Forces Square in CAD

  • Reversed Leader Creation (Start Point Establishes Arrow Location)

  • Text Leader Tool

  • Revision Cloud Tool

  • Layers from Symbols Come With Them When Inserted

  • Setup Text Prompts in CAD Symbols

  • Added New Table Groups to Assembly Level/Assembly Sheets

    • Drawer Box Parts

    • Drawer Boxes

    • Rollout Box Parts

    • Rollout Boxes

 

Counter Tops

  • Editing of Automatic Appliance Cut Outs in Counter Top Editor

  • Define Joint Edges via Counter Top Shape Editor

 

Doors

  • Option to Output Buyout Doors to CNC

 

Drawers & Roll Outs

  • Adjustment for Dovetail Fit

 

Drawings Area

  • Selecting a Scene From the Scene List Will Provide a Scene Preview

  • Align Scenes Using Snap Points to Grid, Paper, or Other Scene Points 

 

General

  • Custom Order Imports via TXT/CSV

  • Custom Part List Exports via Text File

  • Drawer Box and Roll Out Material Schedules Added to ORD File

  • Improved 3D Rotation Functionality

  • Change Unit of Measure Precision via Statusbar Menu

 

Job Properties

  • All Combo-boxes (drop downs) Sorted Alphabetically

 

Layers

  • Turn On/Off Door Drawer Numbering at Section Editor

  • Turn On/Off Door Drawer Numbering at Assembly Sheet

  • Light Layer for User Added Lights

 

Layout (Room) Level

  • Establish a Vertical Position for Object Placement (See Object Placement Also)

  • 0 Entered for Floor to Upper Above Refer Disables Option

  • 0 Entered for Floor to Upper Above Range Disables Option

 

Reports

  • Assembly ID From CVC Added to Cabinets Table in Report.mdb

  • Hinge Material ID From CVC Added to Doors Table in Report.mdb

 

User Created Standards

  • Delete and Reposition a Licensed UCS

  • _MATCH - Set Local or Global Search for Primary Operations
