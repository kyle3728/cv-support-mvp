What's New in Solid Version 4.2

 

Assembly Level

  • Ability to Add System Flutes to any Part

  • Option to Allow UCS Modification on New Parts

  • System Parameter Notes at Notes Tab at Assembly Properties for User Modification

  • CAD Added at the Assembly and/or Part Level Optionally Saved with the Assembly

  • Attributes Added to an Assembly Via UCS or Manually Optionally Saved with the Assembly

  • Notes Added to an Assembly Via UCS or Manually Optionally Saved with the Assembly

  • Drawer Section Editor Added (Allows Cutlery Style Divisions in Drawer Boxes and Roll Outs)

 

Assembly Wizard

  • Assembly Wizard Standard #30 Allows Butting Gussets Against an Inside Nailer

 

Backup & Restore

  • Added the Ability to Specify a Path for the Automatic Backup Location

 

Bid Center

  • Multi-level Group Stock/Modular Pricing by Catalog, Room, and/or Door Style in any Combination

  • Break-Out Rate Tables Per Room (Custom Bids)

  • Break-Out Add-Ons Per Room (Custom Bids)

 

CAD

  • Utility to Allow Importing of 2D DXF's

  • Create Data Tables in CAD

  • CAD Auto-Text Interface Enhanced

  • 242 additional CAD Auto-Text Fields Bringing the Total from 23 to 265

  • Auto-Text ALL System and User Defined Parameters

  • Formatting Options for AutoText

  • Specify the stack Order of CAD Objects

  • Double Click Any Editable CAD Object in CAD Mode to Edit

  • Walls/Objects to Snap to CAD

  • Display Assembly Level CAD at the Room Level

 

Doors

  • "None" Panel Type Added

 

Drawers & Roll Outs

  • New Drawer Box and Roll Out Wizard Added

  • Drawer Box/Roll Out Construction Not Dependent upon a Specific Material Schedule

  • Added "System" Roll Out Type

  • Unlimited Standard Height Designations

  • Unlimited Standard Depth Designations

  • Grain Direction Control for Bottoms

  • Dovetails for Drawer Boxes and Roll Outs

  • Swoop Shape Editor for Roll Out Fronts

  • Side Shape Editor for Roll Out Sides

  • Ability to Define Drawer Box and Roll Out Partitions (Front to Back Split)

  • Ability to Define Drawer Box and Roll Out Dividers (Left to Right Split)

  • Construction Boring Specification (Both full CNC and Fixed (Pattern) Definition)

  • Miter Option for Overlapping Joints

  • Stop and/or Through Dado Definition

  • Drawer Front to Sub-Front Adjustment Machining

 

Drawings Area

  • Generate "Live" Drawing Scenes

  • Convert a "Live" Drawing Scene into a "Static" Drawing Scene

  • Undo/Redo Activated in Drawings Area

  • Assembly Name and Number from Section Editor added to Drawing Scene

  • Bitmap Drawing Scenes set "Visible" By Default

  • Specify Custom Paper Sizes

  • Symbol Library Renamed to Drawings Library to better Represent Enhancements

  • Drawings (Symbol) Library Interface Reorganized

  • Copy Drawings (Symbols)

  • Rename Drawings (Symbols)

  • Drawing (Symbol) Types as well as Multiple Categories for each Type

  • Add, Delete and Rename Symbol Categories from a single area

  • Re-order Drawing Sheets

  • Double Click a Scene to go to CAD

  • Print Landscape and Portrait "Drawings" Simultaneously

 

General

  • New Look And Feel (New Product Icon, Splash, and Buttons)

  • Option for Zoom Window to begin from a Corner instead of Center

  • Zoom via Scroll Wheel all Views

  • Pan via Scroll Wheel Drag in all Views

  • New Catalog "Appliances 2" Added (Include Over 400 New Objects)

  • New Fields Added to System Properties User Tab

  • Give a Description to an Intelli-Joint

  • Set a Text "Type" Parameter

  • Establish New System Parameters as an Attribute (Normal) or as a Note

  • Construction and Intelli-Joints Added to Update Job Utility

User Created Standards

  • Define Public Variables in a UCS

  • Double Clicking a UCS in the UCS List will Open it in the Editor if Previously Modified

  • UCS Editor Macro Recorder (Removed in Version 10)

  • Documented Keyboard and Mouse Shortcuts for the UCS Editor

  • Ability to Set a Parameter's Style via UCS

  • Set A Description (Prompt) for a Parameter via UCS

  • Add Dovetails to Parts Via UCS (Through Wizard or Editor)

  • _MAXTRAD - Specifies the Maximum Tool Radius for a Dado/Route Operation

  • _NESTOP - Specifies that the Operation Should Only be Performed on a Nest

  • _EDGnNPC - Specifies a Nested Part Clearance for the Specified (n) Edge

  • _NESTSIZE - Allows you to Set an Override on Part Size Rules in the S2M Center

  • _DRILLMASK - Allows you to Set a Pattern Mask for Repeat Style Boring Operations

  • CONSTID Parameter Will Now Return Assembly, Drawer, and/or Roll Out Construction ID

  • _BANDMATID - Retunrs Door/Drawer Front Edgeband Material

  • _EDGnRADIUS - Returns Part Edge Radius

  • Remove a Parameter From an Object Via UCS

  • Test for the Existence of a Parameter

 

**Items marked with **(No longer valid in Version 9) can be handled in the new CAD Shape Editor.
