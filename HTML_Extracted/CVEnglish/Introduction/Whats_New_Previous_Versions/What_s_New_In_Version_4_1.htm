What's New in Solid Version 4.1

 

Assembly Level

  • Part Selected in Cabinet Ortho View Side Bar Enhancements

    • Property Displays Corresponding Parameter (X,Y,Z,DX,DY,DZ,AX,AY,AZ) at Field

    • Add a formula directly to these fields without having to open the Object Tree

  • Ability to set default option for Multiple Splits in Section Editor

  • Default Cabinet option to remove Shape Templates and/or User Modified Parameters

  • Ability to specify "No Bottom" in Case

  • Glass Adjustable Shelf added to Horizontal Split type selection

  • Right Click Screen Assembly Properties Added

 

Assembly Wizard

  • Vanity Breakout Added to the Assembly Wizard

  • Construction Method ID Number displayed in the Assembly Wizard

  • Specify Grain Direction for Bottoms for 45 Corner Cabinets in the Assembly Wizard

  • Specify Grain Direction for Tops for 45 Corner Cabinets in the Assembly Wizard

  • Specify Mid Nailers in the Assembly Wizard

  • Specify Full End to Floor or Notched End Toe for Finished Ends and Unfinished Ends

  • Assembly Wizard Window can be resized

 

Bid Center

  • Tie Custom Labor Schemes into your Rate Tables in the Bid Center through a Labor Bid Method

  • New Bid Method, % of Running Total

 

CAD

  • Cut, Copy, and Paste added to CAD Area

  • Rotate Objects in CAD Area

  • Flip Objects in CAD Area

  • Mirror Objects in CAD Area

  • Trim Objects in CAD

  • Crop Objects in CAD

  • CTRL Key to select Multiple Objects in CAD

  • Modify CAD Dimension Text without affecting Dimension Line

  • Easily Change Annotation (Leader) Arrow Size and Style in CAD

  • Undo and Redo activated in the CAD area

 

Closets

  • Closet Class Objects added

  • Closet Properties added to Job Properties

  • Hanging Sections in the Assembly Wizard and the Section Editor

  • Shoe Sections (Cubbies) in the Assembly Wizard and the Section Editor

  • Shoe Shelves in the Assembly Wizard and the Section Editor

  • "Wire Baskets" added to Material Catalog and Material Schedules

  • Wire Baskets in the Section Editor

  • Advanced Closet Layout options added to Object Save options

 

Counter Tops

  • Modify Counter Top Construction Method for Room and/or Job

  • Modify Counter Top Cut Out shapes

  • Counter Top Wizard can now be resized

 

Doors

  • Enhanced Door Catalog Interface

  • New Multi-Panel Tools added to Door Catalog Construction Tab

  • User Defined and Expanded Door Catalog Categories

  • Tool Tip added to Associated Door Image in Door Catalog Associated Door Tab

 

Drawings Area

  • 1 ½ Scale option

  • Rotate Scenes

  • Flip Scenes

  • Mirror Scenes

 

General

  • Restore Utility added to restore from Solid Automatic Backups

  • Import a JobSite Companion Job

  • Create and specify Parameters from the System Level

  • Object Smoothing option for Solid and imported DXF Parts

  • Specify how Textures are applied to Solid and imported DXF Parts

  • Enhanced DXF Import preview

  • Updated Furnishings Catalog to take advantage of new Smoothing and Texture Mapping

  • Updated Appliances Catalog to take advantage of new Smoothing and Texture Mapping

  • Scroll Wheel Zooming in 3D Views

  • Unlimited Window Zooming in 3D Views

  • Toolbar added to the Splash Screen

  • Status bar added to the Splash Screen

  • Cut, Copy, & Paste Operation Line Items in the Part Editor

 

Job Properties

  • Enhanced Interface

  • Selected Door, Drawer Front, and End Panel Style Previews

  • Combo Boxes expanded to show all text

  • Specify Wall Thickness in Job Properties

  • Multi-Line capability for Customer Comments in Job Properties

  • Job Number Field in Job Properties expanded to match Microsoft Long File Name limit

  • Comment Tab added to Job Properties

  • Ship To same as Customer Option added

 

Layers

  • Option to display a "U" label at Unfinished Ends

  • Toe Assembly Layer Control

  • Toe Assembly Dimension Control

  • Layer to control the visibility of User Added Parts

 

Layout (Room) Level

  • Quantity added to Cabinet Tool Tips

  • Establish Cabinet Back Type on Peninsula Walls (Finished, Unfinished, or No Back)

  • Live Section Editor from the Layout Elevation View

  • Tool Tips added to Color/Finish Selector in 3D View

  • Search function for Color/Finish Selection

 

Materials

  • Specify the following items in the Material Catalog:

    • Pull Hole Diameter

    • Drawer Guide Mounting Hole Diameter

    • Drawer Guide Mounting Hole Depth

    • Hinge Cup Bore Depth

    • Hinge Anchor Bore Diameter

    • Hinge Anchor Bore Depth

    • Hinge Plate Bore Diameter

    • Hinge Plate Bore Depth

    • Hinge Plate Bore Spacing

    • Hinge Plate Bore Offset

  • Multi-Line capability in the Address and Comments fields for Vendors

  • JPG file type Added to Valid Texture Import

  • Rough Thickness field added to Materials using BD FT or BD M Unit of Measure

 

Object Tree

  • Resize the Object Tree Window

  • Object Tree Parameter Column Width expanded

  • Parameter list sorted alphabetically

  • Added Icons to represent manually modified and UCS modified Parameters

 

Parts

  • Specify Price per Part (Fixed Value or Formula driven Value)

 

Reports

  • Cabinet Comments now displayed on Assembly Sheets

  • Print Assembly Sheets for the entire Job

  • Materials from the Laminate Tab added to Assembly Sheets.

  • Updated Report Center Engine to Crystal Reports XI

 

User Created Standards
