What's New in 2022.x

Click the links below to get more information on these new features. For quick demonstrations of these new features you can visit the CABINET VISION 2022 area of the CABINET VISION YouTube Channel .

Assembly Level

  • Define dimensions for sectioned doors
  • To Drawing added to door section editor
  • Beaded Frames work smartly with zero width mid stiles and mid rails - 
  • Interior and Exterior Banding available via New Part option
  • New Molding - Molding Profile drop down alphabetized
  • Section Editor Overrides window relocated Remove button and added a Cancel button

  • Finished Side Orientation for Sub Ends
  • Finished Side Orientation for Unfinished Ends
  • No End Connection for Cabinet Parts:

    • Drawer Stretcher
    • Fixed Shelf
    • Nailer
    • Shelf
    • Shelf Nailer
    • Shoe Section

    • Top

    • Adjustable Shelf
    • Fixed Shelf
    • Shelf
  • Associate toe kick position to leg position (fixed position for leg clip)
  • Front Skirt Part definition

Bid Center

  • Set default Rate Table(s) - 

CAD

  • When placing a CAD Symbol it will continue to place until right click to end placement - 
  • The Symbol Library will remember your last selection location - 
  • Alphabetize molding in CAD Symbols - 
  • Add the following profiles to CAD Symbols - 
    • Door Applied
    • Finger Pull
    • Inside Edge
    • Outside Edge
    • Raised Panel
    • Route
  • Crossing (right to left window) selection in CAD

Drawings

  • Specify sectioned door illustrations on assembly sheet containers - 
  • Copy sheet option in Drawings View
  • Define Floor Plan Templates

General

  • VORTEK Spaces export option - 
  • TEVA 3D export option - 
  • Seperate All Graphics and Assigned Material Textures Backup Options
  • Option to disable Zip creation in Backup
  • Protected Assembly Catalog added to Package Utility
  • Distributed Assembly Catalog added to Package Utility
  • Texture Manager thumbnail view uses sorting method selected from list view
  • Create backup copies of autosave files (abv, abs, abr, abz)
  • Export Room uses Room Name as the default filename
  • All list views, such as the Report view, use alternating color bars for easier legibility
  • Measure tool option for picking measurements added to most dimension fields in sidebar
  • Print Floor Plan Sheets along with Elevation Sheets
  • Define alternate letters, numbers, or even symbols for edgebanding
  • Door Sectioning data added to ORDX

Layout

  • Added an option to place the object anyway when getting the "sorry object in the way" error
  • When placing a Catalog Object it will continue to place until right click to end placement
  • Molding is ignored when modifying objects at the room level
  • Cabinet width change dialog defaults to stretch right instead of left
  • Room Level - Molding Properties Material and Profile drop downs alphabetized
  • New Room option opens Room Properties with Room Name field active
  • Support for Leica X- Series devices

User Created Standards

  • BAND and LIBPART parameter now support in-line evaluation
  • Parameter to specify primary edge per machine (MACHID_EDGE_nn)
