User Created Standards

  • _CONFIG - Returns Job Property Scheme Name

  • _DROEPROF - Control which Edges of a Door/Drawer Front the OE Door Profile is Applied to

  • _EXTDADO - Control Dado Extension

  • _LABEL – Returns the Door/Drawer Fronts position within the Assembly

  • _MATCHY - Returns the match point Y position for the front part

  • _MATCHYB - Returns the match point Y position for the back part

  • _POCWDTH - Control Pocket Width (Tool Allowance)

  • _SOFTCORNERS - Apply Soft Corner S2M Center Operation to Parts

  • _SPLITX - Define Part Splits

  • _SPLITXn - Define Part Splits

  • _SPLITY - Define Part Splits

  • _SPLITYn - Define Part Splits

  • BOR_BAL_OPN - Control Bore Balance for Linebore by Opening

  • BOR_FHOL_OPN - Control First Hole Position for Linebore by Opening

  • BOR_OPN - Turn On/Off Linebore per Opening

  • BOTANG - Control Bottom Dado Angle

  • DESC - Controls Assembly Description

  • MACHFA - Control Machine Face for Through Operations

  • NAME - Assembly Name

  • OBJECT - Return Object Type

  • S_MORTB - Control Mortises with the Primary Blind Dado

  • S_MORTDEP - Control Mortise Depth with the Primary Blind Dado

  • S_MORTL - Control Mortises with the Primary Blind Dado

  • S_MORTR - Control Mortises with the Primary Blind Dado

  • S_MORTT - Control Mortises with the Primary Blind Dado

  • S_TENONB - Control Tenons with the Primary Blind Dado

  • S_TENONL - Control Tenons with the Primary Blind Dado

  • S_TENONR - Control Tenons with the Primary Blind Dado

  • S_TENONT - Control Tenons with the Primary Blind Dado

  • TOPANG - Control Top Dado Angle
