Network Administrator

The Network Administrator Utility is used to create users, limit access to various parts of CABINET VISION, point CABINET VISION to the correct physical location for the Network Key, and check the status of CABINET VISION Users.

Users

In the Users Tab, there is a list of the Users that have been created. When a User is selected in the list on the left, the Rights for that User will be displayed on the right. Scroll down the list of Rights and set the level of access that you wish for that User to have for each area listed.

For example, the field labeled S2M CENTER will give you the choice to allow that user access to the S2M CENTER or not allow that user access. The choices for that field are simply 'Yes' or 'No'. However, there are sub-areas under the S2M CENTER such as Machine Catalog and Tool Catalog where you can set that User's access to 'None', 'View', or 'Edit'. Setting a User's access to 'View' will allow them to see that area without the ability to make changes. 'Edit' will allow the User to make changes.

It is recommended to have the default user of CABINET VISION user name match the Windows login name. Doing so will cause the user that is logged into the computer to have his/her user name automatically be selected when attempting to login to CABINET VISION.

Status

The Status Tab shows whether a User is currently logged in to CABINET VISION. Also, there is a button (Unlock Job) that can be used to unlock a Job if it has become locked. This may occur if there is a system crash while a network User has a shared Job open.

The Lockout All Users button is used to deny access to CABINET VISION by any user during periods of maintenance, etc.

Key Server

The Key Server Tab is for compatibility of legacy software only.
