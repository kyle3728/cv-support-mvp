CABINET VISION 2024 Start Group

In Windows 11 click the Windows Start Button then click All apps >scroll to and select the CABINET VISION 2024 Group.

In previous versions of Windows go to Start | All Programs | CABINET VISION 2024

Following are the possible options in the Startup Group:

  1. Backup CV 2024 Settings - Allows the creation of a backup of the CABINET VISION Database Files.  This option also gives access to the settings for the CV Backup Utility. You can also add switches to the short-cut for options when running the Backup Settings from the Start Group:

    • To add switches to Backup Settings right click on the Backup CV 2024 Settings icon in the Start bar and click More | Open file location.

  • Right click the Backup CV 2024 Settings icon in File Explorer and select Properties.

    • Click to place your cursor after the .exe in the Target field.

    • The following switches (command line arguments) are available:

      • /e Create email (will automatically create a mail message and attach a zip file for email)

        • Example: C:\Program Files\Hexagon\CABINET VISION\CV 2024\CVBackup.exe /e

      • /d Delete backups (useful option when you are just emailing)

        • Example: C:\Program Files\Hexagon\CABINET VISION\CV 2024\CVBackup.exe /e /d

      • /g Suppress GUI (Silent backup that uses last settings)

        • Example: C:\Program Files\Hexagon\CABINET VISION\CV 2024\CVBackup.exe /g

      • /p Suppress prompt when you have exceeded backup settings (useful option to use with /g)

        • Example: C:\Program Files\Hexagon\CABINET VISION\CV 2024\CVBackup.exe /g /p

      • /r Make files read only

        • Example: C:\Program Files\Hexagon\CABINET VISION\CV 2024\CVBackup.exe /r

      • /z Creates a Zip file of the Backup

        • Example: C:\Program Files\Hexagon\CABINET VISION\CV 2024\CVBackup.exe /z

  1. CABINET VISION 2024 - Launches CABINET VISION.

  2. Catalog Editor for CV 2024 - A utility designed to facilitate Catalog maintenance (basic version) and for creating Stock/Modular/Distributable Catalogs (full version). See the Catalog Editor Help System for more information.

  3. CV 2024 Help - Opens this Help System.

  4. Network Administrator - A utility that manages User Access Rights for CABINET VISION.

  5. Restore CV 2024 Settings - Allows the Restoration of backups created by using the CV Backup Utility or Automatic Backups that are created automatically during Installations.

  1. System Information - A utility utilized by Technical Support to diagnose, troubleshoot, and repair Software Settings.

  2. Update Version for CV 2024 - Launches the Update Version Utility which compacts and repairs CABINET VISION Database Files.

  3. xRender for CV 2024 - Opens the xRender App for CABINET VISION.
