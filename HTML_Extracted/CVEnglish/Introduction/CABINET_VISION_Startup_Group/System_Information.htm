System Information

The System Information Utility is a support tool that allows the user to view information such as file version and registry data on the Product line.

Application Menu

File

Run Program

This command launches an executable file that is selected in the Application Tree.

Open Folder

Launches Windows Explorer opened to the selected directory.

Exit

Closes the System Information Utility.

Edit

Allow Editing

This feature can be turned on to allow editing .INI files, registering files, un-registering files, and changing the system registry.  This feature requires a password and should be limited to use by CABINET VISION technicians.

Register

To register a file, first select that file and then select File | Register or press F6.

Un-register

To un-register a file, first select that file and then select File  | Un-Register or press F7.

Find

Selecting Find will open the dialog box shown below.  This allows the user to more easily find and item in the application tree or information window.

Find Next

This option will allow you to easily find the next instance of the item searched for last with the Find option.

View

Register Icon

To assist in identifying what files can be registered, this option may be selected.  When this option is selected, the icon next to each .dll and .ocx file will contain a small blue box.  Additionally, a + symbol will appear in the attributes column.

Register Confirmation

Selecting the Register Confirmation option will cause you to receive a confirmation (or error) each time that you attempt to register a file.

Refresh

This option updates the information pane with the most current data.

Tools

Key Test

Opens the Key Test Utility that is used as part of the procedure to troubleshoot problems with the legacy security key.

Job File Version

Opens the File Version Utility that can be used to find out the Version of CABINET VISION a Job was created in.

Windows System Information

Opens the Windows System Information Utility.

File Signature Verification Utility

Opens the Windows File Signature Verification Utility. Not available in newer versions of Windows.

DirectX Diagnostic Tool

Opens the Windows DirectX Diagnostic Tool.

Event Viewer

Opens the Windows Event Viewer.

System Configuration Utility

Opens the Windows System Configuration Editor for editing critical system files. Not available in newer versions of Windows.

Display Property Settings

Opens the Display Properties Window to allow graphics card configuration.

Registry Editor

Opens the Windows Registry Editor.  Do not make Registry changes unless you are qualified to do so, catastrophic system damage can occur.

Links

This menu is no longer used.

Help

About

Shows information about the System Information Utility.

Application Tree

Use the Application Tree to navigate through the file structure of the installed Hexagon Production Software Products.

Information Window

This area shows the details of items selected in the Application Tree.

Information/Status Bar

At the bottom of the System Information Window, the Information/Statusbar shows the current selection. Also, there is a quick filter. The Quick Filter can be used to reduce the contents of the data window. By typing a word in the Quick Filter, the Information Window will only show items that have that word or part of a word in their name. To remove the filter you can either backspace over the text or click the red X to the right of the filter text.
