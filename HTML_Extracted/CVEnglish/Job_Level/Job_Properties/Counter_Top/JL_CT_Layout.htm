Counter Top Layout

Without  or CV Design with Simple Counter Tops enabled.

With  or CV Design with Simple Counter Tops Enabled

Under Automatic Tops, you should enter values for Face Overhang, End Overhang, and Minimum Depth. You can also specify if the AutoBuild Top Feature should automatically add Splashes.

These settings refer to the relationship of the Counter Top to the Base Cabinet below, and apply only when the Auto-Build feature is used. When Manually building Tops these values do not apply.

Splash - Height - Sets the Height of the Splash.

Splash - Thickness - Sets the Thickness of the Splash

Top - Thickness - Sets the Thickness of the Countertop.

Joint - Miter - Establishes that countertops joining in corners utilize miters.

Joint - Butt - Establishes that countertops joing in corners use butt joints.
