Closet Materials

The Materials tab in the Closet Category is the place to choose the Assembly Parts Material Schedule for your Closets. You will also note that there is a button labeled "Modify for this Job". The function of this button is to make changes to the Assembly Parts Material Schedule that will apply ONLY to this Job. Use this feature for one time only changes that you will not be using again in the future. If you need to make changes that you will probably need to make in future Jobs as well, you should create a new Assembly Parts Material Schedule from the Catalogs menu at the System Level.
