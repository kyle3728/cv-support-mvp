Job Properties Toolbar

There are four options in the Job Properties toolbar:

Save Job Property Template - Allows you to save all of the current selections within Job Properties for easy recall for future Jobs.

Job Property Template files are stored as .CFG files in the Database folder. For Network Users the .CFG files are stored in the \Cabinet Vision\Users\Username folder on the Server.

Select Job Property Template - Allows you to select the Job Property Template to use for the Job.

Rename Job Property Template - Allows you to Rename the current Job Property Template.

Delete Job Property Template - Allows you to delete the currently selected Job Property Template.
