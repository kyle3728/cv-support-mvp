Room General

The General Tab allows changing the name from the default "Room 1", Room 2", etc. to a more descriptive name.  Above the Room Name has been changed by clicking the "Name" box, and typing the new name in. Also, the Type of "Room" has been changed to "Floor" indicating that "Jim's Office" is an entire floor of  a multi-floored job-site. This is a cosmetic feature and changes as shown are not necessary in order to do a Job. If you do choose to change the Name, it will appear at the Layout Level.

Material

This option allows you to select the Rooms Material Schedule this Job will use.

Start Cabinet Numbering at

The "Room" option "Start Cabinet Numbering At"  allows custom number sequencing for Rooms within Solid. Assume each Room is to start at a unique value, 100 for room one, 200 for room two, as an example. You would first enable the "Start Cabinet Numbering at" check box. The data box will activate (turn white) and the beginning number can be entered for the Room.

Here, Jim's Office is set to start at Cabinet number 100. Again, the number selected must be one that is previously not assigned. This option should be set prior to placing Cabinets in the Room.
