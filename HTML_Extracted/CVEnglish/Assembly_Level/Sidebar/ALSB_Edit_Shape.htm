When an Assembly is brought from a Catalog and placed on a Wall in a Room, it exists as a standard Assembly or a shaped Assembly. Typically, the corner Cabinets, 45's and 90's are already shaped Assemblies. The remainder of the Assemblies are standard Assemblies. Understanding the advantages of both types of Assemblies allows for greater editing capabilities. There are a few points to understand when defining the Assembly types, the more important of which are:

  • A shaped Assembly need not be "shaped" differently than a standard Assembly, although it may be.

  • Every Assembly, shaped or otherwise, must have a minimum of one Face.

Shaping an Assembly (creating a shaped Assembly)

You must shape an Assembly if the Assembly requires an irregular shape. This shaping can be done in any of the three orthographic views or it can be done at the Plan or Elevation view of the Room level. The view that you choose to edit the shape from MUST be the view that represents the unique shape. In other words, if you want to edit the shape of a the Assembly so as to appear as a roll top desk, you would have to choose the orthographic view allowing access to the Assembly from the side. There are a few tips and tricks that make the process easier to accomplish.

Always edit an object AFTER you have correctly sized it to it's final size. In many cases, resizing an object will trigger a refresh which can rebuild your Assembly to a default configuration. This is avoided by sizing the object to the correct size first. A matter of an inch or two will generally not have that effect on a shaped object, however, whenever possible, get the dimensions close.

There are several options specific to editing the Shape of an Assembly which should be noted.

Selecting a Point on an Assembly's Shape

When selecting a Point on an Assembly's shape you are presented with the following sidebar options:

Start X - Allows you to control the X Position of the Point. You can also activate "Simple Constraints" on the Point. See the Constraints topic for more information.

Start Y - Allows you to control the Y Position of the Point. You can also activate "Simple Constraints" on the Point. See the Constraints topic for more information.

Selecting a CAD Entity on an Assembly's Shape

When selecting a CAD entity on an Assembly's shape you are presented with the following sidebar options:

Start X - Allows you to control the Starting X Position of the selected CAD entity. You can also activate "Simple Constraints" on the Point. See the Constraints topic for more information.

Start Y - Allows you to control the Starting Y Position of the selected CAD entity. You can also activate "Simple Constraints" on the Point. See the Constraints topic for more information.

End X - Allows you to control the Ending X Position of the selected CAD entity. You can also activate "Simple Constraints" on the Point. See the Constraints topic for more information.

End Y - Allows you to control the Ending Y Position of the selected CAD entity. You can also activate "Simple Constraints" on the Point. See the Constraints topic for more information.

Length - Allows you to control the Length of the selected CAD entity.

Angle - Allows you to control the Angle of the selected CAD entity.
