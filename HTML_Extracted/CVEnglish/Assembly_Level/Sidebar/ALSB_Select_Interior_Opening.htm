Select Interior Opening Section View - Section Interior

When selecting an Interior Opening in the Section Editor you are presented with the following sidebar:

You can, if you select one of many Interior Openings within an Assembly, control four different Dimensional properties as well as three Opening Types:

Dimensional Properties

  • Width - Allows you to change the Width of the selected Opening, if there is more than one.

  • Height - Allows you to change the Height of the selected Opening, if there is more than one.

  • Lock Width - Allows you to lock the Width of the selected Opening, if there is more than one.

  • Lock Height - Allows you to lock the Height of the selected Opening, if there is more than one.

Opening Types

  • Open - Selecting this option will change the selected Opening to an Open section.

  • Hanging Section - - Selecting this option will change the selected Opening to a Hanging Section. Only available on a Closet Class Object.

  • Shoe Cubby - Selecting this option will change the selected Opening to a Shoe Section.

  • Object - Selecting this option will change you to Object mode so you can place a Catalog Object as a Sub-Assembly which will fill the selected opening. Once the object is selected you can either double click it, drag it, or click the add button to add it to the selected opening. Once place the object can be edited using the Assembly Selector in the ribbonbar.
