New Part -

In any of the three orthographic views, you may add a New Part manually to the Assembly by clicking the New Part button.

The New Part wizard will begin. In the General area of the New Part Wizard, you will select the Part Group, then the specific Part Type.

Following are the options in the General area:

  • Part Group - Select the Part Group to select the new Part from. These, in general, correspond to the Part Manager Part Categories.

  • Part Type - Select the Part you wish to add. The Parts here come directly from the Part Manager.

  • Description - By Default the Description for the new Part will come from the Part Manager and is what is displayed in your Reports. However, this field is editable and you may change the Description to better describe "this" new Part in your Reports.

  • Material Change - By Default the Material for the new Part will come from the Material Schedule for the Part Group that the Part is selected from.

  • Texture Map - Allows you to set how a Texture is applied to the new Part. See the Room Level Import DXF Texture Map topic for more information.

  • Smooth Surface - Allows you to enable Smooth Surfacing to the new Part. See the Room Level Import DXF Smooth Surface topic for more information.

  • Layout Visibility

    • Hide in Plan - Will hide the new Part in Plan views.

    • Hide in Elevation - Will hide the new Part in Elevation views.

    • Hide in Perspective - Will hide the new Part in Perspective (3D) views.

  • Allow UCS Modification - By Default a UCS can not see a User added "New" Part. This option will allow a UCS to "see" the Part. This option adds the Parameter UCSMOD with a value of 1 on your New Part.

Next is the Size area, enter the dimensions for the new Part that you are adding.

Following are the options in the Size area:

  • Length - Defines the Length of the new Part.

  • Width - Defines the Width of the new Part.

  • Thickness - Defines the Thickness of the new Part. By default this value comes from the selected Material for the Part. It is recommended, especially if this Part goes to a CNC, that this value be allowed to come from the Material.

  • Material Thickness - Instructs the Part Wizard to get the new Parts Thickness from the Material selected for the Part. This is the default and recommended selection.

  • Relate to Assembly Part - This brings up the "Relate to Part Wizard". See the Right Click an Ortho Part Relate to Part topic for more information.

Next is the Orientation area, select the Orientation that you wish the new Part to have.

Finally is the Finish area, select the Finish that you want to have applied to each face of the new Part.

Selecting Interior or Exterior instructs CABINET VISION to use the Finishes defined at the Room/Assembly Finishes Properties for this new Part. Using the Select option allows you to select a specific Finish for the new Part

After you have added the Part to your Assembly, you can change the position of the Part within the Assembly. You may also change the size and orientation of the Part. These controls are found in the sidebar when the part is selected. You also, using Object Intelligence, can give your Parts intelligence.
