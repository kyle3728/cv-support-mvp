Select a Vertical Split Section View - Section Interior

When selecting an Object placed utilizing the Vertical Split option in a Case Opening in the Section Editor you are presented with the following sidebar options:

  • Split Top - Allows you control whether the Partition splits the Top or not. (Only available if the Partition touches the Top.)

  • Split Bottom - Allows you control whether the Partition splits the Bottom or not. (Only available if the Partition touches the Bottom.)

  • Type Partition - Selecting Partition will change the selected Part to a standard Partition as defined in the Construction Method.

  • Type Double Partition - Selecting Double Partition will change the selected Part to a Double Partition. If your Construction Methods is Face Frame or Frame Overlay and the Partition is associated to a Mid Stile then you will get several more options in the Position section (see Boxed, Left Scribe, and Right Scribe below).

  • Type Divider -  Selecting Divider will change the selected Part to a standard Divider as defined in the Construction Method.

  • Position Relative - Lets you choose which side of the Opening the Partition is oriented to. (Float, Left, Center, Right, or Default)

  • Position Boxed - Allows you to choose if you are Boxing your Double Partitions. By Default Double Partitions are sandwiched together like one piece. Boxed Double Partitions will actually move away from each other out flush to the edges of the Mid Stile.

  • Position Left Scribe - When using a Partition set Relative Left or Boxed Double Partitions (Face Frame or Frame Overlay Only) this option allows you to set the Scribe for the Relative Left Partition in relation to the Left Edge of the Mid Stile it is associated to.

  • Position Right Scribe - When using a Partition set Relative Right or Boxed Double Partitions (Face Frame or Frame Overlay Only) this option allows you to set the Scribe for the Relative Right Partition in relation to the Right Edge of the Mid Stile it is associated to.

  • Adjustments Depth - Allows you to adjust the Depth of the Partition to or from the Cases Face. (Default value Controlled by Construction Method)

  • Adjustments Top - Allows you to adjust the Length of the Partition in to or out of the Top. (Default value Controlled by Construction Method)

  • Adjustments Bottom - Allows you to adjust the Length of the Partition in to or out of the Bottom. (Default value Controlled by Construction Method)

  • Hide - Allows you to "Hide" the Partition. Please note, you are not Deleting the Partition, just hiding it. Everything acts like the Partition is still there it is just not visible and will not be on your cut list.
