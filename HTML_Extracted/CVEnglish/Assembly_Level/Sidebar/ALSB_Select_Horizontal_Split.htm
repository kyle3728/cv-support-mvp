Select a Horizontal Split Section View - Section Interior

When selecting an Object placed utilizing the Horizontal Split option in a Case Opening in the Section Editor you are presented with the following sidebar:

Cabinet

Closet

Following are the possible Horizontal Part Types:

  • Shelf - Selecting Shelf will change the selected Part to a Shelf and present you with the following sidebar options:

    1. Position Relative - Lets you choose which side of the Opening the Shelf is oriented to. (Float, Top, Center, Bottom, or Default)

    2. Adjustments Depth - Allows you to adjust the Depth of the Shelf to or from the Cases Face. (Default value Controlled by Construction Method)

    3. Adjustments Left - Allows you to adjust the Width of the Shelf to or from the Left. (Default value Controlled by Construction Method)

    4. Adjustments Right - Allows you to adjust the Width of the Shelf to or from the Left. (Default value Controlled by Construction Method)

  • Adjustable Shelf - Selecting Adjustable Shelf will change the selected Part to an Adjustable Shelf and present you with the following sidebar options.

    1. Position Relative - Previously covered.

    2. Float Shelf/Holes - Enables or disables Shelf alignment forced to linebore, the default value is controlled in the Construction Method.

    3. Adjustments Depth - Previously covered.

    4. Adjustments Left - Previously covered.

    5. Adjustments Right - Previously covered.

  • Glass Adj Shelf - Selecting Glass Adj Shelf will change the selected Part to a Glass Adjustable Shelf Part (Which will follow the Adjustable Shelf Assembly Wizard Properties) and present you with the following sidebar options.

    1. Position Relative - Previously covered.

    2. Float Shelf/Holes - Previously covered.

    3. Adjustments Depth - Previously covered.

    4. Adjustments Left - Previously covered.

    5. Adjustments Right - Previously covered.

  • Fixed Shelf - Selecting Fixed Shelf will change the selected Part to a Fixed Shelf and present you with the following sidebar options.

    1. Position Relative - Previously covered.

    2. Adjustments Depth - Previously covered.

    3. Adjustments Left - Previously covered.

    4. Adjustments Right - Previously covered.

  • Roll Out Shelf - Selecting Roll Out Shelf will change the selected Part to a Roll Out Shelf and present you with the following sidebar options:

    1. Position Relative - Previously covered.

    2. Float Shelf/Holes - Previously covered.

    3. Adjustments Depth - Previously Covered. (Default value Controlled by Shelf Construction Method)

    4. Adjustments Left - Previously Covered. (Default value Controlled by the selected Drawer Guide’s Side Clearance.)

    5. Adjustments Right - Previously Covered. (Default value Controlled by the selected Drawer Guide’s Side Clearance.)

  • Roll Out Tray - Selecting Roll Out Tray will change the selected Part to a Roll Out Tray and present you with the following sidebar options:

    1. Position Relative - Previously covered.

    2. Float Shelf/Holes - Previously covered.

    3. Adjustments Depth - Previously Covered. (Default value Controlled by Roll-out Construction Method Back Clearance)

    4. Adjustments Left - Previously Covered. (Default value Controlled by the selected Drawer Guide’s Side Clearance.)

    5. Adjustments Right - Previously Covered. (Default value Controlled by the selected Drawer Guide’s Side Clearance.)

    6. Adjustments Set Back - Allows you to adjust the amount that the Roll Out Tray sets back from the front of the face. (Default value is 0)

  • Shoe Shelf - - Selecting Shoe Shelf will change the selected Part to a Shoe Shelf and present you with the following sidebar options. Only available on a Closet Class Object.

    1. Position Relative - Previously covered.

    2. Adjustments Depth - Previously covered.

    3. Adjustments Left - Previously covered.

    4. Adjustments Right - Previously covered.

    5. Angle - Allows you to establish the Tilt of the Shoe Shelf as an additional adjustment to the current tilt.

  • Wire Basket - - Selecting Wire Basket will change the selected Part to a Wire Basket. CABINET VISION will make a determination based on the Wire Baskets specified in your Wire Basket Material Schedule the optimum Wire Basket for the Opening it is placed within. Only available on a Closet Class Object.

  1. Adjustments Set Back - Allows you to adjust the amount that the Roll Out Tray sets back from the front of the face. (Default value is 0)

  • Hanging Rod - - Selecting Hanging Rod will change the selected Part to a Hanging Rod and present you with the following three sidebar options.

    1. Bore Style - Lets you choose the Bore Style to use with the Hanging Rod. There are four options:

      • Required - Uses the Boring Specified for the selected Closet Rod from the Material Catalog.

      • Additional - Will bore and additional row of Line Bore for the Closet Rod as defined in the Line Bore of the Construction Method.

      • Existing - Will use the Line Bore specified in the Construction Method.

      • None

    2. Horiz. Ref - Lets you specify whether the Rod is referenced from the Front or Rear of the Assembly Horizontally.

    3. Horizontal - Lets you specify the position Horizontally of the Rod from the specified Horizontal Reference.

  • Drawer Stretcher - Selecting Drawer Stretcher will change the selected Part to a Drawer Stretcher and present you with the following four previously covered sidebar options.

    1. Position Relative - Previously covered.

    2. Adjustments Depth - Previously covered.

    3. Adjustments Left - Previously covered.

    4. Adjustments Right - Previously covered.

  • Top - Selecting Top will change the selected Part to a Top and present you with the following four previously covered sidebar options.

    1. Position Relative - Previously covered.

    2. Adjustments Depth - Previously covered.

    3. Adjustments Left - Previously covered.

    4. Adjustments Right - Previously covered.

  • Bottom - Selecting Bottom will change the selected Part to a Bottom and present you with the following four previously covered sidebar options.

    1. Position Relative - Previously covered.

    2. Adjustments Depth - Previously covered.

    3. Adjustments Left - Previously covered.

    4. Adjustments Right - Previously covered.
