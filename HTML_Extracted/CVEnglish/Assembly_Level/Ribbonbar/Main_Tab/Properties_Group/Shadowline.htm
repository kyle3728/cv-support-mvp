Shadowline

The Shadowline Tab at Cabinet Properties has the following options:

  1. Use Shadowline - Determines if the Shadowline is active on the current Cabinet.

  2. Door - No - Determines if the Doors require a Shadowline.

  3. Door - Horizontal - Determines if the Doors get Horizontal Shadowlines.

  4. Door - Horizontal - Top - Determines if the Doors get Horizontal Shadowlines at the top.

  5. Door - Horizontal - Middle - Determines if the Doors get Horizontal Shadowlines at the middle.

  6. Door - Horizontal - Bottom - Determines if the Doors get Horizontal Shadowlines at the bottom.

  7. Door - Vertical - Determines if the Doors get Vertical Shadowlines.

  8. Door - Vertical - Edges - Determines if the Doors get Vertical Shadowlines at the edges.

  9. Door - Vertical - Middle - Determines if the Doors get Vertical Shadowlines at the middle.

  10. Drawer - No - Determines if the Drawers require a Shadowline.

  11. Drawer Horizontal - Determines if the Drawers get Horizontal Shadowlines.

  12. Drawer Horizontal - Top - Determines if the Drawers get Horizontal Shadowlines at the top.

  13. Drawer Horizontal - Middle - Determines if the Drawers get Horizontal Shadowlines at the middle.

  14. Drawer Horizontal - Bottom - Determines if the Drawers get Horizontal Shadowlines at the bottom.

  15. Drawer - Vertical - Determines if the Drawers get Vertical Shadowlines.

  16. Drawer - Vertical - Edges - Determines if the Drawers get Vertical Shadowlines at the edges.

  17. Drawer - Vertical - Middle - Determines if the Drawers get Vertical Shadowlines at the middle.

  18. Pair Door - Determines if a Pair Door gets a Shadowline.

 

Remember changes made here will only affect the current Assembly. Also, changes made here are attached as permanent (unless you remove them) overrides on the Assembly; which means that if you Save this Assembly into a Catalog these overrides will also be Saved.
