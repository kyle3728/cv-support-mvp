General

The General Tab at Cabinet/Closet Properties has the following options:

  1. Name - You can change the name of the Assembly here.

  2. Quantity - You can change the quantity of the Assembly here.

  3. Description - You can change the description of the Assembly here.

  4. Class - You can change the Assemblies Class here.

  5. Type - You can change the Assemblies Type here.

  6. Price - You can add a Price for the Object here. This value is retrievable in the Bid Center using the Use Cabinet Catalog Pricing Bid Method in your Rate Table.

  7. Labor (minutes) - - You can set an Assembly Labor value here. It must be specified in minutes. This value is retrievable in the Bid Center using the Use Cabinet Assembly Labor line item in your Rate Table.

  8. Size Width - You can change the Width of the Assembly here.

  9. Size Height - You can change the Height of the Assembly here.

  10. Size Depth - You can change the Depth of the Assembly here.

 

Remember changes made here will only affect the current Assembly. Also, changes made here are attached as permanent (unless you remove them) overrides on the Assembly; which means that if you Save this Assembly into a Catalog these overrides will also be Saved.
