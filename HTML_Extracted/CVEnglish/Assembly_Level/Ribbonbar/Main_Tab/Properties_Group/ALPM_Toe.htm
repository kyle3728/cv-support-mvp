Toe

    

The Toe Tab at Cabinet/Closet Properties has the following:

  1. Toe Height - Allows you to specify the Toe Height for the active Assembly.

  2. Front Recess - Allows you to specify the Front Toe Recess for the active Assembly.

  3. Back Recess - This will set the Recess of the Back Toe in relation to the Wall.

  4. Number of Sleepers - Allows you to specify the Quantity of Sleepers and/or the ability to Delete them for the active Assembly.

  5. Left Recess - If utilizing a Finished or Unfinished Toe as on a separate Toe Assembly this will specify the Recess for the Left Side Toe.

  6. Right Recess - If utilizing a Finished or Unfinished Toe as on a separate Toe Assembly this will specify the Recess for the Right Side Toe.

  7. Left Side Toe - There are four options for the Left Side Toe:

    1. Unfinished - If utilizing a separate Toe Assembly this will set the Left Side Toe as Unfinished.

    2. Finished - If utilizing a separate Toe Assembly this will set the Left Side Toe as Finished.

    3. Notched End Toe - This option will set the Left End to run to the floor and be notched for the Front Toe Kick.

    4. Full End - This option will set the Left End to the floor without a Toe Notch.

  8. Right Side Toe - There are four options for the Right Side Toe:

    1. Unfinished - If utilizing a separate Toe Assembly this will set the Right Side Toe as Unfinished.

    2. Finished - If utilizing a separate Toe Assembly this will set the Right Side Toe as Finished.

    3. Notched End Toe - This option will set the Right End to run to the floor and be notched for the Front Toe Kick.

    4. Full End - This option will set the Right End to the floor without a Toe Notch.

  9. Back Toe - There are three options for the Back Toe:

    1. None - This will remove the Back Toe.

    2. Unfinished - If utilizing a separate Toe Assembly this will set the Back Toe as Unfinished.

    3. Finished - If utilizing a separate Toe Assembly this will set the Back Toe as Finished.

  10. Left Sub End Toe

    1. Notched End - Sets the left Sub End to a notched end.

    2. Full End - Sets the left Sub End to a full end.

  11. Right Sub End Toe

    1. Notched End - Sets the right Sub End to a notched end.

    2. Full End - Sets the right Sub End to a full end.

 

Remember changes made here will only affect the current Assembly. Also, changes made here are attached as permanent (unless you remove them) overrides on the Assembly; which means that if you Save this Assembly into a Catalog these overrides will also be Saved.
