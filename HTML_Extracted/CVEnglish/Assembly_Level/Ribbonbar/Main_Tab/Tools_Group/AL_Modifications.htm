Modifications

There are six Modifications options:

Overrides

The Overrides option will display any Overrides that have been placed on the current Assembly. You can remove them by highlighting the item in the list and clicking the Remove button.

Default Assembly

A default Assembly is an Assembly that has been reconstructed by CABINET VISION at the Assembly Level, following the information you have entered in the Assembly Manager and Assembly Parts Material Schedule for the Assembly in question. Additionally, the Assembly is absent features such as Face configuration and Interior options. It is, in short, a box built exactly as you specified when you setup the Construction Method that is effective for the Assembly. Another advantage is that the Default Assembly sheds any unintentional overrides that may have accidentally become part of the Assembly. In practice, it is probably a good idea to begin the process of sectioning an Assembly into a unique configuration by first defaulting the Assembly.

Also note there are two options when Defaulting an Assembly. By default the Default Assembly option will not Remove an Assemblies Shape or any user Added or user Modified Parameters. If you would like them removed as well simply enable these options.

After defaulting the Assembly, it will appear similarly to the Base Cabinet below.

Adjust Parts

The Adjust Parts option allows you to make dimensional and/or positional changes to standard Assembly (Default Assembly) Parts.

There are six main options in the Case Part Adjustments area:

  • Top Depth - Enter a value here to add to or subtract from the current Top Depth.

  • Bottom Depth - Enter a value here to add to or subtract from the current Bottom Depth.

  • Back

    • Height - Enter a value here to add to or subtract from the current Back Height.

    • Elevation - Enter a value here to add to or subtract from the current Back Elevation.

  • Left End

    • Height - Enter a value here to add to or subtract from the current Left End Height.

    • Depth - Enter a value here to add to or subtract from the current Left End Depth.

    • Elevation - Enter a value here to add to or subtract from the current Left End Elevation.

  • Right End

    • Height - Enter a value here to add to or subtract from the current Right End Height.

    • Depth - Enter a value here to add to or subtract from the current Right End Depth.

    • Elevation - Enter a value here to add to or subtract from the current Right End Elevation.

  • Toe Face

    • Position - Enter a value here to add to or subtract from the current Toe Face Position.

    • Length - Enter a value here to add to or subtract from the current Toe Face Length.

    • Width - Enter a value here to add to or subtract from the current Toe Face Width.

As an example of how this may be used, assume a Wall Cabinet above the refrigerator needs to have ends that extend to the floor. There are several methods to create this Cabinet, however, using the Parts Adjustment menu allows an easy method to create the new feature. Because the Cabinet is 69 inches off the floor, I need the ends to increase in length 69 inches. By placing that value in the box marked "Height" for each End, Right and Left, I have increased the length of the ends to 84 inches (69 plus the original 15). Hide Parts

Parts that are added to an Assembly by CABINET VISION that are a part of the Assembly Construction Method are referred to as system Parts. Typically, system Parts lack a "delete" feature for removal. The Hide Parts option allows hiding Parts by placing a check in the box next to the Part to be hidden. To restore a hidden Part, re-access this option and remove the check mark. Notice that the Parts are grouped. Check marks in the group will hide all Parts within the group. If you want to hide only Front Toes, you would not benefit from clicking the Toe Parts group as all Toe Parts would be hidden, instead, select "Front Toe" from the Toe Parts Group and limit the hide to the specific Part you need to hide.

Please note that the keyword here is "Hide" the Assembly still reacts and is constructed as if the Part were still visible.

 

Modify Assembly

The Modify Assembly option has three main options.

Radius

You can specify a Radius value here and then whether you want the Assembly radiused on the inside or outside. This will radius both the Face and Back of the Assembly as shown below:

Inside Radius

Outside Radius

End Clip

The Left and Right Clip option will cause a 45 degree cut across the face of an Assembly with the legs of the angle equal to the value you place in the box. Shown is the default value of 4 inches, which would produce a 4 inch deep by 4 inch wide clip with an angle face of approximately 5 5/8 inches (at 45 degrees).

End Radius

The Left and Right Radius option creates a curved shape at the front of your Assembly with a radius equal to the value you enter.

End Angle

The Left and Right Angle option differs in that it produces a cut that runs from the rear of the Assembly to the Face, at the angle you enter.

If you desire to work on both ends of an Assembly, it is important to do so at the same time. Once you select the Clip, Angle, or Radius options for either end, you will not thereafter, have access to this menu. This menu is available for Assemblies that have not been shaped, and utilizing any of the options here causes the Assembly to become a shaped Assembly.

Reverse Assembly

This option will Reverse/Mirror the current Assembly.

Reverse Assembly, at this time, will not Reverse user modifications and overrides, this would include user added Parts and Formulas.
