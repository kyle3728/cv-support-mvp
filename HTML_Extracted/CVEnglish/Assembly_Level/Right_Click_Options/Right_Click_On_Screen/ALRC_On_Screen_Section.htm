Right Click On Screen -

 

Following are the options when right clicking the screen in the Section Editor:

  •  Assembly - See the Assembly Properties topic for more information.

  •  To Drawing - See the Room Level To Drawing topic for more information.

  •  Save - See the Room Level Save topic for more information.

  •  Save As - See the Room Level Save As topic for more information.

  •  Properties - See the Room Level Layers topic for more information.
