Right Click On Screen -

 

Following are the options when right clicking the screen in 3D view:

  •  To Drawing - See the Room Level To Drawings topic for more information.

  •  To Live Drawing - - See the Room Level To Live Drawing topic for more information.

  •  Save Camera - This option allows you to save the current 3D View as a Camera view. This option saves the Zoom, Rotation, and Pan position of the current 3D View. Use the Camera View controls in the Ribbonbar to work with Camera Views you have Saved.

  •  Save for Assembly Sheet - This option will Save the current view for the active Assembly for use in Assembly Sheets when using the "Additional Saved Views" option.

  •  Properties - There is one option in addition to the normal 3D View options available on the General Tab.

Explode Banding From Part

This option allows you to have Edgebanding Explode when you Explode the Assembly or not.

See the Room Level 3D Screen Properties topic for more information on the other 3D View Properties.
