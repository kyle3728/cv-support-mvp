Right Click Drawer Section View - Section Face

 

In addition to the standard six right click Opening options you get the Properties option. The Properties option for a Drawer has four tabs:

Door Tab

Hinges Tab

Pulls Tab

Drawer Box Tab

There are four options on the Drawer Box Tab:

  • Construction - Allows you to select which Drawer Box Construction Method to use for the selected Drawer Box.

  • Material - Allows you to select which Drawer Box Material Schedule to use for the selected Drawer Box.

  • Guide Type Change - Will bring up the Drawer Guide selector and allow you to change the Drawer Guide for the selected Drawer.

  • Edit Drawer Box - Will open the Drawer Box Editor. This utility will allow you to place Dividers and Partitions, as defined in your Drawer Box Construction Method, in your Drawer.

This option works just like the Assembly Section Editor. You can add Single Horizontal and Vertical Splits as well as Multiple. Also when Openings are highlighted, as shown above, you can control the Width and Depth as well as Lock the Width and Depth of the selected Opening.

Right Click Section Editor Screen

There are two options when right clicking the Drawer Section Editor screen:

To Drawing - Will send the current Drawer Box Section view to the Drawings area.

Properties - This option opens up View Properties allowing you to configure the Drawer Box Section Editor view.

There is one option when editing the Dimensions not previously covered and that is the "Show Splitter" option. This option when enabled will show the thickness of the Vertical/Horizontal Dividers in the Dimensions.
