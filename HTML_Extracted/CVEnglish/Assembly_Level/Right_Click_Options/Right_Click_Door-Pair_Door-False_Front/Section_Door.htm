Section Door

This utility will allow you to not only re-section the current Door but also gives you control over quite a few of the Door's Properties.

Toolbar

Horizontal Split - Splits the selected Door/Panel Horizontally.

Vertical Split - Splits the selected Door/Panel Vertically.

Multiple Horizontal Split - Splits the selected Door/Panel Horizontally into multiple panels. Once you select the Panel to split simply enter the number of equal sections you want to create and click OK.

See the Set As System Defaults topic for more information on it.

Multiple Vertical Split - Splits the selected Door/Panel Vertically into multiple panels. Once you select the Panel to split simply enter the number of equal sections you want to create and click OK.

See the Set As System Defaults topic for more information on it.

Delete - If nothing (Panel/Stile/Rail) is selected it will erase all sections on the Door, otherwise it will delete the selected components.

Right Click Section Editor Screen

There are two options when right clicking the Door Section Editor Screen:

To Drawing - Will send the current Door Section view to the Drawings area.

Properties - This option opens up View Properties allowing you to configure the Door Section Editor view.

There are two options when editing the Dimensions not previously covered:

  • Show Splitter - This option, when enabled, will show the width of the Mid Stiles and Mid Rails in the Dimensions.
  • Show Stile/Rail - This option, when enabled, will show the End Stiles and Top and Bottom Rail Dimensions.

Door Properties

With nothing selected on screen you will see the following sidebar Properties for the active Door.

  • Door - Template - Indicates whether the current Door is a Template Door. This value can not be changed in the Door Section Editor.

  • Door - Dimensions - Width - Indicates the overall Width of the Door. This value can not be changed in the Door Section Editor.

  • Door - Dimensions - Height - Indicates the overall Height of the Door. This value can not be changed in the Door Section Editor.

  • Door - Frame - Left Stile - Specifies the Left Stile Width.

  • Door - Frame - Right Stile - Specifies the Right Stile Width.

  • Door - Frame - Top Rail - Specifies the Top Rail Width.

  • Door - Frame - Bottom Rail - Specifies the Bottom Rail Width.

  • Door - Frame - Miter Corner - Specifies whether the Corners of the perimeter Stiles and Rails are Mitered.

  • Door - Profiles - Outside - Specifies the Outside Edge Profile to use for the Door.

  • Door - Profiles - Inside - Specifies the Inside Edge Profile to use for the Door.

  • Door - Profiles - Panel - Specifies the Raised Panel Profile to use for the Door.

  • Door - Profiles - Route - Specifies the Route Detail to use for the Door. Unless a pre-existing Route exists on the Door this will not change anything.

Stile Properties

Perimeter (Left/Right) Stile Selected

Mid Stile Selected

  • Stile - Dimensions - Width - Specifies the Width of the selected Stile.

  • Stile - Dimensions - Length - Indicates the Length of the selected Stile. This value can not be changed in the Door Section Editor, but can be adjusted with Stile - Adjustments - Length.

  • Stile - Adjustments - Length - Specifies a Length Adjustment to the selected Stile.

  • Stile - Material - Material - Specifies the Material used by the selected Stile.

  • Stile - Material - Finish - Specifies the Finish used by the selected Stile.

Rail Properties

Perimeter (Top/Bottom) Rail Selected

Mid Rail Selected

  • Rail - Dimensions - Width

  • Rail - Dimensions - Length

  • Rail - Adjustments - Length

  • Rail - Material - Material

  • Rail - Material - Finish

Panel Properties

  • Panel - Dimensions - Width

  • Panel - Dimensions - Height

  • Panel - Dimensions - Lock Width - Will lock the current Width of the selected Panel.

  • Panel - Dimensions - Lock Height - Will lock the current Height of the selected Panel.

  • Panel - Panel - Type - Specifies the Panel Type for the selected Panel. Following are the valid Panel Types:

    • Raised (Raised Panel)

    • Flat (Flat Panel)

    • Glass

    • H Slats (Horizontal Slats)

    • V Slats (Vertical Slats)

    • Slab

    • None

  • Panel - Panel - Horizontal Grain - Specifies that the selected Panel is grained Horizontally.

  • Panel - Insets - Horizontal - Specifies the Horizontal inset of the selected Panel.

  • Panel - Insets - Vertical - Specifies the Vertical inset of the selected Panel.

  • Panel - Material - Material

  • Panel - Material - Finish
