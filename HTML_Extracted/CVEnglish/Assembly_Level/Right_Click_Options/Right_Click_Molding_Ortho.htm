Right Click Molding -

 

Following are the options when right clicking on a Molding in the Face, Plan, and End views:

 Copy

This option will copy the selected Molding and allow you to place the copied Molding somewhere else within the Assembly.

 Delete

This option will delete the selected Molding.

 Save To Library

The Save To Library option allows you to Save the selected Molding to a Part Library for retrieval later. Select the Library from the list on the left where you want to save the selected Molding.

Following are the options here:

  • Preserve Finish - This option will Preserve the current Finishes assigned to the Molding.

  • Preserve Size - This option will Preserve the current Size of the Part into the Library. Without this option enabled when you use Replace with Library Part the Library Part would take on the sizing characteristics of the Part being Replaced.

  • Preserve Material - This option will Preserve the current Material of the Part into the Library. Without this option enabled when you use Replace with Library Part the Library Part would take on the Material characteristics of the Part being Replaced.

  • Preserve Attributes - This option will Preserve any Attributes you have added to the Part.

  • Preserve Notes - This option will Preserve any Notes you have added to the Part.

  • Preserve CAD - This option will Preserve any CAD you have added to the Part.

  • Output to CNC - This option if selected will send this Library Part to the S2M CENTER when the rest of your Job is sent. If this option is disabled the Library Part will not be transferred to the S2M CENTER.

 Edit Shape

The Edit Shape option will bring up the Shaping tools to allow you to shape the Molding.

Click the options below for more information

 

  Properties

There are five tabs in the Part Properties screen:

  • General Tab - Following are the options on the General Tab:

    • Molding Type - Select the Molding Type here.

    • Molding Profile - Select the Molding Profile here. If a Molding Material has been selected then this option will be disabled as the actual Profile comes from the Molding Material.

    • Description - By Default, the Description for the Molding will come from the Molding Manager/Molding Material. However, this field is editable and you may change the Description to better describe "this" Molding in your Reports.

    • Material Change - By Default, the Material for the Molding comes from the Material Schedule.

    • Texture Map - Allows you to set how a Texture is applied to the Molding. See the Room Level Import DXF Texture Map topic for more information.

    • Smooth Surface - Allows you to enable Smooth Surfacing on the Molding. See the Room Level Import DXF Smooth Surface topic for more information.

    • Layout Visibility

      • Hide in Elevation - Will hide the Molding in Elevation views.

      • Hide in Perspective - Will hide the Molding in Perspective (3D) views.

  • Comment Tab - You add special notes on this Molding here. Certain cut lists from the Report Center will display this information for the shop.

  • Finish Tab - The Finish Tab allows you to specify what Finish and/or Texture is applied to the selected Moldings Front Face, Back Face, Edges, and Ends.

  • Overrides Tab - The Overrides Tab will display and allow removal of any Overrides on the selected Molding.

 

  • Parameters - This area allows you to see any Parameters on the current Molding. See the Intelligence Workshop for more information on modifying these Parameters.
