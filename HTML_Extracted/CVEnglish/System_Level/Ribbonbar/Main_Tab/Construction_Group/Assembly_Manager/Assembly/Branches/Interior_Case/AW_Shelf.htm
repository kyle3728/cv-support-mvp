Shelf

A Shelf is a horizontal part that does not adjust. It differs slightly from a Fixed Shelf. You can define a setback for the front of the Shelf.

The following diagram shows the options available in the Shelf Branch. Click a specific option in the diagram for more information or continue through this topic for information on all the options.
