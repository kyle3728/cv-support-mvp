Deck

Typically, "Deck" is reserved for the 'bottom' of a Base Cabinet, where the 'bottom' of an Upper Cabinet is referred to as the Bottom.

The following diagrams show the options available in the Deck Branch. Click a specific option in the diagram for more information or continue through this topic for information on all the options.

Delete Deck/Bottom On Drawer Bank (Standard #012)

You have the option to set up your Construction Method to remove the Deck from a Drawer Bank Cabinet.

Deck/Bottom Finished Side Orientation (Standard #136)

For Decks/Bottoms, you have the ability to determine the Material face orientation if you are using 2 sided Materials for the Bottom of a Cabinet. You can select whether the "good side" faces up or down. Remember to add break outs if needed for the different Classes of Cabinets.

Deck/Bottom Grain Alignment (Standard #137)

Here you have the ability to specify Grain Alignment for Bottoms on 45 Corner Cabinets just like with Shelves.
