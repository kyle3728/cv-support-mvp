Reveals

The Reveals option fully control Door sizes in 32mm, Frameless, Frame Overlay, and Closet 32mm Construction Styles. In standard Face Frame Construction the Hinge Overlay controls the Door Sizes except for the Gap for Pair Doors which is controlled here with the Vertical Reveal (Standard #350). Also, if building Face Frame Cabinets with Inset Doors the Doors size is controlled using this Reveals Branch, the Reveals here are used to set the Inset Gaps.

Reveals  are those gaps that appear between the members of a Frameless Cabinets face. Often 1/8 of an inch, or combinations thereof are used, so that face reveals read 1/8 of an inch on adjoining Cabinets.
