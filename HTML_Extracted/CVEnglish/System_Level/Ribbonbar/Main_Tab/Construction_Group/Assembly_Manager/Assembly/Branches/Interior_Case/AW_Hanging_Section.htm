Hanging Section - 

This option sets Hanging Section options, it sets the position as well as drilling options. The Hanging Section Part is only available when using the Closet 32mm Construction Style.

 Regardless of how you answer these questions, you will not see the Hanging Section in a Closet until you edit a Closet and place one there, and save the modified Closet back to your library.

The following diagram shows the options available in the Hanging Section Branch. Click a specific option in the diagram for more information or continue through this topic for information on all the options.

Hanging Section Boring (Standard #500)

Bore Required holes will use the mounting holes associated with the Model from the Material Manager.

Bore Additional Line Bore will use the mounting holes associated with the Model from the Material Manager.

Use Existing Front Line bore will use System Holes and not the mounting holes associated with the Model from the Material Manager.

None will strip the mounting holes associated with the Model from the Material Manager.

Closet Rod Horizontal Position (Standard #505)

This position establishes the point the Center of the Graphic Reference is placed for the Closet Rod.

Closet Rod Minimum Top Clearance (Standard #506)

This position establishes the point the Center of the Graphic Reference is placed for the Closet Rod.
