Build-Up

The following diagram shows the options available in the Build Up object. Click a specific option in the diagram for more information or continue through this topic for information on all the Build Up options.

Build Up Joint Type (Std #4)

Specify whether you Butt your Build-Up together or Miter it. If you select Miter you will need to enter the Build-Up's Width here.

Overlap Build Up at Joint (Std #5)

Select either to Miter or Overlap build up on a joint edge.

Build Up Joint Priority (Std #44)

This option allows you to prioritize your Build-Up. The first Build-Up in the List is always run long. Each subsequent Edge in the list is given priority for running long after it. Simply left click drag the items in the list to prioritize them to your needs.

Build Up Width on Profile Edge (Std #45)

This option allows you to specify the Width of your Build-Up on a Profile Edge. Select None if you do not use Build-Up on your Profile Edges.

Build Up Recess on Profile Edge (Std #46)

This option allows you to specify whether your Profile Edge Build-Up is Recessed or not and what the Recess (Gap) is.

Build Up Width on Finished Edge (Std #47)

This option allows you to specify the Width of your Build-Up on a Finished Edge. Select None if you do not use Build-Up on your Finished Edges.

Build Up Recess on Finished Edge (Std #48)

This option allows you to specify whether your Finished Edge Build-Up is Recessed or not and what the Recess (Gap) is.

Build Up On Joint Edges (Std #6)

This option allows you to specify the Width of your Build-Up on a Joint Edge. Select None if you do not use Build-Up on your Joint Edges.

Build Up Recess on Joint Edges (Std #49)

This option allows you to specify whether your Joint Edge Build-Up is Recessed or not and what the Recess (Gap) is.

Build Up Width on Unfinished Edge (Std #50)

This option allows you to specify the Width of your Build-Up on an Unfinished Edge. Select None if you do not use Build-Up on your Unfinished Edges.

Build Up Recess on Unfinished Edge (Std #51)

This option allows you to specify whether your Unfinished Edge Build-Up is Recessed or Prepped for Scribe and what the Recess (Gap) is.

Unfinished Edge Top Operation Start Position (Std #52)

Enter the span on the top untouched by the back operation.

Unfinished Edge Top Scribe Thickness (Std #53)

Enter the thickness of the top scribe.

Build Up Width on Splash Edge (Std #54)

This option allows you to specify the Width of your Build-Up on a Splash Edge. Select None if you do not use Build-Up on your Splash Edges.

Top Recess on Splash Edge (Std #37)

There are five options for the Top Recess on Splash Edge:

  • No Change - This would be flush as demonstrated in the image for the selection in the Wizard.

  • Recess Top - The Splash is against the Wall and you specify the distance the Top and Build Up is away from the Wall. See sample image for selection in the Wizard.

  • Pre-Scribe Edge - The Splash is against the Wall and you specify the distance the Top and Build Up is away from the Wall. You will next be prompted for the Span which will be cut fully through the Top Surface. See sample image for selection in the Wizard.

  • Prep Edge For Scribe - The Splash is against the Wall and you specify the distance the Top and Build Up is away from the Wall. You will next be prompted for the Span which will be cut to Depth based on an additional prompt for the Top scribe Thickness. See sample image for selection in the Wizard.

  • Recess Build Up - The Splash and Top is against the Wall and you specify the distance the Build Up is away from the Wall. See sample image for selection in the Wizard.

Top Operation Start Position (Std #38)

Specify the Span for the blind Scribe. This option is only available if the "Pre-Scribe Edge" or "Prep Edge For Scribe" options were selected for "Top Recess on Splash Edge (#037).

Top Scribe Thickness (Std #39)

Specify the Thickness for the Top Scribe. This option is only available if the "Prep Edge For Scribe" option was selected for "Top Recess on Splash Edge (#037).

Radius Build Up Width (Std #7)

Additional width is required to radius build up. Enter the maximum width you will allow for radius build up. Note: This value must be greater than standard build up.

Radius Interior (Std #8)

Radius the interior edge of radius build up or leave a straight edge?
