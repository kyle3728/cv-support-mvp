Hinge Plates

This topic covers all the Properties for the Hinge Plate Material Type.

Material Group

Previously covered in the Banding Material Type topic.

Hinge Plate Group

  • Type - Type defines the style of Hinge Plate as either Panel or Frame.

  • Plate Hole Offset - When defining a "Graphic" for your Hinge Plate the primary mounting holes should be placed at the Graphic Reference Point. This value will shift the Graphic when used in an Assembly by this value. This value typically represents the distance from the front of your Case to the center of the primary mounting holes. Sets the Graphic Reference Point placement.
