Sliding Door Rollers

This topic covers all the Properties for the Sliding Door Rails Material Type.

Material Group

Previously covered in the Banding Material Type topic.

Sliding Door Roller Group

  • Type - The Type of Sliding Door Roller being defined. Following are the available Types:

Roller

 

Slot

  • Door Edge Inset - The Mounting Plates location from the Edge of the Door. Sets the Graphic Reference Point placement.

  • Door End Inset - The Mounting Plates location from the End of the Door. Sets the Graphic Reference Point placement.

  • Mount On Back - Specifies if the Sliding Door Roller mounts to the back of the Door.

  • Slot Reference - The reference for the Slot in the Door.

  • Slot Offset - The offset distance from the reference above in the Door.
