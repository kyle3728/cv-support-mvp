Pull Group

  • Type - The Type of Pull being defined.

  • Horizontal Inset - The Horizontal Position of the Pull (Reference Point of Graphic) on the Door/Drawer Front. Sets the Graphic Reference Point placement.

  • Vertical Inset - The Vertical Position of the Pull (Reference Point of Graphic) on the Door/Drawer Front. Sets the Graphic Reference Point placement.
