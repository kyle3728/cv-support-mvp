Finish Type

Toolbar

The toolbar has the following options:

Finish Type Selector - Allows you to select an existing Finish Type to edit.

New - Allows you to create a New Finish Type.

Copy - Copies the currently selected Finish Type.

Delete - Deletes the currently selected Finish Type.

 

The following options can be used to control the Finish Types properties:

Name - The Name of the current Finish.

Description - The Description of the current Finish.

ID - The Finish Type's database ID.

Shader - There are seven Shaders you can choose from. These simulate the real-world reflectance of the stated Material Types.

  • Glass

  • Gloss Metal

  • Gloss Plastic

  • Metal

  • Mirror

  • Plastic

  • Wood

Ambient - Sets the base color, should typically be set at 50%. Defines an Ambient Lights affect on the Material. This is typically set to a high value for complex surfaces to help shade or define folds and creases on the Materials surface.

Diffuse - Values higher than 50% are shinier while values lower than 50% are more matte. Defines how light reflected from the Material strikes other objects in the surrounding area, illuminating them. The Diffuse Factor describes light reflected from surfaces which are not shiny or specular. In real life terms what this means is that light is reflected off non-shiny surfaces such as the ground, walls, or fabric, to reach areas not directly in view of a light source.

Specular - The Specular Factor is distinct from the Diffuse Factor, where incoming light is reflected in a broad range of directions. The most familiar example of the distinction between Specular and Diffuse reflection would be glossy and matte paints. While both exhibit a combination of Specular and Diffuse reflection, matte paints have a higher proportion of Diffuse reflection and glossy paints have a greater proportion of Specular reflection. Very highly polished surfaces, such as high quality mirrors, can exhibit almost complete Specular reflection.

Shininess - This setting sets the smoothness of the surface. Shininess references the Materials ability to reflect. The more towards the right you set the slider the more the Material will reflect.

 Shininess in and of itself will not control Reflection specifically. The roughness of the Finish plays a role in "how" something reflects. A mirror is perfectly smooth and with it reflection is reflection. First you establish a surface is reflective by selecting the Mirror Type then you must tweak the other Finish Properties in order to fully define how the finish reacts to lighting.

Transparency - Transparency references the ability of the Material to allow the Texture below it to show through. Wood grain shows through a transparent color to give a wood tone finish when set correctly.  If the slider bar is moved to the full transparent side (the right side), the Finish will appear as a tint. This may be useful for glass type Finishes, but is too transparent for stain type Finishes.
