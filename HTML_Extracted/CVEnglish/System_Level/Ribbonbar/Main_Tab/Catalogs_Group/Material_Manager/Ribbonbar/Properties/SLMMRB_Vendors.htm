Vendors - 

Opens the Material Vendor Manager for the selected Material. This Utility allows you to assign Vendors from whom you purchase the active Material.

Select your Vendor from the pane on the right and then click the left button to add it to the list of Vendors in the left pane the selected Material can be purchased from. You can also drag a Vendor from the right pane to the left.

The following fields can be edited once a Vendor is added to the Material:

SKU - The identifier used by the Vendor for the selected Material.

Cost - What you pay the Vendor for the selected Material.

Markup - The Markup you apply when using the Material from this Vendor.

Current Default Vendor

If you are assigning multiple Vendors to a Material make sure to select which Vendor is your preferred Vendor.
