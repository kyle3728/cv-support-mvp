Composite

This option allows you to set the structure of your Composite Material. A Composite Material is a sheet Material consisting of several layers of sheet goods.

The Composite Material Manager is only available for the Composite Material type.

Select your sheet good from the pane on the right and then click the left button to add it to the list of Materials that make up the Composite in the left pane. There is a Filter option at the top to assist in finding a specific Material.

You can control the order the sheets are layered by using the Up and Down buttons and remove a sheet from the Composite by clicking the Right button.

The following fields can be edited once a sheet good is added to the Composite:

Face Up - Specifies if the sheet good is Face Up (Enabled) or Face Down (Disabled).

Oversize - An Oversize amount for the sheet good when used in this Composite.

See the Composite Material Type topic for more information.
