Parameters

Opens the Parameter Editor allowing you to add a Parameter to the currently selected Material. See the Parameters topic for more information on the Parameter Editor interface.

To query these Parameters via UCS you will need to use _M:ParameterName on the Object using the Material. The Parameter will not be visible in the Object Tree.

When editing Parameters for a Connector Material you have an additional tool in the toolbar.

When a Connector primary operation creates a secondary operation, the primary operation notifies the Connector a secondary operation has been created. The Connector in response then runs a UCS that has been created for it here. This will allow you to create a UCS to adjust the Z Position of operations after all the internal processing has taken place so the Connector Operations can be adjusted to different thickness Secondary parts.

There is an example within the UCS editor when you click the New tool. See the User Created Standards topic for more information on the UCS interface.

Notice you have the ability to create multiple UCS's here. To actually have a specific Connector UCS run you would use a normal UCS within CABINET VISION to call it. You would assign the normal UCS to the Primary Operation within the Connector and set _MACRO = n. n being the Order number for the UCS in the Connector UCS Editor.
