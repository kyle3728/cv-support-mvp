Texture

A Texture is a pattern that will appear on your finished objects such as wood grain or any other visible feature. Textures will generally show through the Finish color in a Room adding the element of realism to your objects. Typically, Textures are black and white, and receive color from a Finish at the Room Level, however, this is not always the case. A Texture can be a sample of Marble, complete with the color and pattern that you choose to scan into Solid for use in your presentations. You will have to experiment with scanning in order to find the size and type of acceptable (scannable) images that do not distort when you shrink them to the size necessary to appear on a scaled Solid rendering. A hint that may help is to search for books from hobbyists who create miniature items. Examples might be doll house publications, model railroad publications, etc. Typically, the textures you will find there are created at the scale they will be rendered at, and do not need to be reduced, and do not suffer the distortion that often accompanies the reduction process.

Toolbar

The toolbar has the following options:

Import Image -  Allows you to Import an image as a new Texture. See The Bitmap Method below for more information.

 You can use Windows + Click and/or + Click to select and import multiple image files.

 Make sure that the image file you are importing is located somewhere other than where you are importing it to.

Delete - Allows you to Delete the selected Texture.

List View - Switches the Working Area to List View, as shown in the illustration above.

Working Area (List View)

Following are the Texture Properties available in the Working Area (List View):

ID - The internal database ID for the Texture (This value can not be edited).

Name - The Name of the Texture (This value can be edited).

Description - The Description of the Texture (This value can be edited).

Path - The Path to the file the Texture is using (This value can not be edited).

World Width - The Width the image should represent in a real-world dimension.

World Height - The Height the image should represent in a real-world dimension.

Decal - If Enabled this will instruct CABINET VISION to not blend the Finish with the Texture.

Tile - If Enabled instructs CABINET VISION to repeat Texture at the specified World Width/World Height across the surface it is being applied to.

 

The Bitmap Method

Here, a new Texture created using the Paint program that ships with Microsoft Windows™ will be imported for use as speaker cloth on Doors for a Job. The Texture was created in the favored 200 by 200 pixel size, and saved as a Bitmap file to the to my Windows Desktop folder as spk.bmp.

 Make sure that the image file you are importing is located somewhere other than where you are importing it to. In the above example the image file is on the Windows Desktop.

To add a Texture choose the Category where you want it located then click the New option in the ribbonbar and select the Bitmap or JPG file from the Graphics sub-folder where you saved the file you made. Files that you copy to your computer can be saved to the same folder as those you make and imported in the same fashion.

Since this is a file that exists on my computer I will choose the "Load an image from a file" option in the toolbar.

Browse to, and select you image file then click Open.

Go ahead and fill out all the Properties, when you are happy with the Properties click the Save option in the toolbar.

The "Decal" check box, must be checked if you want the Texture to stand alone, I.E., not to be tinted by the Room Finish.

The "Tile" check box should be enabled if you want the pattern to repeat on your object. Without checking this box, the Texture will appear one time, and may not cover all the surfaces you have applied it to. Make sure to set the World Width and World Height at 12 each. After verifying the appearance of your new Texture you can adjust these World dimensions to tweak the appearance of the Texture when displayed.

To add this Texture to a Door as speaker cloth, create the speaker cloth as a "Miscellaneous" Material in the Material Manager, and choose the new Texture for the face Texture. From the Door Material Schedules, create a Schedule referencing the Material as one with speaker cloth (Birch/Spker) and assign the speaker cloth to the "Flat Panel" category. You will see the following if viewed in High Detail in a Job. Note how the speaker cloth is not tinted by the Cabinet color and how the pattern repeats to fill each Door.

The ultimate secret to successful Texture creation and usage is to experiment with the resolution and scale of the items you chose to use. You will quickly understand the relationship and how to improve the image after viewing your Textures in a practice Job. Textures are not limited to patterns, even company logos, initials, etc..., can be used.

Textures can also be added from within a Job in the 3D View when selecting Textures to use in your Room or Assembly.
