Composite

This topic covers all the Properties for the Composite Material Type.

Material Group

Previously covered in the Banding Material Type topic.

Composite Group

  • Pre-Assembled - Establishes if the Composite is assembled before or after Cutting/Machining. If it is not pre-assembled then each Material that makes up the Composite Part is cut and machined separately.

Size Group

  • Length - Displays the overall Length of the Composite. Not available for editing. This is a system calculated value based on the Materials making up the Composite.

  • Width - Displays the overall Width of the Composite. Not available for editing. This is a system calculated value based on the Materials making up the Composite.

  • Thickness - Displays the overall Thickness of the Composite. Not available for editing. This is a system calculated value based on the Materials making up the Composite.
