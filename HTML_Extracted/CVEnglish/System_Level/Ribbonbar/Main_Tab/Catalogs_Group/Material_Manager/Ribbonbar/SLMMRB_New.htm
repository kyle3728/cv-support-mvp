New

The New option opens the New Material wizard which is used to create a New Material in the Material Manager.

There are three options on the first screen of the New Material window:

  • Type - The Type of Material you are creating. There are several options each having a special set of Properties. Some of these Properties will be available on subsequent screens depending upon the Type selected here:

    • Banding - Defines Edgebanding Material.

    • Board Stock - Defines Board Stock Material.

    • Caster - Defines Casters.

    • Composite - Defines Composite Material.

    • Connector - Defines a Connector.

    • Drawer Guide - Defines Drawer Guides.

    • Finger Pull - Defines Finger Pulls.

    • Hinge Plate - Defines Hinge Plates.

    • Hinge  Defines Hinges.

    • Laminate - Defines Laminate.

    • Leg - Defines Legs.

    • Leg Leveler - Defines Leg Levelers.

    • Miscellaneous - Defines Miscellaneous Materials.

    • Molding - Defines Moldings.

    • Molding Set - Defines a Molding Set.

    • Panel Stock - Defines Panel Stock.

    • Pull - Defines Pulls.

    • Rod - Defines Closet Rods.

    • Shadowline Channel - Defines a Shadowline Channel.

    • Sliding Door Rail - Defines Sliding Door Rails.

    • Sliding Door Roller - Defines Door Rollers.

    • Wire Basket - Defines Wire Baskets.

  • Name - The Name you want to use for the Material. The number of characters for a Material Name is limited to 50.

  • Description - The Description you want to use for the Material. The number of characters for a Material Description is limited to 100.

Clicking Next will bring up different screens depending on the Material Type selected.

Drawer Guide

Drawer Guide

Unit of Issue - Previously covered.

Type - The Type of Drawer Guide. There are three choices here:

Length - The Length of the Drawer Guide.

Height - The Height of the Drawer Guide. Only available for the "System" Type.

Side Clearance - This is the value establishes the clearance at the sides of the drawer box for the Drawer Guide. This value, multiplied by 2 and then subtracted from the opening width, will set the Width of a Drawer Box that has this Drawer Guide. This is the area where Drawer Boxes receive the information on their Width. If you want more than a typical 1/2 inch, for a little play for instance, consider entering a value of 17/32.

Top Clearance - This value, and the values entered in the Drawer Box/Roll Out Wizard begin to set the Height of your Drawer Box. You are asked to enter a minimum value here, and that minimum value will always apply to the selected Drawer Guide. This represents the distance between the top of the Drawer Box and the first thing it would otherwise hit, or, the gap between it and another Drawer if there is no Cabinet member in between.

Bottom Clearance - Similar to the minimum top clearance, this defines the distance between the bottom of the Drawer Box side, (or the Drawer bottom, the lower of the two), and the first thing it would hit. If there is no Cabinet member in the way, it would reference the gap.

 

Mount Holes

Diameter - The Diameter of the Mount Holes.

Depth - The Depth of the Mount Holes.

Top Reference - When set to Yes establishes that the Mount Holes are referenced from the top of the Guide.

Screw Center to Ref. - Sets the dimension from the Reference (established with Top Reference) to the center of the Mount Holes.

Position 1 - The Position of the first hole from the Front of the Drawer Box. When defining a "Model" for your Drawer Guide the first hole should be placed at the Graphic Reference Point. This value will shift the Graphic when used in an Assembly by this value. This value typically represents the distance from the front of your Case to the First Bore Hole. Sets the Model Reference Point placement.

Position 2 - The Position of the second hole from the Front of the Drawer Box.

Position 3 - The Position of the third hole from the Front of the Drawer Box.

Position 4 - The Position of the fourth hole from the Front of the Drawer Box.

Position 5 - The Position of the fifth hole from the Front of the Drawer Box.

Position 6 - The Position of the sixth hole from the Front of the Drawer Box.
