Connector

This topic covers all the Properties for the Connector Material Type.

Material Group

Previously covered in the Banding Material Type topic.

Connector Group

  • Type - Specifies the Type of Connector. Following are the possible types:

    • Cabineo 12

    • Cabineo 8

    • Clamex P Medius 14-10 - Requires 4-1/2 Axis machine with Lamello's Clamex Saw. Saw Tool is available in a TechPackage on eSupport.

    • Clamex P-10 - Requires 4-1/2 Axis machine with Lamello's Clamex Saw. Saw Tool is available in a TechPackage on eSupport.

    • Clamex P-14 - Requires 4-1/2 Axis machine with Lamello's Clamex Saw. Saw Tool is available in a TechPackage on eSupport.

    • Connector - This is a generic Connector (no Model) that you can use to define your own Custom Connector from the Model Editor.

    • Divario P-18

    • FastenLink - Requires a Shaped Tool in the S2M CENTER Tool Catalog. FastenLink Tool setup is available in a TechPackage on eSupport.

    • LockDowel

    • Peanut 1

    • Peanut 2

    • Rafix

    • Tenso P-10 - Requires 4-1/2 Axis machine with Lamello's Clamex Saw. Saw Tool is available in a TechPackage on eSupport.

    • Tenso P-14 - Requires 4-1/2 Axis machine with Lamello's Clamex Saw. Saw Tool is available in a TechPackage on eSupport.
