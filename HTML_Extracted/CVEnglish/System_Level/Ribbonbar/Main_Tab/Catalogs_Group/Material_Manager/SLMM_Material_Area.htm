Material Area

This area of the Material Manager displays all of the Materials in the selected Category or all the Materials meeting the Search criteria in an active Search.

Standard Properties for each Material are located on the Material Tab and any special Properties will be stored on the extra tabs. The Material ID and Material Name will always be displayed so you can easily see which Material you are working on. Using the tabs is a good way to quickly change Properties on multiple Materials. For focused configuration of a single Material it is recommended to open the Properties window for that Material.

Right Click Options

Following are the options available when right clicking a Material in the Material Manager:

 

 Copy

 Delete

 Alias

 Redefine

This option allows you to re-run the New Material Wizard on the selected Material.

 Running this option will cause you to loose any changes you have made in the Model Editor including modified Parameters/Formulas.
