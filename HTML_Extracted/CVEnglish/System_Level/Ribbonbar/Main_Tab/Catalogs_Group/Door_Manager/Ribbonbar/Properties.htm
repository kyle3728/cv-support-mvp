Door Properties

When you have a Door selected in the Door Manager you can get to it's Properties by clicking this option in the ribbonbar (alternatively double click the Door or right click and select Properties from the menu).

Following are the Properties available in Door Properties:

ID - The Door Style ID number which can be useful in User Created Standards. The Door Style ID number is auto-generated and can not be edited.

Name - The name of the current Door Style.

Description - The description of the current Door Style.

Tags - Keywords added here will aid in Searches.

Unit Cost - This is where you set a flat price for this Door Style. These prices are retrieved in the Bid Center by using the Use Door Catalog Pricing Bid Method.

Cost /SqUnit - This is where you set a Per Square Unit (Either Meter or Foot depending on your active Unit of Measure) Cost for this Door Style. These prices are retrieved in the Bid Center by using the Use Door Catalog Pricing Bid Method.

Oversize - An amount to add to the overall size of the Door. Typically used to oversize for clean-up on a joiner.

Template - Following are the Template Properties which are available if the selected Door Style is a Template Door, this list will vary depending upon which Template Style is chosen..

  1. Style - See the Ribbonbar topic for a list and description of each of the available Door Template Styles.

  2. Slab - Determines if the Door Type is Slab or Five Piece.

    This option does not determine whether the Door is a Glass Door; that is actually controlled by the Tool Set used on the Route Pattern.

  3. Soften Corners - Determines if the Corners of the Doors need to be softened (radiused). This is typically used when applying RTF or Vinyl to your Doors to keep it from tearing on sharp corners. The Radius value is set in the Preferences are on the Geometry Tab in the S2M CENTER. See the S2M CENTER Help file for more information.

  4. Corner Radius - Allows you to establish a specific Corner Radius for this Door Style when Soften Corners is enabled.

  5. Shape Top - Selecting this option will Shape the Top Outside Edge of the Door to match the internal Route Pattern as demonstrated in the image below.

  6. Shape Bottom - Selecting this option will Shape the Bottom Outside Edge of the Door (Mirror Top Shape has to be active as well) to match the internal Route Pattern as demonstrated in the image below.

  7. Lites - Sets the number and configuration of Lites (Panels) the Template Door has.

    Depending upon the Template you have chosen you will see some of the following Shape options:

    Shape - Crown

    Allows you to specify the Crown Height on the Door as demonstrated in the graphic below.

    Shape - Flat Indent

    Allows you to specify the Flat length before the start of the Crown as demonstrated in the graphic below.

    Shape - Radius

    Allows you to specify the Radius at the transition from the End Stiles to the Crown as demonstrated in the graphic below.

    Shape - Chamfer

    Allows you to specify the Chamfer length on a Chamfer Template Door.

    Shape - Mirror Top Shape

    Allows you to specify that the Bottom of the Route Pattern Mirrors the Top instead of being flat. As demonstrated in the graphic below.

    Shape - All Corners

    No specifies to Chamfer/Radius the top two corners only; Yes specifies to chamfer/radius all four corners of the Route Path.

    Shape - Rise

    Allows you to specify the Rise of the slope of a Slope Template Door.

    Shape - Run

    Allows you to specify the Run of the slope of a Slope Template Door.

    Shape - Width

    Allows you to specify the bumps Width on a Bump Template Door.

    Shape - Height

    Allows you to specify the bumps Height on a Bump Template Door.

Spacing - Following are the Spacing Properties which are available if the selectred Door Style is a Slab or Template Door.'

  1. Left Stile

  2. Right Stile

  3. Top Rail

  4. Bottom Rail

  5. Mid Stile

  6. Mid Rail

Edgebanding - Use these properties to confirm whether the selected Door Style's edges have Edge Banding (Door Edge Banding [D]) or not (None [N]).

  1. Left

  2. Right

  3. Top

  4. Bottom

 Edgebanding will only be applied if the active Material Schedule has a valid Banding Material and the Banding option is enabled.

Edge Profiles - Use these properties to confirm whether the selected Door Style'e edges use the Outside Edge Profile, the Finger Pull Profile/Part or None.

  1. Left

  2. Right

  3. Top

  4. Bottom

Defaults - You can set the following Defaults for a Door Style:

  1. Material Schedule

  2. Outside Edge Profile

  3. Inside Edge Profile

  4. Raised Panel Profile

  5. Finger Pull Profile

  6. Route Path Profile

Ribbonbar

Following are the ribbonbar options in the Door Properties area:

Return

This option will return you to the Door Manager.

New

The New option allows you to create a New Door in the Door Properties area. See the Door Manager New topic for more information.

The new Door will be placed in the same Category as the currently selected Door.

Copy

Allows you to Copy the currently selected Door.

Delete

Allows you to Delete the currently selected Door. After clicking the Delete option you will be prompted to Confirm the Delete operation.

Select

Allows you to Select a different Door to work with in Door Properties.

The sidebar in the Select Material window allows you to Search for a Door or browse for Doors via the Door Categories. Once you have located your Door simply select it from the list of Doors in the Work Area and click OK.

 Associations

Doors that should change Construction style based on pre-determined size requirements or conditions require Associations.

In the above illustration we have established minimum and maximum sizes for the Square Raised Panel Door.

For example I have a square raised panel Door and I would normally not make this Door any less in width or length than 8 inches. Because the Top Rail is 2 inches, the Bottom Rail is 2 inches, my raised panel detail is 1 ¼ inches all around, and I would want a flat area on the panels of no less than 1 ½ inches wide. This all adds up to 8 inches if I make the Door any less than 8 inches in height it gets dangerous to run the panel on my shaper.

When the Square Raised Panel Door is used and the Width or Length of the Door is 8" or less then the Solid Wood Door Style is used.

When the Width is greater than 8" but 24" or less then the Square Raised Panel is used.

When the Width is greater than 24" but 42" or less then the Vertical Split Raised Panel is used.

When the Length is greater than 8" but 32" or less then the Square Raised Panel is used.

When the Length is greater than 32" but 64" or less then the Square Raised Panel is used.

And anything greater than 42" wide and 64" long will be a Slab (plywood).

So using Associations I have set a size limit of 8 inches for the length and the width. When I am in a Job and the Door becomes 8 inches or less in length or width, CABINET VISION will automatically replace the Square Raised Panel Door with the Solid Wood Door. I have also set a size limit of 32 inches for the length. When I am in a job and the Door becomes 32 inches or taller CABINET VISION will automatically replace the Door with a Vertical Split Raised Panel Door.

In situations where the left and right Doors should mirror each other, use the Mirrored Right Style option to assign an alternate Right Door Style to use. Simply create both the Left and Right versions of the Door. Then select the left Door and assign the right Door here. When you choose the left Door as your default Door in a Job, the Mirrored Right Style Door will thereafter be used when right hinging is called for.

 Material Filter

Material Filters are not required but are a good option if you want to limit what Materials are available for the current Door Style. Simply add any of the Available Materials on the left to the list on the right. You will note in the illustration above that I have selected my "Alder", "Ash", "Birch", "Cherry", "Maple", "Oak", and "Paint Grade" Material Schedules for use with this Door.

The Material Cost associated with the selected Material Schedule is also displayed and can be edited from here. See the Door/Drawer Fronts Material Schedule topic for more information.

 

 Profile Filter

Profile Filters are not required but are a good option if you want to limit what Profiles are available for the current Door Style. Simply add any of the Available Profiles on the left to the list on the right. When you start a new Job, you will select the Door at the Job Properties tab and it should give you the opportunity at that time to select the profiles that will be used in that particular Job from the Profiles that you have made available for that Door.

Simply select the Profile Type to see the list of that type.

There are four different types of profiles.

 Inside Edge - The Inside Edge is used when you have a frame around the Door and the frame is assembled with cope & stick construction. Inside Edges are important because they can, depending on the Extended Property, affect the size of your Door parts.

 Outside Edge - The Outside Edge is a shape you will cut around the outside of the Door. Outside Edges do not affect the size of your Door parts.

 Raised Panel - The Raised Panel is the shape you will cut along the outside of the Door panel. Usually done around the edges of a Raised Panel. Panel details do not affect your Door parts sizes.

 Route - The Route is the shape of the cutter you would use if you where going to route a design into the face of the Door using a Door jig.

The Profile Cost associated with the selected Profile is also displayed and can be edited from here.
