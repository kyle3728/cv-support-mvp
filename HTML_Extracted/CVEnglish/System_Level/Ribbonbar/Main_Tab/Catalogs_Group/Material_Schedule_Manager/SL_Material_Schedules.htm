Material Schedule Manager

Material schedules are created so that you can tie Parts from the Part Manager to Materials that you have created in the Material Manager, inside a Job.

A Material Schedule is simply a template of the list of Materials assigned to specific Parts. For example, you might want to create a Material Schedule for an Oak Cabinet. To do so, you would go to the Assembly Parts Material Schedules and create a new one. You would probably want to call the Material Schedule something like "Oak Cabinets." Next, you would go through the Schedule and assign the proper Material to each Assembly Part. Then, any time that you wanted to create a Job that included Cabinets made of oak, all that you would have to do is select the Oak Material Schedule in the Materials Property window, and then draw the Job. In other words, once a Material Schedule has been created, it can be used in any subsequent Jobs that require the same Materials. This should help to save time that would otherwise be used in assigning Materials for Assembly Parts in every Job.

Part List

This area list all the Parts available for Material Assignment in the selected Material Schedule/Material Schedule Type. You will notice a few different icons next to the line items. These represent the following:

- All Parts in this category have the same Material.

- At least one Part in this category has a unique Material.

- This Part is using the categories Parent Material.

- This part has a Material which differs from the categories Parent Material.

There are Right Click options for Parts in the Parts List. See the Right Click Options topic for more information.

Search Pane

Allows Quick Search and Saved Search functionality to quickly find a specific Material or Material meeting specific criteria. See the Material Manager Sidebar topic for more information the Search Pane.

Material List

This area lists all the Materials available to be assigned to your Parts. You can use drag & drop or click the assign button to assign a Material to a selected Part.
