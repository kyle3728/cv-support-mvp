Molding

Materials for the different types of Molding are set in the Molding Material Schedules.

Sidebar/Properties

The following Properties are available for Molding:

ID - The ID number for the selected Material Schedule.

Name - The Name of the selected Material Schedule.

Description - The Description of the selected Material Schedule.

Upcharge - The "Upcharge" value will only be added in a Rate Table that uses the Bid Method "Use Part Catalog Pricing". This value is a Percentage Upcharge you charge above your Standard Part Catalog Price as entered in the Part Manager when this Material Schedule is in use.

When you add Board Stock to your Material Manager, consider the finished dimension of the Molding and the dimension of the Material that you will run in order to get that finished dimension. You may want to have Oak, for example, in 4/4, 6/4, or greater in order to net the sizes that you need.
