Door

Every Door or Drawer Front must have at least one Material Schedule assigned to it. Door and Drawer Front construction is set up in the Door Manager.

The functions of the Material Schedule Editor are the same as for Assembly Parts.

 Assigning a Applied Molding Material to the Applied Inside or Applied Outside Edge Molding Part's will add the selected Applied Molding to the Door the Material Schedule is assigned to. The positional Offset for this Molding is controlled in the Associated Door Profile.

Sidebar/Properties

The following Properties are available for Doors:

ID - The ID number for the selected Material Schedule.

Name - The Name of the selected Material Schedule.

Description - The Description of the selected Material Schedule.

Upcharge - The "Upcharge" value will only be added in a Rate Table that uses the Bid Method "Use Part Catalog Pricing". This value is a Percentage Upcharge you charge above your Standard Part Catalog Price as entered in the Part Manager when this Material Schedule is in use.

Buy Out -  Enable this option you Buy-Out the Door rather than building it yourself. This will tie in to your Reports and Bids.

Cost/Door - This option will simply add the entered amount as a flat rate per Door. This value will only be added in a Rate Table that uses the Bid Method "Use Door Catalog Pricing."

Cost/SqFt (SqM) - This option will multiply the entered amount by the total area of the Door. This value will only be added in a Rate Table that uses the Bid Method "Use Door Catalog Pricing."

Minimum Cost - This option will allow you to set a minimum Door cost for the Door when using this Material Schedule. Note that total door cost is Door + Mat + Profile (Inside Edge + Outside Edge + Raised Panel + Custom Route). This value is only for the Material cost so if the Material "Add Per Door" and the Material "Add Per SqFt" is less than "Min Cost" it will use Min Cost for the Material cost. In other words... the Door Cost and Profile Costs will be added to this Minimum Cost.

Banding - Instructs CABINET VISION to apply the Material assigned to the Door Edge Banding Part to your Door.

Because you control the Banding through the Door and Drawer Front Material Schedules, you may wish to create duplicate Schedules with one having Banding active and another without. Remember to name the Schedules in a way that lets you know that Banding is turned on. For example, you may want to have a Schedule called Birch and one called Birch Band. This will allow you to know by name that one Material Schedule is for a Birch Door that has banding turned on.

Unlike Material Schedules for Assembly Parts, the Door Material Schedules allow you to turn Banding on or off in the Material Schedule itself. If the Material Schedule that you are creating will be used on Doors that you want to Banding simple enable the option.
