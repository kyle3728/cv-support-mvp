Molding Manager

The Molding Manager is where you create and edit the Profiles for Molding you manufacture. See the Material Manager for information on defining Molding you buy-out.

Sidebar

There are several options in the Molding Manager sidebar:

  • Name - Allows you to Name or rename the selected Molding. The character limit for the Molding Name is 50.

  • Type - Allows you to set the Type for the selected Molding. Following are the available Molding Types:

    • Applied

    • Base Board

    • Casing

    • Ceiling

    • Chair Rail

    • Crown

    • Light Rail

    • Scribe

  • Set - Indicates whether the current is a Molding Set or not.

  • Thickness - This value is controlled by the Shape of your Molding.

  • Height - This value is controlled by the Shape of your Molding.

  • Cost Per Job -  - Allows you to set a Cost Per Job for the selected Molding. This value will be used by the Use Profile Catalog Pricing Bid Method in the Bid Center.

  • Cost Per Strip -  - Allows you to set a Cost Per Strip for the selected Molding. This value will be used by the Use Profile Catalog Pricing Bid Method in the Bid Center. A "Strip" is determined by how the molding is designed in your Layout. Each separate connected run including returns is considered one "Strip".

  • Cost Per FT - - Allows you to set a Cost Per Foot for the selected Molding. This value will be used by the Use Profile Catalog Pricing Bid Method in the Bid Center.

  • Sample - Length - A sample Length you can enter to get a preview of your Molding in 3D Mode.

Ribbonbar

Return

This option will Return you from the Shape Edit area. If changes were made to the Shape you will be prompted if you want to Save those changes.

Zoom Group

New

There are two New options:

Molding

This option creates a new Molding.

Molding Set

Each of the Molding Types above can also be defined as a Molding Set by selecting the Molding Set option when creating a new Molding. A Molding Set is a grouping of multiple Moldings you have created, put together so they can be placed in your designs easily as one. The Moldings are broken out independently in Reporting.

Copy

Copies the selected Molding.

Delete

Deletes the selected Molding from the Molding Manager.

This Delete action can not be reversed.

Molding Type Selector

Allows you to select the Molding Type you wish to work with.

Molding Selector

Allows you to select the Molding you wish to work with.

Render Mode

2D Mode

Shows the Molding in a 2D View looking at the end of the Molding.

3D Mode

Shows the Molding in a 3D View.

Import

Allows you to import a 3D Model to represent a Molding with a repeat, such as Dentil or Egg & Dart.

  1. Click the Import option in the ribbonbar.
  2. Browse to, select, and Open your 3D Model.

You can import either Trimble Sketchup or DXF Models.

  1. If you are satisfied with your 3D Model click OK. For more information on the options in this screen see the Import Sketchup Model topic.

You should now see your Model. The model will repeat itself to fill the length of the molding drawn in your Job. You can switch to 3D Mode and change the Sample Length to see how it will react to Length changes.

Reset

Will remove all Shaping that has been applied to the current Molding.

This Reset action can not be reversed.

Techniques for Creating Molding Shapes

The easiest method of adding a Molding Shape is to import it into the Molding Manager. Editing is similar, and uses the same editing tools common to Solid to create the final shape.

Importing A DXF file to Define A Molding

In the following example, a DXF file such as the one shown, will be imported into Molding Shape for use as a Light Rail.

    1. Take your new Molding to the Shape Editor .

    2. Click Select All then click Delete .

    3. Click Insert DXF .

    4. Click the browse button to browse to the location of your DXF file.

    1. Once you have Opened your DXF file from the Browse dialog click Import.

    1. Click OK to Import the DXF into the Shape Editor.

    2. Pick its location in the Shape Editor. It doesn't really matter where you place it.

    3. Once placed select the Reference option and pick the point which you want to be the Reference point on the Molding. See the Shape Editor topic for more information on the Reference option.

  1. Click Return to Save the shape of your new Molding.

  Keep in mind you have full CAD editing capability here so you can Flip, Mirror, Rotate, Scale, Stretch, and modify the imported shape in any way prior to saving it.

 

Importing an Image file to Define A Molding

When Importing an Image file you bring in an Image that will allow you to get the general shape of the Molding by tracing it utilizing the Shape Editor Tools. There are two additional Tools available when a Bitmap is imported that allow you to Scale and/or Align the Bitmap to facilitate proper tracing. Once you have the Bitmap sized as you need simply use the CAD tools to trace the image to get the final shape you desire.

 

Molding Sets

The Molding Shape Editor, when dealing with a Molding Set, allows you to add Moldings to the Set as well as control the added Molding's positions within the Molding Set.

Make sure you have "Snap To Points" turned on as it will make placement of the Moldings within the Molding Set more accurate.

Once you click the Insert option you will be able to select your Category and Molding from the sidebar and drag it into the Shape Editor. Unlike when shaping a Molding you can not control the position of the Reference point but instead must build your Molding Set around the Reference Point. That is the placement point for the Molding when laying it out in your Job. You can drag as many existing Moldings into your Molding Set as you like.

When selecting a specific Molding within the Molding Set you will have the following Property options in the sidebar:

    • ID - Displays the ID of the selected Molding.

    • Name - Displays the Name of the selected Molding.

    • Origin X - Controls the X Position of the selected Molding.

    • Origin Y - Controls the Y Position of the selected Molding.

    • Width - Displays the Width of the selected Molding.

    • Height - Displays the Height of the selected Molding.

    • Angle - Controls the rotation angle for the selected Molding.

    • Mirror - Will Mirror the selected Molding.
