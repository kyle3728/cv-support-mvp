Part Manager

The Part Manager is relatively straight forward in design and usage. It is primarily responsible for holding information on Parts that you need within CABINET VISION. It is relatively complete as it stands, containing the majority of Parts that most shops will use. It is configurable to a degree as it allows you to set Labor values, CNC restrictions, or even Finishes to Parts. There are several Tabs available depending on which Part Group you have active.

Part Tab

Name

The Name is an abbreviated code assigned by CABINET VISION to the Part. Although you can change the Description, you can never change the Name once it is entered. The character limit for the Name is 15.

Description

The Description of the Part can be changed by changing the text here. The Description of the Part here is the "Name" displayed throughout CABINET VISION including all output such as Cut Lists. The character limit for the Part Description is 100.

This will change the "name" of the Part everywhere it is used.

Labor Setup

 

The Labor Setup column allows for a "minute" value to be entered concerning the time you will allow to manufacture the selected Part. The "Labor Setup" value will only occur within the Job once for the part (Setup Value).

Labor Each

The Labor Each column allows for a "minute" value to be entered concerning the time you will allow to manufacture the selected Part. The "Labor Each" values occur for each Part within the Job.

Price

The Price column allows you to establish Pricing for your selected Part. You can type a fixed Price or select one of the pre-defined calculation methods. You can also utilize math to create your own custom price utilizing (addition (+), subtraction (-), multiplication (*), and/or division (/). The formula will be validated by moving to another field. If the formula entered is invalid you will receive a warning message and the invalid formula text will turn red. Following are the available pre-defined calculations:

  • <Cubic IN> - The Cubic Inch value of the Part.

  • <Cubic M> - The Cubic Meters of the Part.

  • <Face SqIN> - The Square Inches of the Face of the Part.

  • <Face SqM> - The Square Meters of the Face of the Part.

  • <Length IN> - The Length, in Inches, of the Part.

  • <Length MM> - The Length, in Millimeters, of the Part.

  • <Matrix Price> - This option will use the active Price Matrix that is selected in the Matrix column within the Price calculation formula you enter. See the Price Matrix topic for more information.

  • <Thick IN> - The Thickness, in Inches, of the Part.

  • <Thick MM> - The Thickness, in Millimeters, of the Part.

  • <Width IN> - The Width, in Inches, of the Part.

  • <Width MM> - The Width, in Millimeters, of the Part.

Matrix

The Matrix column allows you to select the Matrix Price to utilize for this Part. See the Price Matrix topic form more information.

CNC Tab

Optimize

The Optimize column is a toggle. Assuming you have the Optimizer or Nesting, you can toggle Parts on or off as to Optimizing/Nesting.

Ignore Grain

The Ignore Grain column is a toggle. Assuming you have the Optimizer or Nesting, you can toggle Parts on or off as to whether honoring Grain rules for the Part is required.

CNC

The CNC column is a toggle. If you have CNC capabilities, you can choose which Parts are outputted to CNC by toggling them off or on.

Finishes Tab

Exterior Face Finish, Exterior Back Finish, Exterior Edge Finish, and Exterior End Finish

These columns determine whether the Part uses the Exterior (enabled) or Interior (disabled) Finish for the Parts Face, Back, Edge, and/or End. The choices are Enabled (match Exterior Finish in the Job/Room/Assembly) or Disabled (match Interior Finish in the Job/Room/Assembly).

Textures Tab

Exterior Face Texture, Exterior Back Texture, Exterior Edge Texture, and Exterior End Texture

These columns determine whether the Part uses the Exterior (enabled) or Interior (disabled) Texture for the Parts Face, Back, Edge and/or End. The choices are Enabled (match Exterior Texture in the Job/Room/Assembly) or Disabled (match Interior Texture in the Job/Room/Assembly).

 

Right Click Options

Right clicking the Column Headers will bring up the Columns window allowing you to hide and rearrange (drag) the Columns displayed.

The Ribbonbar

Return

Closes the Part Manager and returns you to the opening screen.

New

New Parts can be added by clicking this option. As an example, you use a Part that does not exist on the list, and you want it to be part of your Assemblies and to appear on your Cut List. The first step would be to enter it here by clicking the "Create a new Part" button.

Delete

This option will only be available for Parts you have added and will allow you to Delete them.

Price Matrix -  or 

This option opens the Price Matrix utility to create Price Matrices to be used for your Parts. See the Price Matrix topic for more information.

Adding Banding

You have the ability to add Banding types in addition to the system (Interior/Exterior) banding options. One of the first things you will probably want to do is setup Banding for Drawer Boxes and Roll-Outs.

To do so, click the Drawer Box > Banding Category from the sidebar..

To add new types of banding, click the New option in the ribbonbar and you will see the window pictured below. Select the Banding identifier (Reference Name) you want to use for the Banding and type in its Display Name.

The last letter in the "Name" represents the letter used to identify the Banding throughout the system.

Then you will see the new banding types in the Part Manager and the Material Schedule Manager as well.

Make sure to repeat this process in the Roll Out > Banding category if you need Edge Banding available for your Roll Outs.

These banding codes may be added to a Part either by manually editing the edge banding for a Part or by editing the banding code for the part through a User Created Standard.

You can also change the letter used by default for banding to another letter, a number, or symbol by going to the Banding tab and changing the value in the Symbol field.
