Workstation Setup

Once a license server has been set up, follow this procedure on the client PCs:

  1. Install the Workstation following the Network Install for a Workstation section Install Help document.

  2. From the CV Splash Screen Utilities tab open the License Manager.

  3. On the Server menu, click Search Network to list the Sentinel RMS server(s).

  4. Select the server that you wish to connect to and click Change Active Server.

  5. Confirm that you wish to change the active server, and close the dialog.
