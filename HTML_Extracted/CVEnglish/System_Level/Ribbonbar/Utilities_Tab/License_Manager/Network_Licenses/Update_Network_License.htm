Update Network License

There are several things which might require you to Update your Network License:

  1. When you renew your annual maintenance if your software is tied to annual maintenance.

  2. You purchase additional software modules.

  3. You are on a monthly short term or long term payment plan.

If you are setting up network licensing for the first time, please refer to the Server Setup topic.

Preparation

  1. Ensure any unexpired commuted licenses are checked in (returned).

  2. If you are using a keyed network license, ensure that the key is attached to the server.

Installing the license file

You can install the licenses on the license server from a CLS that is installed on either the server or a client PC (with Administrator Mode enabled).

  1. From the CV Splash Screen Utilities tab open the License Manager.

If the license server is already the active server, go directly to step 3.

  1. Select the server you wish to connect to and click Change Active Server. Confirm that you wish to change the active server, and close the dialog.

  2. Select the license server in the list and click the Install License button.

  3. In the dialog that opens, you have the option to request your license file over the Internet or browse to a file that you have already downloaded or received by email.

To request a license file over the Internet

  1. Click Request License File to request a license file from the CV server over the Internet.

  2. In the license activation dialog, check the details are correctly displayed and click Next.

  3. In the confirmation dialog, click Finish to install the new license file.

To install a file that has been sent to you

  1. Click Use License File.

  2. Browse for and select the license file (the license file will be named .one, .clss or .clsn, i.e. 1234-4567-1234-4567-1234-4567.one for example).

  3. Click Next.

  4. In the confirmation dialog, click Finish to complete the installation.

  5. The last dialog that is displayed includes an option to Remove Existing Licenses. This option will remove the old license file completely from the server before installing the new license file. It is recommended that you check this option when installing a license update.

Your updated licenses are now installed and ready to use.
