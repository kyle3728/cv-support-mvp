License Profiles

License Profiles allow you to save multiple profiles containing groups of Licenses which can then be selected when starting up the application.

  1. In the dialog, enable Use License Profiles.

  2. Enter a Profiles Location. Use the Browse button to select a directory, if required.

  3. Click Ask For Profile On Startup to ensure that a profile is selected when starting an application.

  4. Select the Current Profile to be used from the drop-down list.

  5. Select Save to save an new or existing profile or Save As to make a copy of the current profile.

  6. Click Delete to remove the selected profile.
