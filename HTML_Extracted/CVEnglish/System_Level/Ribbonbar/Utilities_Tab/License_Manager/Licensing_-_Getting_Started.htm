Licensing - Getting Started

You can only install the License after CABINET VISION has been installed on your PC.

When first starting CABINET VISION without an active License you will be presented with the License Setup Wizard.

Choose License Type

Step 1 - Select from the available Licensing methods. The method must match the Licence Delivery Note which was emailed to you.

Step 2 - Click the "Install" button.

Step 3 - Enter the Servercode from the License Delivery Note which was emailed to you. You should have received your Servercode via an email from Vero Software Limited orders@verosoftware.com that contains your Servercode (see image below). If you did not receive this email then you will need to contact your CABINET VISION Software Representative to have it emailed to you.

Step 3 - Click the Install button.
