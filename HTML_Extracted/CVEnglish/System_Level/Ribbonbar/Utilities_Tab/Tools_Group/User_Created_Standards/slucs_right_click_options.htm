Right Click Options

    

When Right Clicking on a UCS in the List View you have five options:

 Open

This option will Open the selected UCS in the UCS Wizard.

 Edit

This option will open the selected UCS in the UCS Editor.

 Copy

This option will Copy the selected UCS.

 Delete

This option will Delete the selected UCS.

 Add Divider

This option allows you to add a Section Divider to help you organize (group) your UCS's together. See the Divider highlighted in the illustration.
