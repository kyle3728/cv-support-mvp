Add (link) A Part

The Add (link) A Part UCS type is very powerful. Let's say for example you would like to add four Shelf Clips to your Adjustable Shelves. Follow along with this Topic to find out How.

Click on the New option in the User Created Standards window.

General Tab

The General Tab is where you enter the Name and select the Type of UCS you would like to create.

  1. Enter a Name for the UCS. (Add 4 Shelf Clips to Each Adj Shelf)

  2. For Type select Add.

  3. From the list to the right select Part for the category then the Shelf Clip Part.

    • If the Shelf Clip is not in the list you must create it. Go to the Parts Manager and add it to the Pulls Part Group. Once added, add a Material named Shelf Pin at 1/8" thick in the Material Manager. Next select this Material for the Shelf Pin Part which should now be listed in your Pulls Material Schedule.

  1. We will leave Macro Type as UCS which is the native macro language for CABINET VISION. If you are familiar with JavaScript and would like to use that language then you can select JavaScript from the list and the Wizard will construct the macro in JavaScript (UCSJS).

  2. Click Next to go to the next step in the Add (Link) A Part UCS Wizard.

Link To Tab

The Link To Tab is where you select which Part you would like to Link your new Part/Operation to.

  1. Select Part from the "Link To Each" section. This option is telling CABINET VISION to link these Shelf Pins to a certain Part.

  2. Select the Adjustable Shelf from the list.

  3. Click Next to go to the next step in the Add (Link) A Part UCS Wizard.

Assembly Class Tab

The Assembly Class Tab allows you to filter out based on certain Assembly Classes and Object Types. In this example we are only going to add shelf pins to Adjustable Shelves that are used in Standard Base Cabinets.

  1. Select Base as the Assembly Class.

  2. Select Standard as the Object Type.

  3. Click Next to go to the next step in the Add (Link) A Part UCS Wizard.

Assembly Properties Tab

The Assembly Properties Tab allows you to filter out based on certain Assembly Properties. In this example we are only going to add shelf pins to Adjustable Shelves that are used in Standard Base Cabinets with a name that starts with SP.

  1. For Assembly Name enter SP*.

  2. Click Next to go to the next step in the Add (Link) A Part UCS Wizard.

End Type Tab

The End Type Tab allows you to filter out based on certain Cabinet End Types. In this example we are not going to specify any End Types.

  1. Since we are not going to Filter based on End Types simply click Next.

Construction Methods Tab

The Construction Methods Tab allows you to filter out based on certain Construction Methods. In this example we are not going to specify any Construction Methods.

  1. Since we are not going to Filter based on Construction Methods simply click Next.

Conditions For Adjustable Shelf Tab

The Conditions For Adjustable Shelf Tab allows you to filter out based on certain Adjustable Shelf Properties.

There are 6 comparators you can use here:

In this example we are not going to specify any Adjustable Shelf Conditions.

  1. Since we are not going to Filter based on Adjustable Shelf Conditions simply click Next.

Shelf Clip Values Tab

The Shelf Clip Values Tab allows you to specify certain Shelf Clip values. In this example we want the Shelf Clip to be 1/8"w x 1/8"l x 1/8"t. We have already specified the Thickness in the Material Manager so here we can specify the Width and Length.

  1. Enter a Fixed Value of 1/8 for the Width.

  2. Enter a Fixed Value of 1/8 for the Length

  3. Click Next to go to the next step in the Add (Link) A Part UCS Wizard.

Shelf Clip Parameters Tab

The Shelf Clip Parameters Tab allows you to specify certain Shelf Clip Parameters. In this example we want the quantity of the Shelf Clip's to be 4.

  1. From the Parameter Name drop down select the QTY parameter.

  2. Enter 4 in the Parameter value field.

  3. Click the Add button to add this Parameter to the UCS.

  4. Click Finish to finish the UCS.

This is how to create a basic add Part/Operation UCS.
