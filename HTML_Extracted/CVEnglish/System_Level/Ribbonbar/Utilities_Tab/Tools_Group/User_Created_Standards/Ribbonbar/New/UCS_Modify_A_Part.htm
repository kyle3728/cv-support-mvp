Modify A Part

The Modify A Part UCS type is very powerful. Let's say for example you would like to modify the Depth of a Shelf placed in a Sink Base Cabinet to 12" to clear plumbing. Follow along with this topic to find out how.

Click on the New option in the User Created Standards window.

General Tab

The General Tab is where you enter the Name and select the Type of UCS you would like to create.

  1. Enter a Name for the UCS. (Shelf Depth in Sink Base Cabinet)

  2. For Type select Modify.

  3. From the list to the right select the Part Category then the Shelf part.

  4. Click Next to go to the next step in the Modify A Part UCS Wizard.

Assembly Class Tab

  1. Select Base for Assembly Class Type.

  2. Click Next to go to the next step in the Modify A Part UCS Wizard.

Assembly Properties Tab

  1. Type in SB* as the Assembly Name. Since I name all my Sink Bases with SB as the leading characters this is all I need to do to activate this UCS for my Sink Base Cabinets.

  2. Click Next to go to the next step in the Modify A Part UCS Wizard.

End Type Tab

The End Type Tab allows you to filter out based on certain Cabinet End Types. In this example we are not going to specify any End Types.

  1. Since we are not going to Filter based on End Types simply click Next.

Construction Methods Tab

The Construction Methods Tab allows you to filter out based on certain Construction Methods. In this example we are not going to specify any Construction Methods.

  1. Since we are not going to Filter based on Construction Methods simply click Next.

Shelf Conditions Tab

This Tab would allow us to specify certain Conditions the Shelf must meet for this UCS to activate. Since we want it to always activate to Sink Base cabinets we would not need to enter anything here.

  1. Since we are not going to Filter based on Shelf Conditions simply click Next.

Part Properties Tab

  1. Select the Width parameter.

  2. Select "Is Equal To" as the How To Modify property.

  3. Enter 12 for the value for the Shelf to be equal to.

  4. Click Finish to finish the UCS.

This is how to create a basic Modify Part/Operation UCS.
