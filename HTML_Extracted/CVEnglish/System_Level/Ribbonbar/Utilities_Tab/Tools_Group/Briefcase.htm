Briefcase

The Briefcase feature works on a Network Workstation and allows you to disconnect the workstation from the Network Server with copy of the Network databases so you can work away from the office. You can later come back to the office and using this option reconnect to the Network Server and any changes you have made to the system data-set will be synced to the network.

Briefcase only handles the databases and not the license. You still need to Commute your Network License if you are not using a standalone License.

This button communicates both states:

 Briefcase disabled - This indicates you are connected to the Network data-set. When you click this option you will be presented with the Briefcase dialog. Click OK to disconnect the workstation from the network. This process can take quite some time depending upon the size of the data-set and your networks file transfer speed.

The progress of the Briefcasing will be shown in the progress bar.

 Briefcase enabled - This indicates you are disconnected from the Network data-set.
