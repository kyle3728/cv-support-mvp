Backup Utility

Backup List Tab

The Include/Ignore area is where you define which CABINET VISION data to include in the Backup.

Anything in the Include pane will be included in the backup and items in the Ignore pane will not be included. Use the corresponding buttons to move items from one pane to another.

There are several options here:

Construction Methods - Including this option will backup all Cabinet and Drawer Construction Methods, User Created Standards and Intelli-Joints. This will backup CVData (SQL Database).

Part / Material / Door Catalogs – Including this option will Back Up all Materials, Material Schedules, Doors and Parts. This will backup CVData (SQL Database).

User added Graphic files used by Textures will be added to the backups automatically when Part/Material Catalog is selected.

Object Catalogs - Including this option will backup all of the Object Catalogs. This will backup CVData (SQL Database).

Drawings – Checking this option will backup all CAD Symbols and Title Blocks. This will backup CVData (SQL Database).

Report Database – Checking this option will backup all Reports, Report Group and Bid Rate Tables. This will back up the file report.mdf.

CNC Database - Checking this option will backup the Tool and Machine catalogs. This will backup CVData (SQL Database).

Posts - Checking this option will backup all Posts.

System Parameters - Checking this option will backup the System Parameters that are stored in the Windows Registry.

Material Summary - The Database that stores Material Summary information for the last Job taken to the Report or Bid Center.

Prices.mdf - The Simple Price Table.

.CFG Files - Your Job Schemes.

xCRM Data - The file that contains the database information for the xCRM multiplier.

Default.dat/DefaultCLST.dat - The files that contain Layer information and custom settings.

BidData.mdf - The BidData database.

All Graphics - All the Graphics files in the \Graphics folder.

Assigned Materials Textures - All the Graphics files which are assigned to Materials.

 

Add Job - Clicking this button will allow you to attach a Job to your backup. This is especially handy when sending a set of Databases to Technical Support. You can attach a Job so the Technician sees exactly what you see on the screen.

Toolbar

Following are the tools in the Backup Utility toolbar.

 Save Backup - Selecting this option will perform a Backup based on the items in the Include pane.

 Email Backup - Selecting this option will allow you to select one of your existing backups to send via email.

 Zip Backup - Selecting this option will perform a backup directly so a zip file.

 Delete Backup - This option you can delete old backups that either you or a CABINET VISION Install has created. Deleting old backups (Archives) will allow you to free up hard-drive space.

 Settings - With this option you can setup automatic deletion of backups (Archives) as well as establish the destination of the Backup files.
