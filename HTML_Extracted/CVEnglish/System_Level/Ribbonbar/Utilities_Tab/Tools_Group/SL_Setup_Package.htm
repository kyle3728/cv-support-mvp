Setup Package

The Setup Package utility is a means to transfer specific objects from one computer running CABINET VISION to another.

Exporting a Setup Package

The Setup Package feature allows you to create a single file that can store and transfer multiple custom items for import into another CABINET VISION user’s or CABINET VISION Tech’s system.

The beauty of the CABINET VISION Setup Package is that any unique identifiers are stored and recreated on the other user’s system. In the simplest terms it is plug and play. A good example of where this comes in handy is setting up a Blum Metabox drawer system. Prior to the implementation of the Setup Package Utility you would have to export each UCS and Intelli-Joint as well as having to recreate each Drawer Guide Material and Drawer Guide Material Schedule. Then link each of these items together in your UCS. Another method would have been to send a Technician or CABINET VISION user your database files thus overwriting their databases. With the Setup Package you simply select the UCS and the Setup Package Utility will scan the UCS and pick out each Intelli-Joint, Material, Finish, Material Schedule, and etc... that is involved in properly creating the Blum Metabox Drawer System.

Vendors

This topic explains how to export a package and send it to Tech Support or another CABINET VISION user.

  1. From the System Level go to the Utilities Tab | Setup Package

  1. Select the item or items you would like to include in the Package from the list on the left side of the screen. After highlighting an item you want to add to the package simply click the "Add Items To Package" button in the lower left corner. There is no limit to the number of items that can be added to a package.

You can use Windows  + Click or  + Click to select multiple items in the active Category.

 

  

  1. After selecting the items to add to the package click on the "Save Current Package" tool in the toolbar.

In the graphic below we selected several Material Schedules, notice that it also picked up the corresponding Materials, Finishes, Finish Types, and Parts.

 

  1. The next screen that pops up is the Create License screen and gives you the ability to put contact information in the Package for distribution. Make sure to type in a License Description if one is not already available and fill out all other details.



    If the Package includes an Object Catalog the Create License dialog will pop up. Depending upon whether it is a Custom or Modular Catalog you will have up to 3 options for the License Type.

    1. None - Normal Assembly Catalog

    2. Protected Assembly Catalog - Allows you to create a Protected (Read Only) Catalog. A Protected Catalog can not be edited in anyway. You also have the ability to Hide the Cutlist as well.

    3. Distributed Assembly Catalog - This function will turn the selected Assembly Catalog into a self-contained Catalog that does not require the end user to have the necessary Materials, Construction Methods, or other items to make the Catalog work. This is useful if the Catalog will be used by outside dealers or others who need access to the Catalog but should not be allowed to make changes to it. This function combines all the information needed for the Catalog into a single read only format file. This file can then be copied to a DVD, thumbdrive, etc... for distribution. There are two main advantages for using this procedure. First, the dealer is not able to make permanent changes to the Catalog, thus giving the manufacturer complete control over Catalog content. Second, the manufacturer does not have to be concerned with distributing and updating all the individual database files associated with a CV Catalog. Be aware that a distributable file cannot be updated, but must be replaced when a change is made to the Catalog.

  2. Select a location to save the Package and type in a Name for the Package then select "Save".

  

  1. You may now either close the Setup Package screen or, if you use Microsoft Outlook you can click on the "E-Mail Current Package" tool in the top toolbar.  This will attach the Package to your e-mail and allow you to e-mail to Tech Support or to another CABINET VISION User.

 

  1. You may now close the Setup Package screen. You have successfully exported a Package from CABINET VISION.

Importing a Setup Package

This section explains how to import a package that has been sent to you by Tech Support or another CABINET VISION user.

  1. From the System Level go to the Utilities Tab | Setup Package

  1. Click the "Open An Existing Package" tool in the toolbar.  

  

  1. Select the package file from the location where it is being stored and click Open.

  

  1. Click the Import tab.

  

  1. Click the "Import Package" button on the Import tab.

Prior to importing you can optionally have all the names of the objects imported from the Package by setting a Prefix using the Add Preix button on the Import tab.

Overwriting Existing Items

Following are the options that allow you to choose how to Overwrite items that already exist in your system.

All Matches - Overwrite - Will overwrite all existing items in your database with the contents of the Package.

All Matches - Use Existing - Will use all the existing items in your database rather than overwrite them with the Package contents.

Child Matches - Overwrite - Will overwrite all existing children of items in the Package that match those in your database.

Child Matches - Use Existing - Will use all the existing children in your database rather than overwrite them with the Package contents.

Toggle Overwrite - Allows you to Overwrite existing items by selecting them individually. This will cause items in the Package to either overwrite the existing item or to use the existing item.

  1. Click on OK for the Import Complete notice.

  1. You may now close the Package screen you have successfully imported a Package into CABINET VISION.

Uninstalling a Setup Package

This section explains how to uninstall a package that you have imported.

  1. From the System Level go to the Utilities Tab | Setup Package

  2. Click the Show History of Imports tool in the toolbar.

  1. Select the Package you want to uninstall from the list of Packages you have previously imported and select Delete.
