The Welcome Screen

When you initially open CABINET VISION, the Welcome screen is the first window that you will see. Pictured below is the typical Welcome screen for CABINET VISION.

Click on the options in the image below for more help

The main area of the Welcome Screen has 4 main options.

New Job

Open Job

Recent Jobs

This area houses a list of the 50 most recent jobs you have worked on. You can use the Search bar to search through all your recent jobs to find the one you need. You can also pin  favorites to the top of the Recent Jobs list.

Right Click Options

When right clicking on a job in the Recent Jobs are you have several options:

 Open

 Open file location

Will open the File Opne window at the location where the selected file resides.

 Open a copy

Opens a copy of the selected file which you can Save As with another name after opening.

 Delete file

Deletes the selected file.

 Copy path to clipboard

Copies the path to the selected file to the clipboard. This can be handy to use in File Explorer.

 Pin to list

Pin's the selected job to the top of the Recent Jobs list.

 Remove from list

Removes the selected job from the Recent Jobs list.

Recent Places

This area houses a list of the most recent locations you have opened jobs from. Clickng a place opens the File Open window at that location. You can also pin  favorites to the top of the Recent Places list.
