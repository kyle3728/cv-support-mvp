Formula for the X (Left/Right)

Look at your cabinet in 3D.  Look at it from a Face ortho view and click on BOARD@4.  Notice where the reference point is located.  Click on the other boards and notice where their reference points are located.  See if you can come up with the formula for the X of BOARD@4 yourself.  Hint; you already know what the formula is because you placed the board left and right in the cabinet to start off with. Click Here To See Formula.

  1. Add a part.  Size the part correctly and place the part correctly.  You've already done that.

  2. Select a part to work on.  We are working on BOARD@4.

  3. What is it you are trying to do to this part?  We are trying to place it left and right.  Write your formula out in English first!  Make sure you write it down so you won't get confused later.  Use the formula you used to manually place the part.  Try to figure it out yourself but if you can't, Click Here to see Formula.

  4. Trace it through the Object Tree, and write it down!  The first level is always "CAB" no matter what the name of the object is.  Try to trace it out yourself but if you can't, Click Here to See the Code.

  5. Go back to the part you are working on (board@4 - After you have traced it through the tree and written the code down).

  6. Select the parameter you want to work on.  We are working on the X.

  7. Select "Edit".

  8. Change from a "Value" to an "Equation".

  9. Write your code on the "THEN" line. Click Here to see Code.

  10. "Test" you code.

  11. Always go back and look at your cabinet in 3D (at the cabinet level) and see if it still looks OK.
