What We Learned

  • You should have noticed that any time the X, Y, or Z is equal to 0" you don't have to write a formula for the placement parameter.

  • You should have noticed that we didn't write any formulas for the thickness of the parts, the thickness of the parts comes from the material the part is made from.

  • You should have noticed that you didn't write any formulas for the AX, AY, or AZ.  You almost never have to write a formula for the rotation because once a part is rotated into place it almost never changes rotation.

  • You should have noticed that if you size and place the part correctly to start with, you are using the formulas you need to size and place the part.
