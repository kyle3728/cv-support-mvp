Shaping the Corbel

Now that you have created the corbel and given the parts intelligence you may wish to shape board@3 so it looks more like a corbel.  When you shape anything in Solid and then later change the size of the object that was shaped, the shape will distort.  In order to control the shape and not have it distort you must "constrain" certain entities which make up the shape. The following exercises are designed to teach you how to shape a part and constrain it so that the shape doesn't distort.

For the first exercise in shaping a part we will take the above corbel and shape it to look like the following corbel.

If you are not already at the Assembly Level, right click on your corbel and select "Edit".

  1. Click on the End ortho view tab.

  2. Right click on Board@3 and select "Edit Shape".

You would not use the "Edit Shape" button in the sidebar, because you would be shaping the entire object or assembly. We have not given the parts any intelligence for changing shape when the assembly changes shape, so shaping the entire assembly would have no effect on any of the parts.

  1. The first thing you want to do is add some reference lines to use to create the clip. We want the clip to start and end at 2 inches. So, use the Modify | Offset option to offset the top and back lines in 2 inches.

  1. Now that you have your reference lines in place make sure that "Snap to Points" is enabled.

  2. Next, select the Drawing | Line option.

  3. Next, with the Line options active, select the front end point of the top reference line.

  1. Then the bottom end point of the back reference line.

  1. Now you can see you have your clip in place. So you can now get rid of the reference lines using the Modify | Delete option.

  2. To establish your new Part perimeter you will need to trim away the unnecessary portions of the Part's shape using the Modify | Trim Delete option.

  3. Simply click the portions of the Part shape you do not need.

  1. To confirm that your shape is fully enclosed select the Shape | Check Shape option. Notice after running Check Shape that all of the lines that make up the Part's perimeter are Green. This indicates the Part shape is complete with no errors and that you are good to go.

  1. So now you have the basic shape of the part, you can click the Return option to Save it.

Now you already have intelligence added to the parts, so that when the object is re-sized the parts will re-size themselves too. And, now you have just changed the shape on the middle part so that it looks like you want it to. So, now test everything by changing the size of the object and then look to see how everything reacts.

Here is an example of what happens to the shape when you change the size of the Object. You can see that the 2" offset we wanted is not being held when you change the size of the Object. So you need to Constrain certain portions of the shape to prevent this from happening.

  1. Return the Object to the standard size and return to the Assembly Level End Ortho View and then right click this part and select Edit.

  2. From a Front Face View at the Part Level click the Edit Shape button in the sidebar to add some Constraints to the Shaped Part.

We only need to add simple Constraints to the lines we wish to remain at 2".

  1. Select the topmost 2" Line.

  1. For Start Y in the sidebar select the Lock Top option.

  1. Select the bottommost 2" Line.

  1. For End X in the sidebar select the Lock Left option.

  1. Click the Return option to Save your Shape changes.

  2. Now that you have Constrained your shaped part, click the Return option to go back to the Room Level and test the shape by changing the size of the object again.

You can see here the Part is sized and shaped properly.
