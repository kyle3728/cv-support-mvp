Reference Point

Imagine an Assembly with no Parts in it; it would be an empty box. All you would have is just the outline of the Assembly. This empty box would have width, height and depth but no Parts. In the left, bottom, back corner of this empty box is an imaginary point called the "Reference Point". An Assembly is positioned on a Wall in reference to this point and the reference point of the Wall. All Parts added to the Assembly will be positioned referenced to this reference point and the reference point of the Part.

Reference point of an Assembly

The reference point for an Assembly is in the left, bottom, and back corner of the Assembly.

Reference point of a Part

The reference point of a Part is in the left, bottom, and back corner of the Part, when that Part is in normal orientation, in other words, if the Part has not been rotated.
