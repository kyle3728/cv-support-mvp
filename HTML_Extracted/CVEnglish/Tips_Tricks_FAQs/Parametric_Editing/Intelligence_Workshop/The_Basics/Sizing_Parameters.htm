Sizing Parameters

The "Sizing" parameters are DX, DY, and DZ these parameters are used to size an object. The letter D in DX, DY, and DZ stands for Dimension. Any time you are doing anything with the dimensions of a Part the parameter will start with a D.

DX = the width of an object

If the object is a Cabinet then DX is the width of the Cabinet, if the object is a Part then DX is the width of the Part. The DX measurement is always across the grain of a Part.

DY = the length or the height of an object

If the object is a Cabinet then DY is the height of the Cabinet, if the object is a Part then DY is the length of the Part. The DY measurement is always with the grain of a Part.

DZ = the thickness or depth of the object

If the object is a Cabinet then DZ is the depth of the Cabinet, if the object is a Part then DZ is the thickness of the Part. The DZ measurement is always from the "Back" of the Part toward the "Face" of the Part.

If the object is a Part, you will most likely never write a formula for its thickness as the thickness of a Part is determined by the thickness of the Material the Part is made from.

Sizing Parameters For an Assembly

  • DX = the width of the Assembly.

  • DY = the height of the Assembly.

  • DZ = the depth of the Assembly.

Sizing Parameters For a Part

  • DX = the width of the Part, across the grain.

  • DY = the length of the Part, with the grain.

  • DZ = the thickness of the Part.
