Sixth Exercise Notes on Formulas

In the section that follows, I want you to think about whether or not we need a formula for the parameter in question.  Do we?  Think about it a little.  What should happen to the parts if we change the width, height or depth of the Wood Top?  If you need help you can click on "Answer".

I want you to try and come up with the formula yourself.  Think about how the parts are related to the overall size of the Wood Top, and the other parts in the Wood Top.  Click on each piece and notice where the reference point is and how it relates to the reference point of the Wood Top (left, bottom, and back corner).  If you need help you can click on "Formula".

Once you have written down your formula, trace it through the tree to get the code and write the code down.  If you need help you can click on "Code".

I have purposely left out the step-by-step instructions for giving the parts intelligence; if you did the five exercises up till now you should know the steps.  If you need help refer to the section "Steps to writing a Formula " in the first exercise.

There is more than one way to skin a cat, so you may come up with a different formula and code than the samples shown here.  If you do, try your code and see what happens.  If it works, great!  Then compare it to what is shown here and see which one you like best.
