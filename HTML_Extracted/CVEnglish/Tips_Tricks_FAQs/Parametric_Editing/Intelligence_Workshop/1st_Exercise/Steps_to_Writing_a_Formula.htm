Steps to writing a Formula

By now you should see a pattern developing.

  1. Add a part. Size the part correctly and place the part correctly.

  2. Select a part to work on. You might have more than one. If you have more than one part that needs intelligence, pick one part to work on.

  3. What is it you are trying to do to this part? Are you trying to place it left and right (X) or up and down (Y) or in and out (Z) or are you sizing it for width (DX) or length (DY)? Pick one parameter to work on at a time.

  4. Write your formula out in English first! Make sure you write it down so you won’t get confused later. Example formula, the height of the cabinet minus the height of the toe kick.

  5. Trace it through the Object Tree, and write it down! If your formula has more than one part trace each part of the formula separately.

  6. The first level of the Object Tree is always "CAB" no matter what the name of the object is.

  7. Every time the tree branches add a period.

  8. When you step through each level of the tree add a period.

  9. Go back to the Part you are working on. After you have traced it through the tree and written the code down, click back on the Part you want to work on.

  10. Select the Parameter you want to work on.

  11. Select "Edit".

  12. Change from a "Value" to an "Equation".

  13. Write your code on the "THEN" line.

  14. Always "Test" your code (If possible, some code may rely on other code you haven’t finished writing yet.).

  15. Always go back and look at your assembly in 3D (at the Assembly Level) and see if it still looks OK.
