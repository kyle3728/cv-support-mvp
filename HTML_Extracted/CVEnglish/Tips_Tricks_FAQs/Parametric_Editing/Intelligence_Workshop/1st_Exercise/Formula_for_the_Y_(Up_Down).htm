Formula for the Y (Up/Down)

Look at your cabinet in 3D.  Look at it from a front ortho view and click on the BOARD.  Notice where the reference point is located.  Look at it from a side view.  Notice where the reference point is located.  When writing code for the position of a part it is important to know where the reference point is.

There are several formulas that will work for the Y position of this BOARD.  You will learn as time goes by that there is always more than one way to do things in Solid.  We could say the Y is equal to the height of the cabinet minus the thickness of the top stretcher.  Or we could say the Y is equal to the height of the cabinet minus 3/4".  It would be better to use the first formula because if the thickness of the top stretcher changed the first formula would take this into consideration whereas the second formula would not.  We could say the Y of the BOARD is equal to the Y of the top stretcher.  If you click on the top stretcher you will note that it and the BOARD have the same up and down position.  This is a good formula so we will use it.  The Y of the BOARD = the Y of the Top Stretcher.  So now that we have our formula.  We need to trace it through the object tree.

  1. Click on one of the ortho views.

  2. Click on "Object Tree".

  3. Open the Object Tree until you find the Top Stretcher, click on it and then click on the Y parameter on the right.  If we trace this through the tree it is CAB.CASE.TS.Y we write this code down so we don’t forget it.

  4. Click on the BOARD then click on the Y parameter.

  5. Click on "Edit". Your screen should now look like the follow screen.

  1. Select Equation and then on the THEN line type the code we wrote down earlier. CAB.CASE.TS.Y.

  2. Click on the test button and it should say "Condition returned: True. Equation returned: 33 3/4 or whatever value you were expecting. Click on OK.

Look at your cabinet in 3D does it look OK? If not you typed something wrong. See "Troubleshooting Error Messages".
