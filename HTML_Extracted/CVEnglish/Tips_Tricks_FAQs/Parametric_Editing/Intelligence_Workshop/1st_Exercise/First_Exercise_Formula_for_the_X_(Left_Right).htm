First Exercise Formula for the X (Left/Right)

Look at your cabinet in 3D.  Look at it from a Face ortho view and click on the BOARD.  Notice where the reference point is located.  Look at it from a side view, click on the BOARD.  Notice where the reference point is located.  When writing code for the position of a part it is important to know where the reference point is.

We will need a formula for the X (Left/Right) because if the thickness of the end changes this part will need to move left or right.

As far as a formula for the X position of the BOARD, there are several that could work.  We could say that the X of the Board was equal to the thickness of the left end.  However, when writing formulas it is best to avoid using the ends in your code if at all possible.  The reason for this is that there are 10 different end "Types".  We would have to write 10 formulas, one for each end type.  See the following picture.

Example if we said the X position of the Board was equal to the thickness of the left unfinished end and there was no left unfinished end, our code would not work.  The part could end up positioned wrong or not appearing in the cabinet at all.  So we need to think up a different formula.

When writing formulas the easiest thing to do is just to find something in the cabinet that already has the value you need and that will change in the way you want the parameter you are working on to change.

If you look at the cabinet in 3D you can see that the Deck, the Top Stretchers, and the Nailers are all butted into the left side of the cabinet the same as this BOARD.  If the thickness of the end changes the Deck, the Top Stretchers, and the Nailers will all move in relation to the thickness of the end.  So we could use the X position of any of these parts as the X position of our board.

When choosing what you want to use as the reference for adding intelligence, be sure that you do not choose something that may change later on.  As an example, if you relate a part to the size of a mid stile, and later on, you delete the mid stile from the cabinet, the formula you wrote will be invalid.

We will use the X position of the Deck as the X position of our Board.  So our formula will be the X of the Board = the X of the Deck.  Now that we know what our formula is, we only have to trace it through the Object Tree.  So we can put it in a language the computer can understand.

  1. Open the Object Tree by clicking on one of the ortho views, then click on "Object Tree".  We then open the tree up until we find the Deck (notice the BOARD we added is showing right under the cabinet), we highlight the deck and then we click on the parameter we want, the X.  We then trace it (the X of the Deck) through the tree starting at the top and working our way down to the X, and the code we come up with is "CAB.CASE.DE.X".  We write this down so we don't forget it, or highlight the path in the Object Tree path bar (highlighted in the illustration below) to the Windows Clipboard.

  1. Now we go to the part we want to write the formula for.  So while still in the Object Tree click on the BOARD.  Then click on the X, this is the parameter we wish to write the formula for.

  1. With the X highlighted, click on the "Edit parameter" button in the Object Tree. This will open the "Parameter Edit" screen. Notice the "Name" of the parameter is "X" and notice that it is set to a "Value" of 3/4".

  1. Select Equation, the "Add", "Delete", and "Test" buttons will all be highlighted.  And the "IF" line and the "THEN" line will also be highlighted.

The IF Line

The "If" line, and the "THEN" line, are the lines you will be writing your formula, or code on.  The "IF" line is used to set a "condition" that must be true before your code will execute.  For example you could say Cab.Name = 'Sink' this means that the name of the cabinet must be "Sink" or the program will not execute the code on the THEN line.  We are not going to go into it that deep at this time, so we will not be changing anything on the IF line.  You will notice that the IF line has a 1 on it, 1 means true, so anything we write on the THEN line will execute.

The THEN Line

The "THEN" line is where we write what we want the program to do, if the condition on the IF line is true.  This is where we will write the code that we wrote down earlier.

  1. Click on the "THEN" line and remove the "0". Then type in the code we wrote down earlier "CAB.CASE.DE.X", do not type in the quotation marks.

Note: It makes no difference to the computer if you type your code all in upper case letters, or all in lower case letters, or in some "Combination" of the two.

Testing your Formula

After typing in the code you always want to "Test" your code to make sure it is working.

  1. Click on the "Test" Button. In the "Test Result" area it should say "Condition returned: True.  Equation returned: 3/4." Or whatever value you were expecting.

  1. Click on "OK".  This will return you to the Object Tree and you will see the result of your equation.

 

  1. Look at your cabinet in 3D does it look OK?  If not you typed something wrong. See "Troubleshooting Error Messages".
