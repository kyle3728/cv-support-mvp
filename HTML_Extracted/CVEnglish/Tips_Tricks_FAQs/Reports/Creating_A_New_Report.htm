Creating A New Report

Although you can certainly create a New Report by simply clicking the New Report option you might be better off finding a Report that has the data you need already within it. If you find that a Report has the data you need then follow these steps:

  1. Create a Copy of the Report.
  2. Right click on the Report and select Properties or click the Report Properties option in the ribbonbar.
    1. Edit the Datasets to get the information you want from the Report.
  3. Edit the report in the Report Designer to suit your needs.

The Report Designer has its own Help file. Please see it for more information on Designing Reports.
