Assign a Specific Outline Tool to a Material

I need to run a specific Outline Tool on a certain Material and the Tool is not my default Outline Tool. How do I do this?

This can be handled easily via a UCS. Let’s say you wanted to use your 3/8 compression bit which is not your Outline Tool to always cut out your 1/4” Unfinished Backs. The first thing you need to do is identify the 3/8 Compression Tool’s Tool ID Number. The Tool ID# is in a grayed out field in your Tool Catalog. Once you have this number let’s say its 13 then your UCS would look like this:

;Make the 3/8 Compression the Unfinished Back Outline Tool

For Each UB Part

TOOLID = 13

If your Unfinished Backs, depending on selected Material Schedule, could be also made out of 3/4" Material that you would want to use your default Outline Tool on. You could add an additional Material filter to the UCS. First you will need to get the Material ID# for your 1/4" Material from the Material Manager. Then write your UCS like this:

;Make the 3/8 Compression the Unfinished Back Outline Tool

For Each UB Part

 If _M:MatID = 123 Then

  TOOLID = 13

 End If
