Troubleshoot No Tool Found Errors

Why am I getting this message after I Optimize saying, "No Tool found for dado or bore..."?

For Line Boring:

Make sure the diameter of a Tool in the Tool Catalog is an exact Diameter match for the Operation. If you defined the Operation in millimeters make sure you also define the Tool in the Tool Catalog in millimeters and vice versa.

Make sure the Vertical Drill tool in the Tool Catalog is assigned to the Machine you are using in the S2M CENTER. The safest assignment is N/A which means any Machine defined in the Machine Catalog can utilize the Tool.

Make sure there is a valid Tool in the Tool Catalog available for this operation.

For Dadoes:

Make sure the width of the Dado is not smaller than the width of the smallest diameter Flat Router bit in the Tool Catalog.

Make sure the Flat Router bit in the Tool Catalog is assigned to the Machine you are using in the S2M CENTER. The safest assignment is N/A which means any Machine defined in the Machine Catalog can utilize the Tool.

Make sure there is a valid Tool in the Tool Catalog available for this operation.
