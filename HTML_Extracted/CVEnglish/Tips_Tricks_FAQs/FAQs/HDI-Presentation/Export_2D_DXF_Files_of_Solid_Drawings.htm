Export 2D DXF Files of CABINET VISION Drawings

Is it possible to export 2D DXF files of CABINET VISION drawings?

Yes...

  1. When looking at the Elevation view, right click and select the "To Drawing" option.

  2. Go to the Drawings area, make sure page is blank, then drag the new Scene on the page.

  3. Go to Utilities | Export DXF.

  4. Choose a location for the file, name it, and save it.

Be careful with the name that you use for the "Room" when setting up the Job. If you use certain punctuation marks like "/,\,." your 3rd party CAD application may not open them.
