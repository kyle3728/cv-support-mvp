Editing/Creating a Title Block

This tutorial is to help you create a title block that you can use when printing from the drawings page. This will cover the basic editing techniques needed, and will allow you to be more creative as you gain experience.

To get started, you need to be working in a new Job. Go to the Drawings Area by selecting the Drawings tab from the Plan view within a new Job.

You now see a blank page to which you can add your Drawings and a Title-Block. This page can then be printed for your customers.

Go to the option Drawings Library.

Select Title Blocks as the "Type" of Template you want to Edit.

Highlight the desired Title-Block Template and click Edit.

For this tutorial, we will use the first title block template labeled "Small Format - Submittal". There are other style and size Title-Blocks available which you can explore at a later time. Highlight "Small Format - Submittal" and click Edit.

This loads (opens) the Title-Block Template in Edit mode. You are now ready to begin editing the Title-Block Template.

To help with the editing, use the Zoom functions to enlarge the area of the Title-Block you wish to work on.

 

Each Text field within this template can be edited.

To begin, move your mouse right on top of the "Name" field and right click (or Double Click). When the menu pops up, left click on the word Edit. The dot, at the lower left of the text field, indicates this field is highlighted and ready for editing. This dot also appears with a left click.

Notice the variable enclosed in Brackets {} to the right of the text Name. This is an Automatic Text field.

You can change the Text or the Auto-Text in any of these Fields. You also can add additional Text utilizing the CAD Text Tool. To add additional Auto-Text simply select what you would like to add from the menu that is available by clicking the Action button. See the Automatic Text topic for further information along with a complete list of the available auto-text variables.

To move any of these Text fields, move your mouse over the dot that appears when that text field is highlighted. Your mouse will change to four arrows and you can then drag this field to position it anywhere you want.

You can continue to edit any of the fields contained within the Title-Block. After you have finished your changes, left click on the home tool to get out of the Editing mode and automatically save your changes.

This new Title Block is now saved and ready to use. In order to set this Title Block as the default, right click and select Clean Sheet. Then reload the new Title Block. Now it will stay until you change it.
