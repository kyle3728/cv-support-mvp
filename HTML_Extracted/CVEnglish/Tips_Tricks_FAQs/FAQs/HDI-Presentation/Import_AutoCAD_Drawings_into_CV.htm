Import AutoCAD Drawings into CABINET VISION

How do you import AutoCAD drawings into CABINET VISION?

3D AutoCAD Drawings

2D DXF's

Go into CAD Mode.

Go to Insert | DXF.

Browse to and select your 2D DXF file.

A 2D DXF file can define quite an assortment of CAD entities that CABINET VISION can not process. If CV can not process an entity it will be ignored and a warning will be issued. The following entities are supported by CV:

  • Arc

  • Circle

  • Line

  • LWPOLYLINE

  • POLYLINE

  • SEQEND

  • VERTEX
