Create a Flip Up Window Seat

  1. Start a new Job using the Construction Method, Material, and Door you want to use for the Window Seat unit.

  2. Place a Base Cabinet on a Wall - size it to the size you want to use for the Window Seat.

  3. Double click on the Cabinet to go to the Assembly Level.

  4. Go to the Section Editor.

  5. Go to Modifications | Default Assembly.

  6. Go to the End Ortho (Side Smiley Face) view, click on the Edit Shape button in the sidebar.

  1. Select the top edge and in the sidebar select Face as the Side Type.

  1. Click the Return option in the ribbonbar.

  1. Click Yes to save the Shape.

  1. Return to the Section view and note the Side Type Selector in the sidebar.

  1. Select the top edge and note it is now viewable in the Sectioning screen so you may set Doors to this face and hinge it correctly.
