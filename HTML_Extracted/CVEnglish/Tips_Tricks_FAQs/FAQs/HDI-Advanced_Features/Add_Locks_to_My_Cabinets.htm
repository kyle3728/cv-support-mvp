Add Locks to My Cabinets

I want to add locks to a Cabinet. How do I do this?

Creating a Model

In the following example we are going to create a Door Lock Model which includes its machining. To start select the Miscellaneous Group and create a new Miscellaneous Material named "Cam Lock".

Once in the Properties area with this new Material make sure to go to the Miscellaneous area and set the Model option to True.

Select the Model option in the ribbonbar.

From the sidebar select the "Assembly" Object. This Object is simply a sample part to help you orient your Graphic. This object is displayed in brown on screen. For the following Tutorial go ahead and change the DZ (Thickness) of the Assembly Object to .75".

From the sidebar select the "Primary Work Plane" Object. This is the default Work Plane and also represents the insertion (Reference Point) of the Model. Think of this as the container for your Model, everything will be referenced from this Object. The center of the Work Plane object is the insertion (Reference Point). Change the X and Y of the "Primary Work Plane" to 4". The true position of the "Primary Work Plane' will be set when the Model is placed within a Job whether automatically, manually, or via UCS.

  1. Set the View Mode to Front.

  1. With the "Primary Work Plane" object selected in the sidebar click the Add Work Plane option in the ribbonbar.

  2. Set the "Front WP" X and Y to 0.

  3. Click the Add Hardware option in the ribbonbar.

  4. Simply browse to the 3D Model you wish to import making sure to set the appropriate file type (highlighted below) then click Open.

The model used in this tutorial are available in the installation extraction folder at:
C:\Users\Public\Documents\CABINET VISION\Download\CV_2023_n_nnnn_nnn_SetupDvdFiles\Support\Sample Models\

  1. The next screen offers several modification options you can perform prior to importing the 3D Model. Click OK to import your model.

See the topics on Importing DXF or Importing SketchUp Models for more information on the modification options on this screen as well as information about the formatting requirements of 3D DXF files.

  1. Once imported I am going to change the following Properties for the Cam Lock:

    1. Name = LOCK

    2. Description = Cam Lock

    3. Finish = Chrome

    4. Finish Type = Metal - Matte

    5. X = -15/32 (This will center my lock horizontally on the Plane)

    6. Y = -15/32 (This will center the cylinder of my lock vertically on the Plane)

    7. Z = -1 7/32 (This will place the lock face flush to the door face)

  1. Now we need to add some machining for the Lock.

  2. Right click on the "Front WP" Object and select Edit.

  3. Select Work Plane - Front from the sidebar to add Operations to it.

  4. Now we will add the hole for the cylinder.

 

See the CAM View Main Tab topic for more information on adding CAM (Machining) Operations.

  1. Click the Return ribbon option to leave CAM mode.

  2. Switch to 3D Mode to see the Operation that was added. If you switch the Render Mode to Fill Mode then select the Front Hole in the sidebar you can see the Operation more clearly.

  1. Click the Return option in the ribbonbar to save your new Model.

Now lets place this piece of hardware on a Part in CABINET VISION.

  1. Save the Model and close the Material Manager.

  2. Now let's create a Part in the Part Manager to associate this Cam Lock to. Open the Part Manager and select the Pull Group.

  1. Click the New option in the ribbon.

  2. Type "CAMLOCK" for the Name and give it a Description of "Cam Lock".

  3. Close the Part Manager.

  4. Open Material Schedules.

  5. Select the Pull Material Schedule Type, and pick a specific Material Schedule.

  6. Now assign the Cam Lock Material we created earlier to the Cam Lock Part.

  7. Start a new Job making sure that Pull Material Schedule is active.

  8. Draw a Wall.

  9. Place the Empty Base Object from the Custom Cabinets | Base Cabinets group.

  10. Re-size the Empty Base Object to 24" wide, 30" high, and 3/4" deep.

  11. Take the Empty base Object to the Assembly Level | Face Ortho View.

  12. Click New Part in the sidebar.

  13. Select Board from the Part Type list.

  14. Click Next.

  15. Make the Length 30" and the Width 24".

  16. Click Next.

  17. Select Front for the Orientation.

  18. The default Finishes and Textures are OK here so click Finish.

  19. Change the Left/Right for the new Part to 0 in the sidebar.

  20. Click anywhere on the screen to de-select the New Part.

Now that we have a new Part in place let's put our Cam Lock on it.

  1. Click New Part in the sidebar.

  2. For Part Group select Pull.

  3. Select Cam Lock from the Part Type list.

  4. Click Next until you get to Finish then click Finish.

  5. With the Cam Lock selected set the Left/Right to 2".

  6. Set the Up/Down to 2".

You will see that your Cam Lock is placed along with all the machining necessary for mounting.

 

Now you can write a User Created Standard to add the Lock to the Cabinets you need it to be listed under. Below is the text to utilize in your UCS:

;Links Cam Lock to Door

 

 For Each DOR Part

 

 If LOCK = 1 Then

 

  Dim CAMLOCK as New Pull

  CAMLOCK.Visible<bool> = True

  CAMLOCK.DX = 0

  CAMLOCK.DY = 0

  CAMLOCK.DZ = 0

  CAMLOCK.Y = :DY - 1.25

  CAMLOCK.Z = 0

  CAMLOCK.AX = 0

  CAMLOCK.AY = 0

  CAMLOCK.AZ = 0

  CAMLOCK.QTY = 1

  CAMLOCK.MatID = 1134

  

  If :HNG = 1 Then               ;Hinged Left Door

 

   CAMLOCK.X = :DX - 2.75

   

  End If

  

  If :HNG = 2 Then               ;Hinged Right Door

  

   CAMLOCK.X = 2.75

   

  End If

   

 End If

 

It is not required to define the Size (DX, DY, DZ) of a Material Model, unless you have written your own sizing Parametrics on the Model).
