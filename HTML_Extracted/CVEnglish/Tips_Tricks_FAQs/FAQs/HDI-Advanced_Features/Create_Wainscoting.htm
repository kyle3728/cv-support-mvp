Create Wainscoting

Step #1

In the Assembly Manager:

  1. Setup a new or copy an existing Assembly Construction Method that will produce the Door Frame parts in the widths and joint configuration that you need.

  2. Use a Mortise & Tenon Connection in the Connection | Top Rail Branch so your cut list will indicate correct lengths for stiles and rails. (Allows for coping)

  3. Note any differences in the Toe, as you may want this panel to go to the floor.

  4. Save this Construction Method with a unique name (Wainscot Method).

Step #2

In the Door Manager:

  1. Create a new Door with no stiles and rails.

  2. Set the Panel Type to be a Raised Panel.

  3. Save the Door with a unique name (Wainscot Panel).

Step #3

In the Material Manager:

  1. Create a new Hinge, call it "nohinge"

  2. Set it to have a Vertical and Horizontal Offset, use a value like .375” so the Panel will Mortise into the Frame.

  3. Give it an inset of 3/4".

Step #4

In the Material Schedule Manager:

  1. Go to Hinges, create a new Hinge Schedule, call it Wainscot, and use the "nohinge" Hinge for this Schedule.

Step #5

In the Material Schedule Manager:

If you do not have the shipping "No Pulls/Knobs" Pull Material Schedule do the following:

  1. Go to Pulls, create a new Pull Schedule, call it Wainscot, and use the "MATERIAL NOT ASSIGNED" Pull for this Schedule.

Step #6

Set the Job Properties:

  1. Start a new Job.

  2. Choose anything but the "nopulls" Pull Schedule. (Job Properties/Cabinet/Hardware)

  3. Choose anything but the "nohinge" Hinge Schedule. (Job properties/Cabinet/Hardware)

  4. Choose anything but the "Wainscot" Door Style. (Job Properties/Cabinet/Doors)

  5. Choose anything but the "Wainscot" Construction Method. (Job Properties/Cabinet/Construction)

  6. We will establish an Override later for each of these selections.

Step #7

Create the Assembly:

  1. Place a Base Assembly (you will need to go through all of the following steps again to create an Upper Wainscot panel if necessary) on a Wall and then take it to the Assembly Level, Section View.

  2. At the Section View, choose Modifications | Hide Parts; Hide all Parts except the Door and Frame Parts.

  3. Go to the Properties Tab, select Construction and change it to Wainscot.

  4. Select the Doors Tab and change the door style to Wainscot Panel.

  5. Select the Hardware Tab and change the Pulls and Hinges schedules each to Wainscot.

  6. All of these choices place overrides on the Assembly. This in effect locks the Construction for this object to Wainscot Method, as well as locking overrides on the doors, hinges, and pulls. This is beneficial because when we save this assembly to a Catalog it will always have these options selected whether the Room it is placed in matches them or not.

  7. Section the assembly as necessary and confirm by looking at all other views that this assembly is correct.

Step #8

Saving the Assembly:

  1. From the Room Level Plan view, set the depth of the Object to 3/4" (or whatever depth yours will be).

  2. Right click on the Object and choose Save As.

  3. Choose a Catalog to save the Object in.

  4. Check Preserve Depth.

  5. Name the Subassembly (Wainscot).

  6. Go to Advanced/Preserve.

  7. Make sure that all of these are checked: Pull Material, Hinge Material, Assembly Construction, Door/Applied End Door Style.

  8. Click on OK, and then OK again to save the Assembly.

  Now whenever you pull the Wainscot panel out, resection it, stretch it or otherwise modify it, it will work flawlessly.
