Setup Multiple Drawer Guide Lengths

How do I properly setup multiple Drawer Guide Length selection?

First off Drawer Guide length is based off of the Drawer Boxes Depth not the Cabinets Depth. The first step in the process is to verify that Drawer Box Depths are being calculated correctly. If they are not it does not matter how you setup the Drawer Guide lengths. Once you have verified that the Depths of the Drawer Boxes are correct then focus on the Drawer Guide Lengths.

Make sure that the Parent guide is set to the shortest length you would use. Then set each increment above that down the line.

Parent --- Accuride 12"

>= 14  --- Accuride 14”

>= 16  --- Accuride 16”

>= 18  --- Accuride 18”

>= 20  --- Accuride 20”

>= 22  --- Accuride 22”

>= 24  --- Accuride 24”

The Greater Than or Equal to references the Drawer Box Depth. A Drawer Box Depth, in the above scenario, Greater than or Equal to 14” would utilize the Accuride 14” Drawer Guide.

Keep in mind that it uses the Parent if the other conditions are not met, in this case a Drawer Box depth that is less than 14” would utilize the Parent, in this case Accuride 12”. If I had my Parent setup as Accuride 22” it would place a 22” guide on Drawer Box Depths less than 14”. As you can see setting the Parent wrong can cause problems.

The key here is to verify that the Drawer Box depths are correct first. Then focus on the Guide Schedules. If you follow these basic "guide" lines you shouldn't have any problems.
