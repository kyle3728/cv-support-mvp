Add Applied Door Molding

  1. Create an Applied Door Molding Material in the Material Manager.

  2. Take that Material to Properties then run the Profile Editor and give it the shape you need.

  3. In a Door Material Schedule assign your new Applied Door Molding to the "Applied Inside Edge Molding" Part.

  4. In a Job use that Door Material Schedule on a Door that is using an Inside Edge Profile.

You should see your Applied Door Molding.
