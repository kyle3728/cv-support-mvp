Create a Door Only Job

Part 1: Creating the Door Only Object

  1. Start a new Job using a Frameless Construction Method.

  1. Place a standard Wall Cabinet on a Wall in this Job.

  1. Go to the Section Editor with this Cabinet.

  1. Go to the Section Editor Menu option Modifications | Default Assembly.

  1. Click "Yes" to rebuild the Cabinet back to its Construction Method defaults.

  1. Click on the Face and change it to a Door and change the Hinge to “None.”

  1. Go to the Section Editor option Modifications | Hide Parts. Check the boxes as shown below then click OK:

  1. Go to the Section Editor option Properties | Reveals and change all of the reveals to zero.

  1. Click the Return option in the ribbonbar to return to the Room Level.

  2. With the Cabinet still highlighted change it’s Depth to your Standard Door Thickness; i.e. 3/4”.

  3. Now, go to the Elevation View and right click on the Cabinet that we have just created and click Save | Save As and click Advanced and go to the Preserves Tab and Preserve the Construction then Save the Door Object in the desired location of the object library and give this Door Object a new name (such as Upper Door).

  4. Repeat this process with a Base Cabinet Door and Base Cabinet False Front with one additional step: After changing reveals to zero, go to the Section Editor option Properties | Toe and change the Toe Height to zero.

Part 2: Creating the Door Only Job

  1. Start a new Job and select the Doors for each type of Cabinet that you wish to use (in other words, select Door & Drawer Styles for Base Cabinets and a Door Style for the Wall Cabinets). Also make sure that you set up your Maximum Door size under the Layout tab of Job Properties to be a fairly large number (such as 50 inches or larger).

  2. Go directly to the Reports View and click the Order Entry button on the sidebar.

  3. Once in the Order Entry area, simply select the Catalog where your Doors are saved and drag it on to the page. This will create a line on the sheet with the dimensions of this Door. You may click on the Width and Height and change them to the desired dimensions. You may also change the Quantity of a particular line item.

  4. Continue to drag and drop Doors out of your Catalog until you have all of the Doors that are needed for the Job.

  5. You can now generate Door Lists and CNC output for them.
