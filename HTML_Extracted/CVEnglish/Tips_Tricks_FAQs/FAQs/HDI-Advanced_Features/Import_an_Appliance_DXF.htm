Import an Appliance DXF

Is it possible to import a drawing of an Accessory/Appliance, if the manufacturer can supply me with an AutoCAD DXF file?

If the manufacturer can supply you with a three dimensional DXF of their product that meets the criteria outlined in the Import DXF topic you should have no problem importing the file into CABINET VISION.

  1. You can import the DXF file into CABINET VISION from Plan view by clicking on Utilities | Import DXF.

  2. At the DXF Import screen click on Browse to locate the file, then click on Import to bring it into the viewing window.

  3. You can rotate the item into the position you want to save it into by using the Flip X, Y and Z buttons.

  4. Highlight a Layer in the Part Type Allocation list, then click on Type to assign a Material to that part/layer (IE: Porcelain from Room Materials).

  5. Once you have the item set the way you want it, click on Save to save it into your Catalog.
