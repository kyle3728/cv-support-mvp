Know When to Use a UCS Parameter or Preserves

I need to know a general rule of thumb as to when one should use (or not use) a UCS, Object Tree Parameters, or just save an Assembly to my Catalog using the Preserve tab (for Construction Method)?

Here is a simple rule of thumb. If the modification is for one Part (or one Assembly) then typically, you wouldn't use a UCS.

However, if the modification is going to apply to multiple Assemblies, for example all sink Cabinets, then write a UCS. The theory behind this rule of thumb is that it is faster to modify one item without writing a UCS. However it is typically faster to modify two or more items with a UCS.
