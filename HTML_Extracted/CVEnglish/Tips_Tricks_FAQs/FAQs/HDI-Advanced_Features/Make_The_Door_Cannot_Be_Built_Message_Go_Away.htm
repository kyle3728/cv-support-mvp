Make "The Door Cannot Be Built" Message Go Away

What does the message "Door Cannot Be Built" mean when I use certain Cabinets?

There is a conflict with the Door parts, meaning the Door can not physically be built at the desired size for the Cabinet.

Go to the Door Catalog, select the Door or Drawer Front used when the message occurs. Check the Door Associations to see if you've selected a Door for CABINET VISION to switch to when it reaches a size where that Door can not be built.  Also, go to the Construction tab and check the stiles, rails, and inside edge profiles. Add them up to see if the Door/Drawer can be built at a certain size. Remember, inside edge profile width's are added to the width of stiles and rails unless the Profile is designated as Extended.
