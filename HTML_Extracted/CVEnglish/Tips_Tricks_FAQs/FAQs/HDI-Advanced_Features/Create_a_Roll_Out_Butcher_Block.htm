Create a Roll Out Butcher Block

I want to do Roll-Out butcher blocks with flip down drawers on one of my Cabinets; what is the best way to achieve this?

  1. From the CABINET VISION System Level go to the Material Manager.

  1. Select the Texture option in the ribbonbar.

  2. Go to the Search box and search for Butcher Block.

    1. Make sure the "Decal" option is enabled.

    2. Make sure Tile option is enabled.

    3. Set both the Width and Height to 24" (610mm).

  3. Click Close to exit the Texture Manager.

  4. Select the appropriate Material group (probably Board Stock) and click the New option in the ribbon.

  5. Make its Type, Board Stock and for the Material name and description type Butcher Block.

  1. Click Next and set the Thickness to 1.5" (38.1mm).

  1. Click Next and change the Finish on all surfaces to Automatic.

  1. Set the Texture for Face, Back, Edges, and Ends to Butcher Block.

  1. Click Finish.

  2. Select the appropriate Material group (probably Drawer Guide) and click the New option in the ribbonbar.

  3. Make its Type, Drawer Guide and for the Material name and description type Butcher Block Guide (or whatever Drawer Guide you use for Butcher Block Pull-Outs).

  1. Click Next and change the Guide Type to Side Mount.

  2. For Length enter 22" (550mm).

  3. Enter the required "Side Clearance" for your Drawer Guide.

  4. Enter .75" (19mm)  for the Top Clearance.

  5. Enter 3" (76mm) for the Bottom Clearance.

  1. You can continue through the New Material screens to setup mounting machining.

  2. Click Finish.

  3. Close the Material Manager and go to the Material Schedule Manager.

  4. Go to the Roll Out Material Schedules.

  5. Create a new Roll Out Material Schedule and name it Butcher Block.

    1. For the Roll Out Back, Divider, Front, Laminate, Partition, and Side right click and select Remove Material.

    2. For the Roll Out Bottom select the Butcher Block Material from the Material Group where you created it.

  6. Go to the Drawer Guide Material Schedules and create a new Drawer Guide Schedule and name it B Block Guide.

    1. For Drawer Guide and Roll Out Shelf Guide leave the MATERIAL NOT ASSIGNED selection.

    2. For Roll Out Guide select the Butcher Block Guide you created.

  7. Exit the Material Schedule Manager.

  8. Go to the Door Manager.

  1. Navigate to and select the Drawer Front you use most often.

  2. Click the Copy option in the ribbon.

  3. Name it Flip Down Drawer.

  1. Click Return or the Exit button to leave the Door Manager.

  2. Go to Drawer Box - Roll Out.area of the Assembly Manager.

  1. Create a new Roll Out named Butcher Block.

  1. Double click the General Branch.

    1. Set Standard #3 to a Height of 1.5" (38mm).

    2. Click Finish for the General Branch.

  1. Double click the Bottom Branch.

    1. Set Standard #10 to "Flush to Box Bottom".

    2. Click Finish for the Bottom Branch

  2. Double click the Box Connection Branch.

    1. Set Standard #157 to "To Outside of Roll Out Side" with an Adjustment of 0.

    2. Set Standard #158 to "To Outside of Roll Out Back" with an Adjustment of 0.

    3. Set Standard #159 to "To Outside of Roll Out Front" with an Adjustment of 0.

    4. Click Finish for the Box Connection Branch.

  3. Click OK to exit the Assembly Manager.

  4. Click Yes to Save changes.

Now Let’s Test

  1. Start a New Job.

  2. Draw a Wall and drag a Std Base Cabinet onto it.

  3. Double click on the Cabinet and go to Section Editor in the Section Face view.

  4. Click on the Drawer Front and on the sidebar change it to a False Front.

  5. Right click on the False Front and select Properties.

  6. Click the Change button and select the Flip Down Drawer and it’s options and click OK to leave False Front Properties.

  1. Now click on the False Front again so it is highlighted.

  2. On the sidebar set the Hinge to Bottom.

    1. You may at this point want to Right Click on the Door and go to the Pull and set a Quantity of 1 if you want a Pull.

  3. Click the Section Interior option in the ribbonbar.

  4. Click in the False Front area to highlight it if you have a Drawer Stretcher otherwise click the Cabinet opening to highlight.

  5. Click the Split Opening Horizontal option in the ribbonbar.

  6. Highlight the Adjustable Shelf you just created by clicking it once.

  7. Change the Shelf Type in the sidebar to Roll Out Tray.

  8. Right click on the Roll Out and click Properties.

    1. Change the Roll Out Tray Construction to Butcher Block.

    2. Change the Roll Out Tray Material to Butcher Block.

    3. Click the Change button and select the Butcher Block Guide then click OK.

  1. Now by placing your cursor over the new Butcher Block Roll Out and clicking and dragging, slide the Roll Out up until it is just center of the Drawer opening.

  2. Now right click it again to open the Roll Out Properties area.

  3. Now slide the Closed/Open slider all the way to Open.

  4. Go to the 3D Ortho view and there you have it.

Want to save this Cabinet for future use?

  1. Click on the Return option in the ribbonbar.

  2. Right Click the Cabinet and click Save…Save As  from the pop-up menu.

  3. For name type in Butcher Block Cab.

  4. Check to make sure you are saving this Cabinet into the correct location.

  5. Also select the icon at the right that you want to represent this Cabinet.

  6. Click OK.

This Butcher Block Cabinet will now be available for use in future Jobs.
