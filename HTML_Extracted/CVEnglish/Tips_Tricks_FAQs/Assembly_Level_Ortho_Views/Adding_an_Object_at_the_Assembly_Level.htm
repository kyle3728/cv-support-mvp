Adding an Object at the Assembly Level

While in any of the Orthographic Views, you may add a Part from your Object Catalog in the same manner as Objects are added from the Floor Plan or Elevation Views.

While in one of the Orthographic views (End Ortho view pictured above), click the Objects button in the sidebar.

The Object library will appear in the sidebar. Simply select the objects that you want to add and then drag them out into the right side of the screen. Use the position and placement fields in the sidebar to place the object in the correct location.
