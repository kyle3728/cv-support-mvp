Install Support Folder

The full install extracts the Support folder to the following location:

C:\Users\Public\Public Documents\CABINET VISION\Download\CV_2023_n_nnnn_nn_SetupDvdFiles\Support

CPUID.exe

A utility that outputs CPU information, including supported CPU features. A troubleshooting utility to verify the CPU is supported.

ZPLView.exe

A utility for viewing ZPL (.zpl) files generated by the Real-time Label Image feature of the S2M CENTER.

AccessDatabaseEngine2010

This folder contains the installation files for the Microsoft Access 2010 Engine. If needed the installation routine will install these files.

AccessDatabaseEngine2016

This folder contains the installation files for the Microsoft Access 2016 Engine. If needed the installation routine will install these files.

BlumDynaplan

Contains the BXFExt.ini (Stores Part Name Localization entries) referred to in the following S2M CENTER Help File Topic: Ribbonbar>Import DXF>Other CAD Importers>Blum Dynaplan

DependencyWalker

A utility used for troubleshooting certain installation or run-time issues. Visit the Dependency Walker website for more information. http://dependencywalker.com/help/html/contents.htm

FontRestore

This fixes a problem with missing Fonts caused by uninstalling Version 3.5 and earlier.

FreeImage

FreeImage component source.

Generic Object Libraries

This contains sample Stock Catalogs for both Cabinets (Generic.pkg) and Closets (Generic Closets.pkg). Simply use the Setup Package Utility to import them.

Images

Miscellaneous background images for use in Renderings.

International

Tools used for performing language and feature Localization in CABINET VISION and the S2M CENTER.

IS Setup

InstallShield documentation for troubleshooting Installation error messages and other installation related issues.

MSdotNET3.5SP1

This folder contains the information for installing Microsoft .NET Framework 3.5 Service Pack 1, which is a prerequisite for installing Windows PowerShell 1.0.

MSdotNET4

Information on MSDotNET4.

MSdotNET4.7.2

Installer for MSdotNET4.7.2

MsiCleanUpUtility

Microsoft Windows Installer Clean-Up Utility used for troubleshooting certain installation and/or uninstall issues.

MsiZap

A command line utility that removes all Windows Installer information for a product installed on a computer. The folder contains documentation for the utility along with Product Codes for CABINET VISION, S2M CENTER, and ALPHACAM to remove their Windows Installer information.

MSSQL2008R2SP1_ExpressEdition

This folder contains the installation files for the Microsoft SQL 2008 R2 Service Pack 1 Express Edition. If needed the installation routine will install these files.

MSSQL2012SP3_ExpressEdition

This folder contains the installation files for the Microsoft SQL 2012 Service Pack 3 Express Edition. If needed the installation routine will install these files.

MSSQL2019_ExpressEdition

This folder contains the installation files for the Microsoft SQL 2019 Express Edition. If needed the installation routine will install these files.

Optimizer

Tools used to set low level parameters used by the S2M CENTER Optimizer and Block Nesting engine. See accompanying documentation for further information.

PartListCreator

The S2M CENTER Part List Creator tool as documented in the following S2M CENTER Help file topic: Ribbonbar>Main Tab>Import DXF>Other CAD Importers>S2M Part List> Creating an S2M Center Part List

PSendMail

A command line utility for e-mailing a file as an attachment.

Sample Models

Contains Sample Models used in Model Editor Tutorials within this Help file.

Sample ORDX

A sample of a documented ORDX file.

Sentinel RMS License Manager 9.7.0

Contains the Sentinel RMS License Manager.

TestAdoDbConnection

Files used to test the ability of OLE DB Provider “Microsoft.ACE.OLEDB.12.0” to connect to a Microsoft Access 2007 (*.accdb) database file.

WindowsInstaller4.5

This folder contains the information for installing Windows Installer 4.5, which is a prerequisite for installing Microsoft .NET Framework 3.5 Service Pack 1.

WindowsManagementFrameworkCorePackage

This folder contains the information for installing the Windows Management Framework Core package.
