Creating a Door with Finger Pull

You will need to add your Finger Pull Material in the Material Manager.

Then assign this new Finger Pull Material to the Finger Pull "Pull" in the Pull Material Schedule.

In the Door Manager.

  1. Select a Category to store the Door in.

  2. Click on the New option in the ribbon.

  3. Name and give a Description for the Door.

  4. Select Slab for the Type.

  5. Click Next.

  6. Click Finish.

  7. Double click the new Door to open Door Properties.

  8. Expand the Edge Profiles Category and select Finger Pull for Top.

This will cover the typical Base Cabinet Door you will want to repeat steps 1 through 8 for an Upper Door with the Finger Pull for the Bottom.
