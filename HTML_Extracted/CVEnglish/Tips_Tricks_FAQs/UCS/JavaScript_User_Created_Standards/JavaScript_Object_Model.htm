JavaScript Object Model

Currently, only a single CV object is exposed in the managed API. It is named CVAsmManaged. CVAsmManaged exposes properties and methods. The properties are shorthand methods for getting and setting common parameters, while methods are exposed for more complex operations.

There are a number of predefined numeric constants used for parameter types and object properties. The constants can be used in conditional checking and method calls. For example…

if ( _cab.Evaluate('ConstID') == _this.QueryID(ID_ASSEMBLYCONSTRUCTION, '32mm') )

ModifyParameter(string name, WORD action, Object value)

Modify a parameter style or description.

example:

_this.ModifyParameter('MYPARM', PARMOD_STYLE, PARSTYLE_ATTRIBUTE);

string name

Parameter name

WORD action

 

 

 

Modification type defined by one of the following constants

  1. PARMOD_DESC

  2. PARMOD_TYPE

  3. PARMOD_STYLE

Object value

 

 

 

 

 

If action is equal to PARMOD_DESC then value represents text for the new description.

If action is equal to PARMOD_TYPE then value represents one of the data type enumeration values.

If action is equal to PARMOD_STYLE then value represents on of the following parameter style constants.

  1. PARSTYLE_DEFAULT

  2. PARSTYLE_ATTRIBUTE

  3. PARSTYLE_NOTE

return

void
