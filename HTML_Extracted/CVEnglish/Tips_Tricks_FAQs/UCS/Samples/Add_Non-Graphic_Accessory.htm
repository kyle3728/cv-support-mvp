Add Non-Graphic Accessory

It is possible to add a non-graphic accessory through a UCS.  Below is an example of such a UCS (in red type).  Note that the parameters that are available for the non-graphic accessory are the same ones that may be added to a non-graphic accessory that is saved into a catalog (see picture below).

;Adds a Non-Graphic Accessory to a Cabinet

 

For Each CAB Assembly

 

  Dim Addon as New NGAcsry

  Addon.Visible = False

  Addon.Name = 'Non-graphic Accessory'

  Addon.Name = Addon

  Addon.PRICE = 5

  Addon.Weight = 20

  Addon.DX = 10

  Addon.DY = 15

  Addon.DZ = 6

  Addon.Upchg = 5

  Addon.Cubes = 900

  Addon.Labor = 45
