The BidData.mdf Database

This Database stores the BidData for the current Job. The following is the breakdown of the tables and their fields:

PriceData

This table stores the Price Data.

Field

Description

ID

A unique number used to reference the Price Data.

ParentID

The Parent ID

Type

 

 

 

 

1 = Bid Data

2 = Labor Data

3 = Labor Worksheet Data

4 = Per Room Total for RateTable

5 = Per Room Total for Add-Ons

Level

Level of item as it would appear in a tree

Description

Item description

Value

Price of item for bid data or minutes of item for labor data

Flag

 

 

1 = Raw Price

2 = Markup

3 = Tax

PriceList

This table stores the Price List.

Field

Description

AccessoryID

A cross-reference to the [Accessories].[Access ID]

CabinetID

A cross-reference to the [Cabinets].[Cabinet ID]

BasePrice

Basic cost of item

ListPrice

BasePrice + Upcharges

CostPrice

ListPrice * Discount

AdditionalPrice

User added additional price

SellPrice

CostPrice + AdditionalPrice

TaxedPrice

SellPrice + Tax

RoomNumber

A cross-reference to the [Rooms].[RoomNumber]
