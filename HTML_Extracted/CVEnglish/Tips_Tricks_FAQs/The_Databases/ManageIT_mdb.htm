The ManageIT.accdb File

The ManageIT.accdb file stores all the data for the xCRM mutliplier. The following is the breakdown of the tables and their fields:

AccountLogs

This Table stores the Account Logs.

Field

Description

AccountLogID

A unique number used to reference the Account Log.

AccountID

A cross-reference to the [Accounts].[AccountID] field.

UserName

The User that created the entry.

UserAccountID

A cross-reference to the [Accounts].[AccountID] field.

Date

Date for the Account Log.

Note

Note for the Account Log.

Description
