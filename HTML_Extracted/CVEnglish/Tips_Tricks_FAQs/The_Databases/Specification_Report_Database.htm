The Report.mdf File

The Report.mdf file contains cut list and summary data for a job. This database is unique in that most of its data is temporary. When either the Bid Center or Report Center buttons are selected from within CABINET VISION most Tables within Report.mdf are flushed and re-populated.

 

The following is the breakdown of the tables and their fields:

MaterialOverride

This table will contain either the Waste from the Material Catalog or the Waste recalculated using the area of the parts and the number of sheets used in Optimization. The waste values will change anytime a filter is applied. Since we are adjusting the Waste Factor the material totals per cabinet will be in a ratio to the number of sheets used (not an even number of sheets). You can get the number of sheets for a single cabinet by applying a filter for that cabinet (or any combination).

Field

Description

A cross-reference to the [Material].[ID] field in CVData

An override to the waste percent from [Material].[Waste] in CVData based on the material usage from the generic Optimizer
