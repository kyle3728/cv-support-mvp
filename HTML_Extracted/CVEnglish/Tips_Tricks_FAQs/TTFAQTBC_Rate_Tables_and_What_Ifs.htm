When the Rate Tables  option is clicked, the "Select Rate Tables" window opens. This window will list the available Rate Tables. Before we look at how we created these Rate Tables, note that they are listed, without check marks by them. To incorporate any of the listed Rate Tables into the current Bid, the box must be checked. Because each Rate Table incorporates different pricing, this can even be used as a "What If" tool for "What If" changes to Bids. What if I had Stain instead of Paint?

Creating Rate Tables

Step 1

Begin this process by identifying each area that you will want to include in your Bid. One area might be for Cabinets based on Construction as was demonstrated previously. Then identify the various Bid Methods available within each area that represent different pricing standards for your shop. Each of these areas can cover items that may not always be present. An example is the Bid above has no price for Molding because CABINET VISION did not find any molding in the Job. You may want to add an item that costs more to make therefore, they need to be included in a Bid when present in the Job.

Step 2

After identifying the items to be priced, have a clear idea as to the Rate Table for which they will be priced and the amount that will be changed. Consider any surcharge that may apply as well.

Step 3

Review your plan. Does it cover the work you do, and allow for the different ways it may be done? Time spent planning at this stage will greatly increase the usefulness of the Bid Center.

Step 4

Define your sub Rate Tables. Remember, when defining each sub Rate Table, the name you assign it will appear on the "Rate Table Window" you will select it from, therefore, name it in such a way that you clearly recognize what the Rate Table represents.

From the Bid Center, click the Rate Tables option to get the Select Rate Tables area. Select the "Setup" button to access the "Setup Rate Tables" area. There are two ways to create a new Rate Table. If you are making a new Rate Table, as you would when making the first one for a new group you are creating, use the "Create New Rate Table" button. If you are making a subsequent Rate Table that is essentially like a previous one, consider copying the previous one and renaming it, as well as changing the values of the pricing.

Creating A New Rate Table Line Item

A new Rate Table is created by adding items to it and selecting a Bid Method by which the Price for that item can be computed. If you are creating a new Rate Table, the original Setup Rate Table Window will have no line items on it. To add a Line Item to it, click the Green Arrowhead and enter the name of the item you wish to Bid.

The above example has chosen Sub-tops as the first item. The user has entered $6.50 for the value of the item, and selected a Bid Method from the available Bid Methods which calculates the value based on the total Lineal Feet of Counter Top. These Bid Methods can not be user defined, but the monetary amount can be defined. As soon as you click in the data box for "Method" you will see a portion of the list. A scroll bar is at the right of the drop down for your use in scrolling through the list. It is a good idea to familiarize yourself with the items on the list prior to constructing a Rate Table, as you are required to relate the items pricing to one of these Bid Methods.
