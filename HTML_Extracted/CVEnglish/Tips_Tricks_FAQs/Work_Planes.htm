Work Planes

Work planes, will allow you to easily add operations to faces or edges of a Part without having to think about positions and rotations. The front and back face of a Part can be considered face work-planes and each edge of the Part will be considered an edge work plane. Face work planes are numbered 1 for the front face and 2 for the back face. Edge work planes are enumerated in a counter clock-wise direction from the shape origin on a shaped part or the bottom left corner of a rectangular part. This functionality will not work on radiused edges.

The edge work planes are oriented by following the Part edges from the origin moving in a counter clock-wise direction. The Y axis will be along the front face of the Part toward the next shape node and the X axis will be from the front face toward the back of the Part as shown on the following sketch.

To position an operation on a work plane, create the operation as if it were in a normal Cartesian (XY) coordinate system. The Z position and AX, AY and AZ rotations will typically be 0. The operation can then be applied to an edge by specifying the edge work plane with an _EDGWP parameter. The value will specify which edge to place the work plane on, eg: _EDGWP=3 will position the operation on the 3rd edge. The operation can also be applied to a face with a _FACEWP parameter, eg: _FACEWP=2 will put the operation on the back face.

There are also additional lower level parameters that can be used to get information about an edge. The values returned are for the edges orientation and position in Parts space. One set of parameters assumes the edge orientation as described for the work planes. The parameters are as follows (# is the edge number):

_edg#x

returns the x origin of the edge

 

_edg#y

returns the y origin of the edge

 

_edg#z

returns the z origin of the edge

 

_edg#dy

returns the length of the edge

 

_edg#ax

returns the x rotation of an operation (work plane) placed on the edge

 

_edg#ay

returns the y rotation of an operation (work plane) placed on the edge

 

_edg#az

returns the z rotation of an operation (work plane) placed on the edge

 

_edg#miter

returns the miter angle of the edge

 

_shpedgcnt

Returns the number of edges a shaped Part has. It will return 0 if the part is not shaped. A bend line or radius node count as 1 edge.

 

Arcs and Curves are ignored in the Work Plane counts and can not be referenced.
