Constraint Tutorial

This Tutorial assumes you understand the Constraint tools and how each Constraint type is applied. See the Constraints topic for more information.

Constraints are a method of applying intelligent parametric rules to Parts. If you are shaping a Part which does not need to ever change size it is unnecessary for you to apply Constraints to it. However, if you want to control the Parts Shape you will need to utilize Constraints.

Part 1 - Automatic Constraints

The best way to learn Constraints and how they are applied is to see how the system applies them.

Let's start by editing the shape of a Standard Base Cabinet.

With the Select Tool active let's select the left-most line and take a look at the Constraints tied to it. There are three Constraints applied to this line. At each end we have Coincidence Constraints which locks the ends of the lines to the ends of the respective lines that they touch. There is also a Vertical Constraint which aligns this line Vertically (Aligned with the Y Axis).

We can also see the Side Type assigned to this selected line in the sidebar. The Type can be changed here.

Next let's select the line representing the back of the Cabinet. There are three Constraints applied to this line. At each end we have Coincidence Constraints which locks the ends of the lines to the ends of the respective lines that they touch. There is also an X-Axis Constraint which aligns this line with the X Axis.

We can also see the Side Type assigned to this selected line in the sidebar. The Type can be changed here.

Next let's select the right-most line and take a look at the Constraints tied to it. There are three Constraints applied to this line. At each end we have Coincidence Constraints which locks the ends of the lines to the ends of the respective lines that they touch. There is also a Y-Axis Constraint which aligns this line with the Y Axis.

We can also see the Side Type assigned to this selected line in the sidebar. The Type can be changed here.

Next let's select the line representing the front of the Cabinet. There are three Constraints applied to this line. At each end we have Coincidence Constraints which locks the ends of the lines to the ends of the respective lines that they touch. There is also an X-Axis Constraint which aligns this line with the X Axis.

We can also see the Side Type assigned to this selected line in the sidebar. The Type can be changed here.

Part 2 - Constrained Assembly Shaping

In Part 2 we are going to setup a 90 Degree Corner Cabinet in a way that allows us to control the Leg Depth separate from the Cabinet Size. The first thing we need to do is create a Parameter we can use to control the Leg Depth,

From the System Level go to Utilities | Parameters.

Configure the Parameter as illustrated in the image below.

Now let's start a new Job and place a 90 Base Corner Cabinet.

Right Click on the 90 Corner Cabinet and select Edit Shape.

The first thing we need to do is add a Logical Fix Constraint instructing CABINET VISION from where this Cabinet need to change size.

We want this Cabinet to change size from the corner so we need to add the Fix Constraint at the Cabinets origin.

Now we just need to add Length Constraints to the two legs of this Cabinet using LEGDEP as the Text.

After Saving the Cabinet Shape you can select the Cabinet and change the LEGDEP Parameter in the sidebar to see the result.

Part 3 - Constrained Part Shaping

In Part 3 we are going to add a Rear Notch to a Left Finished End that is driven by the TOEH System Parameter. The first thing we need to do is expose the TOEH Parameter by setting it up as a System Parameter.

From the Splash Screen go to Utilities | Parameters.

Configure the Parameter as illustrated in the image below.

Now let's start a new Job and place a Base Cabinet, making sure you are using a Construction Method that has Notched Toes. We will take the Base Cabinet to an End Ortho View at the Assembly Level.

Right Click on the Left Finished End and select Edit.

Go to the Front View of the Part and click the Edit Shape button in the sidebar.

The first thing we need to do is add a Logical Fix Constraint instructing CABINET VISION from where this Part need to change size.

We want this Part to change size from the bottom left corner so we need to add the Fix Constraint at the Parts origin.

Next, I am going to use the Chained Line Tool to draw a Notch at the Back of the End.

And next I will Trim out the pieces I do not want using the Trim Delete tool ending up with a rear Notch.

Now we need to add our Dimensional Constraints to this Part.

First let's setup the overall Length of this Part using the Ortho Distance Constraint. Once you activate the Ortho Distance option click the two points indicated in yellow in the image below.

Once you have placed the Constraint you will be presented with the Dimension window. Activate the Text option and type DY as the value. Click Test to verify the result is as you expect.

You should now see your Constraint on the Part.

Next we need to do the same thing for the Part Width using DX as the Text value.

Now that we have the overall Part size Constrained we need to set the Height of our two Toe Notches. For this we will use the Dimension Length Constraint and simply pick the Vertical Line representing the Toe Height and use TOEH as the Text.

This Part is now fully Constrained.

Let's return to the Room Level. From the Room Level I will change the Toe Kick Height to 8" in the sidebar.

And from 3D View you can see our Left End has changed the Toe Kick Height appropriately.
