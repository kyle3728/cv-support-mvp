The Shape Editor (In General)

The Shape Editor consists of several tools designed to allow editing of the shapes of Assemblies, Parts, and Operations. All tools within each Shape Editor work the same, yet not all Shape Editors contain all the tools shown. The Shape Editor will not appear until you are in the "shape edit mode" for the object you are working on. Typically, a right click on the item produces a menu with the "Edit Shape" option. Selecting this option will cause the Shape Editor to display.

The tools defined below are from different Shape Editors, and may not appear in all Shape Editors.

Layer Selector

There are two options in the Edit Shape Layer selector:

Shape - The default Layer and the one all Shapes must be drawn on.

Construction - This Layer allows you to draw Construction/Reference Lines to assist in creating your shape.

Depending on which Layer is active some options in the Modify section of the ribbonbar may or may not be available.

Select

This will change the cursor to an arrowhead and allow selection of lines, arcs, etc. After selecting the item, you will be able to change properties by selecting the desired property off the list in the side bar and selecting the item to apply it to. Lines can be moved, re-sized, etc. Text can be placed, edited, as well as changing the font and font size.

When selecting an entity there are some Simple Constraint options in the sidebar for the entity. See the Constraints topic for more information.

Select A Split - Part

When selecting a Part Split the sidebar will allow modification of two Properties for it depending on whether it is a Horizontal or Vertical Split:

    • Y - Allows you to modify the Y Position of a Horizontal Split.

    •  X - Allows you to modify the X Position of a Vertical Split.

 

Select A Split - Counter Top

When selecting a Counter Top Split the sidebar will allow modification of one Property for it:

    • X - Allows you to modify the X Position of the Counter Top Split.

Select All

The Select All menu option will select all objects in the Shape Editor.

Measure

This option allows you to Measure Objects without leaving a dimension on the drawing.

Delete

This option will allow you to Delete the selected CAD Object(s).

Delete Split

This Tool allows you to delete a Top Split or Part Split you have added.

Delete All

This Tool allows you to delete all Splits on a Part.

Transform

The Transform tools will only be available (not disabled) when selecting objects which are not Constrained.

Flip

This option will Flip the selected object. First select all of the CAD Objects you would like to Flip using the CAD Selection Options.

Then click the Transform | Flip option in the ribbonbar.

Mirror

This option will Mirror the selected object. First select all of the CAD Objects you would like to Mirror using the CAD Selection Options.

Then click the Transform | Mirror option in the ribbonbar.

Rotate...

This option will Rotate the selected object. You will see the following dialog box when you select this option to choose how you want to rotate the object.

Crop

There are four Crop options in the sidebar once the Crop Tool is selected:

Start X - The X Start point for the Crop.

Start Y - The Y Start point for the Crop.

Crop Type - You have three Crop Types available; Rectangle, Circle, and Polygon that determine the shape of the area being cropped.

Crop Area - The Inside option will get rid of what is Outside the Boundary and keep what is Inside. The Outside option specifies that you want to keep what is outside the Crop Boundary and get rid of the objects inside the Boundary.

The Inside option can be a bit aggressive as it will remove all Objects in the current view if it is not inside the boundary. To keep this option from being so aggressive you can limit the Objects that are affected by a Crop by selecting the Objects first.

Remember you can select multiple Objects by holding the  key on the keyboard down as you click the Objects.

Bend

This option allows you to select a Line and turn it into an Arc by dragging it to Bend it.

Scale

This option is made available in the Molding Manager and Profile Manager when Importing a Bitmap. This tool allows you to scale the Bitmap to fit the overall Thickness/Height of your Profile.

Align Image

This option is made available in the Molding Manager and Profile Manager when Importing a Bitmap. This tool allows you to align the Bitmap to the Origin of your Profile.

 Horizontal Split

This option adds a Horizontal Split to your Part. This is very handy, because CABINET VISION will not determine if the Part's length is too large to manufacture in a single piece, this will allow you to instruct CABINET VISION where to split the Part for manufacture.

Vertical Split

This option adds a Vertical Split to your Part. This is very handy, because CABINET VISION will not determine if the Part's width is too large to manufacture in a single piece, this will allow you to instruct CABINET VISION where to split the Part for manufacture.

Line

This option allows the creation of Lines to define the shape of your Assembly or Part.  After selecting and placing a line, the line properties window will display in the sidebar. Clicking on the items listed in this window will allow modification of that item.

Chained Line

This tool allows the continuous creation of connected Lines. Similar to the Polyline option in CAD.

Line Box

This tool allows the creation of a Rectangle. Similar to the Rectangle option in CAD.

Polygon

This tool allows the creation of a fully enclosed Polygon.

Arcs

Arc By Radius

This option allows definition of a point where a radius begins, the radius, and the degrees of radius.  The first mouse click marks the center of the radius, pull the radius that you want and click again.

A circle will appear with a radius line within it. Move the cursor and the circle opening becomes an arc as you do so.

When you reach the correct position, a third click will set the arc at the selected point.

 

3 Points Arc

This option lets you place two points to define the opening of the arc, and the third point pulls the arc increasing the radius.  A 3 Points Arc is very handy when you have three known points to Snap to.

In the example below I want to create the Arc shown in red.

To begin I select the Arcs | 3 Points Arc option from the ribbonbar then Left Click the End Point (With Snap To Points on) of the first line.

Next I Left Click the Mid Point of the second line.

And the last and third point I Left Click the End point of the third line.

And now I have the Arc.
