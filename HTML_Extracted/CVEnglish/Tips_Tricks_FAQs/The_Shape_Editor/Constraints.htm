Constraints

 

When editing the shape of an Assembly, Part or Route Operation, you have the option to apply 2D constraints to the Lines and Arcs making up the shape that must meet critical design specifications or exhibit different kinds of parametric behavior. Solid maintains these constraints when you edit these objects. You can apply the following types of constraints:

Simple Constraints - These constraints are applied in the sidebar when selecting either a Point or a CAD entity that defines a shape.

The other two types, if available, are in the ribbon when the Shape editor is active.

Logical Constraints - Let you maintain logical geometric relationships between lines and arcs. You can apply these relationships between lines and arcs that belong to the same shape.

Dimension Constraints - Let you apply fixed dimension distances or angles between lines and arcs within the shape to meet critical design specifications. You can control these dimensions via Object Tree Parameters as well as UCS's.

Dimension Constraints are not intended for detailing and dimensioning drawing layouts. See CAD Dimensioning for more information on adding detail dimensions.

Basic constraints are applied automatically by Solid as you shape your object.

Right clicking on the screen while in Edit Shape gives you a few options:

 

  •  Paste

  •  Select All

  •  Reverse Selection

  •  Auto-Constraints - Will apply all the Auto-Constraints you have setup.

  •  Remove Constraints - Will remove all Constraints from the Object.

  •  Properties

You can add your own Constraints by using the following tools:

Simple Constraints

Simple Constraints are available when selecting a Point or CAD entity.

Simply click on the little button associated to a selected entity to sequentially change its Simple Constraints:

- The position will be kept at the same distance from the left edge of the Object.

- The position will be kept at the same distance from the right edge of the Object.

- The position will be kept at the same distance from the center (Horizontally) of the Object.

- The position will be kept at the same distance from the top edge of the Object.

- The position will be kept at the same distance from the bottom edge of the Object.

- The position will be kept at the same distance from the center (Vertically) of the Object.

NONE - The position will adjust proportionally to the Object Size.

 

There are two types of Constraints available in the ribbonbar Constraints Group.

Dimension Constraints

There are five Dimension Constraints:

Distance

The Distance option allows you to set a Distance between two CAD elements.

After you select the Distance option, you are prompted to select (in the statusbar) the first CAD Object.

The status bar now instructs you to select the CAD Object to Constrain the first selected CAD Object to...

The status bar now instructs you to select the Constraint position...

Once you have selected the Constraint position you will be presented with the following dialog box. There are two options here:

Value - Allows you to set the fixed value (Distance) of the selected elements.

Text - Allows you to enter a Formula (such as DX/2) or a Parameter that establishes the value for the Distance of the Constraint.

Now you can see the Distance Constraint has been applied.

Logical Constraints

There are ten Logical Constraints:

Fix

The Fix option allows you to fix a Line, Arc, or point to a specific location.

After you select the Fix option, you are prompted to select (in the statusbar) the Line or Arc that you would like to fix to it's current location.

Now you can see the Fix Constraint has been applied.

If a Fix Constraint is applied to an entity, you are just fixing that the entity should be somewhere along the trajectory of the entity. In the diagram below the black lines are the fixed entities and the magenta represent the possible locations of the entity. There is no guarantee of length or position along the magenta line.

A line with its center point fixed, basically spins about the center and can be on any angle of rotation. There is no guarantee of length or angle.

A line with its end fixed, will spin about the fixed end point. There is no guarantee of length or direction.

Coincidence

The Coincidence option allows you to Constrain a Line or Arcs endpoint to another Line or Arcs endpoint so that it is coincident (similar to) to the other. To maintain a closed (valid) Shape each end-point should be Constrained Coincident to the other.

After you select the Coincidence option you are prompted (in the statusbar) to select the first CAD Object.

The status bar now instructs you to select the next CAD Object...The second CAD object selected is always the one that moves (Shrinks/Stretches) to meet the first CAD object selected.

Now you can see the Coincidence Constraint has been applied. Notice the Lines endpoint, which was not Coincident to the other Line's endpoint, is now Coincident to the Line's endpoint. And because we applied the Coincidence Constraint these Line's endpoints will remain Coincident when the object changes its size.

To make it simpler to create and maintain your shapes, Coincidence Constraints are automatically applied to the endpoints of Lines and Arcs endpoints where they meet. That way, connected Lines and Arcs stay connected when you edit the shape of the object. It will seldom be necessary for you to apply them manually.
