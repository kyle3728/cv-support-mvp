Understanding Intelli-Joints

Introduction

The use of Intelli-Joints creates a greater flexibility for construction methods designed for CNC. However, with flexibility comes complexity. This is an advanced concept that will extend your ability to use Solid to its fullest extent.

The Concept

  • What is an Intelli-Joint?  An Intelli-Joint gives a set of rules and functions to separate Parts that come together. The best way to define an Intelli-Joint is to give an example like this. When the two Parts come together in the specified manner (the Parts must either touch or be at an exact specified distance from or into each other in order to trigger the Intelli-Joint to be applied), you may have the Intelli-Joint place operations on either of the two Parts or both of them.

Primary and Secondary

  • A union between two Parts creates a secondary and a primary. The primary dictates to the secondary the required Operations. The primary is the key Part that sets into or demands a place on the secondary. The secondary operations only exist because of the primary.

  • When two Parts intersect; the primary is typically the Part that "butts into" the other Part. The primary's edge touches the secondary’s face.

  • For example a left End (secondary) and an Adjustable Shelf (primary) have joined together. The left End is the secondary because it contains the boring holes for the Shelf to rest upon. Without the Shelf the left End would not need holes, therefore it is dependent on the Shelf for its characteristic, making it the secondary.

2-D Views for Intelli-Joints

  • When you place a "New Part" in an Assembly it looks fine in one view but in another view it may not be positioned correctly. This is a crucial point to understanding Intelli-Joints. Don't take for granted that an Intelli-Joint looks like it is in the correct place in one view that it will be correct in all views. It is best to view the Intelli-Joint in 3-D or in 2 separate 2-D views to verify its proper location. This will be discussed further later.

X, Y, Z of Intelli-Joints

  • Remember "Parts are Parts". Nothing has changed for Intelli-Joints. We will be describing Operations on the Intelli-Joint. The X, Y, Z positions of the Operations will be in reference to the Intelli-Joint.

  • Picture a piece of two-sided tape. It has a rectangular building orientation. On that rectangular shape are objects.

Example Intelli-Joint

  • Looking at this KD (Knock-Down) Fastener notice that it has 2 separate holes. This piece of hardware serves as a perfect example of the need for an Intelli-Joint for Solid.

  • The first suggestion for creating an Intelli-Joint is to draw it on a piece of paper. Draw all aspects of it. The hardware, the Part that it will be going into and where in the finished product it will be. Having the views drawn will allow you to visualize the Intelli-Joint much easier.

Design

    1. Access Intelli-Joints from the System Level and in the Part level. From the System Level select Intelli-Joint on the Main ribbonbar.

    2. Select the New option in the ribbonbar and name the Intelli-Joint KD Fitting.

    1. Click the New Operation button in the toolbar.

    2. Name the operation Front 5mm Hole.

    3. Set it to Primary.

    4. Set it's Type to Bore.

    5. Set it's Face to Left.

    6. Set it's X Position to 1.5.

    7. Set it's Y Position to 1.25

    8. Set it's Depth to 1/4".

    9. Set it's Width/Diam to 5mm.

Since you cannot enter pure Metric values in the Intelli-Joints screen you must qualify a metric value with mm so it is properly converted.

    1. Set it's Rept. to 1.

  • Choose Copy Operation then make the following changes for the 2nd  hole:

    1. Change it's name to Front 15mm Hole.

    2. Change it's X Position to .125.

    3. Change it's Depth to .5.

    4. Change its Width/Diam to 15mm.

  • Notice that the Operation is highlighted to help identify what you are working on and whether it is a Primary or Secondary Operation.

    • Primary = Yellow

    • Secondary = Blue

    • Selected Item = Red

  • Select the New Operation button.

    1. Name the operation Front Peg.

    2. Set to Secondary.

    3. Set it's Type to Bore.

    4. Set it's Face to Normal.

    5. Set it's X Position to .325.

    6. Set it's Y Position to 1.25.

    7. Set it's Depth to .5.

    8. Set it's Width/Diam to 5mm.

    9. Set it's Rept. to 1.

  • Copy all three operations and make the following changes:

    1. Change Front in Operation Name to Rear.

    2. Change Y Position from 1.25 to _MP:DX-1.25.  This equals Primary Part Width - 1 1/4".

  • Now you have completely described the Intelli-Joint.

  • Go in and change the sample joints' width and height to see the changes.  Be sure that as you modify the Intelli-Joint size that the operations are moving correctly.

  • Before leaving the design section jot down what the ID of the Intelli-Joint is.  This will be needed later when you create the User Created Standard to place the Intelli-Joint.

Thickness of an Intelli-Joint

  • A piece of tape has a thickness of "0" this would be an example of a butt joint.

  • The Intelli-Joint has to span the gap between the two Parts exactly or the Intelli-Joint does nothing.  If the gap = 1/8 inch then the Intelli-Joint thickness must = 1/8 inch.

  • When designing an Intelli-Joint for a dado the Intelli-Joint would have a negative thickness equal to the dado depth.

Manually Placing the Intelli-Joint on a Part

  1. Go into a new Job.

  2. Choose a Frameless Construction Method with NO Construction Boring applied, there doesn't need to be any extra information clouding the Intelli-Joint. You will be able to see the result more clearly.

  3. Take a Base Cabinet with a Fixed Shelf to the Assembly Level and go to an orthographic view (Smiley Face).

  4. Right Click on the Fixed Shelf and click Edit.

  1. Click on Reports View .

  2. In the sidebar of the Part Editor click Add New Operation.

  1. Choose Bottom as the Part Face to apply your New Operation to, and click Next.

  1. Choose Intelli-Joint as the Operation type, and click Next.

  1. Select KD Fitting as the Joint Type and Current Assembly for the Scope.  This is the most typical selection although an Intelli-Joint can span two Assemblies as in Closet applications, in that case you would select Entire Room.  Click Next.

  1. Set the Width to Part Thickness + 0.

  2. Set the Length to a Value of 20.

  3. Set the Thickness to a Value of 0.

  1. Set the Distance from Top to a Value of 0.

  2. Set the Distance from Left to a Value 0 and click Finish.

  • The green line in the picture below represents the Intelli-Joint. Look at the Intelli-Joint in different views to make sure it is positioned correctly. If any adjustments are needed, type in the new values at the CNC view.

  • We will leave the Intelli-Joint a little shorter than it needs to be so that we can see it better on the shelf for the moment.

  • Go back to the Cabinet in 3-D view. Adjust the Cabinet and notice that the Intelli-Joint stays right with it. Add in another Shelf and you will notice that it doesn't stay with it.

  • We have covered creating an Intelli-Joint and Manually positioning it.  For automatic positioning you must use a User Created Standard.

Creating a UCS to Automatically Place an Intelli-Joint

  1. Go to the End view.

  2. Go to Utilities | User Created Standards.

  1. Select the New option in the ribbon.

  1. Name the UCS KD Fitting.

  2. Select Add.

  3. Select Intelli-Joint as the Type then select the "KD Fitting" as the Intelli-Joint to add and click Next.

  1. Select Link To Each Part.

  2. Select Fixed Shelf as the Part to link to and click Next.

  1. Click Next for Assembly Class.

  2. Click Next for Assembly Properties.

  3. Click Next for End Type.

  4. Click Next for Construction Method.

  5. Click Next for Conditions for Fixed Shelf.

  6. Select Width Side 1 (Bottom) and Width Side 2 (Top) as the Sides to attach the Intelli-Joint to then click Finish.

  • Exit the UCS and return to the Cabinet. Notice that as additional Fixed Shelves are added the KD Fittings appear automatically.

  • Modify the UCS to include Shelves, Top, Deck & Bottom.

    1. Go back into the User Created Standards screen.

    2. Make sure that KD Fitting is highlighted then click the Code option in the ribbon.

    1. Modify the For Each statement as shown in bold below:

For Each FS|TO|BO|DE|TS|SH Part

    1. Click Close.

    2. Click Yes to save the UCS.

    1. Exit the UCS and return to the Cabinet. Notice that the Intelli-Joints appear on your other Parts now.

  •  Other uses for Intelli-Joints include, blind dado construction, hardware operations (lazy susan, door locks).
