Statusbar Menu

The statusbar menu gives you quick access to control the status of certain Drawing aids. There are six possible options here:

Unit Selection or

Allows you to toggle between Inches (Imperial) and Metric Units on the screen. The Unit displayed on the button represents the Unit of Measure that is currently active. You can turn this button On/Off via the Preferences | Units tab.

Unit Precision

Allows you to select the Unit of Precision for the selected Unit. The Precision of the Unit selected will be dictated by the Unit of precision last Set As System Default for that Unit and View in Units Properties.

Reset Ruler

This option allows you to quickly remove any Snap Marks you have placed on your Drawing.

Snap to Marks

This feature allows snapping to a Snap Point set by use of the Ruler Tool. If this button is highlighted in blue it means it is active simply click it again to disable it.

Snap Grid

This option turns the Grid display on. You will see a grid pattern in the window where you layout Jobs, edit Assemblies, or CAD that you can Snap to. If this button is highlighted in blue it means it is active simply click it again to disable it. Go to Right Click Properties to set the Grid Spacing.

Snap to Points

This is perhaps the most useful tool when using the measure option of the Ruler Tool. Using this snap feature allows precise snapping of dimension lines in CAD, and other precise placement options. This option turns on the Dynamic Snapping functionality in Solid The snap to points appear as small dark dots in the corner of the objects and all parts within the object. These are the snap to points. If these points are present and not desired, reset the snapping mode to OFF or another mode to eliminate them. If this button is highlighted in blue it means it is active simply click it again to disable it.

Snap Tips

    • Use Ruler Points to place complex Peninsula Wall layouts

    • Use Ruler Points for placing Assemblies, Parts, etc.

    • Use Ruler Points to create Architectural Symbols for saving into your Symbol Library

    • Use Ruler Points to draw construction details.

    • Use Ruler Points to edit complex Assembly shapes.

    • All of the CAD Functions in CAD can Snap to Points, Grid or Ruler Points.

    • Use Grid Points to place items on the Grid.

    • Use Points to measure distance on Assemblies, Parts, Walls, etc.

    • Use the Points on an object to place Ruler Points

Build Warnings

This option will display any Build Warnings that are currently active. Simply click this option to bring up the Message Log to view the active Build Warnings.
