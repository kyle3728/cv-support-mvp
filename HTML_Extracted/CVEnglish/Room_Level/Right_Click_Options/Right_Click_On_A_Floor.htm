Right Click On A Floor

Following are the possible options when Right Clicking on a Floor in the Floor Editor.

 

 Delete

This option will Delete the selected Floor.

 Delete All

This option will Delete all Floors in the Room.

 Edit Shape

This menu option allows you to edit the shape of your Floor by bringing up the Shape Editor options.

Click the options below for more information

 Properties

There are three Tabs in the Floor Properties area.

Floor Tab

Following are the possible options in the Floor Tab:

  • Description - The Description of the Floor.

  • Texture - Click the Texture button to select a specific Texture for use with the selected Floor. If you leave the Texture "Undefined" then Solid will apply the Global Floor Texture specified for the Room.

  • Elevation - Allows you to establish the Elevation of the Floor.

  • Thickness - Allows you to specify the Thickness of the Floor.

Comment Tab

This Tab will allow you to add Comments to your Floor.

Parameters Tab

This Tab allows you to View and/or modify any Parameters on the Floor.
