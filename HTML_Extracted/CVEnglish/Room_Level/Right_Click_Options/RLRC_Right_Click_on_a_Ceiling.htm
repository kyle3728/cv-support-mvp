Right Click on a Ceiling

Following are the possible options when Right Clicking on a Ceiling in the Ceiling Editor.

 

 Delete

This option will Delete the selected Ceiling.

 Delete All

This option will Delete all Ceilings in the Room.

 Edit Shape

This menu option allows you to edit the shape of your Ceiling by bringing up the Shape Editor options.

Click the options below for more information

Properties

There are three Tabs in the Ceiling Properties area.

Ceiling Tab

Following are the possible options in the Ceiling Tab:

  • Description - The Description of the Ceiling.

  • Texture - Click the Texture button to select a specific Texture for use with the selected Ceiling. If you leave the Texture "Undefined" then Solid will apply the Global Ceiling Texture specified for the Room.

  • Elevation - Allows you to establish the Elevation of the Ceiling.

Comment Tab

This Tab will allow you to add Comments to your Ceiling.

Parameters Tab

This Tab allows you to View and/or modify any Parameters on the Ceiling.
