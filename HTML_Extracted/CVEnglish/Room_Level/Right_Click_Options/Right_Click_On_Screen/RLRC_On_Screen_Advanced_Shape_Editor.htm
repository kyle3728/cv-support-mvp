Right Click on Screen Shape Editor

There are five options when right clicking on the screen in the Shape Editor:

 

 Paste

 Select All

 Reverse Selection - This option will Reverse your current Selection.

 Auto-Constraints - Use this option to Enable/Disable the addition of Auto Constraints as setup in Preferences.

 Add Constraints - If you have used Remove Constraints, or made changes to your Auto Constraints you can use this option to quickly apply all your current Auto-Constraints to the current Shape.

 Remove Constraints - This option will bring up a windows showing all Constraints Types. Simply select the Constraint Types you want removed and click OK to remove them from the current Shape.
