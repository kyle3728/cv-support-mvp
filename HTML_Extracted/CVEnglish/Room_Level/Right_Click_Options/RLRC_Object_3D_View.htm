Right Click on an Object 3D View -

The following options are available when right clicking on an Object in 3D View:

 

Select... / Select Type...

The Select... option allows you to select from multiple Objects which may fall under your selection cursor such as.

The Select Cabinets... option allows you to select All Cabinets in the Room, All Cabinets on the Wall, or All Cabinets of the Type that you Right Click on.

 

The Select Appliance, Select Ceiling, Select Countertop, Select Molding, Etc... options allow you to select all of that Object Type in the Room or on the Wall where the selected object resides.

The Select Ceiling, Select Floor, and Select Wall options allow you to select all Objects of the selected Type within the Room.
