Right Click On Wall Plan View -

When Right Clicking on a Wall in Plan View you will have the following options:

 

 Cut

Will Cut the selected Wall(s) to the Windows Clipboard.

 Copy

Will Copy the selected Wall(s) to the Windows Clipboard.

 Paste

Will Paste the contents of the Windows Clipboard. Since you are in Wall mode the only valid content which will Paste would be a Wall.

 Delete

Deletes the selected Wall.

 Transform

A fly-out that allows you to Rotate the selected Wall(s).

 Select All

Selects All Walls in the current Room.

 Select None

Deselects any currently selected Walls in the current Room.

 Reverse Selection

Reverses the selected Walls. For example if you have 3 Walls and have Wall #1 and Wall #2 selected the Reverse Selection option will de-select Wall #1 and Wall #2 and select Wall #3.

 Hide

Hides the selected Wall and all Objects attached to it in 3D view.

 Show

Shows the selected Wall and all Objects attached to it in 3D view.

 Show/Hide Walls

If you would like to Show or Hide Walls for more clear 3D Views you can do so by selecting this option. Simply enable or disable the Walls in the list to Show/Hide them.

 Connections

Allows you Connect/Disconnect the selected wall to adjoining Walls.

 

 Show Dimensions

Will turn Dimensions on for the selected Wall if they have been previously Hidden.

 Hide Dimensions

Hides all Dimensions associated with the Wall and Objects on the Wall.

 Properties

Clicking on the option will open the View Properties screen. Following are the tabs in the Wall Properties Screen:

General Tab

This Tab allows you to Modify the Name and/or Description of the selected Wall.

 

Comment Tab

This Tab will allow you to add Comments to the selected Object. This can be displayed in selected Reports you generate.

Finishes Tab

This Tab allows you to modify the Finish or Texture on the Front, Back, and/or Ends of the selected Wall.

Overrides Tab

This Tab allows you to remove any Overrides on the Wall.

Parameters Tab

This Tab allows you to View and/or modify any Wall Parameters.
