Right Click on Splash

Following are the possible options when Right Clicking on a Splash:

 

 Delete

This menu option will delete the selected Splash.

 Delete All

This menu option will delete all the Splashes in the current Room.

 Edit

Takes the Splash to the Assembly Level.

 Edit Shape

This menu option allows you to edit the shape of your Splash by bringing up the Shape Editor options.

Click the options below for more information

 Properties

The Properties menu option opens the Splash Properties area. There are four tabs in the Top Properties area:

Splash Tab

There are four options on the Splash tab:

    • Description - The Description/Name of the selected Splash.

    • Material - Allows you to select a Material Schedule for the selected Splash.

    • Texture - Allows you to select a Texture to apply to the Splash.

    • Thickness - Allows you to set the Thickness of the Splash.

Comment Tab

The Comment tab allows you to add a Comment/Note on the selected Splash.

Overrides Tab

This Tab allows you to View and/or Remove any Overrides that may be present on the selected Splash.

Parameters Tab

This Tab allows you to View and/or modify any Parameters on the selected Splash.
