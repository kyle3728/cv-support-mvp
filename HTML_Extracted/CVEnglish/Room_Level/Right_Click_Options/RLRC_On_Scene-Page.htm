Below are the possible options here:

 Delete

This option is available only if you right-click on a Scene. This allows you to remove (delete) that Scene from the current Drawing Page.

 Convert to Static

This option will, if you are right clicking a Live Drawing Scene, allow you to convert it to a Static Scene.

Once a Live Drawing Scene is converted to a Static Drawing Scene it can not be converted back to Live

 Render

If Render is checked, the Live Drawing Scene will display and print in the view's Render Mode when it was taken (i.e. if it was taken in textured mode, it will display and print in textured mode). If the view was in line drawing it will display and print in a non-concealed line drawing. If you are in Filled or Texture Render Mode when you create the Live Drawing Scene, Render is automatically checked. Otherwise it is not and Preview is checked.

 Preview

Preview mode allows quick display of your Live Drawing Scenes. Live Drawing Scenes have two options "Preview" and "Render". Because of the nature of Drawing Scenes and how they are presented a Render Scene can be very slow to process.

 Save Symbol

Allows you to save the CAD on the current page as a Symbol.

To Save a Symbol:

    1. There must be some CAD on the current Sheet or you must be selecting a Statice Scene.

    2. Right-Click on the Sheet and click Save Symbol.

    3. The Save Symbol Window will appear.

    1. Choose the Category where you want the new Symbol to appear and then give the Symbol a name.

    2. If you want to create a new Category, click the New Category button and the following interface will appear:

    1. Name the new Category and click OK to continue.

    2. When the new Symbol has been named and has been placed in the correct Category, click the OK button to complete the addition of the new Symbol.

 Create Scene

By selecting this option, you can create a blank scene on the current Drawing Page.

 Copy Scene

Create a copy of the currently selected Scene.

 Copy Sheet

Will copy the current sheet and all content to a new sheet.

 Clean Sheet

Will clean everything off of the current Sheet..

 Title Block

Allows you to select a Title Block from the Drawings Library and places it on the Page.

Title blocks may be edited or created, within the Drawings Library, so that your drawings page will be bordered neatly and include important information about the Job and your Company. There are several Title Block Templates that come with your installation of Solid. These Title Block Templates may need to be customized with your information before using them. You may also create your own Title Blocks using the Drawing Library.

By default each new Drawing Sheet you add will automatically add the Title Block you have Set As System Defaults. To clear this out so your new Drawing Sheets are blank simply select the None option here and click OK.

If you change the Title Block, the drawing page will be updated with the new Title Block. This has been made possible by putting the Title Block on its own Layer. Any CAD on the Title Block layer will be removed and replaced by the newly selected Title Block.

 Scale

Displays the Scale for the current Sheet or Scene.  Allows the current sheet to be scaled to fit the paper.

 Properties - Drawing Sheet

Displays the View Properties for the Drawings Sheet.

There are four tabs in the Drawing Page Property area:

Draw Tool Tab

There are four options here:

  • Grid Spacing - Controls the size of the grid squares on the screen when using the Grid Snap option from the Statusbar Menu.

  • Grid Always Visible - Normally the Grid is only displayed when Grid Snapping is on. Simply enable this option to have the Grid Always display.

  • Drag Scene From Snap Point - This option allows you to turn on ability to drag scenes with snap points. This turns on the Snap Points option allowing you to choose what points you would like to snap to. There are three options:

None - Turns off Snap Points for Scenes but still allows points within the Scene to Snap to the Grid.

Paper Points - Will Snap points within the Scene to points on the Grid or on the Paper.

All Scene Points - Will Snap points within the Scene to points on the Grid or from other Scenes on the Paper.

  • Reset Ruler - This removes any snap marks (red Xs) on the view.

Paper Tab

Select your Paper Type from the drop down selector or click the action button to create your own custom Paper size. You can also set the Orientation of your Drawing Page here.

Paper Sizes

Use this area to create your own custom Paper size. You can also Copy, Rename, or Delete Paper sizes here.

Scale Tab

Use this are to select the Scale on your Drawing Page. You can use pre-defined Imperial or Metric Scales, or your own User Defined Scale.

 Properties - Static Drawing Scene

Displays the View Properties for the Static Drawings Scene.

There are two tabs in the Static Drawing Scene Property area:

General

There are three options here:

  • Name - The name of the Drawing Scene

  • Template - Shows the Scene Template used in the Scene and allows you to change it.

  • Hide Scene - Allows you to Hide/Unhide the selected Scene.

Scale

Allows you to set the Scale for the selected Scene.
