 Save

The reference to an Object in Solid includes Cabinets, and other items that you save to your library. Appliances, Cabinets, Closet Objects, Furniture, etc., are all objects. After creating a special Assembly or other object decide whether this will be a desirable item to have available in future Jobs. If this is a one time only creation, it is not necessary to save it as it will be saved when the Job is saved. However, it will only be available in the Job it is saved with. If it is possible that the item will be required for other Jobs it must be saved back into your library.

Save versus Save As

Right clicking on the Assembly at either the Plan or the Elevation view at the Layout or Assembly Level, will provide a menu option similar to the one shown above. Note that there are two options under the "save" feature. It is important to understand the difference between the two options. In the simplest terms, Save keeps the new item in its new form at the expense of loosing the prior item (the pre-edit form). Save As allows for the creation of a new item. In effect, Save As allows for two items when finished, whereas, Save allows for only one. Save overwrites the original item, whereas Save As creates a new item in addition to the original one. Use the Save feature when you want a permanent change to an existing item or Assembly, and the Save As feature when you want the new item in addition to the existing one. Save As will allow you to rename the new item and place it in other libraries you may have made.

Features Common to Save and Save As

The first option allows choosing of a Library by clicking on the Library Symbols in the left window as shown on the above screen capture. Thereafter, if desired, the icon can be changed by typing the selected name in the "Name" box at the top. The Description can be changed to represent your Assembly. An Icon should be chosen from the list at the right that represents the Assembly you are saving. This is done by clicking on the one of your choice.

Preserve Finish

This feature will save the item you are saving to your Library with any Finish overrides you have applied to the Object or its Parts.

Preserve Height

This feature will save the item you are saving to your library at the Height you have edited it to, instead of the default Assembly Height. If you have created an object that is of a different Height than a default Assembly, it is advisable to select this option. You will still be able to stretch or re-size it's Height after you bring it into a new Job regardless of whether you select this option or not.

Preserve Depth

This feature will save the item you are saving to your library at the Depth you have edited it to, instead of the default Assembly Depth. If you have created an object that is of a different Depth than a default Assembly, it is advisable to select this option. You will still be able to stretch or re-size it's Depth after you bring it into a new Job regardless of whether you select this option or not.

Preserve Attributes

This feature will Save any Attributes you have added to your Assembly either via UCS or Manually with your Object in the Catalog.

Preserve Notes

This feature will Save any Notes you have added to your Assembly either via UCS or Manually with your Object in the Catalog.

Preserve CAD

This feature will Save any CAD you have added to your Assembly in the Assembly Ortho Views and/or any CAD you have added to you Assemblies Parts in the Part Level Ortho Views with your Object in the Catalog.

 

The Advanced Button

The Class Tab

The Class tab allows selection of the Class and Type of the object to be saved.

It is a good idea to keep Base Class objects, Wall Class objects, Closet Class objects, etc within their original class as other areas of Solid identify objects by their Class.

The Placement Tab

There are three main areas on the Placement Tab:

Horizontal

Determines whether the Cabinet can be placed along the Wall or in a Corner or both.

Stretching

By enabling or disabling the Width, Height, and/or Depth options you can control whether the Object can be re-sized in those dimensions.

Vertical

Determines the Vertical Placement of your Object. There are thirteen Vertical Placement options:

      • Above Elevation - Will place the Object above the Elevation established for the Object as shown below.

      • Above Positional - Will place the Object above the value set for Vertical Positional on the Room | Layout Tab in Room Properties.

      • Above Soffit - Will place the Object above the Soffit as specified on the Room | Layout Tab in Room Properties.

      • Below Elevation - Will place the Object below the Elevation established for the Object as shown below.

      • Below Positional - Will place the Object below the value set for Vertical Positional on the Room | Layout Tab in Room Properties.

      • Ceiling - Will place the Object at the Ceiling (Wall Height) as specified on the Room | Layout Tab in Room Properties.

      • Counter Top - Will place the Object above the Counter Top.

      • Float - Will allow free placement Vertically on the Wall/Corner.

      • Floor - Will place the Object on the Floor.

      • Toe Height - Will place the Object at the Toe Height.

      • Under Counter - Will place the Object below the Counter Top.

      • Under Soffit - Will place the Object below the Soffit as specified on the Room | Layout Tab in Room Properties.

      • Under Uppers - Will place the Object below the Upper Cabinets.

The Closets Tab -

This Tab allows you to specify Placement options for Objects with the Closet Class. This options will be activated/deactivated based on the Type of Closet Object selected. There are five main options.

Snap Height to 32mm Increment - Used for Sizing Closet Classed Object heights in 32mm increments. This option actually serves two purposes.

      1. Enabling it denotes that you want the Objects Height to size in 32mm increments.

      2. The Height Offset value determines any extra dimension to add to the parts size outside of the 32mm increment Height rule.

Example 1: If you enabled this option with a value of 20 and placed the Object and gave it a Height of 1408mm you would get a 1428mm Height Panel: 1408 + 20 = 1428

Example 2: You can also enter Heights that are not on a 32mm increment. This option will evaluate the nearest 32mm increment so if you entered 1400mm you would also get a 1428mm Height Panel: 1400 = 1408 + 20 = 1428

This option controls and sets a Parameter on the Object named SNAPH32.

Snap Elevation to Increment - Primarily used to snap elevations of Horizontal Types to 32mm centers. This option actually serves two purposes.

      1. Enabling it denotes that you want the Objects Elevation to snap on either 32mm or User Specified increments.

      2. Its value determines the bottom reference of the Object.

Using this Option changes the Reference point of the Shelf to the Center of the Shelf. Thickness/2.

This option controls and sets the Parameter on the Object named SNAPSHY32. If a User Specified increment is set it uses the Object Parameters SNAPTOOSET and SNAPTOINC instead.

Snap Door Heights to 32mm Increment - This option is used to snap elevations of Doors on 32mm centers. Snaps to the top of the lower shelf’s position. This Option actually serves two purposes.

      1. Enabling it denotes that you want the Objects Elevation to snap on 32mm increments.

      2. Its value determines bottom reference of the Object in relation to the shelf below it.

This option controls and sets a Parameter on the Object named SNAPDOY32.

Snap Door Elevation to 32mm Increment - This option is used to snap Door and Drawer Heights to 32mm centers.

This option controls and sets a Parameter on the Object named SNAPOP32.

Stretch to Fit Doors and Drawers in Opening - This option is used to force Solid to Stretch-fit Doors and Drawers in openings. It can be set to Stretch in Width only, Height only, or both.

This option controls and sets a Parameter on the Object named STRETCHFIT. Value is either 1(Stretch-Fit Width Only), 2 (Stretch-Fit Height Only), 3(Stretch-Fit Both Width and Height).

The Preserves Tab

Clicking the Advanced button will present menus that allow preservation of other attributes, such as Material or Construction Methods.

Using this feature will effectively lock your Cabinet from further changes in other Jobs, therefore, it should be used sparingly.

The Aliases Tab

You can create multiple size versions of the same Assembly here as well.

This is an extremely powerful feature. Let's say you build Cabinets on an incremental basis. We start with a STYLE, let's say a Single Door Base Cabinet at 9" wide. We also build this same STYLE Cabinet in 12", 15", 18", 21", and 24". Simply create the STYLE Cabinet and Save it using this tab and it will create individual Cabinets in this STYLE, in differing dimensions, that you specify. See the example of the library below after completion of the above Save As...

To create a new alias, click the new alias button.    This will open the Add/Edit Aliases window.  Make sure to enter the correct nomenclature for the new alias as well as the dimensions for that cabinet.  Click OK to add the new alias to the list.

To delete one of the cabinet's aliases, select that alias and click the delete button.

  Edit

The Edit option takes the selected Assembly to the Assembly Level. You can, alternatively, double click the Assembly to take it to the Assembly Level.

 Section

Selecting this option will put you in the Elevation Section Editor. Simply click on the Assemblies in the Elevation to change the active Section Edit Assembly. This is a quicker way to Section Edit multiple Assemblies than the Right Click...Edit option.

 Edit Shape - 

This menu option allows you to edit the shape of your Assembly by bringing up the Shape Editor options.

Click the options below for more information

In :Core you will not have access to any of these Shape tools but you will have the ability to Move a Line parallel which can come in handy with Shaped Assemblies which ship with the Custom Cabinets Catalog. Clicking any line representing the Shape of the Object will show a small line adjacent to the selected line which can be dragged to alter the Objects shape.

 Edit Part Shape - 

The Edit Part Shape option brings up a list of Part you are hovering over.

Select the Part which you wish the Edit the Shape of which will bring up the Shaping tools in the ribbonbar to allow you to shape the currently selected Part.

The Advanced Layer option "Assembly Outlines Only" must be disabled in order to see and use this option.

 Boxed Ends (Upper Cabinets)

Selecting this option will extend your Wall Cabinets ends to the Floor as in a Refrigerator Cabinet.

 Combine

Selecting this option will allow you to combine the selected Assembly or Toe Kick with another Assembly or Toe Kick. After selecting this command simply left click on the second Assembly or Toe Kick and they will be combined together.

 

 Split

Selecting this option will allow you to split the selected Assembly or Detached Toe equally into two pieces.

 Detach Toe

Selecting this option will separate your Toe Kick from the selected Assembly. This will make the Toe Kick a separate Assembly with its own Assembly Number.

 Rehinge

Selecting this option will allow you change the Hinge side for an Assembly, if it is a Single Door Assembly.

 Finish End

Selecting this option will allow you to change the end condition for the selected Assembly.

 

Lock Width

Selecting this option will allow you to Lock the Width of the selected Cabinet, which keeps the system from automatically adjusting its width.

 Replace

Selecting this option will allow you to Replace the currently selected assembly with another assembly from your catalog. Simply select the assembly to replace it with from the Object Catalog in the sidebar by either a double click or clicking the green plus symbol.

 Normal placement rules apply with this Replace option, for example an object placed with  + Click. This means if the object will not fit for some reason you will get a Windows tone and the object will not be replaced.

 Properties

Selecting this option will open the Assemblies Property Options. There are 6 possible Tabs in the Assembly Properties Screen:

General Tab

Following are the properties you can modify on the General Tab:

Name

The Name field allows you to edit the name of the Object.

Quantity

You can change the Quantity of the selected Object. This will not add a new Assembly to your Layout but will add it to the Report area.

Description

You can modify the Description of the selected Object.

Price

You can modify the Price of the Assembly here.

Labor (In Minutes) -

Enter a value in minutes here representing the Assembly/Misc Labor for the selected Assembly. This value is accessed in the Bid Method Line Item Cabinet Assembly Labor.

Number

Allows you to change the default Assembly Number for this Assembly.

Lock Number

Locks the Assembly Number so Solid can not change it automatically when re-numbering Assemblies.

Snap Points

When selecting an Appliance or other decorative object you have an option to not show Snap Points for those Objects.

Open Doors

You can use this Slide Bar to open the Doors on the selected Assembly.

Open Drawers

You can use this Slide Bar to open the Drawers on the selected Assembly.

Open Trays

You can use this Slide Bar to open the Roll Out Trays on the selected Assembly.

Exposed Interior

Placing a check will apply the Exterior Finish to the Interior of the selected Assembly and will instruct the Cabinet to utilize the Material Schedule selected for Exposed Interior in Job/Room/Cabinet Properties.

Show Plan CAD

Enabling this option will display any CAD drawn at the Assembly Level Plan Ortho View for the Assembly at the Room Level Plan View.

Show Elevation CAD

Enabling this option will display any CAD drawn at the Assembly Level Face Ortho View for the Assembly at the Room Level Elevation View.

 

Comment Tab

This tab will allow you to add Comments to your Assembly. These Comments will be displayed on Assembly Sheets and in selected Reports you generate.

Finishes Tab

This tab allows you to modify the Interior and/or Exterior Finish and/or Texture for the selected Assembly.

Legs Tab

This tab allows you to select the Legs to use on the selected Assembly, this list will include Materials using the following Material Types:

Caster

Leg

Leg Leveler

Before using Legs you need to make sure your active Construction Method has the Toe Kick branch setup so Legs are properly placed.

Overrides Tab

This tab allows you to View and/or Remove any Overrides that may be present on the selected Assembly.

Notes Tab

This tab allows you to modify any Assembly Notes. This tab will only appear if you have created Assembly Level Notes via System Parameters or UCS.

Parameters Tab

This tab allows you to View and/or modify any Assembly Parameters. This area will also display the Name of the UCS modifying a Parameter.
