Right Click On Molding

Following are the possible options when Right Clicking on Molding:

 

 Delete

This menu option will delete the selected Molding.

 Delete All

This menu option will delete all the Molding in the current Room.

 Edit Shape

This menu option allows you to edit the shape of your Molding by bringing up the Shape Editor options.

Click the options below for more information

 Properties

The Properties menu option opens the Molding Properties area. There are six tabs in the Molding Properties area:

Molding Tab

There are six options on the Molding tab:

    • Material - Allows you to select a Material Schedule for the selected Profile.

    • Profile - Allows you to select an Edge Profile for the selected Molding.

    • Description - The Description/Name of the selected Molding.

    • Elevation/Outset - Allows you to modify the Elevation of the selected Molding if it is a Plan View placed Molding, and the Outset if it is an Elevation placed Molding.

    • Mirror Profile Vertical - Will Mirror the Profile vertically for the selected Molding.

    • Mirror Profile Horizontal - Will Mirror the Profile horizontally for the selected Molding.

    • Rotate Profile - Will Rotate the Profile for the selected Molding.

Comment Tab

The Comment tab allows you to add a Comment/Note on the selected Molding.

Overrides Tab

This Tab allows you to View and/or Remove any Overrides that may be present on the selected Molding.

Parameters Tab

This Tab allows you to View and/or modify any Parameters on the selected Molding.
