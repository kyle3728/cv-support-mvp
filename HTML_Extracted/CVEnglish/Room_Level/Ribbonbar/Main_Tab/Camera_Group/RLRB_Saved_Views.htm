Saved Views

These options will allow you to work with any Saved Views you have created via the 3D View Right Click Save View option.

You can either use the drop down menu to select or use the Previous Saved and Next Saved options to scroll through your Saved views.

The Edit Views option allows you to Delete any Views you no longer need in the Job.

The Lock Orientation option keeps you from accidentally Rotating your view. Disable this option if you want to Rotate the Saved View. Note: Rotating the View with this option disabled does not re-save the View at the new Rotation. You would need to use Save View to save a new orientation.

  The Edit Walk Through option allows you to string all, or some, of your Saved Views together into a Walk Through. CABINET VISION will create smooth seamless transitions from one Saved View to another. The left side of the Walk Through Manager will list all of your current Available Saved Views. Selecting and then moving  a Saved View to the right into the Walk Through pane will establish the Walk Through. You can move the Views arround in the list, controlling their order, by using the Move Up and Move Down arrows. The Walk Through created here will be used in +Tour.

 The Walk Through option allows you to "Play" the Walk Through you have created through the Edit Walk Through option. The Walk Through can be played in Wire Mode, Fill Mode, Texture Mode and if you have xRender it will also play in Realtime.
