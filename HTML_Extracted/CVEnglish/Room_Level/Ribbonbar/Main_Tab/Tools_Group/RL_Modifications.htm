Modifications

There are eight Room options:

Quick Room

This option allows you to quickly layout a new Room through a Wizard style interface. Follow the directions in this Wizard to create a Room with Appliances and Cabinets. See the Preferences | AutoPlan topic for default preferences.

First, select the Wall layout that you wish to use and then click Next.

 

 

Then, enter the lengths of the Walls. Each Wall is designated by a letter in red. Enter the length on the corresponding line to the right.

 

 

Next, add a Door and choose its position. Choose the Wall according to the Wall number in red and select the same number in the pull down list to the right.

 

 

Then, add a Window and enter its location.

 

 

Then, you can add a Sink and choose its location.

 

 

Add a Dishwasher next to the Sink if you want.

 

 

Next, add a Range if necessary and choose its location. Choose None if you don't want to add a Range at all.

 

 

Next, add a Refrigerator and choose its location if you want.

 

 

Finally, choose what types of Cabinets, if any, that you wish to add to this Room. You could also choose not to place Cabinets in this view in order to be able to manually add the Cabinets in the Room itself. Here, you can also choose to add Counter Tops, Molding, and a Floor.

 

Click 'Finish' when you have selected all of your options and Solid will automatically create the Room for you.

Copy Room

This option will "Copy the active Room" to a new Room.

Delete Room

This option brings up a list of all the Rooms in your Job and allows you to delete a selected Room. You can also use standard Windows + Click and/or + Click selection here to Delete Multiple Rooms at once.

Center Room

This option will center your Room in the drawing space. If you notice, with the Grid option on in Floor Plan view, there are 2 darker blue lines. One line that is horizontal and one that is vertical. Where these lines cross is the center of the Room and the intersection of the 2 lines is the pivot point of the left to right motion with the mouse in 3D Views.

You can turn the Grid on/off by using the Right Click on Screen Properties menu option.

Reverse Room

This option allows you to "Reverse Plan" a Room.

Export Room

This option allows you to Export a Room to a CVR file for Import into another Job.

Import Room

This option allows you to Import a previously exported Room (CVR), into the current Job.

 Wall Heights

This option allows you to change the Heights of the Walls in the current Room.

Following are the options in the Modify Wall Height utility:

  • New Wall Height - Enter the dimension here for the new Wall Height.

  • Soffit Height - You can choose here either to leave the Soffit at its current height adjusting the assemblies heights to compensate or to leave the assembly heights as is and adjust the soffit height instead.

  • Ignore Modified Walls - With this option enabled the Heights for Walls which have been modified will not be adjusted.

  • Update Room Properties - With this option enabled the New Wall Height entered in this utility will be set as the Wall Height in th Room's Layout Properties.
