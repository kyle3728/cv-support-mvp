Dimension Styles

There are six options on the right side of this screen:

New

Clicking the New button will add a new Style in the Styles list and will give it the name "New Style". You will probably want to rename this with a name that holds some meaning for when and where you may use this new Dimension Style.

Edit

Copy

Clicking this button will copy the selected Dimension Style giving it the name Copy of (Original Name).  You will probably want to rename this with a name that holds some meaning for when and where you may use the copied Dimension Style.

Rename

Clicking this button will open a text editing box around the name of the selected Dimension Style; Simply type in the new name here.

Remove

Clicking this button will Remove (delete) the selected Dimension Style.

Close

Clicking this button will Close the Dimension Styles Editor.
