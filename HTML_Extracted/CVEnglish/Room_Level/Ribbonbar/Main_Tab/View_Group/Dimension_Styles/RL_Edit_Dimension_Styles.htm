Edit Dimension Styles

There are four sections in the Style Edit screen:

General

The General section has two options:

    1. Line. This option determines whether or not your Dimension Style will have extension Lines.

Yes

No

    1. Fixed Scale. This option activates the "Fixed Scale Mode" which allows or prevents some of the dimension properties to be scaled according with the Drawing scene scale factors.

With this mode activated the tick height, line extension, tail offset, arrow/mark size, text offset, text clearance, and text height, will always be the exact values expressed in this list, regardless of any scaling factors.

Yes

No

Lines and Arrows

The Lines and Arrows section has eight options:

    1. Line Width -This setting allows you to establish the width (thickness/weight) of the Dimension lines.

    2. Ticks Height - This setting determines how far the extension line will extend vertically past the dimension line. The area that the Ticks Height option affects is highlighted in red below.

    1. Lines Exten. - This setting determines how far the dimension line will extend horizontally past the extension lines. The area the Lines Exten. option affects is highlighted in red below.

    1. Tail Offset - This setting determines how far the Extension Line will be away from the object being dimensioned. The area the Tail Offset option affects is highlighted in red below.

    1. Left Arrow - This option determines the Arrow type on the Left end of the Dimension Line. There are fifteen choices:

  2. None

  1. Closed

  1. Close Blank

  1. Open 30

  1. Open 90

  1. Dot

  1. Dot Blank

  1. Dot Small

  1. Dot Small Blank

  1. Origin

  1. Origin 2

  1. Architectural Tick

  1. Oblique

  1. Box

  1. Box Blank

    1. Right Arrow - This option determines the Arrow type on the Right end of the Dimension Line. The choices are the same as above for the Left Arrow.

    2. Center Mark - This option determines the Center Mark type for Radius and Diameter dimension types. There are three choices:

      1. None.

      2. Cross   

      3. Target   

    1. Arrow/Mark Size - This setting allows you to control the size of Arrows and Center Marks.

Text

The Text section has nine options:

    1. Alignment - This setting determines how the Text is aligned with the Dimension Line. There are two choices:

Aligned

Horizontal

    1. V Placement - This setting determines how the Text is oriented in relation to the Dimension Line vertically. There are four choices:

Centered

Above

Outside

Below

    1. H Placement -  This setting determines how the Text is oriented in relation to the Dimension Line horizontally. There are five choices:

Centered

At Ext Line 1

At Ext Line 2

Over Ext Line 1

Over Ext Line 2

Selecting to place the text over the Extension Lines will force a 90° angle between the Text and the Dimension Line.

    1. Offset - The Offset setting works in conjunction with the V Placement option. Once a V Placement option is selected a value entered here will move the text away from the Dimension Line.

    2. Clearance - This option controls the gap the text is located within on the Dimension Line. This space is shown in red below.

    1. Frame - This setting allows you to have a Frame placed around your Dimension text. The Frame size is controlled by the Clearance setting.

    1. Font - This setting allows you to choose the Font style for the text in your Dimension Style. All Fonts registered in Windows™ are available here.

    2. Bold - This setting allows you to set the appearance of the Font to Bold.

    3. Height - This setting allows you to control the Text height for the Text in you Dimension Style.

Fit

The Fit section has two options:

    1. Hide Text - If this option is set to Yes, Solid will hide the text on dimensions if it does not fit in the dimension space. If this option is set to No, the text will be moved away from the dimension line in order to be displayed.

    2. Hide Arrow - If this option is set to Yes, Solid will hide one or both arrows if they do not fit in the dimension space. Also if this is set to Yes, the arrow will be hidden to try to fit the text before moving it from the dimension line.
