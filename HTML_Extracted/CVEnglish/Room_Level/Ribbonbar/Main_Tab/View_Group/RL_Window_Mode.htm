Window Mode

CABINET VISION allows viewing of your work in three views simultaneously. This Split Screen feature allows you to alternate between windows as you work, while seeing real time changes in the other windows. Simply select the appropriate orientation from the Split Horizontal or Split Vertical option allowed.

Choosing the Window View

Each window will allow access to the views available at this level, I.E., the Plan, Elevation, 3D and Report views. Click in a window to activate it, and select the View Tab representing the view you would like for that window.

Choosing an Active Window

You will notice that the color band on top of the  window is highlighted when it is the active window. If you wish to work in a window that is not the active window, you must first click in the window to make it the active window. You will have access to the features normally found in that window once it is active.

Maximizing a Particular Window

Each window has a small square button in the upper right hand corner. If this button is clicked, the window will maximize and occupy the entire screen.
