Shop Annotations

There are five main options in the Draw section:

Architectural Hinges

This option turns On and Off the display of Architectural Hinges.

 

              ON                             OFF

 

Reverse Architectural Lines

This option simply reverses the Door swing lines for Architectural Hinges.

Roll Out Tray Outlines

This is the same as the option turns on or off the outlines for Roll Out Trays.

Shelves

There are two options:

 

         Single Line                 Outline

 

There are three main options in the Dimension area:

Opening Width / Open Height

Will display dimensions representing the Opening sizes. These options can not be used in conjunction with the Doors and Drawers Size option. See the image below for example.

Doors and Drawer Size

Will display the dimensions representing the Door and Drawer sizes. This option can not be used in conjunction with the Opening Size options. See image below for example.

Rails/Stiles

See the image below for examples:

 

           Rails                      Stiles

You can also enable/disable the dimensions for Adjustable Shelf Opening Height here.

Frame Overlay

When using a Frame Overlay Construction Style you have two additional dimensioning options. You can Dimension either the Frame Openings or the Case Openings.

Closet Rods Center-Top Clearance

This option will show the Dimension for the Vertical placement of a Closet Rod in a Hanging Section. (Closet Module only)

Font

Clicking on the Font button allows you to change the Font used for Elevation Part Dimensions. You also have the option for the Text to be in Bold Script as well as setting a Fixed Height in the Drawing Views.
