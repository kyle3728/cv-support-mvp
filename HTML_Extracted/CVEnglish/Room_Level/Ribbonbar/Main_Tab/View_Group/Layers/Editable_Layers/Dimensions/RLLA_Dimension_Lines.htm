Dimension Lines

This area (highlighted above) of the Dimension Editor allows you to control which dimensions appear automatically in CABINET VISION. There are seven options.

New

Clicking this button will add a new Line in the Lines list and will give it the name "Dimension (1)". You will probably want to rename this with a name that holds some meaning in this list. Each line in the list represent a separate dimension line in your drawing. See example below:

Edit

Copy

Clicking this button will copy the selected Line giving it the name Copy of (Original Name). You will probably want to rename this with a name that holds some meaning in this list.

Rename

Clicking this button will open a text editing box around the name of the selected Line; Simply type in the new name here.

Remove

Clicking this button will Remove (delete) the selected Dimension Line.

Move Up

This button along with the Move Dn button controls the order in which Auto-Dimensions are placed. The order the Lines are listed in this list is the order the lines are placed in your drawing. In the example below you will notice that Wall Length is first followed by Base Assemblies.
