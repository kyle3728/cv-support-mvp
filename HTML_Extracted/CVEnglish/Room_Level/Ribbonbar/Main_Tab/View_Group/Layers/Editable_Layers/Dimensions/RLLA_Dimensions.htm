Dimensions

The physical appearance of Dimensions is controlled in the Dimension Styles Editor. Where and how the Dimensions appear is controlled here. Following are the main options:

  1. Style - This option allows you to select which Dimension Style to use for the Layer Schedule you are working with. The Action button will open the Dimension Style Editor allowing you to create a new or edit and existing Dimension Style.

  2. Horizontal Line Offset - This setting controls how far away from the object being Auto-Dimensioned the first Horizontal Dimension line should be located.

  3. Horizontal Line Increment - This setting controls how far away any subsequent Horizontal Dimension lines should be located away from the preceding Dimension Line.

  4. Vertical Line Offset - This setting controls how far away from the object being Auto-Dimensioned the first Vertical Dimension line should be located.

  5. Vertical Line Increment - This setting controls how far away any subsequent Vertical Dimension lines should be located away from the preceding Dimension Line.

  6. Tail Dimensions - This option allows you to display your Dimensions with Tails.
