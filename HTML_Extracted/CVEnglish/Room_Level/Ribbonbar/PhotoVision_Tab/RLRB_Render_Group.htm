Render Group

There are three main options in the Render Group.

 Realtime

Takes your current Material and Lighting settings and renders a view with realistic Textures right on the screen. Because it is a PhotoVision Render Lighting is applied. This view is simplified Lighting with no Shadows. It is also a Realtime view so can be rotated and panned while Rendered. See the Realtime 3D Properties tab for more information.

Below is an example of a Render in this mode.

 +Tour

This option switches you to the +Tour mode which is a realtime walkthrough of your camera views. See +Tour tab help topic for information on settings for +Tour.

Advanced

Takes your current Material and Lighting settings and renders a view with realistic Textures and shadows right on the screen.

There are three modes in this option.

  • Preview - This view gives you a quick Rendering with no anti-aliasing.

  • Full - This is a full rendering which includes anti-aliasing.

  • Save - This option allows you to Save the Render. See the Saving Renders topic for more information.

See the Right Click On Screen 3D topic for information on editing this options custom settings.

Below is an example of a Full Render in this mode.

Architectural

Adds realistic reflections to the scene, which surfaces will cast reflections is determined by the Ray-trace Tab in the View Properties dialog box.

There are three modes in this option.

  • Preview - This view gives you a quick Rendering with no anti-aliasing.

  • Full - This is a full rendering which includes anti-aliasing.

  • Save - This option allows you to Save the Render. See the Saving Renders topic for more information.

See the Right Click On Screen 3D topic for information on editing this options custom settings.
