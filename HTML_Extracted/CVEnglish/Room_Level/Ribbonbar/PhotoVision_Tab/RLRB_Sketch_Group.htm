Sketch Group

Following are the options in the Sketch Group:

Cartoon

A Sketch style giving a cartoon-like image (flat colors and outlines around objects).

  • Full - This is a full rendering.

  • Save - This option allows you to Save the Render. See the Saving Renders topic for more information.

Hand Drawn

A Sketch style giving the effect of a hand-drawn outline.
