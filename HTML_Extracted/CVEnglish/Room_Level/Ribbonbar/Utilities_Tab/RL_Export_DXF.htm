Export DXF

If you are in the 3D View the Export DXF option allows you to create a DXF file of the current 3D View for modification in a 3rd party CAD package.

If you are in the Drawings Area this option allows you to Export the current Drawing Page as a 2D DXF file for manipulation in a 3rd Party CAD program. A dialog window will appear and give you the opportunity to name and save the DXF file to any location on your computer.

When you have typed in the name you want and have selected a location, click Save to complete the process.
