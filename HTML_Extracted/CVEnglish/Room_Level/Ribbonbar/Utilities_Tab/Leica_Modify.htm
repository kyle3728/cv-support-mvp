Leica Modify

This Utility will allow you to modify Leica points that you either input directly with the Leica Disto Utility or imported via DXF.

There are three types of Modifications you can perform in the Leica Disto Modify Utility.

Translate

This option allows you to move the group of Leica points in the current room a specified distance in X, Y, and/or Z.

Scale

This option allows you to Scale the group of Leica points in the current room by your specified Factor.

Rotate

This option allows you to Rotate the group of Leica points in the current room by a specific Angle about the X, Y, and/or Z axes.
