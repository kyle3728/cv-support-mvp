Output Data File

The Output Data File option is part of the xReport multiplier and allows you to output part information to certain 3rd Party Applications. Select which 3rd Party Outputs you want from the list in the sidebar and click OK to Output. Use the Tabs to configure the individual Outputs.

PNL File Tab

In the first tab, you can choose to output a .PNL File by checking the "Output .PNL File" box. Then, you have the option to include the Room Number in the file in place of the Cabinet Number. Finally, you can select the destination of the .PNL File output.

See the PNL File Format topic for information.

CUT File Tab

On the second tab, you can choose to output a .CUT File by checking the box labeled "Output .CUT File." On the right side of this window you can select the types of Materials to include in the .CUT File output. Other options for the .CUT file include changing the Grain Setting, limiting the output to 32 fields, and including banding color and Material. You can also select the destination for the .CUT file.

See the CUT File Format topic for more information.

Material Usage Tab

The third tab allows you to output Material usage information about the current Job by checking the "Output Material Usage" box. When you click OK, this will populate the matsum.mdb database file in the Database directory with the usage information from the current Job. If you also check the box labeled "Update Material Data From Material Catalog", clicking OK will also populate the matsum.mdb database with the information about all of the Materials in the Material Manager. This makes it possible to use the matsum.mdb file to create custom reports for the current job outside of Solid.

See the Matsum.mdb topic for more information.

DoorCIM Tab

The fourth tab in the Output Data Files Window is the DoorCIM tab. On this tab, you can choose to Output DoorCIM files which are lists of Door Styles, Door names, and gaps. On this page, you will need to set the default gap sizes and set the amount to adjust the Door sizes. On the right, you have the ability to select the Doors that are included in the output by turning filters on and off. Finally, at the bottom of the window you can choose the destination for the DoorCIM files.

Ottimo Tab

The last tab in the Output Data Files Window relates to Ottimo files. In order to create the Ottimo files, you need to check the box labeled "Output Ottimo File". If you check the box labeled "Create Material Files" and set a path to the desired location for these files, you will output a file for each Material in the Ottimo File. At the bottom of this window you will also have the ability to set the desired path for the Ottimo file output and you can choose to have the Material names shortened in the output by removing any spaces in Material names.

Custom Tab

Any custom Output Configurations you have created will be listed here. Simply enable the file(s) you would like output and click ok.

File Property Area

There are ten general options available to configure your output:

File Name - This field will allow you to specify a specific or partial file name. You may choose to use {JobName} as a variable. Any use of {JobName} will be replaced by the current job.

Example settings:

{JobName}.txt ► Smith.txt

MyOutput.csv ► MyOutput.csv

Out-{JobName}.lst ► Out-Smith.lst

Header - The header is optional. If specified it will be placed once at the top of the file. It can be left blank to omit. Any use of \n will be replaced with a carriage return.

Example Setting: ;This is a heading\nThis is the second line\nThis is the 3rd line.

;This is a heading

This is the second line

This is the 3rd line.

Fixed Width - This will determine if each field will use a specific number of characters in the output. If the actual data has fewer characters then that specified the remaining characters will be padded with space. If the actual data exceeds the character limit then the value will be truncated.

String Length includes the Text Qualifier. If the Field Length is set to 4 and the String is “ABCDEFG” the output will be “AB”. If there is no text qualifier the output would be ABCD.

To set the length of the output field first select the field on the right then enter the Fixed Length at the bottom of the list.

If Fixed Width is false then the Text Length indicates the maximum length of any field. This by default can be left at zero, but can be set to a number it you wish to limit the output length. Unlike Fixed Width, the max length does not include the text qualifier. A max length of 4 with the string ABCDEF would result “ABCD”

Delimiter - This is an optional single character to separate each field. It is commonly a comma (,).

Text Qualifier - This is an optional single character that will surround and text or memo output. Typical values are (“) and (‘).

When the output type is not fixed width if the text qualifier character is found in a text or memo field it will be replaced with either “ or ‘ (whichever is NOT the text qualifier character). The example below uses a double quote (“) as the text qualifier.

A “sample” string with “double” and ‘single’ quotes: “A ‘sample’ string with ‘double’ and ‘single’ quotes.”

Units - The unit can either be metric or imperial. As an option the unit may also be multiplied by a factor of 10.

Unit Precision - This option specifies the unit precision. Value will be rounded to the nearest selection. The unit precision applies to measurements as well as singles and doubles.

Boolean Format - This option allow for multiple Boolean formats.

A string value such as Yes, No, True, False, Y, N, T, F will NOT be surrounded by the Text Qualifier.

Date Format - A string value using (M)onth, (D)ay & (Y)ear as variables. Example settings:

M/D/YY 10/4/10
DD-MM-YYYY 04-10-2010
DDD DD-MM-YYYY Mon 04-10-2010
DDD, MMM DD/YY Mon, Oct 04/10
DDDD, MMMM DD of YYYY Monday, October 04 of 2010

Currency Precision - Similar to unit precision, the value will be rounded to the specified precision.

Field Definition Area

The list on the left are all of the Fields that you can include in your output file. Simply select the fields you would like then click the Right Arrow button to add them to the list on the right. Once a field has been moved over to the list on the right it’s order can be modified by selecting the Up or Down arrow buttons. Multiple fields can be moved at the same time by 1st selecting multiple fields then selecting the Up/Down buttons.

It may be required to output a value that it not represented in the available data list from Cabinet Vision. This type of output is typically not-critical data but is still required by the importing software to fill a space in the output format.
