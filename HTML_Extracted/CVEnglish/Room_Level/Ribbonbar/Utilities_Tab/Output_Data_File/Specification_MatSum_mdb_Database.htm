The MatSum.mdf Database

This Database stores a running Material Summary for every Job the Data is output for. The ability to populate this data is allowed with xReport multiplier.

The following is the breakdown of the tables and their fields:

DB Info

This Table stores general information about the Database.

Field

Description

Version

Displays the current Version for the Database.
