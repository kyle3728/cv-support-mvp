Import SketchUp Model

Objects Imported with this method are not machinable and are only useful as visual representations inside CABINET VISION.

Find the SKP file that you want to import, click on that file and then click Open.

CABINET VISION currently supports the import of SketchUp 2020 and earlier files.

To open a different file you can click the browse button to open a file browse window where you can search for a .SKP file that you want to import.

SketchUp Import Options

Following are the options in the SketchUp Model Import window:

  • Browse and Select - The Browse and Select button brings up the Windows Browse window that allows you to select the SketchUp Model you would like to import.

  • Import - The Import button will Import the SketchUp Model selected by the Browse and Select button.

  • Line Mode - Displays the Imported SketchUp Model in wire frame. This is the default Preview Window view.

  • Fill Mode - Displays the SketchUp Model Part with the selected Part Types Finish (Exterior or Interior).

  • Solid Render Mode - Displays the SketchUp Model Part with the selected Part Types Finish and Texture.

  • Flip X - Will Flip the Imported SketchUp Model 90 degrees along the X axis.

  • Flip Y - Will Flip the Imported SketchUp Model 90 degrees along the Y axis.

  • Flip Z - Will Flip the Imported SketchUp Model 90 degrees along the Z axis.

  • Scale - Allows you to Scale the size of the Imported DXF. It is typically a good idea to create the DXF the size you will be using it. However, if that is not possible you can use this option to Scale the Imported Part to the value you enter in the box below it.

  • Type - Allows you to select the Cabinet Vision Part Type to associate the Imported Part to. You must assign a Part Type to each imported Layer, Material, Color, or Component. To do that simply select each Layer, Material, Color, or Component in the list (highlighted below) and assign it a Part using this option. Each object you select will be highlighted in the image above so you can easily identify each of them.

  • Texture Map - This option allows you to specify how Textures are applied to the Imported Part. There are seven possible map types. See the Texture Map section of the Import DXF topic for more information.

  • Smooth Surface - This option will smooth the surface of some 3D Models. The sphere below demonstrates this functionality.

  •  Single Sided Faces - If your SKP appears distorted or vague in the Preview window this option maybe required.

  • Reverse Faces - This option becomes available and may be necessary when Importing Single Sided Face SKP's.

  • Advanced - This option opens a window allowing you to select how the entities are separated in the source 3D Model.
