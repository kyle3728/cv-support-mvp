Tops -

After selecting this feature, you will see your Room, with an outline around your Assemblies that are Classified as Base or Vanity and a red dashed outline that appears to be a Counter Top on one of the Walls. This red outline represents the Wall that is active. The active Wall is the Wall that the Counter Top is attached to in the hierarchy of CABINET VISION. If you draw a Counter Top with an active Wall other than the Wall the Counter Top is on, and later delete the Wall that was active, you will loose that Counter Top even though it appeared to be attached to a Wall that still exists. Always give thought to which Wall should be the active Wall when creating your Counter Tops.

Selecting the correct active Wall (the red outline) is VERY important. If you have a different Wall activated than the one you are drawing your Counter Top on you will not see your Counter Top in the proper Elevation view. Another VERY important point to note is when using the Bid Center and using the Bid Method Per Lineal Foot of Counter Top or Per Lineal Meter of Counter Top the calculation is based on the active Wall. Not having the correct active Wall selected can cause ERRORS IN YOUR BID.

To change the active Wall, simply click in front of the Wall that you would like to be active. This works equally well on Wall faces and Wall backs.

Creating Counter Tops for your Job requires choosing whether you would like to use the AutoBuild Tops sidebar button or build the Counter Tops using the New Top sidebar button. Simple plans, are easily accomplished using the AutoBuild feature. Click the AutoBuild button, then choose whether you wish the Counter Tops built for the entire Room, or just a single Wall. If you choose the single Wall option, be sure that you have selected the correct Wall as the active Wall.

If you choose to use the New Top method, you will see the same outline with a reference to the active Wall as shown above, however, you will be required to use the Counter Top Shape Editor that will appear in the sidebar.

It is possible to use both methods in a Job, however, if you choose to do so, you must build using the AutoBuild feature before building any Counter Tops by the New Top feature.

The AutoBuild feature will delete any existing Counter Tops in the current Room.

After the shape of the Counter Top was drawn, and the configuration completed, the edges were defined by selecting the edges and setting their Edge in the sidebar.

Edge - Automatic

The system will interrogate an automatically assign edge types.

Edge - Unfinished

This option specifies an Edge on a Counter Top to be Unfinished.

Edge Finished

This option specifies an Edge on a Counter Top to be Finished.

Edge Profile

This option specifies an Edge on a Counter Top to incorporate your selected Counter Top Edge Profile.

Edge Splash

 This option specifies an Edge on a Counter Top to have a Splash.

Edge Butt

This option specifies an Edge on a Counter Top to have your specified Counter Top Joinery.

See the Edit Shape section of the Right Click on a Counter Top topic for more information on the tools you can use to manually draw your Counter Tops.
