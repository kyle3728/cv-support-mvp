Export Report

The Export option allows you to Export the currently active Report to another application or file for further modification. Finished reports can be exported to a number of popular formats, such as spreadsheet, word processor, HTML, XML, and other common data interchange formats. This makes the distribution of information easier. For example, you may want to use the report data to project trends in a spreadsheet package or to enhance the presentation of data in a desktop publishing package.

The exporting process requires you to specify a format via the Direct to... option.

Export Direct To... Types

The export formats supported by the Report Center can be broadly categorized in two groups: page-based formats and record-based formats. Page-based formats tend to produce a more exact output. The emphasis of these formats is layout representation and formatting. Formatting refers to attributes such as font style, text color, text alignment, background color, and so on. Layout refers to object position, object size, and the relationship between these attributes and other objects. Depending on the format of the Report, it may not be possible for the program to preserve all layout and formatting perfectly, but page-based formats, in general, preserve these properties as closely as possible.

With record-based formats, the emphasis is on data rather than the layout and formatting. However, in some formats—such as Microsoft Excel - Data only—you will notice that some formatting is exported.

Please consider:

  • Due to the format, the layout cannot always be taken over 1:1 because there are specific restrictions when converting to these formats.

  • The Microsoft® Word export format creates DOCX files that are compatible with Microsoft® Office 2007 and higher. Tables will be exported on continuous pages to support later editing.

Many formats have various possibilities for configuration. You can make use of these by clicking the "Options" button.

  • With Excel formats, you can reduce the number of columns and rows to the minimum required by means of the "Only data from table object(s)" option. Also all pages can be exported into one sheet and the output formats 'xls' as well as 'xlsx' are available.

  • The picture export to JPEG, TIFF and PNG supports optional automatic cropping of the results to the content. An exported barcode is therefore only exactly as large as required to display the complete content.
