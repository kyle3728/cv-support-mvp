Splash -

Splashes can not only be added when creating Counter Tops in Plan view but can also be added to your Designs from the Elevation View. From Elevation View simply click the Splash button in the sidebar. Adding a New splash uses the Shape Editor. You can also modify the Shape of Splashes placed via the Counter Top tool from Elevation view simply by right clicking on them in Splash mode and selecting Edit Shape.

To access the options for changing the Splashes Thickness and other properties, right click on the Splash as soon as you enter the Editing window. The following menu will appear if you have performed this correctly.

 

If the Splash you are attempting to Edit was placed using the Tops tool, you will need to Edit the Splash in this area before having access to Splash Properties.

Selecting Delete will delete the selected Splash.

Selecting Delete All will delete all the Splashes on the current Wall.

Selecting Edit will take the Splash to the Assembly Level. There is very limited access to modify Splashes at the Assembly Level.

Selecting the Edit Shape right click option will open The Shape Editor options in the ribbonbar allowing you to Edit the Shape of the Splash.

This area allows setting the Thickness of the selected Splash. Note that the Material and Texture can also be adjusted here.
