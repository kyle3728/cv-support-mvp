Finishes -

Change Finishes by clicking the desired "color" swatch and selecting the item whose color you desire to change.

Note: These colors will appear in the "Fill Mode".

The Finishes Window offers several color options depending on the items that are listed. For example, the Finishes Window for Appliances is shown below. Notice that there is a drop down list for Appliance where you can select whether you want the color choice to affect all Appliances or you can choose colors for each Appliance individually. Also, there is a drop down that covers Objects such as the Appliance Panels and the Handles or Pulls.

In the image above "Use Material Finish" is enabled. With this option enabled you can see what colors are assigned to the Objects by the Material Schedule. To change Finishes disable this option.

Now you can select and Finishes that exist on your system. There is an option to quickly Search for a Finish.

You also have the ability to create a New Finish from here.

You can use the sliders to set the color you would like to create or if you know the colors RGB value you can enter it here. Give the new Finish a Name and it will be added to your list of available Finishes.
