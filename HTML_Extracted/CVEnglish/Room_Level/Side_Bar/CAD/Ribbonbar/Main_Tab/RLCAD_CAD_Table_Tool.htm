CAD Table

The CAD Table tool allows you to create Text Look-Up Tables of data similar to a spreadsheet. Depending on the Area or Level of CABINET VISION you are in at the time determines what data is available to populate your Table.

After clicking the Table tool you will specify the start and end point of the Table just like when using the CAD Rectangle Tool.

After clicking the start point (Yellow) and end point (Green), you will be presented with the Table toolbox.

Groups and Columns

This area allows you to create and organize the Groups of data you want displayed in the Table. There are five options here:

Add

This option allows you to add a new Group to the list.

There are three options in the New Group window:

  • Header - This is where you give your Group a name.

  • Filter - This is where you select the type of data you want displayed. The options available here will be different depending on the Area (Paper, Assembly Sheets, Wall Elevations) or the Level (Room, Assembly, Part) of CABINET VISION you are in.

Elevations/Wall Elevations

Assembly Level/Assembly Sheets

  • User - Select this option to build your own Custom Group.

Also notice that once you have added your fields to the Visible area you can move their order in the list by using the Up and Down buttons.

Editing Tables

To edit the look and feel of your table simply click anywhere in the Table using the CAD Select Tool. With the Table active you will see the available Editing options in the CAD sidebar.

To edit the Groups or line items in an existing table simply Right Click Edit the Table or Double Click on it with the CAD Select Tool.

There are 40 possible Table Editing options:

  • Color - Allows you to change the color of the lines making up the Table.

  • Layer - Allows you to change the CAD Layer the Table is on.

  • Line Type - Allows you to change the Line Type of the Lines making up the Table.

  • Line Width - Allows you to change the Line Width of the Lines making up the Table.

  • Left - Allows you to change the Left position of the Table. Referenced from Grid Center.

  • Right - Allows you to change the Right position of the Table. Referenced from Grid Center.

  • Top -  Allows you to change the Top position of the Table. Referenced from Grid Center.

  • Bottom -  Allows you to change the Bottom position of the Table. Referenced from Grid Center.

  • Center X - Allows you to change the Center X position of the Table. Referenced from Grid Center.

  • Center Y - Allows you to change the Center Y position of the Table. Referenced from Grid Center.

  • Width - Allows you to change the Width of the Table.

  • Height - Allows you to change the Height of the Table.

  • Frame - Allows you to turn the Frame of the Table On or Off.

  • Grid - Allows you to turn the Grid of the Table On or Off.

  • Allow Duplicates - Set to Yes will consolidate like Parts on a single line item with the total Quantity reflected.

There are three main areas of each table.

Group

  • Group Font - Allows you to select the Font for the Group area of the Table.

  • Group Bold - Allows you to select the Font weight for the text in the Group area of the Table.

  • Group  Height - Allows you to change the Font Height for the Group area of the Table.

  • Group HAlign - Allows you to select the Horizontal Alignment of the Text in the group area of the Table.

  • Group Padding - Allows you to set the Spacing between one Group and the next, when the Table consists of multiple Groups.

  • Group Show Text - Allows you to display or to suppress the display of the Group Title.

  • Group Show Empty - Allows you to display or to suppress the display of a Group if it contains no data.

Header

  • Header Font - Allows you to select the Font for the Header area of the Table.

  • Header Bold - Allows you to select the Font weight for the text in the Header area of the Table.

  • Header Height - Allows you to change the Font Height for the Header area of the Table.

  • Header HAlign - Allows you to select the Horizontal Alignment of the Text in the Header area of the Table.

  • Header Show Text - Allows you to display or to suppress the display of the Header Title.

Data

  • Data Font - Allows you to select the Font for the Data area of the Table.

  • Data Height - Allows you to change the Font Height for the Data area of the Table.

Selected

Group Selected

    • Caption - Allows you to Name/Rename the Headers Caption.

Header Selected

    • Caption - Allows you to rename the Column Header Caption.

    • Width - Allows you to set the Column Width.

    • Sort - Allows you to set the Sorting of the Column.

Data Selected

    • Width - Allows you to set the Data Column Width.

    • Sort - Allows you to set the Sorting of the Data.

    • Bold - Allows you to select the Font weight for the Data.

    • HAlign - Allows you to select how the Data Column is aligned horizontally.

    • Negative Numbers - Allows you to select how Negative Numbers are displayed.
