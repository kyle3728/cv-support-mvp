CAD Right Click Options

Right Click on Screen

 

 Paste - This option will Paste the CAD Object(s) stored in the Windows Clipboard to the current View.

 Select All - This option will Select All the CAD Objects in the current view.

 Reverse Selection - This option will deselect the currently selected CAD Object(s) and select all other unselected CAD Object(s).

 Line Widths - The Line Widths option allows you to globally change all CAD Line Widths to a different Line Width.

 Properties - The Properties option opens the View Properties screen.

Draw Tool Tab

  • Grid Spacing – Sets the visual spacing of the Grid.

  • Grid Always Visible – Under normal circumstances the Grid will only be visible when the Grid Snap option is enabled. This option will always display the Grid.

  • Reset Ruler – Clears Snap Marks from the screen

Keys Tab

  • Short-cut Keys – These options allow you to set which short-cut keys to use for snapping to Line Mid Point, Line Intersection, and Circle/Arc Center.

Paper Tab

Gives you the ability to control the Paper Type and Orientation.

 Play

This option will Open the selected Media Object in it's Associated Windows Application.

 Edit...

You can also double-click any editable CAD object to get into Edit mode.

There are four editable CAD Objects:

Text

Text Box - This option will re-activate the Text Box for editing.

Dimension

The Dimension Editor will allow you to Edit the Dimensions Value or the Text contained in the Dimension. Activating the Text option in this area will turn the Dimension into Text and allows you to add to or modify the Text without affecting the Dimension Line length.

Media Object - This option will allow you to Edit the selected Media Object in it's Associated Windows application.

 Save Symbol - This option will allow you to Save the selected CAD Object to the Drawings Library.

 Convert To Static - This option will allow you to convert a Table from a Live Table to a Static Table.

 Cut - This option will Cut the selected CAD Object(s) to the Windows Clipboard.

 Copy - This option will Copy the selected CAD Object(s) to the Windows Clipboard.

 Paste - This option will Paste the CAD Object(s) stored in the Windows Clipboard to the current View.

 Delete - This option will Delete the selected CAD object(s).

 Break - This option will allow you to Break a selected CAD object into multiple objects.

With Line Selected

With Circle or Arc Selected

There are three Break Modes:

Break into equal segments - Simply enter the Number of segments you need.

Break into pieces with length - This option will break the Object into segments of a Length specified; with an Object at the end equal to the remainder.

Break at location with length - This option will Break the Object from the Start Point at the Length Specified.

 Group - This option allows you to Group multiple CAD Objects together.

 Ungroup - This option allows you to Ungroup a Group of CAD Objects.

 Transform - There are four possible Transform options:

 Flip - Will Flip the selected CAD Object.

 Mirror - Will Mirror the selected CAD Object.

 Reverse - Will Reverse the selected CAD Object.

 Rotate... - Allows you to specify a Rotation for the selected CAD Object.

 Order

This option allows you to set the view level of the selected Object. When you add pictures, shapes, or other CAD objects drawing, they automatically stack in individual layers as you add them. You see the stacking order when objects overlap — the top object covers a portion of the objects beneath it. See illustration below.

You can move individual pictures, shapes, or other CAD objects or groups (multiple selections) of objects in a stack. For example, you can move objects up or down within a stack one layer at a time, or you can move them to the top or bottom of a stack in one move. You can overlap objects when you draw to create different effects.

There are four Order options:

To bring an object to the front, select the option Bring to Front .

To send an object to the back, select the option Send to Back .

To bring an object one step closer to the front, select the option Bring Forward .

To send an object one step toward the back, select the option Send Backward .

 Select All - The Select All menu option will select all CAD Drawn objects in the current View.

 Select None - The Select None menu option will deselect all of the CAD Drawn objects in the current View.

 Reverse Selection - The Reverse Selection option will deselect all the current Selections and in turn Select all the CAD objects that were not in the previous selection set.

 Line Widths... - The Line Widths option allows you to globally change all CAD Line Widths to a different Line Width.

 Properties - The Properties option opens the View Properties screen.
