Setup Reports

Report Groups may be created to contain Reports that are similar to each other OR Report Groups could be created to contain all Reports needed for a certain type of Job.

Creating a Report Group

When you click Setup Reports, you will see the following screen:

This is a list of the Advanced Bid Reports that are available in CABINET VISION. In the sidebar is a list of all of the Report Groups that have been created. To create a new report group, simply click the Add button and then enter the name that you want to use for this group. When you click OK the new group will be added to the list of Report Groups.

After you have created your new Report Group, you will need to begin assigning Reports to that Group. To add a Report to a Group, click on that Report and then click the check box to add a check next to the desired Report Group. As you scroll down the right side of your screen you will notice that some of the Reports have a green CV icon in front of them. This green CV icon lets you know that the Report has been assigned to a Group. If you click on that Report you will see a check mark next to one or more of the Report "Groups" in the sidebar. Reports can be assigned to more than one group if you like.

Ribbonbar

Sidebar

Groups

The Groups area in the Setup Reports sidebar allows you to view the currently available Report Groups.

Add a Report Group

The Add button in the Setup Reports sidebar allows you to create a new Report Group.

Delete a Report Group

The Delete button in the Setup Reports sidebar allows you to delete the selected Report Group.

You should remove all Reports from a Group before you delete the Group. This means you will have to click on every Report in the Report List that has or icons, and check to see if it is a member of the group you are preparing to delete.

Right Click Options

 

 Copy

Will Copy the selected Report giving you the ability to give it a new Title and Description.

 Delete

Will Delete the selected Report. Only available for User Reports.

 Edit

Will Open the selected Report in the Report Designer. If it is a System Report you will be prompted to Copy the Report first, as System Reports can not be edited directly.

 Properties

Opens the Report's general Properties.
