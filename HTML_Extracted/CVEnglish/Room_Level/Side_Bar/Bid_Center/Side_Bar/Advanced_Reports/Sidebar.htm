It is important to note that the Advanced Reports area will remember the last Printer you printed an individual Report too. This is a handy feature when printing a Report Group with Reports that need to go to different Printers. The Advanced Reports area will send each individual Report in the Group to the last Printer the Report printed to. It is important to note it will only remember the last Printer for that report when and only when Printing that Report from within a Report Group. When Printing it individual the Printer will always default to the last Printer used.
