Simple Reports

The Simple Reports sidebar button takes you to the Report Viewing/Editing area. These are the Reports you use to create a hard copy of your Bid either for interoffice use or for Bid Presentation to your prospective Customer.

Upon first entering the Report area you are presented with the Printable copy of the selected Report.

Following are the options available in the Bid Center Reports area:

Report Selector

This is where you select which Report is active.

Report Mode

There are two options for how to View your Report here:

View Mode

The default View for Reports.

Setup Mode

To customize your Reports you need to click the Setup Mode Report Mode option. You will immediately notice the look of the selected Report will change. Instead of showing the values it shows the variables that store and display those values.

There is one more change you will see in Setup Mode. There is a list of the Report Variables you can add to your Report in the sidebar. This list includes predefined variables; you can not create your own. See the Report Variables topic for more information.

If you need to perform calculations against variables you can do that as well by using the evaluate[] function. For example if I wanted to display the Number of Doors x 2 I would simply enter the following:

evaluate[ *2]

 

Open in Word

This option allows you to edit your Report in Microsoft Word. When first using this tool you will be prompted to browse to the location of the Microsoft Word executable.

Export to RTF

This option allows you to export the selected Report in a Rich Text Format.

New Report

This option allows you to create a new Bid Report from scratch.

Copy Report

This option allows you to make a copy of the current Report.

Delete Report

This option allows you to delete the current Report.

Font Selector

This option allows you to change the Font for the selected text in your Report.

Font Size

This option allows you to change the Font Size for the selected text in your Report.

Bold

This option allows you to change the selected text in your Report to Bold.

Italic

This option allows you to change the selected text in your Report to Italic.

Underline

This option allows you to change the selected text in your Report to Underline.

Left

This option allows you to change the selected texts Justification in your Report to Left.

Center

This option allows you to change the selected texts Justification in your Report to Center.

Right

This option allows you to change the selected texts Justification in your Report to Right.

Indent

This option allows you to Indent the selected text.

Outdent

This option allows you to Outdent the selected text.

Bullets

This option allows you to change the selected text in your Report to a Bulleted List.
