Filter

Allows you to control what Objects are actually included in the Bid. You can Filter by:

  • Room

  • Molding

  • Accessories

  • Assemblies

  • Or any combination of the above…

You can use the Arrow Keys on your keyboard to quickly go through the list and use the spacebar to toggle items on or off.

Select All will select all the listed items.

Select None will deselect all the listed items.

The Invert Selection button allows you to Invert your current Selection.

The Filter you select here is stored within the Job, so when re-entering the Bid Center or entering the Report Center your Filter remains active.
