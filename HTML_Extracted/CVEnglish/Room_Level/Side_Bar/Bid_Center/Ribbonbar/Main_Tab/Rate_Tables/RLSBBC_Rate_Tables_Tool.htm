Rate Tables

This option opens up the Rate Tables area. This is where you select the active Rate Tables as well as where Rate Tables are created.

What are Rate Tables?

First let’s get some of the terminology down so we don't get too confused during the rest of this topic. Rate Tables are basically a store house for a group of Bid Methods. You can create an unlimited number of Rate Tables and utilize them together in any way you want.

Selecting/Creating/Editing Rate Tables

To select/activate a Rate Table simply click on the check box next to it. You can use as many Rate Tables in as many combinations as you want. In the Rate Tables and What If's topic some techniques using multiple Rate Tables to create "What If" Bid Scenarios is covered.

To create or edit Rate Tables click on the Rate Table option in the Bid Center ribbonbar. This will open the Select Rate Tables area.

 To set the current selection of Rate Tables as your default for all new jobs and jobs without previsouly selected Rate Tables simply click the "Set As Default Rate Table" button here.

To edit an existing Rate Table simply highlight it and click the Setup Button. To create a new Rate Table just click the Setup Button. Once you have clicked on the Setup button the Setup Rate Tables area will open.

The Setup Rate Tables Interface

Toolbar Tools

 Rate Table Selector - Use this tool to select which existing Rate Table you would like to modify/view.

 Create New Rate Table - Use this tool to create a new Rate Table.

 Copy Current Rate Table - Use this tool to Copy the current Rate Table with a new Name.

Rename Current Rate Table - Use this tool to Rename the current Rate Table.

 Delete Current Rate Table - Use this tool to Delete the current Rate Table.

 Build A Sample Rate Table - This tool will actually create a sample Rate Table with some common Bid Methods predefined. Depending on how you Bid your Jobs this could save considerable time when setting up a Rate Table from scratch.

 Filter Out Custom Methods - This tool will suppress the display of Custom Bid Methods from the Method drop down. This is handy if you are using Stock Pricing.

 Create New Item Group - This tool allows you to create Item Groups in your Rate Table. Item Groups can be helpful for organizing and categorizing the Bid Methods you have selected within your Rate Table. In the figure below you can see that an Item Group named Installation was created and then under it there is a Cabs Item Group where any Installation charges related to Cabinets is stored. Then there is a Tops Item Group where any Installation charges related to Counter Tops is stored and so on. Although Item Groups don't hold any value they can be handy for organizing very large Rate Tables.

 Change Line Items Position - This tool allows you to change the position of the active Line Item to another physical location within the current Rate Table. This is simply and organization tool.

 Delete Current Line Item - This tool will Delete the currently selected Line Item within your Rate Table.

Line Item Area

There are six fields in this area:

  • Icon - This is an organizational tool you can set this icon to represent the type of Bid Method you have selected for the Line Item.

  • Name - Use this field to describe what the selected Bid Method is actually calculating.

  • Value - This is the Value that the selected Bid Method uses to make its calculations.

  • Method - This is the Bid Method you want this Line Item to use.

  • Markup - This is the Percentage you want to add to the final calculation of this Bid Method that represents your Markup.

  • Tax - This is the Percentage you want to add to the final calculation of this Bid Method that represents the Tax your Customer needs to pay for this Line Item.
