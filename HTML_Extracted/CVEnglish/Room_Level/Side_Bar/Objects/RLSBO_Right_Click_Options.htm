Right Click Options

After clicking the Objects button on the sidebar, you will see the Names (Headers) for your Object Catalogs. If you right-click on a Header, the following options will appear:

 Edit Header - This option returns the window shown in the picture below. This window gives information about the selected Catalog. Nothing in this window can be edited the information is there only for information about the Catalog. This information can be edited via the Catalog Editor. See the Catalog Editor Help file for more information.

 Move - This option allows you to move the Header for this Catalog to a different location on the sidebar.

The remaining Right-Click options are available when you right-click either on an Object or in an empty space on the sidebar. Each option will include in its description where it would be available.

 Add Category - (Right-Click on Item/Right-Click in Empty Space) - This option allows you to add another Category to the list of Objects. You will have the option to enter a label (name) for the Category and select an icon from the list on the right. Click OK to complete the process.

 Edit Item - (Right-Click on Item) - This option allows the user to edit the label and icon for the currently selected item.

If the item is an Assembly, the Edit Item option will open the Save Object dialog box for the Assembly.

 Move Item - This option allows the user to change the position of this item in the list.

 Delete Item - This option will remove the item from the list of objects.

 Add Accessory (Right-Click on Item/Right-Click in Empty Space)- This option allows the user to add a Non-Graphic Accessory (NGA) to the list of Objects. The user can then drag that Accessory out of the Object Catalog and add it to the Room or to another Object. When the user creates a new Accessory in this manner, they will see the window displayed below. In this window, the user may enter values for all of the parameters shown.

 Add to Favorites (Right-Click on Item) - This option allows the user to add the currently selected object to the Favorites Bar at the bottom of the screen.
