Auto Fill -

The following options appear in the sidebar after the Auto Fill button is selected.

There are three sections in the Auto Fill sidebar:

Fill Options

Mode - There are three Fill Mode options:

    • Space - Will fill between two Objects, as is the case when there is less than a full Wall to fill. Space can also be used to fill a Wall.

    • Wall - Will fill the entire Wall with Base or Wall (or both) Cabinets.

    • Room - Will fill all Walls within the Room.

Source - There are two Fill Source options:

    • Catalog - Pulls Objects from the selected Catalog. Will only use Objects specified as Auto Fill in Advanced Object Save Properties.

    • Custom - Uses Standard non-Catalog Base and Upper objects to fill the Room.

Catalog - Select the Catalog to Auto Fill with from the drop down list.

Ask Corner Type - This option is only enabled if you are using the Custom Source. This enables the Corner Options prompt covered later in this topic.

Cabinets

Modular Increments will cause Objects to seek the same size while holding the differences to the increment shown. This might result in an Object that defaults to 30 inches wide (as saved to your Catalog) and increments to 27 or 33 inches when placed. All Objects will be equal width on the same Elevation.

Shrink becomes an option if you choose a modular Catalog for your fill style. Choosing Yes allows those modular Cabinets to be re-sized to fill the space.

Base Corners - Allows you to choose the corner Cabinet type for Auto Fill.

Upper Corners - Allows you to choose the corner cabinet type for Auto Fill.

Room

Molding and Counter Tops become active options if you choose Room as the Fill Mode. Selecting Yes will cause CABINET VISION to include Counter Tops and Molding to the automatic Room.

Using Auto Fill

The Auto Fill feature provides a quick way to place multiple Cabinets on a Wall or Walls within a Room. This is most useful when you have long Elevations with many similar Cabinets. CABINET VISION does not treat Cabinets placed by the Auto Fill feature any differently than Cabinets placed individually, they are edited, stretched, etc. in the same manner as manually placed Objects.

The area shown below is available in the Layout level, by clicking File | Preferences | Layout Tab. You will notice the Clearance area below, these clearances will be followed by the Auto Fill feature, when used. The more important ones are the Floor to Upper clearances as well as the Refer Side Clearance. If the Refrigerator is placed prior to the Auto Fill feature being used, the clearances will be followed.

To use the Auto Fill feature you must first select a Cabinet as the default Base and Wall Cabinet. You are free to do this in each library, but, without so selecting, the Cabinet will not Auto Fill. Typically, a standard Base and standard Wall Cabinet are selected. Right Click one of these Cabinets from the Objects sidebar and select Edit Item.

The following "Save Object" window will appear.  Click the "Advanced" button on this window.

The following "Save Object Advanced Properties" window will appear.

In the bottom right corner of this window is a small box with the word "Auto-fill" beside it. If you want this Cabinet to be your default Auto Fill Cabinet, place a check mark here. Be sure you have not made any other changes to the menu and click OK. Click OK on the window behind the above window to return to the Room Level of CABINET VISION. Do the same with your Wall Cabinet. If you have more than one Catalog, you may select Cabinets in each Catalog to Auto Fill with, and, when using the feature, select the Catalog of your choice.

Corner Options

When Auto Filling a Wall or Space you will be prompted for how you want Cabinet Vision to handle the Corners. The window presented will be similar to this:

Following are the Corner Options:

  • Angled - Will Angle the Cabinet into the Corner.

  • Extended - Places an Extended End Cabinet in the Corner.

  • Gap - Leaves a Gap at the Corner equal to Cabinet Depth plus 12" (304.8mm) (calculated for face at angles).

  • Miter - Places a Miter End Cabinet in the Corner.

  • Return - Places a Return End Cabinet in the Corner.

  • Space - Leaves a Gap at the Corner equal to Cabinet Depth (calculated for face at angles).

  • To Wall - Runs the Cabinets to the Wall in the Corner.
