The Assembly Sheets Layout

There are four main sections of the Assembly Sheet Template that ships with CABINET VISION. To create your own Assembly Sheet Template see the Drawings Library topic.

Cut List Data Table (in Red)

This area of the Assembly Sheet displays the Cut List for your Assembly. This list also will include Hardware included in the Assembly as well as any Comments that have been added to the Assembly.

Container (in Green)

This area will display a graphic of the Primary Face of the Assembly. It also, at the top, includes the Assembly number, Name, Size, and Quantity.

Assembly Properties (in Blue)

This area displays general Assembly Properties.

When values for Scribes are displayed it is important to note that when an Interior Scribes is noted the value for the scribe will be in parenthesis.

Information Area (in Magenta)

This area displays general Job/Room Information.
