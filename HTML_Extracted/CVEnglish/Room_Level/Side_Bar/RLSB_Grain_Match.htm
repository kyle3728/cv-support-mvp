Working With Grain Match Groups

If you click a Group Header or any Assembly within a group the Group will be highlighted on the screen.

Using a Window for selecting Assemblies will create or modify a Group based on what is enclosed within it. Notice below that there are two Groups. Group 1 is the Base Cabinets and Group 2 is the Upper Cabinets. By enclosing all the Cabinets on the Wall in a Window, all Cabinets are placed within a single Group.

If you Right Click on an Assembly in Grain Match mode you have three possible options:

 

  • Create Group - If you Right Click on an Assembly which is not part of an existing group you can select to Create a new Group with it.

  • Add to Group - If you Right Click on an Assembly which is not part of an existing group with an existing group currently selected you can choose to Add the Assembly to the Group.

  • Remove Matching - If you Right Click on an Assembly that is currently Part of a group you can choose to Remove it from the Matching group.

You can also use + Click to Add or Remove Assemblies from a Group.

Auto-Group only Group's adjacent Assemblies. You, however, are not limited to Grouping only adjacent Assemblies.
