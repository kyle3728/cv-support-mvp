Object Tree

The Object Tree allows you to view and edit the parameters for any of the Objects in the current Room, Assembly, or Part.

You can quickly navigate the Object Tree with the arrow keys on your keyboard. Up and Down to move and Left and Right to collapse and expand.

If you right click an Operation in the Object Tree it will highlight all Operations related to the Operation within the Object Tree.

See the Intelligence Workshop for more information on working with the Object Tree.

Values can be entered as Inches (with Xin (12in)) or as Metric (with Xmm (300mm)).
