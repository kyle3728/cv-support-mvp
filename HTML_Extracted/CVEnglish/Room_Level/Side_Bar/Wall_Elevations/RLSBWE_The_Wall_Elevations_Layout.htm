The Plan & Elevation Sheet Layouts

 The Plan Sheets are similar to Elevation Sheets covered below with the only exception being there is no Assembly List.

There are three main sections of the Elevation Sheets that ship with CABINET VISION. To create your own Elevation Sheet Template see the Drawings Library topic.

Container (in Red)

This area of the Wall Elevation displays the Dimensioned Elevation View.

Assembly List Data Table (in Blue)

This area displays a list of all Assemblies on the Elevation along with some general Assembly Properties.

When values for Scribes are displayed it is important to note that when an Interior Scribes is noted the value for the scribe will be in parenthesis.

Information Area (in Magenta)

This area displays general Job/Room/Elevation Information.
