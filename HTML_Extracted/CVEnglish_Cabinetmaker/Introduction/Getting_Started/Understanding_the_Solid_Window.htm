Understanding the Cabinetmaker Window

When you start a new Job in Cabinetmaker, the main Cabinetmaker window opens. This window is your design work space. It contains elements that you use to create your designs and to receive information about them. The following illustration shows the main parts of the Cabinetmaker window.

File Menu

The File Menu is in the upper-left corner of Cabinetmaker. See the File Menu topic for more information.

Quick Access Toolbar

The Quick Access Toolbar is a toolbar containing a set of commands that are independent of the ribbonbar tab that is currently displayed. There are five quick commands available here depending on the Level of Cabinetmaker you are in. New, Open, Save, Undo, and Redo.

The Quick Access Toolbar can be located in one of two places:

  • Upper-left corner next to the Cabinetmaker Button  (default location)

  •  Below the ribbonbar

If you don't want the Quick Access Toolbar to be displayed in its current location, you can move it to the other location. If you find that the default location next to the File menu is too far from your work area to be convenient, you may want to move it closer to your work area. The location below the ribbonbar encroaches on the work area. Therefore, if you want to maximize the work area, you may want to keep the Quick Access Toolbar in its default location.

  1. Click Customize Quick Access Toolbar .

  2. In the list, click Show Below the Ribbon.

You can also use this option to minimize the ribbonbar. Or to quickly Minimize/Maximize the ribbonbar simply double click a ribbonbar tab.

Titlebar

This section displays the current Job Name as well as the Current Level (Layout/Assembly) of Cabinetmaker you are in.

Ribbonbar

This section gives quick and easy access to tools specific to the area of Cabinetmaker you are in. The ribbonbar will change depending on which area, you are in. Tools with small  triangles on the right side have fly-outs containing tools that invoke commands related to the first tool shown.

Minimize, Maximize, and Close Buttons

These buttons hide the Cabinetmaker window, enlarge it to fill the whole screen, and close it, respectively.

Style Selection

Allows you to choose the Style that the Cabinetmaker interface will utilize. There are four choices.

  • Colorful - The default Style

  • White

  • Dark Gray

  • Black

Area Tabs

This series of Tabs allows you to switch your View between several possible areas depending on the Level of Cabinetmaker you are in.

You can right click on the Tabs and select Hide Labels if you do not need them.

Sidebar

This section gives access to functions specific to the area of Cabinetmaker you are in. This section will change depending on which area, controlled by the Area Tabs, you are in.

View/Drawing Area

This is the actual working area of Cabinetmaker for Drawings or the display of other information. The drawing area size varies, depending on the size of the Cabinetmaker window and on the number of other elements that are displayed.

Statusbar

This bar contains your software license information.

Favorites Bar

This is a storage area for quick access to commonly used Objects. To add an Object to this bar simply drag the Object out of your Catalog and instead of dropping the Object into the Layout Area simply drop it onto the Favorites Bar. Up to 32 Objects can be placed on the Favorites Bar.

Statusbar Menu

The Statusbar menu gives you quick access to control the status of certain Drawing aids. There are six possible options here:

  • Unit Selection or - Allows you to toggle between Inches (Imperial) and Metric Units on the screen. The Unit displayed on the button represents the Unit of Measure that is currently active. You can turn this button On/Off via the Preferences | Units tab.

  • Unit Precision - Allows you to select the Unit of Precision for the selected Unit. The Precision of the Unit selected will be dictated by the Unit of precision last Set As System Default for that Unit and View in Units Properties.

  • Reset Ruler - Allows you to remove any Snap Marks you have placed.

  • Snap to Marks - This feature allows snapping to a Snap Point set by use of the Ruler Tool. If this button is highlighted in blue it means it is active simply click it again to disable it.

  • Snap Grid - This option turns the Grid display on. You will see a grid pattern in the window where you layout Jobs, edit Assemblies, or CAD that you can Snap to. If this button is highlighted in blue it means it is active simply click it again to disable it. Go to Right Click Properties to set the Grid Spacing.

  • Snap to Points - This is perhaps the most useful tool when using the measure option of the Ruler Tool. Using this snap feature allows precise snapping of dimension lines in CAD, and other precise placement options. This option turns on the Dynamic Snapping functionality in Cabinetmaker The snap to points appear as small dark dots in the corner of the objects and all parts within the object. These are the snap to points. If these points are present and not desired, reset the snapping mode to OFF or another mode to eliminate them. If this button is highlighted in blue it means it is active simply click it again to disable it.

  • Build Warnings - Will display any Build Warnings that are currently active. Simply click this option to bring up the Message Log to view the active Build Warnings.
