Backup Utility

Backup List Tab

This tab allows you to Back Up your databases. There are several options here:

Construction Methods - Checking this option will Back Up all Cabinet and Drawer Construction Methods. This will back up the CVData (SQL Database).

Part / Material / Door Catalog – Checking this option will Back Up all Materials, Material Schedules, Doors, and Parts.This will back up the CVData (SQL Database).

Report Database – Checking this option will Back Up all Report Group and Bid Rate Tables. This will back up the file report.mdb.

CNC Database - Checking this option will Back Up the Tool and Machine catalogs. This will back-up the S2M CENTER file psnc.mdb (See the S2M CENTER Help file for more information).

Custom Cabinet Library - Checking this option will Back Up the Custom Cabinets Library. This will back up the file custom cabinets.cvc.

Generic Cabinet Library - Checking this option will Back Up the Generic Library. This will back up the file generic.cvc.

Drawings – Checking this option will Back Up all CAD Symbols and Title Blocks. This will back up the file drawings.sym.

Posts - Checking this option will back up all post files that you use to output to CNC.

Others

  • Any items in the other list with the . CVC extension are additional Assembly Catalogs that are available for Back Up.

  • Default.dat is the file that contains Layer information and custom settings.

  • .CFG files are your Job Schemes.

Add Job - Clicking this button will allow you to attach a Job to your Back Up. This is especially handy when sending a set of Databases to Technical Support. You can attach a Job so the Technician sees exactly what you see on the screen.

Toolbar

There are three tools in the Backup Utility toolbar.

Email Backup - With this option selected once the backup is complete, Cabinetmaker will Zip it up and open a new email message (If you are using Microsoft Outlook) with the back up attached. The Backup can then be sent to Technical Support.

Delete Backup - This option you can delete old backups that either you or a CABINET VISION Install has created. Deleting old backups (Archives) will allow you to free up hard-drive space.

 Settings - With this option you can setup automatic deletion of backups (Archives) as well as establish the destination of the Backup files.
