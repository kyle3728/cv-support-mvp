General

The following diagram shows the options available in the General object.

Drawer Box Cost (Standard #137)

The price in this box will be added to the Bid for each box (Drawer or Roll Out) that has been included in the Job.

Drawer Box Labor (Standard #93)

The time (in minutes) in this box will be added to the Labor Estimate.

Height Specifications (Standard #1)

You have two choices for how the Box Height is calculated.

  • To Fit Using Guide Clearance - This option will look at the Drawer Guide in use on the Box and calculate the Height based on the Opening Height where the Drawer is placed minus the Top and Bottom Clearance set on the Drawer Guide in the Material Catalog.

  • Standard Heights - This option will allow you to establish a series of Standard Heights. When this option is selected the next screen will be Standard Heights (#2).

Standard Heights (Standard #2)

This area allows you to list all the possible Standard Heights for the Box. The Heights must be entered from Largest to Smallest. There is no limit to how many Standard Heights you can define. CABINET VISION will choose from that list and build the largest Drawer Box that will fit the opening minus the value set by Top and Bottom Clearance for the Drawer Guide in use. If it happens that the smallest Standard Height is still too large, being that it is the last available size on the list, CABINET VISION will default to the actual required size for the Drawer Box Height. See the Drawer Box sizing topic for more information.

Depth Clearance (Standard #5)

This option allows you to specify the "Minimum Distance Required Between the Drawer Box/Roll Out Back and the Face of the Assembly Unfinished Back"

Depth Specification (Standard #6)

You have two choices for how the Box Depth is calculated.

  • To Fit Using Depth Clearance - This option will look at the value previously established in "Standard Heights (#2)" and subtract that value from the actual Opening Depth where the Drawer is being placed to come up with the exact Drawer Box Depth.

  • Standard Depths - This option will allow you to establish a series of Standard Depths. When this option is selected the next screen will be Standard Depths (#7).

Standard Depths (Standard #7)

This area allows you to list all the possible Standard Depths for the Box. The Depths must be entered from Largest to Smallest. There is no limit to how many Standard Depths you can define. CABINET VISION will choose from that list and build the largest Drawer Box that will fit the opening minus the value set in "Depth Clearance (#5)". If it happens that the smallest Standard Depth is still too large, being that it is the last available size on the list, CABINET VISION will default to the actual required size for the Drawer Box Depth. See the Drawer Box sizing topic for more information.

 

Bottom

The following diagram shows the options available in the Bottom object.

Drawer Box Bottom Banding Definition (Standard #150)

Drawer Box Bottom Grain Direction (Standard #9)

You have two options for the Grain Directions on Bottoms:

  • Front to Back

  • Left to Right

Drawer Box Bottom Scribe (Standard #10)

You have two options for the Box Bottom Scribe:

  • Flush to Box Bottom

  • Space - A positive number represents how far up your want the Bottom to be from the bottom of the Box. A negative number represents how far down from the bottom of the Box you want the Bottom to be located.

Drawer Box Bottom Machining Face (Standard #129)
