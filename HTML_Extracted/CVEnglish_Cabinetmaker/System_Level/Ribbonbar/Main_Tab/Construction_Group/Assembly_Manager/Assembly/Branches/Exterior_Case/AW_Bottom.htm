Deck

Typically, "Deck" is reserved for the 'bottom' of a Base Cabinet, where the 'bottom' of an Upper Cabinet is referred to as the Bottom.

The following diagrams show the options available in the Bottom Branch.

Bottom/Deck Banding Definition (Standard #676)

Bottom/Deck Placement to Case (Standard #586)

Deck Bottom Rail Position (Standard #135)

Notch Deck for Sub-Frame (Standard #9)

Delete Bottom/Deck On Drawer Bank (Standard #12)

You have the option to set up your Construction Method to remove the Deck from a Drawer Bank Cabinet.

Bottom/Deck Finished Side Orientation (Standard #136)

For Bottoms/Decks in a Corner sub-branch, you have the ability to determine the Material face orientation if you are using 2 sided Materials for the Bottom of a Cabinet. You can select whether the "good side" faces in or out (up or down on the Bottom). Remember to add break outs if needed for the different Classes of Cabinets.
