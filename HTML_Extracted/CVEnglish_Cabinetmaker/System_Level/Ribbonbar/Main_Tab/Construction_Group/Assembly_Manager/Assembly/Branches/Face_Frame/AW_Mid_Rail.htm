Mid Rail

This option sets Mid Rail options, it sets the dimensions as well as joinery conditions.

This Branch also allows you to establish the Top opening Height for Top Drawers for a Face Frame Style Construction Method. To set this value for a Frameless Style see the Reveals Branch.
