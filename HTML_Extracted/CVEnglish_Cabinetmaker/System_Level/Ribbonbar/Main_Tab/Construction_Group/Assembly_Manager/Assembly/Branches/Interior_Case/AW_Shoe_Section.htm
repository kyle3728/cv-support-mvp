Shoe Section

This option sets Shoe Section options. The Shoe Section object is automatically joined together using egg-crate style joinery.

 Regardless of how you answer these questions, you will not see the Shoe Section in a Cabinet until you edit the Cabinet and place one there, and save the modified Cabinet back to your library.

The following diagram shows the options available in the Shoe Section Branch.

Vertical Divider Banding Definition (Standard #680)

Horizontal Divider Banding Definition (Standard #681)

Shoe Section Minimum Height (Standard #530)

Shoe Section Minimum Width (Standard #531)

Notch Extension for Horizontal and Vertical Notches in Shoe Section (Standard #548)

Machining Face at Horizontal Divider (Standard #637)
