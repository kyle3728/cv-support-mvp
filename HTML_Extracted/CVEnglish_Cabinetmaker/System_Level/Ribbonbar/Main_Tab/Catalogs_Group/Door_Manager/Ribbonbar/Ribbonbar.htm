Main Tab

 Return

Closes the Door Manager and returns you to the CABINET VISION Opening Screen.

 New

Starts the New Door Wizard to create a new Door.

Give your Door a Name and Description. Select the Type of Door; Slab or Multi Piece. Add Tags (to aid in Searches), which can be separated by a space or a comma, and Notes. Select whether you are designing a Door or Drawer Front (this simply sets a realistic initial sample size) then click Next.

Slab

Following are the options on the Setup Door screen for a Slab Door:

Route Path - Style

If the Slab Door you are defining has a Route Path select the Style here. Following are the Style options:

Shape - Radius

Specifies the Radius if the Radius or Inv Radius Corners Route Paths are selected.

Shape - Chamfer

Specifies the Chamfer size if the Chamfer Route Path Style is selected.

Spacing - Left Stile

Specifies the Left Stile Spacing if a Route Path Style other than None is selected.

Spacing - Right Stile

Specifies the Right Stile Spacing if a Route Path Style other than None is selected.

Spacing - Top Rail

Specifies the Top Stile Spacing if a Route Path Style other than None is selected.

Spacing - Bottom Rail

Specifies the Bottom Stile Spacing if a Route Path Style other than None is selected.

 

Multi Piece

Following are the options on the Setup Door screen for a Multi-Piece Door:

Panel - Type

Select from the following Panel Types:

  • None - Most commonly used if your Door has no Panel and consists only of Stiles and Rails.

  • Raised – most commonly used for solid lumber.

  • Flat – most commonly used for thin plywood or thin panel stock.

  • Glass – most commonly used for glass.

  • H Slats – most commonly used for tambour Doors with the slats going left and right. If you never use this type of panel you could assign it as cloth or some other Material that you do use.

  • V Slats – most commonly used for tambour door with the slats going up and down. If you never use this type of panel you could assign it as cloth or some other Material that you do use.

  • Slab – most commonly used for thick plywood or thick panel stock.

Panel - Panel Inset

Panel Inset is the distance you want the Door panel to stick into one side of the Door.

Panel - Hor Slat Width

This establishes the default "Center-to-Center" spacing for Horizontal Slats when the Horizontal Slat Panel Type is defined for your Door.

Panel - Ver Slat Width

This establishes the default "Center-to-Center" spacing for Vertical Slats when the Vertical Slat Panel Type is defined for your Door.

Frame Stile and Rail Widths

The value entered in these field represents the portion of the Stile or Rail as it exists without any Inside Profile on it. The dimension of the Inside Profile, optionally depending on the 'Extend Part' Property for the Inside Edge Profile, is in addition to the Stile or Rail width. If there is no Inside Profile, then the value set as the default width for the Stile or Rail is the net width of the Stile or Rail.

If you highlight a Stile or Rail in the Section area of the Door Manager, you will see the Stile or Rail width, and an "Extra Width". Remember, the overall width of the Stile or Rail is the default value, PLUS the Profile for the Inside Edge, if any.

You will note that in the graphic above that the Left Stile, Right Stile, Top Rail, and Bottom Rail all preset to 2 1/8”. In the graphic below the Left Stile of a Door is highlighted. If this Door were being produced, the finished width of the Left Stile would be 2 ½”. You will notice that the width of the Stile says "Width" + "Extra" these two measurements must add up to the overall width of the finished Stile. Since this Door is using a 3/8” Inside Edge Profile, setting the Width to 2 1/8” (2 ½ - 3/8 = 2 1/8) under "Default Rail & Stile Widths" will automatically fill in the "Width" portion of the "Width" + "Extra" with the 2 1/8” needed.

You will note that the Mid Rail and the Mid Stile in this scenario should be set to 1 ¾”. This is because to achieve a 2 ½” finished size you have to take into account the 3/8" Inside Edge Profile on both sides of the Mid Stiles and Rails.

2 x 3/8 IEP = ¾

2 ½ - ¾ = 1 ¾

Frame - Miter

The Miter option gives you the ability to choose whether or not your Stiles and Rails Miter.

 Door Properties

 Catalog Properties

You can set the follow main Properties on the currently selected Catalog here:

  1. Name - The Name of the Catalog.

  2. Description - The Description of the Catalog.

Construction Defaults are simply the default settings for any new Door Style that you create. Changing these settings will not have any effect on existing Doors; individual Doors must be modified individually.

This area will allow you to set up your normal, or default Construction Defaults for all of your Doors. This will allow you to set up your Door Styles quickly, especially if your Stiles and Rails, for the most part, are the same size for most of your Doors. Here you have the opportunity to set the fields to their most common values. Typically, a default setting is chosen because it is the one used more often than not. Creating new Door Styles and redoing old Doors is easier if you have your Door Construction Defaults preset.

  1. Frame Stile Width - The Default Frame Stile Width used when creating a new Door in this Catalog.

  2. Mid Stile Width - The Default Mid Stile Width used when creating a new Door in this Catalog.

  3. Frame Rail Width - The Default Frame Rail Width used when creating a new Door in this Catalog.

  4. Mid Rail Width - The Default Mid Rail Width used when creating a new Door in this Catalog.

  5. Joint Adjustment - Joint Adjustment is typically an adjustment only necessary when doing Mortise and Tenon type Doors. Do not use this setting if your Door Construction Method is cope and stick or butt joint. When doing a Mortise and Tenon Door you would the Joint Adjustment of the Mortise and Tenon before creating a Mortise and Tenon type Door. Also make sure you do not select an Inside Edge Profile to use with a Mortise and Tenon Door, as CV could add the Joint Adjustment and the Inside Edge Profile to your Door Parts. After the Door is created come back here and zero out the Joint Adjustment value.

  6. Mitered Frame - Sets whether new Doors default to being Mitered or not.

  7. Panel Type - The Default Panel Type used when creating a new Door in this Catalog.

  8. Panel Inset - The Default Panel Inset used when creating a new Door in this Catalog.

  9. Vertical Slat Width - This establishes the default "Center-to-Center" spacing for Vertical Slats when the Vertical Slat Panel Type is defined for your Door in this Catalog.

  10. Horizontal Slat Width - This establishes the default "Center-to-Center" spacing for Horizontal Slats when the Horizontal Slat Panel Type is defined for your Door in this Catalog.

 List

Changes the main screen view to a List View.

This view allows the viewing and editing of the following Door Properties:

  1. ID (not editable)
  2. Name
  3. Description
  4. Unit Cost
  5. Cost/SqFt
  6. Tags
  7. Note

 Icon

Changes the main screen view to an Icon View.

If the Door Icons are not visible or are not up to date go to Utilities | Generate Icons.

 

Utilities Tab

 Return

Closes the Door Manager and returns you to the CABINET VISION Opening Screen.

 Generate Icons

This option will regenerate the icons that represent your Doors. Use this utility if your icons appear incorrectly. The Door Icons are used to display a sample graphic of the Door on the Door Manager main screen.

 Assign Materials

The Assign Materials Utility can be used to add new or existing Material Schedules to selected Doors within a Door Catalog. Simply select the Material Schedules on the left and the Doors to apply them to on the right.You can also assign the Costing to the Doors at the same time.

 Assign Profiles

The Assign Profiles Utility can be used to add new or existing Profiles to selected Doors within a Door Catalog. Simply select the Profile Type and Profiles on the left and the Doors to apply them to on the right.You can also assign the Costing to the Doors at the same time.
