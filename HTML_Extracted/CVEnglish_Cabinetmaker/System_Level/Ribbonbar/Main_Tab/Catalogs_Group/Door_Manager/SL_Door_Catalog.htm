Door Manager

Doors, Drawer Fronts, Panel Ends, and False Fronts are created in the Door Manager. Materials are assigned to Doors and Drawer Fronts here.

Doors are not sized here, some Door parts  (Stiles, Rails, Panels) are configured and have dimensions set, but the overall size of a finished Door is not set here.

There are three main areas in the Door Manager:

Ribbonbar

Sidebar

There are two panes in the Material Manager sidebar:

Search Pane

At the top of the Search Pane is a quick Search. Simply enter your keywords and the Quick Search will Search through Material Names, Material Descriptions, and Material ID's to bring up Materials containing them. To activate the Quick Search after typing your Search terms press  on the key board or click the Filter button. You can also specifically search for any Tags that are on your Doors by utilizing #. So if you have a Tag named Arc then use #Arc to bring up all Doors that have Arc as a Tag.

Since the Search mechanism uses partial word Searching there is probably no need to use wildcards however they are available. The Search uses standard wildcards and/or typical SQL Search wildcards:

*           Multi-Character

?          Single Character

The search always pads the search text with *'s:              *search text*

The lower section of the Search Pane displays any Saved Searches you have created. To create a Saved Search click the Add Search button. The Search window will display.

 The Add Search button can also be used to open the Search window to modify existing Saved Searches.

Click the Add Search tool in the toolbar to create a new Search. Click New in the toolbar and give it a Name. In the example we are going to create a Saved Search to show any Door Names containing the Arc.

The first line will Search On "Name" Compare using "similar to" Compare to "Arc" Group by "or".

The second line will Search On "Description" Compare using "similar to" Compare to "Arc" Group by "none".

Click OK and Yes to save changes and you will see your Saved Search in the list.

To activate it simple select it from the list.

Group Pane

The Group Pane allows you to easily organize your Materials into an unlimited number of Groups and Sub-Groups. Simply click the Create Group button to create a new Group or Sub-Group.

To Delete an empty Group or Sub-Group simply select it and click the Delete Group button.

Moving Groups and Sub-Groups can be easily accomplished using either Drag and Drop or by selecting the Group/Sub-Group and clicking the Move buttons.

To place Doors in a different Group simply select them and use Drag and Drop to put them in the selected Group.

You can also right click in the Group Pane to Create, Rename, or Delete a Group.

 

Main Screen

This is the area where you select the Door you wish to work on.

Right Click Options

Following are the options available when right clicking a Door in the Door Manager:

 Copy

Makes a Copy of the currently selected Door Style.

 Delete

Deletes the currently selected Door Style.
