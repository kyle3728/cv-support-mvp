Finish

The Finishes utility allows you to add new or edit existing Finishes.

Toolbar

The toolbar has the following options:

Finish Selector - Allows you to select an existing Finish to edit.

 Search - Allows you to Search for a specific Finish.

New - Allows you to create a New Finish.

Copy - Copies the currently selected Finish.

Delete - Deletes the currently selected Finish.

Creating/Editing a Finish

You can edit the following Finish Properties here:

  • Name - The Name of the current Finish.

  • Description - The Description of the current Finish.

  • Color Picker - Click anywhere within the Color Palette to select a starting color for your Finish.

  • Red - The value of the Red tone in the Finish.

  • Green - The value of the Green tone in the Finish.

  • Blue - The value of the Blue tone in the Finish.

The slider bars below the three color swatches can be manipulated in combination to produce any of 16 million colors. These colors are formed using a combination of Red, Green, and Blue. You may also enter the 0-255 number below the bar to achieve a known color once you establish the number code.

  • Luminance - Allows you to change the Luminance of the current Finish.
