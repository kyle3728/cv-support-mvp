Sidebar

There are two panes in the Material Manager sidebar:

Search Pane

At the top of the Search Pane is a quick Search. Simply enter your keywords and the Quick Search will Search through Material Names, Material Descriptions, and Material ID's to bring up Materials containing them. To activate the Quick Search after typing your Search terms press on the key board or click the Filter button.

The lower section of the Search Pane displays any Saved Searches you have created. To create a Saved Search click the New Saved Search button. The Search window will display.

The New Saved Search button can also be used to open the Search window to modify existing Saved Searches.

Click the New tool in the toolbar to create a New Search and give it a Name. In the example we are going to create a Saved Search to show any Material Names or Material Descriptions containing the word Oak.

The first line will Search On "Name" Compare using "similar to" Compare to "Oak" Group by "and".

The second line will Search On "Description" Compare using "similar to" Compare to "Oak" Group by "none".

Click OK and you will see your Saved Search in the list.

To activate it simple select it from the list.

Group Pane

The Group Pane allows you to easily organize your Materials into an unlimited number of Groups and Sub-Groups. Simply click the Create Group button to create a new Group or Sub-Group.

To Delete an empty Group or Sub-Group simply select it and click the Delete Group button.

Moving Groups and Sub-Groups can be easily accomplished using either Drag and Drop or by selecting the Group/Sub-Group and clicking the Move buttons.

To place Materials in a different Group simply select them and use Drag and Drop to put them in the selected Group.

You can use standard Windows selection to Move (or Delete) multiple Materials from one Group to another.
