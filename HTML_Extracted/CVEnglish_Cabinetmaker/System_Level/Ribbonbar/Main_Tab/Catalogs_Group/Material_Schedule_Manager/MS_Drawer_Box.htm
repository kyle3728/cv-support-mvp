Drawer Box

Drawer Box Materials are assigned to the various Drawer Parts via the Drawer Box Material Schedule. The image below defines the Materials necessary to build Drawer Boxes from 1/2 inch Melamine.

Sidebar/Properties

The following Properties are available for Drawer Box Parts:

  • ID - The ID number for the selected Material Schedule.

  • Name - The Name of the selected Material Schedule.

  • Description - The Description of the selected Material Schedule.

  • Buy Out - Enable this option if you Buy-Out the Drawer rather than building it yourself. This will tie in to your Reports (No Parts for Cut Lists) and Bids (No Material Calculation for Drawer Parts).
