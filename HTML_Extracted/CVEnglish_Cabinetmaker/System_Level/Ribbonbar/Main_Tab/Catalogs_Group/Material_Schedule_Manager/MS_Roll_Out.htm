Roll Out

Roll Out Materials are assigned to the various Roll Out Parts via the Roll Out Material Schedule. The image below defines the Materials necessary to build Roll Outs from 1/2 inch Melamine.

Sidebar/Properties

The following Properties are available for Roll Out Parts:

  • ID - The ID number for the selected Material Schedule.

  • Name - The Name of the selected Material Schedule.

  • Description - The Description of the selected Material Schedule.

  • Buy Out - Enable this option if you Buy-Out the Drawer rather than building it yourself. This will tie in to your Reports (No Parts for Cut Lists) and Bids (No Material Calculation for Roll Out Parts).
