Room

The Rooms Material Schedule assigns Materials from your Material Manager to System Components or user defined components that utilize these "System" Parts. The selected Materials are used primarily to assign a Finish to System components used within your Job. This is primarily for Rendering purposes.

Sidebar/Properties

The following Properties are available for Room Parts:

  • ID - The ID number for the selected Material Schedule.

  • Name - The Name of the selected Material Schedule.

  • Description - The Description of the selected Material Schedule.
