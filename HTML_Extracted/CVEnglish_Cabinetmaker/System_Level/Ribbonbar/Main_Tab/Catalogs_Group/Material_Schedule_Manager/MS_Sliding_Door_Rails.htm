Sliding Door Rails

The Sliding Door Rails Material Schedule allows selecting Sliding Door Rails and Rollers with different properties for different applications in your Assemblies.

Sidebar/Properties

The following Properties are available for Sliding Door Rails:

  • ID - The ID number for the selected Material Schedule.

  • Name - The Name of the selected Material Schedule.

  • Description - The Description of the selected Material Schedule.

  • Face Inset - The inset distance from the Face of the Cabinet to the Face of the forward-most Door.

  • Door Gap - The Gap between the Doors.

  • Door Top Clearance - The Gap above the Door.

  • Door Bottom Clearance - The Gap below the Door.
