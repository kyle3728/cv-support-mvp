Hardware

The Hardware Tab at Cabinet Properties has the following options:

  • Pull Schedule - Allows you to specify the Pulls Material Schedule to use with the active Assembly.

  • Drawer Guide Schedule - Allows you to specify the Drawer Guides Material Schedule to use with the active Assembly.

  • Hinge Schedule - Allows you to specify the Hinges Material Schedule to use with the active Assembly.

  • Beaded Frames Bead Profile - This option is only available if a Hinge Schedule utilizing an Inset Hinge is selected. This option allows you to specify the Bead Profile to utilize if doing a Beaded Face Frame Construction. If you are not doing a Beaded Face Frame Construction do not select a Bead Profile.

  • Sliding Door Rail Schedule - Allows you to specify the Sliding Door Rails Schedule to use with the active Assembly.

 

Remember changes made here will only affect the current Assembly. Also, changes made here are attached as permanent (unless you remove them) overrides on the Assembly; which means that if you Save this Assembly into a Catalog these overrides will also be Saved.
