Right Click Part -

     

Following are the options when right clicking on a Part in the Face, Plan, and End views:

Copy

This option will copy the selected Part and allow you to place the copied Part somewhere else within the Assembly.

Edit Shape

The Edit Shape option will bring up the Shape Editor to allow you to shape the Part.

Selecting a Point on a Parts Shape

When selecting a Point on an Part's shape you are presented with the following sidebar options:

Start X - Allows you to control the X Position of the Point. You can also activate "Simple Constraints" on the Point. See the Simple Constraints area of the Shape Editor topic for more information.

Start Y - Allows you to control the Y Position of the Point. You can also activate "Simple Constraints" on the Point. See the Simple Constraints area of the Shape Editor topic for more information.

Selecting a CAD Entity on a Part's Shape

When selecting a CAD entity on an Assemblies shape you are presented with the following sidebar options:

Start X - Allows you to control the Starting X Position of the selected CAD entity. You can also activate "Simple Constraints" on the Point. See the Simple Constraints area of the Shape Editor topic for more information.

Start Y - Allows you to control the Starting Y Position of the selected CAD entity. You can also activate "Simple Constraints" on the Point. See the Simple Constraints area of the Shape Editor topic for more information.

End X - Allows you to control the Ending X Position of the selected CAD entity. You can also activate "Simple Constraints" on the Point. See the Simple Constraints area of the Shape Editor topic for more information.

End Y - Allows you to control the Ending Y Position of the selected CAD entity. You can also activate "Simple Constraints" on the Point. See the Simple Constraints area of the Shape Editor topic for more information.

Length - Allows you to control the Length of the selected CAD entity.

Angle - Allows you to control the Angle of the selected CAD entity.

Miter - Angle - Allows you set the Miter Angle for the selected edge.

Relate To Part

The Relate to Part option will allow you to add intelligence to a new or existing Part and relate the selected Parts properties to an existing Part within your Assembly. In the following Exercise we will add Intelligence to a copy of the Top Front Stretcher relating it to the Top Front Stretcher. This Part could be used as a Sink Skirt.

  1. Select Top Stretcher to be the "Relate To" Part then select Part Length + 0.

  1. Select the X Position to be Part X Position + 0, the Y Position to be Part Y Position + 0, and the Z Position to be Part Z Position + 0.

  1. Select the X Axis Rotation to a Value of 180, the Y Axis Rotation to a Value of 0, and the Z Axis Rotation to Part Z Rotation + -90.

The added Part is now Intelligent and as long as the Top Stretcher it is referenced to exists it will as well.

Edge Banding

The Edge Banding option allows you define which edge of a Part gets edge banded with which edge band type. Once the Edge Band option is selected simply select an edge of the Part and the sidebar will change similar to the image below.

It is important to note that the default values in the sidebar is NOT representative of the current Edge Banding on the selected Part. To use this function simply select which Edge Banding type you would like and whether to Trim the Part for the edge banding or not. It is also important to note that ALL system edge band is applied with Trim Part set to yes. So, it would probably be desirable in most cases to set this option to Yes.

Simply select the Banding and Construction options for the selected edge. In the image below the Front Edge of the End has been selected.

Note that the front edge is now Red which indicates an Exterior Edge Band has been applied. Once you are through defining the Edges simply click the Return ribbonbar option to save the new Edge Properties.

Properties

There are six tabs in the Part Properties screen:

  • General Tab - Following are the options on the General Tab:

    • Description - By Default, the Description for the Part will come from the Part Manager and is what is displayed in your Reports. However, this field is editable and you may change the Description to better describe "this" Part in your Reports.

    • Material Change - By Default, the Material for the Part comes from the Material Schedule for the Part Group that the Part is located in.

    • Texture Map - Allows you to set how a Texture is applied to the Part.

    • Smooth Surface - Allows you to enable Smooth Surfacing on the Part.

  • Comment Tab - You add special notes on this Part here. Certain cut lists from the Report Center will display this information for the shop.

  • Finish Tab - The Finish Tab allows you to specify what Finish and/or Texture is applied to the selected Parts Front Face, Back Face, Edges, and Ends.

  • Overrides Tab - The Overrides Tab will display and allow removal of any Overrides on the selected Part. Since the Sink Skirt we defined in the Relate to Part topic above was selected we can see several overrides on the Part as shown below.
