Transferring Your Standards

NOTICE: This Topic assumes you are familiar with File Management in the Windows Operating System.

Follow these instructions to transfer files into Cabinetmaker 11 from previous versions or different computers.

The local hard drive is being referred to as "C" since that is the most common drive designation, however, it simply represents the drive where Cabinetmaker has been installed. If Cabinetmaker has been installed on a different drive on your computer, then that is the one you will use.

This topic describes the procedures for a standalone computer, but it will be the same if you are using a network server.

Step 1:

Open the Cabinetmaker Backup Utility. From the Cabinetmaker opening screen, click Utilities and then Backup Utility.

Step 2:

At a minimum you should "Include" what is in the "Include" area in the illustration below.

Click OK.

When the backup utility is finished running you must CLOSE Cabinetmaker.

Step 3:

This creates a folder that contains each of these selected files plus a zipped file called "backup.zip" which contains all the files in a compressed format. This new folder will be labeled with today’s date and time and will be placed within the Automatic Backups folder. This folder is located on your hard drive in the following location:

C:\ProgramData\Hexagon\CABINET VISION\CV 2021\Automatic Backups

Now we need to transfer this newly created backup folder to the Automatic Backups folder for the destination computer.

If you are transferring these files from one computer to another, simply copy and paste this backup folder to a USB memory stick or whatever method of transfer you prefer, then copy and paste it to the same location on the new computer.

At this point, make sure that Cabinetmaker is closed.

Now that the new folder has been copied to the Automatic Backups folder, follow these steps:

  1. On your computer, go to the Start button.

  2. Go to All Programs Cabinetmaker \ Restore Settings. The following screen will appear:

  1. The screen that pops up shows the available Backup folders on the left and the Files within each folder on the right.

  2. Make sure you have the folder you just transferred highlighted on the left.

  3. Now check which (or all) files you want to restore (transfer) on the right.

  4. Click OK

This will copy the selected files into the appropriate Database folder and run the appropriate update routine.  Once the update routine has finished running, Cabinetmaker is now ready for use and will contain the updated information.

Moving Existing Jobs

If you want to also move your existing jobs, you can simply copy and paste all the existing job files (.cvj files) from one computer or folder location to another for use in Cabinetmaker. These job files are usually contained within the C:\Users\Public\Documents\CABINET VISION\CV 2021\ folder, but they may be stored in any folder you have designated.

 A job that is edited in the latest version Cabinetmaker can no longer be used in an earlier version. Use a copy of that original job for safety. No update routine has to be run after copying these files.

Door Manager

Now that a new Door catalog has been copied over, you will need to update the Door icons. Open Cabinetmaker, go to Door Manager \ Utiltities and run Generate Icons.

S2M Center

If you use the S2M CENTER, you will have to update these items:

  1. Reselect the Primary and Secondary machines you use for Optimizing.
  2. Create a new Output Directory (path) for your output code folder.
  3. Reselect the path to your Post for each machine you use.
  4. Re-enter your Post code if applicable

Usability of new files

While we make our best effort at making the older files compatible with the newer version of our software, we can’t guarantee all files will work exactly as they did in the older version due to the updates and additions to the new version. Existing jobs may not perform as expected. Also, the Construction Methods and Door designs will have to be reviewed to account for new or updated routines. Please test the new information thoroughly before beginning production.
