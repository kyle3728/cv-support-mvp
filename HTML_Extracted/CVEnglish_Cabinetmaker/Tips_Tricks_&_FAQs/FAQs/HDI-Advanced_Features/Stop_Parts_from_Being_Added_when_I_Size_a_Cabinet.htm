Stop Parts from Being Added when I Size a Cabinet

Sometimes, when I stretch a Cabinet, unwanted parts are added such as, partitions, drawer stretchers, and Doors. How can I keep the Cabinet the same when it stretches?

Go to File Menu | Preferences | Layout Tab and Set your Maximum Door size setting to a higher number (ex. 90) to keep Cabinetmaker from putting a pair of Doors and mid-rails into Cabinets as they stretch.
