Smooth My Renderings

How do I enhance 3D Views and Renderings?

When in 3D view, select the Enhance option in the ribbonbar.

To control the Enhance features right click on the screen and select Properties then go to the CV Render Tab.

Here you can select whether you'd like to have Dithering, Enhanced Lighting, and Watch Enhancements. There is also a slide indicator to control Smoothing.

See the Enhance View topic for more information.
