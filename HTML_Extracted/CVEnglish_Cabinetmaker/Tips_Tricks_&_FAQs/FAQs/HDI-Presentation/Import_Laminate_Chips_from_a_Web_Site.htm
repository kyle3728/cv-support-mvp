Import Laminate Chips from a Web Site

Is it possible to go to a Laminate suppliers web site and pick a laminate color & import it to your laminate color selection?

Yes, if you go to the major laminate companies' web sites they have samples of their laminates available for download. Some even have their full line of textures/colors available for download as well.

You'll want to download the textures somewhere that you'll know where to find them. See the Texture topic for more information.
