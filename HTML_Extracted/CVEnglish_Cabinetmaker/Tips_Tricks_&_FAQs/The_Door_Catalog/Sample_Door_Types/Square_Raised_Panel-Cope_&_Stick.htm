Creating a Square Raised Panel Door (cope & stick)

To create a Square Raised Panel Door with cope & stick construction you must first have an "Inside Edge Profile" that matches your Door cutters.

  1. Select a Category to store the Door on, use the "Raised Panel" category.

  2. Click on the New option in the ribbon.

  3. Name and give a Description for the Door.

  4. Select Multi Piece for the Type.

  5. Click Next.

  6. Select Raised for the Panel Type.

  7. Click Finish.
