Creating a Frame and Panel Door using Mortise & Tenon

You do a mortise & tenon Door pretty much the same way you build any other Door. Except that before adding parts to the Door at the Construction tab you would go to Catalog Properties. Enter the depth of the mortise and tenon you want to use by setting the value for Joint Adjustment. Make sure that after the door is created go back to Catalog Properties and set the Joint Adjustment back to 0.
