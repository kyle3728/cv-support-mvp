Drawer Box 101

Drawer Box setup begins with the Drawer Guide clearances that are set in the Material Manager.

  1. On the Clearance Tab of the Drawer Guide Material, the Side Clearance affects the width of the Drawer Box. Also in that area, the Minimum Below Box and Minimum Above Box Clearances affect the height of the Drawer Box.

  2. A setting in the Constructio Method that affects the Drawer Box Size is the Mid Rail Placement setting in a Face Frame Style Construction Method. This Mid Rail Placement setting will create the Top Drawer opening size which is used in conjunction with the Drawer Guide Clearance settings to give you the Drawer Box Size.

  1. The next area of Cabinetmaker that will affect the Drawer Box is the Drawer Box Material Schedule.

  1. Next, the Assembly Manager is the area where the Drawer Box Construction is set up. In this area, you should enter the sizing and positioning information for the Parts of the Drawer Boxes. This will have no affect on the overall size of the Drawer Box but will determine how the Parts of the Drawer Box are sized and join together.

In the General Object the "Drawer Box Height Specification (#001)" standard, you can restrict the Height and Depth of your Drawer Boxes to specified sizes. If you choose to use this option, you will force the Drawer Box to be the largest listed size for the space available in an Opening.

  1. The last area that has an effect on the Drawer Box size is the Section Editor. When you are inside a Cabinetmaker Job, the Section Editor would be the place that you would make changes that influence the size of a Drawer Box. Changes to the size of the Drawer Opening will obviously influence the height of the Drawer Box.

In the Section Editor, if you Right-Click on the Drawer and click Properties, the Drawer Properties window will open. There is a Drawer Box tab in the Drawer Properties window where you can select a different Drawer Box Construction Method, Drawer Box Material, or Drawer Guide Type. These changes will allow you to have individually adjusted Drawer Boxes that have different construction, etc. from other Drawer Boxes in the same job or even within the same Assembly.
