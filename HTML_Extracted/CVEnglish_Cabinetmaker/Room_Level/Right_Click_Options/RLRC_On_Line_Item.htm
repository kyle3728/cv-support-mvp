Right Click On Line Item -

Following are the options here:

     

Add Copy

Will add a Copy of the selected line item to the list.

 Delete

Will delete the selected line item.

 Edit

Will take the selected line item to the Assembly Level.

 Rehinge

Selecting this option will allow you change the Hinge side for a Assembly, if it is a Single Door Assembly.

To Drawing

Selecting this option will send the current Reports View to the Drawings area as a Drawing Scene.

 Properties

See the Room Level Assembly Properties topic.
