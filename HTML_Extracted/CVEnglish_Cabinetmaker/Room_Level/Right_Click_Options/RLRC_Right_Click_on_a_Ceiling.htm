Right Click on a Ceiling

There are four options when Right Clicking on a Ceiling in the Ceiling Editor.

     

 Delete

This option will Delete the selected Ceiling.

 Delete All

This option will Delete all Ceilings in the Room.

 Edit Shape

This option will allow you to Edit the Shape of the Ceiling using the Shape Editor options.

 Properties

There are three Tabs in the Ceiling Properties area.

Ceiling Tab

There are three options in the Ceiling Tab:

  • Description - The Description of the Ceiling.

  • Texture - Click the Texture button to select a specific Texture for use with the selected Ceiling. If you leave the Texture undefined then Cabinetmaker will apply the Global Ceiling Texture specified for the Room.

  • Elevation - Allows you to establish the Elevation of the Ceiling.

Comment Tab

This Tab will allow you to add Comments to your Ceiling.
