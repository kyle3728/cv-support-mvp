Scene

     

Paper

     

Following are the possible options here:

 Delete

This option is available only if you right-click on a scene.  This allows you to remove (delete) that scene from the current Drawing Page.

Copy Scene

Create a copy of the currently selected Scene.

Clean Sheet

Will clean everything off of the current Sheet..

Title Block

Allows you to select a Title Block from the Drawings Library and places it on the Page.

Title blocks may be edited or created, within the Drawings Library, so that your drawings page will be bordered neatly and include important information about the Job and your Company. There are several Title Block Templates that come with your installation of Cabinetmaker. These Title Block Templates may need to be customized with your information before using them. You may also create your own Title Blocks using the Drawing Library.

By default each new Drawing Sheet you add will automatically add the Title Block you have Set As System Defaults. To clear this out so your new Drawing Sheets are blank simply select the None option here and click OK.

If you change the Title Block, the drawing page will be updated with the new Title Block. This has been made possible by putting the Title Block on its own Layer. Any CAD on the Title Block layer will be removed and replaced by the newly selected Title Block.

Scale

Displays the Scale for the current Sheet/Scene.  Allows the current sheet to be scaled to fit the paper.

Flip

Will Flip the selected Drawing Scene.

Mirror

Will Mirror the selected Drawing Scene.

Rotate...

Allows the user specified Rotation of the selected Drawing Scene.

 Properties - Scene

Displays the View Properties for the selected Scene.

General Tab

There are two options here:

  • Name - Allows you to edit the Name of the Scene.

  • Hide Scene - Allows you to Hide or Unhide the Scene.

Scale Tab

Allows you to set the Scale for the current Scene.

 Properties - Sheet

Displays the View Properties for the Drawings Page.

There are three tabs in the Drawing Page Property area:

Draw Tool Tab

There are four options here:

  • Grid Spacing - Controls the size of the grid squares on the screen when using the Grid Snap option from the Statusbar Menu.

  • Grid Always Visible - Normally the Grid is only displayed when Grid Snapping is on. Simply enable this option to have the Grid Always display.

  • Drag Scene From Snap Point - This option allows you to turn on ability to drag scenes with snap points. This turns on the Snap Points option allowing you to choose what points you would like to snap to. There are three options:

    • None - Turns off Snap Points for Scenes but still allows points within the Scene to Snap to the Grid.

    • Paper Points - Will Snap points within the Scene to points on the Grid or on the Paper.

    • All Scene Points - Will Snap points within the Scene to points on the Grid or from other Scenes on the Paper.

  • Reset Ruler - This removes any snap marks (red Xs) on the view.

See the Set As System Defaults topic for more information on it.

Paper Tab

Select your Paper Type from the drop down selector or click the action button to create your own custom Paper size. You can also set the Orientation of your Drawing Page here.

Paper Sizes

Use this area to create your own custom Paper size. You can also Copy, Rename, or Delete Paper sizes here.

See the Set As System Defaults topic for more information on it.

Scale Tab

Use this are to select the Scale on your Drawing Page. You can use pre-defined Imperial or Metric Scales, or your own User Defined Scale.

See the Set As System Defaults topic for more information on it.
