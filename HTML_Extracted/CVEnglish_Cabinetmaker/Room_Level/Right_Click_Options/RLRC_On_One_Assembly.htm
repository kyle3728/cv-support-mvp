 Copy

The Copy option allows you to Copy the selected Assembly anywhere in the current Room. Right click to cancel the Copy operation.

 Delete

The Delete option deletes the selected Assembly.

Move Label

The Move Label option allows you to Move system added Assembly Labels. The following image shows two Cabinets. The first Cabinet shows the Assembly Label in its default location which could be difficult to read. The second Cabinet shows the Label after using the Move Label Option. This option allows you to place the Assemblies Label anywhere you want.

Save

The reference to an Object in Solid includes Cabinets, and other items that you save to your library. Appliances, Cabinets, Closet Objects, Furniture, etc., are all objects. After creating a special Assembly or other object decide whether this will be a desirable item to have available in future Jobs. If this is a one time only creation, it is not necessary to save it as it will be saved when the Job is saved. However, it will only be available in the Job it is saved with. If it is possible that the item will be required for other Jobs it must be saved back into your library.

Save versus Save As

Right clicking on the Assembly at either the Plan or the Elevation view at the Layout or Assembly Level, will provide a menu option similar to the one shown above. Note that there are two options under the "save" feature. It is important to understand the difference between the two options. In the simplest terms, Save keeps the new item in its new form at the expense of loosing the prior item (the pre-edit form). Save As allows for the creation of a new item. In effect, Save As allows for two items when finished, whereas, Save allows for only one. Save overwrites the original item, whereas Save As creates a new item in addition to the original one. Use the Save feature when you want a permanent change to an existing item or Assembly, and the Save As feature when you want the new item in addition to the existing one. Save As will allow you to rename the new item and place it in other libraries you may have made.

Features Common to Save and Save As

The first option allows choosing of a Library by clicking on the Library Symbols in the left window as shown on the above screen capture. Thereafter, if desired, the icon can be changed by typing the selected name in the "Name" box at the top. The Description can be changed to represent your Assembly. An Icon should be chosen from the list at the right that represents the Assembly you are saving. This is done by clicking on the one of your choice.

Preserve Finish

This feature will save the item you are saving to your Library with any Finish overrides you have applied to the Object or its Parts.

Preserve Height

This feature will save the item you are saving to your library at the Height you have edited it to, instead of the default Assembly Height. If you have created an object that is of a different Height than a default Assembly, it is advisable to select this option. You will still be able to stretch or re-size it's Height after you bring it into a new Job regardless of whether you select this option or not.

Preserve Depth

This feature will save the item you are saving to your library at the Depth you have edited it to, instead of the default Assembly Depth. If you have created an object that is of a different Depth than a default Assembly, it is advisable to select this option. You will still be able to stretch or re-size it's Depth after you bring it into a new Job regardless of whether you select this option or not.

The Advanced Button

The Class Tab

The Class tab allows selection of the Class and Type of the object to be saved.

It is a good idea to keep Base Class objects, Wall Class objects, Closet Class objects, etc within their original class as other areas of Solid identify objects by their Class.

The Placement Tab

There are three main areas on the Placement Tab:

Horizontal

Determines whether the Cabinet can be placed along the Wall or in a Corner or both.

Stretching

By enabling or disabling the Width, Height, and/or Depth options you can control whether the Object can be re-sized in those dimensions.

Vertical

Determines the Vertical Placement of your Object. There are thirteen Vertical Placement options:

      • Above Elevation - Will place the Object above the Elevation established for the Object as shown below.

      • Above Positional - Will place the Object above the value set for Vertical Positional on the Room | Layout Tab in Room Properties.

      • Above Soffit - Will place the Object above the Soffit as specified on the Room | Layout Tab in Room Properties.

      • Below Elevation - Will place the Object below the Elevation established for the Object as shown below.

      • Below Positional - Will place the Object below the value set for Vertical Positional on the Room | Layout Tab in Room Properties.

      • Ceiling - Will place the Object at the Ceiling (Wall Height) as specified on the Room | Layout Tab in Room Properties.

      • Counter Top - Will place the Object above the Counter Top.

      • Float - Will allow free placement Vertically on the Wall/Corner.

      • Floor - Will place the Object on the Floor.

      • Toe Height - Will place the Object at the Toe Height.

      • Under Counter - Will place the Object below the Counter Top.

      • Under Soffit - Will place the Object below the Soffit as specified on the Room | Layout Tab in Room Properties.

      • Under Uppers - Will place the Object below the Upper Cabinets.

The Preserves Tab

The Preserve Tab allows preservation of attributes, such as Material or Construction Methods.

Using this feature will effectively lock your Cabinet from further changes in other Jobs, therefore, it should be used sparingly.

 Edit

The Edit option takes the selected Assembly to the Assembly Level. You can, alternatively, double click the Assembly to take it to the Assembly Level.

Section

Selecting this option will split the screen. The upper section is the Section Editor at the Assembly Level. The lower screen will show your Elevation. Whichever Assembly that is highlighted in the Elevation is the Assembly active in the Section Edit view. Simply click on the Assemblies in the Elevation to change the active Section Edit Assembly. This is a quicker way to Section Edit multiple Assemblies than the Right Click...Edit option.

Boxed Ends (Upper Cabinets)

Selecting this option will extend your Wall Cabinets ends to the Floor as in a Refrigerator Cabinet.

 Detach Toe

Selecting this option will separate your Toe Kick from the selected Assembly. This will make the Toe Kick a separate Assembly with its own Assembly Number.

Rehinge

Selecting this option will allow you change the Hinge side for an Assembly, if it is a Single Door Assembly.

Finish End

Selecting this option will allow you to change the end condition for the selected Assembly.

     

 Properties

Selecting this option will open the Assemblies Property Options. Following are the possible Tabs in the Assembly Properties Screen:

General Tab

Following are the properties you can modify on the General Tab:

Name

The Name field allows you to edit the name of the Object.

Quantity

You can change the Quantity of the selected Object. This will not add a new Assembly to your Layout but will add it to the Report area.

Description

You can modify the Description of the selected Object.

Snap Points

When selecting an Appliance or other decorative object you have an option to not show Snap Points for those Objects.

Exposed Interior

Placing a check will apply the Exterior Finish to the Interior of the selected Assembly and will instruct the Cabinet to utilize the Material Schedule selected for Exposed Interior in Job/Room/Cabinet Properties.

Comment Tab

This Tab will allow you to add Comments to your Assembly. These Comments will be displayed on Assembly Sheets and in selected Reports you generate.

Finishes Tab

This Tab allows you to modify the Interior and/or Exterior Finish for the selected Assembly.

Legs Tab

This Tab allows you to select the Legs to use on the selected Assembly.

This list will include the following Materials to choose from:

  • Bun Foot

  • Caster

  • Leg Leveler

  • Leg Leveler (No Ops)

  • Pedestal Leg

  • Tapered Leg

Overrides Tab

This Tab allows you to View and/or Remove any Overrides that may be present on the selected Assembly.
