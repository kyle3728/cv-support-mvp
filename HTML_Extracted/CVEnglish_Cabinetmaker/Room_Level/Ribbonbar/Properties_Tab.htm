Properties Tab

Following are the options on the Properties tab at the 3D View.

Return

Job Properties

Materials Visual

The Materials Visual option lists all the Materials in the current Job and allows you to change their Color, Texture, and Finish Type. This allows you to quickly change the Rendering for the current Job without returning to the System Level.

Changes made here are for visual purposes only.

Finish Types Visual

The Finish Types Visual option lists all of the Finish Types in the current Job and allows you to change their settings. This allows you to quickly change the Rendering for the current Job without returning to the System Level.

Changes made here are for visual purposes only.
