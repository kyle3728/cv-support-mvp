Saving Renders

Each of the Photovision Render options has a Save option. The Save option is used to create high-resolution bit-map image files that can be opened in an image editing program for enhancement and printing. The image will not be rendered on screen. The resolution that the file will render to is set in the Photovision Tab of the 3D View Properties dialog.

Save Types

Following are the File Types that can be Saved:

PNG Files - A standard graphic format.

JPEG Files - A standard graphic format.
