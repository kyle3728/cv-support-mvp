Edit Group

The Edit Group has one drop down with three options:

Cut

+

This option allows you to "Cut" selected objects to the Windows clipboard for pasting into a different Room to save time in layouts from Room to Room that maybe similar.

Copy

+

This option will allow you to "Copy" selected objects to the clipboard for pasting into a different Room to save time in layouts from Room to Room that maybe similar. If no Objects are selected when this option is selected then an image of the area will be copied to the Windows clipboard which can be "Pasted" into your favorite Windows Image Editor, Word Processor, or other Windows application.

Paste

+

This option allows you to "Paste" an object stored on the Windows Clipboard into a Room.
