Molding

The Molding Tab allows the selection of the default Molding types for the Job. Overhang can be set for the Moldings individually to allow for miter joints or other conditions. Moldings may still be modified or changed in the Job on a Wall by Wall basis if desired.

You may also change the Molding Material Schedule for this Job via the Modify for this Job button.

 

See the Set As System Defaults topic for more information on it.
