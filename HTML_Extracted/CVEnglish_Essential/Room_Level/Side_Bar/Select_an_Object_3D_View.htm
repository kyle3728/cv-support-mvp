Select an Object 3D View

When selecting an Object in 3D View, depending upon the Object Type selected, you gain the ability to change some of that Objects Visual and Material Properties. You can select more than one Object at a time by using + Left Mouse Click.

See the Right Click on an Object 3D View topic for more 3D Object selection options.

The Finish or Texture that is selected will only affect parts using materials that do not have a Finish or Texture assigned.

Selecting a Cabinet or Group of Cabinets

When selecting a Cabinet or a group of Cabinets you get the following options in the Room Level Sidebar:

  • Visual - Exterior Finish - Changes the Exterior Finish on the selected Object(s).

  • Visual - Interior Finish - Changes the Interior Finish on the selected Object(s).

  • Visual - Exterior Texture - Changes the Exterior Texture on the selected Object(s).

  • Visual - Interior Texture - Changes the Interior Texture on the selected Objects(s).

  • Materials - Assembly - Changes the Assembly Material Schedule used for the selected Cabinet.

  • Materials - Pulls - Changes the Pull Material Schedule used for the selected Cabinet.

Selecting an Appliance

  • Visual - Appliance Finish - Changes the selected Appliance's Finish.

  • Visual - Handle Finish - Changes the Handle Finish on the selected Appliance.

  • Visual - Finish Type - Changes the Finish Type on the selected Appliance.

Selecting a Counter Top

  • Visual - Finish - Changes the Finish on the selected Object(s).

  • Visual - Texture - Changes the Texture on the selected Object(s).

  • Materials - Counter - Changes the Counter Top Material Schedule used for the selected Counter Top.

Selecting Molding

  • Visual - Finish - Changes the Finish on the selected Object(s).

  • Visual - Texture - Changes the Texture on the selected Object(s).

  • Visual - Finish Type - Changes the Finish Type on the selected Object(s).

  • Molding - Profile - Changes the Profile used for the selected Molding.

Selecting a Wall

  • Visual - Finish - Changes the Finish on the selected Object(s).

  • Visual - Texture - Changes the Texture on the selected Object(s).

  • Visual - Finish Type - Changes the Finish Type on the selected Object(s).

Selecting a Ceiling

  • Visual - Finish - Changes the Color (Finish) on the selected Object(s).

  • Visual - Texture - Changes the Texture on the selected Object(s).

  • Visual - Finish Type - Changes the Finish Type on the selected Object(s).

Selecting a Floor

  • Visual - Finish - Changes the Color (Finish) on the selected Object(s).

  • Visual - Texture - Changes the Texture on the selected Object(s).

  • Visual - Finish Type - Changes the Finish Type on the selected Object(s).
