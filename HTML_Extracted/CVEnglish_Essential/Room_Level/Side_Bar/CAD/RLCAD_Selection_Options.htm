CAD Selection Options

There are several methods to select CAD objects as well as several functions you can perform on CAD Objects once they are selected.

Left Click Select

Simply Left Click on a single CAD Object to select it.

+ Left Click Select

To select multiple CAD Objects simply hold the key while Left Clicking.

+ Left clicking on a single CAD entity and selecting a CAD node for it will also put that item into Scale/Stretch/Move mode just like using the multi-select options.

Window Selection

You can also Left Click the screen and drag a Window to enclose all of the CAD Objects you want to Select.

Selecting Single CAD Objects

+ Left clicking on a single CAD entity and selecting a CAD node for it will also put that item into Scale/Stretch/Move mode just like using the multi-select options. See the Selecting a Group of CAD Objects section below for more information on Scale/Stretch/Move.

Selecting a Line

There are ten options when selecting a Line.

  • Color - Allows you to set the Color of the selected Line.

  • Layer - Allows you to set the CAD Layer the selected CAD Object is using.

  • Line Type - Allows you to set the Line Type the selected CAD Object is using.

  • Line Width - Allows you to set the Line Width the selected CAD Object is using.

  • Start X - Allows you to set the Starting X position of the selected CAD Object .

  • Start Y - Allows you to set the Starting Y position of the selected CAD Object .

  • End X - Allows you to set the Ending X position of the selected CAD Object .

  • End Y - Allows you to set the Ending Y position of the selected CAD Object .

  • Length - Allows you to set the Length of the selected CAD Object .

  • Angle - Allows you to set the Angle of the selected CAD Object .

Selecting a Rectangle

There are sixteen options when selecting a Rectangle:

  • Color - Previously covered

  • Layer - Previously covered

  • Line Type - Previously covered

  • Line Width - Previously covered

  • Left - Sets the Left position of the CAD Object.

  • Right - Sets the Right position of the selected CAD Object.

  • Top - Sets the Top position of the selected CAD Object.

  • Bottom - Sets the Top position of the selected CAD Object.

  • Center X - Sets the Center X position of the selected CAD Object.

  • Center Y - Sets the Center Y position of the selected CAD Object.

  • Angle - Previously covered

  • Width - Sets the Width of the selected CAD Object.

  • Height - Sets the Height of the selected CAD Object.

  • Hatch - Allows you to select the Hatch to use in the selected CAD Object.

For the Hatch option to be available for a selected Object it must be an enclosed geometry which is fully connected. By default, system CAD Shapes (Rectangle and Circle) will be enclosed.

Selecting a Dimension (Linear and Angular)

There are 26 possible options when selecting a Dimension (Linear):

  • Color - Previously covered

  • Layer - Previously covered

  • Dimension Style - Allows you to select the Dimension Style to use for the selected Dimension.

  • Line - Allows you to show the dimension line or not.

  • Line Width - Previously covered

  • Ticks Height - Sets the Height of Ticks.

  • Lines Extension - Sets the Line Extension.

  • Tail Offset - Sets the Tail Offset.

  • Left Arrow - Sets the Left Arrow Style.

  • Right Arrow - Sets the Right Arrow Style.

  • Arrow Size - Previously covered

  • Internal Left - Allows you to add or remove the Extension Line (Tail) on the Left end of the selected Dimension.

  • Internal Right - Allows you to add or remove the Extension Line (Tail) on the Right end of the selected Dimension.

  • Tailed - Allows you to turn Extension Lines (Tails) on or off for the selected Dimension.

  • Chain - Will remove the Left Extension Line on the selected Dimension when Chaining Dimensions together.

  • Alignment - Sets the Text's Alignment.

  • V Placement - Sets the Text's Vertical Placement.

  • H Placement - Sets the Text's Horizontal Placement.

  • Offset - Sets the Text's Offset.

  • Clearance - Sets the Text's Clearance.

  • Lock Text - Will Lock the Text on the selected Dimension.

  • Frame - Determines whether to show a Frame around the Text.

  • Font - Sets the Text's Font.

  • Bold - Sets whether the Text is Bold or not.

  • Height - Previously covered

  • Show Text - Allows you to turn off the Text on the selected Dimension.

There are several options when selecting CAD Objects you can initiate by simply clicking points on screen around the selected CAD Object(s).

Move

Once a Group of CAD Objects is selected simply left click any point within the object to use that as the Move point, this will allow you to Move an Object or Group of Objects from one known (snap point) position to another (snap point position). You can also Move the selected Group of Objects by using the Tab key to change values in the sidebar.

Since Scale and Stretch are also available by clicking the corners of selected CAD make sure the proper cursor prompt is used.

Move from End Point
