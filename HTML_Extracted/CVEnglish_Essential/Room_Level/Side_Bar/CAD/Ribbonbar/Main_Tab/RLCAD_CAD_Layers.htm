CAD Layers

CAD Layers are used in CAD and within Drawings Scenes when you edit a Drawing Scene in CAD mode. This list may be different depending on what objects are in the active Scene.

The following are the possible default Layers when bringing a Scene into CAD.

Molding - The Molding CAD Layer contains all of the Molding in the selected Scene.

Counter Tops - The Counter Tops CAD Layer contains all of the Counter Tops in the selected Scene.

Assemblies - The Cabinets CAD Layer contains all of the Cabinets in the selected Scene.

Walls - The Walls CAD Layer contains all of the Walls in the selected Scene.

Dimensions - The Dimensions CAD Layer contains all of the Dimensions in the selected Scene.

Annotations - The Annotations CAD Layer contains all of the Annotations in the selected Scene.

When using CAD in "non-Scene" areas you can create your own Custom Layers by clicking New... in the Layer Selector. Whichever Layer is selected in the drop down menu is the current Layer and any CAD entities drawn will be on that Layer.

Hiding the CAD Layers

Each of these Layers can be set to hidden by toggling the Light Bulb to grey. When a Layer is hidden; the elements particular to that Layer will not be visible until that Layer is toggled back to visible. In addition, if you add additional CAD details such as dimensions, lines, etc. to a scene then set that Layer to invisible; all of the added CAD details will be hidden as well.

This can be highly useful when you have a great deal of detail in one area. This can help prevent accidentally editing/deleting the wrong CAD Object as well as making the view clearer for you to work in it. All Drawings sent to the Drawings page as a scene will have a set of Default Layers, which Layers are contained in the Scene is dependent upon the source View of the Scene and/or what is actually contained in the Scene.

Locking the CAD Layers

Each of the CAD Layers include a Lock/Unlock icon. If you click on the Lock icon while it is in the Unlocked position - it will change to Locked and the reverse steps will unlock it. While a CAD Layer is Locked - items in that scene (particular to that Layer) cannot be deleted or modified. Also, items added to that scene (such as Lines, Dimensions, etc) cannot be deleted either. The Lock feature enables you to protect those portions of the scene which you don't want to change.
