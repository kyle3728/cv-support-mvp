Automatic Text

Whenever you add text through CAD to any area of Solid, you not only have the ability to type your desired Text but also have the ability of selecting from Text Variables of information already within your Job. The auto text appears inside the brackets {  } and is sometimes accompanied by additional static text.  The text outside of the brackets may be deleted leaving only that which is inside the brackets to be automatically filled in.  Automatic text is not limited to but is especially useful in Title Blocks.

Clicking the ellipsis button will bring up the list of available Text Variables. The list of Text Variables will be different depending on whether you are on the Paper or within a Scene.

Below is a list of the Text Variables and where the values in the variables are being pulled from.

The list below represents the text variables listed in the menu system. This list does not include "ALL" that are available. In addition to the text variables in the list you can also call out any Parameter in Solid whether it is a System Parameter or a User Parameter by following the format below and replacing the variable name with the Parameter you want to call.

Examples

To call out the AX Rotation of a Cabinet you would simply type {cab.ax}.

Text Format Options

There are three formatting options available specifically to Automatic Text:

u: - Sets first character to upper case.

l: - Sets first character to lower case.

n: - Displays negative numbers within parentheses

Example

{u:cab.cabnote1}
