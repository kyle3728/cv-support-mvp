Main Tab

Click the options below for more information

 

The options specific to the CAD area are below:

Layer Selector

The Layer Selector allows you to select the Current Layer you are drawing on. You can also turn On or Off Layers or Lock or Unlock Layers from here.

Layers

The Layers option allows you to Rename, Remove, or Create CAD Layers. See the CAD Layers topic for more information.

 

Select

This will change the cursor to an arrowhead and allow selection of lines, arcs, etc. After selecting the item, you will be able to change properties by selecting the desired property off the list in the sidebar and selecting the item to apply it to. Lines can be moved, re-sized, etc. Text can be placed, edited, as well as changing the font and font size. Select is covered in more detail in the Selection Options topic.

Select All

The Select All menu option will select all CAD Drawn objects in the current View.

Measure

This option allows you to Measure CAD Objects without leaving a dimension on the drawing.

Delete

This option will allow you to Delete the selected CAD Object(s).

Join

Consolidate

Allows you to Consolidate two lines together. Left Click the first line then Left Click the second and consecutive lines to connect them.

Connect

This option, similar to Fillet, will Connect two Perpendicular lines together creating a corner connection with no radius.

Fillet

This option allows you to Fillet two lines together. Specify the Fillet Radius in the sidebar then Left Click the first line then Left Click the second to connect them.

In the following example, two lines were drawn at a 90 degree angle to one another and the radius in the field on the left of the screen was set to 5”. The dashed lines represent where the lines used to extend.

Chamfer

This option allows you to Chamfer two lines together. Specify the Chamfer Length in the sidebar then Left Click the first line then Left Click the second to connect them.

The example below shows a 5” Chamfer, the dashed lines represent where the lines used to extend.

Bend

This option allows you to select a Line and turn it into an Arc by dragging it to Bend it.

Lines

The Lines tool had three options:

Line

This tool allows the creation of Lines for use in drawings or as overlays for your scenes in Solid.  After selecting and placing a line, the line properties window will display in the sidebar.  

Chained Line

This tool allows you to quickly draw two or more lines that are connected. With the Line tool above you must specify a start and end point for each line drawn. With chained line you specify a start point and when specifying the end point you are automatically specifying the start point for the next line. When you are finished drawing your group of Lines simply right click to end the Chained Line action. The end result will be individual Lines and when selecting them you will be presented with the same sidebar options as Line above.

Line Box

This tool, similar to the Rectangle tool, allows you to quickly draw a Rectangle made up of individual lines. The end result will be individual Lines and when selecting them you will be presented with the same sidebar options as Line above.

Rectangle

This tool provides for creating a Rectangle (or square) on the page by clicking the point where it is to start and clicking on the point where it is to finish. It can thereafter be adjusted by using the Select tool. When using this tool you are creating a fully enclosed shape, you can break the Rectangle into its four individual lines by using the Right Click Break option. The area in the sidebar is similar to the one for the Line tool. You also have the added ability to add a hatch pattern to the enclosed space.

Pressing + Left Mouse Drag will force the Rectangle to be a Square.

Circle

This option allows you to create Circles. Click in the place where you want the center of the circle and pull the mouse to one side or the other and a Circle will develop. The area in the sidebar allows modifying the radius by numerically entering a value, as shown below.

If you select a circle after it has already been drawn, you will have these other options to alter it.

  Arcs

The Arcs tool has three options:

Arc

This option allows creation of an Arc by clicking the point where it is to begin, and an arc will appear.  The arc is sized and positioned by the movement of the mouse.  The final click will set the arc at the selected point.  In the example below you can see that there are two fields on the left that you can enter the X and Y end positions while drawing this arc.  Notice that this arc will be a quarter of a circle (or ellipse).

Arc By Radius

This option allows definition of a point where a radius begins, the radius, and the degrees of radius.  The first mouse click marks the center of the radius, pull the radius that you want and click again.

A circle will appear with a radius line within it. Move the cursor and the circle opening becomes an arc as you do so.

When you reach the correct position, a third click will set the arc at the selected point.

3 Points Arc

This option lets you place two points to define the opening of the arc, and the third point pulls the arc increasing the radius. A 3 Points Arc is very handy when you have three known points to Snap to.

In the example below I want to create the Arc shown in red.

To begin I select the Arcs|3 Points Arc option from the Ribbonbar then Left Click the End Point (With Snap To Points on) of the first line.

Next I Left Click the Mid Point of the second line.

And the last and third point I Left Click the End point of the third line.

And now I have the Arc.

Hatch

Creates and fills shapes on your drawing. An example would be the arrow shown below. Click each point defining the shape of the item, and right click for the final connection. The hatching appears automatically.

If you select a Hatch after it has already been drawn, you will have these other options to alter it.

The hatch design can be changed by selecting a different design by clicking the arrow in the Hatch field to bring up the Hatch selector.

Polyline

This option creates a joined line with multiple segments. Left click to start, and at intermediate points. Right click to set the Polyline. The example below shows a Polyline that has three line segments that have been set and a fourth being drawn.

Dimensions

This option provides real dimensioning capabilities in one of two modes. It dimensions full scale in Paper Space and in scale in Drawing Space. When used with the "Snap To Points" snap feature, the dimensions provided are accurate and easy to place.

Before you draw any Dimension line you will be able to select what type of line you want to draw. This selection can be made from the Dimension option in the sidebar. You have ten different types of dimension lines to choose from.

The design of the dimension line as well as its orientation can be modified from the options in the sidebar. The following options are available after the dimension has been drawn and then selected. There are additional properties available by clicking the double arrows beside the lines labeled “Lines and Arrows” and “Text”.

To quickly Dimension a line you can click the line while holding + and the Dimension tool will grab both end points of the line simultaneously.

See the CAD Dimensioning topic for more information.

Leaders

The Leaders tool has two options:

Leader

This option allows you to draw a segmented line with an arrow point attached. With the options available in the sidebar, you can change the color and line properties. Left click to start and at intermediate points. Right click to finish.

Text Leader

This option, similar to the Leader, allows you to draw a segmented line with an arrow point attached. With the options available in the sidebar, you can change the color and line properties. Left click to start and at intermediate points. Right click to finish the Leader and you are then prompted to enter your Text.

Text

This option will allow placement of Text on the drawing, or the drop down feature can be selected and a variable value can be chosen. See the CAD Automatic Text topic for more information on this option.

Cloud

This option will allow you to draw a revision Cloud. Left click to start the cloud then drag in the direction you wish to move and click to establish each arc length in the Cloud. Or select Free Hand in the sidebar and set a Chord and you can freehand sketch your Revision Cloud.

Image

This option allows you to select and place saved bitmaps onto your drawing page.
