Walls -

Upon entering the Wall Editor you will notice the ribbonbar changes. This topic will focus on four sections.

Layers Section

There are two options in the Layer selector:

Walls - The default Layer and the one the Walls are drawn on.

Construction - This Layer allows you to draw Construction/Reference Lines to assist in laying out your Room.

Depending on which Layer is active that some options in the Modify section of the Ribbonbar may or may not be available.

Drawing Section

There are four options in the Drawing section:

Line

Using the Line option allows you to draw one Wall at a time by selecting the start point then the end point. See the Line option in the CAD Main Tab topic for more information.

Chained Line

The default Wall drawing mode. Using the Chained Line option allows you to draw connected Walls where each click establishes both end of the current Wall and the start of the next Wall. See the Chained Line option in the CAD Main Tab topic for more information.

Line Box

Using the Line Box option allows you to draw four connected Walls as a rectangle with two clicks establishing the two diagonally opposite corners of the Room. See the Line Box option in the CAD Main Tab topic for more information.

Drawing Walls

Click your mouse at the point you desire the Wall to begin. Note that creating the Walls in a clock wise direction around the perimeter of the Room will cause the dimension lines to appear on the outside of the Walls. This would be the normal procedure for placing Assemblies on the inside of the Room. After clicking the origin point for the Wall, release the left mouse button and pull the Wall in the direction you want it to run.

Note the information in the sidebar will change to reflect the Length and angle as the Wall changes. You are free to enter the value for the Wall length at any time by typing in a value. To do this, simply type the value, you do not need to click anywhere or perform any other action prior to entering the value, assuming that the data box for the Wall length is filled with a background color. If you desire to enter values for the other data boxes, use to move down, or +   to navigate upward.

If you are creating the Wall with the mouse only using Chained Line, which is the default, to complete the Wall, left click at the location for the end of the Wall, if you wish to stop drawing Walls simply right click the mouse and select the Select option to complete. To create subsequent Walls, simply drag the mouse to the end point of your next Wall.

Final adjustment to Wall dimensions can be made by left clicking on the Wall, and entering data in the sidebar as described above.

Right click on a Wall for options regarding "Connect Left" or "Connect Right". These options will cause the Walls to connect if they are close enough to one another and not connected to another Wall at the same end. See the Right Click a Wall in Plan View topic for more information.

Creating Angled Walls

To create angled Walls you can simply drag to the Angle desired and eyeball the Angle value in the Left window. Or you can, while dragging the Wall, press the function key F3 on the keyboard to bring up the Angle Calculator. You can use the Angle Calculator to calculate inside angles only, using the 3/4/5 method.
