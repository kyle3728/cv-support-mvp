Right Click On Multiple Assemblies -

There are 4 Options when right clicking on multiple Assemblies in Plan and/or Elevation View:

Equalize Widths

This option will Equalize the Widths of the selected Objects.

Combine

This option will Combine the selected Assemblies into one Assembly.

Delete

This option will Delete the selected Objects.

Move

This option will allow you to Move the selected Objects
