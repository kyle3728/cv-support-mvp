Right Click On Wall Plan View -

When Right Clicking on a Wall in Plan View you will have 14 options:

Cut

Cuts the selected Wall to the Windows Clipboard.

Copy

Copies the selected Wall to the Windows Clipboard.

Paste

Pastes the contents of the Windows Clipboard.

Delete

Deletes the selected Wall.

Select All

Selects All Walls.

Select None

Deselects any Selected Walls.

Reverse Selection

This option will deselect the currently selected Wall(s) and select all other deselected Wall(s).

Hide

Hides the selected Wall and all Objects attached to it in 3D view.

Show

Shows the selected Wall and all Objects attached to it in 3D view.

Show/Hide Walls

If you would like to Show or Hide Walls for more clear 3D Views you can do so by selecting this option. Simply enable or disable the Walls in the list to Show/Hide them.

Connections

Allows you Connect/Disconnect the selected wall to adjoining Walls.

Show Dimensions

Will turn Dimensions on for the selected Wall if they have been previously Hidden.

Hide Dimensions

Hides all Dimensions associated with the Wall and Objects on the Wall.

Properties

Clicking on the option will open the View Properties screen. There are 4 Tabs in the Wall Properties Screen:

General Tab

This Tab allows you to Modify the Name and/or Description of the selected Wall. You can also enter a Labor value in minutes

Comment Tab

This Tab will allow you to add Comments to the selected Object. This can be displayed in selected Reports you generate.

Finishes Tab

This Tab allows you to modify the Finish or Texture on the Front, Back, and/or Ends of the selected Wall.

Overrides Tab

This Tab allows you to remove any Overrides on the Wall.
