Edit Sheets

This option allows you to manage and organize your Drawing Sheets in the Drawings area.

There are seven options available in the Drawing Sheets area:

  • Open - Opens the Selected Drawing Sheet.

  • New - Creates a New Drawing Sheet.

  • Rename - Allows you to Rename the selected Drawing Sheet.

  • Remove - Removes the selected Drawing Sheet.

You can also use standard Windows + Click and/or + Click selection here to Delete Multiple Drawing Sheets at once.

  • Move Up - Moves the selected Sheet up in the Sheet Order.

  • Move Down - Moves the selected Sheet down in the Sheet Order.

  • Close - Closes the Drawing Sheets Editor.
