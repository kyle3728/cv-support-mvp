Update Job

Use this feature to apply System Level changes to the current Job. For example: If you close the Job and go to the Splash Screen, then open the Material Manager and make a change to some Materials or Construction Methods that are being used in the current Job, for those changes to be displayed in the current Job you must open the Job and use this option.
