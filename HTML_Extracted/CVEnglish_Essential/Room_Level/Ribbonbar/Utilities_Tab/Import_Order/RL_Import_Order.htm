Import Order

This option allows you to browse to and import an Assembly list from a 3rd party application.

After clicking the Import Order option you will be presented with the Import Order Browse Window.

 

 

See the ORD File Format topic for more information.

 

 

Simply select the ORD file to Open and click Open.
