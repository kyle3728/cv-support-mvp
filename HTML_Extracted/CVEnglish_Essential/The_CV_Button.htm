The CV Button

The CV Button, depending on the Level of CABINET VISION you are in, has the following possible options.

System Level

Room Level

New

+

When you click on New, Solid will start the Job Properties area to allow you to define a new Job.

Open

+

Clicking on Open will bring up a Windows Open window that you may use to browse to and open a Job file. By default, Jobs are stored in the Jobs folder. However, you may redirect Solid to store Jobs anywhere that you wish.

To open a Solid Job, simply click on the Job that you want to open and click the Open button.

To open an Order Entry File vs. Importing one simply browse to your ORD location, select the file type selector and select Order Entry Files (*.ord), select the Job to open and click Open.

You can also Open XML Order Entry ORDX file here. See the ORDX Format topic for more information.

You also have an option to open Batch Files (*.cvb). A CVB file is simply a text file that contains a list of Jobs (CVJ) or Order Entry Files  to be opened together into a Batch (see next section in this topic for information on Batch Jobs). The file should contain the fully qualified path and file names of the job files.

For example...file name Filename.cvb would contain:

C:\Cabinet Vision\Solid_10_0\Jobs\Job1.cvj

C:\Cabinet Vision\Solid_10_0\Jobs\Job2.cvj

C:\Cabinet Vision\Solid_10_0\Jobs\Job3.cvj

Batch Jobs

This Utility allows you to Batch multiple Jobs or ORD files together into one Cut List. This can be used to either send that list to the S2M Center, Bid Center, or the Report Center.

Clicking on the Add Job... button will bring you into the Standard Windows File Selection Dialog Box. Simply go to the Directory where your Jobs are stored and add them to list. You can select multiple Jobs by using the standard Windows + Click and/or + Click options.

After all your Jobs are selected simply click OK. Solid will then open the Jobs in Batch Mode. You will then have access to the Bid Center, Report Center, and the S2M Center for the Batch.

Job Recovery

This tool will show you all of the .BAK, .ASV, and .TMP files available for possible retrieval. This utility will attempt to restore a Job from a previous state by simply highlighting the file and clicking Open.

The File Types

.BAK - A BAK file is created each time you successfully Open a Job.

.ASV - An ASV file is created each time the AutoSave process runs.

.TMP - It is possible that a TMP file of your Job could be available. Typically TMP files are created during backup and AutoSave generation but are deleted once the system has completed the process. It is unlikely, but possible, that you could restore a Job from a TMP file.

Close

This option will Close the current Job and return to the Splash Screen.

Save

+

Use this option to save the current Job. This updates the current Job file with any changes that have been made to it.

Save As

This option gives you the opportunity to save the current Job with a new name. This allows you to save a new version of the Job under a different name so that the existing Job will not reflect the changes.

When the Save As window appears (as pictured below), simply type in the new name and click Save.

Import Jobsite Drawing

Clicking on Import Jobsite Drawing will bring up a Windows Jobsite Companion Data window that you may use to browse to and open a Jobsite Companion Room.

To open a Jobsite file, make sure your Windows Mobile Device is connected to your computer, then simply click on the Job that you want to open and click the Open button.

If you do not see your Jobsite Jobs make sure your Mobile Device is synchronized and that the path to your Mobile Shared folder has been setup via the Preferences | Modules Tab.

Export Jobsite Drawing

Use this option to export the current Room to a Jobsite Companion file.

Print

+

This option allows you to Print the current View; the following window will appear:

Simply choose the printer that you wish to use and click OK to print  the current view.

Print Setup

This option allows you to set Printer options; the following window will appear:

Use this option to choose the settings for the printer that you plan to use with Solid.

Send

Use this option to create an e-mail message and attach the current Job. This option works if you use Microsoft Outlook as your e-mail program.

Recent Jobs

The last several Jobs that you have opened will be listed here. For quick access to any of those files, simply click on the one that you want to Open. If the location of the Job does not match the default Job Open/Save location (the last path you Open from or Saved to) a portion of the path will be displayed as a hint.
