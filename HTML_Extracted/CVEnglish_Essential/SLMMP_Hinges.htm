Hinge Group

  • Type - Type defines the style of Hinge Plate as either Concealed, Double Door, or Exposed.

  • Horizontal Overlay - The value input here controls the amount that the Door Overlays the Face Frame at the top and bottom of the Door. Therefore, this value only applies to hinges used on Face Frame Style Cabinets.

  • Vertical Overlay - The value input here controls the amount that the Door Overlays the Face Frame at the left and right of the Door. Therefore, this value only applies to hinges used on Face Frame Style Cabinets.

  • Door End to Hinge - References the distance from the Ends (Top and Bottom) of the Door to the center of the Hinge. Sets the Graphic Reference Point placement.

  • Inset - A "0" (zero) value Â would result in the Door being forward of the face frame or face of the case by the thickness of the Door. The amount entered in this field moves the Door backwards, to within the Face Frame or Case, depending on Construction Style. A negative value here will outset the Door away from the Face of the Cabinet.

  • Maximum Spacing - This value controls how far apart Hinges will be placed before Solid will add another Hinge to the Door.

  • Opening Angle - The value you enter here, represents the maximum opening angle of the Hinge in degrees. When working in Solid you will be able to open Doors and Drawers. This value should represent the angle of the Door to the Cabinet when you have opened the Door the maximum amount.

  • Door Edge To Reference Point - When defining a "Graphic" for your Hinge the Graphic will shift when used in an Assembly by this value. This value typically represents the distance from the Edges (Sides) of your Door to the Hinge Center. Sets the Graphic Reference Point placement.

  • Roll Out Clearance - If you place a Roll Out in the same location as a Hinge Solid will automatically space the Roll Out the amount you specify here.
