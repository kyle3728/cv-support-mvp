Right Click On Screen -

There are six options when right clicking on the screen in Plan or Elevation view:

 

 Insert Object

  1. To insert an object using this method, you must:

  2. Go to the floor plan view and draw your wall layout.

  3. Click on the Objects button on the left side of the screen and then select the catalog that you want to pull the object from.

  4. Right Click on the floor plan (away from any walls or other objects). Click Insert Object and the Insert Object box will appear (see below).

  1. Type in the first letter of the object that you want to place and then click OK.

  2. You will see a list of objects in the selected catalog that begin with that letter.  Choose the desired object from the list.

  3. You will then have the desired object as if you had manually pulled it from the library.  Place it on the plan view normally.

 Add Light

This is a shortcut to adding a light to the Room. Find more information about lighting and rendering settings here.

 Show/Hide Walls

If you would like to Show or Hide Walls for more clear 3D Views you can do so by selecting this option. Simply enable or disable the Walls in the list to Show/Hide them.

 To Drawing

Sends a static image of the current View to Drawing Scenes.

 To Live Drawing -

Sends a live view of the current View to Drawing Scenes.

 Properties

Draw Tool Tab

On the view properties window, you have the options to change the grid settings.

Grid Spacing - Controls the size of the grid squares on the screen when using the Grid Snap option from the Status bar Menu.

Grid Always Visible - Normally the Grid is only displayed when Grid Snapping is on. Simply enable this option to have the Grid Always display.

Reset Ruler:  This removes any snap marks (red Xs) on the view.

Layers Tab -

By clicking on the Layers tab, you have a shortcut to the Layers settings for the current layer and the current view. This is the same as making changes to Layers as described here.
