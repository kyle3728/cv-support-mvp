Right Click on Screen Cross Section View -

There are four options when right clicking on the screen in Cross Section View:

 

 To Drawing

Sends a static image of the current View to Drawing Scenes.

 To Live Drawing -

Sends a live view of the current View to Drawing Scenes.

 Save Section

This option allows you to save the current Cross Section View as a Section View. This option saves the Section position of the current Cross Section View. Use the Saved Sections controls in the ribbonbar to work with Saved Sections you have created.
