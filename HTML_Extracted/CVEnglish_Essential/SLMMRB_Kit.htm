This option allows you to add miscellaneous Materials into a Kit attached to the selected Material. For example you can add a Kit for a Wire Pull that includes two screws for mounting. You should specify the quantity as well.

Select your Material from the pane on the right and then click the left button to add it to the Material Kit in the left pane. You can also drag a Material from the right pane to the left.

If a Material has a Kit assigned there is a visual identifier on the Material in the main Material Manager window.
