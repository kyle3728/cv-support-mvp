Use Door Template

This option is only available if you have purchased the MDF DoorMaker module. Clicking the Use Door Template option box allows the creation of MDF Doors for display in Solid and machining through the S2M CENTER.

For your Doors to display and machine properly you will need to first setup the Tools and Tool Sets necessary. See the S2M CENTER Help Tools topic and the Tool Sets topic for more information.

After checking the Use Door Template option the Construction Tab will change to a different layout.

When using Door Templates you will have a group of Properties available that allow you to define the Template to use for your Door as well as several Properties to control how the Door is constructed. There are several Properties available in the sidebar for a Template Door.

Sample Door Size

Width - Allows you to set the Width of the Sample Door displayed to the right.

Height - Allows you to set the Height of the Sample Door displayed to the right.

Sample Profile

Outside - Applies the selected ToolSet to the Outside Edge on the Template Door.

Inside - Applies the selected ToolSet to the Inside Edge on a non-Slab Style Template Door.

Panel - Applies the selected ToolSet to the Panel on a non-Slab Style Template Door.

The Panel Part in a non-Slab Door uses the "Door Glass Panel" Part in the Door Material Schedule. Simply set that Part to the Material Type desired for your Non-Slab Template Door.

Route - The Toolset to use for the Route Pattern representing the Door Style's interior machining.

Template

Template - See the Templates section later in this topic.

Slab - Determines if the Door Type is Slab or Five Piece.

This option does not determine whether the Door is a Glass Door; that is actually controlled by the Tool Set used on the Route Pattern.

Soften Corners - Determines if the Corners of the Doors need to be softened (radiused). This is typically used when applying RTF or Vinyl to your Doors to keep it from tearing on sharp corners. The Radius value is set in the Preferences are on the Geometry Tab in the S2M CENTER. See the S2M CENTER Help file for more information.

Corner Radius - Allows you to establish a specific Corner Radius for this Door Style when Soften Corners is enabled.

Lites - Sets the number and configuration of Lites (Panels) the Template Door has.

  • Shape Top - Selecting this option will Shape the Top Outside Edge of the Door to match the internal Route Pattern as demonstrated in the image below.

  • Shape Bottom - Selecting this option will Shape the Bottom Outside Edge of the Door (Mirror Top Shape has to be active as well) to match the internal Route Pattern as demonstrated in the image below.

Spacing

Top Rail - Sets the Width of the Top Rail.

Bottom Rail - Sets the Width of the Bottom Rail.

Mid Rail - Sets the Width of the Mid Rail.

Left Stile - Sets the Width of the Left Stile.

Right Stile - Sets the Width of the Right Stile.

Mid Stile - Sets the Width of the Mid Stile.

Shape Properties

  • Crown - Allows you to specify the Crown Height on the Door as demonstrated in the graphic below.

  • Flat Indent - Allows you to specify the Flat length before the start of the Crown as demonstrated in the graphic below.

  • Radius - Allows you to specify the Radius at the transition from the End Stiles to the Crown as demonstrated in the graphic below.

  • Chamfer - Allows you to specify the Chamfer length on a Chamfer Template Door.

  • Mirror Top Shape - Allows you to specify that the Bottom of the Route Pattern Mirrors the Top instead of being flat. As demonstrated in the graphic below.

  • All Corners - No specifies to Chamfer/Radius the top two corners only; Yes specifies to chamfer/radius all four corners of the Route Path.

  • Rise - Allows you to specify the Rise of the slope of a Slope Template Door.

  • Run - Allows you to specify the Run of the slope of a Slope Template Door.

  • Width - Allows you to specify the bumps Width on a Bump Template Door.

  • Height - Allows you to specify the bumps Height on a Bump Template Door.
