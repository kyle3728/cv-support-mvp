The CxConstruction.mdf File

The CxConstruction.mdf file stores the Cabinet, Closet, and Drawer & Roll Out Construction data, as well as User Created Standards and Intelli-Joints.

The following is the breakdown of the tables and their fields:

Connection

This table stores Connection data.

Field

Description

ID

A unique number to reference the connection.

Name

The connection name.

Description

The connection description.

rGUID

A global unique identification for the connection

System

A boolean value set true if this is a read-only system connection

ConnectionSetMap

This table stores the Connection Set Map.

Field

Description

ID

A unique number to reference the connection set map.

ConnectionID

A cross reference to [Connection].[ID] for the master connection ID.

MemberID

A cross reference to [Connection].[ID] for the slave connection ID.

UnitOfMeasureID

If non-zero, a cross reference to the [refUnitOfMeasure].[ID] field defining what unit of measure to display the value

UCS

This table stores the User Created Standards.

Field

Description

ID

A unique number used to reference the UCS.

Ordinal

Represents the order that the UCS runs in. This is also the order that the UCS's are displayed in the UCS Editor.

Name

The Name of the UCS.

UCSTypeID

A cross reference to [refUCSType].[ID] indicating the type of UCS: 1 = An Add Part UCS, 2 = A Modify Part UCS, 3 = A Delete Part UCS

UCSOnectTypeID

A cross reference to [refUCSObjectType].[ID] indicating a PartID=1, IJointID=2, Dovetail ToolID=3, Door=4, DrwFrnt=5, Dado=6, HBore=7, VBore=8

ObjectGUID

A Part, Tool or IJoint GUID based on the UCSObjectTypeID

UCSLinkToTypeID

A cross reference to [refUCSLinkToType].[ID] indicating object type the Added Part (UcsTypeID=1) is linked to

LinkToGUID

A cross reference to the [CxMaterials].[Part].[rGUID] of the part this object is linked to

AssemblyName

Represents the assembly Name chosen for the UCS. Wildcards * and ? are allowed.

PartMaterialGUID

Represents the Material selected for the Part in the UCS. Cross referenced to [CxMaterials].[Material].[rGUID]

PartDisplay

A boolean flag set true if the Part is to be displayed graphically.

LibraryPart

The location of the Library Part called in the UCS.

Code

The actual coding that makes up the UCS.

CodeModified

A boolean flag set true if the UCS has been manually modified.

Disabled

A boolean flag set true if the UCS has been disabled.

PreBuild

A boolean flag set true if the UCS should be applied prior to build.

Lic

Licensing data to required to unlock standard.

rGUID

A global unique identification for the user created standard
