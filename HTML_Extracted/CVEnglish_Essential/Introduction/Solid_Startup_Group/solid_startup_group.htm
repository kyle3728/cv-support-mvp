Solid Essential Start Group

In Windows 10 click the Windows Start Button then select All Apps then scroll to the Solid Essential Group.

 

In previous version of Windows go to Start | All Programs | Solid Essential

 

Following are options in the Startup Group:

  1. Backup Solid Essential Settings - Allows the creation of a backup of the Solid Database Files. This option also gives access to the settings for the Solid Backup Utility. You can also add switches to the short-cut for options when running the Backup Settings from the Start Group:

    • To add switches to Backup Settings right click on the Backup Solid Essential Settings icon in the Start bar and click More | Open file location.

    • Right click the Backup Solid Essential Settings icon in File Explorer and select Properties.

  • Click to place your cursor after the .exe in the Target field.

  • The following switches (command line arguments) are available:

    • /e Create email (will automatically create a mail message and attach a zip file for email)

      • Example: C:\Cabinet Vision\Solid_12\CVBackup.exe /e

    • /d Delete backups (useful option when you are just emailing)

      • Example: C:\Cabinet Vision\Solid_12\CVBackup.exe /e /d

    • /g Suppress GUI (Silent backup that uses last settings)

      • Example: C:\Cabinet Vision\Solid_12\CVBackup.exe /g

    • /p Suppress prompt when you have exceeded backup settings (useful option to use with /g)

      • Example: C:\Cabinet Vision\Solid_12\CVBackup.exe /g /p

    • /r Make files read only

      • Example: C:\Cabinet Vision\Solid_12\CVBackup.exe /r

    • /z Creates a Zip file of the Backup

      • Example: C:\Cabinet Vision\Solid_12\CVBackup.exe /z

  1. Help - Opens this Help System.

  2. Restore Settings - Allows the Restoration of backups created by using the Solid Backup Utility or Automatic Backups that are created automatically during Installations.

  1. Solid Essential - Launches Solid Essential.

  2. System Information - A utility utilized by Technical Support to diagnose, troubleshoot, and repair Vero Software Settings.

  3. Update Version - Launches the Update Version Utility which compacts and repairs Solid Database Files.
