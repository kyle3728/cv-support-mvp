Update Version

The progress window shown below will appear during the Update Version process.

The Update Version Utility performs several functions:

Create Backups - The Update Version Utility first creates a full set of database backups which means that you will have a restore point in case there is a need to revert to saved database files for some reason.

Cleanup Databases - This portion of the Update Version Process will take the database files and remove any errors that are found.

Compact Databases - This portion of the Update Version Process removes empty space in the Solid Database files.

Update Databases - This portion of the Update Version Process will make any changes necessary to bring the Solid Databases in line with changes that have been made during the development of new versions and builds of Solid.

Register Files - This portion of the Update Version Process re-registers system files used by Solid.
