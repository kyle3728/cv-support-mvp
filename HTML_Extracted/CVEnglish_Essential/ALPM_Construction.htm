Construction

The Construction Tab at Cabinet/Closet Properties has three possible options:

  • Construction Method Cabinets/Closets - Allows you to change the Construction Method for the active Assembly. This places an override on the Assembly.

  • Construction Method Drawer Box - Allows you to change the Drawer Box Construction for the active Assembly. This places an override on the Assembly.

  • Construction Method Roll Outs - Allows you to change the Roll Out Construction for the active Assembly. This places an override on the Assembly.

Remember changes made here will only affect the current Assembly. Also, changes made here are attached as permanent (unless you remove them) overrides on the Assembly; which means that if you Save this Assembly into a Catalog these overrides will also be Saved.
