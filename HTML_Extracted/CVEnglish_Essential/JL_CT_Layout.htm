Counter Top Layout

Click on the Categories or Tabs below for more information

Under Automatic Tops, you should enter values for Face Overhang, End Overhang, and Minimum Depth. You can also specify if the AutoBuild Top Feature should automatically add Splashes.

These settings refer to the relationship of the Counter Top to the Base Cabinet below, and apply only when the Auto-Build feature is used. When Manually building Tops these values do not apply.
