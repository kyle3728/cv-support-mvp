Model -

The Model Editor allows you to Import 3D DXF or Sketchup 3D Models to represent your selected Material.

Ribbonbar

Following are the ribbonbar options in the Model Editor:

Return

This option will return you to the selected Material's Property page.

Clipboard

Zoom Group

Render Mode

View Mode

The View Mode option allows you to orient your view onto one of the 6 available Views.

3D Mode

Switches the view to a 3D View of your Graphic.

Add Hardware

The Add Hardware option is used to import your 3D Sketchup (.SKP) or 3D DXF (.DXF) file to the selected Work Plane (see Sidebar section later in this topic). See the Creating a Model section of this topic for more information on working with 3D Models.

Add Work Plane

Adds a new Work Plane to the selected View (see View Mode above).

Remove

Removes the object selected in the sidebar tree.

Measure Tools

Object Tree

Help

Sidebar

The sidebar contains a tree view of the Objects contained in the Model along with the Properties of the selected Object. See the Creating a Model section of this topic for more information on working with the sidebar.

Right Click Options

Right Clicking the Screen

 

There is one option when right clicking the screen with nothing selected in the sidebar:

 Properties - Brings up View Properties. There are two tabs available.

Draw Tool - On the view properties window, you have the options to change the grid settings.

Grid Spacing - Controls the size of the grid squares on the screen when using the Grid Snap option from the Statusbar Menu.

Grid Always Visible - Normally the Grid is only displayed when Grid Snapping is on. Simply enable this option to have the Grid Always display.

Reset Ruler - This removes any snap marks (red Xs) on the view.

Set As System Defaults

Layers - By clicking on the Layers tab, you have a shortcut to the Layers settings for the current layer and the current view. This is the same as making changes to Layers as described here.

Right Clicking a Work Plane in the Sidebar

There are 4 options when right clicking a Work Plane, other than the Primary Work Plane which can not be copied) in the sidebar:

 

 Copy - Copies the selected Work Plane and allows you to place it graphically. The Primary Work Plane named "Primary Work Plane" can not be copied.

 Delete - Deletes the selected Work Plane. The primary Work Plane named "Primary Work Plane" can not be deleted.

 Edit - Takes the selected Work Plane to the CAM Editor.

 Properties - Brings up the Properties for the selected Work Plane.

Overrides - Displays, and allows the removal of, and Overrides on the selected Work Plane.

Parameters - Displays, and allows the addition or editing of, Parameters on the selected Work Plane.

Right Clicking an Operation in the Sidebar

 

 Delete - Deletes the selected Operation.

 Properties - Brings up the Properties for the selected Operation. There are two tabs available.

Overrides - Displays, and allows the removal of, and Overrides on the selected Operation.

Parameters - Displays, and allows the addition or editing of, Parameters on the selected Operation.

Right Clicking a 3D Model in the Sidebar

 

 Copy - Copies the selected 3D Model.

 Delete - Deletes the selected 3D Model.

 Edit - Takes the selected 3D Model to the CAM Editor.

Transform - Allows you to Mirror and/or Rotate the selected 3D Model.

 Properties - Brings up the Properties for the selected 3D Model.

Creating a Model

When modifying or creating a Model for a Drawer Guide it is important to note that you define the Left Guide and the system automatically creates a right guide and it's operations (placed on the Primary Work Plane) as a mirror of what you have defined for the left Guide.

In the following example we are going to create an Entry Knob Model which includes an Inside Door Knob and Outside Door Knob and a Strike Plate. To start create a new Miscellaneous Material named "Entry Knob". Once in the Properties area with this new Material make sure to go to the Miscellaneous Group and set the Model option to True.

From the sidebar select the "Assembly" Object. This Object is simply a sample part to help you orient your Graphic. This object is displayed in brown on screen. For the following Tutorial go ahead and change the DZ (Thickness) of the sample Assembly Object to 1.25".

From the sidebar select the "Primary Work Plane" Object. This is the default Work Plane and also represents the insertion (Reference Point) of the Model. Think of this as the container for your Model, everything will be referenced from this Object. The center of the Work Plane object is the insertion (Reference Point). Change the X and Y of the "Primary Work Plane" to 4". The true position of the "Primary Work Plane' will be set when the Model is placed within a Job whether automatically, manually, or via UCS.

  1. Set the View Mode to Front.

  1. With the "Primary Work Plane" object selected in the sidebar click the Add Work Plane option in the ribbonbar.

Additional Work Planes will search for matching surfaces and transfer their non-master operations (such as shaped routes). This search is only within the Part that the primary surface touches..

  1. Set the "Front WP" X and Y to 0.

  1. Click the Add Hardware option in the ribbonbar.

  2. Simply browse to the 3D Model you wish to import making sure to set the appropriate file type (highlighted below) then click Open.

The models used in this tutorial are available on the DVD at \Support\Sample Models\.

  1. The next screen offers several modification options you can perform prior to importing the 3D Model. See the topics on Importing DXF or Importing SketchUp Models for more information on the modification options on this screen. Click OK to import your model.

See the DXF Import topic for information for the formatting requirements of 3D DXF files.

  1. Once imported I am going to change the following Properties for the Inside Knob:

    1. Name = IKNOB

    2. Description = Inside Knob

    3. Finish = Chrome

    4. Finish Type = Metal - Matte

    5. X = -1.25 (This will center my knob horizontally on the Plane)

    6. Y = -1.25 (This will center my knob vertically on the Plane)

  1. Now we need to add some machining for the Inside Knob for mounting.

  2. Right click on the "Front WP" Object and select Edit. You could also double click to edit.

  3. Select Work Plane - Front from the sidebar to add Operations to it.

  4. Now we will add holes for the plate screws and the shaft. See the Part Level Main Tab topic for more information on the CAM Editor options.

    1. Hole #1

    1. Hole #2

    1. Hole #3

 

See the CAM View Main Tab topic for more information on adding CAM (Machining) Operations.

  1. Click the Return ribbonbar option to leave CAM mode.

  2. Switch to 3D Mode to see the Operations that were added.

  1. Next, select the "Primary Work Plane" Object in the sidebar.

  2. We want to add the Outside Entry Knob to the Back of our sample Part so switch the View Mode to Back.

  3. Next, click Add Work Plane to add a new Work Plane to the back of the sample part.

  4. Next, select the new "Back WP" from the sidebar.

  5. Change X to 0.

  6. Change Y to 0.

  7. Now click the Add Hardware option.

  8. I will select and Import my Outside Entry Knob model and click OK.

  1. Once imported I am going to change the following Properties for the Outside Knob:

    1. Name = OKNOB

    2. Description = Outside Knob

    3. Finish = Chrome

    4. Finish Type = Metal - Matte

    5. X = -1.25 (This will center my knob horizontally on the Plane)

    6. Y = -1.25 (This will center my knob vertically on the Plane)

  1. The Outside Knob does not require machining so we can move to the Strike Plate. Switch the View Mode to the Left.

  2. Select the "Primary Work Plane" Object from the sidebar.

  3. Select the Add Work Plane option from the ribbonbar.

  4. Change Y to 0.

  5. Click the Add Hardware option from the ribbonbar.

  6. I will select and Import my Strike Plate model and click OK.

  1. Once imported I am going to change the following Properties for the Strike Plate:

    1. Name = SPLATE

    2. Description = Strike Plate

    3. Finish = Chrome

    4. Finish Type = Metal - Matte

    5. X = -.5 (This will center the Strike Plate horizontally on the Plane)

    6. Y = -1.125 (This will center the Strike Plate vertically on the Plane)

    7. Z = -.0625 (This will let the Strike Plate into the sample Part Edge by 1/16")

  1. Now we need to add some machining for the Strike Plate for mounting. Right click on the "Left WP" Object in the sidebar and select Edit.

  2. Select Left WP from the sidebar to add Operations to it.

  3. Now we will add holes for the mounting screws, shaft, and a pocket to let in the plate.

    1. Hole #1

    1. Hole #2

    1. Hole #3

    1. Pocket (Use Rectangle with Type set to Pocket).

  1. Click the Return ribbonbar option to leave CAM mode.

  2. Switch to 3D Mode to see the Operations that were added.

  1. If you'll notice the knob is too far away from the edge of the sample part (if placed in the real world), let's change that. Select the "Primary Work Plane" Object from the sidebar.

  2. We want to change the X Position to be 2-3/8" away from the edge. Enter 2.375 for the X.

  3. Now you can see our final result. Click the Return option in the ribbonbar to save your new Model.

Now lets place this piece of hardware on a Part in Cabinet Vision.

  1. Save the Model and close the Material Manager.

  2. Now let's create a Part in the Part Manager to associate this Entry Knob to. Open the Part manager and select the Pull Group.

  1. Click the New option in the Ribbonbar.

  2. Type "EKNOB" for the Reference Name and give it a Description of Entry Knob.

  3. Close the Part Manager.

  4. Open Material Schedules.

  5. Select the Pull Material Schedule Type, and pick a specific Material Schedule.

  6. Now assign the Entry Knob Material we created earlier to the Entry Knob Part.

  7. Start a new Job making sure that Pull Material Schedule is active.

  8. Draw a Wall.

  9. Place the Empty Base Object from the Custom Cabinets | Base Cabinets group.

  10. Re-size the Empty Base Object to 36" wide, 80" high, and 1-1/4" deep.

  11. Take the Empty base Object to the Assembly Level | Face Ortho View.

  12. Click New Part in the sidebar.

  13. Select Board from the Part Type list.

  14. Click Next.

  15. Make the Length 80" and the Width 36".

  16. Since I do not have a 1-1/4" Material I will simply select the Thickness option and change it to 1-1/4".

  17. Click Next.

  18. Select Front for the Orientation.

  19. The default Finishes and Textures are OK here so click Finish.

  20. Change the Left/Right for the new Part to 0 in the sidebar.

  21. Click anywhere on the screen to de-select the New Part.

Now that we have a new Part in place let's put our Entry Knob on it.

  1. Click New Part in the sidebar.

  2. For Part Group select Pull.

  3. Select Entry Knob from the Part Type list.

  4. Click Next until you get to Finish then click Finish.

  5. With the Entry Knob selected set the Left/Right to 2-3/8".

  6. Set the Up/Down to 36".

You will see that your Entry Knob is placed along with all the machining necessary for mounting.
