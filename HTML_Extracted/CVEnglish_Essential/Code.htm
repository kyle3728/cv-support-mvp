Toolbar Tools

 Save - Saves the UCS.

Cut - The Cut option allows you to cut text to the Windows Clipboard so you can move it to another location, select the text, then click the Cut tool.

+

 

Copy - The Copy option allows you to copy text so you can paste it in another location, select the text, and then on the Right Click Menu click Copy.

+

 

Paste - The Paste option allows you to paste text you have cut or copied, place the cursor where you want to paste the text, and then on the Right Click Menu click Paste.

+

 

Find - You can quickly search for every occurrence of a specific word or phrase. In the Find field, enter the text that you want to search for. Select any other options that you want.

+

 To find only text with uppercase and lowercase characters, as specified in the Find field, select Match case.

You can specify which direction to search starting from the insertion point in the document. In Direction, click Up to search from the insertion point to the top. Click Down to search from the insertion point to the bottom.

 

Find Next - This will allow you to Find the next instance of the Text string entered in the Find option.

 

Replace - In the Find field, enter the text that you want to search for. In the Replace with field, enter the replacement text. Select any other options that you want. Click Replace to replace the first instance, or Replace All to replace all the matching text found.

+

 

Undo - You can reverse the last action in the UCS Editor by using the Undo command. Each click reverses one more action. Continue clicking Undo to remove as many of the previous changes as necessary.

+

 

Redo - This option will reverse the last Undo operation.

+

 

Toggle Bookmark - Adds or removes a Bookmark from the selected line

 

Next Bookmark - Moves the active line to the Next Bookmark.

 

Previous Bookmark - Moves the active line to the Previous Bookmark.

 

Clear All Bookmarks - Removes all Bookmarks from the UCS.

 

Line Numbers - Enables or disables Line Numbers.

 

Right Click Options

All the toolbar tools above are available in the Right Click menu.
