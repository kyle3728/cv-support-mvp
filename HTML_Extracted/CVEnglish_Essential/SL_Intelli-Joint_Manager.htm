Intelli-Joint Manager

What is an Intelli-Joint?

An Intelli-Joint applies operations to separate Parts that come together. When the two Parts come together in a specified manner (the Parts must either touch or be at an exact specified distance from each other in order to trigger the Intelli-Joint to be applied), you may have the Intelli-Joint place operations on either of the two Parts or both of them.

Intelli-Joint Ribbonbar

Following are the options available in the ribbonbar.

 Return

Will Return you to the System Level.

 New

Option to Name and create a New Intelli-Joint.

 Copy

Option to Copy the selected Intelli-Joint.

 Delete

Option to delete the selected Intelli-Joint

 Rename

Option to rename the selected Intelli-Joint.

 Mirror

Option to Mirror the selected Intelli-Joint. Mirroring the Intelli-Joint creates a copy of that joint and changes the face of operations in the new joint to the mirror opposite of the original operation. For example, if you mirror an Intelli-Joint vertically and there is an operation that is performed on the top face, the mirrored operation in the new joint will appear on the bottom face. Horizontal mirroring changes the new operation from left to right (and vice versa). You also have the ability to mirror both horizontally and vertically at the same time.

Intelli-Joint Selector

Option to select which of the existing Intelli-Joints you would like to edit.

 Preview

Opens the Intelli-Joint Viewer allowing you to simulate your Intelli-Joint on objects.

In the box on the top of the above picture, you can see the "normal" face of the Intelli-Joint. The Intelli-Joint appears in green with operations in blue and red. The origin point for the Intelli-Joint is at the bottom left corner of the green rectangle representing it. Notice that operations may be placed outside of the boundaries of the Intelli-Joint.

The middle box in the picture above shows the view of the Intelli-Joint as if you were looking down the length of the Intelli-Joint with the origin point on the closer end. To assist in your understanding of the perspective, the operations are the same color in all views. The bottom box is a perspective view of the Intelli-Joint.

You may zoom in or out of any of the three views. To zoom in, simply click in that view window. To zoom out, right click in the window.

Operations Tool Bar

The first section of this portion of the interface is the Operations toolbar. There are several tools available here:

New Operation

Allows you to add a New Operation to the list.

Copy Operation

Allows you to copy the selected Operation.

Operation Parameters

This option may be used to add a user defined Parameter to the selected Operation.

To add a Parameter:

    1. Click the Add button.

    2. Name the Parameter in the Name field.

    3. Choose either a value to set the Parameter to or you can choose to set the value through an equation.

Operations List

This is the area where Operations that are included/added to the Intelli-Joint are displayed and modified. There are eleven columns (Properties) you can control here.

Operation - This is a name for the Operation. This Name will display in the Operations area of the Part Editor.

Master - This controls where the Operation goes. It can be placed on either the Master or the Slave Part.

Type -  This option allows you to control the Operation Type (Bore or Dado).

Face - The operation is not limited to being on the face of the Part that actually touches the other Part (the normal face). The operation can be on the Left, Right, Top, or Bottom Face.

Pos X - The X position of the Operation if the Intelli-Joint is viewed in the normal orientation (the top view of the Intelli-Joint at the right of the Intelli-Joint Catalog window). Click the button and the field and you can then type a hard Value, a Variable, a Formula, or a combination of the three.

There are nine pre-configured Variables you can use to help build formulas, they are:

        • DX - This represents the Width of the Intelli-Joint.

        • DY - This represents the Length of the Intelli-Joint.

        • DZ - This represents the Thickness of the Intelli-Joint.

        • _MP:DX - This represents the Width of the Part the Intelli-Joint is attached to.

        • _MP:DY - This represents the Length of the Part the Intelli-Joint is attached to.

        • _MP:DZ - This represents the Thickness of the Part the Intelli-Joint is attached to.

        • _SP:DX - This represents the Width of the Part the Intelli-Joint is contacting.

        • _SP:DY - This represents the Length of the Part the Intelli-Joint is contacting.

        • _SP:DZ - This represents the Thickness of the Part the Intelli-Joint is contacting.

Pos Y - The Y position of the Operation if the Intelli-Joint is viewed in the normal orientation.

Depth - The Depth of the Operation.

Space Length - If the Operation is a Dado, this is the Length of the Dado. If the Operation is a Hole, this is the space between Holes (for multiple Holes).

Diam. Width - This is the Diameter if the Operation is a Hole and is the Width if the Operation is a Dado.

Quantity - This is the number of Holes if the Operation is a Hole. If the Operation is a Dado a Quantity of 0 indicates no dado which could be useful as a result from a formula calculation, any number 1 or higher for a Dado is considered 1.

Tool - This is used to specify a certain Tool to cut out the operation. Selecting this field will display a Tool selector in the Operations Tool Bar. If Auto is selected, normal Automatic Tool Selection Logic is followed. Also, if the selected Tool won't work for some reason, the Automatic Tool Selection Logic will be followed.
