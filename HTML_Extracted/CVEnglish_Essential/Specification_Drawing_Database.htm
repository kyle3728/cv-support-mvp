The Drawings.Sym File

The drawings.sym file stores all CAD library objects (symbols).  The following is the breakdown of the tables and their fields:

Header

This Table stores general information about the Database.

Field

Description

Version

A number indicating the version of the Database. The Database version changes each time fields are added/removed from the Database.

Menus

This Table stores general information about the Drawing Symbols.

Field

Description

MenuID

A unique number used to reference the menu.

Parent

A reference to another [MenuID] if this menu has a parent.

ItemNumber

The sort order for a group of menus.

Label

The Name of the Symbol.

SceneID

A cross-reference to the [Scenes].[ SceneID] field.

Type

 

 

 

 

 

 

Indicates the template type for the drawing symbol

0=None

1=Title Block

2=Assembly Sheets

3=Wall Elevations

4=Floor Plans

5=Drawing Scenes
