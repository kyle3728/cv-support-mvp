Crop -

There are three main Crop options in the sidebar once the Crop Tool is selected:

Crop Type - You have three Crop Types available; Rectangle, Circle, and Polygon that determine the shape of the area being cropped.

Crop Area - The Inside option will get rid of what is Outside the Boundary and keep what is Inside. The Outside option specifies that you want to keep what is outside the Crop Boundary and get rid of the CAD objects inside the Boundary.

The Inside option can be a bit aggressive as it will remove all CAD in the current view if it is not inside the boundary. To keep this option from being so aggressive you can limit the CAD Objects that are affected by a Crop by selecting the CAD Objects first.

Remember you can always click Undo option in the Quick Access Toolbar or use the + Undo shortcut to Undo your changes.

Remember you can select multiple CAD Objects by holding the key on the keyboard down as you click the CAD Objects.

Crop Frame - Selecting yes will leave the Boundary in the View once the Crop is complete.

Lines

The Lines tool has three options:

Line

This option allows the creation of Lines for use in drawings or as overlays for your scenes in CABINET VISION. Â After selecting and placing a line, the line properties window will display in the sidebar.

Chained Line

This option allows you to quickly draw two or more lines that are connected. With the Line tool above you must specify a start and end point for each line drawn. With chained line you specify a start point and when specifying the end point you are automatically specifying the start point for the next line. When you are finished drawing your group of Lines simply right click to end the Chained Line action. The end result will be individual Lines and when selecting them you will be presented with the same sidebar options as Line above.

Line Box

This option, similar to the Rectangle option, allows you to quickly draw a Rectangle made up of individual lines. The end result will be individual Lines and when selecting them you will be presented with the same sidebar options as Line above.

Rectangle

This option provides for creating a Rectangle (or square) on the page by clicking the point where it is to start and clicking on the point where it is to finish. It can thereafter be adjusted by using the Select tool. When using this tool you are creating a fully enclosed shape, you can break the Rectangle into its four individual lines by using the Right Click Break option. The area in the sidebar is similar to the one for the Line tool. You also have the added ability to add a hatch pattern to the enclosed space.

Pressing + Left Mouse Drag will force the Rectangle to be a Square.

Circle

This option allows you to create Circles. Click in the place where you want the center of the circle and pull the mouse to one side or the other and a Circle will develop. The area in the sidebar allows modifying the radius by numerically entering a value, as shown below.

If you select a circle after it has already been drawn, you will have these other options to alter it.

DXF -

The DXF option in the CAD area allows you to import 2D DXF's into the CAD area.

To Import a DXF Part:

  1. Click the Action Â button.

  2. Browse to the location of your DXF file, click on it, and click Open.

  1. Click the Import button.

Once your DXF file is displayed on the screen it is ready to be added to your Job by clicking Save. You can also, optionally, Scale the DXF or exclude Layers from the import by deselecting them from the list.

A 2D DXF file can define quite an assortment of CAD entities that CABINET VISION can not process. If CV can not process an entity it will be ignored and a warning will be issued. The following entities are supported by CV:

  • Arc

  • Circle

  • Line

  • LWPOLYLINE

  • POLYLINE

  • SEQEND

  • VERTEX
