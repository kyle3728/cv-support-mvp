Right Click On Screen -

There are five options when right clicking the screen in a Face, Plan, or End Ortho view:

  • Save - See the Room Level Save topic for more information.

  • To Drawing - See the Room Level To Drawing topic for more information.

  • Add Light - See the Room Level Add Light topic for more information.

  • Properties - See the Room Level Properties topic for more information.
