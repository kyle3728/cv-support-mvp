Select a Part -

There are eight sidebar options when selecting a Part in the Assembly Level Face, Plan, or End Views.

Any changes made to Parts utilizing these options are permanent and place an Override on the Part. Until this Override is removed the Part will loose its System Intelligence.

  • Left/Right - Allows you to move the selected Part Left or Right.

  • Up/Down - Allows you to move the selected Part Up or Down.

  • In/Out - Allows you to move the selected Part In or Out.

  • Length - Allows you to change the Length of the selected Part.

  • Width - Allows you to change the Width of the selected Part.

  • Thickness - Allows you to change the Thickness of the selected Part.

Changing the Thickness of Parts is not recommended, however Solid will allow it. The best case scenario is to allow the Material selected to control the Thickness and not to change the Thickness manually unless absolutely necessary.

  • Radius - Allows you to Radius the selected Part in the Plane visible in the active View (Face/Plan/End).

  • Rotate Part By - Allows you to Rotate the selected Part in the Plane visible in the active View (Face/Plan/End).
