Section Group

The Section Group has seven options:

Section Face

This option changes the View to allow sectioning the Face of your Assembly. See the Editing Assembly Faces topic for more information.

Section Interior

This option changes the View to allow sectioning the Interior of your Assembly. See the Editing Assembly Interiors topic for more information.

Horizontal Split

This option will split the selected Face or Opening Horizontally.

 

Vertical Split

This option will split the selected Face or Opening Vertically.

 

Horizontal Multi Split

This option brings up a screen to allow you to split the selected Face or Opening into multiple horizontal divisions of a selected type.

Multiple Face

Multiple Case

See the Set As System Defaults topic for more information on it.

Vertical Multi Split

This option brings up a screen to allow you to split the selected Face or Opening into multiple vertical divisions of a selected type.

Multiple Face

Multiple Case

See the Set As System Defaults topic for more information on it.

Delete

This option will delete the selected division or Part. If no division or Part is selected it will give you the option to Delete all Face and Case Parts instead.
