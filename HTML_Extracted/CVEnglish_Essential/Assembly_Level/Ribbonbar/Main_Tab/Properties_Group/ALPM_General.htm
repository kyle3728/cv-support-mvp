General

    

The General Tab at Cabinet Properties has 8 options:

  • Name - You can change the name of the Assembly here.

  • Quantity - You can change the quantity of the Assembly here.

  • Description - You can change the description of the Assembly here.

  • Class - You can change the Assemblies Class here.

  • Type - You can change the Assemblies Type here.

  • Size Width - You can change the Width of the Assembly here.

  • Size Height - You can change the Height of the Assembly here.

  • Size Depth - You can change the Depth of the Assembly here.

 

Remember changes made here will only affect the current Assembly. Also, changes made here are attached as permanent (unless you remove them) overrides on the Assembly; which means that if you Save this Assembly into a Catalog these overrides will also be Saved.
