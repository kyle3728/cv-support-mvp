Layers

This is the section of CABINET VISION where you can create, edit, copy, rename, and remove Layer Schedules.

New

Clicking 'New' will result in the addition of a new Layer Schedule. The new Schedule will appear at the bottom of the list and will be named 'New Schedule' until you click Rename and type in a new name for the Schedule.

Check for overlapped lines - Objects such as Cabinets in CABINET VISION are three dimensional objects. However, when two dimensional views of these Objects are sent to the drawings page there will be lines that are "stacked" on top of each other because they represent the top edge and bottom edge of the same Object. Check this option to remove those redundant lines.

Checking this option will greatly increase the time that it takes to send a view to drawings. If this option is not important to you it is best to leave it unchecked.

Copy

If you click Copy, you will create a duplicate of the current Layer Schedule which will be named 'Copy of [Layer Schedule name]' until you click Rename and type in a new name for the Schedule.

Rename

The Rename button allows you to type in a new name for the selected Layer Schedule.

Remove

Remove allows you to delete the selected Layer Schedule.
