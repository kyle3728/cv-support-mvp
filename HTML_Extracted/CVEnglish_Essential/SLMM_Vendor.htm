The Vendor utility allows you to add new or edit existing Vendors.

Ribbonbar

Return

Closes the Vendor utility and returns you to the Material Manager.

New

Creates a new Vendor.

Delete

Deletes the selected Vendor.

Working Area

The following Properties can be set for a Vendor.

Name - The Name of the selected Vendor.

Address 1 - Address #1 for the selected Vendor.

Address 2 - Address #2 for the selected Vendor.

City - The City for the selected Vendor.

State - The State for the selected Vendor.

Zip Code - The Zip Code for the selected Vendor.

Country - The Country for the selected Vendor.

Phone - The Phone number for the selected Vendor.

Mobile Phone - The Mobile Phone number for the primary contact at the selected Vendor.

Fax - The Fax number for the selected Vendor.

Email - The email address for the primary contact at the selected Vendor.

Contact - The primary contact at the selected Vendor.

Comment - Any comments about the selected Vendor.

Account Number - Your Account Number for the selected Vendor.

Sales Tax - The Sales Tax you pay when purchasing items from the selected Vendor.
