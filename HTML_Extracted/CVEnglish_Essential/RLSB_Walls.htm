Walls -

Upon entering the Wall Editor you will notice the ribbonbar changes. This topic will focus on four sections.

Layers

There are two options in the Layer selector:

Walls - The default Layer and the one the Walls are drawn on.

Construction - This Layer allows you to draw Construction/Reference Lines to assist in laying out your Room.

Depending on which Layer is active that some options in the Modify section of the ribbonbar may or may not be available.

Start X - Specifies the Start Point along the X Axis for the Wall.

Start Y - Specifies the Start Point along the Y Axis for the Wall.

Length - Specifies the Length of the Wall.

Angle (F3) - Specifies the Angle of the Wall. Pressing F3 on the keyboard brings up the Angle Calculator.

    • Standard

    • Peninsula

    • Cathedral

    • Vaulted Left

    • Vaulted Left

Type Wall End - Sets the Wall to a Wall End. This is a Wall that represents the end of a run. Unlike a normal Wall establishing a Wall End will turn off automatic Corner options as well as allow you to easily place a Cabinet against it. Wall Ends are also ignored in the Elevation Wall selector as they would not have Assemblies on them.

Dimensions Height - Specifies the Height of the Wall.

Dimensions Soffit - Specifies the Height of the Soffit.

Dimensions Thick - Specifies the Thickness of the Wall.

Dimensions Vault Height - Specifies the Height of the Vault above the current Wall Height.

Dimensions Vault Position - Specifies the Position of the Vault along the Wall.

Drawing Section

There are four options in the Drawing section:

Line

Using the Line option allows you to draw one Wall at a time by selecting the start point then the end point. See the Line option in the CAD Main Tab topic for more information.

Chained Line

The default Wall drawing mode. Using the Chained Line option allows you to draw connected Walls where each click establishes both end of the current Wall and the start of the next Wall. See the Chained Line option in the CAD Main Tab topic for more information.

Line Box

Using the Line Box option allows you to draw four connected Walls as a rectangle with two clicks establishing the two diagonally opposite corners of the Room. See the Line Box option in the CAD Main Tab topic for more information.

Arcs

Using the Arc option allows you to draw curved Walls. See the Arcs option in the CAD Main Tab topic for more information on Arc By Radius and 3 Point Arc.

Arc by Chord is specific to the Wall Editor. See the Creating Curved Walls with Arc by Chord later in this topic for more information.

Drawing Walls

Click your mouse at the point you desire the Wall to begin. Note that creating the Walls in a clock wise direction around the perimeter of the Room will cause the dimension lines to appear on the outside of the Walls. This would be the normal procedure for placing Assemblies on the inside of the Room. After clicking the origin point for the Wall, release the left mouse button and pull the Wall in the direction you want it to run.

Note the information in the sidebar will change to reflect the Length and angle as the Wall changes. You are free to enter the value for the Wall length at any time by typing in a value. To do this, simply type the value, you do not need to click anywhere or perform any other action prior to entering the value, assuming that the data box for the Wall length is filled with a background color. If you desire to enter values for the other data boxes, use to move down, or +   to navigate upward.

If you are creating the Wall with the mouse only using Chained Line, which is the default, to complete the Wall, left click at the location for the end of the Wall, if you wish to stop drawing Walls simply right click the mouse and select the Select option to complete. To create subsequent Walls, simply drag the mouse to the end point of your next Wall.

Final adjustment to Wall dimensions can be made by left clicking on the Wall, and entering data in the sidebar as described above.

Right click on a Wall for options regarding "Connect Left" or "Connect Right". These options will cause the Walls to connect if they are close enough to one another and not connected to another Wall at the same end. See the Right Click a Wall in Plan View topic for more information.

Connecting A Wall to the Face of Another

When Drawing a Wall if you click on the Face of another Wall as the Start point the new Wall will be connected to the face of that Wall and then allow you to slide the start point along the Wall. When you are happy with the start point simply click then drag the Wall to Length.

If you click on the Face connected Wall you will be able to set the Walls position accurately along the Wall using the Clearance options in the sidebar.

Creating Angled Walls

To create angled Walls you can simply drag to the Angle desired and eyeball the Angle value in the Left window. Or you can, while dragging the Wall, press the function key F3 on the keyboard to bring up the Angle Calculator. You can use the Angle Calculator to calculate inside angles only, using the 3/4/5 method.

Simply enter in the 3 values A, B, and C. Click the Calculate button and then OK to lock the Angle. At this point simply click the end of your Wall or type a value for the Length of the Wall and press .

Creating Vaulted,  Cathedral, or Peninsula Walls

Vault and Cathedral -

Either before starting to draw your Wall or by selecting the Wall once drawn you can control the Wall's Style by selecting an option from the Style option in the sidebar.

The Wall Style option provides for Standard (rectangular) Walls, Peninsula Walls, Cathedral Walls (which slope to a high point mid length), and Vaulted Wall (right or left slope to high point). Below is a graphic that explains the Cathedral and Vaulted options.

Note that all changes made in this window are for this wall only. If you desire to change more than one Wall, you must select each Wall that is to be changed. Regardless of any choice you make here, these changes will not be set as system default (except Hatch Type or Back Type), and therefore, will have no effect on future Walls. Remember, the Soffit is not a physical object, but represents the distance you desire to hold your Wall and Tall Cabinets down from the physical ceiling.

Notice how the Cathedral Height and the Vaulted Height are represented by CABINET VISION. The Wall height is set at 96 inches in this example, and the actual height is greater due to the ceiling condition. The angle of the vault of Cathedral is related to the width of the Wall. The height remains constant as the angle flattens with wider Walls. This angle is adjustable for Vaulted Walls.

Cathedral Walls are adjustable as well, and the vault location for the Cathedral may also be adjusted.
