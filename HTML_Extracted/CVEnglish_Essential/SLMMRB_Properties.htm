Properties

When you have a Material selected in the Material Manager you can get to it's Properties by clicking this option in the ribbonbar (alternatively double click the Material or right click and select Properties from the menu).

Ribbonbar

Following are the ribbonbar options in the Material Properties area:

Return

This option will return you to the Material Manager.

New

The New option allows you to create a New Material in the Material Properties area. See the Material Manager New topic for more information.

The new Material will be placed in the same Category as the currently selected Material.

Copy

Allows you to Copy the currently selected Material.

Delete

Allows you to Delete the currently selected Material. After clicking the Delete option you will be prompted to Confirm the Delete operation. If you no longer want to be prompted you can enable the "Do not ask me again" option.

Select

Allows you to Select a different Material to work with in Material Properties.

The sidebar in the Select Material window allows you to Search for a Material or browse for Materials via the Material Categories. Once you have located your Material simply select it from the list of Materials in the Work Area and click OK. If you need access to System (non-User Materials) you can click the System Materials tool . System Materials will be red in the list.
