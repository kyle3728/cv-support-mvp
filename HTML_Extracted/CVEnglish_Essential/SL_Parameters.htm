Parameters

The Parameters option allows you to establish your own Custom Parameters (Attributes or Notes) or establish the default values of System Parameters (Attributes). You can use these Parameters to trigger UCS's, Object Intelligence, or add Notes. There are six values set here.

Prompt - This field allows you to establish a user friendly prompt or name to simplify use by other Solid users. The Prompt value will show up on the Attributes Tab at the Parameters Class (Job/Room/Cabinet) or in the sidebar.

Parameter - This is the name of the Parameter. This is the Name displayed in the Object Tree and also used by UCS's to check or set it's value. It is recommended that you prefix your Parameters with U_ to help differentiate between System Parameters, of course this is only a recommendation as you can name them whatever you like.

Value - This is the default Value for the Parameter. You can also define a choice list of Values for selection here.

The choice selection is handled through a choice list assigned to a parameter. The format of the list is a list designator "", followed by choices separated by the "|" symbol. Each choice consists of a prompt and a value separated by the "=" symbol. The first choice in the list is considered the default. For example, Opt 2=2|Opt 1=1|Opt 3=3 designates a list of 3 choices with option 2 being the default. When displayed, the list will be sorted so the order after the first choice is insignificant.

Choice lists can be added to the System Parameters by typing adding a choice list for the value.

 Choice lists can be added to the Catalog Parameters by typing adding a choice list for the value.

 

Choice lists are imported/exported in the ORD files.

[Parameters]

Attribute="Stock","_CATCH","int",2,"Choice 1=1|Choice 2=2|Choice 3=3"

Choice lists can be added via UCS by typing a choice list for the value.

;Add a choice list through a UCS

 

For Each LF|RF Part

 

If _UCSLST = null Then

_UCSLST<int> = '<lst>Val 2=2|Val 1=1|Val 3=3'

_UCSLST<style> = 1

_UCSLST<desc> = 'UCS'

End If

Choice lists can be added manually through the Parameter Editor. The top item in the list will be the default.

If an Attribute parameter has a choice list, the list will be offered in the sidebar in place of the standard edit field.

If an Note parameter has a choice list, the list will be offered in the Note tab in the object properties.

If a System parameter choice list is modified, the list will be updated in Job objects when the "Update Job" utility is run.

Type - Allows you to specify the Values type. The choices are Measurement, Degrees, Integer, Boolean (True/False), Decimal, or Text.

Class - This allows you to specify which Level the Parameter will be placed in. The choices are Job, Room, Cabinets, Base Cabinet, Wall Cabinet, Tall Cabinet, Vanity, Closets, Closet Standard, Closet Shelf (Horizontal), or Closet Panel (Vertical).

Style - This option allows you to specify whether the Parameter is defining an Attribute or a Note. A Parameter defined as a Note will add a Notes tab to the Class level it is defined for. These Notes can be called up and utilized in CAD Tables and CAD Text Lookups.
