Molding

This topic covers all the Properties for the Molding Material Type. The Molding Material Type is designed specifically for buy-out Molding that has a SKU that covers all its properties. So Crown 123 in Oak 10' long has the SKU C123OAK10. If you are ordering your Molding that way then the Molding Material Type is designed specifically for you.

 Molding you manufature should be defined in the Molding Manager and have a simple Board Stock Material instead assigned to it.

Molding Group

Type - Type defines the style of Molding as one of the following Molding Types:

  • Crown

  • Top Edging

  • Scribe

  • Light Rail

  • Scribe

  • Base Board

  • Chair Rail

  • Casing

  • Applied

  • Ceiling

  • Bead

Length - Defines the Length of the Molding.

Cost Per Joint - Defines the price you charge for each joint.

Miter Trim - Defines the amount you compensate for the Moldings Length at each Miter.

When using a Molding Material Type within the Job the use of the Molding will be optimized to choose the best Lengths of Moldings available. It is highly recommended to setup Aliases for your Moldings which include all of the Lengths you buy your Moldings in.
