UCS Evaluation Order

This topic is a look into the way that CABINET VISION generates Assemblies and the order that it evaluates parameters. There are three basic steps that CABINET VISION goes through to generate an Assembly. These steps are:

  1. The Assembly is generated according to the Construction Method.

  2. User Created Standards are processed for that Assembly. See below for details about the way that User Created Standards are processed for an Assembly.

  3. The Graphics for the Assembly are generated.

UCS Evaluation For a Assembly

When User Created Standards are evaluated for an Assembly, they are evaluated in the order that they (User Created Standards) occur in the list of User Created Standards. UCS #1 is evaluated for the current Assembly, then UCS #2 is evaluated for the current Assembly, etc.
