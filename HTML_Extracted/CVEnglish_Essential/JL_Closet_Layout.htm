Sizes

The sizes area allows setting Heights and Depths of Closet Types for the Job. The values entered here should be the most common Heights and Depths needed for the Job. Entered values may still be changed for a Room or an Object once the Job is created.

Standard - Only affects Objects saved as Closet Class/Standard Type.

Vertical - Only affects Objects saved as Closet Class/Vertical Type.

Horizontal - Only affects Objects saved as Closet Class/Horizontal Type.
