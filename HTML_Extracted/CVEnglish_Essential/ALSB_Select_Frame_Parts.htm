Select Face Frame Parts Section View - Section Face

Select a Left or Right Stile

When selecting a Left or Right Stile you are presented with the following sidebar options:

  • Width - Allows you to change the Width of the selected Frame Part.

  • Length - If enabled, allows you to change the Length of the selected Frame Part.

  • Lock Width - Allows you to lock the Width of the selected Frame Part.

  • Adjustments Width - Allows you to adjust the Width of the selected Frame Part. An adjustment made here will not affect other Parts, or the Assemblies overall Size.

  • Adjustments Height - Allows you to adjust the Length of the selected Frame Part. An adjustment made here will not affect other Parts, or the Assemblies overall Size.

  • Adjustments Elevation - Allows you to adjust the Elevation of the selected Frame Part. An adjustment made here will not affect other Parts, or the Assemblies overall Size.

Select a Top or Bottom Rail

When selecting a Top or Bottom Rail you are presented with the following sidebar options:

  • Width - Previously Covered.

  • Length - Previously Covered.

  • Lock Width - Previously Covered.

Select a Mid Rail

When selecting a Mid Rail you are presented with the following sidebar options:

  • Width - Previously Covered.

  • Length - Previously Covered.

  • Lock Width - Previously Covered.

  • Adjustments Left - Allows you to adjust the Length to the Left of the selected Frame Part. An adjustment made here will not affect other Parts, or the Assemblies overall Size.

  • Adjustments Right - Allows you to adjust the Length to the Right of the selected Frame Part. An adjustment made here will not affect other Parts, or the Assemblies overall Size.
