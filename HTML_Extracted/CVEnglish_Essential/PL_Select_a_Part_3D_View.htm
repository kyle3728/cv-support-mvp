Select a Part 3D View

When selecting a Part in 3D View you gain the ability to change some of that Objects Visual Properties. You can select more than one Object at a time by using + Left Mouse Click.

  • Visual - Color - Changes the Color (Finish) on the selected Part(s).

  • Visual - Texture - Changes the Texture on the selected Part(s).

  • Visual - Finish Type - Changes the Finish Type on the selected Part(s).
