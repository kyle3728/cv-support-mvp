Public Variables

Overview

You have the ability to declare a variable as Public in a User Created Standard. This will allow for easy editing of the variables values without modifying the UCS Code.

Syntax

Each public variable must be declared on a single line of code. The syntax is as follows:

Public VariableName[] [:]= DefaultValue [;Variable Description]

Examples:

Public BSpacing = imp(32) ;Bore spacing

Public Oversized := DX > 40 ;When is the cabinet oversized?

Modifying a Public Variable

Note the public variable declaration at the top of this standard.

These variables are exposed in the public variables property list on the left when that UCS is selected.

You can modify a variables value in the code or directly at the property list.

Once you have modified the variable from the property list the modified flag in the database will be set. From this point on the variable will always get its value from the property list. A workaround for this behavior is to modify the variable name in the Code editor.
