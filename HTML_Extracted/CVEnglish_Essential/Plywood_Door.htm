Creating a Plywood Door

  1. Click on the "Create A New Door Style" tool.

  2. Name the Door.

  3. Select a Category to store the Door on, use the "Slab" category.

  4. Click on the "Construction" tab.

  5. Click in the middle of the Door and select "Slab" as the panel type for this Door.

  6. Check the Door panel size it should match the size of the sample Door.

  7. Click on the "Profiles" tab and select the appropriate Profiles to use with this Door.

  8. Click on the "Materials" tab and select the appropriate Material Schedules to use with this Door. Â Be sure to select Material Schedules that have the correct Material selected for "Slab".
