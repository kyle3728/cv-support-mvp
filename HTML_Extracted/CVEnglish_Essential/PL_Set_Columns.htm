Set Columns -

Allows you to set the Order and choose which Columns in the Reports View to display. Simply drag the Column name to its new location.

Column Definitions

Operation Icons

The first column in the Reports View can not be moved and represents both the Type and Origin of the Operation.

There are 13 Icon shapes representing the Operation Type:

Route

Line Bore

Hole

Dado

Intelli-Joint

Alphacam Drawing

Dado Master

Blind Dado Master

Line Bore Master

Hole Master

Dove Tail Master

FastenLink™

Lamello™ Clamex™

These icons can be represented by one of 4 colors that represent the Operations Origin:

Red - System created operation

Red Speck - User modified System created operation

Yellow - User created operation

Blue - Slave/UCS created operation

Operation

The name of the Operation.

Pos X

The X Position of the Operation.

Pos Y

The Y Position of the Operation.

Pos Z

The Z Position of the Operation.

Width

The Width of the Operation.

Length

The Length of the Operation.

Depth

The Depth of the Operation.

Rept.

The Repetition of the Operation. Only used for Line Bore.

Dir X

The X Direction of the Operation.

Dir Y

The Y Direction of the Operation.

Dir Z

The Z Direction of the Operation.

Priority

Establish the Order in which the Operations will be machined. This is typically not necessary to establish as the S2M CENTER will handle this automatically.

See the Priority and Group Parameter documentation in the System parameters topic for more advanced information on setting Order of Operations.
