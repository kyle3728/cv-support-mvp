Objects -

The object library will appear in the sidebar. This option allows you to place previously saved Assemblies into the current Assembly as sub-assemblies. Simply select the objects that you want to add and then drag them out into the right side of the screen. Use the position and placement fields in the sidebar to place the object in the correct location.

You can Section Edit these sub-assemblies, once placed, by right clicking the line item for them from the Order Entry screen.
