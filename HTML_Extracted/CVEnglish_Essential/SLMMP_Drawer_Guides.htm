Drawer Guides

This topic covers all the Properties for the Drawer Guide Material Type. The properties that accompany each Drawer Guide play an important part in helping determine your Drawer Box size. However, this is only part of the process of Drawer Box sizing.

Material Group

Previously covered in the Banding Material Type topic.

Cost - If you are entering "Cost" information, be aware that the price you enter is for "each" Drawer Guide, not each pair. Therefore, enter 1/2 of the cost of the "pair" of Drawer Guides.

Guide Group

  • Type - Type defines the style of Drawer Guide as either Side Mount, Bottom Mount, Top Mount or System.

With Cabinet Vision the "Mount" indicates how the Guide attaches to the Assembly not how it attaches to the Drawer. Example a typical Blum Tandem Drawer Guide in Cabinet Vision is considered a Side Mount because it Mounts to the side of a typical Cabinet.

  • Height - The Height of the Drawer Guide.

  • Length - The Length of the Drawer Guide.

  • Extension - This value sets the graphic limit of how far the Drawer Box opens when you use the "Open Drawer" feature.

  • Screw Center Reference to Top - This option, if enabled, will set the reference for your mounting bores from the top of the Drawer Box. If not enabled the Reference will be from the Drawer Box Bottom. Sets the Graphic Reference Point placement.

or

  • Screw Center To Box Reference - This value sets your Mounting Bore position from the Reference selected above. Sets the Graphic Reference Point placement.

This value helps determine your Drawer Box Height if you Bore All Line Bore Holes and align your Drawer Guides to System Boring.

  • Back Notch Length - Allows you to specify the Back Notch Length.

  • Back Notch Height - Allows you to specify the Back Notch Height.

  • Back Notch Offset - Allows you to specify the Horizontal position of the Back Notch.

Clearance Group

  • Side Clearance - This is the "thickness" of the Drawer Guide. This value, multiplied by 2 and then subtracted from the opening width, will set the Width of a Drawer Box that has this Drawer Guide. This is the area where Drawer Boxes receive the information on their Width. If you want more than a typical 1/2 inch, for a little play for instance, consider entering a value of 17/32.

  • Minimum Below Box - Similar to Minimum Above Box (below), this defines the distance between the bottom of the Drawer Box side, (or the Drawer bottom, the lower of the two), and the first thing it would hit. If there is no Cabinet member in the way, it would reference the gap.

  • Minimum Above Box - This value, and the values entered in the Drawer Box/Roll Out Construction MethodÂ begin to set the Height of your Drawer Box. You are asked to enter a minimum value here, and that minimum value will always apply to the selected Drawer Guide. This represents the distance between the top of the Drawer Box and the first thing it would otherwise hit, or, the gap between it and another Drawer if there is no Cabinet member in between.
