Molding

Click on the Categories or Tabs below for more information

The Molding Tab allows the selection of the default Molding types for the Job. Overhang can be set for the Moldings individually to allow for miter joints or other conditions. Moldings may still be modified or changed in the Job on a Wall by Wall basis if desired.

This tab can be used to either select Molding you make...Select a Material Schedule which uses Board Stock Material and then a Molding profile as created in the Molding Manager. Or Molding you buy...Select a Material Schedule (or a Material using the Change button) which has the Molding profile as part of the Material. When using the second method the Profile option will be disabled since the Profile used is set by the Material selected. You can always revert a "Changed" Material back to the default Material established by the selected Material Schedule by clicking the Reset button.

You may also change the Molding Material Schedule for this Job via the Modify for this Job button.
