Cabinet Materials

Click on the categories or tabs below for more information

The Cabinet Materials tab of Job Properties is the place to choose the Material Schedule for your Cabinets. You will also note that there are buttons labeled "Modify for this Job." The function of this button is to make changes to the Assembly Parts Material Schedule that will apply ONLY to this Job. Use this feature for one time only changes that you will not be using again in the future. If you need to make changes that you will probably need to make in future Jobs as well, you should create a new Material Schedule from the Catalogs menu at the Splash Screen.

The Material Properties Menu allows selection of the Material Schedules you have previously set up, and assigning the schedules to Base, Upper (Wall) Cabinets, Drawer Boxes, and Roll Outs. Notice how the Material Schedules for the Standard Cabinets is different than the Material Schedules for the Exposed Interior Cabinets.

 To utilize the Exposed Interior Material Schedule in a Cabinet the Cabinet must be set to have an Exposed Interior. To do this simply right click on the Cabinet and select Properties. From here enable the Exposed Interior option.
