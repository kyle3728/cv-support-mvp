Room General

Click on the categories or tabs below for more information

The General Tab allows changing the name from the default "Room 1" to a more descriptive name.  Above the Room Name has been changed by clicking the "Name" box, and typing the new name in. Also, the Type of "Room" has been changed to "Floor" indicating that "Jim's Office" is an entire floor of  a multi-floored job-site. This is a cosmetic feature and changes as shown are not necessary in order to do a Job. If you do choose to change the Name, it will appear at the Layout Level.
