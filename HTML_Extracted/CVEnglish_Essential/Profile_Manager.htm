Profile Manager

The Profile Manager is where you create and edit Bead and Counter Top Edge Profiles.

Sidebar

There are several options in the Profile Manager sidebar:

  • Name - Allows you to Name or rename the selected Profile. The character limit for a Profile Name is 50.

  • Type - Allows you to set the Type for the selected Profile. Following are the available Profile Types:

    • Bead (Defines the Bead Shape/Width for Beaded Inset Face Frame Construction)

    • Top Edging (Counter Top Edge Profile)

  • Thickness - This value is controlled by the Shape of your Profile.

  • Height - This value is controlled by the Shape of your Profile.

  • Cost Per Job - - Allows you to set a Cost Per Job for the selected Profile. This value will be used by the Use Profile Catalog Pricing Bid Method in the Bid Center.

  • Cost Per Strip - - Allows you to set a Cost Per Strip for the selected Profile. This value will be used by the Use Profile Catalog Pricing Bid Method in the Bid Center. A "Strip" is determined by how the molding is designed in your Layout. Each separate connected run including returns is considered one "Strip".

  • Cost Per FT - - Allows you to set a Cost Per Foot for the selected Profile. This value will be used by the Use Profile Catalog Pricing Bid Method in the Bid Center.

Ribbonbar

Return

This option will Return you from the Shape Edit area. If changes were made to the Shape you will be prompted if you want to Save those changes.

Zoom Group

New

This option will allow you to create a new Profile.

Copy

Copies the selected Profile.

Delete

Deletes the selected Profile from the Profile Manager.

This Delete action can not be reversed.

Profile Type Selector

Allows you to select the Profile Type you wish to work with.

Profile Selector

Allows you to select the Profile you wish to work with.

Reset

Will remove all Shaping that has been applied to the current Profile.

This Reset action can not be reversed.

Edit

Takes the current Profile in the Shape Editor to allow you to define the Shape of your Profile (or the Position of Multiple Profiles within a Profile Set).

Techniques for Creating Profile Shapes

The easiest method of adding a Profile Shape is to import it into the Profile Manager. Editing is similar, and uses the same editing tools common to CABINET VISION to create the final shape.

Importing A DXF file to Define A Profile

In the following example, a DXF file such as the one shown, will be imported into Profile Shape for use as a Counter Top Edge.

    1. Take your new Profile to the Shape Editor .

    2. Click Select All then click Delete .

    3. Click Insert DXF .

    4. Click the browse button to browse to the location of your DXF file.

    1. Once you have Opened your DXF file from the Browse dialog click Import.

    1. Click OK to Import the DXF into the Shape Editor.

    2. Pick its location in the Shape Editor. It doesn't really matter where you place it.

    3. Once placed select the Reference option and pick the point which you want to be the Reference point on the Profile. See the Shape Editor topic for more information on the Reference option.

  1. Click Return to Save the shape of your new Profile.

  Keep in mind you have full CAD editing capability here so you can Flip, Mirror, Rotate, Scale, Stretch, and modify the imported shape in any way prior to saving it.

Importing an Image file to Define A Profile

When Importing an Image file you bring in an Image that will allow you to get the general shape of the Profile by tracing it utilizing the Shape Editor Tools. There are two additional Tools available when a Bitmap is imported that allow you to Scale and/or Align the Bitmap to facilitate proper tracing. Once you have the Bitmap sized as you need simply use the CAD tools to trace the image to get the final shape you desire.
