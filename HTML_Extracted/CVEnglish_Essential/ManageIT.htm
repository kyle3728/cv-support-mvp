Manage-IT

Manage-IT is add on option for CABINET VISION.

When you open CABINET VISION, the first screen you will see is the opening screen. The graphic below is the Manage-IT opening screen.

Getting Started

There are four main views in the Manage-IT splash screen:  Customers, Users, Vendors, and Contractors.  Proper setup is necessary in order to manage Job files using Manage-IT.  In each view, you can add Customers, Users, Vendors and Contractors, arrange them by type, as well as generate Manage-IT reports.  The available categories and reports vary, depending upon the view.

 

Jobs are associated with Customers and Customers are associated with Users.  Vendors and Contractors are also associated with Users.  Therefore, it is best to create Users first.

Users View - To setup users, click on the second view button.  To add a new user, click on the "New" button on the right (or right-click in the open area and select "New User"). Note:  Since the "Supervisor" field is a list of users, enter users in order, starting from the highest position within your organization.  The "Type" and "Status" fields are programmed selections that can be used for arranging the list of users.  The "Add" button (next to the "Alt. Phone(s)" field) allows you to setup additional phone numbers for this user.

  When a particular User is selected from the "Type" box, the Jobs for that user will appear in the bottom half of this window.

User Reports

Clicking on the Reports button in the sidebar, in the User View, brings up the options, in the sidebar, to print two reports:

      1. Jobs for Designer - This Report will display all Jobs in Manage-IT that are assigned to the selected User.

      1. Employee Report - This Report will display an Employee Information sheet on the selected User.

Vendors View - To setup vendors, click on the third view button.  To add a new vendor, click on the "New Vendor" button on the right (or right-click in the open area and select "New Vendor").  Note:  Since the "Account Manager" field is a list of users, it is best to enter users first.  The "Type" and "Status" fields are programmed selections that can be used for arranging the list of vendors.  The "Add" button (next to the "Alt. Phone(s)" field) allows you to setup additional phone numbers for this vendor.  The "Add Contact" button (next to the "Email" field) allows you to setup multiple contacts for this vendor.

Vendor Reports

Clicking on the Reports button in the sidebar, in the Vendor View, brings up the options, in the sidebar, to print one report:

      1. Vendor Report - This Report will display information for the selected Vendor.

Contractors View - To setup contractors, click on the fourth view button.  To add a new contractor, click on the "New Contractor" button on the right (or right-click in the open area & select "New Contractor").  Note:  Since the "Account Manager" field is a list of users, it is best to enter users first.  The "Type" and "Status" fields are programmed selections that can be used for arranging the list of contractors.  The "Add" button (next to the "Alt. Phone(s)" field) allows you to setup additional phone numbers for this contractor.  The "Add Contact" button (next to the "Email" field) allows you to setup multiple contacts for this contractor.

 

Contractor Reports

Clicking on the Reports button in the sidebar, in the Contractor View, brings up the options, in the sidebar, to print one report:

      1. Contractor Report - This Report will display information for the selected Contractor.

Customers View - To setup customers, click on the first view button.  To add a new customer, click on the "New Customer" button on the right (or right-click in the open area and select "New Customer").  Note:  Since the "Designer" field is a list of users, it is best to enter users first in the "User" view.  The "Type" and "Status" fields are programmed selections that can be used for arranging the list of customers.  The "Add" button (next to the "Alt. Phone(s)" field) allows you to setup additional phone numbers for this customer.  The "Add Contact" button (next to the "Email" field) allows you to setup multiple contacts for this customer.

 Managing Job Files

This section will focus primarily on how to manage Job files and understand Job Status in the Customer view.

To manage Job files in Manage-IT, a Job must be associated with a Customer.  

When a Job file was previously saved in Solid prior to adding the Manage-IT option, you will first need to highlight a customer in the top half of the Customer view.  Next, right-click in the bottom half of the Customer view and select "Import Job".  You can then browse for your .CVJ file and select "Open" to import it under the selected Customer.

To create a new Job in Manage-IT, you can select "New" in the Solid Button menu item or right-click in the bottom half of the Customer view and select "New Job".  Again, Job files are associated with a Customer, so be sure to highlight a customer in the top half of the Customer view before creating a new Job.

The default "Job Filename" will repeat the Customer Name, followed by an underscore and a number.  The number is automatically sequential, depending upon the number of Job files already associated with this Customer Name.  You can either click on OK to accept the automatic Job name, or type in a different name for your Job.  Additionally, you may add a comment in the "Job Comment" field.

The Job status upon returning to the Manage-IT Splash Screen, until your Job file has been opened and saved, will indicate, "Job file not created yet".

To open a Job, right-click on the Job in the bottom half of the Customer view and select "Open".  After making selections in the "Job Properties" window, click on OK to enter Layout Level.  To return to the Manage-IT splash screen from the Job, select "Close" from the Solid Button menu.  

Manage-IT allows you to save multiple versions of a Job file.  Until the "Save Version" option is used in the Manage-IT splash screen, the Job status will indicate, "New Job (no versions saved)".

To save a version of a Job, right-click on the Job in the bottom half of the Customer view and select "Save Version".  (Note:  if this option is grayed out, either the Job file has not been created yet OR has not been checked out by you.  A Job file must be opened or checked out before a version can be saved.)

You will then be prompted to write a comment for this version of the Job.  This is one of the most critical tools in Manage-IT.  Your Job Version comments will help you identify differences in multiple versions of the same Job.  For example, if you have the "rare" customer that frequently changes his/her mind, having an "initial version" will help you start over without actually having to start over.

There are two toggle boxes in this window.  Each of these choices has a significant effect on the status of the Job file.  By toggling ON "Keep Checked-out to you after saving" the Job status will read, "Checked out to ".  By toggling ON "Undo Check-out (Don’t Save Version)" the Job status will read, "Checked-in".  Additionally, "Undo Check-out" will NOT save a version of the Job and any change made by you will be lost.  Note:  The first time you save a version of a particular Job, "Undo Check-out" will not be available to you.

The Job status upon returning to the Manage-IT splash screen, will either read, "Checked-in" or "Checked-out to ".  Also, a version number is indicated in the "Version" column.

To view ALL versions of your Job file, right-click on the Job and select "Properties".  Highlight which version you would like to check out and use the "Check-out Selected Version" button.  If you want to copy a version for another Job that is similar, use the "Copy Version as New Job" button.

When a version of a Job file is checked out, other users are prevented from checking out the same or different versions of the Job file until you check it back in.

When a version of a Job file has been copied as a new Job, the default "Job Filename" will automatically repeat the Customer Name, followed by an underscore and the original Job number.  Since this is a copy of another Job file, an additional underscore, followed by another number identifies the "copy" number.  You can either click on OK to accept the automatic Job name, or type in a different name for your Job.  Additionally, you may add a comment in the "Job Comment" field.
