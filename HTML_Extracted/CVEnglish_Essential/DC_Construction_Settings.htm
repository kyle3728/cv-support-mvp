Construction Settings

Construction Settings are simply the default settings for any new Door Style that you create. Changing these settings will not have any effect on existing Doors in the Door Catalog. To change the sizes of Parts on an existing Door, you must select that Door and click on the Construction tab to make the changes to that Door.

Selecting Construction Settings will allow you to set up your normal, or default Construction Methods for all of your Doors. This will allow you to set up your Door Styles quickly, especially if your Stiles and Rails for the most part are the same size for most of your Doors. Here you have the opportunity to set the fields to their most common values. Typically, a default setting is chosen because it is the one used more often than not. Creating new Door Styles and redoing old Doors is easier if you have your Door Construction Defaults preset.

There are three areas to this screen.

Default Rail and Stile Widths

The  value entered here represents the portion of the Stile or Rail as it exists without any Inside Profile on it. The dimension of the Inside Profile, optionally depending on the Extended setting for the Inside Edge Profile, is in addition to the Stile or Rail width. If there is no Inside Profile, then the value set as the default width for the Stile or Rail is the net width of the Stile or Rail.  

If you highlight a Stile or Rail on the Construction tab of the Door Catalog, you will see the Stile or Rail width, and an "Extra Width". Remember, the overall width of the Stile or Rail is the default value, PLUS the Profile for the Inside Edge, if any.

You will note that in the graphic above that the Left Stile, Right Stile, Top Rail, and Bottom Rail all preset to 2 1/8”. In the graphic below the Left Stile of a Door is highlighted. If this Door were being produced, the finished width of the Left Stile would be 2 ½”. You will notice that the width of the Stile says "Width" + "Extra" these two measurements must add up to the overall width of the finished Stile. Since this Door is using a 3/8” Inside Edge Profile, setting the Width to 2 1/8” (2 ½ - 3/8 = 2 1/8) under "Default Rail & Stile Widths" will automatically fill in the "Width" portion of the "Width" + "Extra" with the 2 1/8” needed.

You will note that the Mid Rail and the Mid Stile are set to 1 ¾”. This is because to achieve a 2 ½” finished size you have to take into account the 3/8" Inside Edge Profile on both sides of the Mid Stiles and Rails.

2 x 3/8 IEP = ¾

2 ½ - ¾ = 1 ¾

Default Construction Methods

In the Construction Methods area you can setup for "Mortise & Tenon" construction and establish it's Depth. You can preset the Panel Inset you want for raised panel and flat panel type Doors. And you can control the width of the vertical and horizontal slats for slat type (tambour) Doors.

Mortise & Tenon Depth

Mortise & Tenon Depth is only necessary when doing Mortise and Tenon type Doors. Do not use this if your Door Construction Method is cope and stick or butt joint. When doing a Mortise and Tenon Door you would click the check box for Mortise and Tenon and enter the Depth of the Mortise and Tenon before creating a Mortise and Tenon type Door. Also make sure you do not select an Inside Edge Profile to use with a Mortise and Tenon Door, as Solid could add the Mortise and Tenon and the Inside Edge Profile to your Door Parts. After the Door is created come back here and disable Mortise & Tenon Depth.

Panel Inset

Panel Inset is the distance you want the Door panel to stick into one side of the Door. Having this preset makes creating Doors a lot easier.

Width of Vertical Slats

This establishes the default "Center-to-Center" spacing for Vertical Slats when the Vertical Slat Panel Type is defined for your Door.

Width of Horizontal Slats

This establishes the default "Center-to-Center" spacing for Horizontal Slats when the Horizontal Slat Panel Type is defined for your Door.

Default Panel Type

In the Default Panel Type area, you can preset which Type of Panel you want the Doors you are going to be working on to be. So that the Panel Type will automatically be whatever you have set here, when you start a new Door or when you click on the Start Over button on the Construction tab. So if you were going to work on raised panel Doors you could preset the Default Panel Type to Raised Panel. Or if you were going to work on glass Doors you would preset the Default Panel Type to Glass Panel.

Set Unit - Click this button to select your Unit of Measurement and the level of precision that you want to see in the Door Catalog.
