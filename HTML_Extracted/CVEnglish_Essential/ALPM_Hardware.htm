Hardware

The Hardware Tab at Cabinet/Closet Properties has the following possible options:

  • Pulls Schedule - Allows you to specify the Pulls Material Schedule to use with the active Assembly.

  • Drawer Guides Schedule - Allows you to specify the Drawer Guides Material Schedule to use with the active Assembly.

  • Wire Baskets Schedule - - Allows you to specify the Wire Baskets Material Schedule to use with the active Closet.

  • Closet Rod Schedule - - Allows you to specify the Closet Rod Material Schedule to use with the active Closet.

  • Hinges Schedule - Allows you to specify the Hinges Material Schedule to use with the active Assembly.

  • Beaded Frames Bead Profile - - This option is only available if a Hinge Schedule utilizing an Inset Hinge is selected. This option allows you to specify the Bead Profile to utilize if doing a Beaded Face Frame Construction. If you are not doing a Beaded Face Frame Construction do not select a Bead Profile.

  • Sliding Door Rails Schedule - Allows you to specify the Sliding Door Rails Schedule to use with the active Assembly.
