Drawings Library

This utility is specifically designed to help manage your Drawing Libraries. You can Organize Categories, Copy, Rename, and Delete Drawings, as well as move Drawings up and down the list with this Utility. You can also, through this utility, Edit your Drawing Templates (Assembly Sheets, Part Sheets, Elevation Sheets, and Title Blocks).

There are seven Drawings Types:

  • Symbols - This is where you will store any of your custom Drawings.

  • Title Blocks - This is where you will store your Drawing Title Blocks.

Title Blocks are available for Solid Essential users.

  • Assembly Sheets - This is where you will store your Assembly Sheet Templates.

  • Wall Elevations - This is where you will store your Wall Elevation Templates.

  • Drawing Scenes - This is where you will store your Drawing Scene Templates.

  • Part Sheets - This is where you will store your Part Sheet Templates.

  • Hatch Patterns - This is where you will store your Hatch Patterns.

    • Hatches must utilize the PAT file format. The PAT file format is a text file which represents a vector pattern. PAT Hatch pattern files are available via several websites and/or can be easily created in a text editor.

    • When you select the Hatch Type in the Drawings Library simply click the Import button to Browse for your PAT file.

    • PAT files can contain multiple Hatches and the importer will interpret and import each pattern within the PAT file separately.

    • You have a few other options specific to Hatches:

    1. By selecting any Hatch you will be able to Copy them.

    2.  With any Non-System Hatch you can set the CAD Defaults it utilizes (You can Copy any System Hatch in order to control CAD Defaults for it). Meaning how it appears when initially placed in CAD.

    3. You can control the following Defaults for Hatches:

      1. Rotation Angle

      2. Scale

      3. Line Width

The Organize option allows you to Add, Delete, and/or Rename Drawing Categories for the selected Drawing Type.

See the Right Click Save Symbol from the Drawings Area or Right Click Save Symbol from CAD topic for more information on saving your own Symbols.

 

Title Block, Assembly Sheet, Part Sheet, Wall Elevation, and Scene Templates

When selecting the Title Block, Assembly Sheet, Part Sheet, Wall Elevations, or Scene Types the New... and Edit buttons become active. Clicking New, to create a New Template, or Edit, to Edit a selected Template will take you to the Template Editor, where a dashed line shows the current paper (template) size/orientation.

The first step to creating a New Template is to establish your default Paper Size. To do this Right Click on the Sheet and select Paper.

The Paper Size and Orientation is important in Templates, because will instruct Solid how to stretch/adjust the drawings once they are applied to the real paper.

You can add your Custom Paper Sizes to this list by going to Main | Paper Sizes.

Locking Cad Objects

You can use any of the standard CAD Tools available in your Templates. In addition to normal CAD functionality you also, when in Template Mode, have the option to Lock CAD Objects to your Paper. As mentioned previously, the primary feature of a Template is to automatically adjust to the real paper size/orientation.

In order to control this behavior, every Cad Object in a template can be Locked in different ways.

Simply click on the little button associated to a Cad Object property to sequentially change its locking flags:

- The position will be kept at the same distance from the left edge of the Paper.

- The position will be kept at the same distance from the right edge of the Paper.

- The position will be kept at the same distance from the center (Horizontally) of the Paper.

- The position will be kept at the same distance from the top edge of the Paper.

- The position will be kept at the same distance from the bottom edge of the Paper.

- The position will be kept at the same distance from the center (Vertically) of the Paper.

NONE - The position will adjust proportionally to the Paper Size.

Every object is auto-locked when editing a Title Block created in a previous version of Solid for the first time.

The Auto-Lock Mode is applied to every new Cad Object added to a template, and can be deactivated/reactivated with the right-click menu.

The Auto Lock feature chooses the locking edge by dividing the paper size by 3 and checking where the Cad Object's position falls.

 

Container

When creating an Assembly Sheet and/or a Wall Elevation template, you need to place an Container object on the Paper to control the image(s) position and appearance during the printing process.

Only one Image Container Object can be placed on a Template.

You place an Image Container Object the same way you place a Rectangle or Table Object.

When setting up Assembly Sheet templates, the following window pops-up after you finish placing a new Image Container Object, and/or selecting Edit from the right-click menu.

The Standard Tab refers to Standard Assemblies, the Shaped Tab to Shaped Assemblies. A check mark means the selected view is active, a question mark means it's optional, nothing means it won't be printed.

Optional means that it will not Print out if Printing it causes another piece of Paper to be used.

Annotated Parts refer to any Part of that Assembly that has had CAD added to it at the Part Level.

When selecting an End View you will be presented with an option to set the Section Percentage. At 0% you will simply see an End View of the Cabinet. If you would like instead to see a cross section simply set the percentage inside the Cabinet you would like to see the Section line.

When selecting a 3D View you will be presented with Tilt and Rotation options for the 3D View presented.

Selecting the Additional Saved Views option will place any views for the active Assembly saved using the "Save for Assembly Sheet" right click option.

Once the Image Container Object is on the template, you can control it's Type in the sidebar. There are two type options:

1 Graphic

2 Graphics

If you select 2 Graphics you will get two additional sidebar properties.

Horizontal

Reverse
