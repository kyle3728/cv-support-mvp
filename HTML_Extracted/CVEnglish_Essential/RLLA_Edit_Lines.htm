Edit Lines

Highlighting any new or existing item in the list will allow you to Edit the item. Simply click the Edit button.

There are four main options in this screen:

  • Name - This is another area where you can rename the line.

  • Type - This selection box allows you to select which Type of Dimension you want to use with this line. There are three types:

    1. Wall - This selection sets the line to display across the length of the Wall.

    2. Wall End - This selection sets the line to display across the depth of the Wall.

    3. Island Counter Top - This selection sets the line to display around a Counter Top. Make sure to activate Dimension around on the Counter Top.

A Dimension Line may only have one of these selections. You cannot combine a Dimension Line to do both Wall and Wall End Dimensions.

  • Force Dimension Line. This feature is best defined in an example:

Setup the Left Dimension Line to show Wall Ends, Cabinet Top, Cabinet Bottom, Upper Cabinet Top, Upper Cabinet Bottom.  Setup the Right Dimension Line to show Wall Ends, Counter Top Surface, Counter Top Bottom.  If you have a design that does NOT have Counter Tops, Solid is designed to NOT show the Right Dimension Line since nothing in that dimension line actually exists in the design (see above example). However, by choosing "Force Dimension Lines", the Wall Ends will still be dimensioned on the right, even though there is no Counter Top Surface and no Counter Top Bottom.
