Profile

This option is only available for the Molding and/or Molding Set Material Types. This utility allows you to associate a Profile shape with the Molding Material.

There is one option here that is only available for Material Molding:

The Import > Profile option allows you to import a Profile Shape from the System Level Molding Manager. Once the option is selected your Molding Library of Profile Shapes will be available in the sidebar.

Simply double click the Profile in the sidebar to replace the current Profile shape.

See the Molding Manager topic for more information.
