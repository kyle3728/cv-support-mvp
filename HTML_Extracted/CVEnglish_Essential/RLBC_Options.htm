Options

Following are the Tabs in the Bid Center Options Area:

Totals Tab

There are eight options on the Totals Tab:

  • Default Deposit % – Allows you to set the Default Deposit % you require your Customers to pay for the Job.

  • Default Markup % – Allows you to set the Default Markup % for your Jobs.

  • Include Drawer Fronts in Door Count – If you treat your Drawer Fronts as Doors when Bidding by Door select this option. This would come into play when using the Per Door Bid Method.

  • Include False Fronts in Door Count – If you treat your False Fronts as Doors when Bidding by Door select this option. This would come into play when using the Per Door Bid Method.

  • Price Fillers As Cabinets – If you consider your Fillers Cabinets enable this option. Enabling this will include Fillers in Lineal Foot/Meter and Per Cabinet type Bid Methods.

  • Price Detached Toe Kicks As Cabinets – If you consider your Detached Toes Cabinets enable this option. Enabling this will include Fillers in Lineal Foot and Per Cabinet type Bid Methods.

  • Lineal Total For Corner Cabinets is Width + Depth – If you calculate your Lineal totals for Corner Cabinets by both width and depth select this option. See the two figures below for examples:

With this option Disabled the Cabinet below would calculate as 3 Lineal Feet.

With this option Enabled the Cabinet below would calculate as 6 Lineal Feet.

 

  • Include Doors, Drawers & Hardware in Part Count – If you include Doors, Drawers and Hardware in your Part Count calculations Enable this option. This would come into play when using the Per Part Bid Method.

  • Estimate Sheets of Material - With this option enabled the Bid Center will use the results of a generic panel Optimization to estimate Sheet Goods usage rather than simple area estimations. This functionality uses a generic Optimizer with the following limitations:

    • It will not use your Optimization settings even if you have the Optimizer, it uses predefined settings.

    • If there is an error during optimization for any reason (ie. part that won’t fit the sheet) you will get a Warning in the status bar about that Material and it will end Optimization on that Material and use the waste factor from the Material Catalog.

    • Buy out parts are never included in a sheet estimate

    • Any material that has a valid Sheet Size are estimated regardless of their "CNC" or "Optimize" Settings.

    • The grain property of the sheet is honored.

    • The Sheet estimate uses a 1/4” trim, 1/8” kerf, Block Nesting with no restrictions on phases. As a result the actual Optimization results may be better or worse depending on TrueShape versus Block and Yield versus Labor.

Rounding Tab

There are two main options on the Rounding Tab:

  • Line Item Rounding – There are three options for Line Item Rounding:

    • Round To The Nearest Penny – Selecting this option will Round all the Line Items in your Bid to the nearest Penny using standard Rounding rules.

    • Round To The Nearest Dollar – Selecting this option will Round all the Line Items in your Bid to the nearest Dollar using standard Rounding rules.

    • Round Up To The Nearest Dollar – Selecting this option will Round Up all the Line Items in your Bid to the nearest Dollar.

  • Totals Rounding – There are two options for Totals Rounding:

    • Show Cost Of Actual Total – Selecting this option will display the actual Total vs. a Rounded Total.

    • Round Cost To Display Total – Selecting this option will display the Rounded Total based on the selected Line Item Rounding option.

Show / Hide Tab

There are four options on the Rounding Tab:

  • Show Price Lines That Are $0.00 – Selecting this option instructs the Bid Center to display all Line Items in your Bid including those that have a value of 0. This is especially helpful when setting up your Rate Tables for the first time.

  • Show Hidden Lines In Internal Mode – This option is used for Stock Pricing.

  • Show Stock Cabinet Options At Opening Screen – This option is used for Stock Pricing.

  • Show Legacy Line Items & Variables – Selecting this option will allow or suppress the display of Legacy Bid Methods.

Grouping Tab

In the Assembly Prices (This is for Prices stored on the Assemblies not Rate Table generated) area you have three levels you can "Group" your Bid Prices by in the Breakdown area. There are three grouping options which can be used in any order:

  • Catalog

  • Room

  • Door Style

Password Tab

The Password Tab allows you to change the Password to access the Internal Mode of the Bid Center. Internal Mode shows a full Break-Out of what makes up the Bid. The default Password is solid. If you do not want to have to enter a Password to enter Internal Mode you can set a new Password that is blank and you will no longer be prompted for a Password.
