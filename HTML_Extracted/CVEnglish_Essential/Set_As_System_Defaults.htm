Set As System Defaults

Â Use this function to store the current settings as default values. For example, if you are entering the values for the Heights and Depths of the different Cabinet Classes, and you want every Job to have these new values input automatically, click the Set As System Defaults button when you finish entering the numbers. This will allow the values entered to appear each time a new Job is started so that you don't have to re-enter them in every Job. The values may still be modified thereafter.

System Default settings are stored in the Windows Registry.

In Job Properties once you Save a Job Template then the Set As System Defaults is no longer necessary and will not be displayed.
