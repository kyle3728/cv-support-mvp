Automatic Text

Whenever you add text through CAD to any area of CABINET VISION, you not only have the ability to type your desired Text but also have the ability of selecting from Text Variables of information already within your Job. The auto text appears inside the brackets {  } and is sometimes accompanied by additional static text.  The text outside of the brackets may be deleted leaving only that which is inside the brackets to be automatically filled in.  Automatic text is not limited to but is especially useful in Title Blocks.

Clicking the Action button will bring up the list of available Text Variables. The list of Text Variables will be different depending on what Level of CABINET VISION you are in.

Below is a list of the Text Variables and where the values in the variables are being pulled from.

The list below represents the text variables listed in the menu system. This list does not include "ALL" that are available. In addition to the text variables in the list you can also call out any Parameter in CABINET VISION whether it is a System Parameter or a User Parameter by following the format below and replacing the variable name with the Parameter you want to call.

Examples

To call out the AX Rotation of a Cabinet you would simply type {cab.ax}.

OR

To call our a User Parameter named CabNote1 you would simply type {cab.cabnote1}.

Text Format Options

There are three formatting options available specifically to Automatic Text:

u: - Sets first character to upper case.

l: - Sets first character to lower case.

n: - Displays negative numbers within parentheses

Example

{u:cab.cabnote1}

If a variable is not populated with data it will show the Variable name in CAD mode. However, in Normal View modes it will be invisible. This is a very handy feature that allows you to have Notes on Assemblies that maybe populated by a UCS only in certain conditions. This allows you to control the display of the note conditionally.

General

{dim} - Displays the formatted Dimension for the Object. See the CAD Dimensioning topic for more information.

{value} - Displays the content of the Text Objects Value field formatted as specified. If you select or type {value} as your Text you will get additional options in the sidebar for the Text object. Please note you can use {value} in the Text field along with other Text and or Parameters.

Value - The actual value for the {value} parameter to display.

Type - The Type of value to display. There are six possible Types:

    • Measurement

    • Degrees

    • Integer

    • Boolean

    • Decimal

    • Currency
