View Mode

There are five View Modes in the Bid Center.

 

Bid Breakdown

This option changes the Bid Center view to a Bid Breakdown. This is the default View when entering the Bid Center.

Cabinet Prices

This option changes the Bid Center view to a Cabinet Prices view displaying Catalog Pricing. Catalog Pricing would be a fixed Object price passed from a Manufacturers Catalog.

Accessory Prices

This option changes the Bid Center view to an Accessory Prices view displaying Catalog Pricing. Catalog Pricing would be a fixed Object price passed from a Manufacturers Catalog.

Vendors

This option changes the Bid Center view to the Vendors view displaying the Vendors selected and available for the Materials used in the Job.

Your Default Vendor for each Material will be selected. You may select any other Vendor for each Material to use in the Job. There are also some quick selections in the sidebar you can use to make the selections for you.

 

Summary

This option displays a Design Summary for the current Job.

Labor

This option displays the Labor Summary/Calculation area. This is an tool you can use to assist in developing your Labor Costs. Labor is Part and Assembly based using the Bid Methods covered previously.

Labor Schemes are created utilizing the same Bid Method line items as your Rate Tables. To Edit/Create your Labor Schemes click on the Edit Labor Schemes tool in the sidebar.

The Labor Scheme Editor is basically the same as the Rate Table area and setting it up is exactly the same with the exception that instead of entering Currency or Percentage values you are entering Minutes for the selected Bid Method.

Once you have saved your Labor Scheme you can recall it as a Bid Method in any of your Rate Tables for use in calculating your full Bid.
