Boring

The Boring Tab at Cabinet/Closet Properties has twelve options:

  • Boring Style - This option allows you to choose the type of Line Boring the selected Assembly will use. There are four options:

    • No Boring Holes - Will remove all Line Boring from the selected Assembly.

    • Bore All Holes - Will "Bore All Holes" (Shotgun Line Bore) the selected Assembly.

    • Bore Only Required Holes - Will "Bore Only Required Holed" for the selected Assembly.

    • Bore By Opening - Will "Bore By Opening" the selected Assembly.

You must have the active Construction Method fully configured for each of these options in order to get good results.

  • Balanced Panels - Use this option to specify whether you want your Line Boring to be equally spaced from the Top and Bottom of your End Panels. See image below:

  • Balanced Openings - Use this option to specify whether you want your Line Boring, if set to Bore By Opening, to be equally spaced from the top and bottom of the Opening.

  • Align Reveals to System Holes - Use this option to specify whether you want your Reveals to align with you Line Boring. See image below:

  • Top Construction Holes - You can use this option to turn off Top Construction Boring.

  • Bottom Construction Holes - You can use this option to turn off Bottom Construction Boring.

  • Align Shelves to System Holes - Use this option to specify whether you want your Adjustable Shelves to align with your Line Boring.

  • First Hole Location - This sets the location for the first Line Boring hole vertically.

  • Line Bore Hole Depth - This sets the Depth of Line Bores.

  • Line Bore Hole Diameter - This sets the Diameter of Line Bores.

  • Front Line Bore Set-back - Sets the Front Line Bore setback for the active Assembly. The reference side (Front or Back) is controlled in the Construction Method selected for the active Assembly.

  • Rear Line Bore Set-back - Sets the Rear Line Bore setback for the active Assembly. The reference side (Front or Back) is controlled in the Construction Method selected for the active Assembly.
