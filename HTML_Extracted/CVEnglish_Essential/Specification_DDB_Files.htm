The *.DDB Files

The door database(s) consists of a list of Doors. These doors can be built out of various materials; the material information is stored in the CxMaterials file.  The following is the breakdown of the tables and their fields:

Construction Defaults

This table stores general construction information for all Doors

Field

Description

Unit

 

 

Value representing the selected Unit of Measure for your Door Construction:

0 = Imperial

1 = Metric

LSW

Default Width of Left Stile.

RSW

Default Width of Right Stile.

MSW

Default Width of Mid Stile.

TRW

Default Width of Top Rail.

BRW

Default Width of Bottom Rail.

MRW

Default Width of Mid Rail.

Mortice Tennon

A Boolean flag if set true indicates Mortise and Tenon Construction.

Tennon Depth

The default Tenon Depth.

Edge Profile

The default sample Width of the Edge Profile.

Panel Type

 

 

 

 

 

The default Panel Type:

0 = Raised

1 = Flat

2 = Glass

3 = Vertical Slats

4 = Horizontal Slats

Panel Inset

The default amount a Door Panel Insets into a Stile or Rail.

Vertical Slat

The default Width of the Vertical Slats.

Horizontal Slat

The default Width of the Horizontal Slats

RPD

The default Width of the Raised Panel Detail.

DB Info

This Table stores general information about this Database.

Field

Description

Version

A number indicating the Version of this Database. The Database Version changes each time fields are added/removed from the Database. This process typically happens after a Build update during the Update Version process.

Name

A short name for the Catalog. Door graphics for this Catalog will use this name appended with the [Doors].[ID].

Description

A description of this Catalog.

Door Images

This Table is currently not used.

Field

Description

ID

Currently Not Used

Name

Currently Not Used

Description

Currently Not Used

Image

Currently Not Used

Doors

This Table stores a list of all Door designs and their attributes.

Field

Description

ID

A unique number to reference this Door.

Name

The Door name.

Build

A string of information indicating how the Door is to be built.

Tab

A cross reference to the [Tabs].[Index] field, indicating which Category this Door Style is located.

Min Width

The minimum width that the Door will be built.

Min Height

The minimum height that the Door will be built.

Max Width

The maximum width that the Door will be built.

Max Height

The maximum height that the Door will be built.

Min Width Pointer

A reference to the [Doors].[ID] field that indicates when the Door exceeds the Minimum Width to build this door instead.

Min Height Pointer

A reference to the [Doors].[ID] field that indicates when the Door exceeds the Minimum Height to build this Door instead.

Max Width Pointer

A reference to the [Doors].[ID] field that indicates when the Door exceeds the Maximum Width to build this Door instead.

Max Height Pointer

A reference to the [Doors].[ID] field that indicates when the Door exceeds the Maximum Height to build this door instead.

Mirror Pointer

If set, the Door Style built will be determined by whether the Door is hinged on the right or left.

Cost Per Door

Cost associated with this Door Style that is applied per Door.

Cost Per SqFt

Cost associated with this Door Style that is applied per Square Foot.

Horizontal Grain

A Boolean flag if set true indicates the grain for the Door panel runs horizontally.

ApplyOutsideEdgeTo

 

 

 

 

 

A Binary Sum representing which Edges the Outside Edge Profile is applied to. Following are each edges numerical assignment:

1 = Right Edge

2 = Left Edge

4 = Bottom Edge

8 = Top Edge

Notes

A text field used to store User entered Notes for this Door Style.

Template

A Boolean flag if set true will signify this is a Template Door.

TemplateID

 

 

 

 

 

 

 

 

 

 

 

A number representing which Template to use for this Door. Following are each Templates numerical assignment:

1 = Square
2 = Cathedral

3 = Flex Arc Cathedral

4 = Flat Cathedral

5 = Chamfer

6 = Radius Corners

7 = Inv Radius Corners

8 = Arch

9 = ½ Circle Arch

10 = Slope

11 = Bump

TopRail

Template Door Only. Number representing this Template Doors Top Rail Width.

BottomRail

Template Door Only. Number representing this Template Doors Bottom Rail Width.

MidRail

Template Door Only. Number representing this Template Doors Mid Rail Width.

LeftStile

Template Door Only. Number representing this Template Doors Left Stile Width.

RightStile

Template Door Only. Number representing this Template Doors Right Stile Width.

MidStile

Template Door Only. Number representing this Template Doors Mid Stile Width.

Crown

 

 

 

 

Template Door Only. Number representing the Crown Height for this Template Door. Used with:

Cathedral

Flex Arc Cathedral

Flat Cathedral

Arch

FlatIndent

 

 

 

 

Template Door Only. Number representing the Flat Indent for this Template Door. Used with:

Cathedral

Flex Arc Cathedral
Flat Cathedral

Arch

Radius

 

 

 

Template Door Only. Number representing the Radius for this Template Door. Used with:

Flex Arc Cathedral

Radius Corners

Inv Radius Corners

Chamfer

 

Template Door Only. Number representing the Chamfer for this Template Door. Used with:

Chamfer

Lites

 

 

 

 

 

 

 

 

 

 

 

Template Door Only. Number representing the Number or Style of Lites in the Door. Following are each Lites numerical assignment:

1 = 1

2 = 1L

3 = 1R

4 = 2V

5 = 2H

6 = 3

7 = 4

8 = 4V

9 = 6

10 = 8

MirrorTopShape

Template Door Only. Specifies whether the shape specified by the Template is mirrored onto the bottom of the Door.

ApplyToCorners

 

 

 

Template Door Only. Specifies whether corner operations are applied to the bottom of the Template. Used with:

Chamfer

Radius Corners

Inv Radius Corners

Rise

Template Door Only. Specifies the Rise on the Slope Template Door.

Run

Template Door Only. Specifies the Run on the Slope Template Door.

Width

Template Door Only. Specifies the Width of the Bump on a Bump Door.

Height

Specifies the Height of the Bump on a Bump Door.

MultiPiece

Template Door Only. Specifies whether the Template Door is a Slab type or Five Piece.

LongRails

Template Door Only. Boolean specifying whether the five piece Door is produced with Long or Short Rails.

LongMidRails

Template Door Only. Boolean specifying whether the five piece Door is produced with Long or Short Mid Rails.

ShapeTop

Template Door Only. Specifies whether the shape specified by the Template is applied to the Top of the Door by shaping the Top of the Door to match.

ShapeBottom

 

Template Door Only. Specifies whether the shape specified by the Template is applied to the Top of the Door by shaping the Top of the Door to match. Used in conjunction with MirrorTopShape

LegacyIEPW

(Not Used) Used in previous versions of Solid to specify the Inside Edge Profile Width.

SoftenCorners

A boolean set true if the edges of the door should have a fillet to prevent ripping an applied foil.  The fillet radius is defined by the SoftCornerRadius or by the S2M preferences.

SoftCornerRadius

A value other then zero will override the Soft Corner Radius value set in S2M preferences.

Material Schedules

This Table stores additional pricing and laminate information not in Material.CVM.

Field

Description

ID

A unique number used to reference this Material data.

Name

The Material Schedule name.

ClassID

Currently Not Used

Cost per Door

The default Cost associated with this Material Schedule that is applied per Door.

Cost per SqFt

The default Cost associated with this Material Schedule that is applied Square Foot.

Laminate Face

A Boolean flag if set true will add a piece of Laminate to the cut list for the Door Face.

Laminate Back

A Boolean flag if set true will add a piece of Laminate to the cut list for the Door Back.

Band

A Boolean flag if set true will add banding to the cut list for the Door.

Purchase

 

 

Value Representing whether the Door is Manufactured or Purchased.

0 = Manufactured

1 = Buy Out

MinCost

Minimum cost associated with this Door Style.

PriceMatrices

This Table stores any Price Matrices you have created.

Field

Description

PriceMatrixID

 A unique number used to reference this Price Matrix data.

SizeMatrixID

 A cross-reference to the [SizeMatrices].[SizeMatrixID] field.

Name

 The Price Matrix Name.

Prices

This Table stores Size and Pricing information for Price Matrices.

Field

Description

PriceMatrixID

A cross-reference to the [PriceMatrices].[PriceMatrixID] field.

WidthOrder

A numeric value establishing which Column in the grid this Price is located in.

HeightOrder

A numeric value establishing which Row in the grid this Price is located in.

Price

The Price associated with this Width/Height combination.

Profile Types

This Table stores the currently available Profile Types.

Field

Description

ID

A unique number used to reference this Profile Type.

Description

The description of the Profile Type.

Profiles

This Table stores Shape and Cost for all Profiles created for this Door Catalog.

Field

Description

ID

A unique number used to reference this Profile.

Type

A cross-reference to the [Profile Types].[ID] field.

Name

The name of the Profile.

NumberOfPoints

The number of points in the Profile Shape.

Points

A comma delimited string of X, Y coordinates describing the Profile’s shape.

Width

The Width of the Profile.

Height

The Height of the Profile.

Cost per Door

The default Cost associated with this Profile that is applied per Door.

Cost per SqFt

The default Cost associated with this Profile that is applied per Square Foot of Door.

Applied

A Boolean flag if set true will signify this is an ”r;Applied” Inside Edge Profile.

Selected Material Schedules

This table stores the link for the Materials selected for a particular Door Style. This table also stores the Pricing specific for the selected Material Schedule when used in conjunction with the Door Style.

Field

Description

DoorID

A cross-reference to the [Doors].[ID] field.

Schedule

A cross-reference to the [Material Schedules].[ID] field.

Cost per Door

The actual Cost associated with this Material Schedule for this Door Style, applied per Door.

Cost per SqFt

The actual Cost associated with this Material Schedule for this Door Style, applied per Square Foot of Door.

PriceMatrixID

A cross-reference to the [Prices].[PriceMatrixID] field.

Default

A boolean set true if this is the default material schedule to use for the door.

MinCost

Minimum cost associated with this Door Style.

Selected Profiles

This table stores the link for the Profiles selected for a particular Door Style. This table also stores the Pricing specific for the selected Profile when used in conjunction with the Door Style.

Field

Description

DoorID

A cross-reference to the [Doors].[ID] field

Profile

A cross-reference to the [Profiles].[ID] field

Cost per Door

The actual Cost associated with this Profile for this Door Style, applied per Door.

Cost per SqFt

The actual Cost associated with this Profile for this Door Style, applied per Square Foot of Door.

ToolSet

Internal value identifying whether this Selected Profile is a Tool Set.

ProfileType

A cross-reference to the [ProfileType].[ID] field

Default

A boolean set true if this is the default profile to use for the door.

SizeMatrices

This Table stores Size Matrices.

Field

Description

SizeMatrixID

A unique number used to reference this Size Matrix.

Name

A name for this Size Matrix.

Widths

Specifies the number of Width Columns in the Size Matrix.

Heights

Specifies the number of Height Rows in the Size Matrix

Sizes

This Table stores Sizes for the Size Matrices.

Field

Description

SizeMatrixID

A cross-reference to the [SizeMatrices].[SizeMatrixID] field

Size

A numeric value representing the Size for the Matrix.

Width

A Boolean flag if set true will signify this is a ”r;Width” Size. If set to false will signify this is a ”r;Height” Size.

Tabs

This Table stores the User Defined Categories for this Door Catalog.

Field

Description

Index

A unique number used to reference this Door Category.

Name
