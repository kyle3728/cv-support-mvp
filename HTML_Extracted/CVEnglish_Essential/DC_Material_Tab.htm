Once you have selected the Profiles to use with the Door Style. The next step is to click on the "Material" tab and select the Door Material Schedules you wish to use with this Door. You will note in the picture above that I have selected my "Alder", "Ash", "Birch", "Cherry", "Maple", "Oak", and "Paint Grade" Material Schedules for use with this Door.

The Material Cost associated with the selected Material Schedule is also displayed and can be edited from here. See the Door/Drawer Fronts Material Schedule topic for more information.

If you would like the Door to have a default selection for the Material simply select the Material you would like to be the Default selection for the Door and enable the "Set Material as the Default to Use for this Door" option.

If you want to set the Banding Edges for the selected Door Style simply select the Banding Edges option then enable the Edges where you want Banding applied.

Banding will only be applied if the active Material Schedule has a valid Banding Material and the Band Edges option is enabled.

The "Use" Column

For every "Yes" in  the USE column an additional Material will be available.  If you neglect to select any Materials for a Door, you will receive the error message "Unable to find material Schedule, Select another Door" when you attempt to use your new Door in a Job.

You should select at least one Material Schedule for use with a Door. If you don’t select a Material Schedule for use here the system will assign a Default schedule to the Door.

Each of the Material Schedules that you see on the page were created for the Doors and Drawer Faces.  By clicking on the Material, you can see the content of the schedule in the preview screen in the Top Right Corner.  To change the schedule, click on the menu option "Setup...Materials" and choose the correct Material Schedule from the drop down list.
