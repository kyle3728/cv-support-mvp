Right Click On Screen -

 

Following are the options when right clicking the screen in a Face, Plan, or End Ortho view:

  •  Save - See the Room Level Save topic for more information.

  •  To Drawing - See the Room Level To Drawing topic for more information.

  •  To Live Drawing - - See the Room Level To Live Drawing topic for more information.

  •  Add Light - See the Room Level Add Light topic for more information.

  •  Save for Assembly Sheet - This option will Save the current view for the active Assembly for use in Assembly Sheets when using the "Additional Saved Views" option.

  •  Properties - See the Room Level Properties topic for more information.
