Library Part -

The Part Library window will appear.

On the sidebar, select the Catalog where the desired Part is located. Then click the Part that you want to add and click Next.

In the Size area, type in the size that you want to have the Library Part come in at. See the New Part Size topic for more information.

Choose the Orientation that you want the new Part to have. See the New Part Orientation topic for more information.

Select the desired Finish for the new Part. Click Finish to complete the process of importing the Library Part. See the New Part Finish topic form more information.

Use the size and position fields in the sidebar to locate the Library Part properly in your Assembly.
