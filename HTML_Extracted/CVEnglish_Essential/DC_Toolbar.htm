Door Catalog Toolbar

There are five tools in the Door Catalog toolbar.

 

Create a New Door Style

Click this tool to create a new Door from scratch.

 

Copy Current Door Style

Click this tool to create a new Door Style that is a copy of the current Door Style. This is the best way to create a new Door if you already have one that is similar to what you need.

 

Delete Current Door Style

Click this tool to delete the current Door Style.

Remember, as discussed in the Door Catalog topic, the Door Catalog has GLOBAL REACH and therefore if you delete a Door that has been used in a previous Job, that Door will disappear from that Job and a new one must be selected the next time that you open that Job.

 

Mirror Door Style

Click this tool to mirror the currently selected Door Style.

 

Select A Door Style

Click this tool to browse through the Door Catalog to the Door that you wish to edit.
