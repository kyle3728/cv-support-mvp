Right Click On A Counter Top

Following are the possible options when Right Clicking on a Counter Top:

 

 Delete

This menu option will delete the selected Counter Top.

 Delete All

This menu option will delete all the Counter Tops in the current Room.

 Edit

This menu option will take your Counter Top to the Assembly Level and allow you to make Assembly and/or Part Level changes to it.

 Edit Shape

This menu option allows you to edit the shape of your Counter Top by bringing up the Shape Editor options.

Click the options below for more information

There are several options specific to editing the Shape of Counter Tops that should be noted.

Selecting a Point on a Counter Top

When selecting a Point on a Counter Top you are presented with the following sidebar options:

  • Start X - Sets the Start X for the selected Point.

  • Start Y - Sets the Start Y for the selected Point.

  • Build Up Joint Type - Will allow you to select the Joint Type (Build-Up Joint) for the selected Node/Point. There are six options here:

      • Default - The Default Joint type as defined by the Counter Top Construction Method.

      • Push - Configures a Push Build-Up as demonstrated by its image.

      • Pull - Configures a Pull Build-Up as demonstrated by its image.

      • Miter - Configures a Miter Build-Up as demonstrated by its image.

      • Common - Configures a Common (one Piece) Build-Up as demonstrated by its image.

      • None - Configures a no Build-Up Joint.

  • CNC Auto Radius - This function is very handy for creating your own Miter/Butt Joints. If the Point is set to Auto Radius as an Outside Corner the corner will be radiused to match the diameter of the Outline Tool. This will allow an Outside Corner to fit into an Inside Corner.  

Selecting a Line or Arc on a Counter Top

When selecting a Line or Arc on a Counter Top you are presented with the following sidebar option specific to Counter Tops:

    • Edge - Type - Specifies the Edges Type. Following are the options here:

  • Automatic

  • Unfinished - This option allows you to specify an Edge on a Counter Top to be Unfinished.

  • Finished - This option allows you to specify an Edge on a Counter Top to be Finished.

  • Profile - This option allows you to specify an Edge on a Counter Top to incorporate your selected Counter Top Edge Profile.

  • Splash - This option allows you to specify an Edge on a Counter Top to have a Splash.

  • Butt - This option allows you to specify an Edge on a Counter Top to have your specified Counter Top Joinery.

Right Clicking a Counter Top in the Shape Editor

When Right Clicking in the Counter Top Shape Editor you have the following options:

 

  •  Cut - This option will Cut the selected CAD Object(s) to the Windows Clipboard.

  •  Copy - This option will Copy the selected CAD Object(s) to the Windows Clipboard.

  •  Paste - This option will Paste the CAD Object(s) stored in the Windows Clipboard to the current View.

  •  Delete - This option will Delete the selected CAD object(s).

  •  Break - This option will allow you to Break a selected CAD object into multiple objects.

There are three Break Modes:

Break into equal segments - Simply enter the Number of segments you need.

Break into pieces with length - This option will break the Object into segments of a Length specified; with an Object at the end equal to the remainder.

Break at location with length - This option will Break the Object from the Start Point at the Length Specified.

  •  Transform - There are four possible Transform options:

 Flip - Will Flip the selected CAD Object.

 Mirror - Will Mirror the selected CAD Object.

 Reverse - Will Reverse the selected CAD Object.

 Rotate... - Allows you to specify a Rotation for the selected CAD Object.

 

These options will not be available if the selected CAD Object is Connected. See Remove Connections below.

  •  Select All - The Select All menu option will select all CAD Drawn objects in the current View.

  •  Select None - The Select None menu option will deselect all of the CAD Drawn objects in the current View.

  •  Reverse Selection - The Reverse Selection option will deselect all the current Selections and in turn Select all the CAD objects that were not in the previous selection set.

  •  Add Connections - Adds any missing Connections to your current shape.

  •  Remove Connections - Removes all Connections from your current shape.

  •  Properties - The Properties option opens the View Properties screen.

 Add Cutout

The following graphic demonstrates the procedure required to place a Cutout in a Counter Top.

By right clicking on your Counter Top, and selecting "Add Cutout" the Counter Top Cutout window appears. The values that are placed in the boxes of the cutout window are reflected in the small plan view shown below the Counter Top cutout window. To view the cutout as it appears on the Top, use the 3D view at the Room level.

Automatic Cutouts

Appliances that normally correspond with a cutout in the Counter Top now give you the option of automatically creating that cutout when the appliance is placed.  In the example below, the user has right-clicked on the cook-top and clicked Properties.

In the Properties window that appears, the Cutout is defined by values entered under the Cutout tab.

If you do not wish to have the Cutout created automatically by placing the appliance, un-check the "Create Cutout in Counter Top" check box on the Cutout tab of the Object Properties window and re-save the appliance to the appropriate library. You can also change the Type to CNC in order to edit the properties of the cut-out for CNC Output.

 Edit Splits

This menu option brings up the Split Editor options in the ribbonbar.

Add Split

This option adds a Split to your Counter Top. This is very handy, because Solid will not determine if the Counter Top's length is too large to manufacture in a single piece, this will allow you to instruct Solid where to split the Counter Top for manufacture.

Delete Split

This option allows you to delete a Top Split or Part Split you have added.

Move Split

This option allows you to select the Top Split letting you move it by dragging or by Tabbing into the position field on the sidebar and entering a specific dimensional location (Based on the left edge of the Counter Top).

 Properties

The Properties menu option opens the Top Properties area. There are six tabs in the Top Properties area:

Counter Top Tab

Following are the possible options on the Counter Top tab:

    • Description - The Description/Name of the selected Counter Top.

    • Material - Allows you to select a Material Schedule for the selected Counter Top.

    • Texture - Allows you to select a Texture to apply to the Counter Top.

    • Construction - Allows you to select an alternate Construction Method for the selected Counter Top.

    • Edge Profile - Allows you to select an Edge Profile for the selected Counter Top.

    • Top Elevation - Allows you to modify the Elevation of the selected Counter Top.

    • Splash Height - Allows you to set the Height/Width of the Back Splash on the selected Counter Top.

Comment Tab

The Comment tab allows you to add a Comment/Note on the selected Counter Top.

Dimensions Tab

There are four Dimension modes you can set for the selected Counter Top here:

Each of these options refer back to the Counter Tops Dimension Layer for specific Dimensioning instructions.

    • Automatic - Utilizes the standard Counter Top Dimensioning.

    • Dimension Around Counter Top - Dimensions around the Counter Top.

    • Dimension Counter Top Along Wall - Dimensions the Counter Top along the Wall.

    • None - Turns Dimensioning Off for the Counter Top.

Cutouts Tab

Displays and allows removal of Cutouts that are on the selected Counter Top.

Overrides Tab

This Tab allows you to View and/or Remove any Overrides that may be present on the selected Counter Top.

Parameters Tab

This Tab allows you to View and/or modify any Parameters on the selected Counter Top.
