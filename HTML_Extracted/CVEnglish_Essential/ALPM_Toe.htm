Toe

    

The Toe Tab at Cabinet/Closet Properties has nine main options:

  • Toe Height - Allows you to specify the Toe Height for the active Assembly.

  • Front Recess - Allows you to specify the Front Toe Recess for the active Assembly.

  • Back Recess - This will set the Recess of the Back Toe in relation to the Wall.

  • Number of Sleepers - Allows you to specify the Quantity of Sleepers and/or the ability to Delete them for the active Assembly.

  • Left Recess - If utilizing a Finished or Unfinished Toe as on a separate Toe Assembly this will specify the Recess for the Left Side Toe.

  • Right Recess - If utilizing a Finished or Unfinished Toe as on a separate Toe Assembly this will specify the Recess for the Right Side Toe.

  • Left Side Toe - There are four options for the Left Side Toe:

    • Unfinished - If utilizing a separate Toe Assembly this will set the Left Side Toe as Unfinished.

    • Finished - If utilizing a separate Toe Assembly this will set the Left Side Toe as Finished.

    • Notched End Toe - This option will set the Left End to run to the floor and be notched for the Front Toe Kick.

    • Full End - This option will set the Left End to the floor without a Toe Notch.

  • Right Side Toe - There are four options for the Right Side Toe:

    • Unfinished - If utilizing a separate Toe Assembly this will set the Right Side Toe as Unfinished.

    • Finished - If utilizing a separate Toe Assembly this will set the Right Side Toe as Finished.

    • Notched End Toe - This option will set the Right End to run to the floor and be notched for the Front Toe Kick.

    • Full End - This option will set the Right End to the floor without a Toe Notch.

  • Back Toe - There are three options for the Back Toe:

    • None - This will remove the Back Toe.

    • Unfinished - If utilizing a separate Toe Assembly this will set the Back Toe as Unfinished.

    • Finished - If utilizing a separate Toe Assembly this will set the Back Toe as Finished.

 

Remember changes made here will only affect the current Assembly. Also, changes made here are attached as permanent (unless you remove them) overrides on the Assembly; which means that if you Save this Assembly into a Catalog these overrides will also be Saved.
