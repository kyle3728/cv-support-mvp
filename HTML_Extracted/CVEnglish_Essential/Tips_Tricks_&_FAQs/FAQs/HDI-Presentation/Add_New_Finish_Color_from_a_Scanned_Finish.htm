Add New Finish Color from a Scanned Finish

Is there anyway that I can add my own color as a finish by taking the color from a scanned image?

Yes, you can add any color you want. Trick is it will have to be a Texture and not a finish.

Scan it and create a Bitmap of it then import as a new Texture create all the new Materials you need and assign this Texture to them. Make sure when you create this new Texture to enable the "Decal" option.
