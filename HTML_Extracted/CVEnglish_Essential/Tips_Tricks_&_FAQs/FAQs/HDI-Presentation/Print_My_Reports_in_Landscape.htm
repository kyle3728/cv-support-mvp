Print My Reports in Landscape

How can I set my Reports to Print in Landscape?

  1. Go to the Windows Control Panel, Printers option.

  2. Install another copy of your default printer, but give it a different name. IE: Default Printer - Landscape

  3. Change the default settings for this printer to always print to Landscape.

  4. In Solid, when you go to Print your Report, select this new printer (Default Printer - Landscape).
