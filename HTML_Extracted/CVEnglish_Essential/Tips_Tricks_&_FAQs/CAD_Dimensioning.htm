CAD Dimensioning

With the ability to edit dimensions through the Layers in Solid, it is less common for the need to arise that dimensions be added manually through CAD. However, the functionality exists to create dimensions in CAD should something need to display dimensions when the automatic dimensions won’t include that item. It is best to use the snap tool with snap turned to "Snap to Points".

When the snap is set to "Snap to Points", you will see small node points at every spot on the Object that you may snap to. In order to snap to that point, you must click somewhere close to it but it is not necessary to be exact.

When dimensioning an object in the CAD mode, you may select the type of dimension line that you will use. This is combined with the Dimension Style that is selected (Dimension Styles are set up under the View Group of the main ribbonbar, on the Floor Plan or Elevation Views) to give you the characteristics of the dimension lines that show up on the drawing. The types of dimension lines that are available in CAD are:

Simple

The dimension appears exactly where you draw the dimension line. For example, if you pull a dimension from one point to another, the dimension line appears exactly between those two points. Linear

The dimension is displayed either horizontally or vertically, not necessarily in line with the points that are being dimensioned. The picture below shows two Linear Dimensions between the same points as in the previous example. Notice that the Linear Dimension does not reflect the angle between the points that are being dimensioned. This dimension shows the distance between the points either vertically or horizontally.

Aligned

The dimension line follows the angle between the points being dimensioned. This dimension gives the distance straight between the two points.  See the example below.

Internal

Use this type of dimension if you are showing interior dimensions. No tick marks will show because the object itself provides the outline for the dimensions. See the example below.

Linear-Tailed

This option is the same as linear (described above) but with a tail to the points that are being dimensioned. See the example below.

Aligned-Tailed

This option is the same as aligned (described above) but with a tail to the points that are being dimensioned. See the example below.

Radius

Angular
