Creating a Laminate Door

  1. Click on the "Create A New Door Style" tool.

  2. Name the Door.

  3. Select a Category to store the Door on, use the "Slab" category.

  4. Click on the "Construction" tab.

  5. Click in the middle of the Door and select "Slab" as the Panel Type for this Door.

  6. Check the Door panel size, it should match the size of the sample Door.

  7. Click on the "Profiles" tab and select the appropriate Profiles to use with this Door (you most likely will not select any profiles).

  8. Click on the "Materials" tab and select the appropriate Material Schedules to use with this Door. Be sure to pick Material Schedules that have had the laminate options you want turned on and that have the correct Material selected for "Slab".
