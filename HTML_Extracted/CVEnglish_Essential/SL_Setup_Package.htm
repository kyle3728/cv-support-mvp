Setup Package

Are limited to Importing Licensed Setup Packages only. You can not create Setup Packages.

The Setup Package utility is a means to transfer specific objects from one computer running Solid to another.

Exporting a Setup Package

The Setup Package feature allows you to create a single file that can store and transfer multiple custom items for import into another CABINET VISION user’s or CABINET VISION Tech’s system.

The beauty of the CABINET VISION Setup Package is that any unique identifiers are stored and recreated on the other user’s system. In the simplest terms it is plug and play. A good example of where this comes in handy is setting up a Blum Metabox drawer system. Prior to the implementation of the Setup Package Utility you would have to export each UCS and Intelli-Joint as well as having to recreate each Drawer Guide Material and Drawer Guide Material Schedule. Then link each of these items together in your UCS. Another method would have been to send a Technician or CABINET VISION user your database files thus overwriting their databases. With the Setup Package you simply select the UCS and the Setup Package Utility will scan the UCS and pick out each Intelli-Joint, Material, Finish, Material Schedule, and etc... that is involved in properly creating the Blum Metabox Drawer System.

The following list of items can be added to a package for transfer:

Bid Reports - This includes both Custom Reports and Word Forms.

Catalogs

Construction Methods

Doors - If you select a Door it will automatically import any items attached to that Door such as; Profiles, Material Schedules, etc...

Drawer Boxes - If you select a Drawer Box it will automatically import the Material Schedule associated with it.

Drawings - This includes your Title-blocks and Drawing Symbols

Finish Types

Finishes

Intelli-Joints

Labels (Label-IT Labels or Saw Graphic designs only)

Machines

Material Schedules - If you select a Material Schedule it will automatically import all Materials associated with it.

Materials - If you select a Material it will automatically import any Finishes and/or Textures associated with it.

Parameters

Parts

Profiles

Rate Tables

Reports

Roll Outs - If you select a Roll Out it will automatically import the Material Schedule associated with it.

S2M Reports

Textures

Tools - If you select a Tool Set it will automatically import all Tools associated with that Tool Set.

Top Construction Methods

User Created Standards - If you select a UCS it will automatically import any items called within the UCS such as; Intelli-Joints, Materials, Finishes, Material Schedules, and etc...

Vendors

This topic explains how to export a package and send it to Tech Support or another Solid user.

  1. From the System Level go to the Utilities Tab | Setup Package

  1. Select the item or items you would like to include in the Package from the list on the right side of the screen. After highlighting an item you want to add to the package simply click the "Add Items To Package" button in the lower left corner. There is no limit to the number of items that can be added to a package.

  

  1. After selecting the items to add to the package click on the "Save Current Package" tool in the toolbar.

In the graphic below we selected several Material Schedules, notice that it also picked up the corresponding Materials, Finishes, Finish Types, and Parts.

 

  1. Select a location to save the Package and type in a Name for the Package then select "Save".

  

  1. You may now either close the Setup Package screen or, if you use Microsoft Outlook you can click on the "E-Mail Current Package" tool in the top toolbar.  This will attach the Package to your e-mail and allow you to e-mail to Tech Support or to another CABINET VISION User.

 

  1. You may now close the Setup Package screen. You have successfully exported a Package from CABINET VISION.

Importing a Setup Package

This topic explains how to import a package that has been sent to you by Tech Support or another CABINET VISION user.

  1. From the System Level go to the Utilities Tab | Setup Package

  1. Click the "Open An Existing Package" tool in the toolbar.  

  

  1. Select the package file from the location where it is being stored and click Open.

  

  1. Click the "Swap Between Creation and Import Modes" button in the toolbar.

  

  1. Click the "Import Package Contents" tool in the toolbar.

Prior to importing you can optionally have all the names of the objects imported from the Package by setting a Prefix using the Prefix The Item Names tool in the toolbar.

Overwriting Existing Items

There are five options in the toolbar, that allow you to choose how to Overwrite items that already existed in your system.

Overwrite All Matches With Package Contents - Will overwrite all existing items in your database with the contents of the Package.

Use Existing Items For All Matches - Will use all the existing items in your database rather than overwrite them with the Package contents.

Overwrite Child Matches With Package Contents - Will overwrite all existing children of items in the Package that match those in your database.

Use Existing Items For Child Matches - Will use all the existing children in your database rather than overwrite them with the Package contents.

Turn On/Off Overwrite Mode - Allows you to Overwrite existing items by selecting them individually. This will cause items in the Package to either overwrite the existing item or to use the existing item.

  1. Click on OK for the Import Complete notice.

  1. You may now close the Package screen you have successfully imported a Package into CABINET VISION.
