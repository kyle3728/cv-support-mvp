Sliding Door Rail

The Sliding Door Rail Material Schedule allows selecting Sliding Door Rails and Rollers with different properties for different applications in your Assemblies.

Sidebar/Properties

The following Properties are available for Sliding Door Rails:

ID - The ID number for the selected Material Schedule.

Name - The Name of the selected Material Schedule.

Description - The Description of the selected Material Schedule.

Upcharge - The "Upcharge" value will only be added in a Rate Table that uses the Bid Method "Use Part Catalog Pricing". This value is a Percentage Upcharge you charge above your Standard Part Catalog Price as entered in the Part Manager when this Material Schedule is in use.

Face Inset - The inset distance from the Face of the Cabinet to the Face of the forward-most Door.

Door Gap - The Gap between the Doors.

Door Top Clearance - The Gap above the Door.

Door Bottom Clearance - The Gap below the Door.
