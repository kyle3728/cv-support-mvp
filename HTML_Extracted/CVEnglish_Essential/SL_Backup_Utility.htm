Backup Utility

Backup List Tab

This tab allows you to Backup your CABINET VISION data.

Anything in the Include pane will be included in the backup and items in the Ignore pane will not be included. Use the corresponding buttons to move items from one pane to another.

There are several options here:

CNC Database - Checking this option will backup the Tool and Machine catalogs. This will backup the S2M CENTER file psnc.mdb (See the S2M CENTER Help file for more information).

Construction Methods - Checking this option will backup all Cabinet and Drawer Construction Methods, User Created Standards and Intelli-Joints. This will backup the CxConstruction (SQL Database)

Custom Cabinet Library - Checking this option will backup the Custom Cabinets Library. This will backup the file custom cabinets.cvc.

Door Catalog – Checking this option will backup all information in the Door Catalog. This will backup the file door.ddb.

Drawings – Checking this option will backup all CAD Symbols and Title Blocks. This will backup the file drawings.sym.

Generic Cabinet Library - Checking this option will backup the Generic Library. This will backup the file generic.cvc.

Part / Material Catalog – Checking this option will Back Up all Materials, Material Schedules, and Parts. This will backup the CxMaterials (SQL Database).

User added Graphic files used by Textures will be added to the backups automatically when Part/Material Catalog is selected.

Posts - Checking this option will backup all post files that you use to output to CNC.

Report Database – Checking this option will backup all Reports, Report Group and Bid Rate Tables. This will back up the file report.mdb.

System Parameters - Checking this option will backup the System Parameters that are stored in the Windows Registry.

Others

  • Any items in the other list with the . CVC extension are additional Assembly Catalogs that are available for backup.

  • Default.dat is the file that contains Layer information and custom settings.

  • ManageIT.mdb is the file that contains the database information for the Manage It add-on.

  • Matsum.mdb is the Database that stores Material Summary information for the last Job taken to the Report or Bid Center.

  • Prices.spt is the Database that stores Pricing information for the Generic Library.

  • Stock.ddb is the Database that stores Doors for the Generic Library.

  • .CFG files are your Job Schemes.

Add Job - Clicking this button will allow you to attach a Job to your backup. This is especially handy when sending a set of Databases to Technical Support. You can attach a Job so the Technician sees exactly what you see on the screen.

Toolbar

There are three tools in the Backup Utility toolbar.

 Email Backup - With this option selected once the backup is complete, CABINET VISION will Zip it up and open a new email message (If you are using Microsoft Outlook) with the back up attached. The backup can then be sent to Technical Support.

 Delete Backup - This option you can delete old backups that either you or a CABINET VISION Install has created. Deleting old backups (Archives) will allow you to free up hard-drive space.

 Settings - With this option you can setup automatic deletion of backups (Archives) as well as establish the destination of the Backup files.
