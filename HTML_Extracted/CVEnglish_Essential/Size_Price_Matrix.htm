Price Matrix

Part Matrix Pricing allows you to establish standard Part sizes along with fixed Prices or Pricing formulas for each size combination that you offer for your Parts.

The Interface

The Toolbar

The following tools are available in the Size - Price Matrix.

Matrix Selector - Allows you to select an existing Price Matrix to edit.

New - Creates a new Price Matrix.

Copy - Will Copy the selected Price Matrix

Rename - Will Rename the selected Price Matrix.

Delete - Will Delete the selected Price Matrix.

Modify Prices - Allows you to Increase or Decrease the Prices in the active Price Matrix a specific Percentage.

Unit of Measure Selection - Allows you to select the Unit of Measure to work with.

The Matrix Area

Shows the currently selected Matrix. Lengths are shown vertically and Widths horizontally.

Global Pricing Area

There are four options in the Global Pricing Area. This area allows you to establish Global Pricing for the selected Price Matrix. There are two options here:

  • Default Price Formula - The Price or Pricing Formula entered here will be used for any Cells left blank within the selected Price Matrix.

  • No-Match Price Formula - The Price or Pricing Formula entered here will be used for the selected Price Matrix if a Size Match is not found.

  • Exact Width Match - This option, if selected, instructs Solid that the Width must be an exact match to use the Price for the Size.

    • If this is not selected for Width then instead of an exact match it will use >= for the match. The low number in the Width is what is keyed off of. So a 0 to 12 Width will capture from 0" up to 11.999". The next Width grid from 12 to 14 will capture 12" up to 13.999" and so on.

  • Exact Length Match - This option, if selected, instructs Solid that the Length must be an exact match to use the Price for the Size.

    • If this is not selected for Length then instead of an exact match it will use >= for the match. The low number in the Length is what is keyed off of. So a 0 to 24 Length will capture from 0" up to 23.999". The next Length grid from 24 to 36 will capture 24" up to 35.999" and so on.

An Example

The best way to explain how Matrix Pricing functions is by example. This example will cover setting up a Price Matrix for Fixed Shelves with the following Prices.

Create a Size/Price Matrix Named "Fixed Shelves Locked Width-Length".

Next, you will need to setup the Sizes for your Size/Price Matrix. Fixed Shelves are offered in widths from 12" to 24" in 2 inch increments. Lengths are offered from 24" to 96" in 12 inch increments. So in this example the manufacturer has these Fixed Shelves pre-cut and ready to ship and these sizes each have a set Price. Any other sizes are custom and will use a special calculation to cover the cost custom cutting the special sizes. Step one would be to create a Size Matrix to cover Fixed Shelves. To cover the sizes in the example simply enter the required Widths in the Column Headers and the required Lengths in the Row Headers as in the illustration below.

You will also notice two options in the lower left (highlighted above) of the Part Size - Price Matrix area. In this example the Prices are for exact Part Sizes so these two options (Exact Width Match and Exact Length Match) should be selected.

Each cell in the Price Matrix on the right is now active and ready for us to enter our Prices.

You have a few options here:

  1. You can enter a fixed Price in the Cells.

  2. You can enter a Price formula amount in the Cells.

    • You can type a fixed Price or select one of the pre-defined calculation methods. You can also utilize math to create your own custom price utilizing (addition (+), subtraction (-), multiplication (*), and/or division (/). The formula will be validated by moving to another field. If the formula entered is invalid you will receive a warning message and the invalid formula text will turn red. Following are the available pre-defined calculations:

      • Volume

        • - The Cubic Inch value of the Part.

        • -  The Cubic Meters of the Part.

      • Face Area

        • - The Square Inches of the Face of the Part.

        •   - The Square Meters of the Face of the Part.

      • Length

        • - The Length, in Inches, of the Part.

        • - The Length, in Millimeters, of the Part.

      • Thickness

        • - The Thickness, in Inches, of the Part.

        •   - The Thickness, in Millimeters, of the Part.

      • Width

        • - The Width, in Inches, of the Part.

        • - The Width, in Millimeters, of the Part.

  1. You can leave any Cell you need to blank.

    • If you leave a Cell blank then the blank Cell will utilize the Price or Formula that is entered in the Default Formula field at the bottom.

You can use any combination of the 3 options above within the Matrix. This means, each Size cell can be any one of the 3 options above.

In the example there are Fixed Widths and Lengths along with Fixed Prices for each combination so our Price Matrix will look like below with each Cell containing a fixed Price.

Because the Widths and Lengths are fixed we have covered all of the Pricing for our Standard Parts. Next we need to cover the Pricing when there is no Size match. This will be our calculation for Custom Fixed Shelf Sizes. This formula needs to be entered in the "No Match Formula" field.

In this case we are going to use *1. This is the Square Inches of the Part multiplied by 1.
