Quick Access Toolbar

There are six possible options in the Quick Access Toolbar.

New

Open

Save

Print

Undo

+

This menu option allows you to "Undo" the last several changes you made. To setup how many changes are available with the Undo command see the topic on setting up Undo options.

Redo

+

This menu option allows you to Redo the last "Undo" you initiated. See the topic on setting up Redo options.
