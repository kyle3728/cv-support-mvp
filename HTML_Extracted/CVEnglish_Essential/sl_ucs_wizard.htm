New - User Created Standards Wizard

The User Created Standard Wizard is a quick way to make most UCS's. There are three types of UCS's:

  1. Add (Link) A Part: This type of UCS allows you to add a Part/Operation that is linked to another Assembly, Opening, another Part, or another Operation.

  2. Modify A Part: This type of UCS allows you to modify a selected Part /Operation.

  3. Delete A Part: This type of UCS allows you to delete a selected Part/Operation.
