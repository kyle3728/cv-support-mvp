Connection Manager

The Connection Manager is where you will setup your Connection Machining for use on Part Connections for Assemblies and Drawers and Roll Outs in the Assembly Manager.

When selecting the New option in the ribbonbar you are given the choice of creating a Connection or a Connection Set.

Following are the Connections that can be created in the Connection Manager.

Click the Connection Types below for more information.

Set

Connection Sets allow you to tie multiple Connections together. Maybe you have a Connection that defines dowel boring and you also have one which performs pre-drills. You can tie these two together create a single Connection Set which creates both.

Simply add any existing Connections (leftmost pane) to the Connection Set (rightmost pane).

You can also Copy , Delete  or Modify  an existing Connection of Connection Set here.
