Interior Case Branch

The Interior Case Branch contains all of the internal Assembly Parts. This branch is important for all Construction Styles.

Notice that these Parts are typically within a Cabinet or Closet, but not always. For example if a Cabinet is Saved to a Catalog without Drawer Stretchers they will not be placed in the Cabinet when it is pulled out of the Catalog even though the active Construction Method includes them.

Depending on the Construction Style selected the following Branches can be in the Interior Case Section.

Adjustable Shelf

Divider

Drawer Stretcher

Fixed Shelf

Nailer

Partition

Shelf

Shelf Nailer

Shoe Section

Sub End

Unfinished Back

Unfinished End
