Creating Door Profiles

The Profiles area allows you to create Profiles for use with your Doors.

New

To Create a new Profile simply click the "New" tool in the toolbar. Once in the "Add New Profile" screen simply select the "Profile Type". There are four different choices in the Add New Profile dialog box:

  1. Outside Edge Profile (Finger Pull) - This option creates an Outside Edge Profile for use on your Doors. See the sample image below, notice the Outside Edge Profile highlighted in red.

Once you have clicked OK to Add your New Profile, your new Profile will appear in the list as "New ???????" (In place of the Question marks will be the type of Profile you selected).

There are several Property options for your new Profile. You can change any of this items by simply selecting it and modifying it from the Properties area to the right. Each option is described below:

The profile type the property is available for is displayed below.

    • OEP = Outside Edge Profile

    • IEP = Inside Edge Profile

    • RPD = Raised Panel Detail

    • RPTN = Route Pattern

    • TS = Tool Set.

  1. Name - (OEP/IEP/RPD/RPTN/TS) - To give your Profile a name simply click on the Name field to the right in the Properties area, and type in the Name you wish to identify the Profile with.

  2. Cost Per Door - (OEP/IEP/RPD/RPTN) - If needed give this profile a "Cost Per Door" when used.

  3. Cost Per SQFT - (OEP/IEP/RPD/RPTN) - If needed give this profile a "Cost Per Square Foot" when used.

  4. Width - (OEP/IEP/RPD/RPTN) - Give your new Profile a width in the Size field.

  5. Extended - (IEP) - This field allows you to define whether the selected Inside Edge Profile is extended or not.

There is an additional tab in the Profile editor entitled "Extended Profiles" this screen is a sample screen only and allows you to see how an Inside Edge Profile is treated by Solid, in different conditions, depending on whether it is defined as Extended or not.

  1. Machine - (TS) - Assigns the selected Tool Set to a particular Machine from the Machine Catalog. See the NcCenter Help File Tool Set Help topic for more information.

  2. Cut For Glass - (TS) - This field is not editable. Its value will change depending on whether you have defined a Tool in your Tool Set to "Cut / Glass". See the NcCenter Help File Tool Set Help topic for more information.

  3. Follow Glass Shape - (TS) - This option defines whether or not you would like your Tool Set, if set to cut glass, to follow the shape of the opening or to setup to pocket out the opening for a piece of square glass. See the NcCenter Help File Tool Set Help topic for more information.

  1. Rabbet Width - (TS) - Defines the Width of the Rabbet on the Backside for the Glass to fit into. See the NcCenter Help File Tool Set Help topic for more information.

  2. Rabbet Depth - (TS) - Defines the Depth of the Rabbet on the Backside for the Glass to fit into. See the NcCenter Help File Tool Set Help topic for more information.

  3. Router for Back - (TS) - Defines the Route Tool to use to route the Rabbet on the Backside for the Glass to fit into. See the NcCenter Help File Tool Set Help topic for more information.

  4. Nested Part Clearance - (TS) - This sets the space in between Doors and Door Parts on a nested sheet when the selected Tool Set is used. See the NcCenter Help File Tool Set Help topic for more information.

  5. Outside Edge - (TS) - Defines whether the selected Tool Set will be available as an Outside Edge Profile. See the NcCenter Help File Tool Set Help topic for more information.

  6. Inside Edge - (TS) - Defines whether the selected Tool Set will be available as an Inside Edge Profile. See the NcCenter Help File Tool Set Help topic for more information.

  7. Raised Panel Detail - (TS) - Defines whether the selected Tool Set will be available as an Raised Panel Detail. See the NcCenter Help File Tool Set Help topic for more information.

  8. Route - (TS) - Defines whether the selected Tool Set will be available as an Route Pattern. See the NcCenter Help File Tool Set Help topic for more information.

Copy

Click this button to Copy the Selected Profile. This action will copy all properties as well as the shape of the selected Profile.

Delete

Click this button to Delete the Selected Profile.

Shape Tab - Inside Edge/Outside Edge/Raised Panel/Route Pattern

The next step in the process is to give your profile a shape.

The profiles "Shape", when defined for OEP/IEP/RPD/RPTN, does not affect anything except the way your Doors display and Render when used. The width, however, with the Inside Edge Profile can have an effect, depending on the setting for "Extended".

With your Profile selected, click on the "Shape" tab. Using the Shape Tools create your Profile/Moldings shape. When you enter the Profile Shape area you will notice that there is a "Sample" shape in place. This will help you orient your Profile properly.
