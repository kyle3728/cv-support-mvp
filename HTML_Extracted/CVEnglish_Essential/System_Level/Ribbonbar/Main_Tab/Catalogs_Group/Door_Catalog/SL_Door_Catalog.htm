Door Catalog

Doors, Drawer Fronts, Panel Ends, and False Fronts are created in the Door Catalog. Materials are assigned to Doors and Drawer Fronts here, as well as the assignment of Profiles.

Doors are not sized in this Catalog, some Door parts  (Stiles, Rails, Panels) are configured and have dimensions set, but the overall size of a finished Door is not set in this Catalog.

  The Door Catalog has a Global Reach. This means that normal changes made to a Door in your Door Catalog WILL reach Jobs where that same Door existed prior to the change. This could result in a Job being built with incorrect Doors due to the subsequent change. Therefore, to be on the safe side, never change an existing Door. Should you need the same Door in a changed form, use the Copy Current Door Style option, and change the copy of the Door.

Click on the menu options, toolbar tools, and/or tabs below for more information

Note that the name of the selected Door and the name of the Door Catalog in use is in the Titlebar of the Door Catalog.
