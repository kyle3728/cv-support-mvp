Profiles

Click on the tabs below for more information

On this tab, you need to select any Profiles that should be available for the current Door. When you start a new Job, you will select the Door at the Job Properties tab and it should give you the opportunity at that time to select the profiles that will be used in that particular Job from the Profiles that you have made available for that Door.

If you would like the Door to have default selections for Profiles simply select the Profile you would like to be the Default selection for the Door and enable the "Set As Default Profile for..." option.

The top section of these tab will display any cost associated with the selected Profile with the Add Per Door and Add Per SqFt fields. You can not edit these fields here this information is pulled from the Profile Properties as set up in the Setup|Settings menu.

Also, in the Top area you can define which edges the selected Outside Edge Profile will apply to. By default all edges are selected.

There are four different types of profiles.

 Inside Edge Profile - The Inside Edge Profile is used when you have a frame around the Door and the frame is assembled with cope & stick construction. Inside Edge Profiles are important because they can, depending on the Extended Property, affect the size of your Door parts.

 Outside Edge Profile - The Outside Edge Profile is a shape you will cut around the outside of the Door. Outside edge profiles do not affect the size of your Door parts.

 Panel Detail - The Panel Detail is the shape you will cut along the outside of the Door panel. Usually done around the edges of a Raised Panel. Panel details do not affect your Door parts sizes.

 Route Pattern - The Route Pattern is the shape of the cutter you would use if you where going to route a design into the face of the Door using a Door jig.

 

Column Headings

Name

In the Name column, you will see the name that has been given to the selected profile.  You will also see one of the four possible icons that will represent the type of the selected profile.

Use

This column will display the word "Yes" for any profiles that have been selected as options for the currently selected Door.

Width

This column will display the width (size) of the selected Profile.

Profile Type

This is a second area that shows the type of Profile that is selected.  This should match the icon showing the Profile Type in the Name column.

Cost Per Door

This column will show the amount of Cost Per Door for this Profile.  This amount will be added one time to any Door that has this Profile selected for it in a job.

Cost Per Square Foot

This column will show the amount of Cost Per Square Foot for this Profile.  This amount will be multiplied by the square footage and added to any Door that has this Profile selected for it in a job.

Glass

This column will show whether the selected Profile includes a tool that is set to make a pass on the backside of the Door for a Glass Panel to fit into.  

 

Apply Outside Edge Profile To

If the selected Profile is an Outside Edge Profile, the settings in this area will allow the user to select specific sides of the Door to apply the Profile to.
