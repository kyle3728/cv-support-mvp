General Info

Click on the Tabs below for more information on them

There are two main areas on the General Info Tab:

Door Properties

  • Door Name - The Door Name appears in the field below the icon that represents the Door.

  • Category - The Category of the Door is simply the location of that Door in the Door Catalog. When you click the Select a Door button, the Door Catalog should open and the Categories will appear as tabs in the window.

  • Notes - Enter any Notes you would like to appear on the Door Reports, in the Report Center, that include Notes.

  • Per Door - This is where you set a Flat Price for this Door Style.

  • Per SqUnit - This is where you set a Per Square Unit (Either Meter or Foot depending on your active Unit of Measure) Price for this Door Style.

  • Sample Material - Allows you to specify the Sample Material to use in the Preview pane in this area.

If no Sample Material is available for selection then that means not Material has been allowed for the current Door Style. See the Material Tab topic for more information.

  • Sample Outside Profile - Allows you to specify the Sample Outside Edge Profile to use in the Preview pane in this area.

If no Sample Outside Edge Profile is available for selection then that means no Outside Edge Profile has been allowed for the current Door Style. See the Profile Tab topic for more information.

  • Sample Inside Profile - Allows you to specify the Sample Inside Edge Profile to use in the Preview pane in this area.

If no Sample Inside Edge Profile is available for selection then that means no Inside Edge Profile has been allowed for the current Door Style. See the Profile Tab topic for more information.

  • Sample Panel Profile - Allows you to specify the Sample Raised Panel Detail to use in the Preview pane in this area.

If no Sample Raised Panel Detail is available for selection then that means no Raised Panel Detail has been allowed for the current Door Style. See the Profile Tab topic for more information.

  • Sample Route Profile - Allows you to specify the Sample Route Profile to use in the Preview pane in this area.

If no Sample Route Profile is available for selection then that means no Route Profile has been allowed for the current Door Style. See the Profile Tab topic for more information.

Preview

The Preview pane is the largest part of the General Info tab. In the Preview pane, you will see a graphic representation of the Door that is similar to the graphic representation that will appear in a Job. This view may be rotated, zoomed, and rendered as if it were in a Job. Also, the Door may be exploded or bent to give you a more detailed view.

There are six options in the Preview Pane:

Rotate View

By Default Rotate View is active. To rotate your Door simply click in the View area and drag, and the Door will Rotate. If you deactivate Rotate View you can click and drag to Pan the View of the Door.

Zoom

This tool is used for increasing the size of an area on your screen to see a detail or examine a condition. By clicking on the Zoom In tool, then clicking in the center of the area you would like to Zoom In to, and, while holding the mouse button down, dragging the cursor diagonally across the item you wish to zoom in on, finally releasing the mouse button at the end of the area, your screen will zoom in.

Reset View

Use this Tool after Rotating, Panning, or Zooming to bring the View to default.

View Mode

There are three View Modes:

Line Mode - This is the default view. You will see the Door as if it had wire lines at the corners.

Fill Mode - This mode of rendering causes the Door to appear solid, with a Finish added to them. The Door does not display wood grain.

Render Mode - This mode of rendering causes the Door to appear solid, with a Finish and Textures (Wood Grain) applied to it.

Explode Tool

Allows you to incrementally Explode your Door.

Bend Tool

Allows you to incrementally Bend your Door.
