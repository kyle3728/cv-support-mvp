Door Catalog Shape Editor

The Door Catalog Shape Editor consists of several tools designed to allow editing of Door Rails, Door Routes, and Door Profiles.

Add Point

Selecting this tool allows for additional points to be added to your object. Adding points allows for defining shapes by introducing new sides, or splitting sides into more than one element. When a point is added it appears as a yellow dot on the object. These yellow dots will be joined by a dashed line that represents a side. Additional dots mean more sides.

Delete Point

This tool functions opposite of the Add Point Tool. When selected, each point that is clicked on will be deleted.

Move Points

This tool allows existing points to be moved to a new position on your object. Click on the point and release the button, as the point will become your cursor. Reposition the point and click again to set it in its new position. Be careful not to cross other lines when editing the position of a point. If a point on the left end is moved to the right end, it will destroy the shape of an Object, and probably result in the loss of the Object. Points can be moved anywhere along the dashed line UNTIL they reach their neighboring yellow point. If it is necessary to move the dot beyond its neighbor, it is better to delete the current point, and create a new one in the proper area.

Another feature that will accomplish the same purpose as this tool and do it more precisely, is accessed by right clicking on the yellow point.

The Point Position area allows you to control the position of the point by inputting the position of the point along two axes. In this example, the point is located at "300" X, which equates 300mm in X and 0mm Y offset.

Parallel Line

This tool allows two adjacent points to be  moved at the same time. When selected, the yellow position dots will disappear from the corners, and reappear in the middle of lines that are eligible for the parallel move option. Click the yellow dot, and move the line to the new position and click again to set it.

Arc Line

This tool will create a new point when it is clicked on a dashed line. The point can then be pulled and as it does so, an arc will appear between the two reference points adjacent to the area you have clicked. By observing the data boxes in the sidebar, you will see the offset of your arc. This arc starts and ends at the adjoining points, so creative use of the points can create an edge that arcs out, then in, etc.

Radius Points

This tool will radius an existing point on an object. When selected, the menu prompts for the value to use for the radius. The entered value becomes the radius of the arc. When using this Tool it is important to select the radius value so that the tangent points do not cross over, or position themselves on, the neighboring yellow points. This is clearly visible when using this tool.  If they do cross, click the same point that applied the radius to and select a smaller radius.

Reference Point

Occasionally, you may need to change the location of the reference point in an object you are editing. This feature will allow that point to be changed to one of the other points by simply clicking on the new selection. Changing the reference point is only temporary during editing and then returns to it's default position after editing is complete.

Show Sample Profile

This tool shows a sample Door Profile to help orient you when creating the different Door Profile Types. Typically, you would utilize this Tool to orient yourself prior to Shaping then turn it Off to create/edit your Door Profile.

Show/Hide Lines

This Tool is available when creating/editing Custom Routes in the Door Catalog. You can use this option to turn off certain Lines in your Route Shape that would not actually be Routed. Because a Route Shape has to be one continuous line this allows you to turn off certain sections of it.

Scale Image

This tool is made available in the Door Profile Catalog when Importing a Bitmap. This Tool allows you to scale the Bitmap to fit the overall Thickness/Height of your Profile.

Align Image

This tool is made available in the Door Profile Catalog when Importing a Bitmap. This Tool allows you to align the Bitmap to the Origin of your Profile.

Lock Points

This tool allows you to lock a selected point. Locking a Point can be utilized to restrict stretching/shrink when Solid attempts to resize the Shape. Clicking on a Point with this Tool selected will bring up the following screen.

You can Lock the selected point both Horizontally and Vertically. For example if I select a Point to lock Left Horizontally the Length of the line from the Left-most point to the selected point would remain fixed and any stretch/shrink required would appear to the right of this point.

Show Grid

This tool allows you to turn on/off the display of the Grid.

Snap To Grid

This tool turns on the option to Snap to the Grid lines.

Grid Unit Box

Allows for resetting the pattern size and snap precision of the Grid.

Zoom Window

This tool allows you to Zoom In.

Zoom Out

This tool incrementally Zooms Out your image.
