Part Manager

The Part Manager is relatively straight forward in design and usage. It is primarily responsible for holding information on Parts that you need within Solid. It is relatively complete as it stands, containing the majority of Parts that most shops will use.

Part Tab

Name

The Name is an abbreviated code assigned by Cabinet Vision to the Part. Although you can change the Part Description, you can never change this Name once it is entered.

Description

The Description of the Part can be changed by changing the text here. The Description of the Part here is the "Name" used throughout Cabinet Vision including all output such as Cut Lists.

This will change the name of the Part everywhere it is used.

CNC Tab -

Optimize

The Optimize column is a toggle. Assuming you have the Optimizer or Nesting, you can toggle Parts on or off as to Optimizing/Nesting.

Ignore Grain

The Ignore Grain column is a toggle. Assuming you have the Optimizer or Nesting, you can toggle Parts on or off as to whether honoring Grain rules for the Part is required.

CNC

The CNC column is a toggle. If you have CNC capabilities, you can choose which Parts are outputted to CNC by toggling them off or on.

Finishes Tab

Exterior Face Finish, Exterior Back Finish, Exterior End Finish, and Exterior Edge Finish

The next columns determine whether the Part uses the Exterior (enabled) or Interior (disabled) Finish is used for the Parts Face, Back, End and/or Edge. The choices are Enabled (match Exterior Finish in the Job/Room/Assembly) or Disabled (match Interior Finish in the Job/Room/Assembly).

Exterior Face Texture, Exterior Back Texture, Exterior End Texture, and Exterior Edge Texture

The next columns determine whether the Part uses the Exterior (enabled) or Interior (disabled) Texture is used for the Parts Face, Back, End and/or Edge. The choices are Enabled (match Exterior Texture in the Job/Room/Assembly) or Disabled (match Interior Texture in the Job/Room/Assembly).

Right Click Options

Right clicking the Column Headers will bring up the Columns window allowing you to hide and rearrange (drag) the Columns displayed.

The Ribbonbar

Return - Closes the Part Manager and returns you to the Splash Screen.
