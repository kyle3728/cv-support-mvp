Properties

When you have a Material selected in the Material Manager you can get to it's Properties by clicking this option in the ribbonbar (alternatively double click the Material or right click and select Properties from the menu).

Ribbonbar

Following are the ribbonbar options in the Material Properties area:

Return

This option will return you to the Material Manager.

Select

Allows you to Select a different Material to work with in Material Properties.

The sidebar in the Select Material window allows you to Search for a Material or browse for Materials via the Material Categories. Once you have located your Material simply select it from the list of Materials in the Work Area and click OK.
