Board Stock

This topic covers all the Properties for the Board Stock Material Type.

Material Group

Previously covered in the Banding Material Type topic.

Board Group

  • Thickness - Establishes the Boards finished Thickness. The Thickness is the most important property of a Material because the construction of your Cabinets will depend on the Material Thickness in order to give you a correct cut list.

  • Rough Thickness - Establishes the Boards Rough Thickness for proper Board Foot/Meter calculations. This field only comes into play if BD FT or BD M is selected for the Unit of Issue.

Other than selecting a different Unit (BD FT or BD M in place of a Sheet Size) the logic is the same in completing these two areas. You may choose to enter Board Stock as dimensional lumber and define the Thickness in the Thickness box. An example, 4/4 Oak for the Name and 3/4 for it's Thickness. This would appear on your Cut List as:

The 4/4 references the Board Stock's name. This is a matter of taste as the 4/4 reference has no effect on the Material Thickness. To control the purchased (Rough) thickness to get accurate Board Stock Material Requirements enter the thickness in the Rough Thickness field. Additionally, you may need thicker stock for manufacturing other items. Keep in mind that the Name you provide for your Material is the name that will appear on the Cut List, and in the Material Schedule area when accessing the Material from the Material Manager.

CNC Group -

The CNC Properties section allows you to set up the CNC properties for the current Material. For example, the "Minimize Face Chipping" and "Minimize Back Chipping" check boxes will be used by the S2M Center's Automatic Tool Selection Logic.

  • Optimize - If the Material that you are setting up is sheet stock, you should set up the Optimization rules so that Solid and/or the S2M Center will know how to treat the Material in the Optimizer/Nester. This option lets you select whether or not to send this Material to the Optimizer/Nester when it is used in a Job. Although this Property is available for Board Stock it would be unusual to send Board Stock Parts to the Optimizer/Nester.

  • Grain - This option will let you determine whether Parts may be rotated across the Sheet/Board in the Optimizer/Nester. It is a good idea to set up these rules for any new sheet stock Material even if you do not currently have the Optimizer option for Solid/S2M Center. If you upgrade to Optimization/Nesting in the future, it will already be set up.

  • Face Dependent - If this option is enabled the Material is considered Face Dependent (single sided).

  • Drop Width - A value representing the Minimum Width to be considered an Offcut rather than scrap.

  • Drop Length - A value representing the Minimum Length to be considered an Offcut rather than scrap.

  • Feed Rate Percent - A value representing the Percent of optimum Tool Feed Rate to use with this Material.

  • Spindle Speed Percent - A value representing the percentage of optimum Tool Spindle Speed (RPM’s).

  • Minimize Face Chip - If this option is enabled, the S2M Center automatic tool selection logic will try to select a down-shear bit. If both this option and "Minimize Back Chip" are enabled then the S2M Center automatic tool selection logic will try to select a compression bit.

  • Minimize Back Chip - If this option is enabled, the S2M Center will try to select an up-shear bit for that Material. If both this option and "Minimize Face Chip" are enabled then the S2M Center automatic tool selection logic will try to select a compression bit.

  • Climb Cut - Establishes if this Material should be cut with Climb Cut.

  • Maximum Depth Per Pass - Establishes the depth per pass a tool can make when cutting this Material. The smaller of the two amounts (material versus tool) will be used for the path.
