Material Area

This area of the Material Manager displays all of the Materials in the selected Category or all the Materials meeting the Search criteria in an active Search.

Standard Properties for each Material are located on the General Tab and any special Properties will be stored on the extra tabs. The Material ID and Material Name will be displayed on each tab so you can easily see which Material you are working on. Using the tabs is a good way to quickly change Properties on multiple Materials. For focused configuration of a single Material it is recommended to open the Properties window for that Material.

Right Click Options

Right clicking the column headers will bring up the Columns window allowing you to hide and rearrange (drag) the columns displayed.

Following are the options available when right clicking a Material in the Material Manager:

Redefine

This option allows you to re-run the New Material Wizard on the selected Material.
