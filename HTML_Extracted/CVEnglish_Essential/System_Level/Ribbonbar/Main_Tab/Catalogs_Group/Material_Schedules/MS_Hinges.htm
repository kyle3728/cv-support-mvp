Hinges

Hinge Material Schedules are created in the same manner as the Assembly Parts.

Sidebar/Properties

The following Properties are available for Hinges:

  • ID - The ID number for the selected Material Schedule.

  • Name - The Name of the selected Material Schedule.

  • Description - The Description of the selected Material Schedule.

Hinge schedules should be set up for different configurations that you may want to use at the Assembly Level. One thing to keep in mind: When using a Face Frame Style Construction Method, the Assemblies Door size is controlled by the Overlay that is set for the selected Hinge. Therefore, if you use Face Frame Construction and have Hinges that can be used with different plates that will change the overlay, you should create a new Hinge for each overlay possibility (in the Material Manager) and set up a different Hinge Material Schedule for each possibility as well.
