Pull

Use the Pull Material Schedule to assign Pulls to Doors and Drawers. This window functions in the same way as Assembly Parts.

Sidebar/Properties

The following Properties are available for Pulls:

  • ID - The ID number for the selected Material Schedule.

  • Name - The Name of the selected Material Schedule.

  • Description - The Description of the selected Material Schedule.

  • Rotate Door Pull - Indicates to rotate your Door Pull by 90 Degrees for the current Pulls Material Schedule.

  • Rotate Drawer Pull - Indicates to rotate your Drawer Pull by 90 Degrees for the current Pulls Material Schedule.
