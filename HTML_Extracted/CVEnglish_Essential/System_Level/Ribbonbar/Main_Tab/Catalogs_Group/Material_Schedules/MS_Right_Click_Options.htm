Right Click Options

When Right Clicking on a Part, depending on its Part Type and the Material assigned to it you will have several Right Click options.

Materials

  • Match Parent Material - This option allows you to match the Material of the Parent for the selected Line Item. If there is no Parent to the item this option is grayed out.

  • Change Material in Multiple Schedules - This option allows you to set the selected Parts Material in multiple Material Schedules simultaneously.

When entering the Change Material Utility you will see the active Part in the titlebar. You have several options here:

    • Set Material - This option will automatically have the Material from the Part you used to enter the utility with. Simply click this Apply to change all selected Parts to this Material.

    • Match Parent - This option will allow you to change all the selected Parts to match their respective Parent's selected Material.

    • Select - This button allows you to select an alternate Material to "Set Material" on all the selected Parts

  • Remove Material - Removes the Material from the selected Part.

Variable Sizing

These options are available for Drawer Guides and Sliding Door Rails. See the Drawer Guides and/or the Sliding Door Rails topic for more information.
