Right Click Part -

 

Following are the options when right clicking on a Part in the Face, Plan, and End views:

 Copy

This option will copy the selected Part and allow you to place the copied Part somewhere else within the Assembly.

 Delete

This option will delete the selected Part.

This option will only Delete user added parts. You have two options for Deleting System Parts.

      1. (Preferred) Go to the Section Editor, and use the "Hide Parts" option. This adds parameters to the Assembly that keeps CABINET VISION from ever creating the Part in the first place on a rebuild.

      2. Add the parameter QTY with a value of 0 to the Part in the Object Tree. Doing this, CABINET VISION will create the Part on every rebuild, then apply QTY = 0 to it making it no longer display or go to CNC/cut-list.

 Replace

There are two options in the Replace fly-out:

  • Import DXF Part - The only difference between using a DXF to replace a Part rather than simply importing a DXF is the selected DXF will conform to the dimensions and Materials of the Part it is replacing. See the Import topic for more information.

  • Library Part - The only difference between using a Library Part to replace a Part rather than simply bringing in a Library Part is the selected Library Part will conform to the dimensions and Materials of the Part it is replacing. See the Library Part topic for more information.

 Save To Library

The Save To Library option allows you to Save the selected Part to a Part Library for retrieval later. Select the Library from the list on the left where you want to save the selected Part.

Following are the options here:

  • Preserve Size - This option will Preserve the current Size of the Part into the Library. Without this option enabled when you use Replace with Library Part the Library Part would take on the sizing characteristics of the Part being Replaced.

  • Preserve Material - This option will Preserve the current Material of the Part into the Library. Without this option enabled when you use Replace with Library Part the Library Part would take on the Material characteristics of the Part being Replaced.

  • Preserve Attributes - This option will Preserve any Attributes you have added to the Part.

  • Preserve Notes - This option will Preserve any Notes you have added to the Part.

  • Preserve CAD - This option will Preserve any CAD you have added to the Part.

  • Output to CNC - This option if selected will send this Library Part to the S2M CENTER when the rest of your Job is sent. If this option is disabled the Library Part will not be transferred to the S2M CENTER.

 Edit

The Edit option will take the Selected Part to the Part Level for Editing.

 Edit Shape

The Edit Shape option will bring up the Shaping tools to allow you to shape the Part.

Click the options below for more information

There are several options specific to editing the Shape of a Part which should be noted.

Selecting a Point on a Parts Shape

When selecting a Point on an Part's shape you are presented with the following sidebar options:

Start X - Allows you to control the X Position of the Point. You can also activate "Simple Constraints" on the Point. See the Constraints topic for more information.

Start Y - Allows you to control the Y Position of the Point. You can also activate "Simple Constraints" on the Point. See the Constraints topic for more information.

Selecting a CAD Entity on a Part's Shape

When selecting a CAD entity on an Assemblies shape you are presented with the following sidebar options:

Start X - Allows you to control the Starting X Position of the selected CAD entity. You can also activate "Simple Constraints" on the Point. See the Constraints topic for more information.

Start Y - Allows you to control the Starting Y Position of the selected CAD entity. You can also activate "Simple Constraints" on the Point. See the Constraints topic for more information.

End X - Allows you to control the Ending X Position of the selected CAD entity. You can also activate "Simple Constraints" on the Point. See the Constraints topic for more information.

End Y - Allows you to control the Ending Y Position of the selected CAD entity. You can also activate "Simple Constraints" on the Point. See the Constraints topic for more information.

Length - Allows you to control the Length of the selected CAD entity.

Angle - Allows you to control the Angle of the selected CAD entity.

Miter - Angle - Allows you set the Miter Angle for the selected edge.

 

 Relate To Part

The Relate to Part option will allow you to add intelligence to a new or existing Part and relate the selected Parts properties to an existing Part within your Assembly. In the following Exercise we will add Intelligence to a Board relating it to the Top Front Stretcher. This Part could be used as a Sink Skirt.

  1. Select Top Stretcher to be the "Relate To" Part then select Part Length + 0.

  1. Select the X Position to be Part X Position + 0, the Y Position to be Part Y Position + 0, and the Z Position to be Part Z Position + 3.25.

  1. Select the X Axis Rotation to a Value of 0, the Y Axis Rotation to a Value of 0, and the Z Axis Rotation to a Value of -90.

The added Part is now Intelligent and as long as the Top Stretcher it is referenced to exists it will as well.

 Edge Banding

The Edge Banding option allows you define which edge of a Part gets edge banded with which edge band type. Once the Edge Band option is selected you will be placed in Edgeband mode with the following 4 options in the ribbonbar.

  • Select - When entering Edgeband mode this tool is selected by default. Simply click on the edge you would like to apply the Edgeband to and then select its type in the sidebar.

You can use Windows + Click to select multiple edges.

  • Select All - To quickly change all edges to a specific edgeband select this option then select their type in the sidebar.

  • Remove Banding - Will remove Edgebanding from the selected edge(s).

  • Set Banding - Will expose the Edgeband type selector in the sidebar without having to select an edge first.

It is important to note that the default values in the sidebar options are NOT representative of the current Edge Banding on the selected Part. To use this function simply select which Edgebanding type you would like and whether to Trim the Part for the edgebanding or not. It is also important to note that ALL system edgeband is applied with Trim Part set to yes. So, it would probably be desirable in most cases to set this option to yes.

Once the Banding and Construction options are chosen simply select any of the Part Edges you would like these Edgeband properties assigned to. In the image below the Front Edge of the End has been selected.

Note that the front edge is now Red which indicates an Exterior Edgeband has been applied. Once you are through defining the Edges simply click the Return option to save the new Edge Properties.

 Edit Splits

The Edit Splits option allows you to split a Part into multiple Pieces. Visually it will still appear to be one piece. However, in your Cut Lists and CNC Outputs you will get separate Parts for each Split. This menu option brings up the Split Editor options.

Click the options below for more information

 Properties

There are six tabs in the Part Properties screen:

  • General Tab - Following are the options on the General Tab:

    • Part Group - Select the Part Group to select the new Part from. These, in general, correspond to the Part Manager Part Categories.

    • Part Type - Select the Part's type here. The Parts here come directly from the Part Manager.

    • Description - By Default, the Description for the Part will come from the Part Manager and is what is displayed in your Reports. However, this field is editable and you may change the Description to better describe "this" Part in your Reports.

    • Material Change - By Default, the Material for the Part comes from the Material Schedule for the Part Group that the Part is located in.

    • Texture Map - Allows you to set how a Texture is applied to the Part. See the Room Level Import DXF Texture Map topic for more information.

    • Smooth Surface - Allows you to enable Smooth Surfacing on the Part. See the Room Level Import DXF Smooth Surface topic for more information.

    • Layout Visibility (Not available for System Parts)

      • Hide in Plan - Will hide the Part in Plan views.

      • Hide in Elevation - Will hide the Part in Elevation views.

      • Hide in Perspective - Will hide the Part in Perspective (3D) views.

  • Comment Tab - You add special notes on this Part here. Certain cut lists from the Report Center will display this information for the shop.

  • Flutes Tab - - The Flutes Tab allows you to specify whether the Part will receive Flutes. See the Job Level Flutes topic for more information.

  • Finish Tab - The Finish Tab allows you to specify what Finish and/or Texture is applied to the selected Parts Front Face, Back Face, Edges, and Ends.

  • Overrides Tab - The Overrides Tab will display and allow removal of any Overrides on the selected Part. Since an End using a notched toe is selected there is a "Shape - XY Template" on the Part.

 

  • Parameters - - This area allows you to see any Parameters on the current Part. See the Intelligence Workshop for more information on modifying these Parameters.
