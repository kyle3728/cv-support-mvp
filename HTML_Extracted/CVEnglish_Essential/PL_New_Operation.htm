New Operation -

Adding an operation to the Part: To add a manual Operation to a Part, begin by clicking the New Operation Button.

After clicking the New Operation button, you will see the new Operation wizard which will guide you through the process of adding the new Operation. In the Face area, you need to select which Face that you want the Operation to appear on. Generally, if you want the Operation to be on the primary Face so that it gets cut out on the nest, you would choose the Back face. See the Work Planes topic for more information.

In the Operation area, you will need to select the type of Operation that you want to add.

Following are the Operation types to select from:

  • Blind Dado-Master - Allows the addition of a Master Blind Dado to your Part. A Master Blind Dado is applied to the Master Part and the Machining for it actually gets applied to the Slave Part. This is very similar to an Intelli-Joint utilizing a Slave Dado.

  • Clamex-Master - Allows the addition of a Master Clamex Operation on your Part. A Clamex Master is applied to the Master Part and the Machining for it gets applied to both the Master and Slave Part.

  • Dado - Allows the addition of a Dado to your Part.

  • Dado-Master - Allows the addition of a Master Dado to your Part. A Master Dado is applied to the Master Part and the Machining for it actually gets applied to the Slave Part. This is very similar to an Intelli-Joint utilizing a Slave Dado.

  • Dove Tail Master - Allows the addition of a Master Dove Tail (Male) to your Part.  A Dove Tail Master is applied to the Master Part and the Machining for it actually gets applied to both the Master (Male) and the Slave (Female) Part.

  • FastenLink - Allows the addition of a FastenLink Operation.

  • FastenLink Master - Allows the addition of a FastenLink Master Operation.

  • Hole - Allows the addition of a Hole to your Part. A Hole is round and it's shape can not be altered.

  • Hole-Master - Allows the addition of a Master Hole to your Part. A Master Hole is applied to the Master Part and the Machining for it actually gets applied to the Slave Part. This is very similar to an Intelli-Joint utilizing a Slave Bore.

  • Intelli-Joint - Allows the addition of an Intelli-Joint to your Part.

  • Line Bore - Allows the addition of a Line Bore or Line Bore series to your Part.

  • Line Bore-Master - Allows the addition of a Master Line Bore to your Part. A Master Line Bore is applied to the Master Part and the Machining for it actually gets applied to the Slave Part. This is very similar to an Intelli-Joint utilizing a Slave Bore.

  • Route - Allows the addition of a Route to your Part. A Route can be shaped after being added in the Part Shape Editor.

  • ALPHACAM Drawing - This Operation type will allow you to select an ALPHACAM Drawing that has been saved into the S2M CENTER Part Library to add to your Part. See the S2M CENTER Part Library Help topic for more information.

If you select an Intelli-Joint the next area is the Intelli-Joint area. There will be a list of all Intelli-Joints you have created. You also have the following additional options:

  • Joint Search Scope

    • Entire Room - Lets the selected Intelli-Joint search outside of the Assembly to adjacent Assemblies to apply Machining. Very handy for Assembly Connector Machining.

    • Current Assembly - The Default selection, this option only evaluates Slave Parts within the Assembly the Intelli-Joint is located.

If you select an ALPHACAM Drawing the next area is the ALPHACAM Drawings area. There will be a list of all ALPHACAM Drawings you have saved in your S2M CENTER Part Library.

In the Dimensions area, you can set up the size of the operation.

The Dimensions area is skipped if you have selected an ALPHACAM Drawing. ALPHACAM Drawings are not Parametric and must be created and saved in the S2M CENTER Part Library at the size they will be used.

If you select a Blind Dado-Master Operation the next option would be the Blind Dado Dimensions.

 Following are the options here:

  • Top Notch (T) - Specifies the Length of the Top Notch; represented by the letter T in the on screen illustration.

  • Bottom Notch (B) -  Specifies the Length of the Bottom Notch; represented by the letter B in the on screen illustration.

  • Tenon Width (W) -  Specifies the Tenon Width; represented by the letter W in the on screen illustration.

  • Side - Allows you to choose the side of the Part that the Tenon is on.

  • Pocket Width (H) - Specifies the Pocket Width, represented by the letter H in the on screen illustration.

Next is the Orientation are where you will define the location and Orientation of the Operation.

Following are the options here:

  • Rotation - Specifies the Operations Rotation on the Part it is being applied to. Following are the options here:

    • None - Leaves the Operation in it's default Orientation.

    • Rotate 90 - Rotates the Operation 90 degrees.

    • Rotate -90 - Rotates the Operation -90 Degrees.

    • Rotate 180 - Rotates the Operation 180 Degrees.

    • Other - If Other is selected the value box activates and you can enter a specific angle of Rotation for the Operation.

  • Distance from Left - Specifies the distance from the left of the Part to place the Operation. Following are the options here:

    • Value - Takes the Value you enter as is.

    • Part Width + - Takes the Parts Width and adds the entered value to it. A negative value can be used to subtract.

    • Part Length + - Takes the Parts Length and adds the entered value to it. A negative value can be used to subtract.

    • Part Thickness + - Takes the Parts Thickness and adds the entered value to it. A negative value can be used to subtract.

  • Distance from Bottom - Specifies the distance from the bottom of the Part to place the Operation. Following are the possible options here:

    • Value - Takes the Value you enter as is.

    • Part Width + - Takes the Parts Width and adds the entered value to it. A negative value can be used to subtract.

    • Part Length + - Takes the Parts Length and adds the entered value to it. A negative value can be used to subtract.

    • Part Thickness + - Takes the Parts Thickness and adds the entered value to it. A negative value can be used to subtract.

    • Edge Length + - Tales the selected Edges Length and adds the entered value to it. This option is very useful on Shaped Parts after selected a Shaped Edge (Work Plane) to place the Operation on. A negative value can be used to subtract.

  • Zoom - Will Zoom in or out on the Operation.

Next is the Tool area, you will select the Tool that you wish to use for this operation. Unless there is a specific Tool that you wish to use, choose Auto Select to allow normal tool selection logic to apply. This area will only show Tool Types that would be valid with the Operation type defined. See the S2M CENTER Help system for more information on Tools, Tool Properties, and Automatic Tool Selection logic. This is the last area unless you have selected a Route or a Hole.

This is the last area for a Route. This area will only display if a Route Operation is being defined. Following are the options here:

  • Path - Allows you to establish how the Cut Path for the Route, there are three options:

    • Cut Along the Centerline

    • Cut On the Inside

    • Cut On the Outside

  • Pocket - Establishes that the Route is a Pocket.

This is the last area for a Hole if it is being routed through. This area allows you to select whether the Hole is pocketed out or routed out. Following are the options here:

  • Use setting in S2M CENTER geometry preferences - This option will follow the setting in the S2M CENTER Geometry preferences for machining holes. See the S2M Center Help system for more information.

  • Pocket the entire hole - This option will pocket the Hole out when machining so as to not create projectiles of the negative space left from the route operation.

  • Route hole outline only - This option will Route the perimeter of the hole only to machine it out.
