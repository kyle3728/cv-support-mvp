Reports View

The Reports View allows you to print Reports for the selected Catalog on the Catalog Pane.

Ribbonbar

Zoom Group

Following are the options in the Zoom Group.

In - Will incrementally Zoom In your Report.

Out - Will incrementally Zoom Out your Report.

Zooming Out far enough will show multiple pages in the viewing are if your Report contains multiple pages.

Page - Will Zoom to fit the entire page in the viewing area.

Page Width - Will Zoom to fit the page's Width in the viewing area.

Navigation Group

This option allows you to navigate from Page to Page in the currently visible Report.

Tools Group

See the Export Report topic for more information.

Sidebar

There are 4 Standard Reports that are available for each Catalog.

Door Styles

The Door Styles Report shows the available Door Style Names in the selected Catalog along with the Base, Upper, and Drawer Front Door Type for that Door Style. Also shown in this report is the cost for each Door Type and its up-charge.

Line Pricing

This is a report on the Line Pricing Formulas that have been set up for the currently selected Catalog along with information about those Formulas.

Options

This is a report of the Options available for the selected Catalog and the associated modifications (i.e. cost multipliers).

Variables

This is a report of the Variables set up for the currently selected Catalog and their associated value.
