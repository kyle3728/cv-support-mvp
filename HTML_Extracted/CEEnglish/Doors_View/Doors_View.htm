Doors View

Typically, the Door name used in the manufacturing shop may differ from the actual Door Style name used when selling a product. For example the Door can be referred to as a 022 Door. However, when selling that Door in a retail setting, the Door would be known as a Square Flat Panel Door Style. The Door Style view in the Catalog Editor is where the Square Flat Panel would be linked to the 022 Door from the Door Catalog.

Maneuvering Door Styles:

The order in which Door Styles are displayed can be changed by selecting and highlighting the entire line and holding the SHIFT button while using the Arrow keys on the keyboard to move the item up or down the list.

Door Styles can also be Cut, Copied, or Deleted by right clicking on the line that contains the Door Style and selecting Copy, Cut, or Delete. If selecting Copy or Cut, then move to the new line and right click, select Paste and note the additional options to Append, Insert Before, and Insert After. This not only copies and moves the Door Style name, but all associations as well.
