Route Ptrn

Click on the Tabs in the image below for additional information

This window shows the Route Pattern Profiles that are available for the selected Door. The Profiles may be enabled by clicking on the Enabled check box in the Enabled column. An X in the Enabled Box indicates that the Profile is available for that Door. The Upcharge column allows for an amount to be added to the pricing according to which Profile is selected for the Door.

Clear Prices

All  

This button clears all of the prices for the selected Door across all of the categories (Price, Outer, Inner, RP Detail, Route Ptrn, Material)

Current List  

This button clears the prices for the selected Door in the current category only.
