Style Pane

There are six columns in the Style Pane:

Style Name

The Style Name is the name that the end customer would see regarding Door selections. Typically, the Door Name used in the manufacturing shop may differ from the actual Door Style name used when selling a product. For example the Door can be referred to as a Four-Quarter 265 Arch Oak Door. However, when selling that Door in a retail setting, the Door would be known as a Regency Cathedral Door Style. The Door Style view in the Catalog Editor is where the Regency Cathedral would be linked to the Four- Quarter 265 Arch Oak door from the Door Manager.

Price Category

In the Price Category field, choose the appropriate Price Label to link to the Door Style. So, when this Door Style is chosen in a Job, that Price Category will be used for pricing the objects.

If an object has a zero in a Price Category linked to the selected Door Style in CABINET VISION, when an attempt is made to use that object in Solid, there will be a message saying "Not Available in This Door Style". An object with a zero price will get this message. If the object should have a Price, open the Catalog Editor, right click on the object in the Item Pane and select Edit Pricing. Then enter a price for that object in the appropriate Price Category.

Base Door

This column will display the selected Base Door Name (the name from the Door Manager) for a particular Door Style.

Upper Door

This column will display the selected Upper Door Name (the name from the Door Manager) for a particular Door Style.

Drawer Front

This column will display the selected Drawer Front Name (the name from the Door Manager) for a particular Door Style.

Description

This column provides a field to add a field to add an explanation of the selected Door Style.
