Modifications

Use this interface to link Cabinet Modifications to the selected Option.  Modifications are listed by type below:

Type

Toe Height:  Enter the desired Toe Height in the Value field

Single Door -> Pair Doors:  Non Graphic Modification changing a Single Door to a Pair of Doors.

Pair Doors -> Single Door:  Non Graphic Modification changing a Pair of Doors to a Single Door.

Omit Doors:  Selecting this Option and setting the value to 1 will remove Doors from the Object.

Butt Doors:  Non Graphic Modification that changes the Doors to Butt Doors.

Clip Corner - Left:  Enter the amount of the clip in the Value field

Clip Corner - Right:  Enter the amount of the clip in the Value field

Clip Corner - Both:  Enter the amount of the clips in the Value field

Radius Corner - Left:  Enter the amount of the radius in the Value field

Radius Corner - Right:  Enter the amount of the radius in the Value field

Radius Corner - Both:  Enter the amount of the radius in the Value field.

Angled End - Left:  Enter the desired angle (in degrees) for the Cabinet end

Angled End - Right:  Enter the desired angle (in degrees) for the Cabinet end

Angled End - Both:  Enter the desired angle (in degrees) for the Cabinet ends

Add Shelf:  Non Graphic Modification that is used to represent the addition of a shelf.

Delete Shelf:  Non Graphic Modification that is used to represent the removal of a shelf.

Peninsula:  Non Graphic Modification to represent a change to a Peninsula.

Extended Stile - Left:  Enter the desired amount of extension for the stile in the Value field

Extended Stile - Right:  Enter the desired amount of extension for the stile in the Value field

Extended Back:  Enter the amount to extend the back down in the Value field

Extended Left End:  This Modification will extend the Left End down the amount entered in the value field.

Extended Right End:  This Modification will extend the Right End down the amount entered in the value field.

Flush Top/Bottom:  Non Graphic Modification that represents a Flush Top and Bottom.

Recess Toe Left:  Enter the amount of toe recess in the Value field

Recess Toe Right:  Enter the amount of toe recess in the Value field

Recess Toe Back:  Enter the amount of toe recess in the Value field

Flush Front Toe:  Selecting this Option and setting the value to 1 will Flush the Toes

Split Drawer:  Non Graphic Modification to represent splitting a Drawer into two Drawers.

Invert Door/Drawer:  Non Graphic Modification to represent swapping the positions of the Door and Drawer.

Full Depth Shelf:  Non Graphic Modification that represents changing a shelf to Full Depth.

Glass Mullion Doors:  This Option changes the existing Door Panels to Glass.  The value corresponds to the number of lites.

Panelized Ends:  Change the Cabinet Ends to Panelized.  The value should be set to 1 to turn this Modification on.

Omit Pulls:  Non Graphic Modification that represents omitting the Pulls from the selected Object.
