Construction

Use this interface to link a Construction Method to the selected Option. The preferred practice is to link no more than what the selected Option calls for. For example, if the Option is called Face Frame Construction, then the user should only select the desired Face Frame Construction Method in the Cabinets drop down box. It would not be wise to limit the Drawer Box and Roll Outs Construction Methods to the same Cabinet Construction Option, these should probably receive their own Options.
