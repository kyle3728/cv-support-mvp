UCS

Use this interface to link a User Created Standard to the selected Option.

Setting Up a User Created Standard as an Option Effect

The best way to demonstrate this function is to create an example. The following procedure will create an Option that deletes any Mid Stiles from a Cabinet when selected.

  1. First, it is necessary to create a UCS that deletes Mid Stiles. This can be easily accomplished through the UCS Wizard. If you need more instruction on creating a UCS, go through the exercise in the CABINET VISION Help Files. The UCS should look like this:

;Deletes Mid Stile

 For Each MS Part

 QTY<int> = 0

Make sure that the UCS is enabled.

  1. Next, using the Catalog Editor and the Generic Catalog as an example, create the Option "Delete Mid Stile." This should look something like this:

  1. Then, double-click the Delete Mid Stile Option to get the Option Effects Window. Click on the UCS Tab and then select the Delete Mid Stile UCS in the drop down and set the value to 1.

  1. Next, you will need to close the Catalog Editor and open CABINET VISION. Start a new Job and place a Cabinet from the Generic Catalog into this Job (you should choose Face Frame Construction for the Catalog Construction Option). For this example, select the B33 Cabinet. This Cabinet can be found under Generic | Base | Double Door Base | W/2 Drawers.

  1. Then, from the Floor Plan View, right-click on that Cabinet and click Properties. In the Properties Window, click the Options Tab. Select the Delete Mid Stile Option that we created (see below). Then click OK.

  1. Next, you will be back at the Floor Plan View. Right-Click the Cabinet again and click Properties. This time, click the Parameters Tab. Towards the top of the Parameters list you should see a Parameter called _UCS followed by a number. In this example the Parameter is _UCS5 = 1. The Parameter for your Cabinet will most likely be something different. This Parameter represents a number that is automatically assigned to the UCS that we linked to this Option Effect. By selecting the Delete Mid Stile Option for this Cabinet, we have caused this Parameter to be added to this Cabinet.  Therefore, any time this Option is selected for a Cabinet, the _UCS5 Parameter will be added to that Cabinet with a value of 1.

  1. Finally, we need to add an If...Then statement to our UCS in order to apply that UCS only to Cabinets that have the Delete Mid Stile Option selected. Our changes will make the UCS look like this (with changes in bright green):

;Deletes Mid Stile

 For Each MS Part

If _UCS5 = 1 Then

      QTY<int> = 0

End If

Now, the UCS that we have written will only be applied to Objects that have the Delete Mid Stile Option selected.
