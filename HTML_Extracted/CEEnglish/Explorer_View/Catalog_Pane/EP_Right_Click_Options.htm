Right Click Options

There are five options when right clicking on a Catalog or Category:

  • Select Options - Allows you to set the Options for the selected Catalog.

  • New Menu - Allows you to add a New Category in the selected Catalog/Category.

  • New Assembly - Allows you to add a New Assembly in the selected Catalog/Category.

  • New Non-Graphic Accessory - Allows you to add a new Non-Graphic Accessory in the selected Catalog/Category.

  • Delete - This utility allows you to Delete the active Catalog.
