Edit Parent Assemblies

This option will open the Assembly Level of CABINET VISION with direct access to edit all of the Parent Assemblies in the currently selected Catalog. For more information on Parent Assemblies, see the CABINET VISION Help Topic on Aliases. The drop down box (highlighted yellow below) allows for the selection of any of the parent Assemblies in the selected Catalog. For more information on the Assembly Level, see the CABINET VISION Help Files.
