Simple Pricing Table

Simple Pricing is generally used to price single part Objects. This type of Pricing makes allowances on a per item basis and a size basis.

Click an area below for more information.

 

Item

This column shows the list of objects in the selected Catalog.

$ / Item

This column holds the price per item.

$ / Inch h

This column holds the price for the item as calculated per inch of height of the object.

$ / Inch w

This column holds the price for the item as calculated per inch of width of the object.

$ / Sqft w x h

This column holds the price for the item as calculated per square foot of the object width times height.

$ / Sqft w x d

This column holds the price for the item as calculated per square foot of the object width times depth.

$ / Sqft h x d

This column holds the price for the item as calculated per square foot of the object height times depth.

Catalogs

Select the Catalogs to apply a Simple Pricing Table to from a list of the available Catalogs. In the drop down list, the Catalogs using Simple Pricing will appear.
