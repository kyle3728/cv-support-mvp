Stock Pricing

Stock Pricing is the area in the Catalog Editor that allows you to customize the pricing for your specific Catalog. Stock Pricing gathers all pricing information that has been set up in all areas of the Catalog (assembly, accessory, finished good, door/drawer, and option pricing) and allows you to detail how this information is to be used with a series of formulas.

Variables

There are three variables that are present in every Catalog, they are:

  • Discount List Price - This variable is a percentage and will be taken off the total list price of the stock Catalog to arrive at the selling price.

  • MFG Multiplier - This variable is a percentage and will be multiplied with the total list price of the Catalog to arrive at the dealer's cost.

  • Upcharge Dealer Cost - This variable is a percentage that will be multiplied by the dealer cost to arrive at the selling price.

To add a new Variable, right-click anywhere in the Variable grid and select Add New Variable. There is a 20 character limit to the Variable Name. To delete a Variable, right-click on that Variable and select Delete Variable. To change a Variable Name, click on that name and enter the desired new name.

Variables must be added into a formula in order for them to be used in the pricing calculations and reporting.

Options

This area is for reference and simply identifies priced option types that are used in formulas. These Options are the Job Level and Assembly Level Option Types in the Catalog Editor. Option Line Names must match the Option Type Names listed in this area exactly.

Totals

The Totals area is for reference only and displays the Totals that are automatically calculated for the Catalog and may be used in creating Formulas.

Line Name

Only letters may be used in Line Names and there is a 20 character limit. The first six Line Names (CaseList, Accessories, FinishedGoods, DoorUpcharge, DrawerUpcharge, and PartPrice) are standard for every Catalog.

Formula

These are algebraic expressions that are used to arrive at the selling price of the objects in the selected Catalog. Formulas may also be used in arriving at the pricing for accessories and other items (such as counter tops or installation).

The algebraic expressions are created by using the listings for Variables, Options, Totals, Line, and/or dollar or percentage amounts; and using the function characters of (+) addition, (-) subtraction, (*) multiplication, and  (/) division. There is a 255-character limit within the formula. The formula is calculated in the order of the functions, as parenthesis cannot be used.

Example:

You have installation for cabinetry and installation for counter tops where there needs to be a multiplication to arrive at both totals. You will need two formulas if you only want to show one total, and three formulas if you need a separate line for both totals for reference or to show.

($ cabinet installation X #cases) + ($ top installation X Top Lineal Foot) must be:

Install Cabinets                <V:CabinetInstallation> * <T:#Cases>                                            HIDE

Installation                        <V:TopIntallation> * <T:LinFtTops> + <L:InstallCabinets>            SHOW

OR

Install Cabinets               <V:Cabinet Installation> * <T:# Cases>                                  HIDE or SHOW

Install Tops                      <V:Top Intallation> * <T:LinFtTops>                                        HIDE or SHOW

Total Installation              <L:Install Cabinets> + <L:InstallTops>                                             SHOW

Again, any line may be used in a formula cell. Any line used in a formula must be above the formula being created.

Price Per

This is a toggle that determines whether the formula applies to each cabinet/graphic item, or if the formula applies once for the Job.

Select a cell within the column and click on the button that appears within the field to change. Toggle lines to job such as a design fee that your company charges at a percentage of the cabinetry selling price (Subtotal) or a flat fee. If you do not toggle a flat fee to “Job” you will have that amount multiplied by the number of cabinets in the job. I.e. $100.00 toggled to “Job” will add $100.00 as the design fee, and if toggled to “Cab” with 36 cabinets in the job, the design fee will be $3600.00. If you want to vary the flat fee per job you will want to create a variable so that you can easily change the value in the Bid Center. If not, just create a line and enter the value desired in the formula cell.

Example: With Variable

Design Fee                <V:DesignFeeMultiplier> * <L:Subtotal>

Design Fee                <V:DesignFlatFee>

Example: Without Variable

Design Fee                100

Description

Text entered in the Description field will override text in the Line field.  The Description field should be used if more than 20 characters are needed for an adequate description for that line or if numbers and symbols are required.

Total (Show/Hide)

This is to toggle what lines show in the Total field of the stock catalog window in the Bid Center. Select a cell within the column and click on the button that appears within the field to change.

Assembly (Show/Hide)

This is a toggle that determines whether a line is shown in the assembly listing drop-down information and is calculated into the total shown for an individual assembly or item. Select a cell within the column and click on the button that appears within the field to change.

Accessory (Show/Hide)

This is a toggle that determines whether a line is shown in the accessory listing drop-down information and is calculated into the total shown for an individual accessory or item. Select a cell within the column and click on the button that appears within the field to change.

Guidelines for Total, Assembly, and Accessory Show/Hide

You can toggle the Total, Assembly, and Accessory columns to show or hide, depending on how you want the information to appear in the Bid Center.

Below is an example of the typical setup for Show/Hide.

  • CaseList                                                                     Hide/Show/Show

  • Accessories                                                               Hide/Show/Show

  • FinishedGoods                                                          Hide/Show/Show

  • DoorUpcharge                                                           Hide/Hide/Hide

  • DrawerUpcharge                                                       Hide/Hide/Hide

  • Options before Case and Opt Mods                        Hide/Show/Show

  • Case and Opt Mods                                                   Hide/Hide/Hide

  • Options after Case and Opt Mods                            Hide/Show/Show

  • Cabinet Total Show                                                    Show/Hide/Hide

  • Job Level Options after Cabinet Total Show           Show/Show/Show

  • ListPrice                                                                      Hide/Hide/Hide

  • AListPrice                                                                   Hide/Hide/Hide

  • Subtotal ListPrice                                                      Show/Hide/Hide

  • CostPrice                                                                    Hide/Hide/Hide

  • ACostPrice                                                                  Hide/Hide/Hide

  • Subtotal Cost Price                                                    Hide/Hide/Hide

  • Discount                                                                      Hide/Hide/Hide
