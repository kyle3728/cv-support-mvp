Object Pane

When a Category is selected in the Catalog Pane, the Object Pane will show a list of the Objects that can be found in that Category.

Nomenclature

This is the name of the object. It is recommended that the name include some indication as to the type and size of the object.

Width

Indicates the width of the object.

Height

Indicates the Height of the object.

Depth

Indicates the Depth of the object.

Weight

Indicates the Weight of the object.

Volume

Indicates the Volume of the object.

Autofill

This field indicates whether the object is to be used by Solid when the Autofill procedure is performed. See the Auto-Fill topic in the Solid Help file for more information.
