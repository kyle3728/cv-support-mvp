Object Properties

The Object Properties area has 5 possible tabs for a standard Object and one tab for a Non-Graphic Accessory.

General Tab

There are fourteen options on the General Tab:

  1. Nomenclature - Allows you to specify the name of the Object.

  2. Hinge Specifier - On a one Door Unit indicates the character (Typically L or R) that indicates a Hinge swing change.

  3. Description - Allows you to specify a Description for the Object.

  4. Width - Allows you to specify the Width of the Object.

  5. Height - Allows you to specify the Height of an Object, if it is an Alias.

  6. Depth - Allows you to specify the Depth of an Object, if it is an Alias.

  7. Weight - Allows you to specify the Weight of an Object, if it is an Alias.

  8. Cubes - Allows you to specify the Cubic dimension of the Object.

  9. Labor - Allows you to specify the Labor for the Object.

  10. Auto-Fill - Allows you to specify if the Object can be used with the Auto-Fill feature when performing Layouts in CABINET VISION. See the CABINET VISION Help file for more information on the Auto-Fill feature.

  11. Price per Sq. Ft - Allows you to specify if the Object is priced per square foot.

  12. Door/Drawer Overrides Base - This Overrides the Door Count for the Object. Let's say this is a Base Cabinet with two Doors but you need to charge like it has 4 Doors. Simply enter 2 in this field and the Cabinet will return a Door Count of 4.

  13. Door/Drawer Overrides Upper - This Overrides the Door Count for the Object. Let's say this is an Upper Cabinet with two Doors but you need to charge like it has 4 Doors. Simply enter 2 in this field and the Cabinet will return a Door Count of 4.

  14. Door/Drawer Overrides Drawers - This Overrides the Drawers Count for the Object. Let's say this is a Base Cabinet with two Drawers but you need to charge like it has 4 Drawers. Simply enter 2 in this field and the Cabinet will return a Drawer Count of 4.

Class Tab

CABINET VISION identifies objects based on the Class under which those objects have been saved. For example, an object that is saved under the Base Class would derive its Materials and Construction for a Base Cabinet.

Placement Tab

There are three main areas in the Placement Tab:

Horizontal

Check the boxes that apply to where you would want the current Object to be placed. If the Object is a Corner Object, you would probably want to check both Wall and Corner.

Vertical

Determines the Vertical Placement of your Object. There are thirteen Vertical Placement options:

    • Above Positional - Will place the Object above the value set for Vertical Positional on the Room | Layout Tab in Room Properties.

    • Above Elevation - Will place the Object above the Elevation established for the Object as shown below.

    • Above Soffit - Will place the Object above the Soffit as specified on the Room | Layout Tab in Room Properties.

    • Below Positional - Will place the Object below the value set for Vertical Positional on the Room | Layout Tab in Room Properties.

    • Below Elevation - Will place the Object below the Elevation established for the Object as shown below.

    • Ceiling - Will place the Object at the Ceiling (Wall Height) as specified on the Room | Layout Tab in Room Properties.

    • Counter Top - Will place the Object above the Counter Top.

    • Float - Will allow free placement Vertically on the Wall/Corner.

    • Toe Height - Will place the Object at the Toe Height.

    • Under Counter - Will place the Object below the Counter Top.

    • Under Soffit - Will place the Object below the Soffit as specified on the Room | Layout Tab in Room Properties.

    • Under Uppers - Will place the Object below the Upper Cabinets.

Stretching

Select the appropriate directions that you would want this Object to be stretched in. For example, if you did not want the designers to be able to change the Width of this Object when it is placed in a job, simply disable the Width option.

Comment Tab

Allows you to enter a Comment for the Object.

Exclusions Tab

Allows you to control which Options do not apply to the Object.
