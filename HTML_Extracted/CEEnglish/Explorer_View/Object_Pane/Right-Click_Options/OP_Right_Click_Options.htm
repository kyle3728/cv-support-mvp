Right Click Options

When Right Clicking an Object in the Object Pane you will have six Options:

  • Change Icon - Allows you to change the Icon that represents the Object in the Catalog.

  • Insert Label - Allows you to place an informational divider between Objects in the Object List.

 

  • Edit Object - Allows you to Edit the Object at the CABINET VISION Assembly Level. See the Assembly Level area of the CABINET VISION Help file for more information.

  • Edit Pricing - Allows you to edit the Price of the selected Object.

  • Delete - Allows you to Delete the selected Object.

  • Properties - Allows you to edit the Properties of the selected Object. See the Object Properties topic for more information.
