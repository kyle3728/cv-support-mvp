Explorer View Sidebar

The Explorer View Sidebar contains important information about the currently selected Catalog.

Catalog Area

Internal Name

This is a field that is intended to hold a name for the selected Catalog that is more descriptive than the Catalog Name that appears in CABINET VISION for the same Catalog. The recommended practice is to use the Catalog Name + the date of the most recent update.

Version

This is a field for the version number of the currently selected Catalog.

Description

This field may contain any helpful information about the currently selected Catalog.

Type

Allows you to set the Catalog type. There are five types, Undefined, Cabinet, Closet, Decorative, and Appliance.

It is recommended yo define your Catalog with one of valid Types and not the "Undefined" Type.

Modular

This check box turns the Modular Mode on or off.  If Modular Mode is on (the box is checked), then CABINET VISION will look to the information in the Catalog Editor for Construction, Material, and Pricing information for the selected Catalog. If the information is not found in the Catalog Editor for that Catalog, then CABINET VISION will look to the Job/Room Properties.

Hide List

If this box is checked, the List Price for an object from this Catalog will not appear in the CABINET VISION Bid Center.

Hide Cutlist

If this box is checked, the Cutlist for an object from this Catalog will not appear in CABINET VISION.

Doors Area

Database

This field is for the name of the Door Database that will be used for this Catalog when Modular Mode is turned On.

Mix Styles

If this option is checked, the user may select a different style door for Upper/Wall Cabinets than what is selected for Base Cabinets. If this option is not checked, Upper Cabinets and Base Cabinets must use the same Door Style.

Standard Dimensions

Standard Height and Depth

These settings will control the Height and Depth of assemblies from the selected Catalog if the selected Catalog is Modular (i.e. Modular Mode is On). Information entered in these fields will override the Job information for the selected Catalog only.

Two Pull

The value entered here would be the minimum width that a Drawer Front must reach before a single Pull will be automatically changed to two Pulls. Set this value to 0 to disable it.
