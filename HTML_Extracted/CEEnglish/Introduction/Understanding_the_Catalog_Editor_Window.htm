Understanding the Catalog Editor Window

When you start up the Catalog Editor, the main window opens. This window is your Catalog work space. It contains elements that you use to create your Catalogs and to receive information about them. The following illustration shows the main parts of the Catalog window.

File Menu

The File Menu is located in the upper-left corner of the Catalog Editor. See the File Menu topic for more information.

Quick Access Toolbar

The Quick Access Toolbar is a toolbar containing a set of commands that are independent of the ribbon tab that is currently displayed. There are three quick commands available here Print, Undo, and Redo.

The Quick Access Toolbar can be located in one of two places:

  • Upper-left corner next to the File Menu (default location)

  • Below the ribbonbar

If you don't want the Quick Access Toolbar to be displayed in its current location, you can move it to the other location. If you find that the default location next to the File Menu is too far from your work area to be convenient, you may want to move it closer to your work area. The location below the ribbon encroaches on the work area. Therefore, if you want to maximize the work area, you may want to keep the Quick Access Toolbar in its default location.

  1. Click Customize Quick Access Toolbar .

  2. In the list, click Show Below the Ribbon.

You can also use this option to minimize the ribbon. Or to quickly Minimize/Maximize the ribbon simply double click a ribbon tab.

Titlebar

This section displays the currently active Catalog.

Ribbon

This section gives quick and easy access to tools specific to the area of the Catalog Editor you are in. The ribbon will change depending on which area, controlled by the Area Tabs, you are in. Tools with small  triangles on the bottom have fly-outs containing tools that invoke commands related to the first tool shown. Click and hold down the first button to display the fly-out.

Minimize, Maximize, and Close Buttons

These buttons hide the Catalog Editor window, enlarge it to fill the whole screen, and close it, respectively.

Style Selection

Allows you to choose the Style that the Catalog Editor interface will utilize. Following are the choices.

  • Colorful

  • White

  • Dark Gray

  • Black

Area Tabs

This series of Tabs allows you to switch your View between several possible areas.

You can right click on the Tabs and select Hide Labels if you do not need them.

Sidebar

This section gives access to functions specific to the area of the Catalog Editor you are in. This section will change depending on which area, controlled by the Area Tabs, you are in. See the Sidebar topic for more information.

Catalog Pane

This is the area where you select the Catalog to work with. See the Catalog Pane topic for more information.

Object Pane

This is the actual working area of the Catalog Editor. See the Object Pane topic for more information.

Statusbar

This bar contains important information depending on the area of the Catalog Editor you are in.

Statusbar Menu

The Statusbar menu gives you quick access to control the status of certain Drawing aids. There are six possible options here:

  • Unit Selection or - Allows you to toggle between Inches (Imperial) and Metric Units on the screen. The Unit displayed on the button represents the Unit of Measure that is currently active. You can turn this button On/Off via the Preferences | Units tab.

  • Unit Precision - Allows you to select the Unit of Precision for the selected Unit.

  • Reset Ruler - Allows you to remove any Snap Marks you have placed. See the CABINET VISION Help file for more information.

  • Snap to Marks - This feature allows snapping to a Snap Point set by use of the Ruler Tool. See the CABINET VISION Help file for more information.

  • Snap Grid - This option turns the Grid display on. See the CABINET VISION Help file for more information.

  • Snap to Points - This is perhaps the most useful tool when using the measure option of the Ruler Tool. See the CABINET VISION Help file for more information.

  • Build Warnings - Will display any Build Warnings that are currently active.  See the CABINET VISION Help file for more information.
