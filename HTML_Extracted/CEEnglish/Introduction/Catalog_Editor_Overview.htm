Catalog Editor Overview

The Catalog Editor is an interface that may be used to create and edit Object Catalogs. Before studying the Catalog Editor it is important that you have a thorough understanding of CABINET VISION and that you have already completed the setup of Materials, Construction, and Manufacturing standards.

The full Catalog Editor is an add-on option. If you do not have the add-on, the activities you have the ability to perform in the Catalog Editor are limited.

Origins

The Catalog Editor began as what was called the Library Explorer. The Library Explorer functioned much as Windows Explorer in that the user was able to browse, copy, paste, delete, and rename Objects in a Library (or Catalog). Over time, more functions were added to the Library Explorer and it slowly became more of a Catalog creation and maintenance tool. Therefore, the name was changed to better reflect the functionality of this utility and the Catalog Editor was born.
