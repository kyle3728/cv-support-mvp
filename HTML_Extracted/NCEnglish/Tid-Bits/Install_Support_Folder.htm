Install Support Folder

CPUID.exe

A utility that outputs CPU information, including supported CPU features. A troubleshooting utility to verify the CPU is supported.

ZPLView.exe

A utility for viewing ZPL (.zpl) files generated by the Real-time Label Image feature of the S2M CENTER.

BlumDynaplan

Contains the BXFExt.ini (Stores Part Name Localization entries) referred to in the following S2M CENTER Help File Topic: Ribbonbar>Import DXF>Other CAD Importers>Blum Dynaplan

DependencyWalker

A utility used for troubleshooting certain installation or run-time issues. Visit the Dependency Walker website for more information. http://dependencywalker.com/help/html/contents.htm

HASP

Contains Hasp driver files used during the installation of the ALPHACAM engine.

International

Tools used for performing language and feature Localization in CABINET VISION and the S2M CENTER.

IS Setup

InstallShield documentation for troubleshooting Installation error messages and other installation related issues.

MsiCleanUpUtility

Microsoft Windows Installer Clean-Up Utility used for troubleshooting certain installation and/or uninstallation issues.

MsiZap

A command line utility that removes all Windows Installer information for a product installed on a computer. The folder contains documentation for the utility along with Product Codes for CV, S2M CENTER, and ALPHACAM to remove their Windows Installer information.

Optimizer

Tools used to set low level parameters used by the S2M CENTER Optimizer and Block Nesting engine. See accompanying documentation for further information.

PartListCreator

The setup (installer) for the S2M CENTER Part List Creator Utility as described in the Creating an S2M CENTER Part List topic.

PSendMail

A command line utility for e-mailing a file as an attachment.

Sentinel RMS License Manager 8.5.5

Contains the Sentinel RMS License Manager installation files.

VBA

Contains files used during the installation of the ALPHACAM engine.
