The psnc-ac.mdf Database

This Database stores all Machining Information required when processing a S2M CENTER Job in ALPHACAM.

The psnc-cv.mdf Database

This Database stores all Machining Information required when processing a Job from within the CABINET VISION S2M CENTER.

The psnc-nc.mdf Database

This Database stores all Machining Information required when processing a Job in the stand-alone S2M CENTER.

The +Label Job Files (.lld/.pld)

This Database stores all Machining Information required when processing a Job in +Label.

Orientation

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Indicates the Arrow Position, Arrow Flip and Face Up/Down flag.

 

The Flags are as follows:

0x01  Arrow Left

0x02  Arrow Right

0x04  Arrow Up

0x08  Arrow Down

0x10  Arrow Flip

0x20  Face Up

 

These values are repeated for each of the 4 Edges:

  • The Left Edge Uses the first 8 bits: 0x01  0xff

  • The Right Edge Uses the next 8 bits: 0x0100  0xff00

  • The Top Edge Uses the next 8 bits: 0x010000  0xff0000

  • The Bottom Edge Uses the next 8 bits: 0x01000000  0xff000000

 

So a single field can store the Arrow orientation for all 4 Edges:

0x12011821

0x12000000 = Bottom Edge  Arrow Flip Right

0x00010000 = Top Edge  Arrow Left

0x00001800 = Right Edge  Flip Down

0x00000021 = Left Edge  Face Up  Arrow Left

 

The orientation flag is also used for the Front and Back program. The Bits have the same meaning with the exception that

  • The Front Edge Uses the first 8 bits: 0x01  0xff

  • The Back Edge Uses the next 8 bits: 0x0100  0xff00
