Secondary Machine

Use the drop down selector here to select your Secondary Machine , it must be either a Parts Point to Point or a Parts Router. If you do not have an available machine to select here this means that you have either not created a Parts Machine or you have not selected an NC Link that is valid for use as a Parts Machine. See the Topic on the Machine Catalog for how to create a Parts Machine.

There is another option in the Secondary Machine selector.

Pre-Band - This instructs the S2M Center whether you band your cut to size parts prior to placing them on your secondary machine. If you do this set Pre-Band to Yes, this will take into account edge banding thickness in Secondary machining operations, otherwise, leave it at the default of No.
