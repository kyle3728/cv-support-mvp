Optimize

The Optimize button, once depressed, will either begin the Optimization or Nesting process, depending on whether the selected Primary machine is a Panel Saw or Nested Based Router. Once selected the Optimizing Screen will appear.

You can override the Run Number here if necessary.  The run number displayed is based on the RunCounter field in the DBINFO table of the PSNC.mdb. This field is incremented each time a user optimizes or Exports CNC (unless there is a preset run number). This number is then used to determine the actual run number for the Job. The RunCounter is divided by the user defined Run Number Range and its remainder is added to the minimum range.

For example:

If the RunCounter is 723. That number is divided by 100 (600-500=100) and you get a remainder of 23.

That remainder is added to the minimum range (500) and the result is 523.

The Run Number is displayed at each of the Nest, Optimize and P2P dialogs. You can override the current run number by typing in a new value (disabled for a preset number).

If you change the run number to 100 for example. The next time a/the job is optimized the Run Number will default back to your normal range and the next increment based on the calculations above.

  1. Click on the Run button to begin the Optimization process.

  1. Once the Optimization is completed this screen will display each Material optimized with the total number of sheets, the Used percentage, the Offcut percentage, and the Waste percentage. Also displayed are any Tool selection Errors or Warnings.

Warnings in this area reflect the Warnings you have selected to display in the Preferences area. Warnings do not necessarily reflect a major problem. Typically they point out a Tool that was selected for an Operation which may not be ideal.

Errors in this area typically reflect a major problem. Usually they point out an Operation that does not have a Tool available. You will probably need to create a new Tool to handle this Operation.

You can at this point Print the completed section by clicking the Print button or by clicking the OK button you can accept the results and move on to the Patterns Area of the S2M CENTER.
