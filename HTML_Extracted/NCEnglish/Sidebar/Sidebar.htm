At the Pattern/Part View

The sidebar will have two different appearances depending on whether you are Optimizing or Nesting.

Optimizing

There are four selectable areas in the Optimizing sidebar:

  • Cut Patterns - This area displays all optimized sheets. To view the Patterns simply click on the stack image in this area. If you have multiple Primary Machines (Machine Sets) you are outputting to you can use the Machine Selector to view Patterns for each Primary Machine here.

  • Reports - Allows you to view and print the S2M CENTER Reports.

  • NC Code & Labels - Allows you to output NC Code & Labels.

  • Cost and Time Report - Clicking this button will print out a Cost and Time Report. The costs and times shown in this report are based on the numbers provided in the Machine Catalog for your saw. This button is only available if you are Optimizing using a Saw.

Nesting

Following are the selectable areas in the Nesting sidebar:

  • Machine Set Selection - If you are running multiple Machine Sets there will be a combobox above the Cut Patterns which will allow you to select which Machine Set results you want to see in this view.

  • Cut Patterns - This area displays all nested sheets. To view the Patterns simply click on the stack image in this area. If you have multiple Primary Machines (Machine Sets) you are outputting to you can use the Machine Selector to view Patterns for each Primary Machine here.

  • Reports - Allows you to view and print the S2M CENTER Reports.

  • NC Code & Labels - Allows you to output NC Code & Labels.

  • Show Tool Path - Clicking this button will display the Tool Path in the Sheet Display Area.

There are three colors used by Show Tool Path.

      • Red = Default (Centerline)

      • Cyan = Left Compensation

      • Magenta = Right Compensation

  • Machining Simulation - Clicking this button will take the currently selected Pattern to the Machine Simulator.

Right Click Options

 

In the Cut Patterns are there are five possible right click options:

  •  Optimize - Clicking the Optimize command will allow you to re-optimize the selected material. Used in conjunction with the Properties command available here you can improve your optimization on a per material basis.

  •  Print - Clicking the Print command will Print the selected pattern.

  •  Properties - Clicking the Properties command will bring up the optimization/nesting properties for the selected material. This works like an override. The Properties you change here will be for the selected Material only. This will allow you to re-optimize with different results and not adversely affect the other materials for this PNC file.

  •  Add Blank Sheet - Clicking the Add Blank Sheet option will add a Blank Sheet for the selected Material.

  •  Remove Blanks Sheets - This option will only be active if there are Blank Sheets for the selected Material and will allow you to remove them.
