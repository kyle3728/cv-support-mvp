Output CNC Operations

If you are in the S2M CENTER and see this button this means you either have not selected a Primary Machine and only have a Secondary Machine, such as a Point-to-Point, selected or you have not purchased either the Nesting or Optimization packages. Clicking the Output CNC Operations button will generate the type of output your Router and/or Point to Point has setup, via NC Link, in the Machine Catalog.

Clicking Print on this screen will print out the list of parts exactly as shown on this screen.

Click OK to enter the Patterns Section.
