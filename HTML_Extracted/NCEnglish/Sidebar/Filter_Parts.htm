Filter Parts

The Filter Parts option is a very powerful feature that allows you to include or exclude parts from your machines based on specific criteria.

There are four Filter Types you can choose in the Part Filter:

  1. New Filter - This option will not take into account any currently selected Parts or Previously Filtered Parts.

  2. Add to Selected Items - This option will build a Filter that will not de-select any Parts manually selected from the Parts View.

    • For Example: If you enter the Filter Parts screen and select Room 1 and exit the Filter and upon entering the Parts View realize you also want Cabinet #2 in Room 2. You can reenter the Parts Filter select Add to Selected Items and then Select Cabinet #2 in Room 2 and not loose your previous selection.

  3. Filter on Previous Selection - This option will allow you to continue building a Filter based on previously defined criteria.

    • For Example: Let’s say I have a Job with 100 Cabinets in it, and I need to cut only the ¼ Material for Cabinet 1, 8, and 47. So I start a New Filter and select 1, 8, and 47. Then switch to “Filter On Previous Selection” then select my ¼ Material. The only items selected when I return to the Parts List would be the ¼ Material for Cabinets 1, 8, and 47.

  4. Remove Selected Items - This option will build a Filter that will de-select any Parts manually selected from the Parts View.

There are five Tabs in the Part Filter:

Assemblies

The Assemblies Tab allows you to include or exclude Jobs, Rooms, and/or Cabinets from your Machine processing. You can include or exclude Cabinets individually or you can include or exclude Cabinets by Room or by Job.

A check in the box instructs the S2M CENTER to include items in the NC Output. Removing the check mark from the item instructs the S2M CENTER to exclude the item from NC Output.

Simply click the Invert Selection button to quickly reverse your selection.

Materials

The Materials Tab allows you to include or exclude any or all Materials from Machining.

A check in the box instructs the S2M CENTER to include items in the NC Output. Removing the check mark from the item instructs the S2M CENTER to exclude the item from NC Output.

Simply click the Invert Selection button to quickly reverse your selection.

Parts

The Parts Tab allows you to include or exclude any or all Parts from Machining.

A check in the box instructs the S2M CENTER to include items in the NC Output. Removing the check mark from the item instructs the S2M CENTER to exclude the item from NC Output.

Simply click the Invert Selection button to quickly reverse your selection.

Other

The Other Tab is divided into three Categories.

A check in the box instructs the S2M CENTER to include items in the NC Output. Removing the check mark from the item instructs the S2M CENTER to exclude the item from NC Output.

Operations

      1. Part Shape - There are two options for Part Shape:

        • Shaped - This Filter will, when selected, only select Shaped parts for your processing.

        • Rectangular - This Filter will, when selected, only select Rectangular parts for your processing.

      2. Part Face - There are three options for Part Face:

        • None - This Filter will, when selected, only select Parts which have no Machining on either Face.

        • Single - This Filter will, when selected, only select Parts which have Machining on only one Face.

        • Double - This Filter will, when selected, only select Parts which have Machining on both Faces.

      3. Horizontal Boring - There are three options for Horizontal Boring:

        • With Horizontal Boring - This Filter will, when selected, only select parts with Horizontal Boring.

        • Without Horizontal Boring - This Filter will, when selected, only select parts without Horizontal Boring.

        • Only Horizontal Boring - This Filter will, when selected, only select parts that only have Horizontal Boring.

      4. Vertical Boring - There are two options for Vertical Boring:

        • With Vertical Boring - This Filter will, when selected, only select parts with Vertical Boring.

        • Without Vertical Boring - This Filter will, when selected, only select parts without Vertical Boring.

        • Only Vertical Boring - This Filter will, when selected, only select parts that only have Vertical Boring.

      5. Route Operations - There are three options for Route Operations:

        • With Route Ops - This Filter will, when selected, only select parts with Route Operations.

        • Without Route Ops -This Filter will, when selected, only select parts without Route Operations.

        • Only Route Ops - This Filter will, when selected, only select parts that only have Route Operations.

      6. Drill/Dowel Operations - There are three options for Drill/Dowel Operations:

        • With Drill/Dowel Ops - This Filter will, when selected, only select parts with Drill & Dowel Operations.

        • Without Drill/Dowel Ops - This Filter will, when selected, only select parts without Drill & Dowel Operations.

        • Only Drill/Dowel Ops - This Filter will, when selected, only select parts that only have Drill & Dowel Operations.

      7. Cut To Size Operations - There are three options for Cut To Size Operations:

        • With Cut Ops - This Filter will, when selected, only select parts with Cut To Size Operations.

        • Without Cut Ops - This Filter will, when selected, only select parts without Cut To Size Operations.

        • Only Cut Ops - This Filter will, when selected, only select parts that only have Cut To Size Operations.

      8. Other Machine Operations - There are three options for Other Machine Operations:

        • With Other Ops - This Filter will, when selected, only select parts with Other Machine Operations.

        • Without Other Ops - This Filter will, when selected, only select parts without Other Machine Operations.

        • Only Other Ops - This Filter will, when selected, only select parts that only have Other Machine Operations.

Machines

      1. Optimize - There are two options for Optimize:

        • Tagged for Optimization - This Filter will, when selected, only select parts tagged for Optimization for your processing.

        • Not Tagged for Optimization - This Filter will, when selected, only select parts not tagged for Optimization for your processing.

      2. CNC - There are two options for CNC:

        • Tagged for CNC - This Filter will, when selected, only select parts tagged for CNC for your processing.

        • Not Tagged for CNC - This Filter will, when selected, only select parts not tagged for CNC for your processing.

Output

      1. Primary Machine - There are two options for Primary Machine

        • Output to Machine - This Filter will, when selected, only select parts that have already been output to the machine for your processing.

        • Not Output to Machine - This Filter will, when selected, only select parts that have yet to be output to the machine for your processing.

      2. Secondary Machine - There are two options for Secondary Machine

        • Output to Machine - This Filter will, when selected, only select parts that have already been output to the machine for your processing.

        • Not Output to Machine - This Filter will, when selected, only select parts that have yet to be output to the machine for your processing.

      3. Drill/Dowel Machine - There are two options for Drill/Dowel Machine

        • Output to Machine - This Filter will, when selected, only select parts that have already been output to the machine for your processing.

        • Not Output to Machine - This Filter will, when selected, only select parts that have yet to be output to the machine for your processing.

      4. Cut To Size Machine - There are two options for Cut To Size Machine

        • Output to Machine - This Filter will, when selected, only select parts that have already been output to the machine for your processing.

        • Not Output to Machine - This Filter will, when selected, only select parts that have yet to be output to the machine for your processing.

Saved

The Saved Tab allows you to Save and manage your Saved Filters.
