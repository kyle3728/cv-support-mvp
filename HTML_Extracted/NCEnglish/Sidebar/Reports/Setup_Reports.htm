Setup Reports

Report Groups may be created to contain Reports that are similar to each other OR Report Groups could be created to contain all Reports needed for a certain type of Job.

Creating a Report Group

When you click Setup Reports, you will see the following screen:

This is a list of the Reports that are available in the S2M CENTER. In the sidebar is a list of all of the Report Groups that have been created. To create a new report group, simply click the Add button and then enter the name that you want to use for this group. When you click OK the new group will be added to the list of Report Groups.

After you have created your new Report Group, you will need to begin assigning Reports to that Group. To add a Report to a Group, click on that Report and then click the check box to add a check next to the desired Report Group. As you scroll down the right side of your screen you will notice that some of the Reports have a green S2M icon in front of them. This green S2M icon lets you know that the Report has been assigned to a Group. If you click on that Report you will see a check mark next to one or more of the Report "Groups" in the sidebar. Reports can be assigned to more than one group if you like.

Report Mode

Before adding a Report to a Report Group, you may wish to preview that Report to determine whether it is the correct format for your needs. To preview a Report, simply click on that Report and then click the View Mode option in the Report Mode drop-down. As a preview of the Report, you will see a the Report with the information from the current Job.

Magnify

Only active in View Mode this option allows you to Zoom In and Out on the Preview.

Edit Report

This ribbon option opens the Report Editor. See the Report Editor Help file for more information. The Report Editor Help file is accessible via the help button at the top right as illustrated below.

Report Properties

This option allows you to edit the Name, Description, and source data for the selected Report. You can also determine if the Report needs a Part Graphic (including Type and Size), the Banding Symbol Size, and whether to display Part Sizes and Part Names (For Pattern Printouts).

Copy Report

Will Copy the selected Report giving you the ability to give it a new Title and Description.

Delete Report

Will Delete the selected Report. Only available for User Reports.

Part labels

All existing part label reports have indicators on each part label which indicate if a part will run in a "normal" or "mirror" part machining "zone" (M) and, also, if the part has two part CNC programs (2nd):

How to "read" the M and 2nd indicators:

Part on label has one part CNC program (that which is referenced on the part label) that will run in a "normal" part machining "zone".

Part on label has one part CNC program (that which is referenced on the part label) that will run in a "mirror" part machining "zone".

Part on label has two part CNC programs. The part CNC program which is referenced on the part label itself will run in a "normal" part machining "zone." The second part CNC program (referenced implicitly) will run in a "mirror" part machining "zone" (opposite of that indicated for part CNC program referenced on the part label).

Part on label has two part CNC programs. The part CNC program which is referenced on the part label itself will run in a "mirror" part machining "zone." The second part CNC program (referenced implicitly) will run in a "normal" part machining "zone" (opposite of that indicated for part CNC program referenced on the part label).

In addition to this the name of the part program itself also indicates which Face the Program is output to machine.

F - Front CNC program for operations machined from the part face (finished face of material) (example: r01f0001)

B - Back CNC program for operations machined from the part back (unfinished face of material) (example: r01b0001)

Program lists:

All program list reports place an indicator (M) beside each part CNC program to be run in a "mirror" part machining "zone." Those part CNC programs lacking the (M) indicator are to be run in a "normal" part machining zone:
