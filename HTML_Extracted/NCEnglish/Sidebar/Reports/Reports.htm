Reports

When your CNC Output is complete, the Reports area allows you to print information about your Output.

The Reports area is also used to select, and organize your Reports into "Report Groups" that you create and maintain.  A Report Group is a collection of Reports organized under one heading or name. Using Report Groups makes it easy to print multiple Reports; you just select a "Report Group" to print, and then click on the "Print Reports In Group" button and all the Reports contained in that "Report Group" will print.

While it is true that you can start printing information about your Output as soon as your output is complete, as the S2M Center comes preset up with sample Report Groups. It is also true that those Report Groups might not contain the Reports you would like them too. Therefore you will have some preliminary setup work to do to customize the Reports area to your way of doing things. See the Setup Reports section later in this topic.

Once you have the Reports area setup to your satisfaction printing is just a matter of clicking on the "Reports" button then selecting the "Report Group" to print and then click on the "Print Reports In Group" button.

Export Report

The Export option allows you to Export the currently active Report to another Application or File for further modification. Finished reports can be exported to a number of popular formats, such as spreadsheet, word processor, HTML, XML, and other common data interchange formats. This makes the distribution of information easier. For example, you may want to use the report data to project trends in a spreadsheet package or to enhance the presentation of data in a desktop publishing package.

The exporting process requires you to specify a format via the Direct to... option.

Export Direct To... Types

The export formats supported by the Reports area can be broadly categorized in two groups: page-based formats and record-based formats. Page-based formats tend to produce a more exact output. The emphasis of these formats is layout representation and formatting. Formatting refers to attributes such as font style, text color, text alignment, background color, and so on. Layout refers to object position, object size, and the relationship between these attributes and other objects. Depending on the format of the Report, it may not be possible for the program to preserve all layout and formatting perfectly, but page-based formats, in general, preserve these properties as closely as possible.

With record-based formats, the emphasis is on data rather than the layout and formatting. However, in some formats—such as Microsoft Excel - Data only—you will notice that some formatting is exported.

Please consider:

    • Due to the format, the layout cannot always be taken over 1:1 because there are specific restrictions when converting to these formats.

    • The Microsoft® Word export format creates DOCX files that are compatible with Microsoft® Office 2007 and higher. Tables will be exported on continuous pages to support later editing.

Many formats have various possibilities for configuration. You can make use of these by clicking the "Options" button.

    • With Excel formats, you can reduce the number of columns and rows to the minimum required by means of the "Only data from table object(s)" option. Also all pages can be exported into one sheet and the output formats 'xls' as well as 'xlsx' are available.

    • The picture export to JPEG, TIFF and PNG supports optional automatic cropping of the results to the content. An exported barcode is therefore only exactly as large as required to display the complete content.

Following are the supported Outputs:

    • Adobe PDF Format

    • XHTML/CSS

    • Multi-Mime HTML Format

    • Microsoft Excel (XLSX)

    • Microsoft Word Format (DOCX)

    • Rich Text Format (RTF)

    • Microsoft XPS Format

    • Multi-Page Tiff Picture

    • TIFF Picture

    • PNG Picture

    • JPEG Picture

    • Bitmap Picture

    • Metafile Picture (EMF)

    • File

    • HTML Format

    • HTML jQuery Mobile Format

    • Pinwriter (TTY)

    • Text (CSV) Format

    • Text (Layout) Format

    • XML Format

Sidebar

Report Group

The Report Group area in the Setup Reports sidebar allows you to view the currently available Report Groups.

Add a Report Group

The Add button in the Setup Reports sidebar allows you to create a new Report Group.

Delete a Report Group

The Delete button in the Setup Reports sidebar allows you to delete the selected Report Group.

You should remove all Reports from a Group before you delete the Group. This means you will have to click on every Report in the Report List that has or icons, and check to see if it is a member of the group you are preparing to delete.

Print Reports In Group

This button will print all of the Reports in the selected Report Group.
