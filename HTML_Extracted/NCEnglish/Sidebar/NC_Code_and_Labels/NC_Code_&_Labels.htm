NC Code & Labels

Clicking NC Code & Labels brings up the Output Data window. Simply select which data you would like to output. The Output Tasks area will be categorized by each active Machine Set you are outputting to. Each Category has three main areas:

  1. NC Machine Data (Machine Set Name) - The Create Output option can be turned off to not Output Codes for the Machines in this set. All the Machines within the Machine Set along with their Output Directories will be listed here. This includes Primary Machine (Saw/Router), Secondary Machine (P2P/Router), Drill & Dowel, Chop Saw, and/or Other machines.

  2. +Label - Checking this option, typing a +Label Job name, and clicking run will output data for your +Label Station.

  3. Label Images - Checking this option, will output Image files for your Saws or Routers labeling system to utilize.

  4. CV Paperless Traveller - There are three options for the CV Paperless Traveller category:

    • CV PT Upload Work Order - Will create the CV Paperless Traveller data and send it to the cloud.

    • CV PT Work Order Name - Enter the name for the Work Order here. THis is the name shop personnel see when using the CV Paperless Traveller app.

    • CV PT Work Order Path - Local location for the CV Paperless Traveller data.

    •  CV Paperless Traveller has the capability to "Scan Parts" this capability expects a QR Code containing the "work order" and Part ID (WorkOrder_PartID) with an underscore delimiter. You can define a QR Code in +Label utilizing {WorkOrder}_{Part.ID} or in a custom Report/Label utilizing {PSNC-CV.mdf | RunInfo | rGUID}_{PSNC-CV.mdf | Parts | ID}.

  5. Save Job State - There are three options for the Save Job State category:

    • Create Output File - Will create an SNC file of the current Job at it's "Ready for Output" state.

    • SNC File Name - Specifies the file name for your Job State file.

    • SNC Output Path - Specifies the folder to Save the Job State file to.

See the File Menu topic for information on Opening SNC files.
