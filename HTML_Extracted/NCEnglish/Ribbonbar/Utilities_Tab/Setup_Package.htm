Setup Package

The Setup Package utility is a means to transfer specific objects from one computer running S2M CENTER to another.

Exporting a Setup Package

The Setup Package feature allows you to create a single file that can store and transfer multiple custom items for import into another S2M CENTER user’s or S2M CENTER Tech’s system.

The following list of items can be added to a package for transfer:

Labels (Label-IT Labels or Saw Graphic designs only)

Machines

Tools

  • If you select a Tool Set it will automatically import all Tools associated with that Tool Set.

Materials

Library Parts

This topic explains how to export a package and send it to Tech Support or another S2M CENTER user.

  1. Go to the Utilities Tab | Setup Package

  1. Select the item or items you would like to include in the Package from the list on the left side of the screen. After highlighting an item you want to add to the package simply click the "Add Items To Package" button in the lower left corner. There is no limit to the number of items that can be added to a package.

  1. After selecting the items to add to the package click on the "Save Current Package" tool in the toolbar.

In the graphic below we selected two Tool Sets and the Packager automatically imported the all the individual Tools that make up each Tool Set.

  1. Select a location to save the Package and type in a Name for the Package then select "Save".

  2. You may now either close the Setup Package screen or, if you use MS Outlook or Outlook Express, you can click on the "E-Mail Current Package" tool in the top toolbar.  This will attach the Package to your e-mail and allow you to e-mail the SetupPackage to Tech Support or to another S2M CENTER User.

  3. You may now close the Setup Package screen.  You have successfully exported a Package from the S2M CENTER.

Importing a Setup Package

This topic explains how to import a package that has been sent to you by Tech Support or another S2M CENTER user.

  1. Go to Utilities Tab | Setup Package

  1. Click the "Open An Existing Package" tool in the toolbar.

  1. Select the package file from the location where it is being stored and click Open.

  1. Click the Import tab.

  1. Click the "Import Package" button.

Prior to importing you can optionally have all the names of the objects imported from the Package by setting a Prefix using the Add Preix button on the Import tab.

Overwriting Existing Items

Following are the options that allow you to choose how to Overwrite items that already existed in your system.

  • All Matches - Overwrite - Will overwrite all existing items in your database with the contents of the Package.
  • All Matches - Use Existing - Will use all the existing items in your database rather than overwrite them with the Package contents.
  • Chil Matches - Overwrite - Will overwrite all existing children of items in the Package that match those in your database.
  • Child Matches - Use Existing - Will use all the existing children in your database rather than overwrite them with the Package contents.
  • Toggle Overwrite - Allows you to Overwrite existing items by selecting them individually. This will cause items in the Package to either overwrite the existing item or to use the existing item.

  1. Click on OK for the Import Complete notice.

  1. You may now close the Package screen you have successfully imported a Package into the S2M CENTER.
