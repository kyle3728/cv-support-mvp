Material Catalog

When first entering the Material Catalog in the S2M CENTER you will see a screen similar to the one below with no Materials.

If you are using S2M CENTER stand-alone and do not run either CABINET VISION or Cabnetware simply click the New Material tool in the toolbar to start creating your Materials.

If you are using CABINET VISION, the first thing you will want to do is Import your Materials. To do this click the Import Materials tool in the toolbar. Upon entering the Import Materials tool you will be presented with the following screen.

Place a check mark to each Material you would like to Import in the list.

Simply click Import once you have checked off those Materials you would like to Import. You should notice that once you click Import each Material you selected is removed from the list. This will keep you from double Importing Materials.

Simply click Close to return to the Material Catalog. You will notice it is now populated with your Materials.
