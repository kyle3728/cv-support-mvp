Designing +Label Labels

Accessing the Label Designer

To access the Label Designer follow these steps:

  1. Click on Utilities Tab | +Label Setup.

  1. In the Label Administrator click on the Design Tab.

 

Make sure that once you have created your Label Design that you go into the Machine Catalog Setup Tab for your Sheet Router/Saw and select this Label as the active Label.

Label Designer Feature List

The features of the Label Designer are denoted with the red numbers in the graphic below, followed by a brief description of each.

  1. Name of Label currently active in the Label Designer.

  2. Use this button to create a new Label.

  3. Use this button to copy the active Label.

  4. Use this button to delete the active Label. You will be prompted whether or not to delete the Label.

  5. Use this field to define the Width of the active Label.

  6. Use this field to define the Height of the active Label.

  7. This list shows all the fields being utilized on the active Label.

  8. Use this button to add a new field to the active Label.  See the complete list of the fields available with a brief description of the fields content later in this topic.

  9. Use this button to delete a selected field on the active Label.

  10. This is where you can enter sample text to see what the Label will look like with actual data in place.  This is also the area where you will enter permanent text for the text field.

  11. This is where you enter the X (Left to Right) location of a selected field on the Label. This value references the top left corner of the field to the top left corner of the Label.

  12. This is where you enter the Y (Up and Down) location of a selected field on the Label. This value references the top left corner of the field to the top left corner of the Label.

  13. This is where you specify the width of a selected field.

  14. This is where you specify the height of a selected field.

  15. This is where you set the Rotation of the selected field. When selecting a Part Graphic this option changes to Band Symbol Size and can be used to make the Banding Symbol included in a Part Graphic larger or smaller.

  16. This is the area where you format a selected field’s text formats. You can define the following text attributes:

    1. Font style (Typically the 3 of 9 Bar-code font is used for bar codes)

    2. Bold

    3. Italic

    4. Justification (Left/Center/Right)

  17. This is the workspace for the active Label.

The Unit of Measure buttons  allow you to establish the Unit of Measure and Precision used on +Label Output.

Field Modification Options

The graphic below shows a highlighted field.  There are different methods to move and size a field.

  1. You can use the fields X and Y to move the field.

  2. You can use the fields Width and Height to stretch or shrink the field.

  3. Alternatively you may grab the center handle (Yellow Dot) to move the field or the (4) corner handles (Yellow Dots) to stretch or shrink the field.

Changes in Version 6

In Version 6 the Fields utilizing Banding Letter were removed to be replaced with Banding: List. The existing fields on your Labels were converted to new filter fields.

Banding: B Material is now {M}[Filter:B]

The new variables are:

{I} Or {Indicator}
    {Tag}
{C} Or {Color}
{M} Or {Material}
{T} Or {Thick}

Now you can format the banding field anyway you like using those variables:

I.e.:  {T} – {M}: in the color of {C}

In addition you can add either a complex filter    [Filter:E|I|D]     or a Simple Filter [Filter:B]

This allows for superior flexibility and optimizing label real estate.

Specifying Edge Banding on your Label

Banding: Code

The following characters can and may be displayed in the Banding: Code:

L

Length

W

Width

E

Exterior Banding

I

Interior Banding

D

Door Banding

A

Arc/Radius

Here are some examples of simple codes:

1LE - Band 1 Length with Exterior Band

2LE1WI - Band 2 Lengths with Exterior Band and 1 Width with Interior Band

For shaped parts when there is more than 1 side, you will see a dimension represented to indicate which edge gets banding applied.  

Imagine a 34” end panel with a 4” toe notch:

1LE301LI - Band the 30" Length with Exterior and the Full Length with Interior

A part with a radius may look something like this:

1WE241A6E - Band the 24" Width and the 6" Arc/Radius with Exterior band

Banding: Flag

Another alternative to the Banding: Code is the Banding: Flag.  The banding flag begins with the number of sides for a part followed by what each side will be banded with.

N

No Banding

E

Exterior Banding

I

Interior Banding

D

Door Banding

B-Z

Misc (B-Z) Banding

6NNENNN - In this sample part, starting from the reference point and going clockwise you get None, None, Exterior, None, None, None.  

This type of coding can be quite confusing on complicated parts.  The code and/or flag for the part below (24” x 36”) with an edited shape would be:

CODE: 1W19E1W5E1L27E1A5E1W16E1L4E1W4E

FLAG: 8EEEEEEEE

To simplify the edge banding instructions to machine operators, you can use the Part: Graphic field on your Label.  Each edge of the part will be marked with either an E (Exterior), I (Interior), or D (Door), as demonstrated in the graphic below.  With this method you would not need to display either the Banding: Code or the Banding: Flag fields.

 

Primary & Secondary Programs

There is a difference in the definition of Primary and Secondary programs depending on the types of machinery you utilize.

Cell Based Manufacturing (Saw & Point-to-Point)

In the Cell Based Manufacturing model you cut parts to size on a Saw and then machine on your Point-to-Point.  In this situation the Primary program is run first on the Back (Interior) side of the part.  After this the Secondary program is run on the Face (Exterior) side of the part.  This is fairly straightforward.

In Cell Based manufacturing you would need to specify both a Primary program/bar-code as well as a Secondary program/bar-code to include all machining necessary on your Label.

Nested Based Manufacturing

In the Nested Based Manufacturing model you typically utilize one machine to do all of your face machining operations.  When nesting in the S2M CENTER the program number that is generated is for a whole sheet of parts (Sheet program) therefore no individual program number is generated for the Back (Interior) sides of the part.  In this case the Primary program is for the Face (Exterior) side of the part.  In this method of production there would be no Secondary program since both faces of the parts are being covered by the Sheet program and then by the Primary program.

In Nested Based manufacturing you would only need to specify a Primary program/bar-code to include all machining necessary on your Label.
