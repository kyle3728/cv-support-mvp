Using the +Label App in Production

Selecting a Job

Selecting a Job in +Label app is fairly straightforward:

  1. Click on the File tab and click Open.

  1. Browse the Job you want to open, select it then click Open.

You can open a specific +Label job via command line as follows. c:\Program Files\Hexagon\CABINET VISION\S2M 2023\pnclabelit.exe "C:\Users\me\Desktop\Banding.tld"

+Label App Feature List

 

  1. Name of the Job currently open in the +Label app.

  2. Clicking the View option allows you to View the Label for the currently selected Part.

  3. Laser - Turns on/off the Laser Pattern Generator. See the Running A Laser topic for more information.

  4. Mark Job Complete - This option does two things:

    1. When selected this option will remove the Job from being visible at the Label-IT station.

    2. This option at the same time will mark the Label-IT job Done on the S2M CENTER station in the front office.

  5. 1 Label - Prints the selected label.

  6. On Part Select - Prints the selected label.

  7. Sheet - Will print Labels for all Parts on the active Sheet.

  8. Sheet of Labels - Will print sheets of labels for the entire job.

  9. Patterns included in the selected Job.

  10. Current selected part.

    1. Selected parts that have not had labels printed for them are highlighted in green.

    2. Parts that are not selected and that have not had labels printed for them are white.

    3. Selected parts that have had labels printed for them are highlighted red.

    4. Parts that are not selected that have had labels printed for them are highlighted in blue.

    5. This field also has a drop down arrow which will show you the Parts below the current Part if you are Booking (Stack Cuts) using a Panel Saw.

  11. The Composites button allows you to view either the sub-part or the composite part.  The icon will indicate the current view. Any time a Sub-Part label is printed the composite part label will be printed as well. The button will be disabled if the selected Part is not part of a Composite.

  12. Selected Patterns material. This text will also indicate how to "load" one sided Materials on the machine like the Pattern Print outs from the Pattern area.

  13. Selected Sheet’s location and quantity.

  14. Clicking the |< button moves you to the first Sheet/Label in the selected Pattern.

  15. Clicking the >| button moves you to the last Sheet/Label in the selected Pattern.

  16. Clicking the < button moves you to the previous Sheet/Label in the selected Pattern. You can also simply swipe right for the previous Part/Label if you have touch screen.

  17. Clicking the > button moves you to the next Sheet/Label in the selected Pattern.  You can also simply swipe left for the next Part/Label if you have touch screen.

  18. Clicking the Print option prints the Label or Sheet, depending on your Preference, for the currently selected part.

  19. Sheet program bar-code.

  20. Sheet program name.

    1. This name matches the name of the program generated by Intelli-CAM for the Router.

Preferences

Following are the options in Preferences:

  • Print Options - Allows you to choose how you want to Print. You have the following options:

    • Print 1 Label - If you have a Part selected clicking the Print option will print its Label.

    • Print On Part Select - Simply selecting a Part will print its Label.

    • Print Sheet - Will print Labels for all Parts on the active Sheet when clicking the Print option.

    • Print Sheets of Labels - Will print sheets of labels for the entire job.

  • Rotate View - Allows you to Rotate the Pattern view to match the Orientation of your Machine.

  • Shift Origin - Allows you to shift the Origin of your Parts on the Nest. Utilizing Rotate View and Shift Origin in conjunction will allow you to exactly match the Patterns output from the S2M CENTER.

  • Display options - Allows you to choose extra items you may want displayed on-screen. You have the following options:

    • Show Offcuts -  A toggle to Hide or Show Offcuts.

    • Display Part Size on Part

    • Show Label Position - This option will put the Label position symbol as defined in S2M CENTER Preferences on the Pattern View as well as the Part Graphic on the Label; if your Label is designed to display the Part Graphic.

    • Display Part Number on Part

    • Show Part As Cut

Label Placement

Placing your Labels oriented properly on the part is extremely important in your success with Label-IT.  There is a built in feature to help insure that you orient the Label properly onto your part.  The graphic below shows a part with a small dot.  This dot represents how the Label needs to be oriented on the part.  If the Label is placed properly on the part then the machine operators can look at simple graphics on the Label that will instruct them on how to orient the part on the machine as well as which edges to edge band and which edges receive boring on your Drill & Dowel machine. The small line, within the circle, points to the top of the Label.

 

The Label Placement denotation is only displayed on the Label-IT stations screen. Proper Label Placement is important on Parts that receive Secondary operations so the Parts can be oriented correctly on the Secondary Machine. This means it is important that the person responsible for Label Placement has the Label-IT station in his work area.
