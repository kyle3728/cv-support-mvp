Generating +Label Output

+Label output is all generated on the S2M CENTER station.  The process is outlined below:

  1. Click on Utilities Tab | +Label Setup.  

  2. The first tab upon entering +Label Setup is Jobs.  This screen shows currently active and inactive +Label jobs. You can remove any of these jobs from the system by simply clicking the check box beside the job and clicking the "Remove Selected Jobs" button.

You can also select the Job Type from the Jobs Tab. The first option is "Linked (Small)" this +Label Job type requires the psnc.mdb database file be located in the same folder that you are outputting the +Label Job to. The second option "Self-Contained (Large)" creates a stand-alone +Label file that can be used on any +Label station and does not require a machine database for support. Although a bit larger it is the most stable +Label job file as it contains your Label designs and requires no linking back to the S2M CENTER.

An additional capability is the ability to control the Unit of Measure for the +Label. Click the Unit of Measure buttons at the lower left to control this.

  1. Click OK to exit +Label Setup.

  1. Optimize your jobs as you normally would.

  2. Once on the Pattern screen, output your NC instructions as normal.

  3. Enable the "Output to +Label" option. Type in the name of the +Label job. You may want to give the job a name that will tie back to your job name or number. This is the name the +Label operator will select to start processing the job.

  

  1. Click on Run.

You have now successfully output to the +Label Station. See the Topic on "Using +Label in Production" to learn how to access and use this output.
