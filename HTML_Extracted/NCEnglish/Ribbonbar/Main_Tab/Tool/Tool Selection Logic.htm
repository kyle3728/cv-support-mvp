Enhanced Tool Selection Logic for CNC Output

The following represents the automatic tool selection logic, which, the S2M CENTER employs when parts are output to CNC and is organized by operation type. This, in part, is made possible by the many properties that can be assigned to a particular tool in the Tool Catalog.  If you are unfamiliar with the properties that can be assigned to particular tools in the Tool Catalog, it may be best to review the information presented in the Tool Catalog Setup topic in this help file before reviewing the information below.

Part Outline Operation - See Chart #6 at the end of this topic

Search tools for flat routers with Outline = "True"

  1. If found then select tool in this order

    1. Up / Down Shear bit that matches sheet requirements

    2. Compression bit

    3. Down Shear bit

    4. Up Shear bit

    5. "Expansion" bit (Up Shear & Down Shear = "False")

  2. No match  – ERROR

Vertical Hole Operation - See Chart #1 at the end of this topic

Search tools for vertical drill bit with exact size match

  1. If found then select tool in this order

    • Through bore setting matches hole requirements 1st drill with size match

    • Check if Drill Bit is in Gang Drill Head.

      • If found - Check if hold position is in gang drill head limits

    • If no match – WARNING – Search flat routers for size match

      • If found then select tool in this order to create hole in one drop

        1. Down Shear bit

        2. Compression bit

        3. Up Shear bit

        4.  "Expansion" bit (Up Shear & Down Shear = "False")

    • If no exact size match then search for flat router with diameter smaller than hole size

      • If found then select tool in this order to create pocket to match hole size

        1. Down Shear bit

        2. Compression bit

        3. Up Shear bit

        4.  "Expansion" bit (Up Shear & Down Shear = "False")

  1. No match – ERROR

Horizontal Hole Operation - See Chart #2 at the end of this topic

Search horizontal drill bits for a size and direction match

  1. If found then select that horizontal drill

  2. No match - ERROR

Dado Operation - See Chart #3 and Chart #4 at the end of this topic

  1. If dado extends up to or past the edges of the part – Use saw

  2. Search saws for size match

    • If direction is OK then use tool

    •  If direction is opposite then swap dado direction and use tool

    • If direction is not usable then search for saw with smaller kerf

  1. If no size match then use saw with smaller kerf

    • If direction is OK then use tool and create multiple passes

    • If direction is opposite then swap dado direction and use tool with multiple passes

    • If direction is not usable – WARNING – Search for router

  1. If no saw match Use router

    • Search flat routers for size match and Up Shear / Down Shear match (Will Select Down Shear for a Dado)

  1.  If found then create single path using router

    • If no size match then search for flat router with smaller diameter than dado width

    • If found and tool diameter is greater than or equal to ½ dado width then create 4-pass route using tool selected in this order

      1. Up / Down shear bit that matches sheet requirements

      2. Down Shear bit

      3. Compression bit

      4. "Expansion" bit (Up Shear & Down Shear = "False")

      5. Up Shear bit

    • If found and tool diameter < ½ dado width then create multi-pass route (starting from the outside and then moving inward concentrically) using tool in this order

      1. Up / Down shear bit that matches sheet requirements

      2. Down Shear bit

      3. Compression bit

      4. "Expansion" bit (Up Shear & Down Shear = "False")

      5. Up Shear bit

  1. No match – ERROR

Cutout / Pocket Route Operation - See Chart #5 at the end of this topic

  1. Search flat routers for size match and Up Shear / Down Shear match

    • If found then create single path using tool

    •  If no size match then search for smaller diameter flat router with Up Shear / Down Shear match

      1. If found and tool diameter is greater than or equal to ½ dado width then create 4-pass route

      2. If found and tool diameter < ½ dado width then create multi-pass route

        • If no match found then search for any flat router with smaller diameter then the route operation

        • If found and tool diameter is greater than or equal to ½ dado width then select tool in this order to create 4-pass route

          • Cutout route operation

            1. Compression bit

            2. Down Shear bit

            3. Up Shear bit

            4. "Expansion" bit (Up Shear & Down Shear = "False")

          • Pocket route operation

            1. Down Shear bit

            2. Compression bit

            3. "Expansion" bit (Up Shear & Down Shear = "False")

            4. Up Shear bit

        • If found and tool diameter < ½ dado width then select tool in this order to create multi-pass route

          • Cutout route operation

            1. Compression bit

            2. Down Shear bit

            3. Up Shear bit

            4. "Expansion" bit (Up Shear & Down Shear = "False")

          • Pocket route operation

            1. Down Shear bit

            2. Compression bit

            3. "Expansion" bit (Up Shear & Down Shear = "False")

            4. Up Shear bit

  1. No match – ERROR
