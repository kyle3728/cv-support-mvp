Overview of the Tool Catalog

The Tool Catalog is used in creating and defining tools for use in your CNC operations. The Tool Catalog also allows an unlimited quantity of tools, of various types, in any combination, to be assigned to a user-defined "Tool Set".

The following is a brief introduction to the Tool Catalog's features:

Toolbar

 New

Allows you to create a New Tool.

There are 11 possible Tool Types you can create in the Tool Catalog:

      1. Vertical Drill - Used for Boring Part Faces/Backs.

      2. Horizontal Drill - Used for Boring into Part Edges. Will always be output as a Secondary Operation.

      3. Flat Router - Covers the wide range of Standard Straight router bits.

      4. Ball Router - Used for 3 dimensional routing and carving also used for routing slots with rounded bottoms or rounded inside corners (Flutes).

      5. Bull Nose Router - Typically used for decorative bottom cutting.

      6. Tool Set - A method for linking one or multiple tools together to perform a single operation.  The primary purpose for a Tool Set is for the Raised Panel detail for MDF Doors. A couple of other examples may be to preset a router bit used to create a bull nose to drop at the correct depth and offset for the bull nose.  Another example may be to use a flat router to cut out a section then follow with a shaped router bit to alleviate extra wear on the shaped router bit.

      7. Dovetail Tool Set - Allows the definition of a Dovetail Tool as well as the Male and Female toolpath movements required.

      8. Saw - Typically used for Grooving. A common accessory on Point-to-Point Machines. Also an Aggregate option on most CNC Routers.

      9. 5-Axis Saw - Typically used for cutting Miters on Parts.

      10. V Router Bit - A conical bit used for machining Miters in 2 ½ axis.

      11. Shaped Router - Allows definition of Shaped Router bits.

 Copy

Allows you to Copy the current Tool.

 Delete

Allows you to Delete the current Tool.

Tool Display Filter

This option allows you to control which Tools are displayed by Machine assignment. The Any option will show Tools which are available for Any Machine. Selecting a Machine from the list will show Only Tools that are assigned to that Machine and/or Tools available for that Machine (Tools available for Any Machine).

Tool Type Filter

This option allows you to control which Tools are available by Tool Type.

Tool Selector

This option allows you to select which Tool is active for editing here.

Tool Properties

See the Tool Catalog Setup topic for more information.

S2M CENTER for ALPHACAM Tool Catalog Option

When opening the Tool Catalog from inside ALPHACAM there is an additional Tool in the toolbar.

Import Tool - Allows you to browse to your ALPHACAM Tools and Import selected ALPHACAM Tools into the S2M CENTER Tool Catalog.
