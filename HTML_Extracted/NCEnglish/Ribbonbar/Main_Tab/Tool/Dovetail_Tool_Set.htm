Dovetail Tool Set

Other than the standard Tool Properties there are several Tool Properties specific to the Dovetail Tool Sets. Each of these is preceded by a letter which is visually defined on the Tool Shape tab.

Tool Size

Items preceded by a letter designation reference the area noted with that letter in the illustration on the lower right of the Tool Catalog window.

  • Diameter

The S2M Center will use this size to match tools to operations.  In addition it will determine for dado's if they are a single pass, rectangle, or pocket.  This diameter should match the diameter of the tool on the machine; however there are some instances where you may not want them to represent the actual size. One example; if you always wanted your dadoes to create a rectangle or single pass regardless of the actual size of the tool on the Machine. In this case you would enter a Tool Diameter that matched the dado width for a single pass or was smaller than the dado width for a rectangle. See the topic on Automatic Tool Selection Logic for more information.

  • Work Depth

This field is used to Display the Graphic for tools. It holds no other importance.

  • (A) Bottom Radius - Sets the bottom radius of the Dovetail Tool as highlighted below. For proper fit it is important you have a tool no larger than this that can perform the Male cut.

  • (C) Top Radius

  • (D) Min Width

Dovetail Female Operations

  • (H) Pocket Width

  • (I) Pocket Depth

  • (H) Pocket Tool

  • (J) Plunge Depth

  • (K) Cut Depth

  • (L) Cut Adjustment - Allows you to adjust the Width of the Female Operation in order to adjust the fit. This setting will not affect the size of the Male Operation.

  • (L) Tail Width Extension - Adjusts the path for both the Male and Female and allows you to widen your actual Dovetail larger than the actual Tool. This is not a "Fudge" value for making the Female Operation wider for the Male Dovetails. See (L) Cut Adjustment above for fit adjustment.
