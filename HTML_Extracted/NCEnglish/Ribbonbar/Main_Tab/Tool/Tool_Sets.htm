Tool Sets

Tool Sets give the ability to assign multiple Tools to a single Route path with one selection. You have the ability to assign each Tools offset, depth, and other items in relation to a predefined Route Path.

Following are the sections in the Properties section for Tool Sets:

General

  • Machine - Assigns the selected Tool Set to a particular Machine(s) from the Machine Catalog.

  • Name - A name you enter to easily identify the Tool Set.

  • Tool Number - This is typically 0 as each individual Tool within the Tool Set dictates the Tool used by the machine. However, if you want to assign a Tool Set to a DXF Layer on imported DXF Parts you can use this as the Tool Number for identifying the Tool Set. It is recommended in that case that you set this to some unique Tool Number.

  • Tool ID - This field is not editable. Its value is a unique number automatically assigned to each Tool/Tool Set. This value can be called from within your design package to assign a specific Tool to a specific Operation. For Example, within Cabinet Vision, in a User Created Standard you can use the Parameter ToolID equal to the Tools Tool ID number to make an assignment.

Tool Size

  • Diameter - Specifies the Diameter of the Tool Set. Establishes logic for Automatic Tool Selection. Only valid if Allow Auto Select is enabled.

  • Work Depth - Specifies the Working Depth of the Tool Set.

Tool Selection

  • Allow Auto Select - This option allows you to exclude the Tool Set from Automatic Tool Selection Logic. This field will only be available if one of the Tools within the Tool Set has a depth reference of Operation Depth.

  • Outline Tool - This option allows you to establish the Tool Set as an Outline Tool. This field will only be available if one of the Tools within the Tool Set has a depth reference of Bottom of Material or Operation Depth.

  • Down Shear - Establishes logic for Automatic Tool Selection. Only valid if Allow Auto Select is enabled.

  • Up Shear - Establishes logic for Automatic Tool Selection. Only valid if Allow Auto Select is enabled.

Cut For Glass Doors

  • Cut For Glass - This field is not editable. Its value will change depending on whether you have defined a Tool in your Tool Set to "Cut / Glass".

  • Follow Glass Shape - This option defines whether or not you would like your Tool Set, if set to cut glass, to follow the shape of the opening or to setup to pocket out the opening for a piece of square glass. In the images below the Rabbet on the Backside for the Glass to fit into is represented in red.

  • Rabbet Width - Defines the Width of the Rabbet on the Backside for the Glass to fit into.

  • Rabbet Depth - Defines the Depth of the Rabbet on the Backside for the Glass to fit into.

  • Router for Back - Defines the Route Tool to use to route the Rabbet on the Backside for the Glass to fit into.

Nesting

  • Nested Part Clearance - This sets the space in between Doors and Door Parts on a nested sheet when the selected Tool Set is used. This value is added to the Part Clearance set on your Nesting Machine. For example you set a 2” Nested Part Clearance (NPM) and ½” Part Spacing on the Machine, the space between two parts with a NPM of 2” will be 4 ½”. Between a part with a NPM and one without, the spacing will be 2 ½”.

Door Profile Types

  • Outside Edge - Defines whether the selected Tool Set will be available as an Outside Edge Profile.

  • Inside Edge - Defines whether the selected Tool Set will be available as an Inside Edge Profile.

  • Raised Panel Detail - Defines whether the selected Tool Set will be available as an Raised Panel Detail.

  • Route - Defines whether the selected Tool Set will be available as an Route Pattern.

Edit Shape

When defining a Tool Set's Shape you are actually defining the Tools and Offsets used to define the Tool Set. Since this information will transfer to your CNC Machine, the accuracy of the information you enter here is paramount.

  • Add - Adds a new Tool to the Tool Set.

  • Delete - Removes the selected Tool from the Tool Set.

  • Move Up - Moves the selected Tool Up one level for each click.

  • Move Down - Moves the selected Tool Down one level for each click.

  • On Center  - This option displays the Tool Set as it will follow a path defined in the Door, such as a Route Pattern or Template Door Route.

  • On Edge - This option displays the Tool Set as it will follow a path defined on the Edge of a Door.

  • Show Tool - This option toggles the Tools display on/off in the Graphic sample. This can make it a little easier to see your final result.

  • Show Only Selected Tool - Will allow you to only see the currently active tool.

  • Zoom In - Allows you to Zoom In on the Sample Shape.

  • Zoom Out - Allows you to Zoom Out on the Sample Shape.

  • Reference From Left - Establishes that the Offset Dimension for all Tools in this Tool Set are referenced from the Left Side of the Tool Path.

  • Reference From Center - Establishes that the Offset Dimension for all Tools in this Tool Set are referenced from the Center of the Tool Path.

  • Reference From Right - Establishes that the Offset Dimension for all Tools in this Tool Set are referenced from the Right Side of the Tool Path.

Following are the options for Tools added to your Tool Set:

Tool Properties

  • Router Bit - This drop down allows you to select which route Tool, defined in the Tool Catalog, to use in the Tool Set.

  • Offset - The path a Tool Set follows is defined by a single line or path. This setting determines the offset Left or Right the Tool should be from the line or path.

  • Depth - This value determines the depth the Tool should cut within the Tool Set.

  • Drop Offset - This value is for use with under-cut type bits. This value instructs Solid how far away from the edge the center of the Tool needs to be prior to entering the Part to be cut.

  • Cut For Glass - Select this option if you want the selected Tool to define the Glasses position if you are defining a Glass Door Tool Set. This will generate a backside Rabbet for your glass. See the Rabbet definition above.

Operation Properties

The Operation Properties presented here vary depending upon the selected Operation Type. Following are the available Operation Types and their associated Properties:

  • Type - Rough - A standard, and the most common, Operation Type.

    • Tool Side - This option sets the side of the Path the Bit is following that has bit pressure focus. See diagrams below.

Looking at the Tool Set below the Raised Panel Cutter is set to Clean Left, this is because the Left Side of this operation requires a Finish pass and we would want the Left Side to be the cleanest possible cut. The other three Tools are set to Clean Right, this is because the Right Side of these operations requires Finish passes and we would want the Right Side to be the cleanest possible cut. See the second diagram below for more information.

In the diagram below if you will notice the direction of cut in combination with tool rotation places the bits cutting focus on the specified side of the cut or the entry of the cut. Whereas the opposite edge is the exit of the cut and therefore would be more likely to have tear out than the entry.

  • Type - Engrave - ngrave is a special Operation Type designed for Engraving.

Make sure to use a V Router BIt for the Engrave Operation Type the path of the Engrave is calculated from the shape of the Tool. A Flat Router would not create a proper path.

    • Tool Side - Previously Covered.

    • Start Point - Standard Operations start in the Middle, however, when engraving typically you would start in the corner.

  • Type - Pocket - Pocket is a special Operation Type designed for making Pocket Cuts.

  • Type - Clean Up - Clean Up is a special Operation Type designed to Clean Up Corners after a Large Diameter Tool Pass. Simply specify the Tool Radius to Clean Up after. Notice in the animation below that the Clean Up Cutter comes in after the initial pass and only cleans up in the Corners rather than making a complete set of passes to clean up.

  • Type - Vertical & Horizontal Slats - Vertical and Horizontal Slats are special Operation Types designed to Cut Slats allowing the creation of Shaker Style MDF doors. Specify the Distance from the Route Path to the Start/End of Slat; this keeps the Engraver used for the Slats from overlapping your perimeter. There are two options for the Slat Spacing "Optimal" which will vary the Slat Spacing but not exceed the Offset value and "Fixed" will use a Fixed value for the Slat Spacing based on the Offset value. Simply specify the Distance between the Slats in the Offset Column.0

  • Type - Finish - The Finish Operation Type, similar to the Clean Up Operation Type, is designed to allow you to assign a Finish Tool to clean up inside radiuses on the outside of a tool path. There are a few very IMPORTANT things to note when utilizing this Operation Type.

    • The Outline Tool option is disabled unless the depth reference is Bottom of Material or Operation Depth

    • A toolset will not be available for auto-select unless at least one tool is selected for Operation Depth

    • Set the Nest Clearance to 0 unless you need extra clearance

    • You have to manually set the tool size.

    • The outline path should have a clean left edge.

The animation below demonstrates the path of a Finish Operation Type.

  • Type - Profile - The Profile Operation Type, similar to the Clean Up Operation Type, is designed to allow you to assign a Profile Tool to clean up inside and outside radiuses on a profiled toolpath.

    • Maximum Depth Per Cut - This is the depth of cut used if the depth calculated by the Maximum Error exceeds this value. In most cases the default value should be sufficient.

    • Maximum Error - This value is used to calculate the depths of cut and determines the surface finish. The depth of cut is calculated by moving the tool out to the Maximum Error then down till the contact point meets the slope. In most cases the default value should be sufficient.

    • Clean Both Sides - This determines if you want both Outside and Inside Corners cleaned. When disabled it only cleans Outside Corners.

Reference Properties

  • Depth - The Fixed Depth selection allows you to specify the Depth of the Cut from the Top of the Material being cut by a positive amount. Material + Depth allows you to specify the Depth of cut from the Bottom of the Material by a Negative number. See Diagrams below.

Fixed Depth is a Positive Depth Value Measured from the Top of the Material this allows the Depth to stay consistent with a Material thickness change.

Material + Depth is a Negative Thickness Value Measured from the Bottom of the Material; this allows the remaining thickness from the cut to stay consistent with a Material thickness change.

Operation + Depth gets its depth from the defined Operation. Use the Sample Operation Depth at the lower left to see how your Tool Set will be applied utilizing this option.
