Creating an S2M CENTER Part List

The S2M Part List Creator application has been supplied in the Installation extraction folder which can be used to easily create the CncRun.txt file to simplify the process of bringing your DXF's into the S2M CENTER. The installation file is located in the Installation extraction folder at \Support\PartListCreator\. Simply double click Setup.exe to install the Part List Creator on your computer.

S2M CENTER Part List Creator

Open the S2M Part List Creator and see its Help file for information on its use.
