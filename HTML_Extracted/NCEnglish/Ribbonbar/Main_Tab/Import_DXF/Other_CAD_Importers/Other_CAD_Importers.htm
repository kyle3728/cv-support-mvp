Other CAD Importers

The DXF Import utility has some predefined DXF Layer Schemes you can utilize depending upon what CAD application you are using to generate your DXF's. There are six pre-defined Layer Schemes provided which are setup to match your CAD applications default output however each can be modified to suit your needs.

Default

A generic sample you can modify to suit your custom needs. This one and any new ones you create will be this style and require that you set the Open File Type to DXF files (*.dxf) when importing your DXF's.
