KAB-NX

To use the KAB-NX CAD Import Link you will need to get the Import Link License added to your License by from your local S2M CENTER supplier.

If you are using KAB-NX to create the DXF files you want to process in the S2M CENTER then this is the Scheme you need to have selected prior to importing KAB-NX DXF files.

When you output your DXF's from KAB-NX it will create a JobName.txt file which is what you would use to Import the DXF files into the S2M CENTER. Simply change the File Type in the Open Dialog to KAB-NX (*.txt).

Following is the format for the KAB-NX Part List text file.

KAB-NX Part List

JobName.txt File Structure

Typical Part Entry

End_Left_0_f.dxf, End_Left, 12, 30, 0.75, 3/4 Melamine, 1, 1, 1, 0, F

Field Contents

1 DXF filename

2 Part description

3 Part width (x)

4 Part length (y)

5 Part thickness (zz)

6 Part material name

7 Cabinet or unit number

8 Quantity to be generated

9 Rotation flag (0=no 1=yes)

10 Part Mirrored (0=no 1=yes)

11 Part Face (F=Front B=Back )
