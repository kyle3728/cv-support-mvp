Blum Dynaplan

If you are using Blum Dynaplan to create the BXF files you want to process in the S2M CENTER then you are not required to have a DXF Scheme selected as we can read all machining information directly from the file.

Simply select the Blum Dynaplan files (*.bxf) for your file type then select and open your BXF file.

All Parts within the BXF file will be listed in the DXF Import area.

The material list will be filled with materials that match the part size. If there is no match a material will be created to match the part size.

You can then choose which parts you would like to import or import all the parts by selecting the “Save” button.

BXFExt.ini

The BXFExt.ini is located in the installation extraction folder at \Support\BlumDynaplan\ and should be copied to the \NcCenter_12 folder. The BXFExt.ini file serves two purposes, localization and part orientation.

Localization

The data in the BXF file is in German, along with the part names. The .ini file can be used to change the part to recognizable part names. Note: The same part might have different names or may be in a numbered sequence. The following is a sample:

[LocalPartNames]

LinkeSeitenwand - Seitenwand=Left End

LinkeSeitenwand=Left End

Rueckwand=Back

*Rueckwand_2=Back #2

*Rueckwand_3=Back #3

Orientation

The parts in a BXF file do not have a typical orientation like the S2M CENTER is used to, they are simply shapes in space. The orientation is determined by the size and position of the part. The part’s face is based on the part’s location in the cabinet. By default all part faces point out. However there may be some parts, like a back, where the part should be oriented face in. To override the normal orientation of a part use the [PartFaceTowardCenter] section of the BXFExt.ini.

Parts listed in this section can be the actual part name or the localized name. Be sure to account for parts in a numbered series. Any non-zero value (such as 1) will change the part face towards the center of the cabinet.

The following is a sample:

[PartFaceTowardCenter]

Rueckwand=1

Back=1

Back #2=1

Back #3=1

Top=0
