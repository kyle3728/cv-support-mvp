SmartLister

To use the SmartLister CAD Import Link you will need to get the Import Link License added to your License by from your local S2M CENTER supplier.

If you are using  SmartLister to create the DXF files you want to process in the S2M CENTER then this is the Scheme you need to have selected prior to importing SmartLister DXF files.

When you output your DXF's from SmartLister it will create a WORKLIST.txt file which is what you would use to Import the DXF files into the S2M CENTER. Simply change the File Type in the Open Dialog to SmartLister (Worklist.txt).

Following is the format for the SmartLister Part List text file.

SmartLister Part List

WORKLIST.txt File Structure

Typical Part Entry

C:\1-CNC-Input\Test Job 1\End_Left_0_f.dxf, 3/4 Melamine,1,1,"SD, End_Left", JohnDoeKitchen-1,???,EB,EB,EB,EB

Field Contents

1 Full Path to DXF File

2 Part material name

3 Rotation flag (0=no 1=yes)

4 Quantity to be generated

5 Part description

6 Job Name-Part cabinet number

7 Not used

8 Edgeband Info 1 (Optional)

9 Edgeband Info 2 (Optional)

10 Edgeband Info 3 (Optional)

11 Edgeband Info 4 (Optional)
