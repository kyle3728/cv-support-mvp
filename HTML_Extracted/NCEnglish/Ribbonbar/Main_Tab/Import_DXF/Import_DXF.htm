Import DXF

Import DXF Parts allows you to bring in Parts you have created in other CAD packages into the S2M CENTER.

To Import a DXF Part:

  1. Click the Select button.

  2. Browse to the location of your DXF Part, click on it, and click Open. See the Other CAD Importers topic for other import options.

You can use standard Windows Ctrl + Click or Shift + Click combinations to select multiple DXF's here.

  1. Click the Open button.

Once your DXF Parts are displayed on the screen they ready to be added to your Job by clicking Save. However, there are several options prior to Adding the Part to the Job you may want to modify. First, your Parts will appear on the Viewing Screen to the Left.

By selecting each Layer Name in the list at the upper Right of the screen you can see that Operation highlight in the preview screen you will also be able to see below how the S2M CENTER has translated the Layer Names.

Each identified Part or set of Operations will be displayed, when selected, either in Blue or in Red. If the Part or Operation is displayed in Red it means the S2M CENTER has been unable to determine the Operation Type. This typically means that the Layer has not been defined correctly. You can either edit the Layer Scheme for the S2M CENTER or edit the Layer Name within the DXF from this area.

Edit DXF Layer Scheme

This is where you define how your Layers coming from your CAD package are defined.

Toolbar

There are 5 options in the toolbar:

Layer Scheme Selector - Allows you to select which DXF Layer Scheme to use.

New Scheme - Allows you to create a new DXF Layer Scheme.

Copy Scheme - Allows you to copy the current DXF Layer Scheme.

Rename Scheme -  Allows you to rename the current DXF layer Scheme.

Delete Scheme - Allows you to delete the current DXF Layer Scheme.

There are three sections in the Advanced Screen:

Layers -  There are 12 options in the Layers Section.

  1. Part - Specifies which Layer defines the Parts boundary.

  2. Outline - Specifies which Layer defines the Parts Outline.

  3. Vertical Bore - Specifies which Layer defines Vertical Boring Operations.

  4. Horizontal Bore - Specifies which Layer defines Horizontal Boring Operations.

  5. Left Horizontal Bore - Specifies which Layer defines Left Horizontal Boring Operations.

  6. Right Horizontal Bore - Specifies which Layer defines Right Horizontal Boring Operations.

  7. Front Horizontal Bore - Specifies which Layer defines Front Horizontal Boring Operations.

  8. Back Horizontal Bore - Specifies which Layer defines Back Horizontal Boring Operations.

  9. Route - Specifies which Layer defines Route Operations.

  10. Dado - Specifies which Layer defines Dado Operations.

  11. Pocket - Specifies which Layer defines Pocket Operations.

  12. Edge Miter - Specifies which Layer defines an Edge Miter.

Keywords - There are 8 options in the Keywords Section.

  1. Tool Number - Specify the character that represents your Tool Number. Typically it is the letter T.

  2. Operation Depth - Specify the character that represents your Operation Depth. Typically it is the letter D.

  3. Operation Width - Specify the character that represents your Operation Width. Typically it is the letter W.

  4. Offset Left - Specify the character that represents if your Operation needs to machine with a Left Cutter Offset (Comp). Typically it is the letter L.

  5. Offset Right - Specify the character that represents if your Operation needs to machine with a Right Cutter Offset (Comp). Typically it is the letter R.

  6. Tool Inside - Specify the character that represents if your Operation needs to be machined on the Inside.

  7. Tool Outside - Specify the character that represents if your Operation needs to be machined on the Outside.

  8. Secondary Face - Identifies that the Operation is on the Secondary Face. Without this designation on an operation the assumption is the Operation is on the Primary Face. This allows you to define machining on both faces of the Part from one DXF drawing.

Miscellaneous -  There are 3 options in the Miscellaneous Section.

  1. Decimal Separator - Specify what character you use as a decimal separator. Typically it is either the letter p or an underscore _.

  2. Part Face Up - Specify what character you use to indicate the Part is Face Up. By default it is assumed to be Back Up.

  3. Use Thickness as Depth - Set this to True if your CAD geometry includes a true thickness representation.

If there is a duplicate Layer name or Keyword a notification occurs next to the OK button and you will be prompted to fix the problem before exiting, copying or selecting another scheme.

The Part Layer should (but is not required) contain a rectangle representing the extent of the part. The Outline Layer should hold the part shape.

This utility allows you to edit the Layer name passed by the DXF. Simply type the new layer name(s) you wish to use in the corresponding New Layer Name field.

Known Limitations

Although the DXF Import reads a DXF file there are some differences between an output DXF and Input DXF that should be noted.

  1. One DXF file per part. It is common to have a Partition output both a front and back program. However since each DXF file is assigned to a part the DXF file should contain both the face and the back operations rather than having them split between two DXF files.
  2. No mirrored parts. The DXF files input should not be mirrored.  Currently the application will only rotate part 90 degrees or flip the part over if necessary.
  3. Operation size is assumed to be the bounding box of the operation.

For example, if you output the tool path for a dado as centerline, the geometry for the dado is smaller than the actual dado required since the operation will grow by the tool diameter.

  1. A DXF file can define quite an assortment of CAD entities that the S2M CENTER can not process. If the S2M CENTER can not process an entity it will be ignored and a warning will be issued. The following entities are supported by the S2M CENTER:
    • Arc
    • Circle
    • Line
    • LWPOLYLINE
    • POLYLINE
    • SEQEND
    • VERTEX
