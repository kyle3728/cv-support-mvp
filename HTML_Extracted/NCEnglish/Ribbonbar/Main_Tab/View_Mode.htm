View Mode

There are two options in View Mode:

Part View

This is the Default View when only outputting to a Secondary Machine. The Part View displays the Materials and Parts list in the sidebar and the selected Part and it's machining in the Pattern/Part Display Area.

Right Click Menu

 

 To Drawings - Only available with CABINET VISION. This option will send the current View as a Drawing Scene to the CABINET VISION Drawings Area.

 Properties - Clicking on the Properties command brings up the following window.

General Tab

There are two main options on the General Tab:

      1. You can control whether Operations are visible on the Pattern printouts.

      2. You can, via the slide bar, control the speed of the CNC Player when using Show Tool Path in the Patterns section of the S2M CENTER.

Annotations Tab

On the Annotations Tab there are seven main options:

      1. Font - You can control the displayed Font.

      2. Dimensions Horizontal Offset - You can control the Horizontal Offset of your Dimensions.

      3. Dimensions Vertical Offset - You can control the Vertical Offset of your Dimensions.

      4. Dimensions Text Height - You can control the Text Height in your Dimensions.

      5. Dimensions Tick Height - You can control the Tick Height in your Dimensions.

      6. Annotations Heading Text Height - You can control the Text Height in the Heading Annotations.

      7. Annotations Part Labels Text Height - You can control the Text Height in your Part Labels (Labels here refers to the Text marking your Part on the Sheet not the Label Reports you Print.).

Pattern View

This is the Default View when Optimizing or Nesting. The Pattern View displays the Patterns list in the sidebar and the selected Pattern and it's machining in the Pattern/Part Display Area.

This area displays your Sheet/Pattern layouts.

If a part can be rotated (not saw output, not a grain match part/block, not viewing the 6th face).  A point of origin (dark green) and a rotation point (light green) will appear on the part.  When the mouse is near the rotation point   the cursor will change to indicate the part can be rotated.

Click the mouse once to start rotation, click again to finish.

Whenever the rotated part is near 0, 90, 180, or 270, the part will snap to that angle.

The lower Section is the Part List and Clipboard area.  The Parts on the active sheet are listed in the lower section of the screen. There are several columns of information which you can sort the list by clicking on the column heading. A Part selected from either the Pattern are or Part list area will cause that Part to be highlighted in both Views.

Any items Cut or Copied via a Part Right Click will be listed on the clipboard . Simply drag any Part in the Clipboard to the Sheet where you want it placed.

In the Parts List/Clipboard area you have several tools you can use to modify Parts within the Nest:

Part List View - Shows all Parts on the current Pattern in the list view.

Clipboard View - Shows a list of all Copied Parts.

Show 6th Face/Show 1st Face - Allows you to flip the Pattern over to see the 6th Face/1st Face operations if there is a 6th Face option on the selected Pattern.

Rotate Clockwise - Rotates the selected Part clockwise 90 degrees

Rotate Counter Clockwise - Rotates the selected Part counter clockwise 90 degrees.

Rotate 180 - Rotates the selected Part 180 degrees.

Flip Part - This option will flip the selected Part. This action will typically create either a secondary program, modify the secondary program, or modify the 6th face of the sheet.

Right Click Menu

 

There are several options in the Sheet Display Area right click menu.

    1.  Add Parts - Brings up the Part Library and allows you to add a Part from it to the current sheet.

Simply select any Part you have saved into your Part Library enter a Length, Width, Quantity, and Overmake and click Add.

    1.  Copy Part - This option copies a Part to the clipboard  for adding to the current sheet or any other sheet within the Job. (Even a different Material and/or Thickness.)

    2.  Cut Part - This option removes a Part to the clipboard for adding to the current sheet or any other sheet within the Job. (Even a different Material and/or Thickness.)

    3.  Paste Part - This option will take the last part that was either copied or cut and place it on the Current Sheet.

    4.  To Drawing - Will send the current screen to the Cabinet Vision Drawings Area as a Drawing Scene.

    5.  Properties - Clicking on the Properties command brings up the following window.
