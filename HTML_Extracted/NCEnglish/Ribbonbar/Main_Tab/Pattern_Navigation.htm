Pattern Navigation Option

First Page

Clicking the First Page option will display the first Pattern the Pattern View.

Previous Page

Clicking the Previous Page option will display the previous Pattern the Pattern View. Alternatively you can use the key on your keyboard.

Current Page Display

The Current Page Display option displays the current Pattern as well as the total number of patterns in the selected group (material). Clicking in here and changing the first number will allow you to type a specific Pattern number to navigate to.

Next Page

Clicking the Next Page tool will display the next Pattern in the Pattern View. Alternatively you can use the key on your keyboard.

Last Page

Clicking the Last Page tool will display the last Pattern in the Pattern View.
