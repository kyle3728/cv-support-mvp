Edit Shape

The Shape Editor consists of several tools designed to allow editing the shapes of Parts in the S2M Center.

Click the ribbonbar options below for more information

 Return

The Return option will prompt to Save your Part and return you to the main Edit View.

 Clipboard

The Clipboard option has three tools:

 Cut

The Cut option allows you to "Cut" selected objects to the Windows clipboard for pasting.

 Copy

The Copy option will allow you to "Copy" selected objects to the clipboard for pasting.

 Paste

The Paste allows you to "Paste" an object stored on the Windows Clipboard onto your Part.

Zoom Group

See the Zoom Group topic for more information.

Layer Selector

There are two options in the Edit Shape Layer selector:

Shape - The default Layer and the one all Shapes must be drawn on.

Construction - This Layer allows you to draw Construction/Reference Lines to assist in creating your shape.

Depending on which Layer is active some options in the Modify section of the Ribbonbar may or may not be available.

 Select

This option will change the cursor to an arrowhead and allow selection of lines, arcs, etc. After selecting the item, you will be able to change properties by selecting the desired property off the list in the sidebar and selecting the item to apply it to. Lines can be moved, resized, etc.

 Select All

This option will Select All objects in the Shape Editor.

 Measure

This option allows you to Measure Objects.

 Delete

This option will allow you to Delete the selected CAD Object(s).

 Transform

The Transform tools will only be available (not greyed out) when selecting objects which are not Constrained.

 Flip

This option will Flip the selected object. First select all of the CAD Objects you would like to Flip using the CAD Selection Options.

Then click the Transform|Flip option in the Ribbonbar.

 Mirror

This option will Mirror the selected object. First select all of the CAD Objects you would like to Mirror using the CAD Selection Options.

Then click the Transform|Mirror option in the Ribbonbar.

 Rotate

This option will Rotate the selected object. You will see the following dialog box when you select this option to choose how you want to rotate the object.

 Crop

There are 2 main Crop options in the sidebar once the Crop Tool is selected:

Crop Type - You have three Crop Types available; Rectangle, Circle, and Polygon that determine the shape of the area being cropped.

Crop Area - The Inside option will get rid of what is Outside the Boundary and keep what is Inside. The Outside option specifies that you want to keep what is outside the Crop Boundary and get rid of the objects inside the Boundary.

The Inside option can be a bit aggressive as it will remove all Objects in the current view if it is not inside the boundary. To keep this option from being so aggressive you can limit the Objects that are affected by a Crop by selecting the Objects first.

Remember you can always click Undo option in the Quick Access Toolbar or use the + Undo shortcut to Undo your changes.

Remember you can select multiple Objects by holding the key on the keyboard down as you click the Objects.

 Bend

This tool allows you to select a Line and turn it into an Arc by dragging it to Bend it.

 Line

This tool allows the creation of Lines to define the shape of your Part. After selecting and placing a line, the line properties window will display in the sidebar. Clicking on the items listed in this window will allow modification of that item.

 Chained Line

This tool allows the continuous creation of connected Lines.

 Line Box

This tool allows the creation of a Rectangle.

 Arcs

 Arc by Radius

This option allows definition of a point where a radius begins, the radius, and the degrees of radius. The first mouse click marks the center of the radius, pull the radius that you want and click again.

A circle will appear with a radius line within it. Move the cursor and the circle opening becomes an arc as you do so.

When you reach the correct position, a third click will set the arc at the selected point.

 3 Points Arc

This option lets you place two points to define the opening of the arc, and the third point pulls the arc increasing the radius. A 3 Points Arc is very handy when you have three known points to Snap to.

In the example below I want to create the Arc shown in red.

To begin I select the Arcs|3 Points Arc option from the Ribbonbar then Left Click the End Point (With Snap To Points on) of the first line.

Next I Left Click the Mid Point of the second line.

And the last and third point I Left Click the End point of the third line.

And now I have the Arc.
