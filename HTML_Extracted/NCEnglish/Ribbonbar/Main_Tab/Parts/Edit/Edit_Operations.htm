Edit Operations

The Operations Editor consists of several tools designed to allow editing and adding of Operations to Parts in the S2M Center.

Click the ribbonbar options below for more information

Layer Selector

There are two options in the Edit Shape Layer selector:

CAM - The default Layer and the one all Operations must be drawn on.

Construction - This Layer allows you to draw Construction/Reference Lines to assist in creating/editing your Operations.

Depending on which Layer is active some options in the Modify section of the ribbonbar may or may not be available.

See the Edit Shape topic for more information on the Zoom, Inquiry and Modify groups.

 Line Bore

When selecting the Line Bore option you will left click to establish the start point of your row of Line Bores then drag and left click to establish the end point.

When selecting a Line Bore operation there are thirteen options in the sidebar Properties area:

  • Pos X - Establishes the X Position of the start point for the Line Bore.

  • Pos Y - Establishes the Y Position of the start point for the Line Bore.

  • Pos Z - Establishes the Z Position for the Line Bore.

  • End X -  Establishes the X Position of the end point for the Line Bore.

  • End Y - Establishes the Y Position of the end point for the Line Bore.

  • Length - Establishes the Length of the Line Bore.

  • Diameter - Establishes the Line Bore Diameter.

  • Depth - Establishes the Line Bore Depth.

  • Angle - Establishes the Line Bore angle. Rotation occurs around the start point.

  • Spacing - Establishes the Line Bore spacing.

  • Repeat - Establishes the number of Line Bores.

  • Tool - Allows you to assign a specific tool to the Line Bore.

  • Description - The Description of this Line Bore.

 Clamex

Use this option to place machining for Lamello Clamex connectors.

When selecting a Clamex operation there are ten options in the sidebar Properties area:

  • Pos X - Establishes the X Position of the Clamex.

  • Pos Y - Establishes the Y Position of the Clamex.

  • Pos Z - Establishes the Z Position of the Clamex.

  • Width - Displays the Width of the Clamex. The Type selected determines its Width.

  • Length - Displays the Length of the Clamex. The Type selected determines its Length.

  • Depth - Displays the Clamex Depth. The Type selected determines its Depth.

  • Type - Establishes the Type of Clamex to machine for. There are four types available:

    • P10

    • P15

    • P14

    • 15/10

  • Tool - Allows you to assign a specific tool to the Clamex.

  • Description - The Description of this Clamex.

 

 Line

Using the Line option allows you to draw one Line at a time by selecting the start point then the end point. See the Line option in the Edit Shape topic for more information.

When selecting a Line there are ten options in the sidebar Properties area:

  • Pos X - Establishes the X Position of the start point for the Line.

  • Pos Y - Establishes the Y Position of the start point for the Line.

  • Pos Z - Establishes the Z Position for the Line.

  • End X - Allows you to set the Ending X position for the Line.

  • End Y - Allows you to set the Ending Y position for the Line.

  • Length - Establishes the Length of the Line.

  • Depth - Establishes the Line Depth.

  • Angle - Establishes the Line angle. Rotation occurs around the start point.

  • Cut - There are three options for Cut:

    • Centerline - Cuts along the Centerline of the Line.

    • Inside - Cuts along the Inside of the Line.

    • Outside - Cuts along the Inside of the Line.

  • Tool - Allows you to assign a specific tool to the Line.

  • Description - The Description of this Line.

 Circle

When selecting an Circle there are eight options in the sidebar Properties area:

  • Pos Y - Establishes the Y Position of the center-point for the Circle.

  • Pos Z - Establishes the Z Position for the Circle.

  • Diameter - Establishes the Diameter of the Circle.

  • Depth - Establishes Circle Depth.

  • Cut - There are three options for Cut:

  • Tool - Allows you to assign a specific tool to the Circle.

  • Description - The Description of this Circle.

 Rectangle

This tool provides for creating a Rectangle (or square) by clicking the point where it is to start and clicking on the point where it is to finish. It can thereafter be adjusted by using the Select tool. The area in the sidebar is similar to the one for the Line tool.

Pressing + Left Mouse Drag will force the Rectangle to be a Square.

When selecting an Rectangle there are eleven options in the sidebar Properties area:

  • Pos X - Establishes the X Position of the start point for the Rectangle.

  • Pos Y - Establishes the Y Position of the start point for the Rectangle.

  • Pos Z - Establishes the Z Position for the Rectangle.

  • Width - Establishes the Width of the Rectangle.

  • Length - Establishes the Length of the Rectangle.

  • Depth - Establishes the Rectangle Depth.

  • Angle - Establishes the Rectangle angle. Rotation occurs around the start point.

  • Type - There are two choices for Type:

    • Outline - Will cut an Outline of the Rectangle.

    • Pocket - Will Pocket out the Rectangle.

  • Cut - There are three options for Cut:

    • Centerline - Cuts along the Centerline of the Rectangle.

    • Inside - Cuts along the Inside of the Rectangle.

    • Outside - Cuts along the Inside of the Rectangle.

  • Tool - Allows you to assign a specific tool to the Rectangle.

  • Description - The Description of this Rectangle.
