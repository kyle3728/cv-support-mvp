Object Snapping

The S2M CENTER provides a special kind of cursor snap in addition to the typical grid snap. This allows the cursor to automatically snap to different hot-points on or adjacent to another object; for example, end-points, mid-points, center-points, and intersections. These locations are commonly referenced when creating or editing geometry. For most functions, the cursor will also snap to the origin of the object.

For example, you may want to create a line that is connected to the end of an arc. Start creating the line by locating the endpoint of the arc. When located, a round hot-point marker appears on the arc endpoint.

Click the hot-point marker to start the line there. The start of the line snaps precisely to the end of the arc, connecting these objects end-to-end.

You can choose what types of positions and conditions will automatically be located by enabling or disabling options in Preferences | Tools.

Dynamic Snapping

The S2M CENTER also has the ability to Snap Dynamically (also known as Assumptive Snapping). When you are in line with a Snap Point on another Object within the Drawing space the S2M CENTER will project that position to your location. When this projection along with the associated hot-point marker is displayed you can be assured that the Object will be placed in-line with that Objects Snap Point.

For Example you would like to draw two line Parallel to each other. After drawing the first line simply move your cursor in-line with the first lines Start Point and you will see a Snap Line projection assuring you they are in-line.

Hot-Point Markers

When the cursor moves over a valid snap location, the located position displays with a hot-point marker and a special cursor appears to identify the type of location.

The system will not display a hot-point marker for a particular location type if it is disabled in Preferences | Tools.

The list below identifies these special cursors:

 Cursor is snapping to one of the extremities (End Point) of an object.

 Cursor is snapping to the middle position (Mid Point) of an object.

 Cursor is snapping to the center of an arc or circle.

 Cursor is snapping to the intersection of two objects.

 Cursor has one coordinate aligned with any of the snap points.

 Editing CAD line is extending along the original angle.

 Editing CAD line is snapping to one of the smart line angles.

 Editing CAD line has a locked angle.

 Cursor is snapping to any of the points in the editing shape.

 Cursor is snapping to the grid.

 Cursor is snapping to the origin (0,0 position) in the view.
