Edit Shape Selection Options

There are several methods to select Edit Shape objects as well as several functions you can perform on Edit Shape Objects once they are selected.

Left Click Select

Simply Left Click on a single CAD Object to select it.

Left Click Select +

To select multiple CAD Objects simply hold the key while Left Clicking.

Left clicking + on a single CAD entity and selecting a CAD node for it will also put that item into Scale/Stretch/Move mode just like using the multi-select options.

Window Selection

You can also Left Click the screen and drag a Window to enclose all of the CAD Objects you want to Select.

Selecting Single Objects

Left Clicking + on a single entity and selecting a node for it will also put that item into Scale/Stretch/Move mode just like using the multi-select options. See the Selecting a Group of Objects section below for more information on Scale/Stretch/Move.

Selecting a Line

There are six options when selecting a Line.

  • Start X - Allows you to set the Starting X position of the selected CAD Object.

  • Start Y - Allows you to set the Starting Y position of the selected CAD Object.

  • End X - Allows you to set the Ending X position of the selected CAD Object.

  • End Y - Allows you to set the Ending Y position of the selected CAD Object.

  • Length - Allows you to set the Length of the selected CAD Object.

  • Angle - Allows you to set the Angle of the selected CAD Object.

Selecting an Arc (Arc by Radius)

There are five options when selecting an Arc (Arc by Radius).

  • Center X - Sets the Center X position of the selected CAD Object.

  • Center Y - Sets the Center Y position of the selected CAD Object.

  • Begin Angle - Allows you to set the Begin Angle for the selected CAD Object.

  • End Angle - Allows you to set the End Angle for the selected CAD Object.

  • Radius - Allows you to set the Radius of the selected CAD Object.

Selecting an Arc (3 Point Arc)

There are eleven options when selecting an Arc (3 Point Arc).

  • Start X - Previously covered

  • Start Y - Previously covered

  • End X - Previously covered

  • End Y - Previously covered

  • Arc Point X - Allows you set the Arc Point X position for the selected Arc.

  • Arc Point Y - Allows you set the Arc Point Y position for the selected Arc.

  • Center X - Previously covered

  • Center Y - Previously covered

  • Begin Angle - Previously covered

  • End Angle - Previously covered

  • Radius - Previously covered

Selecting a Group of Objects

There are eight options when selecting a Group of CAD Objects:

  • Left - Sets the Left position of the CAD Object.

  • Right - Sets the Right position of the selected CAD Object.

  • Top - Sets the Top position of the selected CAD Object.

  • Bottom - Sets the Top position of the selected CAD Object.

  • Center X - Previously covered

  • Center Y - Previously covered

  • Width - Sets the Width of the selected CAD Object.

  • Height - Sets the Height of the selected CAD Object.

There are several options when selecting CAD Objects you can initiate by simply clicking points on screen around the selected CAD Object(s).

Scale

Once a Group of CAD Objects is selected simply left click one of the corner points to Scale the selected Objects. You either eye-ball the Scale or you can use the Tab key to change the values in the sidebar for the selected objects.

Stretch

Once a Group of CAD Objects is selected simply left click one of the mid-points to Stretch the selected Objects. You can either eye-ball the Stretch or you can use the Tab key to change the values in the sidebar for the selected objects.

You can also Stretch to a Snap point. For example you can pick a single line and grab and end-point and stretch/extend that line to any other Snap point.

Move

Once a Group of CAD Objects is selected simply left click any point within the object to use that as the Move point, this will allow you to Move an Object or Group of Objects from one known (snap point) position to another (snap point position). You can also Move the selected Group of Objects by using the Tab key to change values in the sidebar.

Since Scale and Stretch are also available by clicking the corners of selected CAD make sure the proper cursor prompt is used.
