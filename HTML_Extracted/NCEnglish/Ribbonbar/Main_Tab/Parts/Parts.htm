Parts

This is the default opening view for the S2M CENTER.

You will notice in the figure above that the "Finished Left End" has a Red icon next to it on the Left. Any parts in the list denoted with a red icon is a part that will not be optimized. In this case the part is wider than the 48" wide Material that is specified. An icon that displays just a Router icon, such as the "Nailer" part, indicates that this Part is not tagged to Optimize. An icon that displays both the brown sheet and the yellow router, such as the "Finished Right End" part, indicates that this Part is tagged for both Optimize and CNC. See Right Click options later in this topic for more information.

To find the exact meaning of a particular Part icon simply hover over it to receive a description.

By default all items that can be sent to the Optimizer and/or your CNC machine will be sent to them. However, if you just wanted to send certain items to the Optimizer and/or CNC machine you can select them individually. To select a part simply click the icon next to the part. To select more than one simply hold the Ctrl key down while selecting. You can also select a range of parts by selecting the first part in the list you would like to send and then hold the Shift key down and select the last part. This will select all parts in between the first and last parts selected.

You can also change the Material of any part in the list. Simply click on the Material Name and a drop down will appear and allow you to change the current Part to any other Material available in the current PNC file.

Any item in the list can be modified. You can change the Quantity of parts, the parts size, you can even add an Overage using the Over column.

If you have created a Machine Set you will notice the second column in this view is the Sel column. This column indicates which Machine set the Part will be outputted to. This is a quick method to compare. In the figure below I have three possible Machine Sets, Manual (which is simply Manual output), Saw Set, and Router Set. The column, in order, will show either a | or a .

|.. indicates the Part is going to be output with Manual but not Saw Set or Router Set.

|.| indicates the Part is going to be output with both Manual and the Saw Set.

The red arrow in the view above is there to let you know you can add a new Part to this Part list.

Right Click Options

Right clicking on a Part in the Part list brings up the following menu:

 

There are six possible commands available in this Right Click menu:

    1.  Copy - Creates a Copy of the selected Part.

    2.  Edit

    3.  Delete - Clicking the Delete command will delete the selected part from the Parts List.

    4.  OK to Optimize - This option allows you to determine if the selected Part is OK to go to the Optimizer/Nester.

If you do not see this option check to make sure Show Parts Not Set for Optimization or Output is enabled in Preferences.

    1.  OK to Output CNC - This option allows you to determine if the selected Part is OK to go to your Secondary Machine.

If you do not see this option check to make sure Show Parts Not Set for Optimization or Output is enabled in Preferences.

    1. Save to Library - Clicking the Save to Library command will save the selected part to the Part Library for use in future Jobs.
