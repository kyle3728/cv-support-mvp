There are 5 possible tabs available in Material Properties. Most of theses are available in the Optimizer Parameters sidebar option on the Parts View. However, when utilizing the Options here they are applied to the selected Material avoiding effecting the Optimization/Nesting Parameters for other Materials in the Job.

Offcuts

The Offcuts Tab in Material Properties is the same as in Optimizer Properties with one exception. Offcuts set differently here will hold with the Material. This is handy when utilizing a high cost Material. You may want to set the Offcut Size (piece you save) to a smaller dimension for your high cost Materials.

Tools

The Tools Tab allows you specify specific Tools for the Front and Back Face of the selected Material.

A Tool Selection override is married to the currently selected Primary Machine. So, if you pick a different Primary Machine the Material will use the Tool Override that has been assigned to it.

A selection other than Auto Select made here will stick to your Material for all future Jobs utilizing the modified Material with the current Primary Machine.
