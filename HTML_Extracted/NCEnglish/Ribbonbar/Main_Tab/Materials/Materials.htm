Materials

This changes the View to a Material List. You can modify the Material, Sheet Size, Cost, and Quantity of materials in this list.

The red arrow in the view above is there to let you know you can add a new Material from the S2M Material Catalog, or a copy of a Material within the current Job with an alternate size, to this Material list.

Right Click Options

Right clicking on a Material in the Material list brings up the following menu:

 

There are 4 commands available in this Right Click menu:

    1.  Delete - Clicking the Delete command will delete the selected material from the Materials List.

    2.  Remove Tool Override - If you have set an alternate Tool as the Outline Tool, via Material Properties below, for this Material there will be a small green tool on the Materials icon. This lets you know that a Tool override is present on the Material. Clicking this option will remove that override allowing the S2M CENTER to select the proper Outline Tool based on your Tool Catalog settings.

    3.  Remove Override - If you have modified the Optimization/Nesting Properties for a specific Material either from this menu or from the Patterns Area sidebar there will be a check mark on the Materials icon. This lets you know that an override is present on the Material. Clicking this command will remove that override allowing the Material to be Optimized/Nested utilizing the Machines default properties.

    4.  Properties - Clicking the Properties command will bring up the optimization/nesting properties for the selected material. This works like an override. The Properties you change here will be for the selected Material only. This will allow you to re-optimize with different results and not adversely affect the other materials for this PNC file.
