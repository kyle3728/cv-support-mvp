Render Mode

There are three Render Mode options:

Wire Mode

The Wire Mode option allows you to view your Pattern in a line (wire frame) view.

Fill Mode

The Fill Mode option allows you to view your Pattern in a filled (basic color) view.

The green color represents offcut/waste. The blue color represents trim. The Brown color represents the components being Optimized/Nested.

Texture Mode

The Render Mode option allows you to view your Pattern in a render (color's and textures) view.

The green color represents offcut/waste. The blue color represents trim. The Brown color represents the components being Optimized/Nested.
