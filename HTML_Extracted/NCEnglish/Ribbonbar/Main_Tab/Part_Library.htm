Part Library

The Add Library Parts option allows you to add Parts you have saved into your Part Library into your current Job. To Add a Part simply highlight it in the list and click the Add button. There are several other options here.

Toolbar

There are 4 options in the toolbar:

  1. New - Creates a new Folder.

  2. Copy - Copies the selected folder and its contents.

  3. Rename - Allows you to rename the selected folder.

  4. Delete - Deletes the selected Folder. Be careful once a folder is deleted it can not be restored.

Other Options

  • Quantity - Prior to clicking Add you can set a Quantity for the currently selected Part.

  • Overmake - This field allows you to set how many extra of these Parts can be made if they will fit on a sheet. It will try to get these without a adding an additional sheet to your Nest. If an additional Sheet is required to get them it will not produce them.

  • Material - Select which Material you want the selected part out of.

  • Orientation - If the Material you selected is defined as a one sided Material this option allows you to tell the S2M CENTER which Face you want the Primary Operations performed on.
