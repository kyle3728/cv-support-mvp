TASK Version 5

This is a Parts Router and/or Point-to-Point Link.

TASK version 5 supports both Tool Compensation for routers and Tool Numbers for saws and routers. All saws should be set to output Center Line paths. Feed Rate, Layer Name & Pen Number from the Tool Catalog will be ignored. Tool selection will be made by Tool Number.  If the Tool Number is 0, then tool selection is made by a diameter match between the tool and the operation. Since some tools are selected by matching the width of the tool and the operation, the output unit should match that of your machines tool catalog. This is typically metric.
