TwinCAM 32

This is a Parts Router and/or Point-to-Point Link.

TwinCAM 32 does not support Tool Compensation for saws.  All saw tools should be set to output Center Line paths. Routers can be output with tool compensation. The following information from the Tool Catalog can not be used by TwinCAM and is therefore ignored: Feed Rate, Descent Rate, Maximum Depth Per Pass. Pen and Layer names will be generated automatically to the default settings:

  1. Panel Dimensions: PL   (White)

  2. Routing: RT_TT + Tool Number + Tool Comp   (Red)

  3. Grooving Operations:  RT_TT90   (Yellow)

  4. Vertical Boring: VD_TT1   (Blue)

  5. Vertical Through Boring: VD_TT2   (Blue)

  6. Horizontal Boring: H_TT1   (Green)

Tool selection for routing is made by Tool Number. Be sure to have accurate tool numbers for all routers in the Tool Catalog. Tool selection for boring is made by a size match between the tool and the operation.
