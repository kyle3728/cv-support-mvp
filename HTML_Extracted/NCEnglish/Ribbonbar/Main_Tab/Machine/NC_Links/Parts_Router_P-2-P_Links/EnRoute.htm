EnRoute

This is a Parts Router and/or Point-to-Point Link.

Use this output with Scanvec EnRoute.  EnRoute supports tool compensation for routers and saws. The following information from the tool catalog can not be used by EnRoute and is therefore ignored: Descent rate and maximum depth per pass.  Layer names from the tool catalog are ignored and will be generated automatically to the default settings:

     Panel Dimensions: BD

     Outline: OT_T + Tool Number + D + Tool Diameter + F + Feed Rate + Tool Comp

     Routing: RT_T + Tool Number + D + Tool Diameter + F + Feed Rate + Tool Comp

     Grooving: SW_T + Tool Number

     Vertical Boring: VF_T + Tool Number

     Through Boring: VT_T + Tool Number

     Horizontal Boring On Left: HL_T + Tool Number
