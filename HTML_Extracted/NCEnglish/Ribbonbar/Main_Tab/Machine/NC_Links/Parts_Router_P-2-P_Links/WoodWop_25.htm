WoodWop 25

This is a Parts Router and/or Point-to-Point Link.

WoodWop version 2.5 does not support Tool Compensation. All router and saw tools should be set to output Center Line paths. The following information from the Tool Catalog can not be used by WoodWop 2.5 and is therefore ignored: Tool Compensation, Feed Rate, Descent Rate, Maximum Depth Per Pass. Layer names will be generated automatically to the default settings:

  1. Panel Dimensions: S_ + Part Thickness

  2. Routing: FZ + Distance Off Table + T + Tool Number

  3. Grooving Operations:  NZ + Depth

  4. Vertical Boring: ZG + Depth

  5. Vertical Through Boring: ZD + Depth

  6. Horizontal Boring: HO + Z Position

Tool selection for routing is made by Tool Number. Be sure to have accurate tool numbers for all routers in the Tool Catalog. Tool selection for boring is made by a size match between the tool and the operation. This output will create a JobName_25.bat file that can be used to automate the process of running BPP and converting the DXF files to the WoodWop file format.
