Auto Link

This is a Parts Router and/or Point-to-Point Link.

AutoLink supports tool compensation for routers, Tool Numbers, Feed Rate, Descent Rate & Maximum Depth Per Pass.

Saws should be set to output Center Line paths.  One Saw may be setup with multiple kerf sizes.  If a tool exists in the Tool Catalog with a kerf that matches the dado width the Planit CNC Center will output a single polyline with width. The kerf in AutoLink’s configuration setup will determine the number of passes the saw takes to create the dado width.

Tool selection, for all tools, is made by Tool Number.  If the Tool Number is 0, then AutoLink will attempt to do a tool match automatically.

Each Tool is interrogated for it’s maximum depth for a single pass.  If an operation exceeds the maximum depth per pass, multiple passes will be generated automatically.

Routes are output different for Open and Closed Paths. Open paths are output as line segments. Closed paths are output as polylines. This enables a feature to be used with AutoLink where it splits the distance of the 1st line in a polyline and starts the cut here. This eliminates a common problem of overshot in the corners. Closed Path Route Output adds _RI_RO to the layer name to communicate to AutoLink to synthesize the Lead In / Out  and Ramp In / Out.

A part description and dimension has been added to a description layer. This description will show next to the file name on the controller for easy identification of the part. The batch.lst is generated so that AutoLink can process the files automatically.

Layer Name & Pen Number from the Tool Catalog will be ignored. The DXF files created for AutoLink have the following characteristics:

  1. Border: Border + Part Thickness + LL orientation

  2. Description: Stores part information.  This information will appear in the comment section of Winner90 and Genesis

  3. Vertical Boring:  Vbore + Depth + Tool Number (If not 0) + Multiple passes

  4. Horizontal Boring: Hbore + Z position + Tool number (If not 0) + Multiple passes

  5. Grooving: Groove + Depth + Tool Number (If not 0) + Multiple passes

  6. Routing: Route + Depth + Tool number (If not 0) + Tool compensation + Feed Rate (If not 0) + Descent Rate (If not 0) + Multiple passes + Ramp In/Out (For closed path routing)

Each unique layer gets a consecutive pen number.

All entities are output at 0z with No Thickness.

A _Mn (where n is the number of passes) is added to the layer name automatically to process the instruction with multiple passes.

Parts are automatically output in the Top Left quadrant + X , +Y to match the interface in AutoLink.

Dimensions in the layer name use p for a decimal place. They output a maximum of 2 places in the layer name for metric and 5 places for imperial .

Horizontal boring is created with 3 line segments open to the edge of the part the hole will be generated on. Warning messages will be produced for horizontal boring that is not on the edge of part.

Tool Number, Feed Rate, Descent Rate & Tool Compensation from the Tool Catalog - will only be used in the layer name if they are not 0.
