Alphacam P2P

This is a Parts Router and/or Point-to-Point Link.

Use this output when using Alphacam with a Point to Point machine.  Alphacam supports tool compensation for routers and saws. The following information from the Tool Catalog can not be used by Alphacam and is therefore ignored: Descent rate and Maximum Depth Per Pass.  Layer names from the Tool Catalog are ignored and will be generated automatically to the default settings:

  1. Panel Dimensions: PL + Part Thickness

  2. Routing: RT_Z + Depth + _T + Tool Number + _D + Tool Diameter + _F + Feed Rate + Tool Comp

  3. Grooving Operations: SW_Z + Depth + _T + Tool Number + _D + Kerf + _F + Feed Rate + Tool Comp

  4. Vertical Boring: VD_Z + Depth + _T + Tool Number + _D + Tool Diameter

  5. Horizontal Boring On Left: HL_Z + Z Position + _T + Tool Number + _D + Tool Size

  6. Horizontal Boring On Right: HR_Z + Z Position + _T + Tool Number + _D + Tool Size

  7. Horizontal Boring On Top: HT_Z + Z Position + _T + Tool Number + _D + Tool Size

  8. Horizontal Boring On Bottom: HB_Z + Z Position + _T + Tool Number + _D + Tool Size

Tool selection for routing is made by Tool Number.  Be sure to have accurate tool numbers for all Flat and Shaped Router Bits in the Tool Catalog.
