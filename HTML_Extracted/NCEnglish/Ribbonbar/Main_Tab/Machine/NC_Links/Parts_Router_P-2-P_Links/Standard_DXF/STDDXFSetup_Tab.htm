Standard DXF Machine Catalog Setup Tab

Use this tab to customize a DXF file for your particular CAM package. The Standard DXF is available for both Point to Point machines and Routers.

There are several options available when setting up the Standard DXF Link for a Point to Point and/or Parts Router.

DXF - Layer Format Group

  • Board Layer Name - Define the name of the Layer representing the rough size or border of the part being cut.

You can use text variables to define your Layer Name.

Zone/Run Field

  • Border Color # - Enter the color number you would like to use for Board's Layer.

  • # Decimals in Layer Name - Enter the number of decimal places you would like exported in the Layer names to your CAM package.

  • Output "_" for Decimal Point - A "Yes" here will output an underscore "_" to represent the decimal place, a "No" here will output a letter "p" to represent the decimal place.

DXF - Geometry Group

  • Break Polylines into Lines - A "Yes" here will break route operations into separate lines, a "No" will output the route operations as a continuous polyline.

  • Rectangles for Side Bores - A "Yes" here will output Horizontal boring as rectangles, a "No" will output them as circles.

  • Use Thick on Rect Side Bore For Z Position - A "Yes" here will use the Thickness of the Rectangle Side Bore for the Z Position of the side bore.

  • Apply Depths to Output - A "Yes" here will output your DXF's with depths, a "No" will output a 2D representation of your machining.

  • Set Base of Board as 0 Z - A "Yes" here will set the base of your outputted board to a Z of 0 (Mic @ Spoil Board), a "No" here will set the top of your outputted board to the top of your material.

  • Set Bottom Edge as 0 Y - A "Yes" here will set the bottom edge of your outputted parts at a Y of 0, a "No" will set the top edge to 0.
