Cam 3

This is a Parts Router and/or Point-to-Point Link.

Biesse Cam3 DXF files do not support tool compensation. All router and saw tools should be set to output Center Line paths. The following information from the Tool Catalog can not be used by Cam3 and is therefore ignored: Tool Numbers, Tool Compensation, Feed Rate, Descent Rate, Maximum Depth Per Pass. Layer names will be generated automatically to the default settings:

  1. Panel dimensions: PANEL + Depth

  2. Vertical drills: VERTICAL + Depth

  3. Horizontal drills on left side: SIDEA + Z Position

  4. Horizontal drills on front side: SIDEB + Z Position

  5. Horizontal drills on right side: SIDEC + Z Position

  6. Horizontal drills on back side: SIDED + Z Position

  7. Grooving operations: BLADE + Depth

  8. Generic routing operations: ROUTER + Depth

Tool selection is made by a size match between the tool and the operation. To differentiate between multiple routers bits that have the same diameter, enter slightly different tool diameters for each tool and have those diameters match the tool setup in Cam3.
