Biesse Rover

This is a Parts Router and/or Point-to-Point Link.

On the NC4x x only, starting from the release 4.24, it is possible to add more information to the DXF. These DXF files will support tool compensation for routers and selection of routers by tool names. This output does not support tool compensation for saws.  All saw tools should be set to output Center Line paths. The following information from the Tool Catalog can not be used in this output and are therefore ignored: Tool Numbers, Feed Rate, Descent Rate, Maximum Depth Per Pass. Layer names will be ignored for all tools except routers. Router tools should have a layer name that matches the tool name on the machine. The following are the standard layer names:

  1. Panel dimensions: PANEL + Part Thickness

  2. Vertical drills: VERTICAL + Depth

  3. Horizontal drills on left side: SIDEA + Z Position

  4. Horizontal drills on front side: SIDEB + Z Position

  5. Horizontal drills on right side: SIDEC + Z Position

  6. Horizontal drills on back side: SIDED + Z Position

  7. Grooving operations: BLADE + Depth

  8. Generic routing operations: Layer Name + Depth + Tool Comp + Lead In/Out

Tool selection, for all tools but routers, is made by a size match between the tool and the operation. Lead In / Out will be added based on tool compensation and if the router operation is on the interior or exterior of the part.
