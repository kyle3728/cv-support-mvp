TASK Version 3

This is a Parts Router and/or Point-to-Point Link.

TASK version 3 and earlier do not support Tool Compensation or Tool Number. All saws and routers should be set to output Center Line paths. Feed Rate, Tool Number, Tool Compensation, Layer Name & Pen Number from the Tool Catalog will be ignored. Tool selection is made by a diameter match between the tool and the operation. Since some tools are selected by matching the width of the tool and the operation, the output unit should match that of your machines tool catalog. This is typically metric.
