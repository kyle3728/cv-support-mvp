WoodWop 5

This is a Parts Router and/or Point-to-Point Link.

This link does not support tool compensation for saws.  All saw tools should be set to output center line paths.  Routers can be output with tool compensation.  The following information from the tool catalog can not be used by this link and is therefore ignored: Feed rate, descent rate, maximum depth per pass.  

Layer names will be generated automatically to the default settings:

     Panel Dimensions: ProcPart_ + Part Thickness

     Routing: V_Trim_ + Distance Off Table + T + Tool Number + Tool Comp

     Grooving Operations:  V_Saw0_ + Depth

     Vertical Boring: V_DrillSF_ + Depth

     Vertical Through Boring: V_DrillSFS

     Horizontal Boring: H_Drill_ + Z Position

Tool selection for routing is made by tool number.  Be sure to have accurate tool numbers for all routers in the tool catalog.  Tool selection for boring is made by a size match between the tool and the operation. This output will create a RunNumber_50.bat file that can be used to automate the process of running BPP and converting the DXF files to the WoodWop file format.
