Real time Label Printing Using a S2M Center CPOUT file and a Label picture

Basic setup

To implement this on a CUT 85 saw the design software needs to produce 2 items. These items are a CPOUT file and label pictures. The CPOUT files should be placed in the C:\CPOutPfad folder, and the label pictures in the C:\CV_Pictures folder. Make these folders if they do not exist. Next check that the saw is set to look at the CPOUT folder as in the example below (figure 1). The folders can have any name you want, you just need to enter the proper names in the proper places.

Figure 1 - CPOUT file location

Label design

The next item to do is to design the label file at the saw. Set the label size by selecting “F5 Size”. Enter the label width and length to match the S2M Center Label Designer label size. In “Line definition” set “Type” to “Variable picture”. Set “X-Pos” and “Y-Pos” to “0.000” and width and height to match the label size. The next item “Variable:” needs to match the location of the picture file name in the CPOUT file. Use “14 Element identifier” if the value is in the “PRT1 partno” field. Use “62 Element info 1” if the value is in the “PRT1 drillpat” field. Set “Format:” to 8 to strip out any extra characters. Last set “Path:” to your selected picture folder. Also add “#.XXX” with the XXX being your picture file format       (*.bmp, *.ico, *.jpg, *.jpeg, *.emf, *.wmf). Below is the screen shot of the label design screen in the CUT 85 software (Figure 2).

 

Figure 2 – Label Format

Select label type

Select “F2 Label type” and double click on “Raw cut label” from the selection box (figure 3). Save the label by selecting “F3 Label…” and select “Save as…”. Type a file name that describes the label type and select “OK” (Figure 4)

Figure 4  Save label

Printer Selection

Now that the label style is set and saved, select “F7 Printer” and select the proper printer for your labels (Figure 5). Once the printer is selected Select “F3 Label…” and this time select “Save”
