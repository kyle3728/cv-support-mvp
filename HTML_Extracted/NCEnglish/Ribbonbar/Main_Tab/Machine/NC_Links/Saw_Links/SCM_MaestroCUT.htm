SCM MaestroCUT

This link will create a series of .xml files that can be used with the SCM MaestroCUT line of saws.

Following are the Link Parameters:

  1. Campo[xx] – [string]: This allows the user to define variable text that can be used for labeling in the link
  2. XMix Output Directory – [Path]: this allows the user to define a custom output directory for the .xmix files

The Campo[xx] is actually a series of parameters that start at Campo22 and end at Campo64.  The user can enter variable text and literal text, or even a combination of the two.  For example:

Part Size: {WidthString} x {LengthString}

This could output the following text:

Part Size: 6 x 21

Following are the valid Variables to use:

Each variable should be enclosed in a set of curly braces ({ }) otherwise the text will be considered a string literal.
