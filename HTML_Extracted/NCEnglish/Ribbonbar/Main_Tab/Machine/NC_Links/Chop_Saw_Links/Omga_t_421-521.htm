Omga T 421/521

This is a Chop Saw Machine Link.

Only selected parts made of board stock are output.

 

File Protocol T 421/521 OPT

The Structure must be the following

  •  ‘S’ ; = start transmission

    1. ‘P’ ; = cycle number (max 3 characters)

    2. ‘:’ ; = (optional line) id , cycle alphanumeric name (max 8 characters)

    3. ‘X’ ; = (optional line) n° box ( max 3 characters)

    4. ‘Y’ ; = (if first grade, optional line; if omitted, grade =1) grade values = only 1 or 2

    5. ‘Q’...’T’ ; = length (max 3 numbers+ 3 after the decimal sign), number of repetitions (Limit of 500 Records per File)

  • ‘E’ ; = end transmission

The cycles sent as per above will be written in the programs memory with the same rules as function FUN1; therefore no cycle number grater than the next cycle available will be accepted.

By transmitting cycle number already present in the controller, the cycle on the controller will be overwritten.

IMPORTANT:

Because of programming restrictions it is necessary to:

  1. close each line with a semi-colon’;’

  2. start the line with the character identifying the data that will follow (see above)

  3. use only UPPER-CASE characters for the letters preceding the data on each line

Example of ASCII file with unit of measure Inches (controller memory empty) :

S

;start transmission

 

 

P1

;cycle #1

:first

;id = ‘first’

X1

;box # =1

Y1

;grade 1

Q25.125 T5

;piece length =25 1/8” , Number of pieces (repetitions) 5

 

 

P2

;cycle #2

Q12.500 T8

;piece length =12 1/2” , Number of pieces (repetitions) 8

 

 

P2

;cycle #2

Y2

;grade 2

Q10.750 T10

;piece length =10 3/4” , Number of pieces (repetitions) 10

 

 

P3

;cycle #3

Q30.750 T12

;piece length =30 3/4” , Number of pieces (repetitions) 12

Q15.500 T5

;piece length =15 1/2” , Number of pieces (repetitions) 5

 

 

E

;end transmission

Note 1: new cycles begin each time the Material or Width changes

Note 2: in order to make the data sequence more understandable, it is allowed to:

  • insert blank lines in between the data lines

  • insert spaces (only after the numbers and not the TAB character)

  • insert comments on each line after the semi-colon ’;’ character

Note 3: the unit of measures will be the one selected on the controller with the ‘UN_MIS’ parameter

Note 4: by not transmitting the optional lines those field will be set to ZERO (i.e. ID, width, box, grade).
