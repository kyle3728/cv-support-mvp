Koch Maxi 5

This is a Chop Saw Machine Link.

Output for Koch/Paul Maxi 5 Premium Control.

Only selected parts made of board stock are output.

 

Structure of Cutting List To Be Imported into MAXI 5 Premium Control

The file to be imported must start with the order details comprising 9 lines, each with a maximum of 100 characters. This line will start with #; and end with ;

Then follow the individual lists.

Each list may contain a 'project' line with a maximum of 100 characters. The project name will start with #; and end with ;

Each list may contain max. 400 sizes.

Data have to be in ASCII code, i.e. ASCII characters, hex 20 to hex 7E. Not allowed are: $ , @ ,\ .

Decimals to be separated by a point, not by a comma, each line to end with <CRLF> (Carriage return / line feed).

File name: nnnnnnnnnnnn.stk . Any name with the ending stk.

Line structure:

P xx; Active; Control; Length; Number of Pieces; Sorting Station; Priority; Printing Code; <CRLF>

Thereby:

P xx: Cutting list number 1 .. 60

Active: If the lengths in this line are to be used, set this value to 1, otherwise to 0

Control: For special applications, always set to 1

Length: in millimeters with one decimal place, e.g. 1000.2 mm, min. 20.0 mm, max. 7000.0 mm

Number of pcs. 1 to 5 000000

Sorting station: 1 to 48

Priority: 1 to 300 in %, standard value is 100%

Printing code: If not used, then send “ ;“ (2 blanks and semicolon).

Printing code may comprise max. 30 ASCII characters (no semicolon, no @ within the character string)
