RazorGage

This is a Chop Saw Machine Link.

Only parts made of board stock are output.  Output can include scribe marks for positioning mid stiles and rails.

If you are using Solid to output to the S2M Center and you need the output for Scribe Marks included you will want to download the RazorGage UCS/Intelli-Joint Package from the Downloads section of eSupport.

Razor Gauge File Format

The following is the field information for the Razor Gauge Cutoff Saw.

Material Name

Material ID

Part Quantity

Part Name

Part Width

Part Length

Part Thickness

Room Name

Room ID

Cabinet Name

Cabinet ID

Scribe Marks

The scribe marks are identified in a string formatted using the pipe symbol “|” as the delimiter:

A part that needs scribe marks at 10, 12, 12.5, 18, and 19.75 would be output as:

‘’10 | 2| .5 | 5.5 | 1.75’’

A part that had no scribe mark would be “ “

Currently the output is restricted to imperial

The data in the file is already sorted by:

  1. Material

  2. Part Width (descending)

  3. Part Length (descending)

Materials that are not board stock are not filtered out.
