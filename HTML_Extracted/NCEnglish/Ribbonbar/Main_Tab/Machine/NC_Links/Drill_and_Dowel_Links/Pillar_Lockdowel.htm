Pillar Lockdowel

This is a Drill and Dowel Machine Link.

Output for the Pillar Lockdowel machine.

When a Tool in the Tool Catalog is assigned to this machine you will get two Parameters:

  • Drill (_hmInsert) - When set to Yes indicates the tool can be used to Drill.

  • Insert (_hmDrill) - When set to Yes indicated the tool can be used to Insert.
