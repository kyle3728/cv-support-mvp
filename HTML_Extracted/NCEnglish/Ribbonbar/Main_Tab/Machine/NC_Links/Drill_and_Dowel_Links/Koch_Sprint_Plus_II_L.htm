Koch Sprint Plus II/L

This is a Drill & Dowel Link

Has the following machine parameters:

  • One program per part  (Boolean) - Output one program per part rather than per edge

  • Use part len as prog len (Boolean) - Use part’s length as program length rather than edge against the fence.  Without this set L & R’s length = part width

  • Display RPM (Boolean) - Display RPM (Spindle Speed) in output.  Without this, the spindle speed is set to 0

  • Display Operation Depth (Boolean) - Display operation depth in output.  Without this, the depth is set to 0

Has the following tool parameters:

  • Canned Cycle Number (string) - the canned cycle number to use with this tool

  • Operation Type (string) - The type of operation that this tool represents
