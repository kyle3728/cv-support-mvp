T.O.P. Albatros

This is a Drill and Dowel Machine Link.

Output for the T.O.P Drill & Dowel machine.

When a Tool in the Tool Catalog is assigned to this machine you will get the following Parameters:

  • Drill (_hmDrill) –  Allows you to specify that this tool should perform a drill operation

  • Insert (_hmInsert) – Allows you to specify that this tool should perform an insertion operation

  • Macro (_hmMacro) -  Macro name to use for the insertion operation
