This is a Drill and Dowel Machine Link.

This output will create program files for Drill and Dowel Machines using TpaCAD. The output supports Horizontal Drill / Glue / Dowel and Vertical Boring. Assign tools to this machine and to the boring operations to create output. Additional instructions such as dowel, glue, macro name, programmable depth, and tool number can be set on tools assigned to this machine in tool catalog. One program will be created per edge.

If you need to use the pocket screw attachment for this link, create a new horizontal tool, then assign it to the machine using this link, then set the Macro property to pk.tcn (this is case insenstivite). When that is done, the neccessary changes to the output will be made when the pocket screw attachment is needed.

PSHOWTOOLPROP_ToolNumber

Link Parameters

  • Add Glue - (_hmGlue) [Bool/Tool]: Add glue when using this tool
  • Add Dowel - (_hmDowel) [Bool/Tool]: Add dowel when using this tool
  • Macro - (_hmMacro) [String/Tool]: The macro name to use for Glue/Dowel operation: forainsert.tmcr
  • Base zone # on Part Size - (_hmZoneOnPrtSize) [Bool/Machine]: Option to uset he smaller inside zones if the part will fit
  • Dimension 'A' - (_hmMaxSizeForZone) [Measurement/Machine]: Maximum size for the small zone
  • Programmable Depth - (_hmProgDepth) [Bool/Tool]: When true, this will output operation depth for this tool, when false, depth will equal 0
