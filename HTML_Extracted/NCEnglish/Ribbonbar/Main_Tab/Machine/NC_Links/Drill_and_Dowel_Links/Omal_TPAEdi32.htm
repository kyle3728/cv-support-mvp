Omal TPAEdi32

This is a Drill and Dowel Machine Link.

This output will create program files for Omal using TpaEdi32.  The output support Horizontal Drill / Glue / Dowel and Vertical Boring.  Assign tools to this machine and to the boring operations to create output.  Additional instructions such as dowel, glue and macro name can be set on tools assigned to this machine in tool catalog.  One program will be created per edge.
