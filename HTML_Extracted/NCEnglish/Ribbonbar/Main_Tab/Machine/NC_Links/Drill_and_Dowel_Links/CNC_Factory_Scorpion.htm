CNC Factory Scorpion

This is a Drill and Dowel Machine Link.

This output will create program files for the CNC Factory Scorpion CNC Drill & Dowel Machine.  Assign tools to this machine to create the correct output.  In the tool parameters you can provide information such as tool height offset id, tool number, whether or not the tool is an insertion tool, as well as the insertion depth, insertion offset id, and the insertion gun offset.

When assigning a Tool to a CNC Factory Scorpion Machine you will get the following additional Tool options:

  • Tool Number
  • Height Offset ID
  • Descent Rate

They will also have the following Parameters added:

  • Insert Dowel (Yes/No) - Denotes this tool as an insertion tool
  • Insert Depth (Measurement) - Provides the insertion Depth for this tool
  • Insert Offset (Integer) - The insertion offset id for the tool
  • Gun Offset (Integer) - The gun insertion offset id
  • Safe Rapid Level (Measurement) - Provide safe z level for rapid movement
