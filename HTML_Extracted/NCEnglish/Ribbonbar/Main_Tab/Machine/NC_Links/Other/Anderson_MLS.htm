Anderson MLS

This link will create a .csv file, for each material used, that the Anderson Material List can read to hep with loading of materials onto a machine.  All materials that are to be used will need to have a unique ID assigned to them in the Material parameters of CABINET VISION.  By default the name of the ID parameter is _mhid, but you can change this to whatever you want it to be and then just define what the parameter name should be in this links output section.

If you do not provide a material ID, <invalid> will appear in the file instead
