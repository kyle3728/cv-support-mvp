This part of the topic is a description of the above:

Line beginning with AssembledDoor.bpp,1,0 (is the Name of the assembled Door but has to have the .bpp,1,0)

Followed by the finished size of the door and the profile name for the outside edge, followed by the centerline of each hingecup assembly and hinge name, then handle/knob type/name and X Y location of center line of handle, then material name. All of descriptions below per the sample above are separated by spaces.

Assembled door operation

  • Size & profile outside edges (same tool)
  • Sand profile
  • Drill for hinges (custom locations for base doors)
  • Drill for door pulls
  • Drill for bumper pads (not necessary)

The next line contains the Panel Details.

Door panel operation (before assembly)

  • Size panel edges
  • Profile outside edges (multiple passes depending on wood species)
  • Sand  profile (multiple passes depending on wood species)

The last line contains the Top Rail Shape which is not currently supported by the S2M Center.
