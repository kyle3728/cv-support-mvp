Material Handling

This Machine Type is specifically for automated Material Handling Systems. Following are the Output Tab options for Material Handlers:

Output Tab - Material Handling

To change information in these fields, highlight the field name at the left. Then click on the data at the right and enter the new values. Some of the options below require that you select a specific Machine prior to seeing the option, Offcut Import File is an example, and it appears on the Setup tab only after selecting the Winstore NC Link.

Import Group

  • Import File - Utilize the Browse button in the field to set the Import File and its location for your Material Handling system.

  • Auto Sync on Optimize - If this is set true the Sync will occur each time you Create CNC / Optimize. If false the you would need to click Add Offcuts to Sync the offcuts.

  • Create Import File Command - Option for the Intellistore link.

  • Offcut Import File - Utilize the Browse button in the field to set the Offcut Import File and its location for your Material Handling system.

User Interface

Once you have the Machine(s) setup you still have to turn the system on. This is done in the Offcut Manager:

Simply select your Material Handling System and enable the Sync with option. You can test the link by clicking the Sync Now button.

If Auto Sync is disabled you will need to click the Add Offcuts sidebar option prior to clicking Optimize.

If Auto Sync is enabled simply click the Optimize sidebar option and the Offcuts are automatically added.

If the import file can not be found you will be prompted to select a file.
