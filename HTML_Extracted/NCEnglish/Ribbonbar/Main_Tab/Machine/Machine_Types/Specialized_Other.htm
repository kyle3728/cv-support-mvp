Specialized Other

This Machine Type is specifically for other woodworking machines which do not fit in the other Machine Type Categories. Following are the Setup Tab options for Specialized Chop Saw:

Setup Tab - Other

Parameters Group

  • Barcode Format - This option will appear when an Accu-Systems MTH or Weinig French Miter NC Link is selected. See Sheet Router for information on this field.

  • Tenon Thickness Clearance - This option will appear when an Accu-Systems Universal NC Link is selected. Sets the clearance required for the Tenon Thickness.

  • Tenon Width Clearance - This option will appear when an Accu-Systems Universal NC Link is selected. Sets the clearance required for the Tenon Width.

  • ID Parameter Name - This option will appear when the Anderson MLS NC Link is selected. This allows you to specify the name of the unique material id parameter that will identify this material to the lift system.

  • Swap A/B Notches - This option will appear when the Hoffman Beaded Frame System NC Link is selected. Allows you to swap the A/B notches.

  • Mirror Output - This option will appear when the Hoffman Beaded Frame System NC Link is selected. Enables Mirrored Output.

  • Feed Speed - This option will appear when the Homag Edgebander NC Link is selected. Allows you to set the machine feed speed.

  • Oversize - This option will appear when the Homag Edgebander NC Link is selected. Allows you to set the oversize value.

  • Glue Type - This option will appear when the Homag Edgebander NC Link is selected. Allows you to set the glue type.
