Specialized Drill and Dowel

This Machine Type is specifically for Drill and Dowel Machines. Following are the Setup Tab options for Specialized Drill and Dowel:

Setup Tab - Specialized Drill & Dowel

The fields defined below will only be available if your Drill & Dowel supports the feature the field defines.

Depth Match Group

  • Face Up - This setting will force the program to be written either face up or down regardless of if the part is face up/down on the nest or saw.

You can override the up/down flag on an individual part with the DNDFLIP parameter.

Face Up with Label-IT

Below is sample output for 2 different edge machines. Each machine has a different Face Up setting.

The arrows will indicate how the part should be fed into the machine and since this part was on the saw Face Down that the Face Up program(s) require the part to be flipped.

When two programs share the same name but different Face Up/Down requirements then multiple arrows will be shown to accommodate. The arrows will coincide with the machine tags field.

Note that when this part is BU (back up) it has a Left and Bottom program. When it is flipped Face Up, it has a Right and Bottom program. The L,R,B,T programs are always relative to the primary edge of the part and if the parts is Face Up/Down.

In the past the Label was always rotated so that the Arrow pointed to the primary edge of the part. However there was a disconnect with this rotation and how the part appears on the Label in Label-IT. In Label-IT the part is always rotated 90 degrees since parts are typically longer then they are wide to maximize the part image.

Now the label is always rotated so that it points to a part length.

In the image depicted below we have the same part with 4 different primary edges. The primary edge corresponds with the part number. In the past the 2nd parts label would point to the right and the 4th parts label would point left. In the new scheme the part with the primary edge of 2 or 4 have the same rotation as a part with 1 or 3 respectively.

Note that the position of the label has not changed, only the rotation.

Silent Mirrored Programs

If a part is shaped there may be difficulty lining up the part on the fence of the machine on its shaped side. We now output a silent mirrored program for all shaped parts. The mirrored program can be run on the opposite fence if needed.

Note that the mirrored program will still have the same Face Up/Down orientation as the primary program. In essence only the X-Position is mirrored (not the Z).

These additional programs will only occur for shaped parts.

The Barcodes on the Label Image will be for the primary edge program. The operator will need to be trained/informed that there is an additional program if necessary.

The edge program has a new naming convention:

Previously

R010011L (Run Number R01) (PartID 0011) (Edge L = Left)

New

R01P00BL (Run Number R01) (Primary or Mirrored P/M) (PartID in base 36 00B) (Edge L = Left)

R01M00BL (Run Number R01) (Primary or Mirrored P/M) (PartID in base 36 00B) (Edge L = Left)

With Base 36, the three 3 characters reserved for PartID can now store 46,655 unique IDs from 000 to ZZZ

If the part is shaped the program will output a R01M* program in addition to the R01P*

All D & D output except for the Weeke BEK that describes the entire part now feature the mirrored programs.

Match Depth Explanation

In the grid below I have defined 4 different Match Depths (Triggers) for each Operation Type. The dimension used here are a bit extreme. You would probably define them with about .001” difference when you actually utilize it.

Drill Match Depth

0.125

Dowel Match Depth

0.25

KD Match Depth

0.375

Glue Match Depth

0.5

In this situation if I passed a Horizontal Bore Operation through the S2M Center that was 1/8” Deep the Machine would Drill Only.

If I passed a ¼” Deep Operation it would Drill and Insert a Dowel. This would be used for inserting glue-less dowels.

If I passed a 3/8” Deep Operation it would Drill and Insert a KD Fitting (If your Machine supports it, otherwise this field will not be available.)

If I passed a ½” Deep Operation it would Drill, Inject Glue, and Insert a Dowel.
