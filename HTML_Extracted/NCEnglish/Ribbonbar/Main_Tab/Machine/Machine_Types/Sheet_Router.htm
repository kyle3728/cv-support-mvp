Sheet Router

This Machine Type is specifically for Nested based Routers. Following are the Output Tab options for Sheet Routers:

Output Tab - Sheet Router

To change information in these fields, highlight the field name at the left.  Then click on the data at the right and enter the new values.  Some of the options below require that you select a Machine prior to seeing the option, Small Parts is an example, and it appears on the second tab after selection of a machine.

Intelli-CAM - Post Group

  • Post Processor - Utilize the Browse button in the field to set the Post and its location for your Machine.

Intelli-CAM - Rapid Group

  • Safe Rapid Level - The safe distance above the material the tool will move to the next operation.

  • Rapid Down To - When drilling you can retract to safe level or this level.   The point where the rapid, between operation movements, of the machine slows to or approaches the machining speed.

The Safe Rapid and Rapid Down To values must be different if even only slightly. If the values are the same you will get either Errors or at the very least undesirable results.

  • Apply Comp on Rapid - Only works on some machines. This value instructs the Machine to perform its Cutter Compensation during the safe rapid move.

  • Drill traverse at safe rapid - See Safe Rapid

  • File Extension - Enter the File Extension your Machine requires, here.

  • Save Drawing File - Setting this to True will create an .ARD file for each G-Code file created for any given Job. ARD files are native ALPHACAM Router Drawing files.

  • Output Comment Character(s) - The characters representing comments in the G-Code output.

Intelli-CAM - Tool Movement

  • Compensation Type - (Compensation refers to an adjustment in the path of the tool to compensate for a change in tool size due to tool wear.  This is referred to as Tool Comp. or Tool Compensation). See the document on Output Path for more information.

    • Tool Center No G41/G42 - Tool Comp is accommodated strictly though the tool path.  It is offset from the actual cut path and doesn't look to a register in the machine's control to adjust the path. The outline path of the part is larger than the part's geometry by the tool radius. This path is to be followed by the center of the tool.

    • Tool Center G41/G42 - The outline path is identical to Tool Center no G41/G42. The difference is it triggers to look up tool compensation values on the machine's control. Tool comp is important when you start sharpening tools. When you sharpen a tool it gets smaller. Tool comp allows you to inform the control that the bit has changed size and to compensate for the change. For example: If you use a ½” router bit (.25 radius) and as a result of the bit being sharpened it has a new size (.23 radius). You need to account for this tool size change somewhere.

      • With Centerline - Tool Center no G41/G42 you would go back to the Tool Catalog in the S2M Center and enter the new size of the tool (.46 diameter) and re-output the program. Otherwise your part would cut out larger than it should be.

      • With Centerline - Tool Center G41/G42 you can enter a value of .02 at the machine's control (.25 - .23 = .02) and the machine's control will offset the tool path by .02 and correctly cut out the part.

      • It boils down to user preference whether the sharpened tool size is accounted for in the S2M Center or at the machine's control although you have advantages at the machine since if you need to swap bits you can compensate for different sizes on the fly rather than having to re-output from the S2M Center.

    • Tool Radius - This path follows the outline of the part exactly. Thus if the router were to follow the path centerline, the part would be too small. This path must be compensated for at the machine's control. If you are using a 1/2 bit, the machine's control will need an offset of .25 (radius).  If you sharpen the bit you would need to change the comp value at the machine's control to (.23). In either case the machine's control will offset the path of the tool by the tool radius resulting in the part being milled to the correct size.

  • Force Centerline on Open Interior Paths - Forces open Route paths inside the Part to centerline.

  • Roll Around Corners - Breaks 90 degree corners into a tiny radius.  For many controllers this will give them a hint to slow down before they reach the corner.

  • Limit Output to 3 Axis - Do not set this to No unless your machine has C-Axis capabilities.

  • Y Axis Horizontal on Workplane - Determines the orientation of the Y-Axis on the machining workplane.

  • 5 Axis Tilt Clockwise - Establishes the 5 Axis tilt direction.

  • Optimize Drill Selection - Some older machines and entry level machines can take a long time to change spindle drops on a gang drill head.  This option optimizes the tool path to minimize the number of gang drill head dropped spindle configurations.

  • Mix Hole Diameters - If enabled will mix different Drill Diameters into a spindle drop if possible based on spacing.

  • Slope Arc Lead Spiral Pocket - If enabled will perform a Spiral Lead for Pockets as illustrated.

  • Hover Tool Comp on Arc - This is for controllers that restrict tool comp during an arc.  When this option is enabled a line will be added prior to the arc above the material for compensation

  • Adjust Open Tool Path For Climb vs Conventional Cut - If enabled will adjust open tool paths to account for Climb versus Conventional cuts.

Intelli-CAM - Sheet Reference Group

  • Material Length is X - If the machine(s) longer dimension runs along the X-axis, Intelli-CAM rotates the sheet so that the sheets axis matches the machines axis.  If you require this option, set the option to TRUE (your machine sets length dimensions along the X axis).  A False setting will rotate the material from the normal X-axis for length.

  • Mic at Spoil Board - Select Yes or No here based on how you Touch Off your Tools on your Machine. A Yes here indicates that you touch your Tools off at the top of the spoil board. A No indicates you touch off your Tools at the Top of your Material.

Intelli-CAM - Offset Group

  • Approach Offset - This controls the Approach Offset away from the top of the Part Edge on Sloping Leads.

  • Offset from Material Bottom - This controls the Offset of the Tool away from the Bottom of the Part Edge on a Lead-In. This option can be used to reduce chipping of the Material on the bottom side of your Parts.

  • Add Finish Pass to Pockets - This determines whether you would like a Compensated Finish Pass to Pockets. Pockets generated through the S2M Center are Spiral Pockets with a Centerline Path and can not be compensated. You must set this option to Yes if you want a compensated final pass.

+Label - Label Group

  • Label - If you are using +Label select the Label Style you want to use with the selected Machine. Use the +Label Label Designer to create your own Label styles for selection here.

  • Offcut Label - If you are using +Label select the Label Style that you want to use for Offcuts.

  • Label Output Folder - Select the Default location for your +Label files to be placed.

  • Label Sort Order - Select the Default Sort Order for your Labels to print at the +Label station.

Sort Order is calculated based on the Part's Origin. The quickest way to identify the Part's Origin in +Label is to note the Part Number's location. The Part Number's location is at the Part Origin.

    • Bottom to Top, Left to Right - Orders the parts bottom to top, left to right.

    • Bottom to Top, Right to Left - Orders the parts bottom to top, right to left.

    • Last Out - First In - This order is a common order when cutting parts with a beam saw.  All rips in a sheet are done before any cross cuts.  The last rip cut is the first rip to feed in for cross cuts.  All cross cuts for that rip are done before any 3rd phase cuts.  The last cross cut will be the first to feed for 3rd phase cuts.

    • Left to Right, Bottom to Top - Orders the parts left to right, bottom to top.

    • Left to Right, Top to Bottom - Orders the parts left to right, top to bottom.

    • Right to Left, Bottom to Top - Orders the parts right to left, bottom to top.

    • Right to Left, Top to Bottom - Orders the parts right to left, top to bottom.

    • Rip - Cross Cut - 3rd Phase - This order is a common order when cutting parts with a table saw.  Every part in the first rip will be processed before going to the second rip.  All cross cuts will be processed in the rip prior to processing any 3rd phase cuts.

    • Standard Optimized Order - This order will match the order from the S2M Center Optimizer.  Every part in the first rip will be processed before going to second rip.  3rd phase cuts will immediately follow the preceding cross cut.

    • Top to Bottom, Left to Right - Orders the parts top to bottom, left to right.

    • Top to Bottom, Right to Left - Orders the parts top to bottom, right to left.

Nesting - Outline Group

  • Outline Start Point - This option allows you to specify the Outline start point type.

There are three options here:

    • Center - The standard S2M start point along the mid point of the edge of the Part.

    • Perimeter - An innovative S2M Center start point methodology. The concept is cutting the parts from the outside of the sheet first moving towards the center with the point of origin always being towards the center of the sheet.  Using this pattern each part is attached to a very large section of the sheet when its outline is cut so that its holding power is not just from the size of the part but the size of all other uncut parts. We have found this to be a excellent method with superior results for minimizing part movement with smaller Parts in most cases not requiring any special small Part Handling (Onion Skins, etc...). There are several other options which are available below when this Outline Start Point type is selected.

    • Corner - An alternative start point method for use on Machines where witness marks on along the start/end points are a problem. This method leads in and out at the corners of the part rather than the mid point of the edge.

  • Outline Sort Type - This option allows you to control the general order which Parts are machined in the Nest.

There are three options here:

  1. Size - Sorts by size smallest to largest. This is a good strategy to avoid Part movement making sure the smaller Parts are cut first while the maximum vacuum is available.
  2. Position - Follows a logical sequence around the sheet.
  3. S Pattern - Follows an S Pattern on the sheet.

  • Decelerate At End Of Outline - If you want the tool to decelerate at the end of the outline you can set the value (measurement) here for when the deceleration should begin. This can minimize Part shifting on the final release of the Part.

  • Percent of Feed Rate - The percentage of the current feed rate you would like to use on the decelerate above.

  • Large Part Area - Once defined, any large parts are removed from this routine, they are cut last since they are large enough to have sufficient hold down in themselves. This will increase the hold down of all other parts in the process.

  • Auto-Skin Center Parts - With this feature on, any small part or very small part will automatically get a return onion skin if it is placed in the center of the sheet. Since the center of the board is cut last these parts have a greater possibility for movement. With this feature on you can have Ramp-In/Out set for all parts types and only those parts susceptible to movement will receive a return onion skin, optimizing both part hold down and overall cycle time.

  • Link Part Outlines - This option allows you to turn on Part Bridging in your Nests. Bridging is when all Parts within the Nest are cut with a single Lead-In and Lead-Out for the Sheet. This option can improve Sheet cycle time by reducing the number of times the Tool Lead In/Out on each individual Part.

  • Link Outline Overlap Adjustment - This option allows you to adjust the Overlap where bridges occur.

Nesting - 6th Face Group

  • Allow 6th Face Output - When using this option you will be able to actually machine your Secondary (Flip Side/Face 6) operations on the reverse side of the Sheet prior to machining your Operations and Cutting out your Parts on the Primary Face. This option will use your outline tool to clean up one long edge and one short edge of your sheet (by one half of your Trim Cut) to give you a clean edge for alignment to 0/0 after flipping the Sheet to finish your nest.

When Nest Programs are output they will have one of three suffixes helping the CNC Operator identify 6th Face Output.

  2. A = The back-side (6th face) program to run before Z program.
  3. Z = Main program with nest operations and part outlines; just like N.

Nesting - Common Line Group

  • Common Line Cutting - This option will turn Common Line Cutting on or off. Common Line Cutting output tightly packs the parts in a nest so that the outline path can cut two parts with a single pass.  Although this type of output has a better yield on material and allows for a longer tool life it has its downsides.  A toolpath only has one clean edge.  Based on the tools rotation and the material climb vs. conventional cut the S2M Center instructs the tool to cut around the parts in the optimal direction.  With common line cutting there is no way to do this.  With a common cut one part is going to get a clean edge and the other will experience some blowout.  You should be working with sharp tools to minimize the blowout.  In addition common line cutting uses a centerline path with no compensation.  It is very important to have the correct tool size in the Tool Catalog to ensure the parts are the correct size.  For this reason it is recommended that common line cutting only be used for non-essential parts and less than premium Materials.  Because of this the common line settings are both per machine and per material based.

  • Common Line Maximum Retrace - This value will affect the final tool path for the outlines.  If you specify a retrace value that allows the outline path to move alongside a path that has already been cut resulting in fewer rapid moves.  You should base this preference on how fast you machine transitions from rapid to machine moves.

  • Common Line Retrace by Small Parts - If you are allowing retracing you may want to ensure that retracing does not occur near small parts since they may have shifted after being cut.  The S2M Center automatically avoids retracing near very small parts.  This setting will ensure that retracing also avoids the small parts.  Setting this value true will result in more rapid moves so it should be set to false if you are not having problems with your small parts shifting.

Small Parts - Tabs & Skin Group

  • Tabs - Tabs can be employed to help hold a "Small" or "Very Small" Part. They are areas where the router cuts at less than full depth for a short distance during the lead in thus leaving a "bridge". The lead out of the router removes this "Tab" over the area. During the time the router is cutting the Part free, the Tab provides the security that can help keep the Part from shifting.

  • Onion Skin Thickness - Allows you to set the Onion Skin Thickness when using the Onion Skin Lead Type.

  • First Pass Offset - If you are using an Onion Skin Pass this option allows you to have the first pass cut the Part oversized (rough cut) by the specified amount.

Small Parts - Lead Types

  • Lead Types - The manner in which the Tool enters and exits the operation. There are five Lead Types:

      1. None - A simple tool plunge where the bit plunges straight into the Material (least favored). Cutter Compensation will not be applied.

      1. Sloped - The most common Lead Type. This lead allows for less strain on the bit and provides cleaner machining of the Material. Sloped is also a required Lead on most Machines if Cutter Compensation (G41/G42) is being utilized. The Cutter Compensation is actually applied by the Machine during the Lead In.

      1. Tab - Tabs can be employed to help hold a "Small" or "Very Small" Part. They are areas where the router cuts at less than full depth for a short distance thus leaving a "bridge". A second pass of the router removes this "Tab" over the area. During the time the router is cutting the Part free, the Tab provides the security that can help keep the Part from shifting.

      1. Onion Skin - Typically used for small parts, the path defining the part shape is cut to a depth slightly less than the material thickness.  After the first pass, there remains a thin "Onion Skin" of material that anchors the part in place.  The router then repeats the path, this time removing the onionskin, and freeing the part.

      1. Return Onion Skin - Similar to Onion Skins, except ALL Return Onion Skin parts receive the first pass prior to any Return Onion Skin Part receives the second (or "liberating") pass.
