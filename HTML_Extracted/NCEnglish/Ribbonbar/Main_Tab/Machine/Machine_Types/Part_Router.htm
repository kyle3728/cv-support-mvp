This Machine Type is specifically for Part Routers. Following are the Setup Tab options for Part Routers:

Setup Tab - Part Router

See Sheet Router

Intelli-CAM - Rapid Group

See Sheet Router

Intelli-CAM - File Info Group

See Sheet Router

Intelli-CAM - Tool Movement Group

  • No Sloping Lead In/Outs - With this option enabled Sloping Lead Ins and Outs will not be generated.

See Sheet Router for other settings

Intelli-CAM - Sheet Reference Group

See Sheet Router

Intelli-CAM - Offset Group

  • Offset by 1/2 EDX/EDY - This option is important when using EDX/EDY to oversize a Part and the Part Router is setup to Route the full outline. Use this option to make sure the oversized amount is compensated for.

See Sheet Router for other options

Intelli-CAM - Banding Operation

  • Banding Tool - Select the Tool that applies (represents) the Banding Operation. You post-processor must be setup properly in order to use this option.

Intelli-CAM - Label Format Group

  • Label Format - Defines the formatting for your barcode as required by your machine and/or bar-code scanner manufacturer.

  • Match Programs - With Match Programs enabled the S2M Center will identify Parts which are the same and generate a single Program for them versus unique programs. This is considered a group program and will have a "g' designator within the program name as illustrated in the Machine Catalog image below.

Zone Sizes Group

  • Front and Rear Zones - Allows you to specify whether or not your Machine has both Front and Rear Zones.

  

  • Dimension 'A' - Allows you to specify the Dimension on your Machine denoted by "A" in the graphic.

 Dimension 'B' - Allows you to specify the Dimension on your Machine denoted by "B" in the graphic.

  • Dimension 'C' - Allows you to specify the Dimension on your Machine denoted by "C" in the graphic.

 Alternate Zones - Specifies whether you want the S2M Center to between Zones. For example if a Part is determined to be a Zone 1-2 Part and the next Part also is determined to be a Zone 1-2 Part, with this option on it would send the first Part to Zone 1-2 and the next to Zone 3-4 then the next to Zone 1-2. Alternating Zones allows the machine to process one part while loading another into the alternative zone. This can be a real time saver if you have a multi-zone machine.
