Specialized Chop Saw

This Machine Type is specifically for Chop Saws with automated fence systems. Following are the Setup Tab options for Specialized Chop Saw:

Setup Tab - Chop Saw

Parameters Group

  • Output for T421 SNC - Establishes if the output needs to be for a T4212 SNC when the Omga T 421/521 NC Link is selected.

  • Output Material Name - Establishes whether the Material Name should be output when the Pistorius BF-A NC Link is selected.

  • Mirror Output -  Establishes whether Mirrored Output should be produced when the Pistorius BF-A NC Link is selected.

  • Output to Hoffmann CNC - Enable this option if you are outputting to the Hoffmann CNC with the RazorGage link.

  • Swap A/B Notches - Enable this option to Swap the A/B Notches for the RazorGage link.

  • Add Part ID - Establishes whether the Part ID is output when the RazorGage NC Link is selected.

  • Add Job Description - Establishes whether the Job Description is output when the RazorGage NC Link is selected.

  • Add Program Name - Establishes whether the Program Name is output when the RazorGage NC Link is selected.

  • Strip Quotes - Enable this option to remove Quotation Marks as the Text Qualifier in output when using the TigerStop NC Link.
