Machine Catalog

You must access the Machine Catalog in order to setup your machines. The Machine Catalog has detailed graphics designed to make the setup process intuitive.

Machine Tab

Following are the tools in the Machine Catalog toolbar:

Setup Name - The name of the currently selected Machine (Setup).

Create New Machine - Allows you to Create a New Machine (Setup). There are eight possible Machine Types depending on your level and which add-ons you may have for the S2M CENTER.

    • Saw

    • Sheet Router

    • P2P

    • Part Router

    • Drill and Dowel

    • Chop Saw

    • Other

    • Machine Set

    • Material Handling

Copy Selected Machine - Allows you to Copy the currently selected Machine (Setup).

Change Machine Name - Allows you to Rename the currently selected Machine (Setup).

Delete Selected Machine - Allows you to Delete the selected Machine (Setup).

When selecting a normal (not a Machine Set) Setup you have the following possible options on the Machine tab:

Manufacturer - Use this option to specify the Manufacturer of your Machine, if not using Intelli-CAM output.

NC Link - Use this option to specify the NC Link to use with the current Machine (Setup). See the Topic on NC Links for more information on the types of Links and there use.

Tag - This is a short name for the currently selected Machine (Setup). This option helps a machine operator identify which Barcode on a Label goes with which Machine (Setup). For example the following three examples demonstrate how it is used (displayed). The tags are highlighted in yellow.

Label for one Secondary Machine

Same Barcode Sent to Two Different Machines (Machine Set)

Different Barcode to Two Different Machines (Machine Set)

Output Metric (mm) - Use this option if the Output to your Machine must be Metric (mm).

Output Summary - Determines whether the CNCRun is generated for the selected Machine.

Available - Determines whether the Machine is available for Output.

Output Directory - Specify the Directory where you want the code for the Machine output to. Utilize the Browse button in the field to set the Output Directory for your Machine. You may also use a UNC path for the Output Directory.

To create a new Machine click the "Create New Machine" tool in the toolbar .

Type in a descriptive name for the Machine and select its type.

Click the Machine Types below for Setup Tab Options

  • Type - Machine Set

  • Type - Material Handling

  • Type - Sheet (Use this type for Panel Saws and Nested Based Routers)

    • Saw - Creates a new NC Panel Saw for use with the Optimizer.

    • Router - Creates a new CNC Router. Select this option to setup an Intelli-CAM Nested Based Machine.

  • Type - Part (Use this type for Point-to-Point machines or for your Secondary Operations on your Nested based Router)

    • P2P - Creates a new Point-to-Point Machine. Select this option to setup an Intelli-CAM Parts Machine.

    • Router - Creates a new Parts Machine for your Router.  Select this option to setup an Intelli-CAM Parts Machine.

  • Type - Specialized (Use this type for Drill & Dowel and Chop Saws)

    • Drill and Dowel - Creates a new Drill and Dowel Machine. Select this option to setup a CNC Drill and Dowel Inserter.

    • Chop Saw - Creates a new Chop Saw. Select this option to setup an Automated Chop Saw system.

    • Other - Creates machine types that don't fit into the other categories. At this time the only Machine that needs to be setup as this type is the Hoffmann/RazorGage Haunching System.

Multi-Drill Tab

This tab contains Multi-Drill head Setups for machines using Intelli-CAM for G-Code generation.

Simply select the Multi-Drill head to use with your machine. Make sure to look at the X & Y offsets for the tools from the Master (X=0/Y=0) spindle to verify orientation is the same as your Machine.

The Format button brings up options you can set for your multi-drill if you are using an ALPHACAM post that may not have been specifically written for S2M CENTER output.

There are six options here:

  • Prefix - Establishes the tool number prefix.
  • Delimeter - Establishes the delimeter for the tool numbers
  • Tool Number Offset - Sets the numeric offset for the tool numbers.
  • Format - Sets the numeric format for the tool numbers.

As you set the values for the four fields above you will see the results in the Results For 123 area.

  • Speed up routine by only looking at one axis (X or Y)
  • Hide ALPHACAM legacy multi-drill attributes

The Override Tool Properties check box, when enabled, will ignore your Feed Rate and Spindle Speed settings on your individual Drills utilizing instead the values entered here. When disabled Intelli-CAM will look at the individual drills assigned here for Feeds and Speeds.

To assign tools to the spindles simply click in the Tool field representing the appropriate spindle. A drop down box will appear listing the appropriate Tools for use in the spindle.

To view a layout and graphically assign bits for the spindles click the View button.

The Spindle that is active is highlighted with a red square. You can assign a bit to this Spindle by clicking the drop down at the lower left and select your Drill. To change to a different Spindle click the Next Spindle or Prev Spindle buttons.

To discontinue the viewing/editing of the Multi-drill head simply click OK to Save your changes.

Limitations Tab

This tab was put in specifically for those CNC Machines that have Boring bits in their Multi-Drill head that do not reach all areas of the Machine Table. When Limitations are activated Intelli-CAM looks to the Min X, Max X, Min Y, and Max Y fields for the Minimum and Maximum reach for the Master Spindle in the Multi-Drill head.

There are several other options on this Tab:

  1. Point of Origin - This option allows you to control where the Origin Point for your Machine is located.

  2. Nest Pack - This option allows you to control how parts are packed (placed) on the Nested Sheet.

  3. Material Length is X - When disabled the Material Length is placed along the Y-Axis.

  4. X Positive from Origin - When de-selected the X is calculated as a Negative value from the Origin point.

This option does not change the G-Code Output the Post will require modification to achieve a change in output results.

  1. Y Positive from Origin - When de-selected the Y is calculated as a Negative value from the Origin point.

This option does not change the G-Code Output the Post will require modification to achieve a change in output results.
