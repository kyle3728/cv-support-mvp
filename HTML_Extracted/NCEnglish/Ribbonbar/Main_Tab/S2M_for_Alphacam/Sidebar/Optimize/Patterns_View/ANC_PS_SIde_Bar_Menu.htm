Sidebar

Cut Patterns View

There are three Primary Areas in the sidebar:

Toolbar

There are 11 tools available in the main toolbar:

  • Return - Will return you to the previous sidebar.

  • Save Job State - Allows you to Save a Job State (SNC) File.

  • Print - Allows you to Print the Pattern Layouts.

  • Sheet View - Shows the Sheet stacks below.

  • Part View - Shows the Parts programs below.

  • Rebuild Sheet - Rebuilds the active Sheet Pattern.

  • First - Goes to the first Sheet in the selected stack.

  • Previous - Goes to the Previous Sheet in the selected stack.

  • Next - Goes to the Next Sheet in the selected stack.

  • Last - Goes to the Last Sheet in the selected stack.

  • Show 6th Face - Will show the 6th Face of your sheet if 6th Face Nesting is active and 6th Face operations exist.

Cut Patterns Area

Select the stacks in the list to view.

There are seven tools available in the Cut Patterns View toolbar:

  • Select A Part On The Current Sheet - This tool allows you to select a Part on the active Pattern in the Drawing Area.

  • Add Part (s) from Library - This tool allows you to add a Part from the Part Library to the active Pattern in the Drawing Area.

  • Cut Selected Part - Allows you to Cut the selected Part to the Clipboard for pasting on the active sheet or an alternate sheet.

  • Copy Selected Part - Allows you to Copy the selected Part to the Clipboard for pasting on the active sheet or an alternate sheet.

  • Paste Part - Paste the Part stored on the Clipboard to the active sheet.

  • Show Part List - Shows a Part List of all Parts on the active sheet.

  • Automatically Rebuild after Cut or Paste - When active this tool will automatically rebuild a sheet when a Part is Cut from or Pasted to the active sheet.
