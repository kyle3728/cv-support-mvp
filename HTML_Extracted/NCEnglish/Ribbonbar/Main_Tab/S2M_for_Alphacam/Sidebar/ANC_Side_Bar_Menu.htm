Sidebar

There are three Primary Areas in the S2M for ALPHACAM sidebar:

Toolbar

There are 8 options in the toolbar:

  1. Open a New PNC File - Allows you to open a previously exported PNC file. This will overwrite the currently open Job.

  2. Import a PNC File into Current Job - Allows you to add the contents of a previously exported PNC file into the current Job.

  3. Save Current Job as a PNC File - Allows you to Save the current Job as a PNC file for later use.

  4. Print - Allows printing of the Parts List.

  5. Edit Part Library - Allows you to manage your S2M CENTER Part Library.

  6. Show Par/Material List - Allows you to toggle on/off the Part List/Material List.

  7. Materials List - Displays the Material List for the Current Job.

  8. Parts List - Displays the Part List for the Current Job.

Main Buttons

There are 4 Main Buttons:

  1. Optimizer Parameters - Same as in the Standard S2M CENTER.

  1. Add Offcuts - Same as in the Standard S2M CENTER.

  1. Filter Parts - Same as in the Standard S2M CENTER.

  1. Optimize - Similar to the Standard S2M CENTER.

Machines

There 4 main Machine Types that are selectable in this area.

  1. Primary Machine

  2. Secondary Machine

  3. Drill and Dowel Machine

  4. Chop Saw
