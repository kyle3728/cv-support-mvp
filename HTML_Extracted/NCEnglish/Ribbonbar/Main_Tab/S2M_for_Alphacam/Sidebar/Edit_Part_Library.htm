Edit Part Library

Allows you to Edit Parts in your Part Library with ALPHACAM.

Edit Part Toolbar Tools

There are ten tools in the Edit Part toolbar:

  1. Return - Returns you to the main sidebar.

  2. Create New Category - Allows creation of a new Part Category in the Tree below.

  3. Copy Selected Item - Makes a copy of the selected Part/Category.

  4. Rename Selected Item - Allows you to Rename the selected Part/Category.

  5. Delete Selected Item - Allows you to Delete the selected Part/Category.

  6. Build - Each time you select a Part from the Parts List ALPHACAM draws (Builds) that Part on the screen. If you de-select the Build button it stops this which allows you to move around the Categories a bit quicker.

  7. Display In Metric - Displays your Parts in Metric.

ALPHACAM, unlike your Design/Manufacturing software, does not handle Unit of Measure changes on the fly. Therefore, when you Save a Part to the Part Library there are two Parts Saved. One Part is in Imperial Units and the Other is in Metric Units. When clicking the MM Tool it displays the parts "Alter Ego".

  1. Save Changes to Current Part - Allows changes made to the current ALPHACAM Part to be Saved to the Library overwriting the existing Part.

  2. Create Part Outline - Defines the extents of an ALPHACAM Part. This is required if you are going to save an Alphacam Part into the Part Library. This is also very helpful if you are going to utilize these Parts in your Design/Manufacturing software.

Depending upon your Parts Shape you will want to experiment with each of these Scan types to get the results you desire. Below are examples of each scan types results on the same Part.

As you can see in the preceding images the Radial or Left / Right scan option would be the best choice. As stated earlier it really depends on the shape of the Part the Outline is being generated for. Experiment!

  1. Add Part to Library - Adds the current Part to the Part Library.

Make sure that Parts you Add to the S2M CENTER from ALPHACAM are set to 0 as Top of Material and Depths are Negative Value equal to actual Depths for the Parts to be processed properly in the S2M Center. This must be setup this way regardless of whether you are set up in your Machine Catalog as "Mic at Spoilboard" or not.
