Customize Quick Access Toolbar

The QAT can be located in one of two places:

  • Above the Ribbon (default location)

  •  Below the Ribbon

If you don't want the QAT to be displayed in its current location, you can move it to the other location. If you find that the default location above the ribbon is too far from your work area to be convenient, you may want to move it closer to your work area. The location below the ribbon does encroach on the work area; therefore, if you want to maximize the work area, you may want to keep the QAT in its default location.

  1. Click Customize Quick Access Toolbar .

  2. In the list, click Show Below the Ribbon.

There are several other options available in the Customize Quick Access Toolbar (QAT) menu.

Any commands listed at the top with checkmark can be removed from the QAT by clicking the checkmark.

You can also use the QAT to Minimize the Ribbon. Or to quickly Minimize/Maximize the ribbonbar simply double click a ribbonbar tab.

More Commands...

The More Commands option allows you to Customize the commands available in both the QAT and the Ribbon.

The quickest method to add a command to the QAT is to simply right click that command in the Ribbon and select the Add to Quick Access Toolbar... option.

Quick Access Toolbar

From the sidebar you can select the Quick Access Toolbar option in order to modify the currently active QAT.

The list on the right represents the current QAT. You can select a command and move it up or down in the list to control its order. YOu can Remove a command from the QAT by selecting the <<Remove option.

On the left is where you can select from a list of available commands to add to the QAT using the Add>> option.

Select the Choose commands from: option.

There are four main options here:

Popular Commands - Popular commands.

Commands not in the Ribbon - Commands that are not in the ribbon.

All Commands - All commands are available here.

File - Commands from the File menu.

Depending upon which view you open from the remaining options will simply list the Ribbons available in the current View.

Customize Ribbon

From the sidebar you can select the Customize Ribbon option in order to modify the currently active Ribbons.

The list on the right represents the currently available Ribbon Tabs. You can select a group or command and move it up or down in the list to control its order. You can Remove a command from the QAT by selecting the <<Remove option. You can add new Tabs and Groups by clicking the New Tab and/or New Group buttons. You can rename a Tab or Group by clicking the Rename... button.

On the left is where you can select from a list of available commands to add to the Ribbon using the Add>> option.

Select the Choose commands from: option.

There are four main options here:

Popular Commands - Popular commands.

Commands not in the Ribbon - Commands that are not in the ribbon.

All Commands - All commands are available here.

File - Commands from the File menu.

Main Tabs - Depending upon which view you open from the remaining options will simply list the Ribbons available in the current View.

Keyboard Shortcuts

Clicking the Keyboard Shortcuts Customize... button gives you the ability to add Keyboards shortcuts to the commands in S2M CENTER.
