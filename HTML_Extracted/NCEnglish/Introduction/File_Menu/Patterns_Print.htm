Print

When viewing Patterns this option will allow printing of Pattern diagrams for your Machine Operator which includes a graphic representation of the Pattern, a list of Parts, Program Name and bar code for the Pattern (If a Nest).

With Nested output the print-out also indicates how the Material should be loaded on the Machine.

In the example below, the material being machined is a "one-sided" material with different "sides" (a face-dependent material which has different textures on the material back/unfinished face and material front/finished face). In this example, the nest CNC program machining face indicator is indicating that the material should be loaded "Particle Board" side down for proper nest CNC program machining. "Particle Board" is the name of a texture ascribed to only one of the two "sides" of the material "1/4 1s Melamine."

In the example below, the material being machined is a "one-sided" material with different "sides" (a face-dependent material which has different textures on the material back/unfinished face and material front/finished face). In this example, the nest CNC program machining face indicator is indicating that the material should be loaded "Particle Board" side up for proper nest CNC program machining. In the example below, the material being machined is a "two-sided" material with identical "sides" (a non-face-dependent material which has identical textures on both the material back/unfinished face and material front/finished face). As such, no nest CNC program machining face indicator is present since one is not required for proper nest CNC program machining.
