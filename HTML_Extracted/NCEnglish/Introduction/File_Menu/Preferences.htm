Preferences

There are 7 tabs in the Preferences screen:

General Tab

There are three sections at the General Tab:

  • CNC Programs - This area allows you to select the Maximum Run Number and Run Number Range to utilize for Program Names.

    • Run Number Formats

      • 3 Digit Numeric - With this number type the first 3 characters in the program name are nnn. This allows up to 1000 outputs without repeat. (000-999)

      • Alpha-Numeric - With this number type the first 3 characters in the program name are Rnn/Raa. This allows up to 1296 outputs without repeat. (R00-RZZ)

      • Standard - With this number type the first 3 characters in the program name are Rnn. This allows up to 100 outputs without a repeat. (R00-R99)

    • Run Number Range

      • This option will accommodate those shops that have multiple users outputting NC programs and want to assign allowing each user a range. Jeff = 0-99, Sally=100-199, John=200-299 etc. In addition the user can get a preview of the run number output by specifying the various number formats.

This feature is disabled if there is a preset run number (Cabnetware)

  • ALPHACAM - This area allows you to select which ALPHACAM Module is utilized by the S2M CENTER Ultimate.

  1. Label Position - The first option, Rotate labels to align with sheet length, when enabled, will make it so the Labels are always aligned to the Sheet Length. This allows the operator to stick the Labels in the same orientation each tie and the Part Graphic, if present, will rotate so operations further down stream are oriented properly. You can turn on an Annotation in your Pattern View denoting where the Label should be placed on the individual Parts. You can specify which side of a 6th face nest to place the Label on. You can also control the size of this Annotation. You have ten choices for where the Label is placed on Parts:

    • Automatic

    • Bottom Center

    • Bottom Left

    • Bottom Right

    • Center

    • Center Left

    • Center Right

    • Top Center

    • Top Left

    • Top Right

  • Show Parts Not Set for Optimization or Output - This option allows you to turn on the capability to control whether a Part set to "Not Optimize" or "Not CNC" can be overridden. Doing this will turn on Right Click menu options in the Part List. See the Parts topic for more information.

  • One Click Optimize To Pattern Results - Enable this option to go straight from clicking the Optimize button to the Pattern area.

  • Use Radial Menus - This feature changes the text style context (right click) menu to a graphical menu, similar to below.

  • Enable Side Bar Redraw Clipping - This feature turns on a feature to redraw the side bar more frequently. This feature should only need to be enabled in a VM environment where the buttons are not always visible in the side bar.

Tool Warnings Tab

This option allows you to turn off Warnings that may display during Optimization. Simply de-select the Check Boxes for the Warnings you do not want displayed.

Geometry

There are 10 options on the Geometry Tab:

  • Extend Dado Lengths by Tool Diameter - This option will extend any rectangle path Dadoes on your Parts by the Tool Diameter.

  • Extend Single Pass Dado Lengths by Tool Diameter -  This option will extend any single pass Dadoes on your Parts by the Tool Diameter.

  • Combine Edge Operations with Part Outline - This option will combine any Routes that cut through the Part and touch one of the Part edges together into a single Part Outline cut.

  • Combine Boring to Through Boring - This option will combine any Boring operations that originate from both the Front and Back face that meet or cross in depth and are the same diameter and in the exact same location. This can reduce the number of Secondary programs necessary to generate through bores.

  • Mill Operations from the Unfinished Face Where Possible - This option will machine through Operations from the Unfinished Face.

  • Only Route Outline of Through Holes - This option, if enabled, will only cut the Outline of Holes that go through the Material.

Under certain conditions Routing Holes with this option enabled can produce airborne off fall as the Tool leads out of the Hole.

  • Ignore Operations Outside of Part's Nest Boundary - This option, if enabled, will only machine Operations that fall within the Parts Outline. This does not include Operations which overlap the Outline to be combined into the Outline.

  • Close Grain Match Gaps Larger Than - By default the S2M Center orients Grain Matching exactly as passed by the Design Software. Let's say you had an Upper Cabinet above a Base and they were grouped together. If this is set to 0 then the space between the Upper and the Base would be included between the Doors in the Grain Match. You can set this to a value to determine when the S2M Center needs to close the Gap to the Nest Gap.

A Grain Match Gap that is smaller than the Kerf is ignored. 0 or Kerf x 2 are the typical values used here.

  • Pre-Mill Edgebanding - The S2M CENTER will compensate for Edgebanders with Pre-Mill Stations; by selecting this option you are indicating to the S2M Center that your Edgebander has a Pre-Mill Station. If Max is selected then the value is the maximum amount the banding machine will trim off the edge of the part during the banding operation.  Typically the banding machine trims off a value equal to the banding thickness. If Fixed is selected this is the amount the banding machine will trim off of the edge of the part that include banding.

Modifying the Max pre-mill edge banding will dynamically change the part size of banded parts.

What Edges can be Pre-Milled / Pre-Banded

Pre-Banded edges can only be the Left, Right, Top & Bottom of the part while it is still a rectangle.  Even if the part is shaped and banded on all edges only the L, R, T & Bottom edges will show the part size extended as pre-banded since the part would be banded prior to being placed on the P2P / Router to be shaped.

Pre-Milled edges can be any pre-banded edge as well as any other edge that could be fed into a banding machine. Edges that are excluded would be arcs or any edge that couldnt be fed into the machine due to a collision with another edge of the part.  The part will be resized with only valid pre-mill edges extended.  

Below are examples of parts that are banded on all edges with 1 thick banding with the pre-mill max set to ½ inch. In this example the pre-mill and pre-band edges are the same.  Note that none of the inside edges could be fed into a banding machine and therefore the part size was not extended for these edges.

In this example the pre-mill edge is valid for all 5 edges while the pre-band edge would NOT be valid for the angle.

This marks a change from 2011 R1 as all sides of both shapes would have been extended on a pre-band shape.

In this example the pre-mill edge is only valid for one edge. The edges adjacent to the radius have been excluded because if all edges were banded you would want a single piece of banding to span the radius and adjacent parts. Even if the radius was not banded, there is danger of the end trimmer cutting into the radius. For these reasons an edge adjacent to a radius is excluded.

This marks a change from Version 5 as all banded sides would have been extended on a pre-band shape.

  • Soft Corner Radius - Sets the Soft Corner radius for Doors set through MDF DoorMaker as having Soft Corners.

The maximum Radius allowed is 1/4" (6.35mm).

Paperless Traveler

There are three options here:

  • Customer ID - Enter your Hexagon Customer ID listed in the Customer Portal, on your License delivery email, or Order.

  • CV PT Token - Enter your CABINET VISION Paperless Traveller Token provided to you by the Nexus Connected Worker Team when they assisted you in setting up CABINET VISION Paperless Traveller.

  • Work Center Manager - This button opens up the Work Center Manager.

Enter the Number and Name of each of the Work Centers you setup in CABINET VISION Paperless Traveller here. Then select a Report from either CV or S2M to assign to each Work Center. It is not required to assign a report to a Work Center. At the bottom of this screen you will also want to specify which of your Assembly Sheets to use as well as identify which Work Center the Assembly Sheet is used in. You can also specify if you want one Assembly Sheet report per Assembly or if you want all Assemblies for a Work Center in a single Assembly Sheet report.

CNC

Allow Invalid Pre-Assigned Tools to be Substituted - The purpose of this preference is to ensure operations assigned an Invalid ToolID are still milled with an alternate tool.  It is possible to assign a tool to a specific operation and have a tool assigned to specific machines thus ensuring that certain operations are only performed on specific machines.  This option allows a fail-safe when a pre-assigned tool is invalid for a given operation. The S2M Center has a routine that validates a tool if it is pre-assigned. It will look to see if it is the proper tool type, size, etc. If ALL THIS CRITERIA IS PASSED, it then looks to see if the tool is available on the current machine. If not the operation is passed over for current machine, however, with this option on it would go into the Auto-Select routine to find a suitable alternate tool.

Miter Options

If you have Miters on your Parts coming from your manufacturing package this tab allows you to decide if you want the S2M Center to create Miter Operations and which tool to use to machine those miter options. If you have created a V Router Bit and have enabled the "Use V-Bit tool if angle matches miter" option then that tool will be used to machine the Miter. In certain scenarios such as a Toe Notch you may or may not want the Miter applied considering that it may show up inside the Assembly. Typically this allowed for with proper Deck/Bottom placement and if this is the case make sure to enable the "Allow miters on concave (unsafe) edges".

2 ½ Axis Miters are permitted on a nest, but restricted from 6th face nesting.  It was determined that the miters might damage the integrity of the sheet when it needs to be flipped.

4/5 Axis Miter operations are created on Workplanes and would require a properly configured multi-axis Tool be setup in the Tool Catalog and also a Machine and Machine Post which supports 4/5 axis movement..

Ordering

If you choose to order "By Cabinet", each sheet of Material will be restricted to contain Parts of only one Cabinet. For example, when Cabinet number 1 is completely cut out, you can begin assembly on it. If you choose to overlap Cabinets, you may control how many Cabinets may be on the same sheet.

Minimize 6th Face Patterns - Enabling this option will force 6th Face Operations to as few Sheets as possible, at the expense of Material Yield.

You also have the ability to control the order Operations are processed on your Nests and/or Parts. Simply select the Operation type and move it up or down the list. The first Operation in the list is the first operation type processed and so on.

Reports

Following are the main areas in the Reports Tab:

Part Report - Select which Report to use when Printing Secondary Part Sheets

Pattern Printouts

These options allow you to determine which Pattern Printout Reports to use by default.

Saw Optimization

These options allow you to select whether you want Offcuts and Gap Cuts (when Grain Matching) to be reported.

Measurement Units

The Measurement Units tab allows access to the area where your standard display unit of measure and its precision are set. Keep in mind that the S2M Center stores everything as a double integer in the Imperial (Inches) unit of measure. What you are selecting is how the S2M Center displays this stored information not how it is stored.

A double Integer consists of 15 unit places, meaning a measurement entered as 56mm is actually stored as 2.20472440944881 by the S2M Center. The measurement entered by you at 56mm would display as follows:

Units as Millimeters

Precision as 1 = 56

Precision as 1.5 = 56.0

Precision as 1.1 = 56.0

Precision as 1.01 = 56.00

Precision as 1.001 = 56.000

 

Units as Inches

Precision as 1 = 2

Precision as 1.1 = 2.2

Precision as 1.01 = 2.20

Precision as 1.001 = 2.205

Precision as 1.0001 = 2.2047

Precision as 1.00001 = 2.20472

Precision as 1.000001 = 2.204724

Precision as 1/8 = 2 1/4

Precision as 1/16 = 2 3/16

Precision as 1/32 = 2 7/32

Precision as 1/64 = 2 13/64

The important thing to remember is the S2M CENTER sees this number (56mm), no matter how you have the display units set, as 2.20472440944881.

The "Trim trailing decimal zeros" will simply trim any trailing zeroes off of displayed decimal measurements.

The "Allow unit-switch button on Status Bar" options turns on/off the Unit-switch button in the Statusbar Menu.

Tools Tab (Accessible from Part Edit area)

Smart-lines. In this section you can control the Snap Angle as well as the Tolerance setting of a drawn Line.

A Snap Angle of 45 and a high Tolerance (Sliding the bar all the way to the right) will snap lines only in 45 degree increments.

A Snap Angle of 45 and a low Tolerance (Sliding the bar all the way to the left) will basically disable the Snap feature. This will basically allow you to draw a free-hand line.

Snap Lines. In this section you can control the Color and Type (Dashed or Solid) of line that Dynamic Snapping will use.

Snapping Options. In this section you can control the Snap Hot-Keys as well as which Hot-Points are on for Dynamic Snapping.

Grid Snap - Snaps what you are currently placing to Grid points.

Point Snap - Snaps what you are currently placing to Snap points.

Short-cut Keys. In this section you can modify the default Short-cut Keys for quick commands inside Solid. You may want to change these if they interfere with Short-cut Keys you may already have setup in Windows. There are five functions that can be controlled by short-cut keys:

Pan - The Pan function allows you to set a new View center. When activated the Pan feature will center your current view to the location of the cursor on the screen.

Refresh - The Refresh function instructs Solid to redraw the current View.

Zoom Mode. In this section you control how the Windows for a Zoom Window is drawn.

From Corner - When using this and performing a Zoom Window you specify the corners of your Zoom area diagonally.

From Center - When using this and performing a Zoom Window you start from the Center of the View. This is the way Solid has normally performed a Zoom Window.

Factor - This slider sets the Zoom Factor for Scroll Wheel Zooming. The lower (left) the setting the smaller the Factor.

Reverse Scroll Wheel Zoom Direction - An option allowing you to reverse the Zoom In/Zoom Out direction on your Mouse Scroll Wheel.
