File Menu

New

+

When you click on New, the S2M CENTER will start the Job Properties area to allow you to define a new Job.

Open

+

Clicking on Open will bring up a Windows Open window that you may use to browse to and open a PNC or SNC Job file.

To open a PNC or SNC file, simply click on the PNC or SNC that you want to open and click the Open button.

When opening an SNC file you will be opened directly into the Pattern/Part View area. If you choose to click the Return option you will be returned to the Material/Part List view and will loose the Job State.

Import PNC

+

Import PNC allows you to build a single Job from multiple PNC Imports.

Export PNC

+

Creates a PNC file of the Open Job.

Properties

There are three tabs available in Properties:

Information

Allows you to enter Job Information that will be used in Reports.

Comment

Allows you to enter Comments for the current Job.

Schedule

Allows you to set Schedules for the current Job.

Print

+

This option allows you to Print the current View; the following window will appear:

Simply choose the printer that you wish to use and click OK to print the current view.

See the Print topic for more information.

Print Setup

+

This option allows you to set Printer options; the following window will appear:

Use this option to choose the settings for the printer that you plan to use with the S2M CENTER.
