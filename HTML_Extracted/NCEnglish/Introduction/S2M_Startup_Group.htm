S2M CENTER Start Group

In Windows 10 and 11 click the Windows Start Button then select All apps then scroll to and select the S2M CENTER 2023 Group.

In previous versions of Windows go to Start | All Programs | S2M CENTER 2023

There are five possible options in the Startup Group:

  1. +Label 2023 - Launches +Label. See the Using +Label in Production topic for more information.

  2. S2M CENTER 2023 - Launches the S2M CENTER.

  3. S2M CENTER 2023 Help - Opens this Help System.

  4. Update S2M CENTER 2023 Databases - Launches the Update Database Utility which compacts and repairs S2M CENTER Database Files.
