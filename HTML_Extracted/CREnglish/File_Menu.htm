File Menu

 Open

Clicking on Open will bring up the Open xRender File window that you may use to browse to and open a CVR (CABINET VISION Render) file. By default, Render Files are stored in the Documents folder. However, you may redirect xRender to store Jobs anywhere that you wish.

 

To open a Render file, simply click on the CVR file that you want to open and click the Open button.

 Close

Click Close will close the current xRender file.

 Save

Use this option to save the current Render. This updates the current Render file with any changes that have been made to it.

 Save As

This option gives you the opportunity to save the current Render with a new name. This allows you to save a new version of the Render under a different name so that the existing Render will not reflect the changes.

 

When the Save As window appears (as pictured below), simply type in the new name and click Save.

Export PNG

This option will allow you to save a PNG image file of the current view.

Exit

Use this option to Exit the xRender application.
